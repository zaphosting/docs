## مقدمة

CSTV تتيح لك إمكانية بث الألعاب مباشرة مع تأخير بسيط. هذا يعني أن الأشخاص الخارجيين الذين ليسوا جزءًا من اللعبة يمكنهم متابعة ما يحدث على السيرفر. تُستخدم هذه التقنية أيضًا في بطولات Counter-Strike الشهيرة حتى يتمكن المشاهدون من متابعة الحدث. بالإضافة إلى ذلك، يمكن حفظ البث تلقائيًا بحيث يمكنك مشاهدته مرة أخرى لاحقًا.

## التفعيل

يتم تفعيل خدمة CSTV عبر ملف التهيئة **server.cfg**. يجب إضافة الأوامر اللازمة هناك:

```
tv_enable 1
tv_delay 90
tv_advertise_watchable 1
```

## التهيئة

الأوامر أعلاه إلزامية لتفعيل سيرفر CSTV الرئيسي وجعله متاحًا. لكن هناك أوامر أخرى للتهيئة المتقدمة كما يلي:

|            الأمر            |                         الوصف                         |
| :--------------------------: | :----------------------------------------------------------: |
|  tv_allow_camera_man "0\|1"  |        يمكن للمشاهدين التحكم بالكاميرا        |
| tv_allow_static_shots "0\|1" |    استخدام كاميرات ثابتة الاتجاه للتصوير    |
|     tv_autorecord "0\|1"     | يسجل جميع المباريات تلقائيًا كملفات CSTV. يتم تسجيل كل المباريات تلقائيًا، وصيغة ملف التسجيل هي auto-YYYYYMMDD-hhmm-map.dem |
|     tv_autoretry "0\|1"      | يحاول إعادة الاتصال بعد انقطاع الشبكة |
| tv_chattimelimit "seconds"  | يقيّد المشاهدين بالكتابة كل x ثانية فقط |
|       tv_debug "0\|1"        |             عرض معلومات تصحيح CSTV             |
|   tv_delaymapchange "0\|1"   | يؤخر تغيير الخريطة حتى يكتمل البث |
|      tv_maxclients "n"       |          الحد الأقصى لعدد العملاء على سيرفر CSTV          |
|        tv_maxrate "n"        | الحد الأقصى لعرض النطاق المسموح به لمشاهدي CSTV، 0 == غير محدود |
|        tv_msg "text"         |           إرسال رسالة لجميع المشاهدين           |
|        tv_name "name"        | اسم المضيف لـ CSTV. يحدد اسم CSTV كما يظهر في متصفح السيرفر وشريط الحالة |
|       tv_nochat "0\|1"       |           تفعيل/تعطيل دردشة CSTV           |
|    tv_password "password"    |       حماية بث CSTV بكلمة مرور       |
|     tv_record "filename"     |             بدء تسجيل ملف CSTV             |
|           tv_stop            |                 إيقاف بث CSTV                 |
|        tv_stoprecord         |            إيقاف تسجيل ملف CSTV             |
|           tv_title           |           تحديد اسم بث CSTV           |

## إنشاء الاتصال

إذا كانت الوظيفة مفعلة على السيرفر، يمكن الوصول إليها عبر عنوان IP ومنفذ CSTV المخصص على السيرفر. للاتصال، تحتاج إلى الكونسول داخل اللعبة. هناك يمكنك الاتصال بالأمر التالي:

```
connect SERVERIP:27020
```

التسجيلات اليدوية والتلقائية تُخزن في الدليل الرئيسي /gxxxxx/CS:GO/game/csgo) لسيرفر CS:GO. يمكن تحميلها عبر وصول FTP المقدم. إذا لم تكن تعرف بعد كيفية الوصول عبر FTP، ستجد تعليمات هنا: [الوصول عبر FTP](gameserver-ftpaccess.md).

![](https://screensaver01.zap-hosting.com/index.php/s/w9b4Z7ECoSkSQdT/preview)

مع تشغيل التسجيل من داخل اللعبة، يمكن اختيار التسجيل المحمّل عبر **Load** وتشغيله عبر زر **Play**.

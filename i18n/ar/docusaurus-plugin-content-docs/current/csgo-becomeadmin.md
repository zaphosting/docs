## المقدمة
تعيين صلاحيات الأدمن يتيح لك إدارة سهلة وشاملة مع تحكم كامل في سيرفرك. كأدمن، لديك خيار استخدام كل الخيارات والوظائف المتاحة التي يوفرها اللعبة مباشرة داخل اللعب. كل الخطوات التي تحتاجها لتعيين صلاحيات الأدمن لسيرفرك سيتم شرحها أدناه.  
<InlineVoucher />

## التحضير
لاستخدام صلاحيات الأدمن، يجب أولاً تأسيس قاعدة. بشكل افتراضي، CS:GO لا توفر إدارة شاملة للأدمن. لكن هناك أُطُر مخصصة لهذا الغرض، مثل CounterStrikeSharp. يجب تثبيت الإطار للتمكن من الإعداد. إذا لم تقم بتثبيته بعد، لا تتردد في مراجعة [دليل تثبيت الإضافات](csgo-plugins) الخاص بنا.

## تعريف الأدمنز

إضافة أدمن تتم عبر ملف الإعداد `admins.json` الموجود في المسار `../CS:GO/game/csgo/addons/CounterStrikeShap/addons/configs/`. لتعريف أدمن جديد، يجب إنشاء إدخال جديد يحتوي على اسم، معرف (هوية)، وصلاحيات (flags). الاسم يمكنك تعيينه كما تريد لأنه يستخدم فقط كمعرّف/عنصر نائب. بالنسبة للمعرف "identity"، يجب إدخال SteamID الخاص بحساب Steam. بالإضافة لذلك، يجب تحديد الصلاحيات حسب الحاجة.

```js title="admins.json (مثال)"
{
  "User 1": {
    "identity": "76561198808392634",
    "flags": ["@css/changemap", "@css/generic"]
  },
  "User 2": {
    "identity": "STEAM_0:1:1",
    "flags": ["@css/generic"]
  }
}
```

## تعريف مجموعات الأدمنز
يمكنك أيضاً تعريف مجموعات الأدمنز. بهذه الطريقة، يمكن تحديد صلاحيات الأدمن عبر مجموعته المرتبطة. هكذا، لا تحتاج لتكوين الصلاحيات لكل أدمن بشكل فردي. المجموعات تُعرف في ملف الإعداد admin_groups.json الموجود في المسار `../CS:GO/game/csgo/addons/CounterStrikeShap/addons/configs/`.
```js title="admin_groups.json"
"#css/simple-admin": {
  "flags": [
    "@css/generic",
    "@css/reservation",
    "@css/ban",
    "@css/slay",
  ]
}
```

## تعريف مناعة الأدمن
يمكن تعيين قيمة مناعة للأدمنز. إذا حاول أدمن أو لاعب بقيمة مناعة أقل مهاجمة أدمن أو لاعب بقيمة مناعة أعلى، فإن الأمر سيفشل. يمكنك تحديد قيمة المناعة بإضافة المفتاح immunity لكل أدمن في ملف configs/admins.json.

```
{
  "ZoNiCaL": {
    "identity": "76561198808392634",
    "flags": ["@css/changemap", "@css/generic"],
    "immunity": 100
  }
}
```
:::info
فحص المناعة لا يتم تلقائياً بواسطة CounterStrikeSharp. الأمر متروك للإضافات الفردية لتولي إدارة وفحص المناعة.
:::

## المقدمة
تعيين صلاحيات الأدمن يتيح لك إدارة سهلة وشاملة مع تحكم كامل في سيرفرك. كأدمن، لديك خيار استخدام كل الخيارات والوظائف المتاحة من Sourcemod مباشرة داخل اللعبة. كل الخطوات التي تحتاجها لتعيين صلاحيات الأدمن لسيرفرك سيتم شرحها أدناه.  
<InlineVoucher />

## التحضير

لاستخدام صلاحيات الأدمن، يجب أولاً تأسيس الأساس. بشكل افتراضي، CS:GO لا يوفر إدارة شاملة للأدمن. لكن هناك أُطر عمل لهذا الغرض، مثل Sourcemod. يجب تثبيت الإطار لتتمكن من التهيئة. إذا لم تقم بتثبيته بعد، لا تتردد في الاطلاع على دليل تثبيت الإضافات الخاص بنا لـ CS:GO.

## تعريف الأدمن

يمكن إضافة الأدمن عبر ملف التهيئة `admins_simple.ini` أو `admins.cfg` الموجود تحت `../csgo/cstrike/addons/sourcemod/configs/`. للإعدادات البسيطة، عادةً ما يكون `admins_simple.ini` كافي. هناك يمكنك تعريف الأدمن بواسطة SteamID وتعيين صلاحياته مباشرة.

```ini title="admins_simple.ini (مثال)"
"STEAM_0:1:12345678" "z"
"STEAM_0:1:87654321" "bcdefgjk"
```

في هذا المثال، الأدمن الأول يحصل على صلاحيات كاملة عبر العلم `z`، بينما الأدمن الثاني يحصل على صلاحيات محددة فقط.

إذا كنت تريد استخدام إعدادات متقدمة مثل المجموعات، المناعة، أو كلمات المرور، يجب عليك استخدام `admins.cfg` بدلاً من ذلك.

```
Admins
{
  "User 1"
  {
    "auth"    "steam"
    "identity" "STEAM_0:1:12345678"
    "flags"   "z"
  }

  "User 2"
  {
    "auth"    "steam"
    "identity" "STEAM_0:1:87654321"
    "flags"   "bcdefgjk"
  }
}
```

## تعريف مجموعات الأدمن

يمكنك أيضًا تعريف مجموعات الأدمن. بهذه الطريقة، يمكن إدارة صلاحيات الأدمن عبر مجموعته المرتبطة. هكذا، لا تحتاج لتكوين الصلاحيات لكل أدمن بشكل فردي. المجموعات تُعرف في ملف التهيئة `admin_groups.cfg` الموجود تحت `../csgo/cstrike/addons/sourcemod/configs/`.

```
Groups
{
  "Full Admin"
  {
    "flags"   "z"
    "immunity" "99"
  }

  "Moderator"
  {
    "flags"   "bcdefgjk"
    "immunity" "10"
  }
}
```

بعدها، يمكنك تعيين مجموعة لأدمن في ملف `admins.cfg`:

```
Admins
{
  "User 1"
  {
    "auth"     "steam"
    "identity" "STEAM_0:1:12345678"
    "group"    "Full Admin"
  }
}
```

## تعريف مناعة الأدمن

يمكن تعيين قيمة مناعة للأدمن أو للمجموعات. إذا حاول أدمن بقيمة مناعة أقل استهداف أدمن بقيمة مناعة أعلى، ستفشل العملية. يمكنك تعريف قيمة المناعة بإضافة المفتاح `immunity` لكل أدمن في `admins.cfg` أو مباشرة في `admin_groups.cfg`.

```
Admins
{
  "ZoNiCaL"
  {
    "auth"      "steam"
    "identity"  "STEAM_0:1:12345678"
    "flags"     "z"
    "immunity"  "100"
  }
}
```

قيمة مناعة أعلى تعني مستوى حماية أعلى ضد الإجراءات الإدارية من أدمن آخرين ذوي مناعة أقل.

## الخاتمة

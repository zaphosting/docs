## مقدمة

إعداد عناوين IP على سيرفرك المخصص بنظام ويندوز يتيح لك التحكم في كيفية تواصل السيرفر داخل الشبكة. هذا يشمل تعيين عناوين IP، تحديد البوابة الافتراضية، وإعداد خوادم DNS المطلوبة للاتصال الشبكي.

الإعداد الصحيح لعناوين IP مهم للسيرفرات التي تشغل خدمات مثل سيرفرات ألعاب، سيرفرات ويب، قواعد بيانات، أو أنظمة الوصول عن بُعد، لأنه يضمن تواصل شبكي مستقر وسهولة وصول موثوقة.

:::warning السيرفر غير متاح بسبب فقدان الاتصال بالإنترنت؟
إذا أصبح سيرفرك غير متاح بسبب إعداد شبكة خاطئ أو فقدان الاتصال بالإنترنت، قد لا يعمل الوصول عن بُعد عبر RDP. استخدم [وحدة تحكم iLO HTML](dedicated-ilo.md#the-html5-console) للاتصال بسيرفرك وإصلاح إعدادات الشبكة.
:::

## التحضير

قبل إعداد عناوين IP، تأكد من توفر المعلومات التالية:

- عنوان IP الذي تريد تعيينه
- قناع الشبكة الفرعية (Subnet mask)
- البوابة الافتراضية (Default gateway)
- خوادم DNS

يمكنك العثور على هذه المعلومات في واجهة الويب. افتح **إدارة السيرفر المخصص** وانتقل إلى **عناوين IP**.

## الإعداد

اتصل بسيرفرك المخصص باستخدام **سطح المكتب البعيد (RDP)**. بعد الاتصال، افتح **لوحة التحكم**. انتقل إلى **لوحة التحكم → الشبكة والإنترنت → مركز الشبكة والمشاركة**.

انقر على **تغيير إعدادات المحول** على الجانب الأيسر. سترى واجهات الشبكة المتاحة. انقر بزر الفأرة الأيمن على محول الشبكة النشط (عادةً **Ethernet**) واختر **خصائص**. في القائمة، اختر **بروتوكول الإنترنت الإصدار 4 (TCP/IPv4)** ثم اضغط **خصائص**.

![img](https://screensaver01.zap-hosting.com/index.php/s/3EpqZXg5pgWHoLK/download)

اختر **استخدام عنوان IP التالي** وأدخل القيم المطلوبة:

- **عنوان IP** → عنوان IP المعين لك
- **قناع الشبكة الفرعية** → قناع الشبكة الفرعية الخاص بك
- **البوابة الافتراضية** → بوابتك
- **خوادم DNS →** 1.1.1.1، 8.8.8.8

ثم قم بإعداد خوادم DNS باختيار **استخدام عناوين خوادم DNS التالية** وأدخل خوادم DNS المفضلة لديك. بعد إدخال كل القيم، أكد الإعداد بالضغط على **موافق** وأغلق النوافذ المتبقية.



## التحقق

للتحقق من الإعداد، افتح **موجه الأوامر** ونفذ الأمر:
```
ipconfig
```
![img](https://screensaver01.zap-hosting.com/index.php/s/AK8F3jppqL8xnsx/download)

يعرض هذا الأمر جميع واجهات الشبكة وإعدادات IP الحالية لها. ابحث عن محول الشبكة النشط وتأكد أن **عنوان IPv4** يطابق عنوان IP الذي قمت بإعداده سابقًا. إذا ظهر عنوان IP الصحيح، فهذا يعني أن الإعداد تم تطبيقه بنجاح.





## الخاتمة

مبروك! لقد قمت بإعداد إعدادات IP بنجاح على سيرفرك المخصص بنظام ويندوز. إذا كانت لديك أي أسئلة إضافية أو تحتاج مساعدة، فريق الدعم لدينا متوفر يوميًا لمساعدتك.

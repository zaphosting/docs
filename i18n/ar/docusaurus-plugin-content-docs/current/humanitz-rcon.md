## مقدمة

RCON (وحدة التحكم عن بُعد) هو بروتوكول شبكي يُستخدم للتحكم عن بُعد في سيرفرات الألعاب. يسمح للمسؤولين بتنفيذ الأوامر على السيرفر دون الحاجة للاتصال باللعبة مباشرة. هذا يجعل من الممكن إدارة اللاعبين، تعديل الإعدادات، أو مراقبة نشاط السيرفر من الخارج.

في **HumanitZ**، يمكن استخدام RCON لتنفيذ أوامر إدارية مثل طرد اللاعبين، إرسال الرسائل، أو التحكم بسلوك السيرفر. يتم تأمين الوصول باستخدام كلمة مرور ومنفذ مخصص.

الميزة الأساسية لـ RCON هي أنه يسمح بإدارة السيرفر **دون الحاجة للاتصال كلاعب داخل اللعبة**، مما يتيح إدارة مرنة وفعالة عن بُعد.

## الإعداد

قبل استخدام RCON، يجب تفعيله في إعدادات السيرفر. افتح صفحة إعدادات السيرفر وقم بتفعيل خيار rcon. يمكنك أيضًا تعديله من خلال ملفات الإعداد. افتح لوحة التحكم الخاصة بسيرفر الألعاب، انتقل إلى Configs، وابحث عن ملف `GameServerSettings.ini`.

أضف أو عدّل الإدخالات التالية:

```
RCONEnabled=true
RCONPassword=your_secure_password
RCONPort=XXXXX
```

يمكنك العثور على منفذ RCON المخصص في **نظرة عامة على المنافذ** داخل إدارة السيرفر. تأكد من أن كلمة المرور والمنفذ متطابقان مع إعداداتك.

بعد إجراء التعديلات، احفظ الملف و**أعد تشغيل السيرفر** لتطبيق الإعدادات.

## الاتصال عبر RCON

للاتصال بسيرفر HumanitZ الخاص بك عبر RCON، يمكنك استخدام أداة سطر الأوامر **rcon-cli**. متوفرة في المستودع الرسمي على GitHub:
https://github.com/gorcon/rcon-cli

بعد التثبيت، يمكنك الاتصال بالسيرفر باستخدام الأمر التالي:

```bash
rcon-cli -a <IP>:<PORT> -p <PASSWORD> command
```

استبدل القيم بـ عنوان IP الخاص بسيرفرك، منفذ RCON، وكلمة المرور.

يتيح لك هذا الأمر تنفيذ أوامر RCON مباشرة من جهازك المحلي دون الحاجة لجلسة تفاعلية.

## أوامر RCON

بعد الاتصال عبر RCON، يمكنك تنفيذ أوامر إدارية مختلفة على سيرفر HumanitZ الخاص بك. الأوامر المتاحة تعتمد على تنفيذ اللعبة لكنها عادة تشمل إدارة اللاعبين والتحكم بالسيرفر.

| الأمر | الوصف |
|----------------------------------|------------------------------------------|
| `broadcast <message>`| يرسل رسالة لجميع اللاعبين |
| `kick <playername>`| يطرد لاعب من السيرفر |
| `ban <playername>` | يمنع لاعب من السيرفر |
| `unban <playername>` | يرفع الحظر عن لاعب |
| `save` | يحفظ حالة اللعبة الحالية |
| `restart`| يعيد تشغيل السيرفر |
| `shutdown` | يغلق السيرفر |
| `whitelist add <steamid>`| يضيف لاعب إلى القائمة البيضاء |
| `whitelist remove <steamid>` | يزيل لاعب من القائمة البيضاء |

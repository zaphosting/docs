## تفعيل القائمة البيضاء

القائمة البيضاء في Unturned تتيح لك حماية سيرفرك عن طريق السماح فقط للاعبين المختارين الموجودين في القائمة بالاتصال بالسيرفر. تفعيل القائمة البيضاء يتم داخل اللعبة عبر أمر. هذا يتطلب **صلاحيات المالك**. إذا لم تكن تعرف ما هي هذه الصلاحيات وكيف تعطيها، تفقد الدليل التالي: [تصبح أدمن](unturned-becomeadmin.md)

إذا اتبعت الدليل وأصبحت أدمن بنجاح، يمكنك الآن تفعيل القائمة البيضاء على سيرفرك. للقيام بذلك، افتح الدردشة/الكونسول داخل اللعبة على سيرفرك. هناك يجب تنفيذ الأمر التالي:

```
/Whitelisted
```

يجب أن تكون القائمة البيضاء مفعلة الآن، وبالتالي تسمح فقط للاعبين الموجودين في القائمة بالانضمام للسيرفر. يمكنك إضافة أو إزالة اللاعبين باستخدام الأوامر التالية.

## إدارة القائمة البيضاء

**إضافة لاعب إلى القائمة البيضاء:**

```
Permit [SteamID]/[Tag]
```

**إزالة لاعب من القائمة البيضاء:**

```
Unpermit [SteamID]
```

**عرض جميع اللاعبين المسموح لهم:**

```
Permits
```

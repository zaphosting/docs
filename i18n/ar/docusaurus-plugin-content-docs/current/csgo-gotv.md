## مقدمة

GOTV تتيح لك إمكانية بث الألعاب مباشرة مع تأخير بسيط. هذا يعني أن الأشخاص الخارجيين الذين ليسوا جزءًا من اللعبة يمكنهم متابعة ما يحدث على السيرفر. تُستخدم هذه التقنية أيضًا في بطولات Counter-Strike الشهيرة حتى يتمكن المشاهدون من متابعة الحدث. بالإضافة إلى ذلك، يمكن حفظ البث تلقائيًا بحيث يمكنك مشاهدته مرة أخرى لاحقًا.

## التفعيل

يتم تفعيل خدمة GOTV عبر ملف التهيئة **server.cfg**. يجب إضافة الأوامر اللازمة هناك:

```
tv_enable 1
tv_delay 90
tv_advertise_watchable 1
```

## التهيئة

الأوامر أعلاه إلزامية لتفعيل خادم GOTV الرئيسي وجعله متاحًا. ومع ذلك، هناك أوامر أخرى للتهيئة المتقدمة كما يلي:

|            الأمر            |                         الوصف                         |
| :--------------------------: | :----------------------------------------------------------: |
|  tv_allow_camera_man "0\|1"  |        يمكن للمشاهدين التحكم بالكاميرا ككاميرمان        |
| tv_allow_static_shots "0\|1" |    استخدام كاميرات ثابتة الاتجاه للتصوير    |
|     tv_autorecord "0\|1"     | يسجل جميع الألعاب تلقائيًا كعروض GOTV. يتم التسجيل بصيغة auto-YYYYYMMDD-hhmm-map.dem |
|     tv_autoretry "0\|1"      | يحاول إعادة الاتصال بعد انقطاع الشبكة |
| tv_chattimelimit "seconds"  | يقيّد المشاهدين بالكتابة كل x ثانية |
|       tv_debug "0\|1"        |             عرض معلومات تصحيح GOTV             |
|   tv_delaymapchange "0\|1"   | يؤخر تغيير الخريطة حتى يكتمل البث |
|      tv_maxclients "n"       |          الحد الأقصى لعدد العملاء على سيرفر GOTV          |
|        tv_maxrate "n"        | الحد الأقصى لعرض النطاق المسموح به لمشاهدي GOTV، 0 == غير محدود |
|        tv_msg "text"         |           إرسال رسالة لجميع المشاهدين           |
|        tv_name "name"        | اسم مضيف GOTV. يحدد اسم GOTV كما يظهر في متصفح السيرفر وشريط الحالة |
|       tv_nochat "0\|1"       |           تفعيل/تعطيل دردشة GOTV           |
|    tv_password "password"    |       حماية بث GOTV بكلمة مرور       |
|     tv_record "filename"     |             بدء تسجيل عرض GOTV             |
|           tv_stop            |                 إيقاف بث GOTV                 |
|        tv_stoprecord         |            إيقاف تسجيل عرض GOTV             |
|           tv_title           |           تحديد اسم بث GOTV           |

## إنشاء الاتصال

إذا تم تفعيل الوظيفة على السيرفر، يمكن الوصول إليها عبر عنوان IP ومنفذ GOTV المخصص له على السيرفر. للاتصال، تحتاج إلى الكونسول داخل اللعبة. هناك يمكنك الاتصال بالأمر التالي:

```
connect SERVERIP:27020
```

العروض المسجلة يدويًا وتلقائيًا تُخزن في الدليل الرئيسي /gxxxxx/CS:GO/game/csgo) لسيرفر CS:GO. يمكن تنزيلها عبر وصول FTP المقدم. إذا لم تكن تعرف بعد كيفية الوصول عبر FTP، ستجد تعليمات لذلك هنا: [الوصول عبر FTP](gameserver-ftpaccess.md).

![](https://screensaver01.zap-hosting.com/index.php/s/w9b4Z7ECoSkSQdT/preview)

مع تشغيل العرض من داخل اللعبة، يمكن اختيار العرض المحمّل عبر **Load** وبدء التشغيل عبر زر **Play**.

## مقدمة

[Cashlytics](https://github.com/aaronjoeldev/cashlytics-ai) هو **منصة حديثة لإدارة التمويل الشخصي والميزانية** مصممة لمساعدتك على تتبع الإنفاق، إدارة الميزانيات، والحصول على رؤى حول عاداتك المالية. من خلال تحليل بيانات المعاملات وفئات الإنفاق، يساعدك Cashlytics على فهم أين تذهب أموالك وكيف يمكنك تحسين تخطيطك المالي.

يمكن للمنصة أيضًا دمج **مساعدة مدعومة بالذكاء الاصطناعي** بشكل اختياري، مما يتيح لك تحليل البيانات المالية، التعرف على أنماط الإنفاق، والحصول على رؤى ذكية حول سلوكك في الميزانية.

![img](https://screensaver01.zap-hosting.com/index.php/s/W46gYzqdiPrBDsE/preview)

استضافة Cashlytics بنفسك على سيرفرك تعطيك **تحكم كامل في بياناتك المالية** مع إبقاء أدوات الميزانية متاحة من أي مكان عبر واجهة ويب. تخطط لاستضافة Cashlytics بنفسك؟ هذا الدليل سيرشدك خلال عملية التثبيت باستخدام Docker على VPS Linux الخاص بك.

## حالات استخدام Cashlytics

يمكن استخدام Cashlytics في مجموعة متنوعة من السيناريوهات الشخصية والمهنية حيث التنظيم المالي والميزانية مهمان. تشمل حالات الاستخدام النموذجية:

- تتبع المصاريف الشخصية والميزانيات الشهرية
- مراقبة أنماط الإنفاق والعادات المالية
- إدارة مصادر دخل متعددة وفئات مالية مختلفة
- توليد رؤى مدعومة بالذكاء الاصطناعي لميزانية أذكى
- استضافة لوحة مالية خاصة وتحت سيطرتك الكاملة

بما أنه مستضاف ذاتيًا، فإن Cashlytics مثالي للمستخدمين الذين يريدون بديل يركز على الخصوصية بدلًا من الأدوات المالية السحابية.

## المتطلبات الأساسية

قبل تثبيت Cashlytics، تأكد من أن VPS الخاص بك يلبي المتطلبات التالية.

| العتاد | الحد الأدنى | الموصى به |
|----------|---------|-------------|
| CPU | نواة واحدة | نواتان |
| RAM | 2 جيجابايت | 4 جيجابايت |
| مساحة القرص | 10 جيجابايت | 20 جيجابايت |



بالإضافة إلى ذلك، يجب تثبيت البرامج التالية:

- Docker
- Docker Compose

إذا لم يكن Docker مثبتًا بعد، يرجى اتباع دليلنا لتثبيت [Docker](dedicated-linux-docker.md) قبل المتابعة.

## التحضير

أولًا، أنشئ مجلد مشروع لتثبيت Cashlytics.

```bash
mkdir cashlytics && cd cashlytics
```

بعدها قم بتحميل ملفات تكوين Docker الرسمية لـ Cashlytics.

```
curl -O https://raw.githubusercontent.com/aaronjoeldev/cashlytics-ai/main/docker-compose.selfhost.yml
curl -O https://raw.githubusercontent.com/aaronjoeldev/cashlytics-ai/main/.env.example
```

أنشئ ملف تكوين البيئة الخاص بك بنسخ الملف النموذجي.

```
cp .env.example .env
```

يحتوي ملف `.env` على متغيرات التكوين المطلوبة لتشغيل Cashlytics.



## التثبيت



### افتح ملف `.env`

افتح ملف `.env` وقم بضبط متغيرات البيئة المطلوبة.

```
nano .env
```



### ضبط كلمة مرور قاعدة البيانات

حدد كلمة مرور آمنة لقاعدة بيانات PostgreSQL التي يستخدمها Cashlytics.

```
POSTGRES_PASSWORD=your_secure_password_here
```



### ضبط اتصال قاعدة البيانات

حدّث سلسلة اتصال قاعدة البيانات واستخدم نفس كلمة المرور التي حددتها أعلاه.

```
DATABASE_URL=postgresql://cashlytics:your_secure_password_here@postgres:5432/cashlytics
```



### مساعد الذكاء الاصطناعي الاختياري

يدعم Cashlytics مساعدًا اختياريًا بالذكاء الاصطناعي يمكنه تحليل البيانات المالية وتقديم رؤى للميزانية. لتمكين هذه الميزة، أضف مفتاح API الخاص بـ OpenAI.

```
OPENAI_API_KEY=sk-your-openai-key
```

إذا لم ترغب في استخدام المساعد الذكي، يمكن ترك هذا المتغير فارغًا.



## بدء تشغيل Cashlytics

بعد إكمال التكوين، ابدأ التطبيق باستخدام Docker Compose.

```
docker compose -f docker-compose.selfhost.yml up -d
```

سيقوم Docker الآن بتنزيل صور الحاويات المطلوبة وتشغيل خدمات Cashlytics. للتحقق من أن الحاويات تعمل، يمكنك استخدام:

```
docker ps
```



## الوصول إلى Cashlytics

بعد بدء الحاويات بنجاح، يمكنك الوصول إلى واجهة الويب الخاصة بـ Cashlytics. افتح متصفحك وادخل العنوان التالي:

```
http://YOUR_SERVER_IP:3000
```

استبدل YOUR_SERVER_IP بعنوان IP الخاص بـ VPS. بمجرد تحميل الواجهة، يمكنك البدء في إعداد بياناتك المالية، الميزانيات، والفئات.

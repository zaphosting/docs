## مقدمة

Joplin هو تطبيق مفتوح المصدر لتدوين الملاحظات والمهام يتيح تنظيم الملاحظات المبنية على Markdown في دفاتر ملاحظات قابلة للبحث ومزامنتها عبر الأجهزة. يوفر تشفير من النهاية إلى النهاية، أداة قص صفحات الويب، ومزامنة عبر منصات متعددة. مثالي للمستخدمين الذين يفضلون الخصوصية، المرونة، والتحكم الكامل في بياناتهم!

## التكوين

بعد تثبيت Joplin، لازم تضبط الإعدادات الأساسية لتناسب طريقة عملك. التكوين متاح من خلال قائمة *الإعدادات*.

**المزامنة**  
يدعم Joplin عدة وجهات مزامنة مثل Nextcloud، Dropbox، OneDrive، أو WebDAV. اختار المزود المفضل عندك تحت “المزامنة” وأدخل بيانات الدخول. هذا يسمح لك بتحديث الملاحظات عبر أجهزتك.

**التشفير**  
لتأمين ملاحظاتك المتزامنة، فعّل التشفير من النهاية إلى النهاية في إعدادات “التشفير”. سيتم إنشاء مفتاح لازم تضبطه على كل الأجهزة اللي توصل لملاحظاتك.

**المحرر والمظهر**  
Joplin يستخدم Markdown للملاحظات. في إعدادات “المحرر” تقدر تحدد إذا كان المعاينة تظهر تلقائيًا وتضبط نوع وحجم الخط.

**الإضافات والملحقات**  
مدير الإضافات المدمج يتيح لك تثبيت ميزات إضافية، مثل دعم الرسوم البيانية، تكامل التقويم، أو إدارة متقدمة للوسوم.

**أداة قص صفحات الويب**  
اختياريًا، فعّل إضافة “Joplin Web Clipper” للمتصفح لحفظ صفحات الويب كاملة أو أجزاء مختارة مباشرة كملاحظات.

بهذه الإعدادات الأساسية، Joplin جاهز للاستخدام في سيناريوهات متعددة مثل إدارة المهام، توثيق المشاريع، أو قواعد المعرفة الشخصية.

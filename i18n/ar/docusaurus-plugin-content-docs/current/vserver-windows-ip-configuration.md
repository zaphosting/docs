## مقدمة

إعداد عناوين IP على VPS ويندوز الخاص بك يتيح لك التحكم في كيفية تواصل سيرفرك داخل الشبكة. هذا يشمل تعيين عناوين IP، تحديد البوابة الافتراضية، وإعداد خوادم DNS المطلوبة للاتصال الشبكي.

الإعداد الصحيح لعناوين IP مهم للسيرفرات التي تشغل خدمات مثل سيرفرات ألعاب، سيرفرات ويب، قواعد بيانات، أو أنظمة الوصول عن بُعد، لأنه يضمن تواصل شبكي مستقر وسهولة وصول موثوقة.

:::warning السيرفر غير متاح بسبب فقدان الاتصال بالإنترنت؟
إذا أصبح سيرفرك غير متاح بسبب إعداد شبكة خاطئ أو فقدان الاتصال بالإنترنت، قد لا يعمل الوصول عن بُعد عبر RDP. استخدم [وحدة تحكم VNC](vserver-vnc.md) في واجهة الويب للاتصال بسيرفرك وإصلاح إعدادات الشبكة.
:::

## التحضير

قبل إعداد عناوين IP، تأكد من توفر المعلومات التالية:

- عنوان IP الذي تريد تعيينه
- قناع الشبكة الفرعية (Subnet mask)
- البوابة الافتراضية (Default gateway)
- خوادم DNS

يمكنك العثور على هذه المعلومات في واجهة الويب. افتح **إدارة VPS** وانتقل إلى **عناوين IP**.

## الإعداد

اتصل بـ VPS الخاص بك باستخدام **سطح المكتب البعيد (RDP)**. بعد الاتصال، افتح **لوحة التحكم**. اذهب إلى **لوحة التحكم → الشبكة والإنترنت → مركز الشبكة والمشاركة**.

اضغط على **تغيير إعدادات المحول** على الجانب الأيسر. سترى واجهات الشبكة المتاحة. انقر بزر الفأرة الأيمن على محول الشبكة النشط (عادةً **Ethernet**) واختر **خصائص**. في القائمة، اختر **بروتوكول الإنترنت الإصدار 4 (TCP/IPv4)** واضغط على **خصائص**.

![img](https://screensaver01.zap-hosting.com/index.php/s/3EpqZXg5pgWHoLK/download)

اختر **استخدام عنوان IP التالي** وأدخل القيم المطلوبة:

- **عنوان IP** → عنوان IP المعين لك
- **قناع الشبكة الفرعية** → قناع الشبكة الفرعية الخاص بك
- **البوابة الافتراضية** → بوابتك
- **خوادم DNS →** 1.1.1.1، 8.8.8.8

ثم قم بإعداد خوادم DNS باختيار **استخدام عناوين خوادم DNS التالية** وأدخل خوادم DNS المفضلة لديك. بعد إدخال كل القيم، أكد الإعداد بالضغط على **موافق** وأغلق النوافذ المتبقية.



## التحقق

للتحقق من الإعداد، افتح **موجه الأوامر** وشغل الأمر:
```
ipconfig
```
![img](https://screensaver01.zap-hosting.com/index.php/s/AK8F3jppqL8xnsx/download)

يعرض هذا الأمر جميع واجهات الشبكة وإعدادات IP الحالية لها. ابحث عن محول الشبكة النشط وتأكد أن **عنوان IPv4** يطابق عنوان IP الذي قمت بإعداده سابقًا. إذا ظهر عنوان IP الصحيح، فهذا يعني أن الإعداد تم بنجاح.





## الخلاصة

مبروك! لقد قمت بإعداد إعدادات IP على VPS ويندوز الخاص بك بنجاح. إذا كان لديك أي أسئلة إضافية أو تحتاج مساعدة، فريق الدعم متوفر يوميًا لمساعدتك.

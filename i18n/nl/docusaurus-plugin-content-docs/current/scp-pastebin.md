## Introductie

Een serverbeschrijving is nodig voor de werking van een SCP-server. De implementatie van de gamebeschrijving is gebaseerd op het gebruik van Pastebin. [Pastebin](https://pastebin.com/) is een website waar je teksten online kunt opslaan.

<InlineVoucher />



## Maak een Pastebin aan

Voeg de beschrijving voor je server toe op de Pastebin-pagina in het veld onder **New Paste** en klik daarna onderaan op **Create New Paste**.

:::warning
Zorg ervoor dat je de pastebin niet op privÃ© zet en geen vervaldatum instelt om mogelijke problemen te voorkomen.
:::

![img](https://screensaver01.zap-hosting.com/index.php/s/5jJkBgkNzQT9fym/download)



Als je deze stappen hebt gevolgd en de pastebin hebt aangemaakt, word je doorgestuurd naar de pastebin die je hebt gemaakt. Het ziet er zo uit:

![img](https://screensaver01.zap-hosting.com/index.php/s/RtFrbdGBbowj9Pr/preview)





## Serverbeschrijving toevoegen

Zodra de pastebin is aangemaakt, moet je de pastebin ID invoeren in de serverconfiguratie. Kopieer hiervoor de ID uit de pastebin-URL en voeg deze toe in het tekstveld voor server info pastebin info in de **game server administratie** onder **Instellingen**.

![img](https://screensaver01.zap-hosting.com/index.php/s/gQBjFC6qfwgSXCT/download)

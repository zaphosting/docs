## Vereisten

Extra software is niet nodig voor het standaard gebruik van een Valheim vanilla server. Maar als je hebt gekozen voor de Valheim Plus of BepInEx variant om plugins te gebruiken, moet je deze extensie ook lokaal installeren. Check hier hoe je deze extensies lokaal installeert voor je game:

## Servergegevens ophalen

Je hebt alle relevante info nodig om verbinding te maken met je server. Deze info vind je op het dashboard van het webinterface paneel van je game server. Belangrijke info is onder andere:

- IP-adres en poort
- Servernaam
- Serverwachtwoord (optioneel)

Met deze info kun je je server vinden en zeker weten dat je met de juiste server verbindt.

## Servergegevens ophalen


Je hebt alle relevante info nodig om verbinding te maken met je server. Deze info vind je op het dashboard van het webinterfacepaneel van je game server. Belangrijke gegevens zijn onder andere:

- IP-adres en poort
- Servernaam
- Serverwachtwoord (optioneel)


Met bovenstaande info kun je je server vinden en zeker weten dat je met de juiste server verbindt.

<Tabs>
    <TabItem value="connect_solution_server_browser_ingame" label="Serverbrowser (In-game)" default>

Om via de in-game serverbrowser verbinding te maken met je server, open je de serverbrowser in het hoofdmenu. Klik hiervoor op **Multiplayer** en selecteer daarna **Browse Servers**. Je kunt het **zoekveld** gebruiken om je server te vinden in de serverlijst via de servernaam en vervolgens verbinding maken met de **Join** knop. 

![img](https://screensaver01.zap-hosting.com/index.php/s/8gXjXqijtWiXJoZ/download)

</TabItem>



<TabItem value="connect_solution_direct" label="Direct verbinden (In-game)" default>

In plaats van zoeken via de serverlijst, kun je ook direct verbinden met het IP-adres en de poort. Klik hiervoor op **Multiplayer** en kies **Join via IP**. Er opent een venster waar je je servergegevens invult in het IP/Port veld. Daarnaast moet je een naam voor de server invullen. Klik daarna op **Add server** om de server toe te voegen en verbinding te maken. 

![img](https://screensaver01.zap-hosting.com/index.php/s/sAiEYyaBbRKTACA/download)

</TabItem>

#### Niet zichtbaar


Als je server niet zichtbaar is, kan dat komen doordat de initialisatie niet goed is afgerond. Dit kan bijvoorbeeld door een verkeerde configuratie of corrupte bestanden. Meestal vind je meer info in de serverconsole of logbestanden.

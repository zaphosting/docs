## Servergegevens ophalen


Je hebt alle relevante info nodig om verbinding te maken met je server. Die info vind je op het dashboard van het webinterfacepaneel van je game server. Belangrijke gegevens zijn onder andere:

- IP-adres en poort
- Servernaam
- Serverwachtwoord (optioneel)


Met deze info kun je je server vinden en zeker weten dat je met de juiste server verbindt.

![img](https://screensaver01.zap-hosting.com/index.php/s/HnKL48icqNQsj2p/preview)

## Verbinding maken met de server


Er zijn vaak meerdere manieren om verbinding te maken met een game server. Hieronder leggen we de beschikbare methodes duidelijk uit.

<Tabs>

<TabItem value="connect_solution_server_browser_ingame" label="Serverbrowser (In-game)" default>

Om via de in-game serverbrowser verbinding te maken, open je de serverbrowser in het hoofdmenu. Zoek daar je server op in de **serverlijst** via de servernaam. Selecteer je server en klik op **Join**.

![img](https://screensaver01.zap-hosting.com/index.php/s/nnsaZwcWzNDGswo/download)

</TabItem>

<TabItem value="connect_solution_direct_ingame" label="Direct verbinden (In-game)">

Om direct verbinding te maken via de in-game serverbrowser, open je de serverbrowser in het hoofdmenu. Klik onderaan op **Direct Join** en vul het IP-adres en de poort van je server in. Bevestig dit en maak verbinding via de **Join**-knop.

![img](https://screensaver01.zap-hosting.com/index.php/s/inJDnmdzc4FwDz4/download)

</TabItem>

</Tabs>



## Mogelijke problemen en oplossingen


Er zijn vaak meerdere manieren om verbinding te maken met een game server. Hieronder vind je de beschikbare methodes en uitleg.

#### Niet zichtbaar


Als je server niet zichtbaar is, kan dat komen doordat de initialisatie niet goed is afgerond. Dit kan bijvoorbeeld door een verkeerde configuratie of corrupte bestanden. Meestal vind je meer info in de serverconsole of logbestanden.



#### Geen oplossing gevonden of niet geholpen


Heb je na het proberen van de tips nog steeds problemen? Geen zorgen, ons supportteam staat dagelijks voor je klaar! Maak gewoon een **[Ticket](https://zap-hosting.com/en/customer/support/)** aan op onze website en leg je probleem zo gedetailleerd mogelijk uit. Wij checken het zo snel mogelijk en helpen je het probleem op te lossen!

## Introductie
Last Oasis biedt de optie om meerdere Oases tegelijk te verbinden binnen hetzelfde Realm. In deze gids laten we zien hoe je deze modus activeert voor jouw eigen Realm.

<InlineVoucher />

## Voorbereiding
Ga eerst naar [de officiële Last Oasis](https://myrealm.lastoasis.gg/) website en log in met je favoriete platform (Steam of Xbox).

![](https://screensaver01.zap-hosting.com/index.php/s/d6xZsqYbEF9jSj8/preview)

## Multi hosting inschakelen
Op de website ga je naar de **Realms** sectie. Klik bij jouw huidige Realm op de **Wijzigen** knop en schakel de hostingmodus om van **Single -> Multiple**.

![](https://screensaver01.zap-hosting.com/index.php/s/qDgwQ5D6YwptWER/preview)

:::info
Om meerdere Oases tegelijk te kunnen draaien, moet je per Oasis een aparte game server huren. Gebruik voor elke nieuwe server dezelfde provider key als voor de originele server. Vergeet niet je server te herstarten na het doorvoeren van wijzigingen.
:::

Na deze wijziging kun je de Oases beheren via de **Beheren** knop in deze sectie.

## Nieuwe Oasis aanmaken
Kies een gewenste locatie voor de nieuwe Oasis en klik op **Toevoegen** om te bevestigen.

![](https://screensaver01.zap-hosting.com/index.php/s/A2GLkeBWaBQr6m9/preview)

Er verschijnt een nieuw venster waarin je een naam kunt instellen en een map kunt selecteren.

![](https://screensaver01.zap-hosting.com/index.php/s/6SkCFyAzooKwQAA/preview)

![](https://screensaver01.zap-hosting.com/index.php/s/CBFHBq8TxAxogk9/preview)

Als alles klopt, klik je op **Oasis toevoegen** om de Oasis te creëren en activeer je deze met de **Activeren** knop.

![](https://screensaver01.zap-hosting.com/index.php/s/yoeHTdeAeXneC2q/preview)

Je ziet nu meerdere actieve Oases tegelijk in de **Oases** sectie. Gefeliciteerd, je hebt succesvol meerdere Oases ingesteld voor jouw Realm.

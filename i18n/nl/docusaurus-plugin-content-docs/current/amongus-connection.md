## Voorbereiding - De Download

Om verbinding te maken met je server, hebben we een bestand nodig dat de connectie mogelijk maakt.  
Open eerst [Impostor](https://impostor.github.io/Impostor/) waar we onze gegevens gaan invullen.

Om het bestand aan te maken, hebben we info nodig zoals het IP-adres, de poort en een naam.  
![](https://screensaver01.zap-hosting.com/index.php/s/bQ7KzTAJcbAsLJd/preview)

Vul hier de gegevens in vanuit je ZAP-Hosting interface, dat ziet er bijvoorbeeld zo uit:

![](https://screensaver01.zap-hosting.com/index.php/s/7scfddD5WPFr25b/preview)

Waarbij 31.214.245.215 jouw IP is en 2000 de poort.

De servernaam mag je zelf kiezen. Als alle gegevens ingevuld zijn, klik je op "Download Server File".  
Er wordt nu een bestand gedownload met de naam "regioninfo.json".

Druk daarna op `Win` + `R` en typ daar in:

```
%APPDATA%\..\LocalLow\Innersloth\Among Us
```

Bevestig met Enter, er opent een map. Verplaats het bestand regioninfo.json hierin en overschrijf het bestaande bestand.

![](https://screensaver01.zap-hosting.com/index.php/s/KTrfij6ww64DPCq/preview)


## Server "ingame" aanmaken

Start Among Us op je PC.  
Klik in het hoofdmenu op "online" en daarna op "Create Game".

![](https://screensaver01.zap-hosting.com/index.php/s/SfMwr82R9BbDK2P/preview)

Hier kun je nu de gewenste instellingen voor je server maken:

![](https://screensaver01.zap-hosting.com/index.php/s/imzCR9WZenyrTSX/preview)

Als je klaar bent met de instellingen, klik je op "confirm".  
Je bent nu verbonden met je server!

In de lobby kun je instellen of het een publieke of private server is.  
Dit verander je door te klikken op het icoon "Private" of "Public":

Als de server op "Public" staat, kunnen andere spelers op dezelfde manier verbinding maken.  
Deze spelers hoeven in het hoofdmenu niet op "Create Game" te klikken, maar op "Find Game".  
Belangrijk is dat de chat op "others" staat en dat de filters exact overeenkomen met de serverinstellingen.  
Anders wordt de server niet weergegeven.

Is het een "Private" server, dan moet je in het hoofdmenu op "Enter Code" klikken.  
Hier vul je de code in die de maker van de server in de lobby kan zien.

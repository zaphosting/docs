## Introductie
Je wilt niet dat iedereen zomaar op je server kan? Er zijn verschillende manieren om de zichtbaarheid en toegankelijkheid van een server te beperken. EÃ©n van die opties is bijvoorbeeld het gebruik van een whitelist, wat we hieronder uitleggen.

De whitelist is een lijst waarmee je een bepaalde selectie mensen toegang kunt geven. Dit betekent dat alleen de mensen die op de whitelist staan kunnen verbinden. Mensen die niet op de whitelist staan worden geweigerd en krijgen een bericht dat ze niet op de whitelist staan wanneer ze verbinding proberen te maken.

## Configuratie

Om de whitelist-functie voor FiveM te beheren, moet je de txAdmin-interface openen. Ga dan naar het menu-item **Settings** en open het tabblad **Whitelist**.

![img](https://screensaver01.zap-hosting.com/index.php/s/WJBH63dWtyWWYQ6/download)

Selecteer het type whitelist-modus dat je actief wilt gebruiken. De volgende opties zijn beschikbaar in de administratie:

- Alleen admin (onderhoudsmodus)
- Discord Server Lid
- Discord Server Rollen
- Goedgekeurde License

In dit voorbeeld wordt de optie **Goedgekeurde License** geselecteerd en geactiveerd, waarmee je spelers kunt goedkeuren via hun license.

![img](https://screensaver01.zap-hosting.com/index.php/s/dzTZLTczjWjorkf/download)

De speler license Identifier moet door een txAdmin administrator op de whitelist worden gezet. Dit kan via de **Whitelist-pagina**, of met het `/whitelist` commando.

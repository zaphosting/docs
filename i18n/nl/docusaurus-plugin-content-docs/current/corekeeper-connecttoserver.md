### Zo maak je verbinding met de Core Keeper game server:

Open je ZAP-Hosting game server dashboard en check of je server aan staat en draait:

![](https://screensaver01.zap-hosting.com/index.php/s/2NbQaA6E2CJ4ZE8/preview)

Start het spel en kies in het menu voor "Multiplayer": 

![385-351-max](https://screensaver01.zap-hosting.com/index.php/s/t36H8NxmLa4gx5S/preview)

Selecteer daarna de optie "Direct join": 

![](https://screensaver01.zap-hosting.com/index.php/s/3MdnG7Xb2222EPQ/preview)

Voer vervolgens het IP-adres en de poort in die je ziet op je game server dashboard:

![](https://screensaver01.zap-hosting.com/index.php/s/R9nRFWFrS9Fbwox/preview)

Klik op "Connect" en de client maakt verbinding met je server.

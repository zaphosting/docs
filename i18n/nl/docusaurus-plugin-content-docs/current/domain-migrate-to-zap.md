## Introductie

Ben je nieuw en geïnteresseerd in de producten van ZAP-Hosting? Dan ben je hier aan het juiste adres. Allereerst, welkom bij ZAP-Hosting! We zijn blij dat je interesse hebt in onze diensten en wilt weten hoe je je bestaande domein van een andere provider naar ons kunt overzetten. Deze gids vertelt je welke algemene stappen je moet nemen om je domein succesvol naar ons te migreren.

## Voordelen

Je staat vast voor de keuze welke provider het beste bij je past. Zo’n beslissing kan soms best wat tijd kosten. Daarom willen we je helpen met je keuze en de voordelen van ZAP-Hosting met je delen. ZAP-Hosting biedt al meer dan tien jaar zijn diensten aan en is daarmee een gevestigde en ervaren speler in de serverhostingwereld.

Onze beheerinterface maakt het beheren van je domein makkelijk en overzichtelijk. Daarnaast kun je met één klik subdomeinen aanmaken voor andere ZAP-producten.

Heb je voor ons gekozen? Dan kun je direct starten met het overzetten van je bestaande domein naar ZAP-Hosting. Dit proces bestaat uit drie stappen: voorbereiding, waarbij je de auth code aanvraagt; overdracht, waarbij het domein geregistreerd en naar ons overgezet wordt; en de noodzakelijke configuratie-aanpassingen om alles daarna goed te laten werken.

## Voorbereiding

Voor het overzetten van een al geregistreerd domein heb je een **Auth Code** (ook wel autorisatiecode genoemd) nodig. Dit is een soort wachtwoord voor domeinen. De auth code is vereist wanneer een domein van de ene provider naar de andere wordt overgezet. Deze code zorgt ervoor dat de domeinoverdracht echt wordt aangevraagd door de rechtmatige eigenaar van het domein.

Sommige providers geven de auth code al direct in hun beheerinterface vrij. Is dat niet het geval, dan moet je je huidige provider informeren over je wens om het domein over te zetten en de auth code opvragen.

:::warning Geldigheidsduur

De auth code is maar een beperkte tijd geldig. Na die periode moet je een nieuwe auth code aanvragen. De geldigheidsduur hangt af van de betreffende domeinextensie.
:::

## Overdracht

Als je je auth code hebt, kun je nu je domein bestellen en registreren bij ZAP-Hosting. Ga naar de bestelpagina voor onze [**domeinen**](https://zap-hosting.com/en/shop/product/domain/). Vul eerst je persoonlijke gegevens in en zoek daarna naar het domein dat je wilt overzetten.

![](https://screensaver01.zap-hosting.com/index.php/s/3dmY76dZscz9DPM/preview)

:::info
Om een domein over te zetten, moet je het tijdens het bestelproces met 1 jaar verlengen. Dit is standaardbeleid bij alle providers.
:::

Er verschijnt een lijst met je gewenste domein en alle beschikbare domeinextensies. Domeinen die al elders geregistreerd zijn, herken je aan de grijze knop met het pijltje.

![](https://screensaver01.zap-hosting.com/index.php/s/omnaMqXJgarxsqW/preview)

Zoek het domein dat je wilt overzetten en klik op het pijltje. Er opent dan een nieuw venster waarin je de overdracht naar je ZAP-Hosting account autoriseert.

![](https://screensaver01.zap-hosting.com/index.php/s/fXjwGCX7kFtPnTB/preview)

In dit veld vul je de **Auth Code** in die je in de voorbereidingsstap van je vorige provider hebt gekregen. Klaar? Klik dan op de groene **Toevoegen** knop.

:::danger Controleer je invoer goed

Zorg ervoor dat de **Auth Code** correct en geldig is ingevuld. Anders duurt het bestelproces en de domeinoverdracht langer en kan het mislukken.
::: 

De domeinoverdracht is nu gestart. Hoe lang dit duurt, hangt af van de domeinextensie en de vorige provider. Meestal is een domeinoverdracht binnen 72 uur afgerond. In speciale gevallen kan het langer duren, omdat er veel factoren meespelen.

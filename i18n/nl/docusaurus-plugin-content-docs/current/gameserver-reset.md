## Intro

Je server werkt niet meer zoals het hoort of wil je helemaal opnieuw beginnen maar weet je niet hoe? Geen stress, je kunt je server resetten door hem opnieuw te installeren. In deze gids leggen we uit hoe je dat doet en waar je op moet letten.

## Server resetten

De optie om je server te resetten (herinstalleren) vind je in de game server administratie onder de instellingenpagina. Daar zie je twee knoppen: **Reset all options** en **Reinstall server**, die je moet uitvoeren om je server volledig te resetten.

![img](https://screensaver01.zap-hosting.com/index.php/s/H8Y9Tz8imigEBDZ/download)

:::info
Tijdens de herinstallatie kun je kiezen of je ook de database wilt resetten. Wil je dat niet, dan moet je deze optie uitvinken.
:::

:::danger
Als er nog belangrijke data op de server staat, maak dan eerst een backup voordat je verder gaat, anders raak je die kwijt.
:::

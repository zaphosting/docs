## Voorbereiding

Allereerst is het belangrijk om te checken welke voorwaarden er zijn om Ã¼berhaupt verbinding te kunnen maken met de server. Denk aan het spel zelf, extra content (mods/extensions) en de algemene serverinfo zoals IP-adres, poort en servernaam.



### Spel / Content

Om verbinding te maken met je game server is het verplicht dat je het spel al bezit. Dit wordt niet door ons meegeleverd bij het huren van een server. Verder is er geen extra content nodig.



### Serverinformatie

Als aan alle voorwaarden van **Spel / Content** is voldaan, heb je de info van je server nodig. Klik in je dashboard op je game server service. Je krijgt nu een overzicht met alle algemene info over je server.

![](https://screensaver01.zap-hosting.com/index.php/s/YpiLt9Byz8A292y/preview)

De belangrijkste info is het **IP-adres** en de **Poort**. Zorg dat je deze onthoudt, want die heb je nodig voor je eerste verbinding met je server.





## Verbinden met de server

Als aan alle voorwaarden is voldaan, kun je beginnen met verbinden. Start je game en ga naar het tabblad **Servers**. Klik daar op Direct Connect:

![](https://screensaver01.zap-hosting.com/index.php/s/58L9WNxqydmHDfP/preview)

Voer het IP-adres en de poort van je server in. Klik daarna op de **Connect** knop om verbinding te maken:



![](https://screensaver01.zap-hosting.com/index.php/s/sQN2nPwTnTbZ8qj/preview)

### Niet zichtbaar

Als je server niet zichtbaar is, kan dat komen doordat de server niet goed is opgestart. Dit kan bijvoorbeeld door een verkeerde configuratie, foutieve mods/plugins of corrupte bestanden. Meestal vind je meer info in de serverconsole of logbestanden.

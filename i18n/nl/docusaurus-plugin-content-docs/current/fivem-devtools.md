## Gebruiksscenario's

Het activeren en gebruiken van de dev tools geeft je uitgebreide debug-informatie. Dit biedt je een overzicht en info voor de volgende toepassingen:

- Resource Monitoring
- Event Logger
- Profiler
- Netwerkdiagnostiek
- Console & Debug Output

## Voorbereiding

Om de FiveM dev tools te kunnen gebruiken, moet het `Beta` kanaal geselecteerd en geactiveerd zijn in de FiveM client-instellingen onder Game. Na het aanpassen hiervan moet je de client opnieuw opstarten.

![img](https://screensaver01.zap-hosting.com/index.php/s/FKfXXYAMEF39n25/download)

## Dev tools gebruiken

Je opent de Dev Tools door de console te openen met de `F8` toets. Bovenaan in de console-navigatie vind je dan de sectie Tools, waar je de meest gebruikte tools terugvindt.

![img](https://screensaver01.zap-hosting.com/index.php/s/E5szziipJre6X7Y/preview)

:::warning Ontbreken tool-opties?
Zie je niet alle tool-opties zoals in de screenshot? Dan heb je waarschijnlijk het kanaal niet correct ingesteld zoals in de voorbereiding beschreven, of ben je vergeten de client opnieuw te starten.
:::

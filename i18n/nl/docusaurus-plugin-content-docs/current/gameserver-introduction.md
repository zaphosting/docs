## Configuraties

We bieden verschillende configuratie-opties voor onze gameservers, zodat er voor elke situatie iets passends is. Onze configuraties bestaan meestal uit de volgende opties:

| Configuratie                   | Keuzeoptie |
| ----------------------------- | ---------- |
| Slots                         | Afhankelijk van wat het spel ondersteunt |
| Memory Boosts (extra geheugen) | Tot 32 GB extra RAM bovenop het standaard toegewezen geheugen |
| Extra schijfruimte            | Tot 200 GB extra opslagruimte bovenop de standaard 50 GB |
| CPU & Host Server             | Standaard (SSD, 2.0-3.4 GHz, DDR3/4 geheugen) of Premium (M.2 SSD, Gaming CPU 3.4-4.4GHz, DDR4 geheugen) |
| Eigen IPv4-adres              | Eigen IPv4-adres / geen eigen IPv4-adres |

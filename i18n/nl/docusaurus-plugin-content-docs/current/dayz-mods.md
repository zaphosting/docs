## Intro

Mods geven je de mogelijkheid om je server nog meer te customizen. Hieronder leggen we uit waar je de populairste servermods kunt vinden en hoe je ze installeert.

<InlineVoucher />

## Voorbereiding

Via de [Steam Workshop](https://steamcommunity.com/app/221100/workshop/) kun je mods op je server installeren. De Steam Workshop is perfect om nieuwe en toffe mods te ontdekken en te beheren. Begin met het browsen door de Steam Workshop op zoek naar mods die je aanspreken. Als je een mod vindt die je tof vindt, is het belangrijk om het bijbehorende Workshop ID te noteren. Dit ID vind je in de URL. Dit Workshop ID heb je nodig voor de installatie.

## Installatie

De volgende stap is het installeren van de mods. Dankzij de Steam Workshop optie is dit super makkelijk. Ga naar het beheer van je game server en navigeer naar Steam Workshop. Daar kun je de mods invoeren in het daarvoor bestemde veld. Vul de Workshop IDs in die je eerder hebt genoteerd en voeg ze toe. Als je meerdere mods toevoegt, zorg dan dat je ze scheidt met een komma.

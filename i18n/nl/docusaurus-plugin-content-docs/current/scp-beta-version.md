## Introductie

Wil je de beta versie van **SCP: Secret Laboratory** spelen en vraag je je af hoe je dat doet? We leggen uit wat je moet weten en hoe je de versie succesvol installeert op zowel je game als je server!

<InlineVoucher />


## Game

Om de beta versie voor je game te gebruiken, moet je in je Steam-bibliotheek de eigenschappen van de game openen en vervolgens het tabblad **Betas** selecteren. Kies in het dropdown-menu onder Beta deelname de optie **Beta versie** in plaats van **Geen**.

![img](https://screensaver01.zap-hosting.com/index.php/s/BSn8E3D7JBgM6Dy/preview)



## Server

De beta versie van de SCP Secret Laboratory Server wordt aangeboden als een aparte game. Dit betekent dat je het spel moet wijzigen als je deze versie gebruikt. De optie hiervoor vind je in de game server administratie onder Games. Als je hier hulp bij nodig hebt, raden we je aan ook onze [Game wijzigen](gameserver-gameswitch.md) gids te bekijken.



![img](https://screensaver01.zap-hosting.com/index.php/s/BZwaxoknbmFKCJB/preview)



:::warning Plugin Support
Het gebruik van de Exiled extensie voor plugins wordt niet ondersteund voor de beta versie.
:::

:::danger Game wijzigen (SCP: EXILED naar SCP: BETA)
Het wijzigen van het spel van **SCP: Exiled** naar **SCP: Beta** vereist een wijziging van het host-systeem vanwege de andere voorwaarden. Tijdens dit proces worden bestaande data verwijderd en verandert ook het IP-adres als je geen eigen IP hebt geboekt.
:::

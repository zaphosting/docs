## Tebex koppelen

Tebex instellen met ZAP is super simpel. De installatie en configuratie vragen maar een paar stappen, die we hieronder wat uitgebreider uitleggen.

### Kies een ondersteunde game

Kies een game die door Tebex wordt ondersteund en voeg ‚Äòm toe aan je gameserver. Klik op je gameserver en kies `Games` in de sidebar. Kies een ondersteunde game onder `Available Games` en klik op de downloadknop om ‚Äòm te installeren.

### Vraag een Tebex invite aan

Als de game die je koos Tebex ondersteunt, zie je een tab ‚ÄúTebex Shop‚Äù in de sidebar. Klik op de afbeelding die verschijnt zodra de pagina geladen is.

![](https://screensaver01.zap-hosting.com/index.php/s/SztAWeeNX8fx5kF/preview)

Tebex stuurt nu een invite naar het e-mailadres dat gekoppeld is aan je ZAP-Hosting account. Volg de instructies in de mail om je Tebex account te activeren. Je wordt door de basisconfiguratie van je shop geleid.

### Installeer de Tebex plugin

Dit proces is meestal geautomatiseerd. Als we geen plugin voor automatische setup op onze servers hebben of als de pingback van Tebex mist, moet je de setup handmatig doen.

### Configureer de Tebex plugin

Installeer de juiste gameserver plugin voor jouw game handmatig. De setup verschilt per game. Basisinstructies voor het installeren van de plugins vind je [hier](https://docs.tebex.io/store/integrating-with-your-game-server-or-website/minecraft-java-edition).

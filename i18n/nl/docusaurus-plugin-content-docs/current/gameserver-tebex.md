## Tebex koppelen

Het instellen van Tebex met ZAP is super simpel. De installatie en configuratie vragen maar een paar stappen, die hieronder wat uitgebreider worden uitgelegd. 

### Kies ondersteunde game

Kies een game die door Tebex wordt ondersteund en voeg deze toe aan je gameserver. Klik op je gameserver en kies `Games` in de zijbalk. Kies een ondersteunde game onder `Beschikbare Games` en klik op de downloadknop om het te installeren.

### Vraag Tebex uitnodiging aan

Als de game die je hebt gekozen Tebex ondersteunt, zie je een tabblad genaamd "Tebex Shop" in de zijbalk. Klik op de afbeelding die verschijnt zodra de pagina geladen is.

![](https://screensaver01.zap-hosting.com/index.php/s/SztAWeeNX8fx5kF/preview)

Tebex stuurt nu een uitnodiging naar het e-mailadres dat gekoppeld is aan je ZAP-Hosting klantaccount. Volg de instructies in de mail om je Tebex-account te activeren. Je wordt begeleid bij de basisconfiguratie van je shop.

### Installeer de Tebex plugin

Dit proces is meestal geautomatiseerd. Als we geen plugin voor automatische setup op onze servers hebben of als we de pingback van Tebex missen, moet je de setup handmatig doen.

### Configureer de Tebex plugin

Installeer de juiste gameserver-plugin voor jouw game handmatig. Het installatieproces verschilt per game. Basisinstructies voor het installeren van de plugins vind je [hier](https://docs.tebex.io/store/integrating-with-your-game-server-or-website/minecraft-java-edition).

## Verifieer verbinding met Tebex

Tot slot moet je bevestigen dat je de gameserver-plugin succesvol hebt geïnstalleerd.

![](https://screensaver01.zap-hosting.com/index.php/s/DkDtPMtij2QTKGS/preview)

In sommige gevallen moet je je Tebex webwinkel plugin API-sleutel invoeren om de verbinding van ZAP naar je Tebex store tot stand te brengen. Je ziet dan dat de verbinding gelukt is:
![](https://screensaver01.zap-hosting.com/index.php/s/4YS2SwHfSTgn4DE/preview)

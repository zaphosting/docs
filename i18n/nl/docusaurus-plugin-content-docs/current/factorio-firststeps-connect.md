## Servergegevens ophalen


Je hebt alle relevante info nodig om verbinding te maken met je server. Deze gegevens vind je op het dashboard van het webinterfacepaneel van je game server. Belangrijke info is onder andere:

- IP-adres en poort
- Servernaam
- Serverwachtwoord (optioneel)


Met bovenstaande info kun je je server vinden en zeker weten dat je met de juiste server verbindt.

#### Niet zichtbaar


Als je server niet zichtbaar is, kan dat komen doordat de initialisatie niet goed is afgerond. Dit kan bijvoorbeeld door een verkeerde configuratie of corrupte bestanden. Meer info vind je meestal terug in de serverconsole of logbestanden.

## Introductie
Het toewijzen van administratorrechten geeft je eenvoudige en uitgebreide controle over je server. Als admin kun je alle beschikbare opties en functies van het spel direct in de game gebruiken. Hieronder leggen we stap voor stap uit hoe je administratorrechten voor jouw server instelt.  
<InlineVoucher />

## Stel het admin wachtwoord in  
Je kunt het admin wachtwoord bekijken en aanpassen in de interface van je server onder het tabblad "Instellingen".  
Je vindt de instellingen hier:

![](https://screensaver01.zap-hosting.com/index.php/s/3yB6Zym5LKc8FPr/preview)

<InlineVoucher />

## Verbind met de server  
Verbind nu met je server.  
Open Assetto Corsa Competizione op je PC.  
Ga vervolgens naar "Multiplayer".  
Klik rechtsonder op "Serverlist".  
In de zoekbalk kun je de naam van je server invoeren, zodat je niet hoeft te zoeken.

LET OP: Het kan ongeveer 30 minuten duren voordat de server in de lijst verschijnt.

## Word admin  
Als je verbonden bent met de server, druk je op ENTER om iets in de chat te typen.  
Typ hier "/admin ADMINPW".  
Nu ben je admin op je server!

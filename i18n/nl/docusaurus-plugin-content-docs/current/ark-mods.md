## Mods installeren

Mods installeren op ARK-servers kan via de Steam Workshop. De mod-community van ARK is een van de actiefste in het survival-genre. Meer dan 6000 mods zijn te downloaden via de Steam Workshop, en het aantal groeit elke dag. Hieronder leggen we uit hoe je deze mods aan je server toevoegt.



![](https://screensaver01.zap-hosting.com/index.php/s/rPT5rKaKtbxgJnr/preview)

## Mods zoeken

Eerst moet je uitzoeken welke mods je op je server wilt installeren. Hiervoor open je de Steam Workshop voor ARK. Die vind je hier:

https://steamcommunity.com/app/346110/workshop/



Let op: niet alle items die je ziet zijn servermods. Zorg er dus voor dat je alleen servermods toevoegt aan je server.



## Mods toevoegen

Heb je een mod gekozen? Dan moet je de ID van het workshop-item van die mod kopiëren en toevoegen in de interface. Afhankelijk van of je de Steam Workshop via je browser of de Steam Client bekijkt, kopieer je de ID als volgt:

- **Steam Client**

  Rechtsklik op de Steam Workshop-pagina van de mod en kies 'Kopieer paginalink'. Kopieer de ID van de mod uit de URL.

  ![](https://screensaver01.zap-hosting.com/index.php/s/tQAcSrYYBSRAZEf/preview)

  

- **Browser**

  In de URL zie je een reeks cijfers achter **.../?id=**. Kopieer die.

  ![](https://screensaver01.zap-hosting.com/index.php/s/HzZjRF7eb2maE6a/preview)



Deze voeg je vervolgens toe in de interface onder Steam Workshop. Wil je meerdere mods gebruiken? Scheid de ID’s dan met een komma:



![](https://screensaver01.zap-hosting.com/index.php/s/fc7piqsTJQoABeL/preview)



Na een herstart van je game server worden de mods automatisch gedownload en correct geïnstalleerd.

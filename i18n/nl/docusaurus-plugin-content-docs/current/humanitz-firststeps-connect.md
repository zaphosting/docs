## Servergegevens ophalen


Je hebt alle relevante info nodig om verbinding te maken met je server. Deze info vind je op het dashboard van het webpaneel van je game server. Belangrijke gegevens zijn onder andere:

- IP-adres en poort
- Servernaam
- Serverwachtwoord (optioneel)


Met bovenstaande info kun je je server vinden en zeker weten dat je met de juiste server verbindt.

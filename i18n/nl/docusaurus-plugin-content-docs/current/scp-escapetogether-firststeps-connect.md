## Servergegevens ophalen


Je hebt alle relevante info nodig om verbinding te maken met je server. Deze info vind je op het dashboard van het webinterfacepaneel van je game server. Belangrijke gegevens zijn onder andere:

Als je server niet zichtbaar is, kan dat komen doordat de initialisatie niet succesvol is afgerond. Dit kan bijvoorbeeld door een verkeerde configuratie of corrupte bestanden zijn. Meer info vind je meestal terug in de serverconsole of logbestanden.

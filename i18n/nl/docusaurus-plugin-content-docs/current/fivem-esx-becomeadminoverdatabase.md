## Introductie
Het toewijzen van administratorrechten geeft je eenvoudige en uitgebreide controle over je server. Als admin kun je alle beschikbare opties en functies van het spel direct in-game gebruiken. Hieronder leggen we stap voor stap uit hoe je administratorrechten aan je server toekent.  
<InlineVoucher />

## Configuratie

Eerst moeten we verbinding maken met onze database. Zo pak je dat aan.

:::info
**Belangrijk:** Je moet zelf een karakter hebben aangemaakt, zodat je deze in de database kunt terugvinden.
:::


**1** Selecteer het Database TAB

![](https://screensaver01.zap-hosting.com/index.php/s/cCweqBDBZ623iNF/preview)

**2** Klik op Beheren

![](https://screensaver01.zap-hosting.com/index.php/s/CEsFgpjYHZ26QSj/preview)

**3** Log in met je gegevens

Log indien nodig in met de gegevens die je hier vindt:
![](https://screensaver01.zap-hosting.com/index.php/s/6bktDHRRJqTNEHM/preview)

**4** Open de tabel users

![](https://screensaver01.zap-hosting.com/index.php/s/Js9ngeei9xQEm48/preview)

**5** Zoek je karakter

Zoek nu je karakter op en dubbelklik in de kolom group.  
Vul hier "admin" in.  
![](https://screensaver01.zap-hosting.com/index.php/s/GjdTo5GxkgX26BA/preview)

Dat was het! Vanaf nu ben je admin op je eigen server.

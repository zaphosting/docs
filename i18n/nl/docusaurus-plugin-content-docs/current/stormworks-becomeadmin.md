## Introductie
Het toewijzen van administratorrechten geeft je een eenvoudige en uitgebreide administratie met volledige controle over je server. Als admin kun je alle beschikbare opties en functies die het spel biedt direct in het spel gebruiken. Hieronder leggen we stap voor stap uit hoe je adminrechten toekent voor jouw server.  
<InlineVoucher />

### Configuratie

Klik in de webinterface van je game server onder Instellingen op "Configs":

![](https://screensaver01.zap-hosting.com/index.php/s/JgnbqrjwwZB7gsT/preview)

Je ziet nu het bestand "Stormworks server_config.xml". Klik op het blauwe potloodje om het te bewerken:

![752-69-max](https://screensaver01.zap-hosting.com/index.php/s/n7MdydSkB2CaBmW/preview)

Daarna kun je in de sectie "admin" een Steam64ID toevoegen die adminrechten moet krijgen:

![](https://screensaver01.zap-hosting.com/index.php/s/jTbdLBEpTT7rasF/preview)

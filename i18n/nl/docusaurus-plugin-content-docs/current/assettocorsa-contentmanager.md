## Voorbereiding

Open eerst je Content Manager en klik rechtsboven op het sandwich-menu, waar je "Server" selecteert:

![](https://screensaver01.zap-hosting.com/index.php/s/LGKdDPSCMCEMEZj/preview)

Hier maken we nu een nieuwe preset aan en selecteren deze:

![](https://screensaver01.zap-hosting.com/index.php/s/XLTcJkwrAAwB65o/preview)

## Server Slots Instellen

We moeten eerst instellen hoeveel capaciteit onze server heeft, in ons geval is dat 10 slots, dus selecteren we '10' bij Capacity:

![](https://screensaver01.zap-hosting.com/index.php/s/XreNRjbpSJJqEsQ/preview)

## Map Instellen

Standaard staat Imola geselecteerd als map, door op de map te klikken kun je je eigen map kiezen:

![](https://screensaver01.zap-hosting.com/index.php/s/B87ywSwXHL6qzFD/preview)



## Auto’s Instellen

Ga nu naar het tabblad "Entry List", hier kun je nieuwe auto’s toevoegen:


![](https://screensaver01.zap-hosting.com/index.php/s/bY5Q5WB7nDq7f8q/preview)


![](https://screensaver01.zap-hosting.com/index.php/s/w6oxDfGJifFZbd7/preview)


:::info
Let erop dat alle car slots gevuld zijn en dat je niet meer auto’s toevoegt dan het aantal beschikbare slots.

![](https://screensaver01.zap-hosting.com/index.php/s/tSZn2QJLzfDx4r9/preview)
:::

## FTP Upload Configureren

Via het menu "Advanced" kun je een FTP uploader instellen zodat je met één klik je serverconfig uploadt.

![](https://screensaver01.zap-hosting.com/index.php/s/7TmdJPGKAbAJnDP/preview)


Vul hier nu je FTP gegevens in:

![](https://screensaver01.zap-hosting.com/index.php/s/7R9xNeEbDQpF4BD/preview)

:::info
Bij "Folder" moet je de ID van je game server invullen zoals je die in de URL van je game server ziet, bijvoorbeeld:

> https://zap-hosting.com/en/customer/gameserver/show/**427814**/ftpBrowser/
:::

In ons geval wordt dat:

```
/g427814/assetto-corsa/
```

Voor AssettoServer:

```
/g427814/assettoserver/
```

Daarna klik je op "Upload Content", de config, track en auto’s die je hebt ingesteld worden automatisch geüpload, daarna hoef je alleen nog maar je server te starten!

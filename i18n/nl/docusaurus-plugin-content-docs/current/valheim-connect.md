Om verbinding te maken met je Valheim-server, openen we eerst onze webinterface en zorgen we dat we een veilig wachtwoord hebben ingesteld in de instellingen:

![](https://screensaver01.zap-hosting.com/index.php/s/XC7Cxd2zt75jyap/preview)

:::info
Let op: Zonder wachtwoord start je Valheim-server niet.
:::

Na het instellen van het wachtwoord kunnen we de server starten, het kan een paar minuten duren voordat hij op "Online" staat:

![](https://screensaver01.zap-hosting.com/index.php/s/GZQRqpGC6GGaAXa/preview)

## Direct verbinden via Valheim

Eerst kopiëren we het IP-adres van de server uit de webinterface:

![](https://screensaver01.zap-hosting.com/index.php/s/KMkbMyx4bTQkLJT/preview)


Nu openen we Valheim en klikken we in de serverlijst op de knop "Join IP":

![](https://screensaver01.zap-hosting.com/index.php/s/zqKp6sx5tEYRebx/preview)


Hier voegen we ons server-IP toe:

![](https://screensaver01.zap-hosting.com/index.php/s/EniEzmaP3E9JpLp/preview)


We kunnen nu direct verbinden met de server via de knop "Connect".


## Direct verbinden via Steam

Eerst kopiëren we het IP-adres van de server uit de webinterface:

![](https://screensaver01.zap-hosting.com/index.php/s/FkYSA8AFm53d8kK/preview)


Om direct verbinding te maken met de Valheim-server, openen we de serverlijst in Steam:

![](https://screensaver01.zap-hosting.com/index.php/s/yHxMdtTBe7xTWeZ/preview)

Hier voegen we onze server als volgt toe:

![](https://screensaver01.zap-hosting.com/index.php/s/QQcjzriQ5K4Mj9k/preview)

:::info
Let op: De poort moet met +1 worden berekend, dat betekent dat als je poort 27043 is, je in Steam 27044 moet invoeren.
:::

![](https://screensaver01.zap-hosting.com/index.php/s/RgsszmnKQLCnYSa/preview)

Na het toevoegen zouden we onze server nu in de serverlijst moeten zien:

![](https://screensaver01.zap-hosting.com/index.php/s/54XtiJzn7xndfPT/preview)

Nu kunnen we direct verbinden met de server via de knop "Connect".


## Verbinden via de Serverlijst
Eerst openen we Valheim en klikken op `Start Game`. Als je nog geen character hebt, moet je er nu een aanmaken, daarna klikken we op `Start`.


Hier willen we geen nieuwe wereld aanmaken, dus drukken we op "Cancel"

![](https://screensaver01.zap-hosting.com/index.php/s/ZnKWT8eXCcArMwX/preview)

Nu klikken we bovenaan in het menu op `Join Game` en typen we onze servernaam in het "Filter"-veld, we zetten ook een vinkje bij "Community"

![](https://screensaver01.zap-hosting.com/index.php/s/kFmZP5wyFQbpLiR/preview)

:::info
Als je je server nog niet ziet, wacht dan even. Afhankelijk van het aantal servers kan het een paar minuten duren voordat de lijst ververst is.
:::

Nu zouden we onze server al in de lijst moeten zien, we klikken erop en daarna op `Connect`. Zodra we verbonden zijn, hoeven we alleen nog het serverwachtwoord in te voeren dat we in de instellingen hebben opgegeven en op Enter te drukken!

Klaar! Je kunt nu spelen op je eigen Valheim-server!

## Introductie

Weet je niet precies wat je nodig hebt om verbinding te maken met je server, of hoe je dat precies doet? Geen zorgen! We leggen je stap voor stap uit wat je nodig hebt om verbinding te maken met je server, hoe je dat doet en waar je op moet letten. Zo kun je zonder problemen succesvol verbinding maken met je server! :)

## Voorbereiding

Allereerst is het belangrijk om te weten welke voorwaarden er zijn om verbinding te kunnen maken met de server. Dit omvat bijvoorbeeld het spel, extra content (mods) en de algemene serverinformatie zoals IP-adres, poort en servernaam.



### Spel / Content

Voor het verbinden met je game server is het verplicht dat je het spel al bezit. Dit wordt niet door ons meegeleverd bij het huren van een server. Verder is er geen extra content nodig om te kunnen spelen.



### Serverinformatie

Als aan alle voorwaarden van **Spel / Content** is voldaan, heb je de gegevens van je server nodig. Klik in je dashboard op je game server service. Het overzicht toont nu alle algemene informatie over je server.

![](https://screensaver01.zap-hosting.com/index.php/s/wYgykBmWqtynSm4/preview)

De relevante info is het **IP-adres** en de **Poort**. Zorg dat je deze onthoudt, want je hebt ze nodig voor je eerste verbinding met de server.



## Verbinden met de server

Als aan alle voorwaarden is voldaan, kun je beginnen met verbinden. Start je game en ga naar het menu Server Manager in het spel. Klik op "Server Toevoegen" om je server toe te voegen.

![](https://screensaver01.zap-hosting.com/index.php/s/E2MFTm9NskpmseS/preview)



Er opent een popup waar je de servergegevens invult. Vul het IP-adres en de poort van je game server in en klik op **Bevestigen**. Het spel vraagt je daarna om een naam en admin-wachtwoord in te stellen.

![](https://screensaver01.zap-hosting.com/index.php/s/fw7WsiRBqqQ9o2M/preview)



De volgende en laatste stap is het aanmaken van een **Savegame**. Dit doe je door op het tabblad **Game Aanmaken** te klikken bij de toegevoegde server en een naam en type wereld te kiezen.

![](https://screensaver01.zap-hosting.com/index.php/s/pDgZA5Ha56c9kJx/preview)



Na het aanmaken van de game kun je verbinden met de server. Klik op het tabblad **Status** en daarna op de knop **Spel Joinen**.

![](https://screensaver01.zap-hosting.com/index.php/s/Q7AeEzNz8sqYs2s/preview)

### Niet zichtbaar

Als de server niet zichtbaar is, kan dat komen doordat de initialisatie niet goed is afgerond. Dit kan bijvoorbeeld door een verkeerde configuratie of corrupte bestanden. Meer info vind je vaak in de serverconsole of logbestanden.

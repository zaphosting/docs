## Productbeheer

In het **Productbeheer** gedeelte vind je alle benodigde pagina’s om jouw server te beheren. Hieronder staan de opties voor het beheren van je server in detail beschreven. 



### Dashboard

Het **Dashboard** van je server is het kloppende hart van het beheer. Hier wordt alle basis- en essentiële info over je service overzichtelijk weergegeven. Denk aan locatie, status, IP: Poort, gebruikt geheugen, lopend spel, servernaam, map en het aantal beschikbare slots. Ook het gebruik van CPU, geheugen en RAM wordt getoond.

![](https://screensaver01.zap-hosting.com/index.php/s/xHPj3XxqwfxmFLs/preview)

In het dashboard kun je je service starten, stoppen of herstarten, zodat je volledige controle hebt over de werking. Daarnaast heb je toegang tot de live console voor realtime info over de serverstatus en kun je direct actie ondernemen als dat nodig is.






### DDoS Manager


Bekijk eerdere DDoS-aanvallen om bedreigingen en patronen beter in te schatten. Schakel meldingen in voor lopende DDoS-aanvallen op jouw server, zodat je snel kunt reageren bij een incident.

Afhankelijk van de serverlocatie en de beschikbare DDoS-bescherming kun je ook actieve aanvallen realtime monitoren. Dit omvat gedetailleerde verkeersinzichten, waarbij zowel legitiem verkeer als het door mitigatiesystemen geblokkeerde verkeer wordt getoond. Deze transparantie helpt je te begrijpen hoe de bescherming werkt en stelt je in staat potentiële risico’s vroegtijdig te herkennen.

:::info Geen toegang tot DDoS Manager?
Het gebruik van de DDoS Manager vereist een dedicated IP-adres of de extra optie "DDoS Manager Overzicht". Als deze onderdelen niet in je huidige pakket zitten, kun je ze altijd toevoegen door je service te upgraden.
:::

![](https://screensaver01.zap-hosting.com/index.php/s/ScCCCY52CMLgfyE/preview)





:::warning
De DDoS Manager is alleen te gebruiken als je een eigen IP-adres hebt of de optie "DDoS Manager Overzicht" als extra optie hebt geboekt. Ontbreken deze onderdelen in je huidige pakket, dan kun je ze altijd toevoegen via een upgrade.
:::



### Logbestanden

Heb je een technisch probleem en weet je niet waar het door komt? Een eerste stap is om de **Logbestanden** te checken. Deze bestanden bevatten veel info over de server, zoals actuele gebeurtenissen, aanwijzingen of foutmeldingen.

Foutmeldingen kunnen vooral helpen om de oorzaak van een probleem te vinden en beter te begrijpen. Vaak kun je problemen opsporen en oplossen door gericht in de logbestanden te duiken.

:::warning
Helpen de gevonden info niet of snap je het niet? Geen stress! We zijn dagelijks bereikbaar via support. Maak gewoon een **[Ticket](https://zap-hosting.com/en/customer/support/)** aan en leg je probleem uit. We kijken er zo snel mogelijk naar en helpen je het op te lossen! :)
:::


### Gebeurtenislog

De **Gebeurtenislog** geeft je een compleet overzicht van alle activiteiten rondom het serverbeheer. Hier zie je wanneer een server gestart, gestopt of gecrasht is, en wanneer updates, backups of nieuwe installaties zijn uitgevoerd.

![](https://screensaver01.zap-hosting.com/index.php/s/xNzsaMbW5BS5KJC/preview)



### Live Console

De **Live Console** geeft je een realtime overzicht van alle actuele gebeurtenissen op je server. Je ziet hier info over de acties van spelers, maar ook foutmeldingen en systeemberichten. Zo krijg je een compleet beeld van wat er op je server gebeurt en kun je problemen vroegtijdig signaleren.

![](https://screensaver01.zap-hosting.com/index.php/s/dktDADkq3bzG3F6/preview)

Naast het monitoren van gebeurtenissen kun je in de **Live Console** ook direct commando’s naar de server sturen. Handig om instellingen aan te passen, acties uit te voeren of snel te reageren op problemen.



### Serviceonderbrekingen

Was je server tijdelijk offline en weet je niet waarom? Kijken in de gebeurtenislog en logbestanden gaf ook geen duidelijkheid? Het kan zijn dat het probleem niet bij jouw server ligt, maar bij het host-systeem waarop jouw server draait.

Als ons monitoringsysteem een algemene storing detecteert, worden onze techneuten automatisch ingeschakeld om het probleem zo snel mogelijk op te lossen.

In dit onderdeel vind je een overzicht van de storingsgeschiedenis van het host-systeem waarop jouw game server draait. Meer info en de laatste status vind je op onze nieuwe [statuspagina](https://status.zap-hosting.com/).

## Tools

In het **Tools** gedeelte van je navigatiepaneel vind je verschillende sub-items die je helpen bij het beheren van je server. Dit zijn onder andere **FTP Browser**, **Databases** en **Backups**. Hieronder krijg je een uitgebreide introductie van deze onderdelen.



### FTP-Browser

Met de **FTP Browser** krijg je alle info die je nodig hebt om verbinding te maken met de FTP-toegang. FTP geeft je directe toegang tot de bestanden van je game server. We raden aan een externe FTP-client te gebruiken zoals **FileZilla** of **WinSCP** voor optimaal gebruik. Ben je niet bekend met FTP? Check dan de [Toegang via FTP](gameserver-ftpaccess.md) gids.

![](https://screensaver01.zap-hosting.com/index.php/s/sX85i9HyzR9wmAM/preview)

**FileZilla:** FileZilla is waarschijnlijk de makkelijkste en bekendste FTP-client. Gewoon installeren, starten en bovenin je inloggegevens invullen die je op deze pagina iets hoger vindt. Met een klik op "Verbinden" ben je direct op de server en kun je bestanden beheren.

**WinSCP:** WinSCP is eigenlijk bedoeld voor SCP en SFTP (de versleutelde varianten van FTP), maar kan ook prima met gewoon FTP overweg. Het programma is wat uitgebreider dan FileZilla en meer geschikt voor gevorderde gebruikers.



### Databases

Naast onze game server producten bieden we ook 10 inbegrepen databases aan. Deze zijn nodig als data in een database opgeslagen moet worden. Onder **Databases** kun je een database aanmaken en de inloggegevens bekijken. Je kunt ook de backup-optie activeren en bestaande database-backups importeren.

![](https://screensaver01.zap-hosting.com/index.php/s/WFLwCbZSQfYjrmD/preview)



### Backups

Onder **Backups** kun je handmatig en automatisch backups maken van je server en bijbehorende database. Zo herstel je makkelijk je data bij problemen of dataverlies.

In de instellingen bepaal je of backups automatisch gemaakt worden en of dat dagelijks of wekelijks moet gebeuren. Ook kun je instellen of de oudste backups verwijderd mogen worden als de opslagruimte vol raakt.

![](https://screensaver01.zap-hosting.com/index.php/s/nm9za5LTMEcNsga/preview)

We bieden standaard 10 GB opslagruimte voor backups op onze opslagserver bij game server producten. Heb je meer ruimte nodig? Dat kun je tegen betaling upgraden. Zo heb je altijd genoeg plek voor je backups en kun je ook oudere backups bewaren.

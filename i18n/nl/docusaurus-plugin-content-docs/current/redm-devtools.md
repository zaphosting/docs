## Introductie

De RedM Dev Tools zijn een set ontwikkeltools voor het RedM-platform die ontwikkelaars helpen hun servers en resources sneller te ontwikkelen, debuggen en optimaliseren.

![img](https://screensaver01.zap-hosting.com/index.php/s/rNMcaSHBrNyD8Aw/preview)

<InlineVoucher />

## Gebruiksscenario's

Door de dev tools te activeren en te gebruiken krijg je uitgebreide debug-informatie. Dit geeft je een overzicht en info voor de volgende toepassingen:

- Resource Monitoring
- Event Logger
- Profiler
- Netwerkdiagnose
- Console & Debug Output

## Voorbereiding

Om de RedM dev tools te kunnen gebruiken, moet het `Beta` kanaal geselecteerd en geactiveerd zijn in de RedM client instellingen onder Game. Na het aanpassen moet je de client opnieuw opstarten.

![img](https://screensaver01.zap-hosting.com/index.php/s/YnxrfpzBr8ZZA95/download)

## Dev tools gebruiken

Je opent de Dev Tools door de console te openen met de `F8` toets. Bovenin de console-navigatie zie je dan de Tools sectie, waar je de meest gebruikte tools vindt.

![img](https://screensaver01.zap-hosting.com/index.php/s/24NqpsmSbJxnHaR/preview)

:::warning Ontbreken tool-opties?
Zie je niet alle tool-opties zoals op de screenshot? Dan heb je waarschijnlijk het kanaal niet correct ingesteld zoals in de voorbereiding beschreven, of ben je vergeten de client te herstarten.
:::

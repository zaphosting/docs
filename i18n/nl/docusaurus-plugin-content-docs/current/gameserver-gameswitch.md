## Introductie

Met een ZAP-Hosting Gameserver kun je snel en makkelijk wisselen tussen verschillende games die wij aanbieden via je webinterface. In deze gids lichten we onze game switch-functie toe en laten we zien hoe je deze kunt gebruiken om verschillende games op je server te beheren en activeren.

## Webinterface Toegang

Ga eerst naar het dashboard van je gameserver product op onze website. Navigeer vervolgens naar **Instellingen->Games** in het menu aan de linkerkant.

Je bent nu in de Game Switch-sectie, waar je game-installaties kunt beheren, een game kunt activeren en waar nodig de slotprijs kunt aanpassen.

![](https://screensaver01.zap-hosting.com/index.php/s/NT6d4f7iZa62iPf/preview)

## Games Beheren

In de **Games** sectie van je Webinterface zie je een lijst met geïnstalleerde games. Daarnaast zie je de gameversie en verschillende knoppen om de game-installatie te beheren.

:::info Lifetime Gameservers
Bij onze Lifetime Gameserver werkt de prijsstelling van de game switch-functie anders. Voor elke game die je permanent wilt bezitten op je lifetime server, betaal je een **eenmalige** vergoeding van **€30** per game. Na aankoop is de game **permanent** beschikbaar op die specifieke lifetime server en kun je er dus altijd naar wisselen.

Afgezien van de prijsstelling voor lifetime servers, werkt het game switch-proces hetzelfde, maar koop je games in plaats van te werken met slotprijzen.
:::

![](https://screensaver01.zap-hosting.com/index.php/s/JJfYoRSi3M26qbc/preview)

### Nieuwe Game Installeren

Om een nieuwe game op je server te installeren, selecteer je het dropdownmenu **Beschikbare Games**, waar je games kunt bekijken die je zonder extra kosten kunt installeren.

![](https://screensaver01.zap-hosting.com/index.php/s/iN7rNje3zaBPMgf/preview)

Hier vind je ook het dropdownmenu **Meer Games**. Binnen ons brede game-aanbod kunnen sommige games een andere slotprijs hebben, wat betekent dat je de game niet kunt installeren zonder extra kosten. Om dit op te lossen, moet je een toeslag betalen om je slotprijs te verhogen. Zie het gedeelte Slotprijs Wijzigen voor meer info.

:::info Minimale Slot Aantal
Sommige games in onze collectie vereisen een minimum aantal slots voordat je ze kunt installeren. Zweef over het waarschuwingsicoon om meer te weten te komen. Gebruik de upgrade/downgrade functie om je server aan te passen op basis van deze info.

![](https://screensaver01.zap-hosting.com/index.php/s/AfAonXCqmLFDyay/preview)
:::

Heb je een game gevonden die je wilt installeren? Klik dan op de groene downloadknop om het installatieproces te starten.

![](https://screensaver01.zap-hosting.com/index.php/s/EjCQK6WYac7Ejfr/preview)

:::caution Host Systeem Wissel
Afhankelijk van de game kan het zijn dat je moet wisselen naar een ander host-systeem om de installatie voort te zetten. Dit komt meestal doordat de game op een ander besturingssysteem draait (Linux of Windows) en dus een andere host nodig heeft.

Het wisselen van host is een **vernietigend** proces, wat betekent dat alle data op de server **verwijderd** wordt. Zorg dat je backups maakt van bestanden of saves die je wilt bewaren voordat je doorgaat.
:::

Kort na het klikken op de downloadknop krijg je een pop-up om de installatie te bevestigen. Accepteer deze om het automatische installatieproces te starten. Je ziet de voortgang bovenaan de pagina.

Na voltooiing zie je de nieuwe game onder **Geïnstalleerde Games** staan. Je hebt succesvol een nieuwe game geïnstalleerd op je server.

### Game Activeren & Verwijderen

Voordat je gaat spelen, moet je de game activeren. Dit zet de game als actieve game die op je server draait.

Dit doe je simpel door in de **Geïnstalleerde Games** sectie op de blauwe activeerknop te klikken. De actieve game herken je aan het groene vinkje. Na klikken verschijnt een bevestigingsprompt die je moet accepteren.

![](https://screensaver01.zap-hosting.com/index.php/s/XT8jwyDq6j5zXPn/preview)

Je server activeert de game automatisch en herstart tijdens dit proces. Zodra klaar is de game actief en kun je beginnen met spelen. Gefeliciteerd, je hebt een game geactiveerd op je server.

### Game Verwijderen & Opnieuw Installeren

Games verwijderen of opnieuw installeren gaat ook makkelijk via de **Geïnstalleerde Games** sectie.

Zoek de game die je wilt aanpakken en klik op de oranje herinstallatieknop om de game opnieuw te installeren. Dit is een vernietigend proces en verwijdert alle data van die game, zoals servers of configuraties.

![](https://screensaver01.zap-hosting.com/index.php/s/x8kmeor6S7mXe9g/preview)

Wil je een game verwijderen? Gebruik dan de rode verwijderknop naast de game in de **Geïnstalleerde Games** sectie om deze van je server te halen.

![](https://screensaver01.zap-hosting.com/index.php/s/fwgGGzjA5mE2qRs/preview)

## Slotprijs Wijzigen

Sommige games in onze collectie hebben verschillende slotprijzen, wat kan voorkomen dat je ze kunt installeren. In dat geval moet je je slotprijs verhogen en een toeslag betalen om de game weer beschikbaar te maken.

Dit kan op twee manieren: via de **Slotprijs Wijzigen** knop bovenaan de pagina, of door eerst games te zoeken en de slotprijzen van niet-beschikbare games te bekijken.

We raden aan eerst het dropdownmenu **Meer Games** te gebruiken om games te vinden en makkelijk hun slotprijzen te checken. Heb je een game gevonden? Klik dan op de blauwe koopknop.

![](https://screensaver01.zap-hosting.com/index.php/s/FnrqX4SFQq2MkPL/preview)

Er opent een prompt om de slotprijs aan te passen, met de prijs al ingevuld voor de gekozen game. Je ziet hier ook de toeslag die je moet betalen en de nieuwe pakketprijs na aanpassing. Je kunt hier ook nogmaals games bekijken indien nodig.

![](https://screensaver01.zap-hosting.com/index.php/s/EGaZRMzjnF3QPiL/preview)

Klaar? Klik op volgende om naar de betaalmethode te gaan om de toeslag te voldoen. Kies je betaalmethode en klik op nu kopen om de slotprijswijziging af te ronden.

![](https://screensaver01.zap-hosting.com/index.php/s/83oPQocpmSX58R4/preview)

Na succesvolle betaling zie je de aangepaste slotprijs terug in je serverpakket. Je wordt gevraagd je server te herstarten om alles definitief door te voeren.

Je kunt nu terug naar **Beschikbare Games** en de game normaal installeren. Gefeliciteerd, je hebt de slotprijs van je server succesvol aangepast.

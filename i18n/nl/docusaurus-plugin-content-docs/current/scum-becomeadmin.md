## Introductie
Het toewijzen van administratorrechten geeft je eenvoudige en uitgebreide controle over je server. Als admin kun je alle beschikbare opties en functies van het spel direct in de game gebruiken. Hieronder leggen we stap voor stap uit hoe je adminrechten aan je server toewijst.

<InlineVoucher />



## Configuratie

Je kunt de adminrechten instellen via de **instellingenpagina van je game server**. Open hiervoor de instellingen en ga naar de optie **Admin List**. Vul hier je **SteamID64** in om jezelf als admin toe te wijzen. Wil je meerdere spelers als admin toevoegen? Scheid dan elke SteamID met een **komma**.

![img](https://screensaver01.zap-hosting.com/index.php/s/AiMDoy5mdEHjnj3/download)



## Admin Commands

Hieronder vind je een lijst met veelgebruikte admin commands die je als admin op je server kunt gebruiken.

| Command           | Beschrijving                                               | Voorbeeld                    |
| ----------------- | ---------------------------------------------------------- | ---------------------------- |
| #LIST             | Geeft een complete lijst van SCUM admin commands.          | #list                       |
| #LISTPLAYERS      | Toont een lijst van alle spelers die momenteel op de server zijn. | #listplayers                |
| #KICK             | Kickt een specifieke speler van de server.                 | #kick playername            |
| #BAN              | Bant een specifieke speler permanent van de server.        | #ban playername             |

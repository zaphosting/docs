## Introductie
Het toewijzen van administratorrechten maakt eenvoudig en uitgebreid beheer mogelijk met volledige controle over je server. Als admin kun je alle beschikbare opties en functies die het spel biedt direct in het spel gebruiken. Hieronder vind je alle stappen die je moet volgen om administratorrechten aan je server toe te wijzen.  
<InlineVoucher />

## De token ophalen

Om je als admin te registreren op je Terraria-server, heb je je unieke token nodig.  
Deze token wordt gegenereerd wanneer je de server voor het eerst opstart. Open daarna de live console, waar je de token kunt vinden:

![](https://screensaver01.zap-hosting.com/index.php/s/5cEQYgBgxAYQRcx/preview)

![](https://screensaver01.zap-hosting.com/index.php/s/HEYwEWe5c3DPZ6E/preview)

In dit voorbeeld is de token **4914096**.

## De token in het spel invoeren

Om je als admin te registreren, join je je server in Terraria en voer je het commando **/setup 4914096** in het spel in.  
Vervang natuurlijk de token **4914096** door jouw eigen token.

![](https://screensaver01.zap-hosting.com/index.php/s/tadkJkQf5cE3dTB/preview)

### Het volgende bericht verschijnt dan in de chat:

![](https://screensaver01.zap-hosting.com/index.php/s/JDfKNpaeB63pCeB/preview)


## Een admin toevoegen

Nu kun je een nieuwe admin toevoegen aan je server met het commando **/user add GEBRUIKERSNAAM WACHTWOORD owner**.  
Vervang **GEBRUIKERSNAAM** door je naam en kies als **WACHTWOORD** een wachtwoord naar keuze,  
dat je in de toekomst kunt gebruiken om in te loggen als admin op je server.

![](https://screensaver01.zap-hosting.com/index.php/s/iX62CLtn577NfFQ/preview)  
![](https://screensaver01.zap-hosting.com/index.php/s/xeP8Y8sx66LkSJQ/preview)

:::info
Let op: je wachtwoord moet minimaal 4 tekens lang zijn!
:::

## Inloggen als admin

Nadat je je in het spel als admin hebt geregistreerd, kun je inloggen met het volgende commando:  
**/login GEBRUIKERSNAAM WACHTWOORD**

Vervang natuurlijk **GEBRUIKERSNAAM** door je naam en **WACHTWOORD** door het eerder ingestelde wachtwoord.

![](https://screensaver01.zap-hosting.com/index.php/s/ewTra2Fi2yia9jS/preview)

Je bent nu ingelogd als admin op je server en kunt deze beheren met verschillende commando’s.

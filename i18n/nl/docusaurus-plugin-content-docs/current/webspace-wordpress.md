## Introductie

WordPress is het meest gebruikte gratis contentmanagementsysteem wereldwijd. Het werd vanaf 2003 geprogrammeerd door Matthew Mullenweg als software voor weblogs en wordt nog steeds ontwikkeld als een open source project. In deze gids leggen we uit hoe je deze software installeert op ons webspace-product.

![](https://screensaver01.zap-hosting.com/index.php/s/j3Ctfont64EnpcH/preview)

## Voorbereiding

Voordat je WordPress daadwerkelijk kunt installeren, moeten er een paar dingen geregeld worden. Dit omvat het verkrijgen van de software, het aanmaken van de database die je gaat gebruiken en het instellen van de mailserver (e-mailadres).



**Software**

De software kun je downloaden van de officiële WordPress-website. De download vind je hier: [WordPress Download](https://en.wordpress.org/download/)

![](https://screensaver01.zap-hosting.com/index.php/s/69YD2QWG8iHzzaa/preview)

De download bevat een gecomprimeerd zip-bestand, dat je op je lokale computer moet uitpakken. Binnenin vind je een map met de naam **wordpress**. De inhoud hiervan heb je nodig en moet je uploaden via **FTP of File Manager**. In dit voorbeeld laten we de setup zien via FTP. Je kunt een FTP-gebruiker aanmaken onder **Websites & Domeinen** bij **FTP-toegang**.



![](https://screensaver01.zap-hosting.com/index.php/s/nkKsBpyb6GM6Rkp/download/chrome_7Y5hmuXn5f.gif)



Nu verbinden we met de webspace via het aangemaakte FTP-account. Daar navigeer je naar de **httpdocs**-map en upload je de bestanden van de software.



![](https://screensaver01.zap-hosting.com/index.php/s/c9JSb9XEjfwGwwQ/preview)



**Database**

Vervolgens moet je de database aanmaken, waarin straks alle informatie van je website wordt opgeslagen. Klik hiervoor weer op **Websites & Domeinen** en daarna op **Databases**. Klik dan op Database toevoegen en maak een database aan:  



![](https://screensaver01.zap-hosting.com/index.php/s/99BJeGCHjS9QHkz/download/chrome_Jzw3adOg7G.gif)



Klik op **OK** en de database wordt aangemaakt.



**Mailserver (e-mailadres)**

Voor het registreren van een account op het forum heb je een mailserver met een e-mailadres nodig, zodat registratie-mails naar gebruikers gestuurd kunnen worden. Instructies voor het instellen van zo’n e-mailadres vind je hier: [E-mails Verzenden](webspace-plesk-sendmail.md)



## Installatie

Als je alle stappen uit de voorbereiding hebt afgerond, kun je beginnen met de daadwerkelijke installatie van WordPress. Open hiervoor je website in je internetbrowser. Dit zou er zo uit moeten zien:

![](https://screensaver01.zap-hosting.com/index.php/s/8AcGsKyHpsSgKxL/preview)



Klik hier op **Let’s go!** om de installatie te starten. Het eerste wat je doet is de database configureren, die je al hebt aangemaakt tijdens de voorbereiding. Vul hier de databasegegevens van de aangemaakte database in.



![](https://screensaver01.zap-hosting.com/index.php/s/9jnjxz8B9TaeZSD/preview)



Klik op **Verzenden** en daarna op **Installatie uitvoeren**.



![](https://screensaver01.zap-hosting.com/index.php/s/ZLZdC5NeWDoQBkr/preview)



WordPress vraagt nu nog wat extra info. Je kunt nu de titel van je website, gebruikersnaam, wachtwoord en e-mailadres invullen.

 

![](https://screensaver01.zap-hosting.com/index.php/s/WzckYZ47QJLd7yM/preview)



Als alle velden ingevuld zijn, klik je op **WordPress installeren**. De installatie wordt binnen een paar seconden afgerond en het inlogscherm verschijnt. Hier kun je inloggen met je aangemaakte account. Daarna kun je je WordPress helemaal naar eigen wens inrichten!

![](https://screensaver01.zap-hosting.com/index.php/s/MnLRYMDJCXjJJ8i/download/chrome_4KNjihGpo0.gif)

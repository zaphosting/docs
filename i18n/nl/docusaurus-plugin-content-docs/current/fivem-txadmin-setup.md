## Introductie

txAdmin is een volledig gratis te gebruiken, compleet webpaneel om je FiveM-server te beheren en te monitoren. Het biedt een breed scala aan functies die het beheren van een FiveM-server super makkelijk maken. Het opzetten van txAdmin is helemaal gratis, begin vandaag nog met het meest simpele maar toch functionele FiveM server webpaneel.

## Toegang

Je vindt een link en de inloggegevens voor txAdmin in het txAdmin-gedeelte van het dashboard van je game server. Hiermee heb je alle info die je nodig hebt om makkelijk in te loggen op de txAdmin interface en te beginnen met de setup.

![img](https://screensaver01.zap-hosting.com/index.php/s/69LdTK3FyNZNXid/download)

Kopieer de gebruikersnaam en het wachtwoord en klik op de link (IP-adres:poort) om de txAdmin interface te openen en in te loggen. De txAdmin interface opent dan in een nieuw venster, waar je met de gegeven inloggegevens moet inloggen.

![](https://screensaver01.zap-hosting.com/index.php/s/pp8GLQBoX4LoqTA/preview)



## Setup

Nu je succesvol bent ingelogd, kun je beginnen met het instellen van txAdmin en je server. Je ziet nu de txAdmin homepage (Dashboard). Linksboven zie je een melding dat je server nog geconfigureerd moet worden. Klik op **Ga naar de setup pagina** om te starten met de setup.

![](https://screensaver01.zap-hosting.com/index.php/s/oXakf3qoJaim7ex/download)



### Welkom en servernaam

Geef nu een servernaam op die je voor je project wilt gebruiken. Deze naam is niet bedoeld voor de serverlijst, maar alleen voor gebruik binnen de txAdmin interface en voor chat/Discord berichten.

![](https://screensaver01.zap-hosting.com/index.php/s/FCmd5xQ89wSPHfe/preview)



### Deployment Type

Bij Deployment Type kies je nu hoe je je server wilt opzetten. Je hebt de volgende opties: **Popular Recipes**, Bestaande Server Data, **Remote URL Template**, en **Custom Template**. Hieronder leggen we elke optie wat uitgebreider uit, zodat je kunt kiezen wat het beste bij jou past.

![](https://screensaver01.zap-hosting.com/index.php/s/52HfyJSNLscApNE/preview)



### Popular Template

Popular Templates zijn kant-en-klare pakketten die je meteen kunt gebruiken, dit is de makkelijkste manier om je FiveM-server snel aan de praat te krijgen. Als dit je eerste server is, raden we deze optie sterk aan. Deze templates bevatten alle benodigde configuraties en resources om snel een werkende server op te zetten, zodat jij je kunt focussen op customizen en gamen in plaats van de basis setup. Kies simpelweg een template die bij je past.

![](https://screensaver01.zap-hosting.com/index.php/s/PSsf22NeebNBRw7/preview)



### Bestaande Server Data

Gebruik deze optie als je al een server met txAdmin hebt gedraaid en die nu bij ons wilt hosten. Je kunt dan simpelweg je serverdata via FTP uploaden en het pad opgeven tijdens de setup. Ben je nog niet bekend met FTP? Check dan zeker onze [FTP toegang](gameserver-ftpaccess.md) gids. Die helpt je stap voor stap om je serverbestanden soepel over te zetten, zodat je snel live bent op ons platform.

![img](https://screensaver01.zap-hosting.com/index.php/s/KS4raRtHWmmw5iN/preview)





### Remote URL Template

Wil je een template gebruiken die niet standaard in txAdmin zit? Met deze optie kun je een custom recipe via een externe URL opgeven. Deze wordt dan opgehaald en uitgevoerd op je server. Handig als je een eigen serverconfig of specifieke setup wilt gebruiken die niet in de standaard txAdmin opties zit. Door de URL te geven, download en pas txAdmin de template automatisch toe, zodat je server precies zo wordt ingericht als jij wil.

![](https://screensaver01.zap-hosting.com/index.php/s/jrGzTGp9FwLc82i/preview)

### Custom Template

Deze optie is voor gebruikers die al een eigen recipe hebben gemaakt en die opnieuw willen laden. Hiermee kun je snel en makkelijk je eigen configuraties toepassen zonder alles opnieuw te hoeven instellen. Selecteer gewoon je bestaande recipe en txAdmin regelt de rest, zodat je server precies zo wordt ingesteld als jij wilt.

![](https://screensaver01.zap-hosting.com/index.php/s/Z75q5RKakwfpHGy/preview)



### Recipe Deployer

Nadat je een Deployment Type hebt gekozen en de stappen hebt gevolgd, klik je op **Ga naar Recipe Deployer**. Hier rond je de laatste stappen af: **Review Recipe**, **Input Parameters**, **Run Recipe**, en **Configure server.cfg**.

In stap 1 kun je de gekozen of toegevoegde recipe nog verder aanpassen. Als dat niet nodig is, ga je door naar stap 2. Hier vul je je eigen [Eigen License Key](fivem-licensekey.md) in en controleer je de databasegegevens. Als alles klopt, klik je op **Run Recipe**. Dit kan even duren. Tot slot kun je de serverconfig (`server.cfg`) aanpassen waar nodig.

![](https://screensaver01.zap-hosting.com/index.php/s/wFMD576sBQAAdxZ/download)

Klik daarna op **Opslaan & Server starten**. Daarmee is de txAdmin setup klaar en kun je verbinding maken met je server.


## Extra support

Voor txAdmin-specifieke support kun je contact opnemen met het txAdmin-team via hun [officiële website](https://txadm.in/) of join de community op [Discord](https://discord.gg/txAdmin/). De community en support staan klaar om je te helpen met vragen of problemen, zodat jij de beste ervaring hebt met het beheren van je server.

## Plugins installeren op een Unturned Rocket Mod Server

### Stap 1
Eerst moet je verbinding maken met de server via [Toegang via FTP](gameserver-ftpaccess.md).

Zodra de FTP-verbinding staat, open je de map "Plugins".  
Deze vind je onder:  
```
/unturned-rocket/Servers/ZAP001/Rocket/Plugins. 
```
Standaard zou deze map leeg moeten zijn: 

![](https://screensaver01.zap-hosting.com/index.php/s/wYY7jLji9YaNcBp/preview)

### Stap 2

Nu kun je een plugin uploaden.  
Plugins voor Rocketmod vind je bijvoorbeeld op Github: https://github.com/RocketModPlugins  
Nadat je de gewenste plugin hebt gedownload, moet je deze uitpakken.  
Afhankelijk van je besturingssysteem kun je dit direct doen, maar soms heb je extra programma’s nodig zoals 7Zip om het archief te openen.

Op Windows is het bijvoorbeeld genoeg om het ZIP-bestand dubbel te klikken en de inhoud naar je bureaublad te slepen. 

![](https://screensaver01.zap-hosting.com/index.php/s/qnpy29HySQzJTBL/preview)

Als dat klaar is, kun je de plugin uploaden via “drag and drop”:  

![](https://screensaver01.zap-hosting.com/index.php/s/o4ZNQtyosnw5eHo/preview)

De plugin staat nu op je server:  

![](https://screensaver01.zap-hosting.com/index.php/s/DYXpnZ2n5ibxW5t/preview)

### Stap 3

Start nu je game server via de webinterface:  

![](https://screensaver01.zap-hosting.com/index.php/s/pgyBsYcoXNP7dnL/preview)

Je server is nu online met de gewenste plugin:  

![](https://screensaver01.zap-hosting.com/index.php/s/ZZLPBprWoBZfTQ6/preview)

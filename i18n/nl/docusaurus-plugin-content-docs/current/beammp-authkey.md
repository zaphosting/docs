## Waarom heb ik een key nodig?

Een Auth Key is nodig als je server in de serverlijst getoond moet worden, anders is deze alleen via directe verbinding bereikbaar.

<InlineVoucher />

## Voorbereiding
Om je eigen authkey aan te maken, log je eerst in op https://beammp.com/keymaster, dit vereist een Discord-account.
Als je bent ingelogd, klik je links op "Keys", hiermee kom je in het overzicht van je licentiesleutels:

![](https://screensaver01.zap-hosting.com/index.php/s/Zp72q2WR85pxJgq/preview)

Klik hier nu op het blauwe "here" om de key aan te maken:

![](https://screensaver01.zap-hosting.com/index.php/s/ARqCQyEbF6BYnH4/preview)


## Gegevens invullen

Vul nu de gegevens als volgt in:

- **Servernaam:** Een naam voor je server, dit hoeft niet hetzelfde te zijn als de naam in de instellingen.
- **Server IP:** Het IP-adres van je server, zonder poort.


### IP opgeven

Het IP moet zonder poort worden opgegeven, deze vind je bovenaan in je interface:

![](https://screensaver01.zap-hosting.com/index.php/s/8MJeXxm87EdLykg/preview)

## Key aanmaken

Klik nu op "Create", er wordt een key gegenereerd.

![](https://screensaver01.zap-hosting.com/index.php/s/Ebyk5tPCHnppcWC/preview)

## Key instellen

Open nu de instellingen van je game server en scroll naar beneden, hier kun je onder "Auth Key" je eigen key invullen. Het resultaat zou er zo uit moeten zien:

![](https://screensaver01.zap-hosting.com/index.php/s/5p7LdSDCJzrxKDy/preview)

Sla de instellingen op en start de server opnieuw op.

Klaar! De key is nu ingesteld en je server zou zichzelf binnenkort in de lijst moeten tonen.

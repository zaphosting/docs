## Waar haal ik mods vandaan?
Omdat BeamMP is gebouwd op BeamNG, kun je natuurlijk ook de BeamNG Mods gebruiken.  
Er zijn verschillende plekken waar je mods kunt downloaden, de officiële vind je op beamng.com/resources

## Data uploaden
Allereerst moet je via FTP verbinding maken met je BeamMP-gameserver. De inloggegevens vind je in de "FTP-Browser" in je ZAP Interface.

![](https://screensaver01.zap-hosting.com/index.php/s/98Yz5M7K6YDP9Fd/preview)

Verbind met een ftp-programma dat jij fijn vindt en open de map

```
/g*****/beammp/Resources/Client/
```

Upload hier de .zip van je mod.

:::info
Na een server restart is de mod beschikbaar en wordt deze automatisch naar je client gedownload.
:::

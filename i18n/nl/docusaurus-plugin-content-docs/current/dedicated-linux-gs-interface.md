## Introductie

Als game server producten niet helemaal jouw ding zijn of niet aan je verwachtingen voldoen, of als je je service op een andere manier nodig hebt, is het vaak slimmer om een VPS of dedicated server te gebruiken. Maar dan loop je vaak tegen het probleem aan dat niet alle services geautomatiseerd of vooraf geïnstalleerd zijn. Het opzetten van services zoals game servers, Teamspeak 3 servers en muziek bots moet je dan zelf regelen.

Om dit makkelijker te maken, hebben we een handige oplossing ontwikkeld voor onze Linux-gebaseerde VPS of dedicated servers. De zogenaamde Gameserver/Teamspeak 3 Server interface (GS/TS3 Interface)!



## Gebruikssituatie

Met de gebruiksvriendelijke GS/TS3 interface kun je met slechts een paar klikken Teamspeak 3 of andere geselecteerde game server services installeren op je Linux VPS of dedicated server. Je hebt geen voorkennis nodig voor deze installatie. Het installeren van alle benodigde pakketten, serverbestanden en de daadwerkelijke installatie is volledig geautomatiseerd, wat jou tijd en moeite bespaart.

:::warning
De GS/TS3 interface functie is te gebruiken met de volgende besturingssystemen:

- Debian: 10, 11, 12
- Ubuntu: 20.04, 22.04

(*) We zijn al bezig om binnenkort ook andere en nieuwere besturingssystemen en versies beschikbaar te maken.
:::



## Beschikbare services & games

De GS/TS3 interface ondersteunt het opzetten van game servers, voice servers en muziek bots. De meeste games die we aanbieden voor onze game server producten zijn ook beschikbaar via de GS/TS3 interface optie. Dit geldt voor alle games die we actief onder Linux draaien. Games die alleen op Windows draaien, kunnen helaas niet geïnstalleerd worden. Daar is alleen een handmatige installatie mogelijk, mits de game een Linux server versie ondersteunt.

| Services    | Ondersteund |
| ----------- | ---------------- |
| Gameserver  | ✔️                |
| Voiceserver | ✔️                |
| Musicbot (Sinusbot)  | ✔️                |



## Installatie en configuratie

Om de GS/TS3 interface te installeren, ga je in de serverbeheer van je product naar het navigatiepaneel **Instellingen** en klik je op het submenu **Gameserver / TS3**. 

De eerste keer dat je de GS/TS3 interface opent, word je doorgestuurd naar de gebruiksvriendelijke setup wizard. Om de interface op te zetten en te koppelen aan de server, moet je een ondersteund besturingssysteem selecteren en de inloggegevens van de root gebruiker invullen. 

![](https://screensaver01.zap-hosting.com/index.php/s/dLeLDKdmdiZ74CP/download)



Zodra de server en de website succesvol gekoppeld zijn, kan de GS/TS3 interface worden aangemaakt. 



![](https://screensaver01.zap-hosting.com/index.php/s/FK9mP3BgzrPmH7S/preview)



![](https://screensaver01.zap-hosting.com/index.php/s/JL7jyTKbCEx8FBZ/preview)

:::info
**Let op**: Ga niet verder met het proces totdat de interface volledig geïnstalleerd is. Je herkent een succesvolle installatie doordat de voortgangsbalk verdwenen is en er bij elk benodigd pakket "Geïnstalleerd" staat in plaats van "Ontbreekt".
:::



### Game server aanmaken

Als de creatie en installatie van de GS/TS3 interface succesvol is afgerond, kun je nu beginnen met het opzetten van services. Het installeren van een game server doe je via de knop **Game server installeren** in de categorie **Game server**.

![](https://screensaver01.zap-hosting.com/index.php/s/QinM7KtFwcAp5pE/preview)



In de volgende stap kun je het IP-adres kiezen van de gewenste game server (als je server meer dan één IP heeft). Ook kun je de gewenste poort, het aantal slots en de RAM boost zelf instellen.

![](https://screensaver01.zap-hosting.com/index.php/s/cqWwZbXT77okeDa/preview)

Check goed of alle gegevens kloppen en aan jouw wensen voldoen. Als je alles hebt ingesteld, klik je op "Game server nu installeren" om het installatieproces te starten. De game server wordt dan volgens jouw specificaties opgezet en je kunt snel aan de slag. Je ziet je nieuw aangemaakte game server nu in de lijst staan.



![](https://screensaver01.zap-hosting.com/index.php/s/9WkJnxzkaEHmri7/preview)



Je bent nu in het beheer van je game server. Daar kun je in het navigatiepaneel **Instellingen** onder **Games** het gewenste spel voor je game server selecteren en installeren.

![](https://screensaver01.zap-hosting.com/index.php/s/6pxEbWttos6HAYt/preview)

Hier vind je een lijst met alle games die je op je server kunt installeren. In dit voorbeeld wordt een server met het spel (pakket) Minecraft: Paperspigot geïnstalleerd. Kies je gewenste game en klik op de groene knop om de installatie te starten.

![](https://screensaver01.zap-hosting.com/index.php/s/gazW2itexCJd7cY/preview)

Er verschijnt een overzicht met alle benodigde voorwaarden. Als aan alle voorwaarden voldaan is, kun je het proces bevestigen met de knop **Accepteren en installeren**.

![](https://screensaver01.zap-hosting.com/index.php/s/jeQC7dp6zpe3ny4/preview)

Het opzetten van het spel kan even duren, afhankelijk van het spel. De voortgang zie je in de statusweergave. Daarna kun je je game server starten en ermee verbinden in het spel!



### Voice server aanmaken

Het installeren van een **Voiceserver (Teamspeak 3 Server)** werkt vergelijkbaar met het installeren van een game server. Je klikt eerst weer op "Gameserver / TS3". In dit gedeelte vind je de optie **Voiceserver installeren**, waarmee je de installatie van de Teamspeak 3 server start.

![](https://screensaver01.zap-hosting.com/index.php/s/mi8p3NTfwBBExsD/preview)

In de volgende stap kun je de gewenste instellingen voor je Teamspeak 3 server opgeven, zoals het IP-adres, de poort en het aantal beschikbare slots. Als je alle gegevens hebt ingevuld, rond je de installatie af door op **Voiceserver nu installeren** te klikken.

![](https://screensaver01.zap-hosting.com/index.php/s/ajfzxsJfCFdfBac/preview)

Check goed of alle informatie klopt en aan jouw wensen voldoet. Als je alles hebt ingesteld, klik je op **Teamspeak 3/5 Server nu installeren** om het installatieproces te starten. Je ziet de geïnstalleerde Teamspeak 3 server nu in het overzicht **Voiceserver Overzicht**.

![](https://screensaver01.zap-hosting.com/index.php/s/YaEYorRG7TJGpmB/preview)

:::info
**Let op**: Dit is de instantie waarop de Teamspeak 3 server daarna geïnstalleerd wordt.
:::

Je bent nu in de interface van de instantie van je geïnstalleerde Teamspeak 3 server. Je kunt de instantie starten door op de knop **Start** te klikken.

![](https://screensaver01.zap-hosting.com/index.php/s/SmqHB24ozJimBY9/preview)


Klik nu op het menu-item **Virtual Servers**. Hier staan alle geïnstalleerde Teamspeak 3 servers op jouw instantie overzichtelijk weergegeven. Om de interface van een specifieke Teamspeak 3 server te openen, klik je simpelweg op het "oog" icoon.

![](https://screensaver01.zap-hosting.com/index.php/s/E3ZqxC9rPjWwC5F/preview)

In de Teamspeak 3 server interface vind je alle benodigde info en instellingen om je server helemaal naar wens te beheren en in te richten. Bijvoorbeeld onder het tabblad "Autorisatiesleutel" kun je een token aanmaken waarmee je adminrechten op je server krijgt. Zo heb je volledige controle over gebruikersbeheer en toegang tot de server.

Onder het menu-item "Instellingen" heb je nog meer opties om je server aan te passen. Hier kun je bijvoorbeeld de naam van je server wijzigen, een wachtwoord instellen om toegang te beperken, of een persoonlijke welkomstboodschap opslaan die gebruikers krijgen als ze de server betreden.

:::caution Info over Teamspeak-licenties
Standaard is er geen licentie geïnstalleerd op de Teamspeak 3 server. Dit betekent dat je een Teamspeak 3 server alleen kunt gebruiken met een capaciteit van 32 slots. Wil je meer capaciteit of extra functies activeren voor je Teamspeak 3 server, dan kun je een bijbehorende licentie aanschaffen bij Teamspeak.
:::

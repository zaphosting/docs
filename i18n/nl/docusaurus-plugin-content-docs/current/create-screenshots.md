## Screenshots maken

Soms is het nodig om screenshots naar de support te sturen.  
Hiervoor moeten de screenshots gedeeld worden via een link.  
Die link kan verwijzen naar je eigen cloud of naar een "web uploader", ook wel web screenshot tools genoemd.

Als je eigen cloud of de web screenshot tool geen functie heeft om direct een screenshot te maken, kun je meestal op je pc met simpele tools een screenshot maken.  
Op Windows is het genoeg om de tool "Knipprogramma" te gebruiken. Hiermee sla je een screenshot lokaal op, die je daarna kunt uploaden.  
Op Linux kun je meestal de al geïnstalleerde tool "Screenshots" gebruiken of andere tools zoals "FlameShot" om lokaal screenshots te maken, die je daarna kunt uploaden.

Hieronder vind je een korte uitleg van de 2 populairste screenshot tools die gratis beschikbaar zijn:

:::info
Het is niet aan te raden om privé- of gevoelige data openbaar te delen. Als er data zichtbaar is op de screenshot, maak deze dan onherkenbaar of vervang het door voorbeelddata.
:::


### Imgur

Imgur biedt een uploader voor eigen bestanden zoals afbeeldingen/video’s.  
Na het uploaden wordt er een link aangemaakt.  
Deze link kun je vervolgens met anderen delen.

Belangrijk: Omdat Imgur zelf geen screenshot functie heeft, moet je de screenshot eerst maken en daarna uploaden (zie hierboven bij "Screenshots maken").

[Imgur-Website](https://imgur.com/upload)

Upload optie:

![](https://screensaver01.zap-hosting.com/index.php/s/4sCEeKkyGEm3EXd/preview)

Na het uploaden kun je kleine aanpassingen doen.

Daarna kun je de screenshot als Post naar Community plaatsen (wordt openbaar weergegeven) of als Verborgen (wordt niet openbaar weergegeven).

Als dat gedaan is, verschijnt de link die je kunt gebruiken om te delen:

![](https://screensaver01.zap-hosting.com/index.php/s/dJCpsKtka9rHX6T/preview)

### Lightshot

Lightshot biedt geen directe upload voor afbeeldingen/screenshots. Uploaden kan alleen via de eigen client.  
De client kun je direct van de website downloaden en die biedt ook een paar tools om iets te markeren/tonen op de screenshot of om delen onherkenbaar te maken.

Download de client [Download](https://app.prntscr.com/en/)  
Belangrijk: Er is geen versie voor Linux beschikbaar.

![](https://screensaver01.zap-hosting.com/index.php/s/iGEccwZZDXqLGKp/preview)

Na het downloaden kun je de client installeren.  
Standaard kun je daarna een screenshot maken of een gebied selecteren met de "Print" knop.

Na het selecteren van het gebied kun je de screenshot nog bewerken.

Als je klaar bent met bewerken, kun je de screenshot eventueel uploaden door op de "Cloud" te klikken.

Na het uploaden kun je de link kopiëren en met anderen delen.

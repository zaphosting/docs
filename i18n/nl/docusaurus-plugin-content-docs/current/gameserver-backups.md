## Intro
Je kunt bij ons met één klik een back-up maken. Zo maak je makkelijk en zonder gedoe back-ups en kun je ze later altijd weer importeren. Deze functie is beschikbaar voor de serverbestanden van je gameserver én de bijbehorende databases. Alle back-ups worden opgeslagen op je opslagserver, waar standaard 10GB gratis opslagruimte bij zit. Heb je meer nodig? Dan kun je upgraden naar Premium Storage.

Naast handmatig back-ups maken, kun je ze ook volledig automatisch laten aanmaken. Dit kan dagelijks of wekelijks op een door jou gekozen tijdstip. Na het terugzetten van een back-up staat je server weer precies zoals op het moment van de back-up.

## Back-ups handmatig maken

Klik op de groene **+** knop naast de back-uplijst om handmatig een back-up te maken. Het back-upproces start daarna direct.

:::info
Het maken van een back-up kan een paar minuten duren, afhankelijk van de grootte van je serverbestanden!
:::

## Back-ups automatisch maken

Onderaan de pagina vind je meer opties, waaronder de instellingen voor automatische back-ups. Zet de optie **Back-ups automatisch maken** aan om deze functie te activeren. Daarna kies je het interval.

![](https://screensaver01.zap-hosting.com/index.php/s/gB9n6JspdNW73bj/preview)

Afhankelijk van het spel is het meestal ook slim om de database mee te back-uppen, want belangrijke data zoals scores, permissies en meer worden daar vaak opgeslagen.

>⚠️ Genoeg opslagruimte is super belangrijk om regelmatig back-ups te kunnen maken. Zorg dat je altijd genoeg ruimte beschikbaar hebt.

## Back-ups downloaden/verwijderen

Wil je back-ups downloaden of verwijderen? Dan moet je via FTP verbinding maken met je opslagserver. Hier vind je de instructies voor FTP: [Toegang via FTP](gameserver-ftpaccess.md)

![](https://screensaver01.zap-hosting.com/index.php/s/tfJoBpaELEPKMij/preview)

![](https://screensaver01.zap-hosting.com/index.php/s/q3E8XTX8gRQoasY/preview)

Daar zie je de gebruikersgegevens van je opslagserver en hoeveel opslagruimte je gebruikt. Als je via FTP verbonden bent, kun je daar back-ups downloaden of verwijderen.

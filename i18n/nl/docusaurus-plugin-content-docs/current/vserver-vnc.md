## Introductie

VNC (Virtual Network Computing) is software die het scherm van een externe computer toont op je lokale computer en tegelijkertijd de toetsenbord- en muisbewegingen van je lokale computer naar de externe computer stuurt. Zo kun je je server op afstand bereiken, zelfs als deze niet goed opstart of draait.

Bij opstart- of netwerkproblemen heb je dus altijd nog de mogelijkheid om in te loggen en actie te ondernemen.

## Gebruik van VNC
De VNC webclient vind je in het VPS-dashboard. Klik hiervoor op de optie serial console in het tools-gedeelte van de administratie en open vervolgens de VNC-verbinding via de **Start VNC tunnel** knop.

![](https://screensaver01.zap-hosting.com/index.php/s/AgSL8QcynHSfXFA/preview)



Daarna opent de webclient van de VNC-console. Hier kun je de informatie bekijken die de server op dat moment toont. Dit kan bijvoorbeeld het opstartproces zijn, het inlogscherm of de GUI van het besturingssysteem.

Het volgende voorbeeld toont een Windows VPS. Deze server is succesvol opgestart, dus de VNC-console toont nu het inlogscherm. Je kunt hier ook inloggen, zelfs als de server geen actieve internetverbinding heeft. Klik op de knop "Send CtrlAltDel" en log in met je gebruikersnaam en wachtwoord.

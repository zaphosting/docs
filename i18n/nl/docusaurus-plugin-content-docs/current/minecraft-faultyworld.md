## Intro

Je wereld is deels overbelast of buggy en dat beperkt je gameplay, waardoor je niet meer goed kunt spelen? Dit kan gebeuren als er te veel items in bepaalde gebieden van je wereld liggen of als er beschadigde chunks in zitten. Een Chunk Error is een vrij veelvoorkomende glitch in Minecraft die optreedt wanneer een chunk niet goed laadt.

![img](https://screensaver01.zap-hosting.com/index.php/s/A5bj6posqkTfGK8/preview)

## Voorbereiding

Er zijn verschillende tools om Minecraft-werelden aan te passen. Een van de populairste tools is [MCA Selector](https://github.com/Querz/mcaselector), die we in deze gids gebruiken. Download hiervoor het bestand `MCA_Selector_Setup.exe`.

Daarnaast heb je de bestanden van je wereld nodig op je pc. Je kunt de wereld via FTP downloaden. Als je niet bekend bent met FTP, raden we je aan onze [Toegang via FTP](gameserver-ftpaccess.md) gids te checken. Download de map `world` uit de game server directory.

## Wereld laden

Als je de tool en de wereld hebt gedownload, kun je de wereld in de tool importeren. Klik in de tool op `File -> Open World`. Daarna zou je je wereld moeten zien zoals in ons voorbeeld.

## Chunk(s) verwijderen

Je kunt individuele vakjes (chunks) selecteren met een muisklik. Zoek en selecteer de chunks die problemen geven en verwijderd moeten worden. Klik daarna in het menu op `Selection -> Delete selected Chunks`.

![img](https://screensaver01.zap-hosting.com/index.php/s/HDCAEX3iWyjjXQm/download)

De geselecteerde chunks zijn nu verwijderd en je kunt de wereld weer via FTP naar je server uploaden.

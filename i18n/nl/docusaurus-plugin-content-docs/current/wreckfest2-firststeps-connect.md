## Servergegevens ophalen


Je hebt alle relevante info nodig om verbinding te maken met je server. Deze info vind je op het dashboard van het webinterfacepaneel van je game server. Belangrijke gegevens zijn onder andere:

#### Niet zichtbaar


Als de server niet zichtbaar is, kan dat komen doordat de initialisatie niet succesvol is afgerond. Dit kan bijvoorbeeld door een verkeerde configuratie of corrupte bestanden komen. Meer info vind je meestal in de serverconsole of logbestanden.

Heb je nog steeds problemen na het troubleshooten? Geen zorgen, ons supportteam staat dagelijks voor je klaar.

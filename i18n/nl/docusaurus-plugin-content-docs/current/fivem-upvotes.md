## Upvotes vs Burst Upvotes

FiveM Upvotes en FiveM Burst Upvotes zijn beide manieren om de zichtbaarheid en populariteit van een FiveM-server te vergroten. FiveM Upvotes zijn doorlopende stemmen die over een langere periode worden ingezet voor langdurige zichtbaarheid in de server rankings. FiveM Burst Upvotes zijn daarentegen bedoeld voor korte acties, waarbij veel stemmen in korte tijd worden ingezet om de server snel omhoog te krijgen in de rankings.



## Upvotes Kopen

De FiveM Upvotes kun je kopen in [onze shop](https://zap-hosting.com/en/shop/product/fivem-upvotes/). Tijdens het bestelproces kun je het aantal upvotes, de looptijd, je FiveM gebruikersnaam en betaalmethode kiezen. 

:::warning Let op bij het betaalproces
FiveM Upvotes kunnen niet met ZAP Coins worden betaald omdat het een extern product is. Betalen kan via geaccepteerde methodes zoals PayPal of creditcard.
:::

Voor het gebruik van de FiveM upvotes moet je je FiveM-account koppelen. De upvotes worden aan dit account toegekend, waarna je ze in-game kunt toewijzen zoals jij wilt. 

![](https://screensaver01.zap-hosting.com/index.php/s/2fT6CyCfzo4wEe5/download)


Zodra dit bevestigd is, kun je de bestelling afronden. De info wordt direct doorgestuurd naar FiveM en daar verwerkt. Zodra dit proces klaar is, zijn de upvotes voor jou beschikbaar en kun je ze via de interface beheren. De looptijd van de upvotes start ook op dat moment. 



## Upvotes Toewijzen

Zodra de upvotes zichtbaar zijn in de interface, kun je ze via de FiveM client toewijzen. Klik hiervoor op het Upvotes-icoon in de serverlijst bij de server van jouw keuze. 
Als bevestiging verschijnt er een pop-up die aangeeft dat de upvotes succesvol zijn toegewezen.

![](https://screensaver01.zap-hosting.com/index.php/s/onkKXJGpWoGYWa2/download)



## Upvotes Resetten

Mocht je de upvotes aan een andere server willen toewijzen of is er een fout opgetreden bij het toewijzen, dan kun je ze via het dashboard resetten. Het resetten van de upvotes duurt een paar minuten voordat Cfx.re dit heeft overgenomen.

## Introductie
Last Oasis-servers vereisen een aantal stappen om te authenticeren met de officiële site. In deze gids nemen we je mee door de stappen om je eigen realm aan te maken in Last Oasis en een oasis binnen je nieuwe realm op te zetten.

<InlineVoucher />

## Voorbereiding
Een van de vereisten om een nieuw realm op te zetten is het genereren van een key via de officiële site, zodat je kunt verifiëren en authenticeren met de officiële online services.

Ga simpelweg naar [de officiële Last Oasis](https://myrealm.lastoasis.gg/) website en log in met je favoriete platform (Steam of Xbox).

![](https://screensaver01.zap-hosting.com/index.php/s/Z2sPJLFks9LY8cg/preview)

## Server key instellen
Eenmaal ingelogd, heb je al een **CustomerKey** die de key is voor jouw game server.

![](https://screensaver01.zap-hosting.com/index.php/s/DgdDoCHf7DDjZ2s/preview)

Ga naar de **Settings** sectie in de webinterface van je game server en vul deze code in het veld **Customer Key** in. Vergeet niet op te slaan na het aanpassen.

## Provider key instellen
Je moet ook een provider key aanmaken om je game server uniek te identificeren. Klik op de **Add Key** knop om het menu te openen.

![](https://screensaver01.zap-hosting.com/index.php/s/JMQPTZrHcCxgEz2/preview)

Vul in het menu een handige naam en beschrijving voor de key in en klik op **Add**.

![](https://screensaver01.zap-hosting.com/index.php/s/bfiaA2gjjjxESpM/preview)

:::tip
Wil je meerdere Oases tegelijk actief hebben binnen je Realm? Check dan onze [Multiple Oases](lastoasis-multiple-oases.md) gids om te leren hoe je dit aanzet.
:::

Ga daarna weer naar de **Settings** sectie van je game server en vul deze code in het veld **ProviderKey** in. Vergeet niet op te slaan.

## Een Realm aanmaken
Je kunt nu een realm aanmaken. Op dezelfde [officiële Last Oasis](https://myrealm.lastoasis.gg/) website ga je naar de **Realms** sectie en maak je een nieuw Realm aan.

![](https://screensaver01.zap-hosting.com/index.php/s/ydzjKZTgG4L7QpQ/preview)

Kies een naam en vul eventueel extra details in. Als je klaar bent, sla je het Realm op. Dit is nodig om verbinding te maken met je server.

![](https://screensaver01.zap-hosting.com/index.php/s/4rJq5r2zfHx66Qs/preview)

Na het aanmaken zie je een overzicht van het hele Realm waar je extra instellingen kunt aanpassen en nuttige info kunt bekijken.

![](https://screensaver01.zap-hosting.com/index.php/s/j5egcHeicf3QPiR/preview)

## Een Oasis aanmaken
De laatste stap is het aanmaken van een Oasis binnen je Realm. Voor elke game server die je hebt, moet je een aparte Oasis aanmaken. Heb je bijvoorbeeld 2 gameservers, maak dan voor beide een eigen oasis aan.

Klik simpelweg op de **Manage** knop in het Realm overzicht.

![](https://screensaver01.zap-hosting.com/index.php/s/GNDZ7DaeLT2qAoC/preview)

Op de nieuwe pagina kies je een gewenste locatie voor je nieuwe Oasis en klik je op **Add** om te bevestigen.

![](https://screensaver01.zap-hosting.com/index.php/s/dXKXKod6y8NE8SC/preview)

Er verschijnt een nieuw venster waar je een naam kunt instellen en een map kunt selecteren.

![](https://screensaver01.zap-hosting.com/index.php/s/aKJmGmGQmz65Xnz/preview)

![](https://screensaver01.zap-hosting.com/index.php/s/Hrq49c3TArdYXdG/preview)

Als je klaar bent, klik je op **Add Oasis** om de Oasis aan te maken en activeer je deze met de **Activate** knop.

![](https://screensaver01.zap-hosting.com/index.php/s/tfEXd4GgAMDd8sE/preview)

:::tip
Wil je meerdere Oases tegelijk actief hebben binnen je Realm? Check dan onze [Multiple Oases](lastoasis-multiple-oases.md) gids om te leren hoe je dit aanzet.
:::

Je hebt nu succesvol een nieuw Realm en een Oasis opgezet voor je Last Oasis game server.

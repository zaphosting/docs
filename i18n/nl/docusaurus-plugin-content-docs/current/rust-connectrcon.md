## Wat is RCON?

RCON is een protocol om commando’s op de server uit te voeren zonder directe toegang tot de console.

<InlineVoucher />

### WebRCON inschakelen

Voordat je kunt verbinden, moet je eerst zeker weten dat WebRCON aan staat.  
Ga hiervoor naar het tabblad **Instellingen**.

![](https://screensaver01.zap-hosting.com/index.php/s/RqpZcnBcEHY3gN2/preview)

En zet het daarna direct aan op deze pagina.  
![](https://screensaver01.zap-hosting.com/index.php/s/wJ5psnbAneZ6rMD/preview)

:::info
Je moet de server herstarten voordat de wijziging actief wordt. 
:::

### RCON Programma installeren

Je hebt een RCON programma nodig om verbinding te maken. Er zijn verschillende opties beschikbaar.  
Voor dit voorbeeld, en als aanrader, kun je [RustAdmin](https://www.rustadmin.com/) downloaden.

Je kunt het direct via de knop op hun website installeren.  
![](https://screensaver01.zap-hosting.com/index.php/s/zNtESocaQPFwzCg/preview)

## Verbinden via RCON

Open WebAdmin, klik bovenaan op **Server** en daarna op **Connect**.

:::info
Je server moet volledig ONLINE zijn om via RCON bereikbaar te zijn. 
:::

![](https://screensaver01.zap-hosting.com/index.php/s/bHLwGDCqH52Bye8/preview)

Je wordt dan doorgestuurd naar een configuratiepagina.  
Hier vul je de gegevens van jouw server in.

![](https://screensaver01.zap-hosting.com/index.php/s/qmQFd7S79EpYWTE/preview)

:::info
TIP: Je RCON Poort & RCON Wachtwoord vind je terug in het **Instellingen** tabblad waar je WebRCON hebt ingeschakeld. 
:::

Als je alles hebt ingevuld, sla je de configuratie op.

![](https://screensaver01.zap-hosting.com/index.php/s/FDMqdsfAHrbiTpt/preview)

Klik daarna op Server -> Connect.

![](https://screensaver01.zap-hosting.com/index.php/s/4DLGZiQ5X6WJrzA/preview)

Als de verbinding succesvol is, zie je onderaan **Connected** in groene letters.

![](https://screensaver01.zap-hosting.com/index.php/s/qzq68ENHzXGK69T/preview)

Je kunt nu RCON gebruiken om je server te beheren! :)

## Intro

Ben je op zoek naar je eigen server met een krachtige configuratie voor een eerlijke prijs? Dan zit je bij ZAP-Hosting helemaal goed! Bij ZAP-Hosting huur of koop je jouw perfecte dedicated server en krijg je hardware voor een maximale performance-ervaring!

In tegenstelling tot een virtuele server heb je bij een dedicated server je eigen fysieke hardware. Dat betekent dat wanneer je een Dedicated Server huurt/koopt, de volledige kracht van alle componenten alleen voor jou en jouw applicaties beschikbaar is. Virtuele servers draaien daarentegen op een host-systeem waarvan de resources gedeeld worden met andere virtuele servers.

![](https://screensaver01.zap-hosting.com/index.php/s/9YMG6QfsNzzjCSQ/preview)



## Bestellen

Het bestelproces is super simpel. Kies je gewenste configuratie voor je [Dedicated Server](https://zap-hosting.com/en/dedicated-server-hosting/). De setup-tijd van de dedicated server hangt af van de beschikbaarheid. Je checkt dit via het statusicoon naast de vlag bij elke configuratie. De status **Groen** betekent dat de configuratie beschikbaar is en binnen een paar minuten klaarstaat. Staat er **Oranje**, dan is de configuratie niet direct beschikbaar en moet deze nog opgebouwd worden. De setup-tijd is dan ongeveer 5 werkdagen (ma-vr).

![img](https://screensaver01.zap-hosting.com/index.php/s/SD6jF4MENeqaJFZ/preview)

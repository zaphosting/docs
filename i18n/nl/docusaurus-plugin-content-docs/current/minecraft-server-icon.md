## Het icoon toevoegen

Om het icoon toe te voegen heb je FTP-rechten en het icoon zelf nodig.

>Belangrijk: Het icoon moet 64x64 pixels zijn en de naam server-icon.png dragen.

Eerst maak je verbinding via FTP met de game server, hoe dat precies werkt zie je hier: [Toegang via FTP](gameserver-ftpaccess.md)

Zodra de verbinding staat, zie je de serverbestanden zoals:

![](https://screensaver01.zap-hosting.com/index.php/s/jKDxAfWzgiepiTE/preview)

Het icoon moet nu in de hoofdmap geüpload worden, niet in een submap:

![](https://screensaver01.zap-hosting.com/index.php/s/dNWE4W5KQ8XgsQs/preview)

Als dit gedaan is, moet de server gestart of herstart worden via de knoppen rechtsboven in de webinterface van de server:

![](https://screensaver01.zap-hosting.com/index.php/s/D4RGqboisxJoBa9/preview)

## Het icoon checken

Als naam en formaat kloppen, wordt het icoon in multiplayer voor de server getoond:

![](https://screensaver01.zap-hosting.com/index.php/s/qzoySiJAPReZDfc/preview)

Je kunt het icoon op elk moment op deze manier vervangen of verwijderen.

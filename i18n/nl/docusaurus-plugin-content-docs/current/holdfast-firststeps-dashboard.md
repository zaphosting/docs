## Productbeheer

In het **Productbeheer** gedeelte vind je alle benodigde pagina’s om jouw server te beheren. De opties voor het beheren van je server worden hieronder in detail opgesomd. 



### Dashboard

Het **Dashboard** van je server is het hart van het beheer. Hier worden alle basis- en essentiële info over je service overzichtelijk weergegeven. Denk aan informatie over locatie, status, IP: Poort, gebruikt geheugen, lopend spel, servernaam, map en het aantal beschikbare slots. Daarnaast zie je ook het gebruik van CPU, geheugen en RAM.

![img](https://screensaver01.zap-hosting.com/index.php/s/p6RGzMJnAGRJk9q/preview)

In het dashboard kun je ook je service starten, stoppen of herstarten, zodat je volledige controle hebt over de werking. Verder heb je toegang tot de live console om realtime info over de serverstatus te krijgen en direct actie te ondernemen indien nodig.






### DDoS Manager


Bekijk eerdere DDoS-aanvallen om bedreigingen en patronen beter in te schatten. Schakel meldingen in voor lopende DDoS-aanvallen op jouw server, zodat je snel kunt reageren als er iets gebeurt.

Afhankelijk van de serverlocatie en de beschikbare DDoS-bescherming kun je ook actieve aanvallen in realtime monitoren. Dit omvat gedetailleerde verkeersinzichten, waarbij zowel legitiem verkeer als het door mitigatiesystemen geblokkeerde verkeer wordt getoond. Deze transparantie helpt je te begrijpen hoe de bescherming werkt en stelt je in staat potentiële risico’s vroeg te signaleren.

:::info Geen toegang tot DDoS Manager?
Voor het gebruik van de DDoS Manager heb je een dedicated IP-adres of de extra optie "DDoS Manager Overzicht" nodig. Als deze onderdelen niet in je huidige pakket zitten, kun je ze altijd toevoegen door je service te upgraden.
:::

![img](https://screensaver01.zap-hosting.com/index.php/s/ScCCCY52CMLgfyE/preview)





:::warning
De DDoS Manager is alleen te gebruiken als je een eigen IP-adres hebt of de optie "DDoS Manager Overzicht" als extra optie hebt geboekt. Ontbreken deze onderdelen in je huidige pakket, dan kun je ze altijd toevoegen door te upgraden.
:::



### Logbestanden

Heb je een technisch probleem en weet je niet waar het door komt? Een eerste mogelijke oplossing is het checken van de **Logbestanden**. Deze bestanden bevatten veel info over de server, zoals actuele gebeurtenissen, aanwijzingen of foutmeldingen.

Foutmeldingen kunnen vooral helpen om de oorzaak van een probleem te vinden en beter te begrijpen. Vaak kun je problemen opsporen en oplossen door gericht naar de logbestanden te kijken.

:::warning
Helpen de gevonden info niet of snap je het niet? Geen stress! We zijn dagelijks bereikbaar via support. Maak gewoon een **[Ticket](https://zap-hosting.com/en/customer/support/)** aan en leg je probleem uit. We kijken er zo snel mogelijk naar en helpen je het probleem op te lossen! :)
:::


### Gebeurtenislog

De **Gebeurtenislog** geeft je een compleet overzicht van alle activiteiten rondom het serverbeheer. Hier zie je wanneer een server gestart, gestopt of gecrasht is, en wanneer updates, backups of nieuwe installaties zijn uitgevoerd.

![img](https://screensaver01.zap-hosting.com/index.php/s/xNzsaMbW5BS5KJC/preview)



### Serviceonderbrekingen

Was je server tijdelijk offline en snap je niet waarom? Kijken in de gebeurtenislog en logbestanden gaf ook geen info? Het kan zijn dat het probleem niet direct bij jouw server ligt, maar bij het host-systeem waarop jouw server draait.

Als ons monitoringsysteem een algemene storing detecteert, worden onze techneuten automatisch geïnformeerd en pakken ze het probleem zo snel mogelijk aan.

In dit onderdeel vind je een overzicht van de storingsgeschiedenis van het host-systeem waarop jouw game server draait. Meer info en de laatste status van onze services vind je op onze nieuwe [statuspagina](https://status.zap-hosting.com/).












## Instellingen

In het **Instellingen** gedeelte vind je alle pagina’s die je nodig hebt om je server te configureren. De beschikbare opties voor het instellen van je server worden hieronder in detail opgesomd.



### Instellingen

Onder Instellingen vind je een overzicht van de meest gebruikte opties voor je server, inclusief een korte uitleg. Deze pagina is vooral aan te raden voor beginners in serverbeheer.

Als je al wat zekerder bent en preciezere aanpassingen wilt doen, raden we aan om de Config Editor te gebruiken. Meer info daarover vind je in de categorie **Configs**.



### Configs

De config-bestandseditor laat je de configuratiebestanden direct via de website bewerken, op de klassieke manier zonder vereenvoudigde weergave. Hiermee kun je de instellingen van je server helemaal naar eigen wens aanpassen.

Met de editor heb je directe toegang tot alle opties en kun je ze precies instellen. Dit is vooral handig voor gevorderde gebruikers die volledige controle willen en precies weten wat ze willen veranderen.

:::warning
Merk je dat sommige wijzigingen deels worden overschreven? Houd er rekening mee dat bepaalde opties worden overgenomen van de instellingenpagina. Wil je deze direct in de config aanpassen, schakel dan eerst de betreffende opties uit op de instellingenpagina.
:::

## Tools

In het **Tools** gedeelte van je navigatiepaneel vind je verschillende sub-items die je helpen bij het beheren van je server. Dit zijn onder andere **FTP Browser**, **Databases** en **Backups**. Hieronder krijg je een uitgebreide introductie van deze onderdelen.



### FTP-Browser

Met de **FTP Browser** krijg je alle info die je nodig hebt om verbinding te maken met de FTP-toegang. Via FTP heb je directe toegang tot de bestanden van je game server. Voor optimaal gebruik raden we een externe FTP-client aan zoals **FileZilla** of **WinSCP**. Ben je niet bekend met FTP? Check dan onze [FTP Access](gameserver-ftpaccess.md) gids, die kan je helpen.

![img](https://screensaver01.zap-hosting.com/index.php/s/sX85i9HyzR9wmAM/preview)

**FileZilla:** FileZilla is waarschijnlijk de makkelijkste en bekendste FTP-client. Gewoon installeren, starten en in de bovenste balk je inloggegevens invullen die je iets hoger op deze pagina vindt. Met een snelle klik op "Verbinden" ben je direct op de server en kun je bestanden beheren.

**WinSCP:** WinSCP is eigenlijk bedoeld voor SCP en SFTP (de versleutelde varianten van FTP), maar kan ook prima met gewoon FTP overweg. Het programma is wat uitgebreider dan FileZilla en meer geschikt voor gevorderde gebruikers.



### Databases

Naast onze game server producten bieden we ook 10 inbegrepen databases aan. Deze zijn nodig als er data in een database opgeslagen moet worden. Onder **Databases** kun je een database aanmaken en de inloggegevens bekijken. Je kunt ook de backup-optie activeren en bestaande database-backups importeren.

![img](https://screensaver01.zap-hosting.com/index.php/s/WFLwCbZSQfYjrmD/preview)



### Backups

Onder **Backups** kun je handmatig en automatisch backups maken van je server en bijbehorende database. Zo kun je makkelijker je data herstellen bij problemen of dataverlies.

In de instellingen kun je aangeven of backups automatisch gemaakt moeten worden en of dit dagelijks of wekelijks moet gebeuren. Ook kun je instellen of de oudste backups verwijderd mogen worden als de opslagruimte beperkt raakt.

![img](https://screensaver01.zap-hosting.com/index.php/s/nm9za5LTMEcNsga/preview)

We bieden standaard 10 GB opslagruimte voor backups op onze opslagserver bij game server producten. Heb je meer ruimte nodig, dan kun je dit tegen betaling upgraden. Zo zorg je dat je altijd genoeg ruimte hebt voor backups en ook oudere backups kunt bewaren.

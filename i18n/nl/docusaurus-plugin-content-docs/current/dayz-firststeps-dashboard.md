## Productbeheer

In het onderdeel **Productbeheer** vind je alle benodigde pagina’s om jouw server te beheren. De opties voor het beheren van je server worden hieronder in detail opgesomd.


### Dashboard

Het **Dashboard** van je server is het kloppende hart van de administratie. Alle basis- en essentiële info over jouw service wordt hier overzichtelijk weergegeven. Denk aan gegevens over de locatie, status, IP: Poort, gebruikt geheugen, lopend spel, servernaam, map en het aantal beschikbare slots. Daarnaast zie je ook het gebruik van CPU, geheugen en RAM.

![](https://screensaver01.zap-hosting.com/index.php/s/TnzM6NSMzjJAQmr/preview)

In het dashboard kun je ook je service starten, stoppen of herstarten, zodat je volledige controle hebt over de werking. Verder heb je toegang tot de live console om real-time info over de serverstatus te krijgen en direct actie te ondernemen als dat nodig is.




### Statistieken

Onder **Statistieken** krijg je een uitgebreid overzicht van de belasting van je server. Je vindt hier info over verschillende aspecten zoals de bezetting van de beschikbare slots, de CPU en het geheugen. Je kunt deze data bekijken voor verschillende periodes, bijvoorbeeld de laatste **60 minuten, 24 uur, 3 dagen, 7 dagen** of **30 dagen.**

![](https://screensaver01.zap-hosting.com/index.php/s/N8jcmdqqG2Xt4Bk/preview)

Dankzij deze info krijg je goed inzicht in het gebruik van je server. Daarnaast helpen deze statistieken vaak om mogelijke prestatieproblemen te herkennen en de oorzaken te achterhalen. Zo kun je bijvoorbeeld terugzien wanneer de belasting is gestegen en of dit verband houdt met bepaalde gebeurtenissen of wijzigingen. Dit is vooral belangrijk bij plotselinge problemen, zodat je gericht stappen kunt ondernemen om te troubleshooten en systematisch mogelijke oorzaken te analyseren.



### DDoS Manager


Bekijk eerdere DDoS-aanvallen om bedreigingen en patronen beter in te schatten. Schakel meldingen in voor lopende DDoS-aanvallen op jouw server, zodat je snel kunt reageren als er iets gebeurt.

Afhankelijk van de serverlocatie en de beschikbare DDoS-bescherming kun je ook actieve aanvallen in realtime monitoren. Dit omvat gedetailleerde verkeersinzichten, waarbij zowel legitiem verkeer als het door mitigatiesystemen geblokkeerde verkeer wordt getoond. Deze transparantie helpt je te begrijpen hoe de bescherming werkt en stelt je in staat potentiële risico’s vroegtijdig te herkennen.

:::info Geen toegang tot DDoS Manager?
Het gebruik van de DDoS Manager vereist een dedicated IP-adres of de extra optie "DDoS Manager Overzicht". Als je huidige pakket deze onderdelen niet bevat, kun je ze altijd toevoegen door je service te upgraden.
:::

![](https://screensaver01.zap-hosting.com/index.php/s/ScCCCY52CMLgfyE/preview)







### Logbestanden

Heb je een technisch probleem en weet je niet waar het door komt? Een eerste mogelijke oplossing is het controleren van de **Logbestanden**. Deze bestanden bevatten veel info over de server, zoals actuele gebeurtenissen, aanwijzingen of foutmeldingen.

Foutmeldingen kunnen vooral helpen om de oorzaak van een probleem te identificeren en beter te begrijpen. Vaak kunnen problemen worden opgespoord en opgelost door gericht naar de logbestanden te kijken.

:::info
Helpen de gevonden info niet of snap je het niet? Geen stress! We zijn dagelijks bereikbaar via support. Maak gewoon een **[Ticket](https://zap-hosting.com/en/customer/support/)** aan en leg je probleem uit. We kijken er zo snel mogelijk naar en helpen je het probleem op te lossen! :)
:::



### Gebeurtenislog

De **Gebeurtenislog** geeft je een compleet overzicht van alle activiteiten rondom serverbeheer. Hier kun je altijd terugzien wanneer een server is gestart, gestopt of gecrasht, en wanneer updates, backups of nieuwe installaties zijn uitgevoerd.

![](https://screensaver01.zap-hosting.com/index.php/s/xNzsaMbW5BS5KJC/preview)



### Serviceonderbrekingen

Was je server tijdelijk offline en snap je niet waarom? Kijken in de gebeurtenislog en logbestanden gaf ook geen info? Het kan zijn dat de oorzaak niet direct bij jouw server ligt, maar bij het host-systeem waarop jouw server draait.

Als ons monitoringsysteem een algemene storing detecteert, worden onze techneuten automatisch geïnformeerd en pakken ze het probleem zo snel mogelijk aan.

In dit onderdeel vind je een overzicht van de storingsgeschiedenis van het host-systeem waarop jouw game server draait. Meer info en de laatste status van onze services vind je op onze nieuwe [statuspagina](https://status.zap-hosting.com/).












## Instellingen

In het onderdeel **Instellingen** vind je alle benodigde pagina’s om je server te configureren. De beschikbare opties voor het instellen van je server worden hieronder in detail opgesomd.



### Instellingen

Onder instellingen vind je een overzicht van de meest gebruikte opties voor je server. Deze worden ook kort toegelicht. Het gebruik van de instellingenpagina is vooral aan te raden voor beginners in serverbeheer.

![](https://screensaver01.zap-hosting.com/index.php/s/9Q4cpnSePrgfPTy/preview)

Hier kun je bijvoorbeeld de volgende aanpassingen doen: 

- Basisinstellingen: **Servernaam**, Game server wachtwoord, Mods, Server mods, Admin wachtwoord, Missie, RCON wachtwoord, RCON aan/uit zetten

Als je wat zekerder bent en preciezere aanpassingen wilt maken, raden we aan om de Config Editor te gebruiken. Meer info hierover vind je in de categorie **Configs**.



### Configs

De config-bestandseditor laat je de configuratiebestanden direct via de website bewerken, op de klassieke manier zonder vereenvoudigde weergave. Hiermee kun je de instellingen van je server helemaal naar eigen wens aanpassen.

Met de editor heb je directe toegang tot alle beschikbare opties en kun je ze precies instellen. Dit is vooral handig voor gevorderde gebruikers die volledige controle willen over hun serverinstellingen en precies weten wat ze willen aanpassen.

:::info
Merk je dat sommige wijzigingen deels worden overschreven? Houd er rekening mee dat bepaalde opties worden overgenomen van de instellingenpagina. Wil je deze direct in de config aanpassen, dan moet je eerst de betreffende opties op de instellingenpagina uitschakelen.
:::

Onder **Games** kun je altijd de game-variant van je bestaande game wijzigen of een andere game installeren. Games met dezelfde slotprijs kun je direct kiezen onder **Beschikbare Games**. Voor alle andere games moet eerst de slotprijs worden aangepast. Meer info hierover vind je in de [Game Change](gameserver-gameswitch.md) gids.

### Versies

Onder **Versies** kun je de huidige versie van je game server bekijken en beheren. Hier kun je altijd updaten naar de nieuwste versie of automatische updates inschakelen, zodat je server altijd up-to-date is.

![](https://screensaver01.zap-hosting.com/index.php/s/w3f4FSLpaPKgbcE/preview)

Daarnaast kun je ook e-mailmeldingen voor automatische updates inschakelen, zodat je op de hoogte blijft van updates en altijd up-to-date bent.

Wil je serverherstarts op vaste tijden plannen? Met de Herstartplanner kun je automatische herstarts van je game server inplannen. Hiermee stel je herstarts in op specifieke tijden of met vaste tussenpozen.

![](https://screensaver01.zap-hosting.com/index.php/s/Y6WciDS7YP98P4m/preview)

## Tools

In het onderdeel **Tools** in je navigatiepaneel vind je verschillende sub-items die je helpen bij het beheren van je server. Dit zijn onder andere **FTP Browser**, **Databases** en **Backups**. Hieronder krijg je een uitgebreide introductie van deze onderdelen.



### FTP-Browser

Met de **FTP Browser** krijg je alle info die je nodig hebt om verbinding te maken met de FTP-toegang. De FTP-toegang geeft je directe toegang tot de bestanden van je game server. Voor optimaal gebruik raden we een externe FTP-client aan zoals **Filezilla** of **WinSCP**. Als je niet bekend bent met FTP, kan de [Toegang via FTP](gameserver-ftpaccess.md) gids je helpen.

![](https://screensaver01.zap-hosting.com/index.php/s/fZSYpSi4KwPos5P/preview)

**FileZilla:**  FileZilla is de makkelijkste en meest bekende FTP-client. Gewoon installeren, starten en je inloggegevens invullen in de bovenbalk, die je op deze pagina iets hoger vindt. Na een klik op "Verbinden" ben je direct ingelogd op de server en kun je bestanden beheren.

**WinSCP:**  WinSCP is eigenlijk bedoeld voor SCP en SFTP (de versleutelde opvolgers van FTP), maar kan ook FTP aan. Het programma is uitgebreider dan FileZilla en geschikt voor gevorderde gebruikers.



### Databases

Naast onze game server producten bieden we ook 10 inbegrepen databases aan. Deze databases zijn nodig wanneer data in een database opgeslagen moet worden. Onder **Databases** kun je een database aanmaken en de inloggegevens bekijken. Je kunt ook de backup-optie activeren en bestaande databasebackups importeren.

![](https://screensaver01.zap-hosting.com/index.php/s/WFLwCbZSQfYjrmD/preview)



### Backups

Onder **Backups** kun je handmatige en automatische backups maken van je server en de bijbehorende database. Zo kun je makkelijker je data herstellen bij problemen of dataverlies.

In de instellingen kun je aangeven of backups automatisch gemaakt moeten worden en of dit dagelijks of wekelijks moet gebeuren. Daarnaast kun je bepalen of de oudste backups verwijderd mogen worden als de opslagruimte beperkt raakt.

![](https://screensaver01.zap-hosting.com/index.php/s/nm9za5LTMEcNsga/preview)

We bieden standaard 10 GB opslagruimte op de opslagserver voor backups bij onze game server producten. Heb je meer ruimte nodig, dan kun je dit tegen betaling upgraden. Zo zorg je dat je altijd genoeg ruimte hebt voor je backups en ook oudere backups kunt bewaren.

## Introductie

Je bent geïnteresseerd in een **Dedicated Server** of hebt er al één, maar mist nog een duidelijk overzicht van de administratie en waar je alle benodigde opties kunt vinden? Vooral in het begin kan dat best een uitdaging zijn. Geen zorgen, hieronder geven we je een uitgebreide introductie in het beheer van jouw service, zodat je het beste overzicht krijgt.

Na het doorlopen van deze gids kun je jouw Dedicated Server makkelijk en efficiënt beheren en vind je alle benodigde opties en functies snel en eenvoudig terug. Hier krijg je een gestructureerd overzicht dat je stap voor stap helpt om vertrouwd te raken met de Dedicated Server administratie en je kennis gericht uit te breiden. Zo zorg je ervoor dat jouw **Dedicated Server** optimaal is ingesteld op jouw wensen en haal je het maximale uit de beschikbare opties.



## Productbeheer

In het onderdeel **Productbeheer** vind je alle benodigde pagina’s om jouw Dedicated Server te beheren. De opties voor het beheren van je Dedicated Server worden hieronder gedetailleerd opgesomd.



### Dashboard

Het dashboard geeft een centraal overzicht van alle belangrijke informatie rondom de Dedicated Server. Je ziet de actuele status, serveradres, hostname, geïnstalleerd besturingssysteem, systeembelasting inclusief CPU- en RAM-gebruik, en netwerkverkeer. Dit overzicht ondersteunt je bij het effectief monitoren en beheren van de virtuele serveromgeving.

![img](https://screensaver01.zap-hosting.com/index.php/s/QEnaS6N7MqHejtk/preview)



### Event log

Houd alle admin taken en systeemgebeurtenissen op één plek in de gaten. De Event log verzamelt alle relevante activiteiten, zodat je altijd weet wie wat heeft aangepast, wanneer dat gebeurde en wat er precies is beïnvloed.

![img](https://screensaver01.zap-hosting.com/index.php/s/akKpTx2XzDKy7qc/preview)



### Statistieken
Gedetailleerde verkeerscijfers laten precies zien hoeveel data jouw server verwerkt, zowel inkomend als uitgaand. Actuele temperatuurmetingen en ventilatorsnelheden helpen je om je hardware altijd binnen een veilige operationele marge te houden.

![img](https://screensaver01.zap-hosting.com/index.php/s/B7yLamtJrdALpPb/preview)



### Hardware status

Houd alle essentiële hardwaregegevens overzichtelijk bij elkaar. Check welke firmwareversies er momenteel geïnstalleerd zijn, inclusief System ROM, iLO en de Storage Controller, en monitor de status van je CPU, RAM, netwerkinterfaces en aangesloten schijven realtime.

Gedetailleerde hardwaredata helpt je om potentiële problemen vroegtijdig te signaleren en zorgt ervoor dat je server betrouwbaar blijft draaien. Je hebt ook toegang tot een duidelijk logboek van alle onderhoudswerkzaamheden, zodat je altijd weet welke updates of reparaties zijn uitgevoerd en wanneer.

![img](https://screensaver01.zap-hosting.com/index.php/s/9CsZGarzsdMP5Ea/preview)

## Instellingen

In het onderdeel **Instellingen** vind je alle benodigde pagina’s om je Dedicated Server te configureren. De beschikbare opties voor het instellen van je Dedicated Server worden hieronder uitgebreid beschreven.

### Eerste installatie
Voordat je je dedicated server productief kunt gebruiken, is een eerste setup nodig. Dit omvat het activeren van de iLO managementinterface, het selecteren en mounten van een ISO-image, en het installeren van het gewenste besturingssysteem. De setup-assistent begeleidt je stap voor stap door dit proces, zodat je ook zonder ervaring soepel aan de slag kunt. Gedetailleerde instructies vind je in onze [Eerste Setup](dedicated-setup.md) gids.

![img](https://screensaver01.zap-hosting.com/index.php/s/MnZKXAGGTqs9Xdp/download)

### Instellingen

Pas configuratie-opties aan voor je dedicated server. Hier heb je volledige controle om instellingen zoals de power mode te wijzigen, zodat je je server optimaal kunt afstemmen op jouw specifieke wensen.

![img](https://screensaver01.zap-hosting.com/index.php/s/WiHSELJNc5icsyQ/preview)



### DDoS Manager

Bekijk eerdere DDoS-aanvallen om bedreigingen en patronen beter in te schatten. Schakel notificaties in voor lopende DDoS-aanvallen op je server, zodat je snel kunt reageren als er iets gebeurt.

Afhankelijk van de serverlocatie en de beschikbare DDoS-bescherming kun je ook actieve aanvallen realtime monitoren. Dit omvat gedetailleerde verkeersinzichten, waarbij zowel legitiem verkeer als het door mitigatiesystemen geblokkeerde verkeer wordt getoond. Deze transparantie helpt je te begrijpen hoe de bescherming werkt en stelt je in staat potentiële risico’s vroeg te herkennen.

![img](https://screensaver01.zap-hosting.com/index.php/s/ScCCCY52CMLgfyE/preview)



### Gameserver / TS3

Met deze functie zet je eenvoudig gameserver- en voiceserverdiensten op en beheer je ze met een paar klikken. De webinterface regelt het hele installatieproces voor je, dus je hebt geen diepgaande technische kennis nodig.

Je kunt nieuwe gameservers of voiceservers aanmaken wanneer je wilt, instellingen zoals slots of resources aanpassen en ze comfortabel bedienen vanuit je ZAP webdashboard. Dit maakt het beheren van meerdere servers snel, flexibel en simpel. Alles op één plek.

:::warning Compatibiliteit Besturingssysteem 
Deze functie werkt alleen met geselecteerde Linux-besturingssystemen. Voor meer info over het opzetten en gebruiken van de GS/TS3-interface, check onze [GS/TS3 Interface](dedicated-linux-gs-interface.md) gids.
:::

![img](https://screensaver01.zap-hosting.com/index.php/s/nd6YF93omGcApC8/preview)



### rDNS aanpassen

Pas de reverse DNS-entry (PTR-record) aan voor het IP-adres van je server zodat deze naar de juiste hostname verwijst. Een correct geconfigureerde rDNS zorgt ervoor dat de naam van je Dedicated Server goed wordt opgelost en is vooral belangrijk als je een mailserver draait. Zonder een geldige rDNS kunnen uitgaande mails als spam worden gemarkeerd of niet betrouwbaar worden afgeleverd.

![img](https://screensaver01.zap-hosting.com/index.php/s/LpdMgD6T39tXiNK/preview)



### IP-adressen

Bekijk in één oogopslag alle IP-adressen die aan je server zijn toegewezen. Dit overzicht toont belangrijke configuratiegegevens zoals IP-adres, subnetmasker en gateway. Je kunt ook checken of een IP bereikbaar is, wat helpt om te verifiëren dat je netwerkinstellingen correct zijn ingesteld in het besturingssysteem.

![img](https://screensaver01.zap-hosting.com/index.php/s/a64XRkWn2EA4Nef/preview)

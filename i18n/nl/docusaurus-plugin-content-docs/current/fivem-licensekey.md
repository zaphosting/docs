## Intro

Standaard krijg je bij je FiveM Game Server een automatisch gegenereerde license key. Het aanpassen hiervan en het gebruiken van je eigen FiveM license key kan handig zijn om verschillende redenen, zoals toegang tot je FiveM Element Club abonnement voordelen zoals Onesync, het veranderen van de server eigenaar naar jouw naam en nog veel meer. In deze guide leggen we uit hoe je je eigen license key maakt en toevoegt aan je FiveM Game Server.

## Key Toevoegen aan Server

Met je license key klaar, moet je nu de bestaande license key op je FiveM game server aanpassen en vervangen door de nieuwe. Hiervoor log je in op het txAdmin paneel van je FiveM server via het webinterface dashboard van je server. Gebruik de inloggegevens die op die pagina staan.

Eenmaal in txAdmin, ga je links naar de **CFG Editor** om het `server.cfg` bestand te openen. Zoek in de editor de regel met de `sv_licensekey` parameter en vervang deze door jouw custom license key. Klik op opslaan als je klaar bent.

![](https://screensaver01.zap-hosting.com/index.php/s/2E8j9jtykcjwF7L/preview)

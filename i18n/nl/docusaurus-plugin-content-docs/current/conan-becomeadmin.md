## Introductie
Het toekennen van administratorrechten geeft je eenvoudige en uitgebreide controle over je server. Als admin kun je alle beschikbare opties en functies van het spel direct in de game gebruiken. Hieronder leggen we stap voor stap uit hoe je administratorrechten voor jouw server instelt.  
<InlineVoucher />

## Configuratie
Om jezelf of andere spelers adminrechten te geven, moet je eerst een admin-wachtwoord instellen. Dit wachtwoord stel je in bij de **Instellingen** van je server, te vinden in het dashboard.

![](https://screensaver01.zap-hosting.com/index.php/s/gpHQXB9tk46RpLL/preview)

Op de **Instellingen** pagina zoek je naar "Admin Password" en verander je het naar een wachtwoord dat jij wilt gebruiken.

![](https://screensaver01.zap-hosting.com/index.php/s/bKWx4qCj3aj6agA/preview)

Nadat je het wachtwoord hebt ingesteld, sla je de wijzigingen op en herstart je je server Ã©Ã©n keer. Daarna kun je verbinding maken met je server en via de ESC-toets naar Serverinstellingen gaan. Daar voer je het Admin Password in om admin te worden.

![](https://screensaver01.zap-hosting.com/index.php/s/giLP794irsw8bjK/preview)

Vanaf dat moment kun je alle servercommands als admin uitvoeren. Hieronder vind je een overzicht van de meest gebruikte commands.

## Veelgebruikte Commands

| Command                                                | Beschrijving                                                  |
| ------------------------------------------------------ | ------------------------------------------------------------- |
| MakeMeAdmin [AdminPassword]                            | Zet de Admin Mode aan                                         |
| MakeMeNormal                                           | Zet de Admin Mode uit                                         |
| LearnEmote [emote]                                     | Typ /emote [emote] in de chat of houd R ingedrukt voor het emote-menu. Admin mode vereist |
| God                                                    | Zet God mode aan/uit                                          |
| PlayerCanBuildEverywhere [PlayerName]                  | Laat de opgegeven speler overal bouwen, ook waar normaal niet mag |
| Teleport                                               | Teleporteer naar een oppervlak of de lucht                   |
| TeleportPlayer X[float] Y[float] Z[float]              | Teleporteer jezelf of een andere speler naar de opgegeven coÃ¶rdinaten |
| TeleportToPlayer [PlayerName]                          | Teleporteer jezelf naar de opgegeven speler                   |
| Fly                                                    | Zet vliegmodus aan                                           |
| Ghost                                                  | Zet ghostmodus aan (onzichtbaar, noclip)                      |
| SetServerSetting serverSetting[FString] Value[FString] | Hiermee kun je serverinstellingen uit ServerSettings.ini tijdens het spel aanpassen |

Meer admin commands vind je op de [Gamepedia](https://conanexiles.gamepedia.com/Admin_Panel) van Conan Exiles.

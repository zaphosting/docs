## Productbeheer

In het **Productbeheer** vind je alle benodigde pagina’s om jouw server te beheren. Hieronder staan de opties voor het beheren van je server in detail beschreven.

### Dashboard

Het **Dashboard** van je server is het kloppende hart van de administratie. Hier worden alle basis- en essentiële info over jouw service overzichtelijk weergegeven. Denk aan locatie, status, IP: Poort, gebruikt geheugen, lopend spel, servernaam, map en het aantal beschikbare slots. Ook de belasting van CPU, geheugen en RAM wordt getoond.

![img](https://screensaver01.zap-hosting.com/index.php/s/GEB3nyMA3pRdjBy/preview)

In het dashboard kun je je service starten, stoppen of herstarten, zodat je volledige controle hebt over de werking. Daarnaast heb je toegang tot de live console voor realtime info over de serverstatus en kun je direct actie ondernemen als dat nodig is.

### Statistieken

Onder **Statistieken** krijg je een uitgebreid overzicht van de belasting van je server. Je vindt hier info over verschillende aspecten zoals de bezetting van de slots, CPU en geheugen. Je kunt deze data bekijken voor verschillende periodes, bijvoorbeeld de laatste **60 minuten, 24 uur, 3 dagen, 7 dagen** of **30 dagen.**

![img](https://screensaver01.zap-hosting.com/index.php/s/N8jcmdqqG2Xt4Bk/preview)

Dankzij deze info krijg je goed inzicht in het gebruik van je server. Deze statistieken helpen ook vaak bij het opsporen van mogelijke prestatieproblemen en het terugvinden van de oorzaak. Zo kun je bijvoorbeeld zien wanneer de belasting is gestegen en of dat samenhangt met bepaalde gebeurtenissen of wijzigingen. Dit is vooral handig bij plotselinge problemen, zodat je gericht kunt troubleshooten en systematisch mogelijke oorzaken kunt analyseren.

### DDoS Manager

Bekijk eerdere DDoS-aanvallen om bedreigingen en patronen beter in te schatten. Schakel notificaties in voor lopende DDoS-aanvallen op jouw server, zodat je snel kunt reageren bij een incident.

Afhankelijk van de serverlocatie en de beschikbare DDoS-bescherming kun je ook actieve aanvallen realtime monitoren. Dit omvat gedetailleerde verkeersinzichten, waarbij zowel legitiem verkeer als het door mitigatiesystemen geblokkeerde verkeer wordt getoond. Deze transparantie helpt je te begrijpen hoe de bescherming werkt en stelt je in staat om potentiële risico’s vroegtijdig te herkennen.

:::info Geen toegang tot DDoS Manager?
Voor het gebruik van de DDoS Manager is een dedicated IP-adres of de extra optie "DDoS Manager Overzicht" vereist. Als deze onderdelen niet in je huidige pakket zitten, kun je ze altijd toevoegen door je service te upgraden.
:::

![img](https://screensaver01.zap-hosting.com/index.php/s/ScCCCY52CMLgfyE/preview)

:::warning
De DDoS Manager is alleen te gebruiken als je een eigen IP-adres hebt of de optie "DDoS Manager Overzicht" als extra optie hebt geboekt. Ontbreken deze onderdelen in je pakket, dan kun je ze altijd toevoegen via een upgrade.
:::

### Logbestanden

Heb je een technisch probleem en weet je niet waar het door komt? Een eerste stap is het checken van de **Logbestanden**. Deze bestanden bevatten veel info over de server, zoals actuele gebeurtenissen, hints of foutmeldingen.

Foutmeldingen helpen vaak om de oorzaak van een probleem te vinden en beter te begrijpen. Vaak kun je problemen opsporen en oplossen door gericht in de logbestanden te kijken.

:::warning
Snap je de info niet of helpt het niet? Geen stress! We zijn dagelijks bereikbaar via support. Maak gewoon een **[Ticket](https://zap-hosting.com/en/customer/support/)** aan en leg je probleem uit. We kijken er zo snel mogelijk naar en helpen je het probleem op te lossen! :)
:::

### Gebeurtenislog

De **Gebeurtenislog** geeft je een compleet overzicht van alle activiteiten rondom serverbeheer. Hier zie je wanneer een server is gestart, gestopt of gecrasht, en wanneer updates, backups of nieuwe installaties zijn uitgevoerd.

![img](https://screensaver01.zap-hosting.com/index.php/s/xNzsaMbW5BS5KJC/preview)

### Live Console

De **Live Console** toont je realtime alle actuele gebeurtenissen op je server. Je ziet hier info over spelersactiviteiten, foutmeldingen en systeemberichten. Dit overzicht geeft je een compleet beeld van wat er op je server gebeurt en helpt je problemen vroegtijdig te signaleren.

![img](https://screensaver01.zap-hosting.com/index.php/s/bDqw5G9fLi5mxAg/preview)

Naast het monitoren van events kun je ook direct commando’s naar de server sturen via de **Live Console**. Handig om instellingen aan te passen, acties uit te voeren of snel te reageren op problemen.

### Serviceonderbrekingen

Was je server tijdelijk offline en weet je niet waarom? Kijken in de gebeurtenislog en logbestanden gaf ook geen info? Het kan zijn dat het probleem niet bij jouw server ligt, maar bij het host-systeem waarop je server draait.

Als ons monitoringsysteem een algemene storing detecteert, worden onze techneuten automatisch geïnformeerd en pakken ze het probleem zo snel mogelijk aan.

In deze sectie vind je een overzicht van de storingsgeschiedenis van het host-systeem waarop jouw game server draait. Meer info en de laatste status vind je op onze nieuwe [statuspagina](https://status.zap-hosting.com/).

## Instellingen

In de sectie **Instellingen** vind je alle pagina’s om je server te configureren. Hieronder staan de beschikbare opties voor het instellen van je server in detail.

### Instellingen

Onder Instellingen vind je een overzicht van de meest gebruikte opties voor je server, inclusief korte uitleg. Deze pagina is vooral aan te raden voor beginners in serverbeheer.

Ben je wat zekerder en wil je preciezere aanpassingen doen? Gebruik dan de Config Editor. Meer info hierover vind je in de categorie **Configs**.

### Configs

De config-bestanden editor laat je de configuratiebestanden direct via de website bewerken, zonder de vereenvoudigde weergave. Hiermee kun je de instellingen van je server helemaal naar eigen wens aanpassen.

Met de editor heb je directe toegang tot alle opties en kun je ze precies instellen. Dit is vooral handig voor gevorderde gebruikers die volledige controle willen en precies weten wat ze willen wijzigen.

:::warning
Merk je dat sommige wijzigingen deels worden overschreven? Houd er rekening mee dat bepaalde opties worden overgenomen van de instellingenpagina. Wil je deze direct in de config aanpassen, schakel dan eerst de betreffende opties uit op de instellingenpagina.
:::

## Tools

In het **Tools** gedeelte van je navigatiepaneel vind je verschillende subitems die je helpen bij het beheren van je server. Dit zijn onder andere **FTP Browser**, **Databases** en **Backups**. Hieronder krijg je een uitgebreide introductie van deze onderdelen.

### FTP-Browser

Met de **FTP Browser** krijg je alle info die je nodig hebt om verbinding te maken met de FTP-toegang. FTP geeft je directe toegang tot de bestanden van je game server. We raden aan een extern FTP-programma te gebruiken zoals **Filezilla** of **WinSCP** voor optimaal gebruik. Ben je niet bekend met FTP? Bekijk dan onze [FTP Access](gameserver-ftpaccess.md) gids, die je kan helpen.

![img](https://screensaver01.zap-hosting.com/index.php/s/sX85i9HyzR9wmAM/preview)

**FileZilla:** FileZilla is waarschijnlijk de eenvoudigste en bekendste FTP-client. Gewoon installeren, starten en in de bovenste balk je inloggegevens invullen die je op deze pagina iets hoger vindt. Met een snelle klik op "Verbinden" ben je direct op de server en kun je bestanden beheren.

**WinSCP:** WinSCP is eigenlijk bedoeld voor SCP en SFTP (de versleutelde varianten van FTP), maar kan ook prima met gewoon FTP overweg. Het programma is wat uitgebreider dan FileZilla en meer geschikt voor gevorderde gebruikers.

### Databases

Naast onze game server producten bieden we ook 10 inbegrepen databases aan. Deze zijn nodig als data in een database opgeslagen moet worden. Onder **Databases** kun je een database aanmaken en de inloggegevens bekijken. Je kunt ook de backup-optie activeren en bestaande databasebackups importeren.

![img](https://screensaver01.zap-hosting.com/index.php/s/WFLwCbZSQfYjrmD/preview)

### Backups

Onder **Backups** kun je handmatige en automatische backups maken van je server en bijbehorende database. Zo herstel je makkelijker je data bij problemen of dataverlies.

In de instellingen kun je aangeven of backups automatisch gemaakt moeten worden en of dat dagelijks of wekelijks moet gebeuren. Ook kun je instellen of de oudste backups verwijderd mogen worden als de opslagruimte beperkt raakt.

![img](https://screensaver01.zap-hosting.com/index.php/s/nm9za5LTMEcNsga/preview)

We bieden standaard 10 GB opslagruimte voor backups op de opslagserver bij onze game server producten. Heb je meer ruimte nodig? Dat kan tegen een meerprijs worden uitgebreid. Zo heb je altijd genoeg ruimte voor je backups en kun je oudere backups blijven benaderen.

## Upvotes vs Burst Upvotes

RedM Upvotes en RedM Burst Upvotes zijn beide manieren om de zichtbaarheid en populariteit van een RedM-server te vergroten. RedM Upvotes zijn doorlopende stemmen die over een langere periode worden ingezet voor langdurige zichtbaarheid in de server rankings. RedM Burst Upvotes zijn juist voor korte acties, waarbij veel stemmen in een korte tijd worden ingezet om de server snel omhoog te krijgen in de rankings.

Voor het gebruik van de RedM upvotes moet je je RedM-account koppelen. De upvotes worden aan dit account gekoppeld, zodat je ze in-game naar wens kunt toewijzen.

Zodra dit bevestigd is, kun je de bestelling afronden. De info wordt direct doorgestuurd naar RedM en daar verwerkt. Zodra dat klaar is, zijn de upvotes voor jou beschikbaar en kun je ze via de interface beheren. De looptijd van de upvotes start dan ook meteen.



## Upvotes Toewijzen

Zodra de upvotes zichtbaar zijn in de interface, kun je ze via de RedM client toewijzen. Klik hiervoor op het Upvotes-icoon in de serverlijst bij de server die jij wilt boosten. 
Als bevestiging verschijnt er een pop-up die aangeeft dat de upvotes succesvol zijn toegewezen.

![](https://screensaver01.zap-hosting.com/index.php/s/zD7b9awkCDW7bcx/download)

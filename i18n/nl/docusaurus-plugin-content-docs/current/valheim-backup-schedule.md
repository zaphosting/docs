Helaas biedt Valheim geen directe optie voor automatische backups, dus gebruiken we een kleine workaround.

Ga in je dashboard naar "Restart Planner".

![](https://screensaver01.zap-hosting.com/index.php/s/FspW5eG7XJNqE4k/preview)

Klik nu op het "+"-teken, er opent een nieuw venster waarin je de gewenste restart-tijden kunt instellen.

![](https://screensaver01.zap-hosting.com/index.php/s/me5tSbwc8YWT7me/preview)

Kies eerst "Dagelijks" als je wilt dat er elke dag een backup wordt gemaakt op het door jou gekozen tijdstip.
In de volgende stap stel je de gewenste restart-tijd in.

:::info
Start offline server - Dit betekent dat je server wordt gestart als hij offline is, zodra het ingestelde herstartmoment bereikt is.
:::

Klik nu op "Opslaan".

:::info
Je kunt een geplande restart instellen om de 6 uur, wat betekent dat er elke 6 uur een automatische backup wordt gemaakt.
:::

![](https://screensaver01.zap-hosting.com/index.php/s/rFPWnSH7EkHxoN9/preview)

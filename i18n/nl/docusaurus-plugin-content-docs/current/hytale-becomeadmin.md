## Intro
Adminrechten toekennen op een Hytale-server geeft vertrouwde spelers de mogelijkheid om mee te helpen met het beheren van de server. Admins, ook wel operators genoemd, kunnen servercommands uitvoeren, spelers beheren en helpen met servermoderatie. Dit maakt het makkelijker voor servereigenaren om routineklussen te delegeren en zorgt voor een soepele gameplay-ervaring.

## Rechten toekennen



Admins op een Hytale-server krijgen operatorstatus, wat hen uitgebreide commandorechten geeft. Dit kan via de **serverconsole** of **in-game commands** door een bestaande operator. Om operatorrechten aan een speler te geven, typ je het volgende in de serverconsole of in-game met voldoende rechten:

```
/op add <spelernaam>
```



## Rechten verwijderen

Als een speler geen admintoegang meer nodig heeft of geen uitgebreide rechten meer mag hebben, kan zijn operatorstatus op elk moment worden ingetrokken:

```
/op remove <spelernaam>
```

Zodra dit is gedaan, krijgt de speler weer standaard permissies en kan hij geen admincommands meer gebruiken.

## Intro
Adminrechten geven je makkelijke en volledige controle over je server. Als admin kun je alle opties en functies van het spel direct in-game gebruiken. Hieronder leggen we stap voor stap uit hoe je adminrechten aan je server toevoegt.  
<InlineVoucher />

### Configuratie
Om jezelf als admin toe te voegen, heb je eerst je SteamID64 nodig. Gebruik bijvoorbeeld de [Steam ID Finder](https://steamidfinder.com/).

Voer hier je Steam URL in:

![](https://screensaver01.zap-hosting.com/index.php/s/e8H8Y4P9ojW9sD2/preview)

Klik daarna op `Find Steam ID`.

Je ziet nu je Steam-profiel, kopieer hier je "Steam64ID (Dec)".

Open vervolgens het webinterface van je server en ga naar de instellingen. Daar vind je de optie "Admins" waar je je Steam ID kunt toevoegen:

![](https://screensaver01.zap-hosting.com/index.php/s/aJkGRMcmWrnZyiM/preview)

Na het toevoegen van je Steam ID sla je de instellingen op en herstart je de server.

## Console openen

De console in Valheim open je door tijdens het spelen op "F5" te drukken. Hier kun je alle admin commands typen.

### Admin Commands

|  Command   |           Arguments          |                  Effect               |
| :--------: | :--------------------------: |  :---------------------------------:  |
|   kick     |        Naam/PlayerID/IP      |     Kickt de opgegeven speler         |
|   ban      |        Naam/PlayerID/IP      |     Bant de opgegeven speler          |
|   unban    |        PlayerID/IP           |     Haalt de ban van de speler af     |
|   banned   |           Geen               |    Toont alle gebande spelers         |

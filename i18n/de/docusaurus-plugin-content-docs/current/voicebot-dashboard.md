## Produktverwaltung

Im Bereich **Produktverwaltung** findest du alle wichtigen Seiten zur Verwaltung deines Voicebots. Die Optionen zur Steuerung deines Voicebots sind unten detailliert aufgelistet.



### Dashboard

Das **Dashboard** des Voicebots dient als zentrale Verwaltungsübersicht. Es zeigt alle grundlegenden und wichtigen Infos zum Voicebot übersichtlich an, darunter den Online-Status, Login-Daten, den Link zur externen Weboberfläche und die Festplattennutzung.

![img](https://screensaver01.zap-hosting.com/index.php/s/YnXaQDPywcoxFPP/preview)



### Webinterface

Die meisten Funktionen des Voicebots sind über eine externe Weboberfläche verfügbar. Dort kannst du die gewünschte Musikquelle auswählen und Vorschläge zur Verbesserung des Bots einreichen. Außerdem kannst du das Passwort für den Zugriff auf die Weboberfläche ändern und den Voicebot neu installieren.

![img](https://screensaver01.zap-hosting.com/index.php/s/cmKrJ2YpzmZcm2L/preview)

### Server

Bevor du den Bot nutzen kannst, musst du festlegen, auf welchem TeamSpeak-Server er laufen soll. Die IP-Adresse und der Port des Servers müssen angegeben werden, ebenso ein frei wählbarer Name für den Bot. Falls der Server durch ein Passwort geschützt ist, muss dieses angegeben werden, damit der Bot Zugriff auf den Server hat.

![img](https://screensaver01.zap-hosting.com/index.php/s/sbbRr9f5HexcMsG/preview)

### Erweiterte Einstellungen

Erweiterte Funktionen wie das automatische Verschieben von Nutzern oder das Abspielen einer Begrüßungsnachricht, wenn ein Nutzer dem Server beitritt, können durch entsprechende Einstellungen aktiviert werden. Diese Features ermöglichen eine persönlichere und automatisierte Serververwaltung.

![img](https://screensaver01.zap-hosting.com/index.php/s/yz94fyZssG53Rjd/preview)

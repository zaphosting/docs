## Produktverwaltung

Im Bereich **Produktverwaltung** findest du alle wichtigen Seiten zur Steuerung deines Voicebots. Die Optionen zur Verwaltung deines Voicebots sind im Folgenden detailliert aufgelistet.

### Dashboard

Das **Dashboard** des Voicebots dient als zentrale Verwaltungsübersicht. Es zeigt alle grundlegenden und wichtigen Infos zum Voicebot übersichtlich an, darunter den Online-Status, Login-Daten, den Link zur externen Weboberfläche und die Speicherauslastung.

![img](https://screensaver01.zap-hosting.com/index.php/s/YnXaQDPywcoxFPP/preview)

### Webinterface

Die meisten Funktionen des Voicebots sind über eine externe Weboberfläche verfügbar. Dort kannst du die gewünschte Musikquelle auswählen und Vorschläge zur Verbesserung des Bots einreichen. Außerdem kannst du das Passwort für den Zugriff auf die Weboberfläche ändern und den Voicebot neu installieren.

![img](https://screensaver01.zap-hosting.com/index.php/s/cmKrJ2YpzmZcm2L/preview)

### Server

Bevor du den Bot nutzt, musst du festlegen, auf welchem TeamSpeak-Server er laufen soll. Die IP-Adresse und der Port des Servers müssen angegeben werden, ebenso ein frei wählbarer Name für den Bot. Falls der Server passwortgeschützt ist, muss dieses angegeben werden, damit der Bot Zugriff auf den Server hat.

![img](https://screensaver01.zap-hosting.com/index.php/s/sbbRr9f5HexcMsG/preview)

### Erweiterte Einstellungen

Erweiterte Features wie das automatische Verschieben von Nutzern oder das Abspielen einer Willkommensnachricht beim Betreten des Servers können durch passende Einstellungen aktiviert werden. Diese Funktionen ermöglichen eine persönlichere und automatisierte Serververwaltung.

![img](https://screensaver01.zap-hosting.com/index.php/s/yz94fyZssG53Rjd/preview)

## Einführung
Admin-Rechte auf einem Hytale-Server zu vergeben, ermöglicht vertrauenswürdigen Spielern, bei der Serververwaltung zu helfen. Admins, auch Operator genannt, können Serverbefehle ausführen, Spieler verwalten und bei der Moderation unterstützen. So wird die Delegation von Routineaufgaben für Serverbesitzer einfacher und das Gameplay bleibt reibungslos.

## Rechte vergeben



Admins auf einem Hytale-Server erhalten den Operator-Status, der ihnen erweiterte Befehlsrechte gibt. Das kannst du entweder über die **Server-Konsole** oder **Ingame-Befehle** durch einen bestehenden Operator machen. Um einem Spieler Operator-Rechte zu geben, gib folgenden Befehl in der Server-Konsole oder ingame mit ausreichenden Rechten ein:

```
/op add <Spielername>
```



## Rechte entziehen

Wenn ein Spieler keine Admin-Rechte mehr braucht oder keine erweiterten Rechte mehr haben soll, kannst du den Operator-Status jederzeit entziehen:

```
/op remove <Spielername>
```

Nach dem Entfernen hat der Spieler wieder Standardrechte und kann keine Admin-Befehle mehr nutzen.



## Fazit

Admin auf einem Hytale-Server zu werden, indem du Operator-Rechte vergibst, ist ein zentraler Teil der Serververwaltung. So können vertrauenswürdige Spieler bei Moderation und administrativen Aufgaben unterstützen. Bei Fragen oder wenn du Hilfe brauchst, steht dir unser Support-Team täglich zur Seite! 🙂

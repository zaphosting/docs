## Serverdetails besorgen

Du brauchst alle relevanten Infos, um dich mit deinem Server zu verbinden. Diese findest du im Dashboard deines Gameserver-Webinterface-Panels. Wichtige Infos sind unter anderem:

- IP-Adresse und Port
- Servername
- Serverpasswort (optional)

Mit diesen Angaben kannst du deinen Server finden und sicherstellen, dass du dich mit dem richtigen Server verbindest.

![img](https://screensaver01.zap-hosting.com/index.php/s/igcm7GkoB8KmsyD/preview)

## Verbindung zum Server herstellen

Die Verbindung zu einem Gameserver kann oft auf verschiedene Arten erfolgen. Die verfügbaren Methoden werden hier detailliert beschrieben und erklärt.

<Tabs>
    <TabItem value="connect_solution_server_browser_ingame" label="Direkt verbinden (Im Spiel)" default>

Um dich über den Ingame-Serverbrowser mit deinem Server zu verbinden, starte dein Spiel. Klicke im Spiel auf **Play** und erstelle einen Zwerg (Charakter), falls du noch keinen hast. Klicke dann auf **Join Other World** und anschließend auf **Direct Connect**. Gib die IP-Adresse und den Port deines Servers bei **Direct Join** ein und klicke auf **Join server**.

## Mögliche Probleme und Lösungen

Die Verbindung zu einem Gameserver kann oft auf verschiedene Arten erfolgen. Die verfügbaren Methoden werden hier detailliert beschrieben und erklärt.

#### Nicht sichtbar

Wenn der Server nicht sichtbar ist, kann das daran liegen, dass die Initialisierung nicht erfolgreich abgeschlossen wurde. Das kann z.B. an einer fehlerhaften Konfiguration oder beschädigten Dateien liegen. Weitere Infos findest du meist in der Serverkonsole oder in den Logdateien.



#### Keine Lösung gefunden oder nicht hilfreich?

Du hast nach der Fehlerbehebung immer noch Probleme? Kein Problem, unser Support-Team steht dir täglich zur Seite. Erstelle einfach ein **[Ticket](https://zap-hosting.com/en/customer/support/)** auf unserer Website und beschreibe dein Problem so detailliert wie möglich. Wir schauen uns das schnellstmöglich an und helfen dir, das Problem zu lösen!

## Introduzione
Non sai come connetterti al tuo server **Subnautica 2** o quali informazioni ti servono per iniziare? Questa guida ti mostra come ottenere i dettagli necessari, stabilire la connessione e risolvere i problemi più comuni.

:::info Avviso Early Access
Subnautica 2 potrebbe essere ancora in sviluppo attivo a seconda della fase di rilascio attuale. Di conseguenza, le funzionalità multiplayer, il comportamento del browser server e i flussi di connessione possono cambiare nel tempo.
:::

## Ottieni i dettagli del server

Ti serviranno tutte le informazioni rilevanti per connetterti al tuo server. Puoi trovare i dettagli nel pannello web del tuo game server nella dashboard. Le informazioni importanti includono:

- Indirizzo IP e porta
- Nome del server
- Password del server

Con queste informazioni potrai identificare il tuo server e assicurarti di connetterti a quello giusto.

| Dettaglio | Descrizione |
| --- | --- |
| `Indirizzo IP` | L’indirizzo pubblico del tuo server Subnautica 2 |
| `Porta` | La porta di gioco usata per la connessione |
| `Nome del server` | Il nome mostrato in gioco se è disponibile una lista server o browser di sessioni |
| `Password del server` | Necessaria solo se il server è protetto da password |

## Stabilire la connessione al server

Il modo esatto per connetterti a un server **Subnautica 2** dipende dalle opzioni multiplayer disponibili nella versione del gioco che stai usando. In generale, ti serviranno l’**indirizzo IP** e la **porta** del server, e in alcuni casi anche la **password**.

### Entra tramite il menu multiplayer in gioco

Avvia **Subnautica 2** e apri la sezione multiplayer o join server dal menu principale. Se il gioco offre un’opzione di connessione diretta, inserisci l’`indirizzo IP` e la `porta` esattamente come mostrato nel pannello web ZAP-Hosting. Se è configurata una password, inserisci la `password del server` quando richiesto.

Dopo aver confermato i dati, il gioco tenterà di stabilire la connessione al server.

:::note Disponibilità Connessione Diretta
La terminologia di menu e pulsanti può variare a seconda della versione del gioco. Se non vedi un’opzione di connessione diretta, verifica se il gioco richiede di unirsi tramite lista server, sistema di inviti o browser di sessioni.
:::

### Entra tramite lista server o browser di sessioni

Se **Subnautica 2** usa un browser server integrato invece di un campo di connessione diretta, cerca il tuo server usando il `nome del server` configurato. Per questo è importante controllare il nome del server nel pannello web prima di provare a connetterti.

Una volta trovato il server corretto nella lista:

1. Seleziona la voce del server
2. Verifica che il nome mostrato corrisponda al tuo server
3. Entra nel server
4. Inserisci la password se richiesta

:::tip Usa Nomi Server Unici
Se il tuo server non è facile da identificare in lista, assegna un nome chiaro e unico nella configurazione. Così sarà più semplice trovare la voce giusta ed evitare di connetterti al server sbagliato.
:::

### Se il server non è subito visibile

Se il server non appare nel browser in gioco, non significa sempre che sia offline. A volte le liste server impiegano tempo a aggiornarsi o il gioco mostra i server solo dopo un’inizializzazione riuscita.

Verifica quanto segue:

- Il server è online nel pannello web ZAP-Hosting
- La porta di gioco corretta è in uso
- Il server è completamente avviato
- Il nome del server è configurato correttamente
- È stata inserita la password se richiesta

## Problemi comuni e soluzioni

La connessione a un game server può essere influenzata da diversi fattori. Qui sotto trovi i problemi più frequenti e come risolverli.

### Server non visibile

La mancata visibilità del server può succedere se l’avvio non è stato completato con successo. Potrebbe dipendere da una configurazione errata o file corrotti. Di solito puoi trovare maggiori informazioni nella console del server o nei file di log.

:::caution Avvio Server Necessario
Non tentare di connetterti prima che il server abbia completato il processo di avvio. Se il server sta ancora caricando o riavviando, i giocatori potrebbero non riuscire a connettersi o il server potrebbe non apparire in nessuna lista.
:::

### Dati di connessione errati

Se non riesci a connetterti, controlla di usare l’`indirizzo IP`, la `porta` e la `password` corretti. Anche un piccolo errore di battitura può bloccare la connessione.

| Controlla | Cosa verificare |
| --- | --- |
| `Indirizzo IP` | Deve corrispondere a quello mostrato nella dashboard del server |
| `Porta` | Deve corrispondere alla porta di gioco assegnata |
| `Password` | Deve essere inserita esattamente come configurata, rispettando maiuscole e minuscole |
| `Nome del server` | Deve corrispondere al nome configurato se cerchi tramite browser |

### Server online ma connessione fallisce

Se il server sembra online ma non riesci comunque a connetterti, controlla la console e i log nel pannello web. Gli errori possono indicare se il server non si è caricato correttamente o se c’è un problema di configurazione che blocca le connessioni.

:::tip Controlla Prima i Log
La console e i file di log sono il modo più rapido per capire perché la connessione fallisce. Ti aiutano a verificare se il server sta ascoltando correttamente e se i tentativi di join arrivano al server.
:::

### Nessuna soluzione o aiuto insufficiente

Hai ancora problemi dopo aver provato a risolverli? Il nostro supporto è disponibile ogni giorno per aiutarti. Crea un **[Ticket](https://zap-hosting.com/en/customer/support/)** sul nostro sito e spiega il problema con più dettagli possibile. Includi IP, porta del server e una breve descrizione di cosa succede quando provi a connetterti.

## Introduzione

Sei interessato a un **Voiceserver** o ne possiedi già uno, ma ti manca ancora una panoramica completa dell’amministrazione e di dove trovare tutte le opzioni necessarie? Soprattutto all’inizio, può essere una sfida. Ma niente panico, qui sotto ti diamo una guida dettagliata sull’amministrazione del tuo servizio, così avrai la miglior overview possibile.

Dopo aver letto questa guida, riuscirai a gestire il tuo Voiceserver in modo super efficiente e a trovare tutte le opzioni e funzionalità necessarie in modo rapido e semplice. Qui avrai una panoramica strutturata che ti aiuterà passo dopo passo a familiarizzare con l’amministrazione del Voiceserver e ad ampliare le tue conoscenze in modo mirato. Così potrai assicurarti che il tuo **Voiceserver** sia configurato al meglio per le tue esigenze e sfruttare al massimo tutte le opzioni disponibili.



## Amministrazione Prodotto

Nella sezione **Amministrazione Prodotto** trovi tutte le pagine necessarie per gestire il tuo Voiceserver. Le opzioni per la gestione del tuo Voiceserver sono elencate nel dettaglio qui sotto. 



### Dashboard

La dashboard ti offre una panoramica centrale di tutte le info importanti relative al voice server. Mostra lo stato attuale, l’indirizzo del server, il nome del server, il numero di utenti connessi e la password configurata. Sono disponibili anche dettagli aggiuntivi per supportare una gestione e un monitoraggio efficaci del server.

![img](https://screensaver01.zap-hosting.com/index.php/s/ZryFnMckWetXbTx/preview)

### Statistiche

La pagina delle statistiche ti dà insight sull’utilizzo del server. Ad esempio, mostra quanti utenti si sono connessi al server negli ultimi minuti o giorni. Per avere dati significativi, il server deve essere stato attivo per un certo periodo. Le statistiche saranno disponibili solo dopo aver raccolto abbastanza informazioni.

![img](https://screensaver01.zap-hosting.com/index.php/s/zYcyRdeHfaHqXt9/preview)



### DDoS Manager

Visualizza gli attacchi DDoS passati per valutare meglio potenziali minacce e pattern. Attiva le notifiche per attacchi DDoS in corso sul tuo server, così puoi reagire subito in caso di incidente.

A seconda della location del server e della protezione DDoS disponibile, puoi anche monitorare gli attacchi attivi in tempo reale. Questo include dettagli sul traffico, mostrando sia il traffico legittimo che quello bloccato dai sistemi di mitigazione. Questo livello di trasparenza ti aiuta a capire come funziona la protezione e ti permette di individuare rischi potenziali in anticipo.

:::info Accesso al DDoS Manager mancante?
L’uso del DDoS Manager richiede l’opzione aggiuntiva "Panoramica DDoS Manager". Se il tuo pacchetto attuale non include questi componenti, puoi aggiungerli in qualsiasi momento aggiornando il tuo servizio.
:::



![img](https://screensaver01.zap-hosting.com/index.php/s/ScCCCY52CMLgfyE/preview)

### Registro Eventi

Il registro eventi offre una panoramica completa di tutte le attività legate all’amministrazione del server. Ti permette di tracciare in qualsiasi momento quando il server è stato avviato o spento, se si è verificato un crash o quando sono stati fatti aggiornamenti, backup o nuove installazioni. Questa funzione di logging aiuta a garantire una documentazione completa di cambiamenti ed eventi e supporta l’identificazione rapida di eventuali problemi.

![img](https://screensaver01.zap-hosting.com/index.php/s/DRFJR6pnL5XRLa5/preview)



## Amministrazione Prodotto

Nella sezione **Amministrazione Prodotto** trovi tutte le pagine necessarie per gestire il tuo voiceserver. Le opzioni per la gestione del tuo voiceserver sono elencate nel dettaglio qui sotto.

### Impostazioni

Il nome del server e il messaggio di benvenuto possono essere modificati senza dover essere connessi direttamente al server. Si possono anche regolare le impostazioni per ban automatici, ad esempio in caso di tentativi di login falliti o comportamenti scorretti ripetuti. Tutte le modifiche entrano in vigore subito dopo aver cliccato su Salva.

![img](https://screensaver01.zap-hosting.com/index.php/s/B6W4ftB6TPGms8K/preview)



### Chiavi Privilegio

Per ottenere certi permessi sul server TeamSpeak serve una chiave privilegio. Questa chiave, chiamata anche token, si inserisce direttamente sul server e dà accesso a livelli di permessi predefiniti. Queste chiavi possono essere create, gestite e rimosse a piacimento.

![img](https://screensaver01.zap-hosting.com/index.php/s/6MjdtBmP7BbQ9cZ/preview)



### Utenti

Tutti gli utenti TeamSpeak 3 attualmente connessi o che si sono connessi in passato sono elencati in una panoramica chiara. È possibile rimuovere utenti dal server o bannarli definitivamente.

![img](https://screensaver01.zap-hosting.com/index.php/s/N8J8ApX5HWEgnYR/preview)



### TSDNS

Il modulo ZAP TSDNS permette di creare indirizzi di connessione personalizzati per il server TeamSpeak. Gli alias TS DNS possono essere configurati come sottodomini su un dominio fornito da ZAP o sul tuo dominio personale gestito tramite l’interfaccia web. Usando un alias del genere per connettersi, non serve più inserire manualmente la porta perché l’indirizzo punta automaticamente al server corretto.

![img](https://screensaver01.zap-hosting.com/index.php/s/c26gTpzprpky54H/preview)



## Strumenti

La sezione strumenti offre varie funzioni per gestire il server. Una di queste è la possibilità di creare backup sotto forma di snapshot. 

### Snapshot

I backup del server TeamSpeak 3 permettono di fare uno snapshot completo dello stato attuale del server. Vengono salvati tutti i canali esistenti, gli utenti registrati e le impostazioni dei permessi. Questi snapshot sono un modo efficace per ripristinare il server a uno stato precedente, ad esempio dopo modifiche strutturali o errori di configurazione accidentali.

![img](https://screensaver01.zap-hosting.com/index.php/s/XyT2q647ENeEZNA/preview)

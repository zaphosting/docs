## Introduzione
Assegnare i permessi di amministratore ti permette una gestione semplice e completa con il controllo totale del tuo server. Come admin, puoi usare tutte le opzioni e funzioni disponibili direttamente nel gioco. Qui sotto trovi tutti i passaggi per assegnare i permessi admin al tuo server.  
<InlineVoucher />

### Configurazione
Per aggiungerti come Admin, prima devi ottenere il tuo SteamID64, puoi usare uno strumento come il [Steam ID Finder](https://steamidfinder.com/)

Inserisci qui il tuo URL Steam:

![](https://screensaver01.zap-hosting.com/index.php/s/e8H8Y4P9ojW9sD2/preview)

Poi clicca su `Find Steam ID`

Ora dovresti vedere il tuo profilo Steam, copia il tuo "Steam64ID (Dec)".

Apri il Webinterface del tuo server e vai alla pagina Impostazioni, qui troverai l’opzione "Admins" dove puoi aggiungere il tuo Steam ID:

![](https://screensaver01.zap-hosting.com/index.php/s/aJkGRMcmWrnZyiM/preview)

Dopo aver aggiunto il tuo Steam ID, salva le impostazioni e riavvia il server.

## Accesso alla Console

La console di Valheim si apre premendo "F5" mentre sei in gioco, qui puoi inserire tutti i comandi admin.


### Comandi Admin

|  Comando  |        Argomenti         |               Effetto                |
| :-------: | :----------------------: | :--------------------------------: |
|   kick    |    Nome/PlayerID/IP      |   Espelle il giocatore specificato  |
|   ban     |    Nome/PlayerID/IP      |   Bannare il giocatore specificato  |
|  unban    |       PlayerID/IP        |   Rimuove il ban dal giocatore      |
|  banned   |          Nessuno         |   Mostra tutti i giocatori bannati  |

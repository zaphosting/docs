## Introduzione

Sei interessato a un **Voicebot** o ne possiedi già uno, ma ti manca ancora una panoramica completa dell’amministrazione e della reperibilità di tutte le opzioni necessarie? Soprattutto all’inizio, può essere una sfida. Ma niente stress, qui sotto ti diamo una guida dettagliata per l’amministrazione del tuo servizio, così avrai la miglior panoramica possibile.

Dopo aver letto questa guida, potrai gestire il tuo Voicebot in modo semplice ed efficiente, trovando tutte le opzioni e funzionalità necessarie in modo rapido e intuitivo. Qui riceverai una panoramica strutturata che ti aiuterà passo dopo passo a familiarizzare con l’amministrazione del Voicebot e ad ampliare le tue conoscenze in modo mirato. Così potrai assicurarti che il tuo **Voicebot** sia configurato al meglio per le tue esigenze e sfruttare al massimo tutte le opzioni disponibili.

## Amministrazione Prodotto

Nella sezione **Amministrazione Prodotto** trovi tutte le pagine necessarie per gestire il tuo Voicebot. Le opzioni per la gestione del Voicebot sono elencate dettagliatamente qui sotto. 



### Dashboard

La **Dashboard** del VoiceBot è il centro di controllo principale. Ti mostra tutte le informazioni base ed essenziali sul VoiceBot in modo chiaro, inclusi lo stato online, i dati di login, il link all’interfaccia web esterna e l’utilizzo del disco.

![img](https://screensaver01.zap-hosting.com/index.php/s/YnXaQDPywcoxFPP/preview)



### Interfaccia Web

La maggior parte delle funzioni del VoiceBot è accessibile tramite un’interfaccia web esterna. Qui puoi scegliere la sorgente musicale desiderata e inviare suggerimenti per migliorare il bot. È anche possibile cambiare la password per accedere all’interfaccia web e reinstallare il VoiceBot.

![img](https://screensaver01.zap-hosting.com/index.php/s/cmKrJ2YpzmZcm2L/preview)

### Server

Prima di usare il bot, devi definire su quale server TeamSpeak deve girare. Devi inserire l’indirizzo IP e la porta del server, insieme a un nome a tua scelta per il bot. Se il server è protetto da password, devi inserirla così il bot può accedervi.

![img](https://screensaver01.zap-hosting.com/index.php/s/sbbRr9f5HexcMsG/preview)

### Impostazioni Avanzate

Funzioni avanzate come spostare automaticamente gli utenti o far partire un messaggio di benvenuto quando un utente entra nel server possono essere attivate configurando le impostazioni appropriate. Queste funzioni permettono una gestione del server più personalizzata e automatizzata.

![img](https://screensaver01.zap-hosting.com/index.php/s/yz94fyZssG53Rjd/preview)

## Introduzione
Assegnare i permessi di amministratore ti permette una gestione semplice e completa con il pieno controllo del tuo server. Come admin, puoi usare tutte le opzioni e funzioni disponibili direttamente in game. Qui sotto trovi tutti i passaggi per assegnare i permessi di amministratore al tuo server.  
<InlineVoucher />

## Ottieni il token

Per registrarti come admin sul tuo server Terraria, ti serve il tuo token unico.  
Questo token viene generato al primo avvio del server, poi apri la console live dove puoi trovare il token:

![](https://screensaver01.zap-hosting.com/index.php/s/5cEQYgBgxAYQRcx/preview)

![](https://screensaver01.zap-hosting.com/index.php/s/HEYwEWe5c3DPZ6E/preview)

In questo esempio il token è **4914096**.

## Inserisci il token in game

Per registrarti come admin, entra nel tuo server Terraria e digita il comando **/setup 4914096** in game.  
Ovviamente sostituisci il token **4914096** con il tuo personale.

![](https://screensaver01.zap-hosting.com/index.php/s/tadkJkQf5cE3dTB/preview)

### Nel chat apparirà questo messaggio:

![](https://screensaver01.zap-hosting.com/index.php/s/JDfKNpaeB63pCeB/preview)


## Aggiungere un Admin

Ora puoi aggiungere un nuovo admin al server con il comando **/user add USERNAME PASSWORD owner**.  
Sostituisci **USERNAME** con il tuo nome e scegli una **PASSWORD** a piacere, che userai poi per fare login come admin sul server.

![](https://screensaver01.zap-hosting.com/index.php/s/iX62CLtn577NfFQ/preview)
![](https://screensaver01.zap-hosting.com/index.php/s/xeP8Y8sx66LkSJQ/preview)

:::info
Ricorda che la password deve essere lunga almeno 4 caratteri!
:::

## Login come Admin

Dopo esserti registrato come admin in game, puoi fare login con questo comando:  
**/login USERNAME PASSWORD**

Ovviamente sostituisci **USERNAME** con il tuo nome e **PASSWORD** con la password che hai scelto.

![](https://screensaver01.zap-hosting.com/index.php/s/ewTra2Fi2yia9jS/preview)

Ora sei loggato come admin sul server e puoi gestirlo con vari comandi.

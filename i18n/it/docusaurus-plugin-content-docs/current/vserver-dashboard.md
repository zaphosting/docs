## Introduzione

Sei interessato a un **VPS** o ne possiedi già uno, ma ti manca ancora una panoramica completa dell’amministrazione e della reperibilità di tutte le opzioni necessarie? Soprattutto all’inizio, può essere una sfida. Ma niente paura, qui sotto ti offriamo una guida dettagliata all’amministrazione del tuo servizio, così avrai la miglior panoramica possibile.

Dopo aver seguito questa guida, potrai gestire il tuo VPS in modo semplice ed efficiente, trovando tutte le opzioni e funzionalità necessarie in modo rapido e intuitivo. Qui troverai una panoramica strutturata che ti aiuterà passo dopo passo a familiarizzare con l’amministrazione del VPS e ad ampliare le tue conoscenze in modo mirato. Così potrai assicurarti che il tuo **VPS** sia configurato al meglio per le tue esigenze e sfruttare al massimo tutte le opzioni disponibili.



## Amministrazione Prodotto

Nella sezione **Amministrazione Prodotto** trovi tutte le pagine necessarie per gestire il tuo VPS. Le opzioni per la gestione del VPS sono elencate dettagliatamente qui sotto. 



### Dashboard

La dashboard offre una panoramica centrale di tutte le informazioni importanti relative al VPS. Mostra lo stato attuale, l’indirizzo del server, l’hostname, il sistema operativo installato, il carico di sistema inclusi CPU e RAM, e il traffico di rete. Questa panoramica supporta un monitoraggio e una gestione efficaci dell’ambiente del server virtuale.

![img](https://screensaver01.zap-hosting.com/index.php/s/JfJKCwtG9opJZ2H/preview)



### Accesso & Sicurezza

Configura un accesso sicuro al tuo server creando le tue credenziali di login. Puoi impostare una password robusta per l’accesso tramite password oppure generare una chiave SSH per connetterti in modo sicuro usando l’autenticazione con chiave SSH. Così scegli il metodo migliore per gestire gli accessi e mantieni il server protetto.

![img](https://screensaver01.zap-hosting.com/index.php/s/KR2KBzfm7kre6mN/preview)



### Registro Eventi

Tieni sotto controllo tutte le attività di amministrazione e gli eventi di sistema in un unico posto. Il registro eventi raccoglie tutte le attività rilevanti, così sai sempre chi ha fatto modifiche, quando sono avvenute e cosa è stato interessato.

![img](https://screensaver01.zap-hosting.com/index.php/s/w9AHMNnPtBtZY2L/preview)



## Impostazioni

Nella sezione **Impostazioni** trovi tutte le pagine necessarie per configurare il tuo VPS. Le opzioni disponibili per la configurazione del VPS sono elencate dettagliatamente qui sotto.

### Impostazioni

Le impostazioni offrono opzioni per apportare modifiche chiave alla configurazione del server. Questo include definire un server DNS, configurare il ballooning o reinstallare il server. Queste funzioni permettono una gestione flessibile e mirata dell’ambiente server.

![img](https://screensaver01.zap-hosting.com/index.php/s/esY87qKYozR4mj9/preview)



### ISO

Vuoi provare un nuovo sistema operativo o controllare il server quando diventa irraggiungibile? Hai a disposizione una vasta gamma di file ISO. Con pochi clic puoi montarli come disco virtuale e usarli subito per installazioni o diagnostica.

![img](https://screensaver01.zap-hosting.com/index.php/s/Ti4CNm3qp8ka6Tw/preview)

### Gameserver / TS3

Con questa funzione puoi configurare e gestire facilmente servizi di gameserver e voiceserver in pochi clic. L’interfaccia web si occupa di tutto il processo di installazione, quindi non serve una conoscenza tecnica approfondita.

Puoi creare nuovi gameserver o voiceserver quando vuoi, regolare impostazioni come slot o risorse e controllarli comodamente dal tuo dashboard ZAP. Così gestire più server diventa veloce, flessibile e semplice. Tutto in un unico posto.

![img](https://screensaver01.zap-hosting.com/index.php/s/dQ6d8LctpNcsd6H/preview)



### Modifica rDNS

Modifica la voce reverse DNS (record PTR) per l’indirizzo IP del tuo server in modo che punti all’hostname corretto. Un rDNS configurato correttamente aiuta a garantire che il nome del VPS venga risolto correttamente ed è particolarmente importante per gestire un mail server. Senza un rDNS valido, le email in uscita potrebbero finire nello spam o non essere consegnate in modo affidabile.

![img](https://screensaver01.zap-hosting.com/index.php/s/jmG7RJHoA22dtpx/preview)



### Indirizzi IP

Visualizza tutti gli indirizzi IP assegnati al tuo server in un colpo d’occhio. Questa panoramica mostra dettagli importanti di configurazione come indirizzo IP, subnet mask e gateway. Puoi anche verificare se un IP è raggiungibile o meno, aiutandoti a controllare che le impostazioni di rete siano configurate correttamente sul sistema operativo.

![img](https://screensaver01.zap-hosting.com/index.php/s/8gWkFMJFpBPNwiW/preview)



## Strumenti

La sezione **Strumenti** contiene tutti gli strumenti essenziali per gestire il VPS. Le opzioni disponibili per configurare il server sono descritte in dettaglio qui sotto e permettono una personalizzazione precisa in base alle esigenze specifiche.

### Console VNC

Con la Console VNC puoi interagire con il tuo server come se fossi direttamente davanti alla macchina. Questo accesso remoto ti permette di controllare schermo, tastiera e mouse, così puoi eseguire operazioni di manutenzione anche senza una connessione di rete attiva. È ideale per riparare impostazioni di rete o risolvere problemi di configurazione.

![img](https://screensaver01.zap-hosting.com/index.php/s/pdiHAkFJGYTmdjp/download)

### Reset traffico

Tieni sotto controllo il traffico mensile consumato. Questa sezione mostra quanto traffico incluso hai già utilizzato. Se raggiungi il limite prima della fine del mese, hai anche la possibilità di resettare il traffico in anticipo pagando un costo aggiuntivo.

![img](https://screensaver01.zap-hosting.com/index.php/s/AjQ7xFjGBHLBJY7/preview)



## Monitor Risorse

Dashboard di monitoraggio per il tuo server virtuale con metriche di performance in tempo reale: utilizzo CPU, consumo memoria, throughput di rete e statistiche di I/O disco.

![img](https://screensaver01.zap-hosting.com/index.php/s/aQWRa6yEbF7BbEC/preview)





### DDoS Manager

Visualizza gli attacchi DDoS passati per valutare meglio potenziali minacce e schemi. Attiva notifiche per attacchi DDoS in corso sul tuo server per reagire rapidamente in caso di incidente.

A seconda della posizione del server e della protezione DDoS disponibile, puoi anche monitorare gli attacchi attivi in tempo reale. Questo include dettagli sul traffico, mostrando sia il traffico legittimo sia la quantità bloccata dai sistemi di mitigazione. Questo livello di trasparenza ti aiuta a capire come funziona la protezione e ti permette di identificare rischi potenziali in anticipo.

![img](https://screensaver01.zap-hosting.com/index.php/s/ScCCCY52CMLgfyE/preview)

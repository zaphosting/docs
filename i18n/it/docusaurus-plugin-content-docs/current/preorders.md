## Introduzione

Alcuni prodotti possono essere ordinati su ZAP-Hosting.com come pre-ordini. Questo significa che il prodotto non è ancora stato integrato nei nostri sistemi, ma lo sarà in futuro. Il server verrà configurato automaticamente non appena il prodotto sarà disponibile.

## Come riconoscere un pre-ordine

I pre-ordini sono contrassegnati come tali nella pagina prodotto. Avranno un'icona a forma di cronometro per rappresentarlo.

![](https://screensaver01.zap-hosting.com/index.php/s/Wx9nEXzees3Bx3B/preview)

## Attivazione dei pre-ordini

I server pre-ordinati verranno configurati in modo completamente automatico non appena il prodotto sarà pronto e disponibile nel nostro game server cloud. Riceverai una mail non appena il server sarà attivo. Il tempo necessario per l’installazione dei server pre-ordinati può variare molto, perché dipende da quando vengono rilasciati i file del server. Cerchiamo di installare i prodotti pre-ordinati appena possibile.



## Pagamento e durata

Il primo periodo di un prodotto pre-ordinato deve essere pagato direttamente al momento dell’ordine. Il primo periodo inizia con la messa a disposizione del server. Il secondo pagamento (in caso di contratto o rinnovo di un server prepagato) è dovuto alla fine del primo periodo. Un server pre-ordinato non genera ulteriori costi finché non è stato configurato.



## Annulla il tuo prodotto pre-ordinato
Per i server pre-ordinati che non sono ancora stati configurati e che possono ancora essere ordinati come pre-ordine, l’importo già pagato verrà rimborsato. Basta contattare il nostro supporto.

## Introduzione

Sei interessato a un **Day of Defeat: Source** o ne possiedi già uno, ma ti manca ancora una panoramica completa sull’amministrazione e su come trovare tutte le opzioni necessarie? Soprattutto all’inizio, può essere una sfida. Ma niente panico, qui sotto ti diamo una guida dettagliata sull’amministrazione del tuo servizio, così avrai tutto sotto controllo.

Dopo aver seguito questa guida, potrai gestire il tuo server in modo efficiente e trovare tutte le opzioni e funzionalità necessarie in modo rapido e semplice. Qui troverai una panoramica strutturata che ti aiuterà passo dopo passo a familiarizzare con l’amministrazione del server e ad ampliare le tue conoscenze in modo mirato. Così potrai assicurarti che il tuo **Day of Defeat: Source** sia ottimizzato per le tue esigenze e sfruttare al massimo tutte le opzioni disponibili.

## Amministrazione del Prodotto

Nella sezione **Amministrazione del Prodotto** trovi tutte le pagine necessarie per gestire il tuo server. Le opzioni per la gestione del server sono elencate dettagliatamente qui sotto.

### Dashboard

Il **Dashboard** del tuo server è il cuore dell’amministrazione. Qui trovi tutte le informazioni base ed essenziali sul tuo servizio in modo chiaro e immediato. Per esempio, informazioni sulla location, stato, IP:Porta, memoria usata, gioco in corso, nome del server, mappa e numero di slot disponibili. Inoltre, viene mostrato anche l’utilizzo di CPU, memoria e RAM.

![img](https://screensaver01.zap-hosting.com/index.php/s/j34HLtkGoLdwbeo/preview)

Dal dashboard puoi anche avviare, fermare o riavviare il servizio, così hai il pieno controllo sul suo funzionamento. Inoltre, puoi accedere alla console live per avere info in tempo reale sullo stato del server e intervenire subito se serve.

### Statistiche

Nella sezione **Statistiche** ottieni una panoramica completa dei valori di carico del tuo server. Troverai dati su vari aspetti come il carico degli slot disponibili, CPU e memoria. Puoi visualizzare questi dati per diversi intervalli di tempo, ad esempio per gli ultimi **60 minuti, 24 ore, 3 giorni, 7 giorni** o **30 giorni.**

![](https://screensaver01.zap-hosting.com/index.php/s/N8jcmdqqG2Xt4Bk/preview)

Queste info ti danno una buona visione sull’utilizzo del server. Inoltre, le statistiche sono utili per individuare eventuali problemi di performance e restringere le cause. Per esempio, puoi capire quando l’utilizzo è aumentato e se è legato a eventi o cambiamenti specifici. Questo è fondamentale in caso di problemi improvvisi, così puoi intervenire in modo mirato per risolverli e analizzare le cause.

### DDoS Manager

Visualizza gli attacchi DDoS passati per valutare meglio minacce e pattern. Attiva le notifiche per attacchi DDoS in corso sul tuo server e reagisci subito in caso di incidente.

A seconda della location del server e della protezione DDoS disponibile, puoi anche monitorare gli attacchi attivi in tempo reale. Questo include dettagli sul traffico, mostrando sia il traffico legittimo che quello bloccato dai sistemi di mitigazione. Questo livello di trasparenza ti aiuta a capire come funziona la protezione e a individuare rischi potenziali in anticipo.

:::info Accesso al DDoS Manager mancante?
L’uso del DDoS Manager richiede un IP dedicato o l’opzione aggiuntiva "DDoS Manager Overview". Se il tuo pacchetto attuale non include questi componenti, puoi aggiungerli in qualsiasi momento aggiornando il servizio.
:::

![](https://screensaver01.zap-hosting.com/index.php/s/ScCCCY52CMLgfyE/preview)

:::warning
L’uso del DDoS Manager è possibile solo se hai prenotato un IP dedicato o l’opzione "DDoS Manager Overview" come extra. Se questi componenti mancano nel tuo pacchetto attuale, puoi aggiungerli in qualsiasi momento con un upgrade.
:::

### File di Log

Hai un problema tecnico e non sai da dove viene? Una prima strategia è controllare i **File di Log**. Questi file contengono tante info sul server, come eventi attuali, suggerimenti o messaggi di errore.

I messaggi di errore in particolare aiutano a identificare e capire meglio la causa del problema. Spesso si riesce a scovare e risolvere il problema dando un’occhiata mirata ai file di log.

:::warning
Non ti aiutano o non li capisci? Tranquillo! Siamo disponibili ogni giorno nel supporto. Apri un **[Ticket](https://zap-hosting.com/en/customer/support/)** e spiegaci il problema. Lo controlleremo il prima possibile e ti aiuteremo a risolverlo! :)
:::

### Registro Eventi

Il **Registro Eventi** ti offre una panoramica completa di tutte le attività legate all’amministrazione del server. Puoi vedere quando un server è stato avviato, spento o crashato, e quando sono stati fatti aggiornamenti, backup o nuove installazioni.

![](https://screensaver01.zap-hosting.com/index.php/s/xNzsaMbW5BS5KJC/preview)

### Console Live

La **Console Live** ti mostra in tempo reale tutti gli eventi correnti sul server. Qui vedi info sulle attività dei giocatori, eventuali errori e messaggi di sistema. Questa panoramica ti permette di capire cosa succede sul server e di individuare problemi in anticipo.

![](https://screensaver01.zap-hosting.com/index.php/s/dktDADkq3bzG3F6/preview)

Oltre a monitorare gli eventi, puoi anche inviare comandi direttamente al server dalla **Console Live**. Utile per modificare impostazioni, eseguire azioni o rispondere velocemente a problemi.

### Interruzioni di Servizio

Il tuo server è stato offline per un po’ e non sai perché? Il registro eventi e i file di log non hanno dato info? Potrebbe essere un problema non del server ma del sistema host su cui gira.

Se il nostro sistema di monitoraggio rileva un’interruzione generale, i tecnici vengono avvisati automaticamente e si occupano del problema il prima possibile.

Qui trovi una panoramica della cronologia delle interruzioni del sistema host su cui gira il tuo server. Per info aggiornate e lo stato attuale, visita la nostra nuova [pagina di status](https://status.zap-hosting.com/).

## Impostazioni

Nella sezione **Impostazioni** trovi tutte le pagine per configurare il tuo server. Le opzioni disponibili per la configurazione sono elencate qui sotto.

### Impostazioni

Qui trovi una panoramica delle impostazioni più comuni per il tuo server, con una breve spiegazione. Consigliamo questa pagina soprattutto ai principianti nell’amministrazione server.

Se sei più esperto e vuoi fare modifiche più precise, ti consigliamo di usare l’Editor Config. Trovi più info nella categoria **Configs**.

### Configs

L’editor dei file di configurazione ti permette di modificare direttamente i file dal sito, senza la vista semplificata. Così puoi personalizzare le impostazioni del server secondo i tuoi gusti.

Con l’editor hai accesso diretto a tutte le opzioni e puoi modificarle con precisione. Perfetto per utenti avanzati che vogliono il controllo totale sulle impostazioni.

:::warning
Hai notato che alcune modifiche vengono sovrascritte? Ricorda che certi valori vengono presi dalla pagina impostazioni. Se vuoi modificarli direttamente nel config, devi prima disattivarli nella pagina impostazioni.
:::

Per gli altri giochi serve prima modificare il prezzo per slot. Maggiori info nel tutorial [Cambio Gioco](gameserver-gameswitch.md).

## Strumenti

Nella sezione **Strumenti** del pannello di navigazione trovi varie sottosezioni che ti aiutano a gestire il server. Ci sono **FTP Browser**, **Database** e **Backup**. Qui sotto ti spieghiamo nel dettaglio.

### FTP-Browser

Con l’**FTP Browser** trovi tutte le info per connetterti via FTP. L’accesso FTP ti dà accesso diretto ai file del server. Consigliamo di usare un client FTP esterno come **Filezilla** o **WinSCP** per un’esperienza ottimale. Se non sai come usare FTP, dai un’occhiata alla guida [Accesso via FTP](gameserver-ftpaccess.md).

![](https://screensaver01.zap-hosting.com/index.php/s/sX85i9HyzR9wmAM/preview)

**FileZilla:** FileZilla è probabilmente il client FTP più semplice e conosciuto. Basta installarlo, avviarlo e inserire i dati di accesso che trovi qui sopra. Con un click su "Connetti" sei subito dentro al server e puoi gestire i file.

**WinSCP:** WinSCP è pensato per SCP e SFTP (versioni criptate di FTP), ma funziona benissimo anche con FTP normale. È un po’ più complesso di FileZilla, ideale per utenti più esperti.

### Database

Oltre ai server di gioco, offriamo anche 10 database inclusi. Servono per esempio quando devi salvare dati in un database. In **Database** puoi crearne uno e vedere le credenziali di accesso. Puoi anche attivare i backup e importare backup esistenti.

![](https://screensaver01.zap-hosting.com/index.php/s/WFLwCbZSQfYjrmD/preview)

### Backup

In **Backup** puoi creare backup manuali e automatici per il server e il database associato. Così recuperi i dati facilmente in caso di problemi o perdite.

Nelle impostazioni scegli se fare backup automatici giornalieri o settimanali. Puoi anche decidere se eliminare i backup più vecchi quando lo spazio si riduce.

![](https://screensaver01.zap-hosting.com/index.php/s/nm9za5LTMEcNsga/preview)

Per i nostri server di gioco forniamo 10 GB di spazio per backup sul server di storage. Se ti serve più spazio, puoi fare un upgrade a pagamento. Così hai sempre spazio a sufficienza e accesso ai backup più vecchi.

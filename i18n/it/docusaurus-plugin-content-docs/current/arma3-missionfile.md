## Carica la missione

Per caricare la tua missione personalizzata devi prima connetterti al tuo server via FTP: [Accesso via FTP](gameserver-ftpaccess.md).  
Qui devi caricare il file della missione nella cartella **mpmissions** che trovi sotto **/gxxxxxx/arma3/**.  
In questo esempio carichiamo il file missione **King_of_the_hill.Altis** sul nostro server.

![](https://screensaver01.zap-hosting.com/index.php/s/jkfK9X48qj2e7Pn/preview)

<InlineVoucher />

## Avviare la missione sul server

Per avviare la missione dal server, vai nell’interfaccia del tuo server Arma 3 e clicca sulla scheda **Configs** nel menu a sinistra:

![](https://screensaver01.zap-hosting.com/index.php/s/PD98ATq8xHdjy8Q/preview)

Qui modifica il file **server.cfg** e inserisci il nome del file missione come mostrato nell’immagine:

![](https://screensaver01.zap-hosting.com/index.php/s/aG7QESxyTNpATsG/preview)

![](https://screensaver01.zap-hosting.com/index.php/s/6TZ7mc578pcHpbF/preview)

Dopo clicca su **Salva** e potrai avviare il server. La nuova missione verrà caricata.

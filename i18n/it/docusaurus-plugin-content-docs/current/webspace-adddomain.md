## Introduzione

Il pacchetto webspace offre la possibilità di creare e gestire sia siti web che indirizzi email tramite l’interfaccia esterna Plesk. Lo spazio web funge da storage web. File e cartelle possono essere caricati tramite una connessione FTP.

Ogni webspace prenotato tramite Zap-Hosting riceve automaticamente un sottodominio generato. Questo sottodominio è composto dal numero cliente, dal server host e dal dominio "zap-webspace.com". Le istruzioni in "Collega Dominio" mostrano come collegare il tuo dominio personale allo spazio web.

## Primi passi

Dopo aver completato l’ordine, lo spazio web sarà visibile nel Dashboard di Zap-Hosting.

### Apri Plesk

I dati di login sono visibili nella pagina appena aperta. La password non è salvata in chiaro. Per mostrarla, devi cliccare sull’icona a forma di “occhio”.

![Bildschirmfoto vom 2022-05-13 05-16-48](https://screensaver01.zap-hosting.com/index.php/s/B3stJeKMAsYRRer/preview)

:::info
Se la password generata automaticamente non ti convince, puoi cambiarla direttamente sulla stessa pagina cliccando sul pulsante "Cambia password".
:::

Una volta visualizzati username e password, puoi aprire l’interfaccia Plesk cliccando sul pulsante "**Apri Amministrazione**". Dopo il reindirizzamento all’interfaccia Plesk, dovrai inserire i dati di login appena visualizzati.

![Bildschirmfoto vom 2022-05-13 05-17-21](https://screensaver01.zap-hosting.com/index.php/s/tZLEJZTzo7kqn4J/preview)

:::info
La lingua dell’interfaccia utente può essere cambiata da tedesco a inglese; il valore predefinito è il tedesco.
:::

### Collega dominio

Il tuo dominio personale può essere inserito nell’interfaccia Plesk sotto la voce di menu "**Aggiungi Dominio**".

![Bildschirmfoto vom 2022-05-13 05-18-15](https://screensaver01.zap-hosting.com/index.php/s/LHJkPqAsC7zGysW/preview)

In "**Aggiungi Dominio**" puoi inserire direttamente l’URL del tuo dominio e scegliere le impostazioni desiderate riguardo la posizione dei file, SSL, ecc.

![Bildschirmfoto vom 2022-05-13 05-19-06](https://screensaver01.zap-hosting.com/index.php/s/JXYg4ycDpwqYxCN/preview)

Le modifiche si confermano cliccando sul pulsante "**OK**".

:::info
La conferma potrebbe richiedere un po’ di tempo. Se durante questa operazione ricevi errori, una possibile causa è che il tuo dominio sia già stato creato come alias nel dominio.
:::

Se la modifica va a buon fine, lo vedrai confermato nell’interfaccia Plesk. Per poter usare questo dominio con lo spazio web, devi configurare le impostazioni DNS nella gestione del dominio. Qui puoi vedere come funziona con i domini ordinati da noi.

![Bildschirmfoto vom 2022-05-13 05-19-40](https://screensaver01.zap-hosting.com/index.php/s/TecSRSimmCrRmYj/preview)

## Einführung

Mods sind mega wichtig, um das Gameplay in **Arma Reforger** richtig zu erweitern. Im **Arma Reforger Workshop** veröffentlichen Spieler und Creator coole Custom-Missionen, Gameplay-Änderungen, Maps und extra Content, den du auf deinem Server nutzen kannst.

Mit Mods kannst du als Server-Owner das Gameplay anpassen, neue Mechaniken einbauen oder einzigartige Missionen für deine Community bereitstellen. Sobald die Mods installiert sind, laden sich Spieler die nötigen Mods automatisch runter, bevor sie connecten.

## Mods finden

Du brauchst zuerst die **Workshop-ID** von dem Mod, den du installieren willst. Öffne den **Arma Reforger Workshop** und such dir den Mod raus. Jeder Workshop-Eintrag hat eine einzigartige ID, die dein Server nutzt, um den Mod automatisch runterzuladen.

![img](https://screensaver01.zap-hosting.com/index.php/s/jMNGbRAQdjKcDyn/preview)

## Konfiguration

Um Workshop-Mods auf deinem Server zu installieren, öffne dein **Gameserver-Management-Panel** und geh zu **Einstellungen**. Such das Konfigurationsfeld namens **Workshop-Mods**. Dort gibst du die **Workshop-ID** des Mods ein, den du installieren willst. Beispiel mit einem Mod:

```
1234567890
```
Wenn du mehrere Mods installieren willst, trenn die Workshop-IDs einfach mit einem **Komma**. Beispiel mit mehreren Mods:

```
1234567890,9876543210,1357924680
```
Speichere danach die Einstellungen und starte den Server neu. Beim Start lädt der Server automatisch die angegebenen Mods aus dem Arma Reforger Workshop runter und lädt sie.

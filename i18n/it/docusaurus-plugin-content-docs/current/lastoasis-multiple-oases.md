## Introduzione
Last Oasis offre la possibilità di collegare più Oasi contemporaneamente nello stesso Realm. In questa guida vedremo come attivare questa modalità per il tuo Realm.

<InlineVoucher />

## Preparazione
Inizia andando sul [sito ufficiale di Last Oasis](https://myrealm.lastoasis.gg/) e accedi con la piattaforma che preferisci (Steam o Xbox).

![](https://screensaver01.zap-hosting.com/index.php/s/d6xZsqYbEF9jSj8/preview)

## Attivare il multi hosting
Sul sito, vai alla sezione **Realms**. Sul tuo Realm attuale, clicca il pulsante **Change** e cambia la modalità di hosting da **Single -> Multiple**.

![](https://screensaver01.zap-hosting.com/index.php/s/qDgwQ5D6YwptWER/preview)

:::info
Per poter far girare più Oasi contemporaneamente, devi noleggiare un game server per ogni Oasis. Usa la stessa chiave provider del server originale per ogni nuovo server. Ricordati di riavviare il server dopo ogni modifica.
:::

Dopo aver fatto questa modifica, potrai gestire le Oasi usando il pulsante **Manage** in questa sezione.

## Crea una nuova Oasis
Scegli la posizione desiderata per la nuova Oasis e premi **Add** per confermare.

![](https://screensaver01.zap-hosting.com/index.php/s/A2GLkeBWaBQr6m9/preview)

Apparirà una nuova finestra dove potrai impostare un nome e selezionare una mappa.

![](https://screensaver01.zap-hosting.com/index.php/s/6SkCFyAzooKwQAA/preview)

![](https://screensaver01.zap-hosting.com/index.php/s/CBFHBq8TxAxogk9/preview)

Una volta confermato, premi **Add Oasis** per creare l’Oasi e attivarla con il pulsante **Activate**.

![](https://screensaver01.zap-hosting.com/index.php/s/yoeHTdeAeXneC2q/preview)

Ora dovresti vedere più Oasi attive contemporaneamente nella sezione **Oases**. Hai configurato con successo più Oasi per il tuo Realm.

## Amministrazione del Prodotto

Nella sezione **Amministrazione del Prodotto** trovi tutte le pagine necessarie per gestire il tuo server. Qui sotto ti elenchiamo nel dettaglio tutte le opzioni per la gestione del server.

### Dashboard

Il **Dashboard** del tuo server è il cuore dell’amministrazione. Qui trovi tutte le info base ed essenziali sul tuo servizio in modo chiaro. Per esempio, informazioni sulla location, stato, IP:Porta, memoria usata, gioco in corso, nome del server, mappa e numero di slot disponibili. Inoltre, viene mostrato anche l’utilizzo di CPU, memoria e RAM.

![](https://screensaver01.zap-hosting.com/index.php/s/pYy8nmAG9Z7L7H9/preview)

Dal dashboard puoi anche avviare, fermare o riavviare il servizio, così hai il pieno controllo del suo funzionamento. Inoltre, puoi accedere alla console live per avere info in tempo reale sullo stato del server e intervenire subito se serve.

### DDoS Manager

Visualizza gli attacchi DDoS passati per valutare meglio potenziali minacce e pattern. Attiva le notifiche per attacchi DDoS in corso sul tuo server, così puoi reagire velocemente in caso di problemi.

A seconda della location del server e della protezione DDoS disponibile, puoi anche monitorare gli attacchi attivi in tempo reale. Questo include dettagli sul traffico, mostrando sia il traffico legittimo che quello bloccato dai sistemi di mitigazione. Questo livello di trasparenza ti aiuta a capire come funziona la protezione e a individuare rischi potenziali in anticipo.

:::info Accesso al DDoS Manager mancante?
L’uso del DDoS Manager richiede un indirizzo IP dedicato o l’opzione aggiuntiva "Panoramica DDoS Manager". Se il tuo pacchetto attuale non include questi componenti, puoi aggiungerli in qualsiasi momento aggiornando il servizio.
:::

![](https://screensaver01.zap-hosting.com/index.php/s/ScCCCY52CMLgfyE/preview)

### File di Log

Hai un problema tecnico e non sai da dove venga? Una prima mossa è controllare i **File di Log**. Questi file contengono un sacco di info sul server, come eventi attuali, suggerimenti o messaggi di errore.

I messaggi di errore in particolare aiutano a capire meglio la causa del problema. Spesso basta dare un’occhiata mirata ai log per scovare e risolvere le difficoltà.

:::info
Non ti aiutano o non li capisci? Tranquillo! Siamo disponibili ogni giorno nel supporto. Apri un **[Ticket](https://zap-hosting.com/en/customer/support/)** e spiegaci il problema. Lo controlleremo il prima possibile e ti aiuteremo a risolverlo! :)
:::

### Registro Eventi

Il **Registro Eventi** ti dà una panoramica completa di tutte le attività legate all’amministrazione del server. Puoi vedere quando un server è stato avviato, spento o crashato, e quando sono stati fatti aggiornamenti, backup o nuove installazioni.

![](https://screensaver01.zap-hosting.com/index.php/s/xNzsaMbW5BS5KJC/preview)

### Console Live

La **Console Live** ti offre una vista in tempo reale di tutti gli eventi attuali sul tuo server. Qui puoi vedere info sulle attività dei giocatori, messaggi di errore e messaggi di sistema. Questa panoramica ti permette di capire cosa succede sul server e di individuare problemi in anticipo.

![](https://screensaver01.zap-hosting.com/index.php/s/J9SsmSrPXnG7P9w/preview)

Oltre a monitorare gli eventi, puoi anche inviare comandi direttamente al server dalla **Console Live**. Utile per modificare impostazioni, eseguire azioni o rispondere velocemente a problemi.

### Interruzioni di Servizio

Il tuo server è stato offline per un po’ e non capisci perché? Controllare il registro eventi e i file di log non ha aiutato? Forse il problema non è direttamente sul tuo server, ma sul sistema host che lo ospita.

Se il nostro sistema di monitoraggio rileva un’interruzione generale, i nostri tecnici vengono avvisati automaticamente e si occupano del problema il prima possibile.

In questa sezione trovi la cronologia delle interruzioni del sistema host su cui gira il tuo server di gioco. Per info aggiornate e lo stato attuale del servizio, visita la nostra nuova [pagina di status](https://status.zap-hosting.com/).

## Impostazioni

Nella sezione **Impostazioni** trovi tutte le pagine necessarie per configurare il tuo server. Qui sotto ti elenchiamo nel dettaglio tutte le opzioni di configurazione disponibili.

### Impostazioni

Qui trovi una panoramica delle impostazioni più comuni per il tuo server, con una breve spiegazione. Consigliamo questa pagina soprattutto a chi è alle prime armi con l’amministrazione server.

Puoi modificare per esempio:

### Configs

L’editor dei file di configurazione ti permette di modificare direttamente i file di config dal sito, senza la vista semplificata. Così puoi personalizzare le impostazioni del server secondo i tuoi gusti.

Con l’editor hai accesso diretto a tutte le opzioni e puoi modificarle con precisione. Perfetto per utenti avanzati che vogliono il controllo totale sulle impostazioni.

:::info
Hai notato che alcune modifiche vengono sovrascritte? Ricorda che certi valori vengono presi dalla pagina impostazioni. Se vuoi modificarli direttamente nel config, devi prima disattivare le opzioni corrispondenti nella pagina impostazioni.

In **Giochi** puoi cambiare la variante del gioco esistente o installarne un altro. I giochi con lo stesso prezzo per slot si possono selezionare direttamente sotto **Giochi Disponibili**. Per gli altri giochi serve prima modificare il prezzo per slot. Maggiori info nel nostro [guida al cambio gioco](gameserver-gameswitch.md).

![](https://screensaver01.zap-hosting.com/index.php/s/QB82f3CSKmJgwqk/preview)

### Versioni

In **Versioni** puoi vedere e gestire la versione attuale del tuo server di gioco. Puoi aggiornare sempre all’ultima versione o attivare gli aggiornamenti automatici per tenere il server sempre aggiornato.

![](https://screensaver01.zap-hosting.com/index.php/s/BH2JzyRHTeLdKHz/preview)

Puoi anche attivare le notifiche email per gli aggiornamenti automatici, così resti sempre informato.

Fatto ciò, puoi usarli sul server e aggiungere varietà e divertimento.

![](https://screensaver01.zap-hosting.com/index.php/s/AaL9sQeye5gKzWQ/preview)

Crea attività programmate che si eseguono automaticamente a orari definiti per automatizzare processi sul server. Puoi scegliere se farle partire una volta o ripetutamente. Tipi di attività supportate: avviare, fermare o riavviare servizi, riavviare il server se online, fare backup o eseguire comandi personalizzati.

### Pianificatore Riavvii

Vuoi programmare riavvii del server a orari specifici? Il Pianificatore Riavvii ti permette di impostare riavvii automatici programmati. Puoi scegliere orari precisi o intervalli regolari.

## Strumenti

Nella sezione **Strumenti** del pannello di navigazione trovi varie sottosezioni che ti aiutano a gestire il server. Ci sono **FTP Browser**, **Database** e **Backup**. Qui sotto ti spieghiamo tutto nel dettaglio.

### FTP Browser

Con l’**FTP Browser** trovi tutte le info per connetterti via FTP. L’accesso FTP ti dà accesso diretto ai file del server di gioco. Consigliamo di usare un programma FTP esterno come **Filezilla** o **WinSCP** per un’esperienza ottimale. Se non sai come usare FTP, dai un’occhiata alla guida [Accesso via FTP](gameserver-ftpaccess.md).

![](https://screensaver01.zap-hosting.com/index.php/s/fZSYpSi4KwPos5P/preview)

**FileZilla:** Il client FTP più semplice e conosciuto. Basta installarlo, avviarlo e inserire i dati di accesso nella barra in alto, che trovi poco sopra in questa pagina. Clicca su "Connetti" e sei dentro al server per gestire i file.

**WinSCP:** Pensato per SCP e SFTP (versioni criptate di FTP), ma gestisce anche FTP. È più completo di FileZilla ed è adatto a utenti avanzati.

### Database

Oltre ai server di gioco, offriamo anche 10 database inclusi. Servono per esempio quando devi salvare dati in un database. In **Database** puoi crearne uno e vedere le credenziali di accesso. Puoi anche attivare i backup e importare backup esistenti.

![](https://screensaver01.zap-hosting.com/index.php/s/WFLwCbZSQfYjrmD/preview)

### Backup

In **Backup** puoi creare backup manuali o automatici per il server e il database associato. Così recuperi i dati facilmente in caso di problemi o perdite.

Nelle impostazioni scegli se fare backup automatici, giornalieri o settimanali. Puoi anche decidere se eliminare i backup più vecchi quando lo spazio si riduce.

![](https://screensaver01.zap-hosting.com/index.php/s/nm9za5LTMEcNsga/preview)

Per i nostri server di gioco offriamo 10 GB di spazio per backup sul server di storage di default. Se ti serve più spazio, puoi aggiungerlo a pagamento. Così hai sempre spazio a sufficienza per i backup e puoi accedere anche a quelli più vecchi.

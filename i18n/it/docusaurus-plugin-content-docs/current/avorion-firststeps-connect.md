## Ottieni i dettagli del server


Ti serviranno tutte le informazioni necessarie per collegarti al tuo server. Le info sul tuo server le trovi nel pannello web del tuo game server, nella dashboard. Le info importanti sono:

- Indirizzo IP e porta
- Nome del server

## Introduzione
I server di Last Oasis richiedono diversi passaggi per autenticarsi con il sito ufficiale. In questa guida vedremo come creare il tuo Realm su Last Oasis e configurare un Oasis all’interno del tuo nuovo Realm.

<InlineVoucher />

## Preparazione
Uno dei prerequisiti per configurare un nuovo Realm è generare una chiave dal sito ufficiale per poter verificare e autenticare i servizi online ufficiali.

Vai semplicemente su [il sito ufficiale di Last Oasis](https://myrealm.lastoasis.gg/) e accedi con la piattaforma che preferisci (Steam o Xbox).

![](https://screensaver01.zap-hosting.com/index.php/s/Z2sPJLFks9LY8cg/preview)

## Configurazione della chiave server
Una volta loggato, avrai già una **CustomerKey**, che è la chiave per il tuo server di gioco.

![](https://screensaver01.zap-hosting.com/index.php/s/DgdDoCHf7DDjZ2s/preview)

Vai nella sezione **Settings** dell’interfaccia web del tuo server di gioco e inserisci questo codice nel campo **Customer Key**. Ricordati di salvare dopo aver modificato.

## Configurazione della chiave provider
Dovrai anche creare una chiave provider per identificare univocamente il tuo server di gioco. Usa il pulsante **Add Key** per aprire il menu.

![](https://screensaver01.zap-hosting.com/index.php/s/JMQPTZrHcCxgEz2/preview)

Nel menu, inserisci un nome e una descrizione utili per la chiave e premi **Add**.

![](https://screensaver01.zap-hosting.com/index.php/s/bfiaA2gjjjxESpM/preview)

:::tip
Se vuoi collegare più Oasi insieme nel tuo Realm e tenerle attive contemporaneamente, dai un’occhiata alla nostra guida su [Multiple Oases](lastoasis-multiple-oases.md) per scoprire come abilitarlo.
:::

Torna nella sezione **Settings** dell’interfaccia web del server e inserisci questo codice nel campo **ProviderKey**. Ricordati di salvare dopo aver modificato.

## Creazione di un Realm
Ora puoi procedere a creare un Realm. Sempre sul sito ufficiale [Last Oasis](https://myrealm.lastoasis.gg/), vai nella sezione **Realms** e crea un nuovo Realm.

![](https://screensaver01.zap-hosting.com/index.php/s/ydzjKZTgG4L7QpQ/preview)

Scegli un nome e inserisci eventuali dettagli aggiuntivi a piacere. Quando sei pronto, salva il Realm. Questo passaggio è necessario per poterti connettere al tuo server.

![](https://screensaver01.zap-hosting.com/index.php/s/4rJq5r2zfHx66Qs/preview)

Dopo aver creato il Realm, potrai vedere una panoramica completa dove potrai modificare altre impostazioni e visualizzare info utili.

![](https://screensaver01.zap-hosting.com/index.php/s/j5egcHeicf3QPiR/preview)

## Creare un Oasis
L’ultimo step è creare un Oasis all’interno del tuo Realm. Per ogni server di gioco che possiedi, devi creare un Oasis separato. Ad esempio, se hai 2 server, crea un Oasis per ciascuno.

Clicca sul pulsante **Manage** nella panoramica del Realm.

![](https://screensaver01.zap-hosting.com/index.php/s/GNDZ7DaeLT2qAoC/preview)

Nella nuova pagina, scegli la posizione desiderata per il tuo nuovo Oasis e premi **Add** per confermare.

![](https://screensaver01.zap-hosting.com/index.php/s/dXKXKod6y8NE8SC/preview)

Apparirà una finestra dove potrai impostare un nome e selezionare una mappa.

![](https://screensaver01.zap-hosting.com/index.php/s/aKJmGmGQmz65Xnz/preview)

![](https://screensaver01.zap-hosting.com/index.php/s/Hrq49c3TArdYXdG/preview)

Una volta confermato, puoi finalmente cliccare su **Add Oasis** per creare l’Oasis e attivarlo con il pulsante **Activate**.

![](https://screensaver01.zap-hosting.com/index.php/s/tfEXd4GgAMDd8sE/preview)

:::tip
Se vuoi collegare più Oasi insieme nel tuo Realm e tenerle attive contemporaneamente, dai un’occhiata alla nostra guida su [Multiple Oases](lastoasis-multiple-oases.md) per scoprire come abilitarlo.
:::

Hai configurato con successo un nuovo Realm e un Oasis per il tuo server di Last Oasis.

## Imposta la modalità di gioco

Per cambiare la modalità di gioco sul tuo server Arma 3 devi caricare il file missione appropriato per la modalità di gioco sul tuo server.

Attualmente offriamo i seguenti file missione senza che tu debba caricarli:

- Arma 3 Wasteland

Questi sono già presenti nella tua cartella **mpmissions**.

<InlineVoucher />

## Carica il file missione

Per caricare il file missione devi prima connetterti al tuo server via FTP: [Accesso via FTP](gameserver-ftpaccess.md).  
Qui carichi il file missione corrispondente nella tua cartella **mpmissions** che trovi sotto **/gxxxxxx/arma3/**.  
In questo esempio carichiamo la modalità di gioco **King of the Hill** sul nostro server.

![](https://screensaver01.zap-hosting.com/index.php/s/rDS7DsEfQskZ9Y3/preview)


## Caricare il file missione sul server

Per caricare il file missione dal server, vai all’interfaccia del tuo server Arma 3 e clicca sulla scheda **Configs** nel menu a sinistra:

![](https://screensaver01.zap-hosting.com/index.php/s/dxDWHiFJy5e4qYq/preview)

Qui modifica il file **server.cfg** e inserisci il nome del tuo file missione come mostrato nell’immagine:

![](https://screensaver01.zap-hosting.com/index.php/s/7JQED8wC9WGBdYB/preview)

![](https://screensaver01.zap-hosting.com/index.php/s/7jtRK3YRD7wWiij/preview)

Dopodiché clicca su **Salva** e puoi avviare il server. Il nuovo file missione verrà caricato.

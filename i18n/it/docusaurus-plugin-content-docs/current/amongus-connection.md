## Preparazione - Il Download

Per poterti connettere al tuo server, ci serve un file che abiliti la connessione.
Per prima cosa apri [Impostor](https://impostor.github.io/Impostor/) dove dovremo inserire le nostre informazioni.

Per creare il file, ci servono dati come IP, porta e un nome.
![](https://screensaver01.zap-hosting.com/index.php/s/bQ7KzTAJcbAsLJd/preview)

Inserisci qui i dati dalla tua interfaccia ZAP-Hosting, ad esempio così:

![](https://screensaver01.zap-hosting.com/index.php/s/7scfddD5WPFr25b/preview)

Dove 31.214.245.215 è il tuo IP e 2000 la porta.

Il nome del server può essere scelto liberamente, una volta inseriti tutti i dati clicca su "Download Server File".
Ora verrà scaricato un file chiamato "regioninfo.json".

Poi premi `Win` + `R` e inserisci

```
%APPDATA%\..\LocalLow\Innersloth\Among Us
```

e conferma con Invio, si aprirà una cartella. Sposta il file regioninfo.json dentro e sovrascrivi quello esistente.

![](https://screensaver01.zap-hosting.com/index.php/s/KTrfij6ww64DPCq/preview)


## Crea il Server "in game"

Avvia Among Us sul tuo PC.
Clicca su "online" nel menu principale e poi su "Create Game".

![](https://screensaver01.zap-hosting.com/index.php/s/SfMwr82R9BbDK2P/preview)

Qui puoi ora impostare il server come preferisci:

![](https://screensaver01.zap-hosting.com/index.php/s/imzCR9WZenyrTSX/preview)

Quando hai finito con le impostazioni clicca su "confirm".
Ora sei connesso al tuo server!

Nella lobby puoi scegliere se il server sarà pubblico o privato.
Per cambiarlo clicca sull’icona "Private" o "Public":

Se il server è impostato su "Public", altri giocatori possono connettersi allo stesso modo.
Questi giocatori non devono cliccare su "Create Game" nel menu principale, ma su "Find Game".
È importante che la chat sia impostata su "others" e che i filtri corrispondano esattamente alle impostazioni del server.
Altrimenti il server non verrà mostrato.

Se è un server "Private", bisogna cliccare su "Enter Code" nel menu principale.
Qui inserisci il codice che il creatore del server vede nella lobby.

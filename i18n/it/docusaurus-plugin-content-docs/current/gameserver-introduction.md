## Configurazioni

Offriamo varie opzioni di configurazione per i nostri prodotti gameserver, così da adattarci a ogni esigenza. Le configurazioni di base includono queste opzioni:

| Configurazione            | Opzioni di scelta |
| --------------------------------- | ---- |
| Slot                             | In base a cosa supporta il gioco |
| Memory Boost (RAM aggiuntiva) | Fino a 32 GB di RAM extra oltre la RAM base assegnata |
| Spazio Disco Aggiuntivo             | Fino a 200 GB di spazio disco extra oltre i 50 GB base |
| CPU & Host Server           | Standard (SSD, 2.0-3.4 GHz, RAM DDR3/4) o Premium (M.2 SSD, CPU Gaming 3.4-4.4 GHz, RAM DDR4) |
| IPv4 dedicato             | IPv4 dedicato / senza IPv4 dedicato |

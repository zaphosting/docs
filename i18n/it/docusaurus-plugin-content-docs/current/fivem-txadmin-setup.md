## Introduzione

txAdmin è un pannello web completamente gratuito e ricco di funzionalità per gestire e monitorare il tuo server FiveM. Offre una vasta gamma di strumenti pensati per rendere la gestione del server FiveM un gioco da ragazzi. Configurare txAdmin è totalmente gratuito, inizia subito a usare il pannello web per server FiveM più semplice ma super funzionale.

## Accesso

Trovi il link e i dati di accesso per entrare in txAdmin nell’area txAdmin del pannello di controllo del tuo game server. Qui hai tutte le info per accedere facilmente all’interfaccia txAdmin e iniziare la configurazione.

![img](https://screensaver01.zap-hosting.com/index.php/s/69LdTK3FyNZNXid/download)

Copia username e password e clicca sul link (indirizzo IP:porta) per aprire l’interfaccia txAdmin e fare il login. L’interfaccia si aprirà in una nuova finestra, dove dovrai inserire i dati di accesso forniti.

![](https://screensaver01.zap-hosting.com/index.php/s/pp8GLQBoX4LoqTA/preview)



## Configurazione

Ora che sei dentro, puoi iniziare a configurare txAdmin e il tuo server. Dovresti vedere la homepage di txAdmin (Dashboard). In alto a sinistra noterai un messaggio che ti dice che il server deve ancora essere configurato. Clicca su **Vai alla pagina di configurazione** per partire con la configurazione.

![](https://screensaver01.zap-hosting.com/index.php/s/oXakf3qoJaim7ex/download)



### Benvenuto e nome server

Per prima cosa, scegli un nome per il tuo server da usare nel progetto. Questo nome non comparirà nella lista server, serve solo per l’interfaccia txAdmin e per i messaggi in chat/Discord.

![](https://screensaver01.zap-hosting.com/index.php/s/FCmd5xQ89wSPHfe/preview)



### Tipo di Deployment

Sotto Tipo di Deployment devi scegliere come vuoi configurare il tuo server. Hai queste opzioni: **Ricette Popolari**, Dati Server Esistenti, **Template URL Remoto** e **Template Personalizzato**. Ti spieghiamo ogni opzione nel dettaglio così puoi scegliere quella che fa per te.

![](https://screensaver01.zap-hosting.com/index.php/s/52HfyJSNLscApNE/preview)



### Template Popolari

I Template Popolari sono pacchetti preconfigurati pronti all’uso, il modo più semplice per far partire il tuo server FiveM. Se è il tuo primo server, ti consigliamo caldamente questa opzione. Questi template includono tutte le configurazioni e risorse necessarie per mettere su un server funzionante in un attimo, così puoi concentrarti su personalizzazioni e gameplay invece che sulla configurazione iniziale. Scegli semplicemente il template che ti serve.

![](https://screensaver01.zap-hosting.com/index.php/s/PSsf22NeebNBRw7/preview)



### Dati Server Esistenti

Usa questa opzione se hai già gestito un server con txAdmin e vuoi ora ospitarlo da noi. Puoi caricare i dati del server via FTP e indicare il percorso durante la configurazione. Se non hai mai usato FTP, dai un’occhiata alla nostra guida su [Accesso FTP](gameserver-ftpaccess.md). Ti spieghiamo passo passo come trasferire i file del server senza problemi, così il tuo server sarà online velocemente sulla nostra piattaforma.

![img](https://screensaver01.zap-hosting.com/index.php/s/KS4raRtHWmmw5iN/preview)





### Template URL Remoto

Se vuoi usare un template che non è incluso direttamente in txAdmin, questa funzione ti permette di specificare una ricetta personalizzata tramite un URL remoto. La ricetta verrà scaricata ed eseguita sul tuo server. È perfetta per configurazioni custom o setup specifici non presenti nelle opzioni standard di txAdmin. Inserendo l’URL, txAdmin scaricherà e applicherà automaticamente il template, assicurandoti che il server sia configurato esattamente come vuoi tu.

![](https://screensaver01.zap-hosting.com/index.php/s/jrGzTGp9FwLc82i/preview)

### Template Personalizzato

Questa opzione è pensata per chi ha già creato una propria ricetta e vuole ricaricarla. Ti permette di applicare facilmente le tue configurazioni custom senza dover rifare tutto da zero. Seleziona la tua ricetta esistente e txAdmin farà il resto, configurando il server esattamente come l’hai pensato.

![](https://screensaver01.zap-hosting.com/index.php/s/Z75q5RKakwfpHGy/preview)



### Recipe Deployer

Dopo aver scelto il Tipo di Deployment e seguito i passaggi, clicca su **Vai a Recipe Deployer**. Qui completerai gli ultimi step: **Rivedi la Ricetta**, **Inserisci Parametri**, **Esegui Ricetta** e **Configura server.cfg**.

Nel primo step puoi personalizzare ulteriormente la ricetta scelta o aggiunta. Se non serve modificare nulla, vai al Passo 2. Qui puoi inserire la tua [Licenza Personale](fivem-licensekey.md) e controllare i dati del database. Quando è tutto ok, clicca su **Esegui Ricetta**. Ci vorrà qualche istante. Infine, puoi modificare il file di configurazione del server (`server.cfg`) come preferisci.

![](https://screensaver01.zap-hosting.com/index.php/s/wFMD576sBQAAdxZ/download)

Per concludere, clicca su **Salva & Avvia Server**. Così hai finito la configurazione di txAdmin e puoi collegarti al tuo server.

## Scarica i Salvataggi

:::info
Nota: Prima di scaricare il salvataggio devi fermare il server per salvare il salvataggio attivo; dopo aver fermato il server, il salvataggio apparirà ricaricando la pagina una volta.
:::

Per scaricare il tuo salvataggio puoi usare il `Savegame-Manager` nell’interfaccia web:

![](https://screensaver01.zap-hosting.com/index.php/s/FryL7EXmEYBFBXD/preview)

Qui puoi facilmente scaricare il tuo salvataggio cliccando sul pulsante verde "Download":

![](https://screensaver01.zap-hosting.com/index.php/s/2o2gnQ3GKpwc22w/preview)

:::info
Attenzione, devi scaricare sia i file `.db` **che** `.fwl`.
:::

Fatto! Ora puoi usare il backup per altri scopi o caricarlo di nuovo in seguito.

## Carica & Attiva il Salvataggio

Per caricare i tuoi salvataggi, apriamo prima il nostro savegame manager.

Qui possiamo caricare il nostro salvataggio tramite drag&drop, dovrebbe apparire così:

![](https://screensaver01.zap-hosting.com/index.php/s/H2RJSyMGeiojfxD/preview)

:::info
Assicurati che il nome del salvataggio sia lo stesso con cui è stato creato dal gioco.
:::

Nel nostro caso i file di backup si chiamano `380622.fwl` e `380622.db`.

Ora inseriamo il nome del backup sopra, sotto `Savegame (worlds)`:

![](https://screensaver01.zap-hosting.com/index.php/s/Ggrys4xKjDm7inz/preview)

Sotto `ZAP Backups` cerchiamo il nostro salvataggio:

![](https://screensaver01.zap-hosting.com/index.php/s/beRMYmE4CicDNsz/preview)

Ora basta premere il pulsante giallo "Restore" su entrambi i file e confermare, così il salvataggio verrà caricato.

Dopo il caricamento, comparirà un messaggio in basso a destra:

![](https://screensaver01.zap-hosting.com/index.php/s/Zs9F4rNRen4QMcs/preview)

Ora puoi avviare il server, il salvataggio caricato sarà attivo sul server.



## Salvataggi Automatici

Purtroppo Valheim non supporta ancora i salvataggi automatici; per salvare il progresso devi fermare il server. Puoi usare il Restart Planner nell’interfaccia web per riavviare automaticamente il server a orari specifici:

![](https://screensaver01.zap-hosting.com/index.php/s/gTp5yJDQyHKX7eF/preview)

Qui puoi aggiungere facilmente un orario per un riavvio giornaliero o settimanale:

![](https://screensaver01.zap-hosting.com/index.php/s/nyqbMzjNL78qz9K/preview)

Dopo aver salvato il riavvio, puoi aggiungere più di un riavvio giornaliero o settimanale.

![](https://screensaver01.zap-hosting.com/index.php/s/BddatyGLAZZWnCA/preview)

Fatto! Il tuo server si riavvierà automaticamente agli orari impostati, e il salvataggio verrà effettuato durante questi riavvii.

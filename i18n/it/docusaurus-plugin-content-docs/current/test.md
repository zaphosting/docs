## Preparazione

La prima cosa da fare è scaricare le mod che vuoi installare sul tuo server. Puoi trovare una vasta selezione di mod su [7daystodiemods.com](https://7daystodiemods.com/). Lì troverai più di 1000 mod e ne vengono aggiunte regolarmente altre. Una volta scaricata una mod, dovrebbe contenere questa struttura di file:

```
  Config/
  ItemIcons/
  Resources/
  ModInfo.xml
```



## Installazione

Una volta scaricate le mod desiderate, puoi iniziare con l’installazione vera e propria. La configurazione delle mod avviene tramite FTP. Ti serve un client FTP per poter caricare i file sul tuo server. Se non sai cos’è l’FTP e come funziona, dai un’occhiata a questa guida: [Accesso via FTP](gameserver-ftpaccess.md)

![](https://screensaver01.zap-hosting.com/index.php/s/9Q86iArComw55cH/preview)



Nella directory ``/gXXXXX4/7dtd-linux-exp/`` devi creare una cartella chiamata **Mods**. Per farlo, clicca col tasto destro e seleziona **Crea directory**. 

![](https://screensaver01.zap-hosting.com/index.php/s/RE2n6WodsWq38Pr/preview)



Dopodiché, carica i file della mod nella cartella mods:

![](https://screensaver01.zap-hosting.com/index.php/s/WjNY5tMnAt7jfga/preview)



Così hai già completato l’installazione della mod. Alla prossima accensione del server, le mod dovrebbero caricarsi automaticamente.

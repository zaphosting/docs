Per connetterti al tuo server Valheim, apriamo prima l'interfaccia web e assicuriamoci di aver impostato una password sicura nelle impostazioni:

![](https://screensaver01.zap-hosting.com/index.php/s/XC7Cxd2zt75jyap/preview)

:::info
Nota: Senza la password il tuo server Valheim non si avvierà.
:::

Dopo aver impostato la password possiamo avviare il server, potrebbero volerci alcuni minuti prima che passi a "Online":

![](https://screensaver01.zap-hosting.com/index.php/s/GZQRqpGC6GGaAXa/preview)

## Connessione Diretta usando Valheim

Per prima cosa copiamo l’IP del server dall’interfaccia web:

![](https://screensaver01.zap-hosting.com/index.php/s/KMkbMyx4bTQkLJT/preview)

Ora apriamo Valheim e clicchiamo sul pulsante "Join IP" nella lista server:

![](https://screensaver01.zap-hosting.com/index.php/s/zqKp6sx5tEYRebx/preview)

Qui inseriamo l’IP del nostro server:

![](https://screensaver01.zap-hosting.com/index.php/s/EniEzmaP3E9JpLp/preview)

Ora possiamo connetterci direttamente al server cliccando su "Connect".


## Connessione Diretta usando Steam

Per prima cosa copiamo l’IP del server dall’interfaccia web:

![](https://screensaver01.zap-hosting.com/index.php/s/FkYSA8AFm53d8kK/preview)

Per connetterci direttamente al server Valheim, dobbiamo aprire la lista server su Steam:

![](https://screensaver01.zap-hosting.com/index.php/s/yHxMdtTBe7xTWeZ/preview)

Ora aggiungiamo il nostro server così:

![](https://screensaver01.zap-hosting.com/index.php/s/QQcjzriQ5K4Mj9k/preview)

:::info
Nota: La porta va calcolata aggiungendo +1 alla fine, quindi se la tua porta è 27043, devi inserire 27044 su Steam.
:::

![](https://screensaver01.zap-hosting.com/index.php/s/RgsszmnKQLCnYSa/preview)

Dopo aver aggiunto il server, dovremmo vederlo nella lista server:

![](https://screensaver01.zap-hosting.com/index.php/s/54XtiJzn7xndfPT/preview)

Ora possiamo connetterci direttamente cliccando su "Connect".


## Connessione tramite la Lista Server

Apriamo Valheim e clicchiamo su `Start Game`. Se non hai ancora un personaggio, creane uno ora, poi clicchiamo su `Start`.

Qui non vogliamo creare un nuovo mondo, quindi premiamo "Cancel".

![](https://screensaver01.zap-hosting.com/index.php/s/ZnKWT8eXCcArMwX/preview)

Ora clicchiamo su `Join Game` in alto nel menu e inseriamo il nome del nostro server nella sezione "Filter", mettendo anche la spunta su "Community".

![](https://screensaver01.zap-hosting.com/index.php/s/kFmZP5wyFQbpLiR/preview)

:::info
Se non vedi ancora il tuo server, aspetta un attimo: a seconda del numero di server, l’aggiornamento può richiedere qualche minuto.
:::

Ora dovremmo vedere il nostro server nella lista, clicchiamoci sopra e poi su `Connect`. Una volta connessi, inseriamo la password del server che abbiamo impostato nelle impostazioni e premiamo Invio!

Fatto! Ora puoi giocare sul tuo server Valheim tutto tuo!

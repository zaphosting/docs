## Panoramica
ZAP-Hosting offre tantissime opzioni di pagamento diverse con cui puoi noleggiare e attivare subito i tuoi server. Il processo di pagamento è semplice e veloce. Ecco le opzioni di pagamento che offriamo:
*  **ZAP Coins:** I ZAP Coins si caricano tramite la Cashbox (ricarica soldi) e possono essere usati per quasi tutti i prodotti.
* **Carta di credito:** Pagamento con carta di credito.
* **PayPal:** Pagamento tramite PayPal.
* **PaySafeCard:** Le PaySafeCard sono carte prepagate acquistabili in oltre 25.000 punti vendita in Germania. Per pagare serve il codice della carta.
* **Bitcoin:** Paga con Bitcoin. La transazione viene gestita dal provider Coinify.
* **Bonifico bancario:** Disponibile solo per ricaricare ZAP Coins. Dopo la ricezione del pagamento, i ZAP Coins verranno attivati.

## Processo di pagamento
Il processo di pagamento è semplice: configuri il server nella pagina di configurazione, in fondo alla pagina scegli il metodo di pagamento desiderato. Cliccando su "Compra ora", se necessario verrai reindirizzato al provider di pagamento selezionato per completare la transazione. Dopo aver ricevuto il pagamento, il server viene attivato in pochi secondi.

## ZAP Coins: Ricarica
Puoi ricaricare ZAP Coins tramite la tua Cashbox usando uno qualsiasi dei metodi di pagamento disponibili. Anche i server a contratto possono essere pagati con ZAP Coins. Nell’interfaccia web puoi ricaricare ZAP Coins sotto la voce **Cash in** nella tua Cashbox.

:::info
Non è possibile pagare parzialmente con ZAP Coins. Se hai già ZAP Coins, ad esempio grazie a un codice ZAP Coins, puoi aggiungere credito quanto vuoi. Consiglio: se paghi con ZAP Coins, tieni d’occhio le nostre offerte Cash In: se l’offerta è attiva, riceverai credito extra aumentando la ricarica.

I bonifici possono essere usati per ricaricare ZAP Coins. Appena riceviamo il bonifico, il credito viene attivato e puoi usarlo subito.
:::

## Cashbox
La Cashbox è accessibile dall’interfaccia web sotto la voce **Cashbox**. Oltre a ricaricare soldi, la Cashbox offre queste funzioni extra:
* **Aggiungi carta di credito:** Puoi salvare carte di credito. Salvando una carta, non dovrai più inserire i dati ogni volta che paghi con carta.
* **Ricarica soldi:** Puoi ricaricare soldi anche tramite la Cashbox.
* **Riscatta voucher:** Qui puoi riscattare voucher che ti danno ZAP Coins. Dopo aver inserito il codice voucher, i ZAP Coins vengono accreditati subito sul tuo account.
* **Ricarica automatica ZAP Coins:** Puoi attivare la ricarica automatica dei ZAP Coins. Utile se hai ordinato un server a contratto. Per la ricarica automatica serve usare PayPal.
* **Panoramica transazioni:** Una panoramica di tutte le transazioni effettuate finora.

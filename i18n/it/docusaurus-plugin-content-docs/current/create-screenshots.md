## Crea screenshot

A volte è necessario inviare screenshot al supporto.  
Per farlo, lo screenshot deve essere condiviso tramite un link.  
Questo link può puntare a un cloud personale o a un "web uploader", chiamato anche tool per screenshot web.

Se il tuo cloud personale o il tool web non offre una funzione per creare screenshot direttamente, puoi farlo facilmente sul PC con tool semplici.  
Su Windows basta usare lo strumento "Strumento di cattura" (Snipping Tool). Con questo puoi salvare lo screenshot localmente e poi caricarlo.  
Su Linux puoi usare lo strumento già installato "Screenshot" o altri tool come "FlameShot" per creare screenshot locali da caricare poi.

Qui sotto trovi una breve descrizione dei 2 tool per screenshot più popolari e gratuiti sul mercato:

:::info
Non è consigliato condividere dati privati o sensibili pubblicamente. Se nello screenshot ci sono dati visibili, vanno oscurati o sostituiti con dati di esempio.
:::

### Imgur

Imgur offre un uploader per file propri come immagini/video.  
Dopo l’upload, viene creato un link.  
Questo link può essere condiviso con altri.

Importante: Imgur non ha una funzione per fare screenshot direttamente, quindi devi prima creare lo screenshot (vedi sopra "Crea screenshot") e poi caricarlo.

[Imgur-Website](https://imgur.com/upload)

Opzione di upload:

![](https://screensaver01.zap-hosting.com/index.php/s/4sCEeKkyGEm3EXd/preview)

Dopo l’upload puoi fare piccole modifiche allo screenshot.

Poi puoi pubblicare lo screenshot come Post nella Community (visibile pubblicamente) o come Nascosto (non visibile pubblicamente).

Quando fatto, ti viene mostrato il link da condividere:

![](https://screensaver01.zap-hosting.com/index.php/s/dJCpsKtka9rHX6T/preview)

### Lightshot

Lightshot non offre upload diretto per immagini/screenshot.  
L’upload si fa solo tramite il client proprio.  
Il client si scarica dal sito e offre anche strumenti per evidenziare o oscurare parti dello screenshot.

Scarica il client [Download](https://app.prntscr.com/en/)  
Importante: non esiste versione per Linux.

![](https://screensaver01.zap-hosting.com/index.php/s/iGEccwZZDXqLGKp/preview)

Dopo il download, installa il client.  
Poi, di default, puoi creare uno screenshot o selezionare un’area con il tasto "Print".

Dopo aver selezionato l’area, puoi modificare lo screenshot.

Quando hai finito, se vuoi, puoi caricarlo cliccando sulla "Cloud".

Dopo l’upload, puoi copiare il link e condividerlo con chi vuoi.

## Introduzione

Creare e condividere screenshot aiuta a migliorare la comunicazione con il supporto fornendo informazioni visive chiare. Questa guida spiega come creare screenshot sul tuo computer e caricarli direttamente nei ticket di supporto.

## Creare Screenshot

Il modo migliore per condividere screenshot in modo sicuro è crearli localmente sul tuo dispositivo e caricarli direttamente nel ticket di supporto tramite drag and drop. Così gli screenshot restano privati e vengono condivisi in modo sicuro senza usare link esterni.

### Windows: Snipping Tool

Windows include lo *Snipping Tool* integrato, che ti permette di catturare un’area specifica, una finestra o l’intero schermo e salvare l’immagine localmente.

1. Apri lo **Snipping Tool** cercandolo nel menu Start.
2. Clicca su **Nuovo** per iniziare uno screenshot.
3. Seleziona l’area che vuoi catturare.
4. Salva lo screenshot in una posizione comoda sul tuo PC.
5. Carica il file dello screenshot salvato direttamente nel ticket di supporto trascinandolo nell’interfaccia del ticket.

### Linux: Screenshot Tool o FlameShot

La maggior parte delle distribuzioni Linux ha uno strumento *Screenshot* di default. In alternativa, tool di terze parti come *FlameShot* offrono funzionalità avanzate.

1. Apri lo strumento Screenshot o FlameShot.
2. Seleziona l’area o la finestra da catturare.
3. Salva lo screenshot localmente.
4. Carica il file dello screenshot salvato direttamente nel ticket di supporto usando drag and drop.

## Caricare Screenshot nel Supporto

Dopo aver salvato lo screenshot localmente, apri il tuo ticket di supporto nel pannello ZAP-Hosting o nel portale di supporto.

- Trascina e rilascia il file dello screenshot nell’area messaggi del ticket.
- Aspetta che il caricamento sia completato.
- Invia il ticket con lo screenshot allegato.

Questo metodo evita di condividere link esterni e mantiene i tuoi dati al sicuro.

:::info Condivisione Sicura degli Screenshot
Evita di condividere screenshot pubblicamente o tramite servizi di upload di terze parti. Se lo screenshot contiene informazioni sensibili o private, assicurati di sfocarle o oscurarle prima di caricarlo.
:::

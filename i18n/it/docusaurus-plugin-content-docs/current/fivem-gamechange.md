## Cambiare la versione di FiveM
Grazie al nostro gamecloud, puoi passare facilmente e gratuitamente tra tutte le versioni di FiveM.

:::info
Il passaggio tra i giochi basati su Linux (txAdmin, ESX Roleplay, FiveM Linux) e Windows cancellerà tutti i dati.
:::

Assicurati sempre di avere un backup dei tuoi dati, puoi crearne uno facilmente nella scheda "Backup" nell’interfaccia.

## Il cambio nel dettaglio
Per cambiare il gioco (la versione) del tuo server FiveM, devi aprire "Games" nella nostra interfaccia.

![](https://screensaver01.zap-hosting.com/index.php/s/n82YtN2DEkAeNpq/preview)

Qui vedrai tutte le versioni di FiveM sotto "Available Games", cerca semplicemente "FiveM"

![](https://screensaver01.zap-hosting.com/index.php/s/drTykzYs2527b7D/preview)

Scegli la versione che preferisci e premi il pulsante verde per avviare l’installazione. Apparirà un popup che richiede la conferma per iniziare il processo.

:::info
Il cambio tra Linux e Windows modifica l’indirizzo IP del tuo server, a meno che tu non abbia prenotato un IP dedicato.

Se passi tra ESX RP, Linux o txAdmin, dopo l’installazione devi abilitare la versione con il pulsante blu in Games.
:::

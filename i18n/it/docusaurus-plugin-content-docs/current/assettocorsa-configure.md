## Collegati via FTP

Prima di poter configurare il server, devi impostare l’[Accesso via FTP](gameserver-ftpaccess.md):

![](https://screensaver01.zap-hosting.com/index.php/s/GdzeM4wyAGtGk8e/preview)

Una volta fatto, puoi connetterti e aprire la cartella del server:

![](https://screensaver01.zap-hosting.com/index.php/s/dkSSLWF5QGGf4yR/preview)


## Preparazione

Per prima cosa apriamo la cartella di Assetto Corsa, puoi farlo aprendo Steam e cliccando col tasto destro su "Assetto Corsa" nella lista giochi.

Ora apri "File Locali" per accedere ai file locali del gioco:

![](https://screensaver01.zap-hosting.com/index.php/s/XKBgdiNbZ5AXtia/preview)

Adesso apri la cartella "server", lì troverai il file "acServerManager", aprilo.

## Imposta la Mappa

Ora sei nell’interfaccia "AC Server Manager", con cui puoi creare e gestire le configurazioni del server. Prima di tutto crea una nuova config.

![](https://screensaver01.zap-hosting.com/index.php/s/Z37toTgBHDmf54n/preview)

Ora puoi configurare il server, nel nostro caso vogliamo solo modificare piste e auto, quindi apriamo la tab "Tracks".

Qui puoi cliccare sulla mappa che preferisci, sopra puoi regolare il numero di auto in pista.

![](https://screensaver01.zap-hosting.com/index.php/s/Hr9AMt9SHyas4CN/preview)

:::info
Assicurati di non impostare più auto rispetto ai slot disponibili sul server, altrimenti il server non partirà.
:::

## Imposta le Auto

Ora passiamo alla tab "Cars / Entry List", qui puoi aggiungere nuove auto:


Per attivare un’auto, devi mettere la spunta a sinistra.

![](https://screensaver01.zap-hosting.com/index.php/s/4w643p56GLXK9cP/preview)

Al centro vedi le auto che possono essere piazzate sulla griglia, qui puoi scegliere le auto con i colori che vuoi far apparire in griglia:

![](https://screensaver01.zap-hosting.com/index.php/s/T5SfpsdAPxfMwnY/preview)

:::info
Per selezionare più auto, puoi mettere più spunte per aggiungere le diverse auto al centro.

Ricorda che tutti gli slot auto devono essere riempiti e non devi superare il numero di slot disponibili.

![](https://screensaver01.zap-hosting.com/index.php/s/FTeJX3BJipBRFQP/preview)
:::

## Carica la Config

Ora puoi salvare la config con il pulsante "Save" ed esportarla con "Export":

![](https://screensaver01.zap-hosting.com/index.php/s/b7co7wfcegmwPYd/preview)

Qui seleziona una cartella libera dove salvare i dati della config.

Ora puoi chiudere AC Server Manager e aprire la cartella in cui hai esportato le config.

![](https://screensaver01.zap-hosting.com/index.php/s/4Nysjo24BAAGbqe/preview)

Queste config possono essere copiate via Drag&Drop nella cartella "cfg" del tuo server di gioco.

![](https://screensaver01.zap-hosting.com/index.php/s/YKHtnDMSqBgssDc/preview)

Ora puoi avviare il server con la nuova config.

## Prefazione

Grazie al nostro sistema cloud, puoi sempre modificare il tuo game server ZAP-Hosting in base ai tuoi desideri e necessità.  
Ad esempio, se ti servono più slot o più RAM, nessun problema, puoi cambiarlo in qualsiasi momento.  
Tieni presente però che se hai usato un voucher, un downgrade non sarà più possibile durante il periodo di validità del voucher.

## Come funziona

Quando cambi il tuo pacchetto, il periodo residuo viene sempre ricalcolato, quindi se vuoi fare un upgrade del server, dovrai pagare la differenza per il periodo rimanente.  
Il prezzo del prodotto cambierà automaticamente al prossimo rinnovo, in base alle nuove condizioni.  
Ricorda che purtroppo i coupon non possono essere applicati agli upgrade, valgono solo per ordini nuovi.

## Upgrade

Puoi sempre trovare il Manager di Up-/Downgrade nella dashboard del tuo prodotto, non importa che tipo di prodotto sia.  
Gli upgrade sono attualmente possibili per tutti i prodotti tranne i Dedicated Server.

Il manager di upgrade/downgrade si trova nel menu a sinistra della dashboard del prodotto:

![](https://screensaver01.zap-hosting.com/index.php/s/f9z7xyEpB55ZZ52/preview)

### Opzioni di upgrade

Una volta dentro il nostro manager, troverai i parametri del tuo prodotto che puoi personalizzare.

Per i gameserver, è possibile spostarli tra diverse location.

![](https://screensaver01.zap-hosting.com/index.php/s/tW3sbKxSszQpb8a/preview)

:::info
**Importante** Questo processo può causare perdita di dati, assicurati di avere sempre un backup dei tuoi dati!
:::

Se vuoi spostare il server, clicca semplicemente sulla location corrispondente per selezionarla.

Qui sotto trovi altri parametri, nel nostro esempio di un server FiveM il numero di slot e il boost RAM, oltre al tipo di sistema e all’IP dedicato:

![](https://screensaver01.zap-hosting.com/index.php/s/eb5FsMSdwJszjWK/preview)

*** 

* Slot Gameserver: Se ti servono più slot sul tuo game server, sposta il cursore verso destra per avere più spazio per giocatori simultanei.

* RAM Boost: Utile per risorse, mod, add-on o simili. Aumenta la RAM disponibile del tuo prodotto.

* CPU & Hostserver: Se ti serve più potenza, puoi passare a un sistema premium che ti dà un boost di performance.

* Indirizzo IP: Se ti serve un IP fisso che non cambia finché il prodotto esiste, ti serve questa opzione.

:::info
⚠ L’IP dedicato non va confuso con un dominio, un IP dedicato è comunque assegnato in modo casuale e non può essere modificato.
:::

### Pagamento

Pagare l’upgrade è super semplice, scorri in basso fino ai metodi di pagamento:

![](https://screensaver01.zap-hosting.com/index.php/s/PFsBtsMr7iyRadf/preview)

* 1: Il nuovo prezzo mensile del pacchetto, in questo esempio il pacchetto costa 16,84€ al mese, già con gli upgrade selezionati inclusi.

* 2: Il sovrapprezzo è l’importo da pagare per il periodo residuo del prodotto, va pagato subito.

![](https://screensaver01.zap-hosting.com/index.php/s/XtyJ6X4jjfz3AEY/preview)

Scegli il metodo di pagamento che preferisci e completa l’upgrade cliccando su "Upgrade".

Ci vorrà solo un attimo per applicare l’upgrade, di solito è disponibile in pochi secondi.

:::info
Di solito è necessario riavviare il prodotto per applicare le nuove specifiche.
:::

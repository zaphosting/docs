## Accesso all’Interfaccia Web

Inizia andando sulla dashboard del tuo prodotto server di gioco sul nostro sito. Da lì, naviga nella sezione **Impostazioni->Giochi** che trovi sulla sinistra.

Ora sei nella sezione Game Switch, dove puoi gestire le installazioni dei giochi, attivare un gioco e modificare il prezzo per slot se necessario.

![](https://screensaver01.zap-hosting.com/index.php/s/NT6d4f7iZa62iPf/preview)

## Gestione Giochi

Nella sezione **Giochi** della tua Interfaccia Web, vedrai la lista dei giochi attualmente installati. Accanto potrai vedere la versione del gioco e vari pulsanti per gestire l’installazione.

:::info Server di Gioco Lifetime
Con il nostro prodotto Lifetime Game Server, il prezzo della funzione game switch funziona diversamente. Per ogni gioco che vuoi possedere permanentemente sul tuo server lifetime, c’è una **tariffa una tantum** di **30€** per ogni gioco che vuoi acquistare. Dopo l’acquisto, il gioco sarà **disponibile permanentemente** su quel server lifetime e potrai switchare a quel gioco in qualsiasi momento.

A parte il prezzo per i server lifetime, il processo di game switch è identico, ma con l’acquisto dei giochi invece di basarsi sul prezzo per slot.
:::

![](https://screensaver01.zap-hosting.com/index.php/s/JJfYoRSi3M26qbc/preview)

### Installare un Nuovo Gioco

Per installare un nuovo gioco sul tuo server, seleziona il menu a tendina **Giochi Disponibili**, dove potrai sfogliare i giochi che puoi installare senza costi aggiuntivi.

![](https://screensaver01.zap-hosting.com/index.php/s/iN7rNje3zaBPMgf/preview)

Qui trovi anche il menu a tendina **Altri Giochi**. Nel nostro ampio catalogo, alcuni giochi possono avere prezzi per slot diversi, il che può significare che non puoi installare quel gioco senza costi extra. Per risolvere, dovrai pagare un sovrapprezzo per aumentare il prezzo per slot. Dai un’occhiata alla sezione Cambia Prezzo Slot per più info.

:::info Numero Minimo di Slot
Alcuni giochi della nostra collezione richiedono un numero minimo di slot per poter essere installati sul server. In questi casi, passa col mouse sull’icona di avviso per saperne di più. Usa la funzione di upgrade/downgrade per aggiornare il server in base alle info.

![](https://screensaver01.zap-hosting.com/index.php/s/AfAonXCqmLFDyay/preview)
:::

Quando trovi un gioco che vuoi installare, clicca il pulsante verde di download per partire con l’installazione.

![](https://screensaver01.zap-hosting.com/index.php/s/EjCQK6WYac7Ejfr/preview)

:::caution Cambio Sistema Host
A seconda del gioco, potresti dover cambiare sistema host per continuare l’installazione. Questo succede perché il gioco che vuoi installare gira su un sistema operativo diverso (Linux o Windows) e quindi richiede un host differente.

Il cambio host è un processo **distruttivo**, cioè tutti i dati attualmente sul server **verranno** cancellati. Assicurati di fare il backup di file o salvataggi se vuoi tenerli al sicuro prima di procedere.
:::

Dopo aver cliccato il download, ti apparirà un pop-up per confermare l’installazione. Accetta per iniziare l’installazione automatica. Potrai vedere il progresso in alto nella pagina.

Quando finisce, vedrai il nuovo gioco nella sezione **Giochi Installati**. Hai installato con successo un nuovo gioco sul tuo server.

### Attivare & Cancellare un Gioco

Prima di giocare, devi attivare il gioco, cioè impostarlo come gioco attivo sul server.

Basta andare nella sezione **Giochi Installati** e cliccare il pulsante blu di attivazione. Il gioco attivo è indicato con un’icona verde di spunta. Dopo aver cliccato attiva, ti apparirà un prompt da accettare per procedere.

![](https://screensaver01.zap-hosting.com/index.php/s/XT8jwyDq6j5zXPn/preview)

Il server attiverà automaticamente il gioco e si riavvierà. Quando è pronto, il gioco sarà attivo e puoi iniziare a giocare. Hai attivato con successo un gioco sul tuo server.

### Cancellare & Reinstallare un Gioco

Cancellare o reinstallare un gioco è semplice e si fa dalla sezione **Giochi Installati**.

Trova il gioco su cui vuoi agire e usa il pulsante arancione di reinstallazione per reinstallarlo. Questo processo è distruttivo e cancellerà tutti i dati di quel gioco, come server o configurazioni.

![](https://screensaver01.zap-hosting.com/index.php/s/x8kmeor6S7mXe9g/preview)

Se vuoi cancellare un gioco, usa il pulsante rosso di cancellazione accanto al gioco nella sezione **Giochi Installati** per rimuoverlo dal server.

![](https://screensaver01.zap-hosting.com/index.php/s/fwgGGzjA5mE2qRs/preview)

## Cambiare Prezzo Slot

Alcuni giochi nel nostro catalogo hanno prezzi per slot diversi, che possono impedirti di installarli. In questi casi, devi aumentare il prezzo per slot e pagare un sovrapprezzo per rendere di nuovo disponibili quei giochi.

Puoi farlo in due modi: cliccando il pulsante **Cambia Prezzo Slot** in alto o cercando prima i giochi per vedere i prezzi slot associati a quelli non disponibili.

Ti consigliamo di usare prima il menu a tendina **Altri Giochi** per trovare i giochi e vedere facilmente i prezzi slot. Quando trovi il gioco, clicca il pulsante blu di acquisto.

![](https://screensaver01.zap-hosting.com/index.php/s/FnrqX4SFQq2MkPL/preview)

Si aprirà la finestra per modificare il prezzo slot con il prezzo preimpostato per il gioco scelto. Vedrai il sovrapprezzo da pagare e il nuovo prezzo del pacchetto basato sul cambio prezzo slot. Puoi anche sfogliare di nuovo i giochi se vuoi.

![](https://screensaver01.zap-hosting.com/index.php/s/EGaZRMzjnF3QPiL/preview)

Quando sei pronto, clicca il pulsante avanti che ti porterà alla selezione del metodo di pagamento per coprire il costo del sovrapprezzo. Scegli il metodo e usa il pulsante compra ora per finalizzare il cambio prezzo slot.

![](https://screensaver01.zap-hosting.com/index.php/s/83oPQocpmSX58R4/preview)

Se tutto va a buon fine, il prezzo slot prenotato per il tuo server sarà aggiornato. Ti verrà chiesto di riavviare il server per completare tutto.

Ora puoi tornare alla sezione **Giochi Disponibili** e installare il gioco normalmente. Hai cambiato con successo il prezzo slot prenotato per il tuo server.

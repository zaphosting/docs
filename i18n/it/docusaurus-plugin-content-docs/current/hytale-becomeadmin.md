## Introduzione
Assegnare i permessi da admin su un server Hytale permette ai giocatori di fiducia di aiutarti a gestire il server. Gli admin, chiamati anche operatori, possono eseguire comandi server, gestire i giocatori e supportare la moderazione del server. Così il proprietario può delegare le attività di routine e mantenere un’esperienza di gioco fluida.

Hytale è uscito il 13 gennaio 2026 ed è attualmente in Early Access. Essendo ancora in fase di sviluppo attivo, il software del server, i file di configurazione, il supporto alle mod e i processi di installazione potrebbero cambiare nel tempo.

## Assegnare i permessi



Gli admin su un server Hytale ottengono lo status di operatore, che dà loro permessi avanzati sui comandi. Puoi farlo tramite la **console del server** o con **comandi in-game** da un operatore già esistente. Per dare i permessi operatore a un giocatore, inserisci questo comando nella console o in-game con i permessi giusti:

```
/op add <playername>
```



## Rimuovere i permessi

Se un giocatore non ha più bisogno dei permessi admin o non deve più avere privilegi elevati, puoi togliere lo status operatore in qualsiasi momento:

```
/op remove <playername>
```

Una volta rimosso, il giocatore tornerà ai permessi standard e non potrà più usare i comandi admin.



## Conclusione

Diventare admin su un server Hytale tramite i permessi operatore è fondamentale per la gestione del server, permettendo ai giocatori di fiducia di aiutarti con moderazione e compiti amministrativi.

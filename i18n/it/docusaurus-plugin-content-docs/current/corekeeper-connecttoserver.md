### Puoi connetterti al server di gioco Core Keeper così:

Apri il pannello di controllo del tuo game server ZAP-Hosting e assicurati che il server sia attivo e funzionante:

![](https://screensaver01.zap-hosting.com/index.php/s/2NbQaA6E2CJ4ZE8/preview)

Avvia il gioco e seleziona la voce di menu "Multiplayer": 

![385-351-max](https://screensaver01.zap-hosting.com/index.php/s/t36H8NxmLa4gx5S/preview)

Poi seleziona la voce di menu "Accesso diretto": 

![](https://screensaver01.zap-hosting.com/index.php/s/3MdnG7Xb2222EPQ/preview)

Qui inserisci l’IP e la porta che trovi nel pannello del tuo game server:

![](https://screensaver01.zap-hosting.com/index.php/s/R9nRFWFrS9Fbwox/preview)

Il client si connetterà al server cliccando su "Connetti".

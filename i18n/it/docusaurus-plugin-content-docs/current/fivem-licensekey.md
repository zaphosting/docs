Dopo il login, vai nella sezione **Servers** dalla barra in alto per accedere alla gestione delle key. Questo sostituisce il vecchio sito FiveM Keymaster con un portale tutto-in-uno. Puoi anche rinnovare una license key usando lâ€™opzione **Re-Activate** su questa pagina.

Ora vedrai la nuova key nella tabella. Copiala usando i pulsanti a destra e tienila pronta per il prossimo step.

![](https://screensaver01.zap-hosting.com/index.php/s/3Hd8tQqJA4xPKWk/preview)

## Aggiungere la Key al Server

Con la license key pronta, devi sostituire quella esistente sul tuo FiveM game server con quella nuova. Per farlo, accedi al pannello txAdmin del tuo server FiveM tramite la dashboard web del server. Usa le credenziali indicate per il login.

Una volta dentro txAdmin, vai su **CFG Editor** nel menu a sinistra per aprire il file `server.cfg`. Trova la riga con il parametro `sv_licensekey` e sostituiscila con la tua license key personalizzata. Salva le modifiche cliccando su salva.

![](https://screensaver01.zap-hosting.com/index.php/s/2E8j9jtykcjwF7L/preview)

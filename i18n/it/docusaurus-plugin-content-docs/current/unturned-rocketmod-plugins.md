## Plugins auf einem Unturned Rocket Mod Server installieren

### Schritt 1
Zuerst muss eine Verbindung zum Server über den [FTP-Zugang](gameserver-ftpaccess.md) hergestellt werden.

Sobald die FTP-Verbindung steht, öffnest du das Verzeichnis „Plugins“.  
Dieses findest du unter  
```
/unturned-rocket/Servers/ZAP001/Rocket/Plugins. 
```
Standardmäßig sollte es leer sein: 

![](https://screensaver01.zap-hosting.com/index.php/s/wYY7jLji9YaNcBp/preview)

### Schritt 2

Jetzt kannst du ein Plugin hochladen.  
Rocketmod Plugins findest du zum Beispiel auf Github: https://github.com/RocketModPlugins  
Nachdem du das gewünschte Plugin runtergeladen hast, musst du es entpacken.  
Je nach Betriebssystem kannst du das Archiv direkt entpacken, manchmal brauchst du aber Programme wie 7Zip.

Unter Windows reicht es z.B., das ZIP-Archiv per Doppelklick zu öffnen und den Inhalt per Drag & Drop auf den Desktop zu ziehen. 

![](https://screensaver01.zap-hosting.com/index.php/s/qnpy29HySQzJTBL/preview)

Anschließend kannst du das Plugin per Drag & Drop hochladen:  

![](https://screensaver01.zap-hosting.com/index.php/s/o4ZNQtyosnw5eHo/preview)

Jetzt ist das Plugin auf deinem Server verfügbar:

![](https://screensaver01.zap-hosting.com/index.php/s/DYXpnZ2n5ibxW5t/preview)

### Schritt 3

Starte jetzt den Gameserver über das Webinterface: 

![](https://screensaver01.zap-hosting.com/index.php/s/pgyBsYcoXNP7dnL/preview)

Dein Server läuft jetzt mit dem gewünschten Plugin: 

![](https://screensaver01.zap-hosting.com/index.php/s/ZZLPBprWoBZfTQ6/preview)

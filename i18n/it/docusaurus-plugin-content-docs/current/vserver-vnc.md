## Introduzione

VNC (Virtual Network Computing) è un software che mostra il contenuto dello schermo di un computer remoto su un computer locale e, in cambio, invia i movimenti di tastiera e mouse del computer locale a quello remoto. Questo ti permette di accedere al tuo server da remoto anche se non è avviato correttamente.

In caso di problemi di avvio o di rete, hai comunque la possibilità di accedere al sistema e intervenire.

## Uso del VNC
Il client web VNC si trova nel pannello di controllo del VPS. Per accedervi, clicca sull’opzione console seriale nella sezione strumenti dell’amministrazione e poi apri la connessione VNC tramite il pulsante **Avvia tunnel VNC**.

![](https://screensaver01.zap-hosting.com/index.php/s/AgSL8QcynHSfXFA/preview)



Dopodiché si aprirà il client web della console VNC. Qui puoi vedere le informazioni che il server fornisce in quel momento. Può essere, ad esempio, il processo di avvio, la schermata di login o l’interfaccia grafica del sistema operativo.

L’esempio seguente mostra un VPS Windows. Questo server è stato avviato con successo, quindi la console VNC mostra la schermata di login in questo stato. Puoi effettuare il login anche se il server non ha una connessione internet attiva. Clicca sul pulsante "Invia CtrlAltDel" e accedi con il tuo username e password.

## Amministrazione Prodotto

Nella sezione **Amministrazione Prodotto** trovi tutte le pagine necessarie per gestire il tuo server. Le opzioni per la gestione del server sono elencate dettagliatamente qui sotto.

### Dashboard

La **Dashboard** del tuo server è il cuore dell’amministrazione. Qui trovi tutte le informazioni base ed essenziali sul tuo servizio in modo chiaro. Questo include, ad esempio, informazioni sulla location, stato, IP: Porta, memoria usata, gioco in corso, nome server, mappa e numero di slot disponibili. Inoltre, viene mostrato anche l’utilizzo di CPU, memoria e RAM.

![](https://screensaver01.zap-hosting.com/index.php/s/ieTPe2kN9FjrTTe/preview)

Dalla dashboard puoi anche avviare, fermare o riavviare il tuo servizio, così hai il pieno controllo del suo funzionamento. Inoltre, puoi accedere alla console live per avere info in tempo reale sullo stato del server e intervenire subito se serve.

### Statistiche

Nella sezione **Statistiche** ottieni una panoramica completa dei valori di carico del tuo server. Trovi info su vari aspetti come il carico degli slot disponibili, CPU e memoria. Puoi visualizzare questi dati per diversi intervalli di tempo, ad esempio per gli ultimi **60 minuti, 24 ore, 3 giorni, 7 giorni** o **30 giorni.**

![](https://screensaver01.zap-hosting.com/index.php/s/N8jcmdqqG2Xt4Bk/preview)

Queste info ti danno un’ottima visione sull’utilizzo del server. Inoltre, le statistiche sono utili per identificare eventuali problemi di performance e restringere le cause. Ad esempio, puoi capire quando l’utilizzo è aumentato e se è legato a eventi o cambiamenti specifici. Questo è fondamentale quando sorgono problemi improvvisi, così puoi agire in modo mirato per risolverli e analizzare sistematicamente le cause.

### DDoS Manager

Visualizza gli attacchi DDoS passati per valutare meglio minacce e pattern. Attiva le notifiche per attacchi DDoS in corso sul tuo server per reagire velocemente in caso di incidente.

A seconda della location del server e della protezione DDoS disponibile, puoi anche monitorare gli attacchi attivi in tempo reale. Questo include dettagli sul traffico, mostrando sia il traffico legittimo che quello bloccato dai sistemi di mitigazione. Questo livello di trasparenza ti aiuta a capire come funziona la protezione e a individuare rischi potenziali in anticipo.

:::info Accesso DDoS Manager mancante?
L’uso del DDoS Manager richiede un IP dedicato o l’opzione aggiuntiva "Panoramica DDoS Manager". Se il tuo pacchetto attuale non include questi componenti, puoi aggiungerli in qualsiasi momento aggiornando il servizio.
:::

![](https://screensaver01.zap-hosting.com/index.php/s/ScCCCY52CMLgfyE/preview)

:::warning
L’uso del DDoS Manager è possibile solo se hai prenotato un IP dedicato o l’opzione "Panoramica DDoS Manager" come extra. Se mancano questi componenti nel tuo pacchetto, puoi aggiungerli in qualsiasi momento con un upgrade.
:::

### File di Log

Hai un problema tecnico e non sai da dove viene? Una prima strategia è controllare i **File di Log**. Questi file contengono tante info sul server, come eventi attuali, suggerimenti o messaggi di errore.

I messaggi di errore in particolare aiutano a identificare e capire meglio la causa del problema. Spesso si riesce a scovare e risolvere il problema dando un’occhiata mirata ai file di log.

:::warning
Non ti aiutano o non li capisci? Tranquillo! Siamo disponibili ogni giorno nel supporto. Apri un **[Ticket](https://zap-hosting.com/en/customer/support/)** e spiegaci il problema. Lo controlleremo il prima possibile e ti aiuteremo a risolverlo! :)
:::

### Registro Eventi

Il **Registro Eventi** ti offre una panoramica completa di tutte le attività legate all’amministrazione del server. Puoi vedere quando un server è stato avviato, fermato o crashato, così come quando sono stati fatti aggiornamenti, backup o nuove installazioni.

![](https://screensaver01.zap-hosting.com/index.php/s/xNzsaMbW5BS5KJC/preview)

### Console Live

La **Console Live** ti mostra in tempo reale tutti gli eventi attuali sul server. Qui vedi info sulle attività dei giocatori, eventuali errori e messaggi di sistema. Questa panoramica ti permette di capire cosa succede sul server e di individuare problemi in anticipo.

![](https://screensaver01.zap-hosting.com/index.php/s/dktDADkq3bzG3F6/preview)

Oltre a monitorare gli eventi, puoi anche inviare comandi direttamente al server dalla **Console Live**. Utile per modificare impostazioni, eseguire azioni o rispondere rapidamente a problemi.

### Interruzioni di Servizio

Il tuo server è stato offline temporaneamente e non sai perché? Il registro eventi e i file di log non ti hanno dato info? Potrebbe essere un problema non del server ma del sistema host su cui gira.

Se il nostro sistema di monitoraggio rileva un’interruzione generale, i tecnici vengono avvisati automaticamente e si occupano del problema il prima possibile.

Qui trovi una panoramica della cronologia delle interruzioni del sistema host dove gira il tuo server. Maggiori info e lo stato aggiornato del servizio li trovi sulla nostra nuova [pagina status](https://status.zap-hosting.com/).

## Strumenti

Nella sezione **Strumenti** del pannello di navigazione trovi varie sottosezioni che ti aiutano a gestire il server. Ci sono **FTP Browser**, **Database** e **Backup**. Qui sotto ti spieghiamo nel dettaglio queste aree.

### FTP Browser

Con l’**FTP Browser** trovi tutte le info per connetterti via FTP. L’accesso FTP ti dà accesso diretto ai file del server. Consigliamo di usare un client FTP esterno come **Filezilla** o **WinSCP** per un’esperienza ottimale. Se non sai come usare FTP, dai un’occhiata alla guida [Accesso via FTP](gameserver-ftpaccess.md).

![](https://screensaver01.zap-hosting.com/index.php/s/sX85i9HyzR9wmAM/preview)

**FileZilla:** FileZilla è probabilmente il client FTP più semplice e conosciuto. Basta installarlo, avviarlo e inserire i dati di accesso che trovi qui sopra. Con un clic su "Connetti" sei subito dentro al server e puoi gestire i file.

**WinSCP:** WinSCP è pensato per SCP e SFTP (versioni criptate di FTP), ma funziona bene anche con FTP normale. È un programma più completo di FileZilla, più adatto a utenti avanzati.

### Database

Oltre ai server di gioco, offriamo anche 10 database inclusi. Servono ad esempio quando devi salvare dati in un database. In **Database** puoi creare un database e vedere le credenziali di accesso. Puoi anche attivare i backup e importare backup esistenti.

![](https://screensaver01.zap-hosting.com/index.php/s/WFLwCbZSQfYjrmD/preview)

### Backup

In **Backup** puoi creare backup manuali e automatici per il server e il database associato. Così recuperi i dati facilmente in caso di problemi o perdite.

Nelle impostazioni scegli se fare backup automatici giornalieri o settimanali. Puoi anche decidere se eliminare i backup più vecchi quando lo spazio si riduce.

![](https://screensaver01.zap-hosting.com/index.php/s/nm9za5LTMEcNsga/preview)

Per i nostri server di gioco forniamo 10 GB di spazio per backup sul server di storage di default. Se ti serve più spazio, puoi fare un upgrade a pagamento. Così hai sempre spazio a sufficienza e puoi accedere anche ai backup più vecchi.

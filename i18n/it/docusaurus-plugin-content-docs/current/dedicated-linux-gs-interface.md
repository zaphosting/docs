## Introduzione

Se i prodotti game server non fanno al caso tuo o non soddisfano le tue aspettative, oppure se hai bisogno del tuo servizio in modi diversi, usare un VPS o un server dedicato è spesso la scelta migliore. Però spesso ti trovi davanti alla sfida che non tutti i servizi sono automatizzati o preinstallati. Configurare servizi come game server, server Teamspeak 3 e music bot va fatto in autonomia.

Per risolvere questo problema, abbiamo sviluppato una soluzione pratica per i nostri VPS o server dedicati basati su Linux. La cosiddetta interfaccia Gameserver/Teamspeak 3 Server (interfaccia GS/TS3)!



## Caso d’uso

Con l’interfaccia GS/TS3 super user-friendly puoi installare Teamspeak 3 o altri servizi selezionati di game server sul tuo VPS o server dedicato Linux con pochi click. Non serve nessuna conoscenza tecnica per questa installazione. La configurazione di tutti i pacchetti necessari, i file server e l’installazione vera e propria sono completamente automatizzati, risparmiandoti tempo e fatica.

:::warning
La funzione interfaccia GS/TS3 è utilizzabile con i seguenti sistemi operativi:

- Debian: 10, 11, 12
- Ubuntu: 20.04, 22.04

(*) Stiamo già lavorando per rendere disponibili altri sistemi operativi e versioni più recenti a breve.
:::



## Servizi & giochi disponibili

L’interfaccia GS/TS3 supporta l’installazione di game server, server vocali e music bot. La maggior parte dei giochi che offriamo per i nostri prodotti game server sono disponibili anche nell’opzione interfaccia GS/TS3. Questo vale per tutti i giochi che gestiamo attivamente su Linux. Purtroppo tutti i giochi che girano su sistemi Windows non possono essere installati qui. In quei casi è possibile solo un’installazione manuale, a patto che il gioco supporti una versione server Linux.

| Servizi    | Supportato |
| ----------- | ---------------- |
| Gameserver  | ✔️                |
| Voiceserver | ✔️                |
| Musicbot (Sinusbot)  | ✔️                |



## Installazione e configurazione

Per installare l’interfaccia GS/TS3, vai nel pannello di navigazione **Impostazioni** nell’amministrazione del tuo server e clicca sulla voce secondaria **Gameserver / TS3**. 

La prima volta che apri l’interfaccia GS/TS3 verrai reindirizzato a un setup assistant super intuitivo. Per configurare l’interfaccia e collegarla al server devi selezionare un sistema operativo supportato e inserire i dati di login dell’utente root. 

![](https://screensaver01.zap-hosting.com/index.php/s/dLeLDKdmdiZ74CP/download)



Una volta collegati con successo server e sito, l’interfaccia GS/TS3 può essere creata. 



![](https://screensaver01.zap-hosting.com/index.php/s/FK9mP3BgzrPmH7S/preview)



![](https://screensaver01.zap-hosting.com/index.php/s/JL7jyTKbCEx8FBZ/preview)

:::info
**Attenzione**: Non proseguire finché l’interfaccia non è completamente installata. Puoi riconoscere l’installazione riuscita perché la barra di progresso sparisce e per ogni pacchetto richiesto appare “Installed” invece di “Missing”.
:::



### Crea Game server

Se la creazione e installazione dell’interfaccia GS/TS3 è andata a buon fine, puoi iniziare a configurare i servizi. L’installazione di un game server si fa cliccando sul bottone **Installa game server** nella categoria **Game server**.

![](https://screensaver01.zap-hosting.com/index.php/s/QinM7KtFwcAp5pE/preview)



Nel passaggio successivo puoi scegliere l’indirizzo IP del game server desiderato (se il tuo server ha più IP). Puoi anche impostare la porta, il numero di slot e il boost RAM a piacimento.

![](https://screensaver01.zap-hosting.com/index.php/s/cqWwZbXT77okeDa/preview)

Controlla che tutti i dati siano corretti e rispecchino le tue esigenze. Quando hai finito, clicca su "Installa game server ora" per avviare l’installazione. Il game server verrà configurato secondo le tue specifiche e potrai iniziare a giocare in poco tempo. Ora puoi vedere e gestire il tuo nuovo game server nella lista.



![](https://screensaver01.zap-hosting.com/index.php/s/9WkJnxzkaEHmri7/preview)



Sei ora nell’amministrazione del tuo game server. Qui puoi selezionare e installare il gioco che vuoi nel pannello **Impostazioni** sotto **Giochi**.

![](https://screensaver01.zap-hosting.com/index.php/s/6pxEbWttos6HAYt/preview)

Troverai la lista di tutti i giochi installabili sul server. In questo esempio si installa un server con il gioco (pacchetto) Minecraft: Paperspigot. Seleziona il gioco che vuoi e clicca sul bottone verde per partire con l’installazione.

![](https://screensaver01.zap-hosting.com/index.php/s/gazW2itexCJd7cY/preview)

Ti verrà mostrata una panoramica con tutte le condizioni necessarie. Se tutto è a posto, conferma con il bottone **Accetta e Installa**.

![](https://screensaver01.zap-hosting.com/index.php/s/jeQC7dp6zpe3ny4/preview)

La configurazione del gioco può richiedere un po’ di tempo a seconda del titolo. Lo stato di avanzamento è visibile nella barra di stato. Dopo puoi avviare il game server e connetterti in game!



### Crea Voice server

L’installazione di un **Voiceserver (Teamspeak 3 Server)** è simile a quella di un game server. Devi cliccare di nuovo su "Gameserver / TS3". Qui trovi l’opzione **Installa voice server**, cliccaci per avviare l’installazione del server Teamspeak 3.

![](https://screensaver01.zap-hosting.com/index.php/s/mi8p3NTfwBBExsD/preview)

Nel passaggio successivo puoi impostare le opzioni per il tuo Teamspeak 3 server, come IP, porta e numero di slot disponibili. Quando hai inserito tutto, clicca su **Installa voice server ora** per completare l’installazione.

![](https://screensaver01.zap-hosting.com/index.php/s/ajfzxsJfCFdfBac/preview)

Controlla che tutte le info siano corrette e rispecchino le tue esigenze. Quando hai finito, clicca su **Installa Teamspeak 3/5 Server ora** per partire con l’installazione. Ora puoi vedere e cliccare sul Teamspeak 3 server installato nella **Panoramica Voiceserver**.

![](https://screensaver01.zap-hosting.com/index.php/s/YaEYorRG7TJGpmB/preview)

:::info
**Attenzione**: Questa è l’istanza su cui verrà installato il server Teamspeak 3.
:::

Sei ora nell’interfaccia dell’istanza del tuo Teamspeak 3 server installato. Puoi avviare l’istanza cliccando sul bottone **Avvia**.

![](https://screensaver01.zap-hosting.com/index.php/s/SmqHB24ozJimBY9/preview)


Ora clicca su **Virtual Servers** nel menu. Qui trovi tutti i server Teamspeak 3 installati sulla tua istanza, elencati chiaramente. Per aprire l’interfaccia del singolo server, clicca sull’icona “occhio”.

![](https://screensaver01.zap-hosting.com/index.php/s/E3ZqxC9rPjWwC5F/preview)

Nell’interfaccia del server Teamspeak 3 trovi tutte le info e le opzioni per personalizzare e gestire il tuo server. Per esempio, nella scheda "Chiave di autorizzazione" puoi creare un token che ti dà i diritti admin sul server. Così hai il controllo totale su utenti e accessi.

Nel menu "Impostazioni" hai altre opzioni per personalizzare il server. Puoi cambiare il nome, mettere una password per limitare l’accesso o scrivere un messaggio di benvenuto che gli utenti vedono entrando.



:::caution Info sulle licenze Teamspeak
Di default il server Teamspeak 3 non ha nessuna licenza installata. Questo significa che puoi usare un server Teamspeak 3 con massimo 32 slot. Se vuoi più slot o funzioni extra, puoi acquistare una licenza direttamente da Teamspeak.
:::

## La Console
Per eseguire comandi sul tuo server, devi essere admin. Se non sai come diventarlo, dai un’occhiata alla nostra guida.
Tutti i comandi vanno eseguiti tramite la console degli sviluppatori. Puoi aprirla premendo `F1`

La console si presenta così:

![](https://screensaver01.zap-hosting.com/index.php/s/Lyi4qrT5MjzJLkL/preview)

## I Comandi
I comandi più importanti:

| Comando    | Descrizione                                                 |
| ----------------- | ------------------------------------------------------------ |
| giveself [item id] [quality level 1-6] | Ti aggiunge un oggetto, con il livello di qualità scelto |
| debugmenu on/off | Attiva il Debug Menu, si apre premendo G |
| giveselfxp [amount] | Aggiunge XP al tuo personaggio |
| creativemenu on/off | Attiva il Creative Menu, si apre premendo U |
| spawnsupplycrate | Fa spawnare una supply crate nella tua posizione |
| admin [add] [remove] [list] | Comando per aggiungere/rimuovere admin |
| kick [playername] [motivo] | Espelle un giocatore dal server |
| tele [Player A] [Player B] | Teletrasporta Player A da Player B |
| weather | Cambia le condizioni meteo |
| help | Mostra tutti i comandi disponibili |

[altri comandi](https://commands.gg/7dtd)

## Introduzione

Svuotare la cache del server su un server FiveM assicura che i file temporanei vengano eliminati. Questo forza il server a riscaricare o ricreare tutti i file necessari al prossimo avvio. Aiuta a rimuovere dati obsoleti o corrotti, risolvendo problemi di prestazioni, errori negli script, crash o problemi nel caricamento del mondo di gioco.

## Metodo semplice

Cancellare la cache sul tuo server FiveM Ã¨ davvero facile. Prima di tutto, vai al pannello web del tuo game server. Trova e apri la sezione **Impostazioni** nel pannello web del tuo server.



![img](https://screensaver01.zap-hosting.com/index.php/s/nizHMSk7oXCsJS4/download)



Poi scorri fino in fondo alla pagina per trovare diversi pulsanti sotto la sottosezione **Azioni**. Qui clicca sul pulsante **Cancella file cache** per svuotare la cache del tuo server di gioco.

![img](https://screensaver01.zap-hosting.com/index.php/s/A2RiTo8gJiTibGR/download)

:::warning

## Metodo alternativo

Prima di tutto, devi connetterti al tuo server via FTP. Se non hai mai usato FTP, ti consigliamo di dare unâ€™occhiata alla guida su [Accesso FTP](gameserver-ftpaccess.md). Nel passo successivo, devi trovare la cartella cache ed eliminarla. Segui questo percorso `/gXXXXXX/fivem/server-data/cache` per accedere alla cartella cache.

![img](https://screensaver01.zap-hosting.com/index.php/s/dfpssTy8KL7B3cK/download)

Ora devi semplicemente svuotare la cache. Per farlo clicca sulla cartella **Cache** e poi su **Files**. Ora seleziona tutte le cartelle presenti nella cartella **Cache** ed eliminale.

![img](https://screensaver01.zap-hosting.com/index.php/s/MnHpWiDmBgHa63B/download)

Adesso riavvia il tuo server FiveM per applicare le modifiche. Al riavvio, il server ricreerÃ  la cache.

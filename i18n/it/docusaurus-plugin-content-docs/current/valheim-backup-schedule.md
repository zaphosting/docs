Purtroppo Valheim non offre un modo diretto per fare backup automatici, quindi dobbiamo usare un piccolo trucco.

Vai nel tuo dashboard su "Pianificatore Riavvii".

![](https://screensaver01.zap-hosting.com/index.php/s/FspW5eG7XJNqE4k/preview)

Ora clicca sul "+" che apre una nuova finestra, dove possiamo definire gli orari di riavvio desiderati.

![](https://screensaver01.zap-hosting.com/index.php/s/me5tSbwc8YWT7me/preview)

Per prima cosa scegli se vuoi "Giornaliero", cioè che ogni giorno venga creato un backup all’orario che scegliamo.
Nel passo successivo definiamo l’orario di riavvio.

:::info
Avvia server offline - Significa che il server verrà avviato se è offline, quando arriva l’orario di reinstallazione.
:::

Ora clicchiamo su "Salva".

:::info
Puoi impostare un riavvio programmato ogni 6 ore, così ogni 6 ore verrà creato un backup automatico.
:::

![](https://screensaver01.zap-hosting.com/index.php/s/rFPWnSH7EkHxoN9/preview)

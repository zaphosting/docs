## Introduzione
Minecraft ti permette di aggiungere un'icona personalizzata per il server, che viene mostrata nel multiplayer davanti al nome del server, a patto che il server sia stato salvato.

## Aggiungere l'icona

Per aggiungere l'icona ti servono i diritti FTP e l'icona stessa.

>Importante: L'icona deve essere di 64x64 pixel e chiamarsi server-icon.png.

Per prima cosa, stabilisci una connessione FTP al tuo game server, qui puoi vedere come fare: [Accesso via FTP](gameserver-ftpaccess.md)

Appena sei connesso, vedrai i dati del server tipo:

![](https://screensaver01.zap-hosting.com/index.php/s/jKDxAfWzgiepiTE/preview)

Ora devi caricare l'icona nella directory principale, non deve finire in nessuna sottocartella:

![](https://screensaver01.zap-hosting.com/index.php/s/dNWE4W5KQ8XgsQs/preview)

Una volta fatto, riavvia o avvia il server usando i pulsanti di controllo nell’interfaccia web in alto a destra sul server interessato:

![](https://screensaver01.zap-hosting.com/index.php/s/D4RGqboisxJoBa9/preview)

## Verifica dell'icona

Se nome e dimensioni sono corretti, l’icona apparirà nel multiplayer davanti al nome del server:

![](https://screensaver01.zap-hosting.com/index.php/s/qzoySiJAPReZDfc/preview)

Puoi cambiare o cancellare l’icona in qualsiasi momento con lo stesso metodo.

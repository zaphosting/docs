## Introduktion

Elementor är en kraftfull och användarvänlig sidbyggare för WordPress som låter dig enkelt skapa professionella webbplatser utan någon kodningskunskap.

:::info
Elementor finns i både gratis- och betalversioner. Med gratisversionen av Elementor (Elementor Core) får du allt du behöver för att komma igång med din webbplats. Elementor Pro bygger vidare på gratisversionen och erbjuder fler funktioner och Pro-widgets. Du hittar mer info om de exakta skillnaderna i den officiella dokumentationen för **[Elementor](https://elementor.com/help/elementor-pro-vs-free/)**.
:::

## Installation

För att använda och installera Elementor behöver du redan ha WordPress installerat och igång på ditt webbhotell. Om det inte är gjort än, måste du börja med det. Om du är osäker på hur du gör, rekommenderar vi att du kollar in vår **[Installera WordPress](webspace-wordpress.md)** guide.

När det är klart och WordPress körs kan du börja installera Elementor. Elementor installeras som ett plugin i WordPress. Hur du installerar plugins i WordPress kan du läsa om i vår [**WordPress-plugins**](webspace-wordpress-plugins.md) guide.

Gå till Plugins i WordPress, klicka på knappen **Lägg till nytt plugin** högst upp på sidan. Sök sedan efter Elementor och klicka på installera vid valet **Elementor Website Builder - More than just a page builder**.

![img](https://screensaver01.zap-hosting.com/index.php/s/xcj9stZtAmY9cgJ/preview)



## Konfiguration

När installationen är klar kan du fortsätta med konfigurationen. Nästa steg är att du bör se Elementors välkomstfönster. Här guidas du genom följande steg: Skapa ett Elementor-konto, installera temat “Hello”, välja önskade funktioner och slutligen bekräftelsen “Good to go”. Följ bara instruktionerna för att ställa in Elementor helt.

![img](https://screensaver01.zap-hosting.com/index.php/s/6QN5trndZgfSano/download)

Att skapa ett Elementor-konto är inte absolut nödvändigt, om du inte planerar att använda Elementor Pro direkt från start. Därefter kan du installera temat och lägga till de extra funktioner du behöver.

![img](https://screensaver01.zap-hosting.com/index.php/s/YgXwPiEnBZTQsC4/preview)



## Kom igång

Nu kan du börja designa din webbplats med hjälp av Elementor. Börja med att skaffa dig en överblick över hur Elementor fungerar, vilka widgets som finns, hur du redigerar design och layout samt hur du skapar individuella sidor och mallar.

### Tillgängliga Widgets

Både gratis- och betalversionerna av Elementor erbjuder en mängd **widgets** som du kan använda för att designa och anpassa din webbplats.


#### Grundläggande Widgets
Gratisversionen av Elementor ger dig ett urval av widgets som du kan använda för att designa och anpassa din webbplats utan extra kostnad:

- Inner Section, Heading, Image, Text Editor, Video, Button, Star Rating, Divider, Google Maps, Icon, Image Box, Icon Box, Basic Gallery, Image Carousel, Icon List, Counter, Spacer, Testimonial, Tabs, Accordion, Toggle, Social Icons, Progress Bar, Sound Cloud, Shortcode, HTML, Menu Anchor, Alert, Sidebar, Text Path, Container och Link in Bio.

#### Pro Widgets

Utöka designen av din webbplats ännu mer med extra Pro-widgets. Elementor Pro-versionen inkluderar följande widgets:

- Posts, Share Buttons, Portfolio, Slides, Form, Login, Nav Menu, Animated Headline, Price Table, Price List, Gallery, Flip Box, Call to Action, Media Carousel, Testimonial Carousel, Nested Carousel, Loop Carousel, Table of Content, Countdown, Facebook Page, Blockquote, Template, Reviews, Facebook Button, Facebook Embed, Facebook Comments, PayPal Button, Lottie Widget, Code Highlight, Video Playlist, Hotspot, Progress Tracker, Stripe Button, Mega Menu, Off-Canvas






## Design och Layout

### Skapa Sida

Skapa en ny sida och designa den som du vill med **Elementor**. Öppna Elementor via sidomenyn och välj **"Skapa sida"**. Editorn ger dig massor av **widgets och inställningar** för att flexibelt anpassa design och innehåll.

![img](https://screensaver01.zap-hosting.com/index.php/s/mdMbnXNkngXWJHt/download)

### Mallar

Skapa mallar för **containers, sidor eller sektioner** som du kan återanvända flera gånger. Den stora fördelen är att ändringar bara behöver göras på ett ställe. Alla områden där mallen används uppdateras automatiskt. Det sparar tid och ger en enhetlig design för din webbplats.

![img](https://screensaver01.zap-hosting.com/index.php/s/mdMbnXNkngXWJHt/download)

## Felsök ditt problem

Startar inte din server? Eller har du problem som du inte kan lösa själv? Inga problem! Vårt supportteam finns här för dig.

Men innan du kontaktar vår support rekommenderar vi starkt att du testar några felsökningssteg som kan lösa eller begränsa problemet.

:::note
Tänk på att vi inte erbjuder direkt kontorelaterad support via våra sociala kanaler, endast via vårt ticketsystem. Detta eftersom vi värnar om din integritet och därför endast ger officiell support genom vårt eget ticketsystem.
:::

### Kolla din servers status
Du kan enkelt se aktuell status för alla våra tjänster, inklusive din server, genom att gå till vår [Statussida](https://status.zap-hosting.com/) och välja rätt tjänst.

:::tip
På vår statussida kan du till och med prenumerera på e-postnotiser för vilken tjänst som helst i vårt nätverk! Det gör du genom att klicka på pilknappen uppe till höger, välja omfattning och skriva in den e-postadress du vill få notiser till.

![](https://screensaver01.zap-hosting.com/index.php/s/fjFET6mF4XCrBSX/preview)
:::

Om produkten du har problem med just nu är påverkad, rekommenderar vi att du väntar tills problemet är löst. Våra tekniker får automatiska larm vid avbrott och jobbar på att fixa det så snabbt som möjligt.

### Kolla dina loggar
Ett annat felsökningssteg är att kolla loggarna för din server. Oavsett vilket spel eller tjänst du kör, bör du titta i konsolen eller i logg-mappen för att se om några felmeddelanden finns. Detta kan skilja sig mycket beroende på spel och tjänst.

:::tip
Om du skapar en ticket kan det vara väldigt hjälpsamt att bifoga loggar eller felmeddelanden i ditt ärende.
:::

Du kan också se händelselogg för din tjänst genom att gå till **Event log**-sektionen i produktens webbgränssnitt. Där ser du händelser som omstarter, påslagningar och mer som hänt nyligen.

## Så här kommer du åt ticketsystemet

Du når vårt ticketsystem via ticket-ikonen i toppmenyn eller direkt via [ticket-sidan](https://zap-hosting.com/en/customer/support/). Om du inte är inloggad kommer du bli ombedd att logga in först.

På sidan kan du se dina aktiva och tidigare tickets samt skapa en ny ticket.

![](https://screensaver01.zap-hosting.com/index.php/s/7E5xtP3oZpGyr79/preview)

## Skapa en ny ticket

Om du fortfarande behöver support, kontakta vår officiella support genom att gå till [ticket-sidan](https://zap-hosting.com/en/customer/support/) i ditt webbgränssnitt efter att du loggat in.

Börja med att klicka på **Create Ticket**-knappen. Här kan du välja mellan en vanlig ticket eller en **Speed-UP** ticket.

![](https://screensaver01.zap-hosting.com/index.php/s/K2xxDEYkJFPN5C2/preview)

Fyll sedan i formuläret med alla nödvändiga detaljer om ditt problem och vilka felsökningssteg du redan testat.

Här är en lista på användbar info att ta med:
- Välj det ämne som passar bäst för din ticket, inklusive vilken server det gäller (om relevant).
- En ämnesrubrik som sammanfattar vad ärendet handlar om.
- En detaljerad beskrivning av problemet. Fråga dig själv: När började det? Hur ofta händer det? Har du testat några felsökningssteg innan du skickade ticketen?
- Extra info, som skärmdumpar på fel, loggar eller annat som kan vara hjälpsamt.

![](https://screensaver01.zap-hosting.com/index.php/s/mL5Zsi7EmqQiLnC/preview)

När du är klar med formuläret klickar du på **Send Support Ticket** för att skicka ärendet till vårt supportteam!

Sen är det bara att vänta på att någon av våra grymma supportmedarbetare svarar på din ticket. Vi meddelar dig via e-post till den adress som är kopplad till ditt konto när vi svarat.

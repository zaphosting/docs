## Användningsområden

Tack vare det schyssta priset, prestandan, de generösa åtkomsträttigheterna och alla möjligheter som följer med, kan ett Webspace användas för flera olika ändamål, bland annat:

- Webbplatser
- Databaser
- Mailtrafik

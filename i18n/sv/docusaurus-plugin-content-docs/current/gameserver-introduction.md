## Konfigurationer

Vi erbjuder olika konfigurationsmöjligheter för våra gameserver-produkter, som passar alla behov. Våra konfigurationer består generellt av följande valmöjligheter:

| Konfiguration            | Valmöjlighet |
| --------------------------------- | ---- |
| Slots                             | Beroende på vad spelet stödjer |
| Memory Boosts (extra minne)       | Upp till 32 GB extra RAM utöver standardallokerat RAM |
| Extra Diskutrymme                 | Upp till 200 GB extra diskutrymme utöver standard 50 GB |
| CPU & Host-server                | Standard (SSD, 2.0-3.4 GHz, DDR3/4-minne) eller Premium (M.2 SSD, Gaming-CPU 3.4-4.4 GHz, DDR4-minne) |
| Egen IPv4-adress                 | Egen IPv4-adress / ingen egen IPv4-adress |

## Skapa skärmdumpar

I vissa fall behöver du skicka skärmdumpar till supporten.  
För att det ska funka måste skärmdumpen/dumparna delas via en länk.  
Denna länk kan vara till en egen molntjänst eller till en "web uploader", även kallad webbaserade skärmdumpsverktyg.  

Om din egen molntjänst eller webverktyget inte har en funktion för att ta skärmdumpar direkt, kan du oftast skapa dem på datorn med enkla verktyg.  
På Windows räcker det med att använda verktyget "Snipping Tool". Med det kan du spara skärmdumpen lokalt, som du sedan kan ladda upp.  
På Linux kan du använda det oftast redan installerade verktyget "Screenshots" eller andra verktyg som "FlameShot" för att skapa lokala skärmdumpar som sedan kan laddas upp.

Nedan hittar du en kort beskrivning av de 2 mest populära skärmdumpsverktygen som finns fritt tillgängliga på marknaden:

:::info
Det rekommenderas inte att dela privata eller känsliga uppgifter offentligt. Om data syns på skärmdumpen bör det göras oigenkänneligt eller ersättas med exempeldata.
:::


### Imgur

Imgur erbjuder en uploader för egna filer som bilder/videor.  
När uppladdningen är klar skapas en länk.  
Den kan sedan delas med andra.

Viktigt: Eftersom det inte finns någon skärmdumpsfunktion där måste skärmdumpen tas innan så att den kan laddas upp (se ovan under "Skapa skärmdumpar").

### Lightshot

Lightshot erbjuder ingen direkt uppladdning för bilder/skärmdumpar. Uppladdningen kan bara göras via deras egen klient.  
Klienten kan laddas ner direkt från hemsidan och erbjuder även några verktyg för att markera/peka ut något på skärmdumpen eller göra delar oigenkännliga.

Ladda ner klienten [Download](https://app.prntscr.com/en/)  
Viktigt: Det finns ingen version för Linux.

![](https://screensaver01.zap-hosting.com/index.php/s/iGEccwZZDXqLGKp/preview)

Efter nedladdning kan klienten installeras.  
Därefter kan du som standard ta en skärmdump eller välja ett område med "Print"-knappen.

När området är valt kan skärmdumpen fortfarande redigeras.

När redigeringen är klar kan skärmdumpen, om du vill, laddas upp genom att klicka på "Cloud".

Efter uppladdning kan länken kopieras och delas med andra.

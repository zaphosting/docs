## Introduktion

Att skapa och dela skärmdumpar hjälper till att förbättra supportkommunikationen genom att ge tydlig visuell information. Den här guiden förklarar hur du tar skärmdumpar på din dator och laddar upp dem direkt till supportärenden.

## Skapa Skärmdumpar

Det bästa sättet att dela skärmdumpar säkert är att skapa dem lokalt på din enhet och ladda upp dem direkt till ditt supportärende via drag & drop. Den här metoden säkerställer att skärmdumparna förblir privata och delas säkert utan att använda externa länkar.

### Windows: Snipping Tool

Windows har det inbyggda *Snipping Tool* som låter dig fånga ett specifikt område, ett fönster eller hela skärmen och spara bilden lokalt.

1. Öppna **Snipping Tool** genom att söka efter det i Start-menyn.
2. Klicka på **Ny** för att börja en skärmdump.
3. Välj det område du vill fånga.
4. Spara skärmdumpen på en lämplig plats på din dator.
5. Ladda upp den sparade skärmdumpsfilen direkt till ditt supportärende genom att dra och släppa den i ärendegränssnittet.

### Linux: Screenshot Tool eller FlameShot

De flesta Linux-distributioner har ett standardverktyg för *Screenshot*. Alternativt erbjuder tredjepartsverktyg som *FlameShot* avancerade funktioner.

1. Öppna Screenshot-verktyget eller FlameShot.
2. Välj området eller fönstret du vill fånga.
3. Spara skärmdumpen lokalt.
4. Ladda upp den sparade skärmdumpsfilen direkt till ditt supportärende med drag & drop.

## Ladda upp skärmdumpar till support

Efter att du sparat skärmdumpen lokalt, öppna ditt supportärende i ZAP-Hosting-panelen eller supportportalen.

- Dra och släpp skärmdumpsfilen i meddelandeområdet för ärendet.
- Vänta tills uppladdningen är klar.
- Skicka in ärendet med den bifogade skärmdumpen.

Den här metoden undviker att dela externa länkar och håller dina data säkra.

:::info Säker delning av skärmdumpar
Undvik att dela skärmdumpar offentligt eller via tredjepartstjänster för uppladdning. Om din skärmdump innehåller känslig eller privat information, se till att sudda ut eller maskera den innan du laddar upp.
:::

## Introduktion

Webbplatsutrymmespaketet ger dig möjligheten att skapa och hantera både webbplatser och e-postadresser via det externa Plesk-gränssnittet. Webbplatsutrymmet fungerar som webb-lagring. Filer och mappar kan laddas upp via en FTP-anslutning.

Varje webbplatsutrymme som bokas via Zap-Hosting tilldelas en automatiskt genererad subdomän. Denna subdomän består av kundnumret, värdservern och domänen "zap-webspace.com". Instruktionerna i "Länka domän" visar hur din egen domän kan kopplas till webbplatsutrymmet.

## Kom igång

När beställningen är klar visas webbplatsutrymmet i Zap-Hosting Dashboard.

### Öppna Plesk

Inloggningsuppgifterna kan ses på den nu öppnade sidan. Lösenordet sparas inte i klartext. För att visa det måste du klicka på "ögon"-symbolen.

![Bildschirmfoto vom 2022-05-13 05-16-48](https://screensaver01.zap-hosting.com/index.php/s/B3stJeKMAsYRRer/preview)

:::info
Om du inte gillar det automatiskt genererade lösenordet kan du ändra det direkt på samma sida med knappen "Byt lösenord".
:::

Så fort användarnamn och lösenord är visade kan Plesk-gränssnittet öppnas genom att klicka på knappen "**Öppna administrationsknappen**". Efter att ha skickats vidare till Plesk-gränssnittet måste de tidigare visade inloggningsuppgifterna anges.

![Bildschirmfoto vom 2022-05-13 05-17-21](https://screensaver01.zap-hosting.com/index.php/s/tZLEJZTzo7kqn4J/preview)

:::info
Gränssnittets språk kan ändras från tyska till engelska, standardvärdet är tyska.
:::

### Länka domän

Din egen domän kan läggas till i Plesk-gränssnittet under menyalternativet "**Lägg till domän**".

![Bildschirmfoto vom 2022-05-13 05-18-15](https://screensaver01.zap-hosting.com/index.php/s/LHJkPqAsC7zGysW/preview)

Under "**Lägg till domän**" kan du skriva in din domänadress direkt och välja dina önskade inställningar för filernas plats, SSL med mera.

![Bildschirmfoto vom 2022-05-13 05-19-06](https://screensaver01.zap-hosting.com/index.php/s/JXYg4ycDpwqYxCN/preview)

Ändringarna bekräftas genom att klicka på knappen "**OK**".

:::info
Bekräftelsen kan ta lite tid. Om du får felmeddelanden under ändringen kan en möjlig orsak vara att din domän redan är skapad som en alias i domänen.
:::

Om ändringen gick igenom utan problem bekräftas detta i Plesk-gränssnittet. För att kunna använda denna domän för webbplatsutrymmet måste DNS-inställningarna konfigureras i domänhanteringen. Hur detta fungerar med domäner beställda via oss kan du se här.

![Bildschirmfoto vom 2022-05-13 05-19-40](https://screensaver01.zap-hosting.com/index.php/s/TecSRSimmCrRmYj/preview)

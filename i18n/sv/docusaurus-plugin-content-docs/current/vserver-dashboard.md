## Introduktion

Du är intresserad av en **VPS** eller äger redan en, men saknar fortfarande en komplett översikt över administrationen och hur du hittar alla nödvändiga inställningar? Speciellt i början kan det kännas överväldigande. Men oroa dig inte, vi ger dig en detaljerad introduktion till administrationen av din tjänst nedan, så att du får bästa möjliga överblick.

Efter att ha gått igenom den här guiden kommer du enkelt kunna hantera din VPS effektivt och snabbt hitta alla nödvändiga inställningar och funktioner. Här får du en strukturerad översikt som hjälper dig steg för steg att bekanta dig med VPS-administrationen och utöka din kunskap på ett målmedvetet sätt. På så sätt kan du säkerställa att din **VPS** är optimalt konfigurerad efter dina behov och att du får ut maximalt av de tillgängliga möjligheterna.



## Produktadministration

I avsnittet **Produktadministration** hittar du alla nödvändiga sidor för att hantera din VPS. Alternativen för att administrera din VPS listas i detalj nedan. 



### Dashboard

Dashboarden ger en central översikt över all viktig information relaterad till VPS:en. Den visar aktuell status, serveradress, hostname, installerat operativsystem, systembelastning inklusive CPU- och RAM-användning samt nätverkstrafik. Denna översikt stödjer effektiv övervakning och hantering av din virtuella servermiljö.

![img](https://screensaver01.zap-hosting.com/index.php/s/JfJKCwtG9opJZ2H/preview)



### Access & Security

Ställ in säker åtkomst till din server genom att skapa egna inloggningsuppgifter. Du kan konfigurera ett starkt lösenord för lösenordsbaserad inloggning eller generera en SSH-nyckel för att ansluta säkert via SSH-nyckelautentisering. Detta låter dig välja den bästa metoden för att hantera åtkomst och håller din server skyddad.

![img](https://screensaver01.zap-hosting.com/index.php/s/KR2KBzfm7kre6mN/preview)



### Event log

Håll koll på alla adminuppgifter och systemhändelser på ett och samma ställe. Eventloggen samlar alla relevanta aktiviteter, så du alltid vet vem som gjort ändringar, när de skedde och vad som påverkades.

![img](https://screensaver01.zap-hosting.com/index.php/s/w9AHMNnPtBtZY2L/preview)



## Inställningar

I avsnittet **Inställningar** hittar du alla nödvändiga sidor för att konfigurera din VPS. De tillgängliga alternativen för att konfigurera din VPS listas i detalj nedan.

### Inställningar

Inställningarna erbjuder möjligheter att göra viktiga konfigurationsändringar på servern. Det inkluderar att definiera en DNS-server, konfigurera ballooning eller installera om servern. Dessa funktioner möjliggör flexibel och målinriktad hantering av servermiljön.

![img](https://screensaver01.zap-hosting.com/index.php/s/esY87qKYozR4mj9/preview)



### ISOs

Vill du testa ett nytt operativsystem eller felsöka din server när den blir otillgänglig? Ett brett utbud av ISO-filer finns tillgängligt för dig. Med bara några klick kan du montera dem som en virtuell disk och använda dem direkt för installation eller diagnostik.

![img](https://screensaver01.zap-hosting.com/index.php/s/Ti4CNm3qp8ka6Tw/preview)

### Gameserver / TS3

Med den här funktionen kan du enkelt sätta upp och hantera gameserver- och voiceserver-tjänster med bara några klick. Webbgränssnittet sköter hela installationsprocessen åt dig, så du behöver ingen djup teknisk kunskap.

Du kan skapa nya gameserver- eller voiceserver när du behöver dem, justera inställningar som slots eller resurser och styra dem bekvämt från din ZAP-webbdashboard. Det gör hanteringen av flera servrar snabb, flexibel och enkel. Allt på ett ställe.

![img](https://screensaver01.zap-hosting.com/index.php/s/dQ6d8LctpNcsd6H/preview)



### Redigera rDNS

Justera den omvända DNS-posten (PTR-record) för din servers IP-adress så att den pekar på rätt hostname. En korrekt konfigurerad rDNS hjälper till att säkerställa att din servers VPS-namn löses korrekt och är särskilt viktigt för att driva en mailserver. Utan giltig rDNS kan utgående mejl markeras som spam eller misslyckas med leverans.

![img](https://screensaver01.zap-hosting.com/index.php/s/jmG7RJHoA22dtpx/preview)



### IP-adresser

Se alla IP-adresser som är tilldelade din server på en och samma vy. Denna översikt visar viktiga konfigurationsdetaljer som IP-adress, subnet mask och gateway. Du kan även kontrollera om en IP är nåbar eller inte, vilket hjälper dig att verifiera att dina nätverksinställningar är korrekt satta i operativsystemet.

![img](https://screensaver01.zap-hosting.com/index.php/s/8gWkFMJFpBPNwiW/preview)



## Verktyg

Avsnittet **Verktyg** innehåller alla viktiga verktyg för att hantera VPS:en. De tillgängliga alternativen för att konfigurera servern beskrivs i detalj nedan och möjliggör precis anpassning efter specifika behov.

### VNC-konsol

Med VNC-konsolen kan du interagera med din server som om du satt direkt vid maskinen. Denna fjärråtkomst låter dig styra skärm, tangentbord och mus, så att du kan utföra underhåll även om det inte finns någon aktiv nätverksanslutning. Perfekt för att reparera nätverksinställningar eller lösa konfigurationsproblem.

![img](https://screensaver01.zap-hosting.com/index.php/s/pdiHAkFJGYTmdjp/download)

### Nollställ trafik

Håll koll på din månatliga trafikförbrukning. Denna sektion visar hur mycket av din inkluderade trafik du redan använt. Om du når din gräns innan månaden är slut har du även möjlighet att nollställa din trafik i förväg mot en extra kostnad.

![img](https://screensaver01.zap-hosting.com/index.php/s/AjQ7xFjGBHLBJY7/preview)



## Resursmonitor

Övervakningsdashboard för din virtuella server med live prestandamått: CPU-användning, minnesförbrukning, nätverkstrafik och disk I/O-statistik.

![img](https://screensaver01.zap-hosting.com/index.php/s/aQWRa6yEbF7BbEC/preview)





### DDoS Manager

Se tidigare DDoS-attacker för att bättre bedöma potentiella hot och mönster. Aktivera notiser för pågående DDoS-attacker på din server för att snabbt kunna reagera när en incident inträffar.

Beroende på serverplats och tillgängligt DDoS-skydd där kan du även övervaka aktiva attacker i realtid. Detta inkluderar detaljerad trafikinsikt, som visar både legitim trafik och mängden trafik som blockeras av skyddssystemen. Denna transparens hjälper dig att förstå hur skyddet fungerar och gör det möjligt att tidigt identifiera potentiella risker.

![img](https://screensaver01.zap-hosting.com/index.php/s/ScCCCY52CMLgfyE/preview)

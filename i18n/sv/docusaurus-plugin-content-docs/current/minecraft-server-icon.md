## Lägg till ikonen

För att lägga till ikonen behöver du FTP-rättigheter och själva ikonen.

> Viktigt: Ikonen måste vara 64x64 pixlar stor och heta server-icon.png.

Först kopplar du upp dig via FTP till din spelserver, exakt hur du gör det kan du se här: [Åtkomst via FTP](gameserver-ftpaccess.md)

När du är uppkopplad ser du serverns filer, till exempel:

![](https://screensaver01.zap-hosting.com/index.php/s/jKDxAfWzgiepiTE/preview)

Ikonen ska nu laddas upp i huvudmappen, den får inte ligga i någon undermapp:

![](https://screensaver01.zap-hosting.com/index.php/s/dNWE4W5KQ8XgsQs/preview)

När det är klart måste servern startas om eller startas på nytt via kontrollknapparna i webbgränssnittet uppe till höger på din server:

![](https://screensaver01.zap-hosting.com/index.php/s/D4RGqboisxJoBa9/preview)

## Kontrollera ikonen

Om namn och storlek stämmer visas ikonen nu i multiplayer framför servern:

![](https://screensaver01.zap-hosting.com/index.php/s/qzoySiJAPReZDfc/preview)

Ikonen kan när som helst bytas ut eller tas bort på samma sätt.

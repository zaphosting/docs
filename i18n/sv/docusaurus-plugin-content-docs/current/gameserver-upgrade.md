## Inledning

Tack vare vårt molnsystem kan du alltid ändra din ZAP-Hosting gameserver efter dina önskemål och behov.  
Om du till exempel behöver fler slots eller mer RAM är det inga problem och kan justeras när som helst.  
Tänk på att om du har använt en voucher går det inte att nedgradera under voucher-perioden.

## Så funkar det

När du ändrar ditt paket räknas den återstående perioden om, så om du vill uppgradera din server måste du betala mellanskillnaden för den återstående perioden.  
Ditt produktpris ändras också automatiskt vid nästa förnyelse, baserat på dina nya villkor.  
Observera att tyvärr kan inte kuponger användas vid uppgraderingar, de gäller endast för nya beställningar.

## Uppgraderingar

Du hittar alltid Upp-/Nedgraderingshanteraren i din produkts dashboard, oavsett vilken produkt det är.  
Uppgraderingar är för närvarande möjliga för alla produkter utom Dedikerade Servrar.

Upp-/nedgraderingshanteraren visas i vänstermenyn i din produktdashboard:

![](https://screensaver01.zap-hosting.com/index.php/s/f9z7xyEpB55ZZ52/preview)

### Uppgraderingsalternativ

När du är inne i vår hanterare hittar du de parametrar för din produkt som går att anpassa.

För gameservers går det att flytta dem mellan olika platser.

![](https://screensaver01.zap-hosting.com/index.php/s/tW3sbKxSszQpb8a/preview)

:::info
**Viktigt** Denna process kan orsaka dataförlust, se till att alltid ha en backup av dina data!
:::

Om du vill flytta servern klickar du bara på den plats du vill välja.

Nedan hittar du fler parametrar, i vårt exempel med en FiveM-server är det slot-antal, RAM-boost, systemtyp och egen IP-adress:

![](https://screensaver01.zap-hosting.com/index.php/s/eb5FsMSdwJszjWK/preview)

*** 

* Gameserver Slot: Om du behöver mer kapacitet på din gameserver, dra bara reglaget åt höger för att få plats för fler samtidiga spelare.

* RAM Boost: Bra för mycket, som resurser, mods, add-ons eller liknande. Ökar mängden tillgängligt RAM för din produkt.

* CPU & Hostserver: Om du behöver mer kraft kan du byta till ett premiumsystem som ger dig motsvarande prestanda-boost.

* IP-adress: Om du behöver en fast IP som inte ändras så länge produkten finns, behöver du detta alternativ.

:::info
⚠ Den egna IP-adressen ska inte förväxlas med en domän, en egen IP tilldelas också slumpmässigt och kan inte ändras.
:::

### Betalning

Att betala för uppgraderingen är superenkelt, scrolla bara ner till betalningsmetoderna:

![](https://screensaver01.zap-hosting.com/index.php/s/PFsBtsMr7iyRadf/preview)

* 1: Ditt nya månadspaketpris, i detta exempel kostar paketet 16,84€ per månad, detta inkluderar redan de valda uppgraderingarna.

* 2: Tillägget är summan som måste betalas för den återstående perioden av din produkt, detta betalas direkt.

![](https://screensaver01.zap-hosting.com/index.php/s/XtyJ6X4jjfz3AEY/preview)

Välj din betalningsmetod och slutför uppgraderingen genom att klicka på "Uppgradera".

Det tar en liten stund innan uppgraderingen är genomförd, oftast är den klar inom några sekunder.

:::info
En omstart av din produkt krävs vanligtvis för att de nya inställningarna ska börja gälla.
:::

## Förberedelser - Nedladdningen

För att du ska kunna ansluta till din server behöver vi en fil som möjliggör anslutningen.  
Öppna först [Impostor](https://impostor.github.io/Impostor/) där vi ska fylla i vår info.

För att skapa filen behöver vi info som IP, port och ett namn.  
![](https://screensaver01.zap-hosting.com/index.php/s/bQ7KzTAJcbAsLJd/preview)

Fyll i datan från ditt ZAP-Hosting-konto här, det ser ut så här till exempel:

![](https://screensaver01.zap-hosting.com/index.php/s/7scfddD5WPFr25b/preview)

Där 31.214.245.215 är din IP och 2000 är porten.

Servernamnet kan du välja helt fritt, när all data är ifylld klickar du på "Download Server File".  
En fil med namnet "regioninfo.json" laddas nu ner.

Tryck sedan på `Win` + `R` och skriv in

```
%APPDATA%\..\LocalLow\Innersloth\Among Us
```

och tryck Enter, en mapp öppnas. Flytta in regioninfo.json där och skriv över den befintliga filen.

![](https://screensaver01.zap-hosting.com/index.php/s/KTrfij6ww64DPCq/preview)


## Skapa server "ingame"

Starta Among Us på din dator.  
Klicka på "online" i huvudmenyn och sedan på "Create Game".

![](https://screensaver01.zap-hosting.com/index.php/s/SfMwr82R9BbDK2P/preview)

Här kan du nu göra dina önskade inställningar för servern:

![](https://screensaver01.zap-hosting.com/index.php/s/imzCR9WZenyrTSX/preview)

När du är klar med inställningarna klickar du på "confirm".  
Nu är du ansluten till din server!

I lobbyn kan du välja om servern ska vara publik eller privat.  
För att ändra detta klickar du på ikonen "Private" eller "Public":

Om servern är inställd på "Public" kan andra spelare ansluta på samma sätt.  
De behöver inte klicka på "Create Game" i spelets huvudmeny utan på "Find Game".  
Här är det viktigt att chatten är inställd på "others" och att filtren matchar serverns inställningar exakt.  
Annars visas inte servern.

Om det är en "Private" server måste man klicka på "Enter Code" i huvudmenyn.  
Där skriver man in koden som serverägaren ser i lobbyn.

### Så här ansluter du till Core Keeper-spelservern:

Öppna din ZAP-Hosting spelserverpanel och se till att din server är igång:

![](https://screensaver01.zap-hosting.com/index.php/s/2NbQaA6E2CJ4ZE8/preview)

Starta spelet och välj menyalternativet "Multiplayer": 

![385-351-max](https://screensaver01.zap-hosting.com/index.php/s/t36H8NxmLa4gx5S/preview)

Välj sedan menyalternativet "Direkt anslutning": 

![](https://screensaver01.zap-hosting.com/index.php/s/3MdnG7Xb2222EPQ/preview)

Där anger du IP-adressen och porten som visas i din spelserverpanel:

![](https://screensaver01.zap-hosting.com/index.php/s/R9nRFWFrS9Fbwox/preview)

Klicka på "Anslut" så kopplar klienten upp sig mot din server.

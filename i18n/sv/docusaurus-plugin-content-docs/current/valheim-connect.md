För att ansluta till din Valheim-server öppnar vi först vårt webbgränssnitt och ser till att vi har satt ett säkert lösenord i inställningarna:

![](https://screensaver01.zap-hosting.com/index.php/s/XC7Cxd2zt75jyap/preview)

:::info
Obs: Utan lösenord startar inte din Valheim-server.
:::

Efter att ha satt lösenordet kan vi starta servern, det kan ta några minuter innan den byter till "Online":

![](https://screensaver01.zap-hosting.com/index.php/s/GZQRqpGC6GGaAXa/preview)

## Direktanslutning via Valheim

Först kopierar vi IP-adressen från servern i webbgränssnittet:

![](https://screensaver01.zap-hosting.com/index.php/s/KMkbMyx4bTQkLJT/preview)


Nu öppnar vi Valheim och klickar på knappen "Join IP" i serverlistan:

![](https://screensaver01.zap-hosting.com/index.php/s/zqKp6sx5tEYRebx/preview)


Här lägger vi till vår server-IP:

![](https://screensaver01.zap-hosting.com/index.php/s/EniEzmaP3E9JpLp/preview)


Vi kan nu ansluta direkt till servern via knappen "Connect".


## Direktanslutning via Steam

Först kopierar vi IP-adressen från servern i webbgränssnittet:

![](https://screensaver01.zap-hosting.com/index.php/s/FkYSA8AFm53d8kK/preview)


För att ansluta direkt till Valheim-servern behöver vi öppna serverlistan i Steam:

![](https://screensaver01.zap-hosting.com/index.php/s/yHxMdtTBe7xTWeZ/preview)

Här lägger vi till vår server så här:

![](https://screensaver01.zap-hosting.com/index.php/s/QQcjzriQ5K4Mj9k/preview)

:::info
Obs: Porten måste räknas med +1 i slutet, det betyder att om din port är 27043, måste du ange 27044 i Steam.
:::

![](https://screensaver01.zap-hosting.com/index.php/s/RgsszmnKQLCnYSa/preview)

Efter att ha lagt till detta bör vi nu se vår server i serverlistan:

![](https://screensaver01.zap-hosting.com/index.php/s/54XtiJzn7xndfPT/preview)

Nu kan vi ansluta direkt till servern via knappen "Connect".


## Anslut via Serverlistan
Först öppnar vi Valheim och klickar på `Start Game`, om du inte har en karaktär än måste du skapa en nu, sedan klickar vi på `Start`


Här vill vi inte skapa en ny värld, så vi trycker på "Cancel"

![](https://screensaver01.zap-hosting.com/index.php/s/ZnKWT8eXCcArMwX/preview)

Nu kan vi klicka på `Join Game` högst upp i menyn och skriva in vårt servernamn i "Filter"-fältet, vi sätter också en bock på "Community"

![](https://screensaver01.zap-hosting.com/index.php/s/kFmZP5wyFQbpLiR/preview)

:::info
Om du inte ser din server än, vänta lite. Beroende på antalet servrar kan en uppdatering ta några minuter.
:::

Nu bör vi redan se vår server i listan, vi kan klicka på den och sedan på `Connect`. När vi har anslutit behöver vi bara skriva in serverlösenordet vi satte i inställningarna och trycka Enter!

Klart! Nu kan du spela på din alldeles egna Valheim-server!

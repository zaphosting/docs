## Skapa konto
Att skapa ett ZAP-Hosting-konto är första steget för att komma åt och hantera alla tjänster och funktioner. Registreringsprocessen är snabb och kräver bara några få uppgifter.

För att starta registreringen, besök ZAP-Hostings webbplats och gå till startsidan. Där hittar du inloggningsområdet med en **Registrera dig!**-knapp. Klicka på **Registrera dig!** för att öppna registrerings-popupen.

![img](https://screensaver01.zap-hosting.com/index.php/s/5b6Wzc27CcHW3Zx/download)


I popup-fönstret måste du ange följande information:

- Användarnamn
- E-postadress
- Lösenord

Efter att du fyllt i dina uppgifter, bekräfta säkerhets-captchan. Detta steg krävs som skydd mot automatiska registreringar. När alla fält är ifyllda och captchan bekräftad, skicka in registreringsformuläret för att skapa ditt konto.

![img](https://screensaver01.zap-hosting.com/index.php/s/k6P3wSKWWRRiYax/preview)

## Kontoaktivering

Efter att du skickat in registreringsformuläret skickas ett aktiveringsmail till den angivna e-postadressen. Mailet innehåller en aktiveringskod som måste bekräftas för att slutföra kontoskapandet.

Det kan ta några minuter innan aktiveringsmailet dyker upp. Om det inte syns i inkorgen, kolla även i din skräppost eller spam-mapp.

:::warning Begränsad funktionalitet utan aktivering
Du kan logga in direkt efter registreringen. Men för att få tillgång till alla funktioner och tjänster måste aktiveringsmailet bekräftas.
:::

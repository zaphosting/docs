## Byta din FiveM-version
Tack vare vår gamecloud kan du enkelt byta mellan alla FiveM-versioner helt gratis.

:::info
Bytet mellan Linux-baserade spel (txAdmin, ESX Roleplay, FiveM Linux) till Windows kommer att radera all data
:::

Du bör alltid ha en backup av din data, det kan du enkelt skapa i din "Backup"-flik i gränssnittet.

## Bytet i detalj
För att byta spel (version) på din FiveM-server behöver du öppna "Games" i vårt gränssnitt.

![](https://screensaver01.zap-hosting.com/index.php/s/n82YtN2DEkAeNpq/preview)

Där ser du alla FiveM-versioner under "Available Games", sök bara på "FiveM"

![](https://screensaver01.zap-hosting.com/index.php/s/drTykzYs2527b7D/preview)

Välj den version du vill ha och tryck på den gröna knappen för att starta installationen. En popup kommer att kräva bekräftelse för att starta processen.

:::info
Ett byte mellan Linux och Windows ändrar IP-adressen på din server, om du inte har bokat en egen


Om du byter mellan ESX RP, Linux eller txAdmin måste du aktivera versionen efter installationen med den blå knappen i Games
:::

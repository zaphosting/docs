## Aperçu
ZAP-Hosting propose un grand nombre d’options de paiement différentes avec lesquelles tu peux louer et déployer tes serveurs de jeux immédiatement. Le processus de paiement est simple et rapide. Voici les options de paiement que nous proposons :
*  **ZAP Coins :** Les ZAP Coins sont ajoutés via la Caisse (dépôt d’argent) et peuvent être utilisés pour presque tous les produits.
* **Carte bancaire :** Paiement par carte bancaire.
* **PayPal :** Paiement via PayPal.
* **PaySafeCard :** Les PaySafeCards sont des cartes prépayées que tu peux acheter dans plus de 25 000 points de vente en Allemagne. Le code de la carte est nécessaire pour le paiement.
* **Bitcoin :** Paye avec Bitcoin. La transaction est traitée par le prestataire Coinify.
* **Virement bancaire :** Disponible uniquement pour déposer des ZAP Coins. Après réception du paiement, les ZAP Coins seront activés.

## Processus de paiement
Le processus de paiement est simple : la configuration du serveur est réalisée sur la page de configuration, en bas de la page tu sélectionnes l’option de paiement souhaitée. Après un clic sur « Acheter maintenant », une redirection vers le prestataire de paiement sélectionné aura lieu si nécessaire, via laquelle le paiement peut être effectué. Après réception du paiement, le serveur est configuré en quelques secondes.

## ZAP Coins : Dépôt
Tu peux déposer de l’argent en ZAP Coins via ta caisse en utilisant n’importe quelle option de paiement proposée. Les serveurs sous contrat peuvent aussi être payés avec des ZAP Coins. Dans l’interface web, tu peux déposer des ZAP Coins sous la rubrique **Cash in** dans ta Caisse.

:::info
Le paiement partiel avec des ZAP Coins n’est pas autorisé. Si tu as déjà des ZAP Coins, par exemple grâce à un code ZAP Coins, tu peux ajouter du crédit comme tu veux. Astuce : Si tu payes avec des ZAP Coins, fais attention à nos Cash In Deals : si le Cash In Deal est activé, tu recevras du crédit supplémentaire en augmentant ton dépôt.

Les virements peuvent être utilisés pour déposer des ZAP Coins. Dès que nous avons reçu le virement, le crédit sera activé et utilisable.
:::

## Caisse
La Caisse est accessible via l’interface web sous l’onglet **Caisse**. En plus de la possibilité de déposer de l’argent, la Caisse offre les fonctions supplémentaires suivantes :
* **Ajouter une carte bancaire :** Les cartes bancaires peuvent être enregistrées. En enregistrant une carte bancaire, il n’est plus nécessaire de saisir toutes les données à chaque paiement par carte.
* **Déposer de l’argent :** La fonction de dépôt peut aussi être utilisée via la Caisse.
* **Utiliser un bon :** Les bons donnant des ZAP Coins peuvent être utilisés ici. Après avoir saisi le code du bon, les ZAP Coins sont crédités immédiatement sur le compte.
* **Recharger automatiquement les ZAP Coins :** Les ZAP Coins peuvent être rechargés automatiquement. C’est utile si un serveur sous contrat a été commandé. Pour pouvoir recharger automatiquement le crédit, PayPal doit être utilisé.
* **Vue d’ensemble des transactions :** Un aperçu de toutes les transactions effectuées jusqu’à présent.

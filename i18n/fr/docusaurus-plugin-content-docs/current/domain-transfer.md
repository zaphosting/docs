## Introduction

Gérer votre domaine est important, y compris la possibilité de transférer votre domaine entre différents fournisseurs. Dans ce guide, nous allons voir comment transférer votre domaine existant, aussi bien vers que depuis votre compte ZAP-Hosting.

<InlineVoucher />

## Transfert de Domaine vers ZAP-Hosting

Transférer un domaine vers votre compte ZAP-Hosting est simple. Commencez par vous rendre sur la [page du configurateur de domaine](https://zap-hosting.com/en/shop/product/domain/) tout en étant connecté, puis recherchez votre domaine existant.

:::info
Pour transférer un domaine, vous devrez le prolonger d’1 an lors du processus de commande.
:::

Une fois que vous avez trouvé le domaine exact que vous souhaitez transférer, sélectionnez-le dans la liste. Il est normal que le domaine apparaisse comme indisponible, mais vous devriez voir l’icône de transfert à côté.

![](https://screensaver01.zap-hosting.com/index.php/s/omnaMqXJgarxsqW/preview)

Cela ouvrira une nouvelle fenêtre vous permettant d’autoriser le transfert vers votre compte ZAP-Hosting.

Vous devrez maintenant contacter votre registrar actuel pour demander le **Code d’Autorisation** de votre domaine. En général, les registrars les plus populaires disposent d’un système automatisé pour demander facilement ce code, par exemple via un formulaire ou votre tableau de bord client. En cas de doute, contactez le support de votre registrar.

Une fois que vous avez le **Code d’Autorisation**, saisissez-le dans la fenêtre et confirmez. Continuez ensuite normalement le processus de commande.

![](https://screensaver01.zap-hosting.com/index.php/s/fXjwGCX7kFtPnTB/preview)

Une fois votre commande validée, la demande de transfert est généralement finalisée sous 48 heures. Cependant, ce processus peut parfois prendre plus de temps car votre ancien fournisseur doit d’abord approuver le transfert, donc merci de faire preuve de patience :)

En cas de problème avec la commande, par exemple un souci avec le Code d’Autorisation, nous vous contacterons rapidement pour résoudre cela au plus vite.

## Transfert de Domaine depuis ZAP-Hosting

Ça nous rend triste de vous voir envisager de déplacer votre domaine ailleurs, mais on respecte nos clients et c’est pourquoi nous avons rendu le transfert de domaine depuis ZAP-Hosting tout aussi simple.

Rendez-vous dans la section **Tableau de bord** de l’interface web de votre domaine. Sur cette page, vous pourrez directement voir votre **Code d’Autorisation** actif pour le domaine.

![](https://screensaver01.zap-hosting.com/index.php/s/TTHm6qFA9rfgFBn/preview)

Vous devrez fournir ce code d’autorisation à votre nouveau fournisseur lors de la configuration pour transférer votre domaine.

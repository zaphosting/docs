## Introduction

Vous êtes nouveau et intéressé par les produits de ZAP-Hosting ? Alors vous êtes au bon endroit. Tout d’abord, bienvenue chez ZAP-Hosting ! Nous sommes ravis que vous vous intéressiez à nos services et que vous souhaitiez apprendre comment transférer votre domaine existant depuis un autre fournisseur vers nous. Ce guide vous expliquera les étapes générales à suivre pour réussir le transfert de votre domaine chez nous.

## Avantages

Vous êtes sûrement en train de vous demander quel fournisseur est le mieux pour vous. Ce genre de décision peut parfois prendre du temps. C’est pourquoi nous voulons vous aider à faire votre choix en partageant avec vous les avantages de ZAP-Hosting. ZAP-Hosting propose ses services depuis plus d’une décennie, ce qui en fait un fournisseur reconnu et expérimenté dans le secteur de la location de serveurs.

Notre interface de gestion permet une gestion simple et intuitive des domaines. De plus, nous offrons la possibilité de créer des sous-domaines pour d’autres produits ZAP en un seul clic.

Si vous avez choisi ZAP, vous pouvez commencer à transférer votre domaine existant vers ZAP-Hosting. Ce processus se compose de trois étapes : la préparation, où vous demandez le code d’autorisation ; le transfert, où le domaine est enregistré et transféré chez nous ; et enfin les réglages de configuration nécessaires pour que tout fonctionne parfaitement après.

## Préparation

Pour le transfert d’un domaine déjà enregistré, un **code d’autorisation** (Auth Code) est nécessaire. C’est une sorte de mot de passe pour les domaines. Ce code est requis lorsqu’un domaine doit être transféré d’un fournisseur à un autre. Il garantit que la demande de transfert provient bien du propriétaire légitime du domaine.

Certains fournisseurs fournissent directement le code d’autorisation dans leur interface de gestion. Sinon, vous devrez informer votre fournisseur actuel de votre souhait de transférer le domaine et demander ce code.

:::warning Durée de validité

Le code d’autorisation n’est valable que pour une période limitée. Une fois ce délai dépassé, vous devrez demander un nouveau code. La durée de validité dépend de l’extension du domaine concernée.
:::

## Transfert 

Une fois que vous avez votre code d’autorisation, vous pouvez passer à la commande et à l’enregistrement de votre domaine chez ZAP-Hosting. Rendez-vous sur la page de commande de nos [**domaines**](https://zap-hosting.com/en/shop/product/domain/). Commencez par remplir vos informations personnelles, puis recherchez le domaine que vous souhaitez transférer.

![](https://screensaver01.zap-hosting.com/index.php/s/3dmY76dZscz9DPM/preview)

:::info
Pour transférer un domaine, vous devrez le renouveler d’1 an lors du processus de commande. C’est une règle standard chez tous les fournisseurs.
:::

Une liste de votre domaine souhaité avec toutes les extensions disponibles s’ouvrira alors. Les domaines déjà enregistrés ailleurs sont identifiables par le bouton gris avec une flèche.

![](https://screensaver01.zap-hosting.com/index.php/s/omnaMqXJgarxsqW/preview)

Trouvez le domaine que vous voulez transférer et cliquez sur la flèche. Une nouvelle fenêtre s’ouvrira pour autoriser le transfert vers votre compte ZAP-Hosting.

![](https://screensaver01.zap-hosting.com/index.php/s/fXjwGCX7kFtPnTB/preview)

Dans ce champ, vous devrez entrer le **code d’autorisation** que vous avez reçu de votre ancien fournisseur lors de la préparation. Une fois prêt, validez avec le bouton vert **Ajouter**.

:::danger Vérifiez bien votre saisie

Assurez-vous que le **code d’autorisation** est correct et valide. Sinon, le processus de commande et le transfert du domaine prendront plus de temps et risquent d’échouer.
::: 

Le transfert du domaine est maintenant lancé. La durée du transfert peut varier selon l’extension du domaine et le fournisseur précédent. En général, un transfert est finalisé avec succès sous 72 heures. Cependant, dans certains cas, cela peut prendre plus de temps car plusieurs facteurs entrent en jeu.

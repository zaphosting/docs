## Obtenir les infos du serveur


Vous aurez besoin de toutes les infos nécessaires pour vous connecter à votre serveur. Ces infos se trouvent dans le tableau de bord de votre panneau web du serveur de jeux. Les infos importantes sont les suivantes :

- Adresse IP et port
- Mot de passe (optionnel)


Avec ces infos, vous pourrez localiser votre serveur et vous assurer de vous connecter au bon serveur.

![img](https://screensaver01.zap-hosting.com/index.php/s/psBsqYP23j9NMBo/preview)

## Établir la connexion au serveur


La connexion à un serveur de jeux peut souvent se faire de plusieurs façons. Les méthodes disponibles sont décrites et expliquées en détail ci-dessous.

<Tabs>
    <TabItem value="connect_solution_direct_ingame" label="Connexion directe (In game)" default>

Pour vous connecter à votre serveur via le navigateur de serveurs intégré au jeu, cliquez sur **Multijoueur** puis ouvrez le **Navigateur de serveurs** dans le menu. Une fois là, cliquez sur **Se connecter à une adresse** à gauche. Remplissez l’adresse IP et le port de votre serveur, puis cliquez sur **Ok** pour établir la connexion.

## Problèmes potentiels et solutions

#### Serveur non visible


Le serveur peut ne pas être visible si l’initialisation n’a pas été correctement terminée. Cela peut être dû par exemple à une mauvaise configuration ou à des fichiers corrompus. Vous pouvez généralement trouver plus d’infos dans la console du serveur ou les fichiers logs.

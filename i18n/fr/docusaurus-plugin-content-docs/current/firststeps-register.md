## Créer un compte
Créer un compte ZAP-Hosting est la première étape pour accéder et gérer tous les services et fonctionnalités. Le processus d’inscription est rapide et ne nécessite que quelques informations.

Pour commencer l’inscription, rends-toi sur le site de ZAP-Hosting et va sur la page d’accueil. Tu y trouveras la zone de connexion avec un bouton **S’inscrire !**. Clique sur **S’inscrire !** pour ouvrir la fenêtre popup d’inscription.

![img](https://screensaver01.zap-hosting.com/index.php/s/5b6Wzc27CcHW3Zx/download)

Dans la fenêtre popup, tu dois renseigner les informations suivantes :

- Nom d’utilisateur
- Adresse e-mail
- Mot de passe

Après avoir saisi tes infos, confirme le captcha de sécurité. Cette étape est obligatoire pour éviter les inscriptions automatisées. Une fois tous les champs remplis et le captcha validé, envoie le formulaire d’inscription pour créer ton compte.

![img](https://screensaver01.zap-hosting.com/index.php/s/k6P3wSKWWRRiYax/preview)

## Activation du compte

Après avoir envoyé le formulaire d’inscription, un e-mail d’activation est envoyé à l’adresse e-mail fournie. Cet e-mail contient un code d’activation qu’il faut confirmer pour finaliser la création du compte.

La réception de l’e-mail d’activation peut prendre quelques minutes. Si tu ne vois pas le message dans ta boîte de réception, vérifie aussi ton dossier spam ou courrier indésirable.

:::warning Fonctionnalités limitées sans activation
Tu peux te connecter immédiatement après l’inscription. Cependant, pour accéder à toutes les fonctionnalités et services, il faut confirmer l’e-mail d’activation.
:::

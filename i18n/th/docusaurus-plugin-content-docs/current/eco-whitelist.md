## แนะนำ

Whitelist คือรายการอนุญาตที่จำกัดว่าใครสามารถเชื่อมต่อกับเซิร์ฟเวอร์เกมของคุณได้ ใน ECO whitelist จะถูกเก็บไว้ในไฟล์ `Users.eco` และจะเริ่มทำงานอัตโนมัติเมื่อมี SteamID64 อย่างน้อยหนึ่งรายการถูกเพิ่มเข้าไป

## การเปิดใช้งาน Whitelist

เชื่อมต่อกับเซิร์ฟเวอร์เกมของคุณผ่าน FTP แล้วหาไฟล์ `Users.eco` เปิดไฟล์นั้นและค้นหาส่วน `WhiteList` เพิ่มค่า SteamID64 ลงใน `$values`

```json
"WhiteList": {
  "System.String": {
"$type": "System.Collections.Generic.List`1[[System.String, System.Private.CoreLib]], System.Private.CoreLib",
"$values": [
  "76561198000000000"
]
  }
}
```

บันทึกไฟล์แล้วรีสตาร์ทเซิร์ฟเวอร์เกมเพื่อให้รายการอนุญาตใหม่ถูกนำไปใช้

## การจัดการผู้เล่นใน Whitelist

การเพิ่มผู้เล่นยังสามารถทำได้ผ่าน Live Console ในการจัดการเซิร์ฟเวอร์เกมของ ZAP-Hosting หากคุณมีสิทธิ์ในเกมที่จำเป็น ใช้คำสั่ง whitelist พร้อมชื่อผู้เล่น

```text
/whitelist PlayerName
```

การลบผู้เล่นต้องแก้ไขไฟล์ `Users.eco` โดยลบ SteamID64 ออกจาก `$values` บันทึกไฟล์แล้วรีสตาร์ทเซิร์ฟเวอร์เกม เพราะไม่มีคำสั่งในเซิร์ฟเวอร์เกมสำหรับลบรายการนี้

## การตรวจสอบการทำงานของ Whitelist

หลังจากรีสตาร์ท ลองเชื่อมต่อด้วยบัญชีที่ไม่ได้อยู่ใน `$values` จะต้องถูกบล็อก จากนั้นลองเชื่อมต่อด้วยบัญชีที่อยู่ใน whitelist

ถ้ายังเข้าได้ ให้ตรวจสอบว่ามี SteamID64 อย่างน้อยหนึ่งรายการและไฟล์ `Users.eco` ที่แก้ไขตรงกับอินสแตนซ์เซิร์ฟเวอร์เกมที่กำลังทำงานอยู่

## สรุป

ถ้าทำตามขั้นตอนทั้งหมดข้างต้นถูกต้อง whitelist ของคุณจะเปิดใช้งานแล้วและคุณสามารถควบคุมได้ว่าใครสามารถเข้าร่วมเซิร์ฟเวอร์เกมได้ หากยังมีปัญหาในการเข้าใช้งาน ให้รีสตาร์ทเซิร์ฟเวอร์เกมอีกครั้งและตรวจสอบไฟล์หรือผลลัพธ์คำสั่งเพื่อยืนยันว่าการเปลี่ยนแปลงถูกนำไปใช้แล้ว

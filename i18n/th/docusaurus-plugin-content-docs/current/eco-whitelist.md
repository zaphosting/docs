## แนะนำ

รายชื่อขาวคือรายการอนุญาตที่จำกัดว่าใครสามารถเชื่อมต่อกับเซิร์ฟเวอร์ของคุณได้ ใน ECO รายชื่อขาวจะถูกเก็บไว้ในไฟล์คอนฟิก `Users.eco` และจะเริ่มทำงานโดยอัตโนมัติเมื่อมี SteamID64 อย่างน้อยหนึ่งรายการ

## การเปิดใช้งานรายชื่อขาว

เชื่อมต่อกับเซิร์ฟเวอร์ของคุณผ่าน FTP แล้วหาไฟล์ `Users.eco` เปิดไฟล์นั้นและค้นหาส่วน `WhiteList` เพิ่มค่า SteamID64 ลงใน `$values`

```json
"WhiteList": {
  "System.String": {
"$type": "System.Collections.Generic.List`1[[System.String, System.Private.CoreLib]], System.Private.CoreLib",
"$values": [
  "76561198000000000"
]
  }
}
```

บันทึกไฟล์แล้วรีสตาร์ทเซิร์ฟเวอร์เพื่อให้รายการอนุญาตใหม่ถูกนำไปใช้

## การจัดการผู้เล่นในรายชื่อขาว

การเพิ่มผู้เล่นสามารถทำได้ผ่าน Live Console ในการจัดการเซิร์ฟเวอร์เกมของ ZAP-Hosting หากคุณมีสิทธิ์ในเกม ใช้คำสั่ง whitelist พร้อมชื่อผู้เล่น

```text
/whitelist PlayerName
```

การลบผู้เล่นทำได้โดยแก้ไขไฟล์ `Users.eco` ลบ SteamID64 ออกจาก `$values` บันทึกไฟล์แล้วรีสตาร์ทเซิร์ฟเวอร์ เพราะไม่มีคำสั่งในเซิร์ฟเวอร์สำหรับลบรายการนี้

## การตรวจสอบการทำงานของรายชื่อขาว

หลังรีสตาร์ท ลองเชื่อมต่อด้วยบัญชีที่ไม่ได้อยู่ใน `$values` จะต้องถูกบล็อก จากนั้นลองเชื่อมต่อด้วยบัญชีที่อยู่ในรายชื่อขาว

ถ้ายังเข้าได้ ให้ตรวจสอบว่ามี SteamID64 อย่างน้อยหนึ่งรายการและไฟล์ `Users.eco` ที่แก้ไขตรงกับอินสแตนซ์เซิร์ฟเวอร์ที่กำลังทำงานอยู่

## สรุป

ถ้าทำตามขั้นตอนทั้งหมดข้างต้นถูกต้อง รายชื่อขาวของคุณจะเปิดใช้งานแล้วและคุณสามารถควบคุมได้ว่าใครสามารถเข้าร่วมเซิร์ฟเวอร์ได้ หากยังเข้าไม่ได้ตามที่คาดไว้ ให้รีสตาร์ทเซิร์ฟเวอร์อีกครั้งแล้วตรวจสอบไฟล์หรือผลลัพธ์คำสั่งเพื่อยืนยันว่าการเปลี่ยนแปลงถูกนำไปใช้แล้ว

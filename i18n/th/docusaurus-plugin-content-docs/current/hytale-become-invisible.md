## แนะนำ

การเป็นล่องหนมีประโยชน์บนเซิร์ฟเวอร์ Hytale สำหรับการดูแลระบบ ทดสอบ หรือสร้างคอนเทนต์ ช่วยให้แอดมินสามารถสังเกตผู้เล่นโดยไม่ถูกสังเกต ตรวจสอบกิจกรรมที่น่าสงสัย หรือสำรวจพื้นที่โดยไม่รบกวนการเล่นเกม ขึ้นอยู่กับการตั้งค่าเซิร์ฟเวอร์และสิทธิ์ที่มี การเป็นล่องหนมักจะเปิดใช้งานได้ผ่านคำสั่งในเกมหรือคอนโซลสด

<InlineVoucher />

## ความต้องการ

เพื่อใช้ฟีเจอร์การเป็นล่องหน คุณต้องมีสิทธิ์เพียงพอบนเซิร์ฟเวอร์ โดยส่วนใหญ่ต้องมีสิทธิ์ผู้ดูแลระบบหรือสิทธิ์เข้าถึงคำสั่งดูแลและเล่นเกม หากคุณไม่มีสิทธิ์ที่จำเป็น คำสั่งจะถูกปฏิเสธและไม่สามารถเปิดใช้งานการเป็นล่องหนได้ หากต้องการสิทธิ์ที่จำเป็น ให้ทำตาม [คู่มือการเป็นแอดมิน](hytale-becomeadmin.md) ของเรา

## วิธีการเป็นล่องหน

เพื่อเป็นล่องหน ให้เปิดแชทในเกมแล้วพิมพ์คำสั่งการเป็นล่องหนที่เซิร์ฟเวอร์รองรับ ใช้คำสั่งนี้:

```
/hide <playername>
```

เมื่อคำสั่งทำงานสำเร็จ ตัวละครของคุณจะไม่ปรากฏให้ผู้เล่นคนอื่นเห็นในโลกเกม คุณยังสามารถเคลื่อนที่ โต้ตอบ และตรวจสอบกิจกรรมได้ ขึ้นอยู่กับกฎของเซิร์ฟเวอร์และการตั้งค่าการมองเห็น

นอกจากคำสั่งซ่อนพื้นฐานแล้ว Hytale ยังมีตัวเลือกเพิ่มเติมที่ให้คุณควบคุมว่าใครจะเห็นผู้เล่นได้ เช่น ซ่อนผู้เล่นจากผู้เล่นเป้าหมายเฉพาะ หรือทำให้ทุกคนบนเซิร์ฟเวอร์ล่องหน

คำสั่งที่ใช้ได้มีดังนี้:
- `/hide <playername>`  ทำให้ผู้เล่นที่ระบุเป็นล่องหน
- `/hide <playername> --target <playername>`  ทำให้ผู้เล่นที่ระบุเป็นล่องหนเฉพาะกับผู้เล่นเป้าหมาย
- `/hide all`  ทำให้ผู้เล่นทุกคนเป็นล่องหน

## การปิดการเป็นล่องหน

เพื่อกลับสู่การมองเห็นปกติ ให้รันคำสั่งที่เหมาะสมอีกครั้งเพื่อปิดการเป็นล่องหน หลังจากคำสั่งยืนยัน ผู้เล่นคนอื่นจะเห็นตัวละครของคุณอีกครั้งตามปกติ

```
/hide show <playername>
```

ยังมีคำสั่งเพิ่มเติมสำหรับการยกเลิกการเป็นล่องหนในสถานการณ์เฉพาะ เช่น คืนการมองเห็นเฉพาะกับผู้เล่นเป้าหมาย หรือยกเลิกการเป็นล่องหนจากทุกคน

คำสั่งที่ใช้ได้มีดังนี้:
- `/hide show <playername>`  ยกเลิกการเป็นล่องหนของผู้เล่นที่ระบุ
- `/hide show <playername> --target <playername>`  ยกเลิกการเป็นล่องหนของผู้เล่นที่ระบุสำหรับผู้เล่นเป้าหมาย
- `/hide showall`  ยกเลิกการเป็นล่องหนของผู้เล่นทุกคน

## สรุป

การเป็นล่องหนบนเซิร์ฟเวอร์ Hytale เป็นเครื่องมือที่มีประโยชน์สำหรับแอดมินและม็อดที่ต้องการตรวจสอบการเล่นเกมอย่างลับๆ หากมีคำถามเพิ่มเติมหรือต้องการความช่วยเหลือ อย่าลังเลที่จะติดต่อทีมซัพพอร์ตของเรา ซึ่งพร้อมช่วยเหลือคุณทุกวัน! 🙂

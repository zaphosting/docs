## แนะนำ

การตั้งค่าการแชทใน **Over the Top WWI** ควบคุมวิธีที่ผู้เล่นสื่อสารผ่านข้อความระหว่างเล่นเกม การปรับแต่งการตั้งค่านี้ช่วยให้คุณเปิดใช้งานการสื่อสารทั่วโลก จำกัดการใช้งานแชท หรือปิดเสียงผู้เล่นทั้งหมดได้ถ้าจำเป็น

การตั้งค่าการแชทอย่างถูกต้องช่วยรักษาความเรียบร้อยบนเซิร์ฟเวอร์และเพิ่มประสบการณ์การเล่นโดยรวมให้ดียิ่งขึ้น

## การตั้งค่า

การตั้งค่าการแชทจะถูกกำหนดในไฟล์คอนฟิกของเซิร์ฟเวอร์ คุณสามารถเข้าถึงและแก้ไขไฟล์นี้ได้ใน **แผงควบคุมเซิร์ฟเวอร์เกม** ของคุณภายใต้เมนู **Configs** ค้นหาและเปิดไฟล์คอนฟิก `ServerConfiguration.ini` ภายในไฟล์นี้ ให้หาพารามิเตอร์ดังต่อไปนี้:

```
AllowAllChat = 1
ServerMuteAll = 0
BannedWordsPathString = BannedWords.txt
```

- `AllowAllChat` เปิดหรือปิดการสื่อสารแชททั่วโลก

- `0` → ปิดใช้งาน
- `1` → เปิดใช้งาน

- `ServerMuteAll` ปิดเสียงแชทของผู้เล่นทั้งหมดเมื่อเปิดใช้งาน

- `0` → ปิดใช้งาน
- `1` → เปิดใช้งาน

- `BannedWordsPathString` กำหนดไฟล์ที่ใช้กรองคำต้องห้าม ไฟล์นี้จะมีรายชื่อคำที่ถูกบล็อกไม่ให้ใช้ในแชท

ปรับแต่งการตั้งค่าเหล่านี้เพื่อควบคุมวิธีที่ผู้เล่นสื่อสารและบังคับใช้กฎการดูแลแชทบนเซิร์ฟเวอร์ของคุณ

หลังจากแก้ไขไฟล์ `ServerConfiguration.ini` ให้บันทึกไฟล์และรีสตาร์ทเซิร์ฟเวอร์ของคุณ การตั้งค่าการแชทที่อัปเดตจะถูกนำไปใช้โดยอัตโนมัติ

## สรุป

ยินดีด้วย! คุณได้ตั้งค่าการแชทบน **เซิร์ฟเวอร์ Over the Top WWI** ของคุณเรียบร้อยแล้ว ซึ่งช่วยให้คุณจัดการการสื่อสารของผู้เล่นและรักษาสภาพแวดล้อมที่ควบคุมได้และสนุกสนาน

หากมีคำถามเพิ่มเติมหรือต้องการความช่วยเหลือ อย่าลังเลที่จะติดต่อทีมซัพพอร์ตของเรา ซึ่งพร้อมให้บริการคุณทุกวัน! 🙂

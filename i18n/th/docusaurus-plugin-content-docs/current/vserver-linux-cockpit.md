## แนะนำ

Cockpit คือเว็บอินเทอร์เฟซสำหรับจัดการเซิร์ฟเวอร์ Linux หนึ่งเครื่องหรือหลายเครื่อง (คลัสเตอร์) ตั้งแต่การดูล็อกง่ายๆ ไปจนถึงการจัดการ RAID Cockpit มีฟีเจอร์ครบครัน  
Cockpit เหมาะสำหรับมือใหม่และผู้ใช้ขั้นสูงที่อยากเห็น/จัดการทุกอย่างที่สำคัญได้ในเวลาสั้นๆ นอกจากนี้ยังเข้าถึงได้จากแทบทุกอุปกรณ์  
เว็บไซต์โปรเจกต์สามารถเข้าได้ที่ https://cockpit-project.org/  

## ติดตั้ง Cockpit ด้วย One Click Apps Installer

คุณสามารถติดตั้ง **Cockpit** ได้โดยตรงผ่าน **One Click Apps Installer** ในเว็บอินเทอร์เฟซของ VPS หลังจากตั้งค่าแอปเริ่มต้นเสร็จแล้ว ให้เปิดแคตตาล็อกแอป ค้นหา **Cockpit** แล้วเริ่มติดตั้งพร้อมตั้งค่าโปรเจกต์ สภาพแวดล้อม และโดเมนที่ต้องการ วิธีนี้ช่วยให้คุณติดตั้งและจัดการ **Cockpit** ได้อย่างรวดเร็วและง่ายดายโดยไม่ต้องตั้งค่าผ่านคำสั่งในเทอร์มินัลเอง แถมยังได้ประโยชน์จากการจัดการผ่านเว็บ, รองรับโดเมนแบบกำหนดเอง และการจัดเตรียม SSL เมื่อมีให้บริการ

:::info
คู่มือนี้อธิบายการติดตั้งสำหรับ Debian (ตั้งแต่ Buster) / Ubuntu (ตั้งแต่ Bionic Beaver) การติดตั้งยังมีในหน้าโปรเจกต์สำหรับ Fedora, Red Hat, Fedora CoreOS, CentOS, Clear Linux, Arch Linux และ Tumbleweed ด้วย  
:::

## การติดตั้ง

อันดับแรก ต้องอัปเดตเซิร์ฟเวอร์ก่อน:
```
// อัปเดต
sudo apt update; sudo apt upgrade -y
```
จากนั้นติดตั้งได้เลย: 
```
// ติดตั้ง Cockpit
sudo apt install cockpit -y
```
เมื่อติดตั้งเสร็จแล้ว สามารถเข้าถึงแผงควบคุม Cockpit ผ่าน IP:9090  
ล็อกอินด้วยข้อมูลล็อกอินปกติของเซิร์ฟเวอร์/ผู้ใช้

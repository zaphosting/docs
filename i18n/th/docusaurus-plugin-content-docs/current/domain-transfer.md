## แนะนำ

การจัดการโดเมนของคุณเป็นเรื่องสำคัญ รวมถึงความสามารถในการโอนโดเมนระหว่างผู้ให้บริการ ในคู่มือนี้ เราจะพาคุณไปดูวิธีการโอนโดเมนที่คุณมีอยู่ ทั้งการโอนเข้าและโอนออกจากบัญชี ZAP-Hosting ของคุณ

<InlineVoucher />

## การโอนโดเมนเข้า ZAP-Hosting

การโอนโดเมนเข้าไปยังบัญชี ZAP-Hosting ของคุณนั้นง่ายมาก เริ่มต้นด้วยการไปที่ [หน้าคอนฟิกูเรเตอร์โดเมน](https://zap-hosting.com/en/shop/product/domain/) ขณะล็อกอินอยู่ และค้นหาโดเมนที่คุณมีอยู่

:::info
เพื่อโอนโดเมน คุณจะต้องต่ออายุโดเมนนั้นเพิ่มอีก 1 ปีในขั้นตอนการสั่งซื้อ
:::

เมื่อคุณเจอโดเมนที่ต้องการโอน ให้เลือกโดเมนนั้นจากรายการ เป็นเรื่องปกติที่โดเมนอาจแสดงว่าไม่ว่าง แต่คุณจะเห็นไอคอนโอนโดเมนอยู่ข้างๆ

![](https://screensaver01.zap-hosting.com/index.php/s/omnaMqXJgarxsqW/preview)

นี่จะเปิดหน้าต่างใหม่ขึ้นมาเพื่อให้คุณยืนยันการโอนโดเมนเข้าสู่บัญชี ZAP-Hosting ของคุณ

ตอนนี้คุณต้องไปที่ผู้จดทะเบียนโดเมนเดิมของคุณและขอ **Auth Code** สำหรับโดเมนนั้น โดยปกติผู้จดทะเบียนที่ได้รับความนิยมจะมีระบบอัตโนมัติให้คุณขอรหัสอนุญาตนี้ได้ง่ายๆ เช่น ผ่านการกรอกแบบฟอร์มหรือแดชบอร์ดลูกค้า หากไม่แน่ใจ คุณควรติดต่อฝ่ายสนับสนุนของผู้จดทะเบียนโดเมนของคุณ

เมื่อคุณมี **Auth Code** พร้อมแล้ว ให้กรอกลงในหน้าต่างที่เปิดขึ้นและยืนยัน จากนั้นดำเนินการสั่งซื้อเหมือนปกติ

![](https://screensaver01.zap-hosting.com/index.php/s/fXjwGCX7kFtPnTB/preview)

หลังจากส่งคำสั่งซื้อแล้ว คำขอโอนโดเมนของคุณจะเสร็จสมบูรณ์ภายใน 48 ชั่วโมงโดยทั่วไป แต่บางครั้งอาจใช้เวลานานกว่านี้เนื่องจากผู้ให้บริการเดิมต้องอนุมัติการโอนก่อน ดังนั้นเราขอให้คุณอดทนในช่วงเวลานี้ :)

หากมีปัญหาเกี่ยวกับคำสั่งซื้อ เช่น ปัญหาเกี่ยวกับ Auth Code เราจะติดต่อคุณโดยเร็วที่สุดเพื่อแก้ไขปัญหาอย่างรวดเร็ว

## การโอนโดเมนออกจาก ZAP-Hosting

เรารู้สึกเสียดายที่คุณพิจารณาย้ายโดเมนออกจากเรา แต่เรายึดมั่นในความเคารพลูกค้า ดังนั้นเราจึงทำให้การโอนโดเมนออกจากเราง่ายเหมือนกัน

ไปที่ส่วน **แดชบอร์ด** บนแผงเว็บอินเทอร์เฟซของโดเมนคุณ ในหน้านี้คุณจะเห็น **Auth Code** ที่ใช้งานอยู่สำหรับโดเมนของคุณโดยตรง

![](https://screensaver01.zap-hosting.com/index.php/s/TTHm6qFA9rfgFBn/preview)

คุณควรให้รหัสอนุญาตนี้กับผู้ให้บริการรายใหม่ของคุณในขั้นตอนการตั้งค่าเพื่อย้ายโดเมน

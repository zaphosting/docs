## แนะนำ

ในคู่มือนี้ คุณจะได้เรียนรู้วิธีจัดการเซฟเกม Myth of Empires บนเซิร์ฟเวอร์เกมของคุณ รวมถึงการสำรองข้อมูลเซฟเกมทั้งในเครื่องและบนเซิร์ฟเวอร์ รวมถึงการอัปโหลดไปยังเซิร์ฟเวอร์ของคุณ

<InlineVoucher />

## การเตรียมตัว

เพื่อเข้าถึงไฟล์บนเซิร์ฟเวอร์ของคุณ คุณจะต้องใช้ FTP Client อย่างเช่น [WinSCP](https://winscp.net/eng/index.php) หรือ [FileZilla](https://filezilla-project.org/) ซึ่งทั้งสองโปรแกรมเป็นโอเพ่นซอร์สและดาวน์โหลดฟรี

คุณต้องเชื่อมต่อกับเซิร์ฟเวอร์ของคุณผ่าน FTP Client โดยใช้ข้อมูลล็อกอินที่อยู่ใน **เครื่องมือ->FTP-Browser** บนเว็บอินเทอร์เฟซของเซิร์ฟเวอร์เกมของคุณ โปรดใช้ [คู่มือการเข้าถึงผ่าน FTP](gameserver-ftpaccess.md) ของเราเพื่อช่วยในการเชื่อมต่อกับเซิร์ฟเวอร์

## การสำรองข้อมูลเซฟเกม

### เซฟเกมในเครื่อง

เซฟเกมในเครื่องคือเซฟที่คุณสร้างขึ้นขณะโฮสต์เกมมัลติเพลเยอร์บนเครื่องของคุณเอง MOE จะเก็บไฟล์เหล่านี้ไว้ในโฟลเดอร์ภายในตำแหน่งติดตั้งเกม

เปิด Steam บนเครื่องของคุณ คลิกขวาที่เกม **Myth of Empires** แล้วเลือก **เรียกดูไฟล์ในเครื่อง** ขณะเลื่อนเมาส์ไปที่ส่วน **จัดการ**

![](https://screensaver01.zap-hosting.com/index.php/s/Cmj325wLSWgNGif/preview)

ไปที่เส้นทางนี้:
```
../WindowsPrivateServer/MOE/Saved/SaveGames/
```

ในโฟลเดอร์นี้ คุณจะเห็นเซฟเกมในเครื่องทั้งหมดที่มีอยู่ตอนนี้

### การเข้าถึงเซฟเกมผ่าน FTP

การสำรองข้อมูลเซฟเกมจากเซิร์ฟเวอร์ทำได้ง่ายมาก เมื่อคุณเชื่อมต่อกับเซิร์ฟเวอร์เกมผ่าน FTP Client แล้ว ให้ไปที่เส้นทางนี้:
```
../moe-dedicated/MOE/Saved/SaveGames/
```

ที่นี่คุณจะเห็นโฟลเดอร์ซึ่งเป็นเซฟเกมของคุณ แค่คลิกขวาที่โฟลเดอร์แล้วกดปุ่ม **ดาวน์โหลด** เพื่อบันทึกลงเครื่องของคุณ

![](https://screensaver01.zap-hosting.com/index.php/s/3yN723tY8eiBb2w/preview)

### การสำรองข้อมูลอัตโนมัติ

เรายังมีตัวเลือกให้สำรองข้อมูลเซฟเกม (และไฟล์คอนฟิก) อัตโนมัติผ่านเว็บอินเทอร์เฟซของเราได้เลย แค่ไปที่เว็บอินเทอร์เฟซของเซิร์ฟเวอร์เกม แล้วเข้าไปที่ **เครื่องมือ->สำรองข้อมูล** ที่นี่คุณสามารถตั้งค่าตัวเลือกต่างๆ เพื่อกำหนดเวลาสำรองข้อมูลอัตโนมัติสำหรับเซิร์ฟเวอร์ของคุณ เรามีพื้นที่สำรองข้อมูลฟรี 10GB สำหรับเก็บไฟล์สำรองของคุณ หากต้องการข้อมูลเพิ่มเติมเกี่ยวกับการสำรองข้อมูล ดูได้ที่คู่มือ [สำรองข้อมูล](gameserver-backups.md) ของเรา

## การอัปโหลดเซฟเกม

เหมือนกับการสำรองข้อมูล การอัปโหลดเซฟเกมก็ง่ายมาก ก่อนอื่นให้แน่ใจว่าคุณเชื่อมต่อกับเซิร์ฟเวอร์เกมผ่าน FTP Client แล้ว จากนั้นไปที่เส้นทางนี้:
```
../moe-dedicated/MOE/Saved/SaveGames/
```

![](https://screensaver01.zap-hosting.com/index.php/s/KLD26KTYZYpfoqP/preview)

แค่ลากและวางเซฟเกมของคุณลงในโฟลเดอร์นี้ผ่าน FTP Client แล้วไฟล์จะถูกอัปโหลดไปยังเซิร์ฟเวอร์ของคุณ

## การเปิดใช้งานเซฟเกม

การใช้เซฟเกมของคุณจะต้องเลือกเซฟนั้นในคอนฟิกเซิร์ฟเวอร์ วิธีที่ง่ายที่สุดคือไปที่ส่วน **การตั้งค่า** บนเว็บอินเทอร์เฟซของเซิร์ฟเวอร์เกม ใช้ [คู่มือการตั้งค่าเซิร์ฟเวอร์](moe-configuration.md) ของเราเพื่อข้อมูลเพิ่มเติม

![](https://screensaver01.zap-hosting.com/index.php/s/QDPzFgWRrfB49HB/preview)

ในส่วนนี้ ให้หาพารามิเตอร์ **ชื่อแผนที่** ในการตั้งค่าพื้นฐาน แล้วเซฟเกมที่คุณอัปโหลดจะปรากฏที่นี่ หากไม่เห็น ให้แน่ใจว่าได้รีสตาร์ทเซิร์ฟเวอร์หลังจากอัปโหลดแล้ว

คุณได้เพิ่มเซฟเกมลงในเซิร์ฟเวอร์เกมของคุณเรียบร้อยแล้ว

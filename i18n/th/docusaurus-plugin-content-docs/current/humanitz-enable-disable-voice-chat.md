## บทนำ

การสื่อสารด้วยเสียงสามารถเปลี่ยนบรรยากาศของเซิร์ฟเวอร์ HumanitZ ได้อย่างมาก การเปิดแชทเสียงช่วยให้ผู้เล่นสื่อสารกันโดยตรงในระยะใกล้ เพิ่มความสมจริงและความร่วมมือ ในขณะที่การปิดแชทเสียงอาจกระตุ้นให้ใช้แพลตฟอร์มสื่อสารภายนอก หรือสร้างประสบการณ์เอาชีวิตรอดที่เงียบสงบและโดดเดี่ยวมากขึ้น

ใน HumanitZ การตั้งค่าแชทเสียงจะถูกกำหนดโดยตรงในไฟล์คอนฟิกของเซิร์ฟเวอร์

## การตั้งค่า

การตั้งค่าแชทเสียงถูกควบคุมในไฟล์ `GameServerSettings.ini` คุณสามารถเข้าถึงไฟล์นี้ได้ในแผงควบคุมเซิร์ฟเวอร์ของคุณภายใต้เมนู **Configs** ในไฟล์คอนฟิก ให้ค้นหาพารามิเตอร์ดังนี้:

```
;When set to 1 or true voice chat will be enabled, set 0 or false to disable
Voip=true
```

การตั้งค่านี้ทำงานดังนี้:

- `Voip=true` หรือ `Voip=1` – เปิดแชทเสียง  
- `Voip=false` หรือ `Voip=0` – ปิดแชทเสียง  

เพื่อเปลี่ยนการตั้งค่าแชทเสียง แค่แก้ไขค่าตามที่คุณต้องการ หลังจากแก้ไขไฟล์แล้ว ให้บันทึกการเปลี่ยนแปลงและรีสตาร์ทเซิร์ฟเวอร์เพื่อให้การตั้งค่าใหม่มีผล



## สรุป

ยินดีด้วย! โดยการแก้ไขค่า `Voip` ในไฟล์ `GameServerSettings.ini` คุณได้เปิดหรือปิดแชทเสียงบนเซิร์ฟเวอร์ HumanitZ ของคุณเรียบร้อยแล้ว หากมีคำถามเพิ่มเติมหรือต้องการความช่วยเหลือ อย่าลังเลที่จะติดต่อทีมซัพพอร์ตของเรา ซึ่งพร้อมให้บริการคุณทุกวัน! 🙂

## Introduction
ไม่แน่ใจว่าจะเชื่อมต่อกับเซิร์ฟเวอร์ **Hytale** ของคุณอย่างไรหรือจำเป็นต้องมีอะไรบ้างเพื่อเริ่มต้น? ไม่ต้องกังวล เราจะช่วยคุณเอง! เราจะพาคุณผ่านทุกขั้นตอนที่จำเป็น ตั้งแต่เครื่องมือและข้อมูลที่ต้องใช้ ไปจนถึงกระบวนการเชื่อมต่อจริง พร้อมคำแนะนำสำคัญเพื่อให้การเชื่อมต่อของคุณราบรื่นและไม่มีปัญหา แค่ตามคู่มือเรา คุณก็จะเชื่อมต่อได้ในเวลาไม่นาน!

## Obtain server details
ก่อนเชื่อมต่อ คุณต้องเก็บข้อมูลเซิร์ฟเวอร์ที่เกี่ยวข้องจากเว็บอินเทอร์เฟซเซิร์ฟเวอร์เกม ZAP-Hosting ของคุณ สำหรับ Windrose รายละเอียดที่สำคัญที่สุดคือ **รหัสเชิญ**

คุณสามารถหาโค้ดเชิญได้ในส่วนจัดการเซิร์ฟเวอร์เกมของคุณ ภายใต้ไฟล์การตั้งค่าเซิร์ฟเวอร์ในไฟล์ `InviteCode.txt`

:::info ข้อกำหนดรหัสเชิญ
สำหรับ Windrose รหัสเชิญคือค่าหลักที่ใช้เชื่อมต่อกับเซิร์ฟเวอร์ หากใช้รหัสผิดหรือรหัสเก่า การเชื่อมต่ออาจล้มเหลว
:::

ข้อมูลต่อไปนี้มีประโยชน์ก่อนเริ่มต้น:

| ข้อมูล | จุดประสงค์ |
| --- | --- |
| เนื้อหาใน `InviteCode.txt` | จำเป็นสำหรับการเชื่อมต่อกับเซิร์ฟเวอร์ Windrose |
| ชื่อเซิร์ฟเวอร์ | ช่วยให้คุณระบุเซิร์ฟเวอร์ที่ถูกต้อง |
| รหัสผ่านเซิร์ฟเวอร์ (ถ้ามี) | อาจจำเป็นขึ้นอยู่กับการตั้งค่าของคุณ |

![](https://screensaver01.zap-hosting.com/index.php/s/boTwmEC5HSjbKYK/preview)

## Establish server connection
เมื่อคุณมีรหัสเชิญแล้ว คุณสามารถเชื่อมต่อกับเซิร์ฟเวอร์ Windrose ได้โดยตรงผ่านเกม

### หาโค้ดเชิญในส่วนจัดการเซิร์ฟเวอร์
เปิดเว็บอินเทอร์เฟซเซิร์ฟเวอร์เกม ZAP-Hosting ของคุณและไปที่ส่วนการตั้งค่าเซิร์ฟเวอร์ ค้นหาไฟล์ชื่อ `InviteCode.txt` แล้วเปิดไฟล์นั้น

ในไฟล์นี้ คุณจะพบรหัสเชิญปัจจุบันสำหรับเซิร์ฟเวอร์ของคุณ คัดลอกค่าตรงตามที่แสดง

:::note ค่าตัวแทน
ถ้าคุณเห็นค่าที่เป็น `[your_invite_code]` ในคู่มือนี้ ให้แทนที่ด้วยรหัสจริงจากไฟล์ `InviteCode.txt` ของคุณเอง
:::

### เข้าร่วมเซิร์ฟเวอร์ใน Windrose
เริ่มเกม Windrose และเปิดเมนูเชื่อมต่อเซิร์ฟเวอร์หรือโหมดผู้เล่นหลายคนในเกม มองหาตัวเลือกที่ให้คุณเข้าร่วมเซิร์ฟเวอร์โดยใช้รหัสเชิญ

กรอกรหัสเชิญจาก `InviteCode.txt` ลงในช่องที่กำหนดและยืนยันการเชื่อมต่อ หากเซิร์ฟเวอร์ของคุณมีการป้องกัน อาจต้องกรอกรหัสผ่านในขั้นตอนนี้ด้วย

| ช่องกรอก | กรอกข้อมูล |
| --- | --- |
| รหัสเชิญ | `[your_invite_code]` |
| รหัสผ่าน | `[your_server_password]` หากตั้งค่าไว้ |

หลังจากส่งรหัสเชิญ เกมจะพยายามเชื่อมต่อกับเซิร์ฟเวอร์ Windrose ของคุณ

:::tip คัดลอกรหัสให้ตรง
เพื่อหลีกเลี่ยงปัญหาการเชื่อมต่อ ให้คัดลอกและวางรหัสเชิญโดยตรงจากไฟล์ `InviteCode.txt` ถ้าเป็นไปได้ เพราะแม้แต่การพิมพ์ผิดเล็กน้อยก็อาจทำให้เชื่อมต่อไม่สำเร็จ
:::

![](https://screensaver01.zap-hosting.com/index.php/s/t3R6by5GrswnZsQ/download)

## Potential issues and solutions
ถ้าการเชื่อมต่อเซิร์ฟเวอร์ไม่สำเร็จทันที คุณสามารถตรวจสอบสาเหตุที่พบบ่อยได้ดังนี้

### รหัสเชิญใช้ไม่ได้
ถ้ารหัสเชิญถูกปฏิเสธ ให้ตรวจสอบก่อนว่าคุณคัดลอกเนื้อหาใน `InviteCode.txt` ครบถ้วนและถูกต้องหรือไม่ ตรวจสอบให้แน่ใจว่าไม่มีช่องว่างเกินก่อนหรือหลังรหัส

นอกจากนี้ รหัสเชิญอาจเปลี่ยนหลังจากเซิร์ฟเวอร์รีสตาร์ท อัปเดต หรือปรับแต่งการตั้งค่า ในกรณีนั้น ให้เปิดไฟล์ `InviteCode.txt` อีกครั้งและใช้รหัสล่าสุด

### เซิร์ฟเวอร์ยังไม่พร้อม
ถ้าเซิร์ฟเวอร์กำลังเริ่มต้นหรือยังไม่เสร็จสิ้นการโหลด ผู้เล่นอาจไม่สามารถเชื่อมต่อได้แม้จะใช้รหัสเชิญถูกต้อง

ตรวจสอบคอนโซลและบันทึกในส่วนจัดการเซิร์ฟเวอร์เกมของคุณเพื่อยืนยันว่าเซิร์ฟเวอร์เริ่มทำงานสำเร็จและออนไลน์เต็มที่แล้ว

:::caution รอให้เซิร์ฟเวอร์เริ่มทำงานเต็มที่
อย่าพยายามเชื่อมต่อขณะที่เซิร์ฟเวอร์กำลังโหลดหรือรีสตาร์ท เพราะอาจทำให้การเชื่อมต่อล้มเหลวแม้รหัสเชิญจะถูกต้อง
:::

### ปัญหาการตั้งค่าหรือไฟล์
ถ้าเซิร์ฟเวอร์ยังไม่สามารถเข้าถึงได้ อาจมีปัญหาการตั้งค่าหรือไฟล์เสียหายที่ขัดขวางการเริ่มต้นอย่างถูกต้อง ในกรณีนี้ ให้ตรวจสอบการเปลี่ยนแปลงล่าสุดที่คุณทำกับเซิร์ฟเวอร์และดูบันทึกข้อผิดพลาด

ถ้าจำเป็น ให้คืนค่าจากสำรองข้อมูลที่ใช้งานได้หรือย้อนกลับการตั้งค่าที่เปลี่ยนแปลงล่าสุด

### ไม่มีวิธีแก้ไขหรือไม่ช่วย
ถ้าคุณยังไม่สามารถเชื่อมต่อได้หลังตรวจสอบรหัสเชิญ สถานะเซิร์ฟเวอร์ และการตั้งค่า ให้ติดต่อทีมซัพพอร์ต ZAP-Hosting ผ่าน [หน้าซัพพอร์ตอย่างเป็นทางการ](https://zap-hosting.com/en/customer/support/)

เมื่อส่งคำขอ กรุณาระบุรายละเอียดให้ครบถ้วน เช่น:

- ปัญหาที่เกิดขึ้นอย่างชัดเจน
- เซิร์ฟเวอร์เริ่มทำงานสำเร็จหรือไม่
- รหัสเชิญใน `InviteCode.txt` มองเห็นหรือไม่
- ข้อความแสดงข้อผิดพลาดจากคอนโซลหรือบันทึก

## Conclusion
ยินดีด้วย คุณเชื่อมต่อกับเซิร์ฟเวอร์ Windrose ของคุณสำเร็จโดยใช้รหัสเชิญแล้ว สำหรับคำถามเพิ่มเติมหรือต้องการความช่วยเหลือ อย่าลังเลที่จะติดต่อทีมซัพพอร์ตของเรา ซึ่งพร้อมให้บริการทุกวันเพื่อช่วยเหลือคุณ! 🙂

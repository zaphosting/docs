## แนะนำ
ไม่แน่ใจว่าจะเชื่อมต่อกับ **Minecraft** เซิร์ฟเวอร์ยังไง หรือจำเป็นต้องมีอะไรบ้างเพื่อเริ่มต้น? ไม่ต้องห่วง เราจะช่วยคุณเอง! เราจะพาคุณผ่านทุกอย่างที่ต้องใช้ – ตั้งแต่เครื่องมือและข้อมูลที่จำเป็น ไปจนถึงขั้นตอนการเชื่อมต่อจริง พร้อมคำแนะนำสำคัญเพื่อให้การเชื่อมต่อของคุณราบรื่นและไม่มีปัญหา แค่ตามคู่มือเรา คุณก็จะเชื่อมต่อได้ในไม่ช้า!



## ความต้องการ

อาจต้องใช้ซอฟต์แวร์เพิ่มเติมเพื่อเชื่อมต่อกับเซิร์ฟเวอร์เกมสำหรับเกมนี้ สำหรับเวอร์ชัน vanilla ของ Minecraft หรือที่ไม่มีม็อดหรือม็อดแพ็คเพิ่มเติม จะไม่ต้องใช้ซอฟต์แวร์เสริมใดๆ แต่ถ้าคุณต้องการใช้ม็อด/ม็อดแพ็ค คุณจะต้องมีตัวเปิดเกม (launcher) ที่ตรงกันด้วย คุณสามารถดูได้ว่าต้องใช้ launcher ตัวไหนสำหรับม็อดหรือม็อดแพ็คแต่ละตัวจากชื่อเกมในแดชบอร์ด

| เวอร์ชัน Minecraft                      | ต้องใช้ซอฟต์แวร์เพิ่มเติมไหม |
| --------------------------------------- | ---------------------------- |
| Minecraft: Vanilla                      | X                            |
| Minecraft: Paper Spigot                 | X                            |
| Minecraft: Spigot                       | X                            |
| Minecraft: Bukkit                       | X                            |
| Minecraft: Forge                        | ✓                            |
| Minecraft: Fabric                       | ✓                            |
| Minecraft กับม็อด/ม็อดแพ็คโดยทั่วไป    | ✓                            |




#### ม็อด
ถ้าคุณอยากใช้ม็อด คุณจะต้องมีตัวโหลดม็อดที่ตรงกันด้วย ขึ้นอยู่กับว่าคุณจะใช้ม็อดจาก Forge หรือ Fabric คุณจะต้องใช้ตัวใดตัวหนึ่งในสองนี้

- [Minecraft: Forge](https://files.minecraftforge.net/net/minecraftforge/forge/) - เพื่อใช้ม็อด Forge จำเป็นต้องดาวน์โหลดและติดตั้ง Forge
- [Minecraft: Fabric](https://fabricmc.net/) - เพื่อใช้ม็อด Fabric จำเป็นต้องดาวน์โหลดและติดตั้ง Fabric

ม็อด Forge และ Fabric ต้องติดตั้งด้วยตัวเองทั้งในเกมและในซอฟต์แวร์เซิร์ฟเวอร์เพื่อให้สามารถเชื่อมต่อได้

:::danger อย่าผสมม็อด Fabric กับ Forge
Fabric และ Forge ไม่เข้ากัน ม็อด Forge ต้องรันบน Forge และม็อด Fabric ต้องรันบน Fabric การผสมกันอาจทำให้ทั้งไคลเอนต์และเซิร์ฟเวอร์ล่มได้
:::


#### ม็อดแพ็ค
ถ้าคุณอยากใช้ม็อดแพ็ค คุณจะต้องมีตัวเปิดเกม (launcher) ที่ตรงกันด้วย คุณสามารถดูได้ว่าต้องใช้ launcher ตัวไหนสำหรับม็อดแพ็คแต่ละตัวจากชื่อเกมในแดชบอร์ด
- [CurseForge](https://www.curseforge.com/) - เพื่อใช้ม็อดแพ็คจาก Curse / Twitch Launcher จำเป็นต้องดาวน์โหลดและติดตั้งม็อดแพ็คจากที่นั่น
- [Feed The Beast](https://www.feed-the-beast.com/) - เพื่อใช้ม็อดแพ็คจาก Feed The Beast Launcher จำเป็นต้องดาวน์โหลดและติดตั้งม็อดแพ็คจากที่นั่น
- [Technic Launcher](https://www.technicpack.net/) - เพื่อใช้ม็อดแพ็คจาก Technic Launcher จำเป็นต้องดาวน์โหลดและติดตั้งม็อดแพ็คจากที่นั่น

## ปัญหาที่อาจเกิดขึ้นและวิธีแก้ไข

การเชื่อมต่อกับเซิร์ฟเวอร์เกมมักจะทำได้หลายวิธี วิธีที่มีให้เลือกจะถูกอธิบายและอธิบายอย่างละเอียดด้านล่างนี้

#### ไม่เห็นเซิร์ฟเวอร์

เซิร์ฟเวอร์ไม่แสดงอาจเกิดจากการเริ่มต้นไม่สำเร็จ อาจเป็นเพราะการตั้งค่าผิดพลาดหรือไฟล์เสียหาย ข้อมูลเพิ่มเติมมักจะพบได้ในคอนโซลเซิร์ฟเวอร์หรือไฟล์ล็อก

#### การเชื่อมต่อล้มเหลว, การเชื่อมต่อหมดเวลา
ข้อความแสดงข้อผิดพลาดนี้หมายความว่าไม่สามารถเชื่อมต่อกับเซิร์ฟเวอร์ได้ภายในเวลาที่กำหนด สาเหตุอาจมีหลายอย่าง ข้อมูลเพิ่มเติมมักจะพบได้ในล็อกหรือคอนโซลสด

#### รายการช่องม็อดไม่ตรงกัน
ข้อความแสดงข้อผิดพลาดนี้เกิดขึ้นเมื่อม็อดระหว่างไคลเอนต์กับเซิร์ฟเวอร์ไม่ตรงกัน เช่น ม็อดบางตัวหายไปหรือถูกติดตั้งสำหรับเวอร์ชันที่ต่างกัน ในกรณีนี้ควรติดตั้งม็อดใหม่และตรวจสอบทั้งไคลเอนต์และเซิร์ฟเวอร์



#### ไม่มีวิธีแก้ไขหรือไม่ช่วย

ยังเจอปัญหาอยู่หลังจากลองแก้ไขแล้ว? ในกรณีนี้ ทีมซัพพอร์ตของเราพร้อมช่วยเหลือทุกวัน แค่สร้าง **[ตั๋ว](https://zap-hosting.com/en/customer/support/)** บนเว็บไซต์ของเราและอธิบายปัญหาของคุณให้ละเอียดที่สุด เราจะรีบดูและช่วยแก้ไขปัญหาให้คุณอย่างรวดเร็ว!

## แนะนำ
การมอบสิทธิ์แอดมินบนเซิร์ฟเวอร์ Hytale ช่วยให้ผู้เล่นที่ไว้ใจได้ช่วยจัดการเซิร์ฟเวอร์ แอดมิน หรือที่เรียกว่า operator สามารถสั่งคำสั่งเซิร์ฟเวอร์ จัดการผู้เล่น และช่วยดูแลเซิร์ฟเวอร์ได้ ซึ่งช่วยให้เจ้าของเซิร์ฟเวอร์มอบหมายงานประจำได้ง่ายขึ้นและรักษาประสบการณ์การเล่นเกมให้ลื่นไหล

:::info แจ้งเตือน Early Access

Hytale เปิดตัวเมื่อวันที่ 13 มกราคม 2026 และตอนนี้ยังอยู่ในช่วง Early Access เนื่องจากเกมยังอยู่ในช่วงพัฒนาอย่างต่อเนื่อง ซอฟต์แวร์เซิร์ฟเวอร์ ไฟล์คอนฟิก การรองรับม็อด และขั้นตอนการติดตั้งอาจมีการเปลี่ยนแปลงได้เรื่อยๆ

:::

## การมอบสิทธิ์



แอดมินบนเซิร์ฟเวอร์ Hytale จะได้รับสถานะ operator ซึ่งให้สิทธิ์คำสั่งที่สูงขึ้น สามารถทำได้ผ่าน **คอนโซลเซิร์ฟเวอร์** หรือ **คำสั่งในเกม** โดย operator ที่มีอยู่แล้ว ในการมอบสิทธิ์ operator ให้ผู้เล่น ให้พิมพ์คำสั่งนี้ในคอนโซลเซิร์ฟเวอร์หรือในเกมโดยมีสิทธิ์เพียงพอ:

```
/op add <playername>
```



## การยกเลิกสิทธิ์

ถ้าผู้เล่นไม่ต้องการสิทธิ์แอดมินอีกต่อไป หรือไม่ควรมีสิทธิ์สูงขึ้น สามารถยกเลิกสถานะ operator ได้ตลอดเวลา:

```
/op remove <playername>
```

เมื่อถูกยกเลิก ผู้เล่นจะกลับไปใช้สิทธิ์ระดับปกติและจะไม่สามารถใช้คำสั่งแอดมินได้อีก



## สรุป

การเป็นแอดมินบนเซิร์ฟเวอร์ Hytale ผ่านสิทธิ์ operator เป็นส่วนสำคัญของการจัดการเซิร์ฟเวอร์ ช่วยให้ผู้เล่นที่ไว้ใจได้ช่วยดูแลและจัดการงานแอดมินต่างๆ หากมีคำถามหรืออยากได้ความช่วยเหลือ ติดต่อทีมซัพพอร์ตของเราได้เลย พร้อมช่วยเหลือทุกวัน! 🙂

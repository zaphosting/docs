## แนะนำ

การตั้งค่าสภาพอากาศใน **Over the Top WWI** มีผลโดยตรงต่อการเล่นเกม การมองเห็น และบรรยากาศ ด้วยการปรับค่าต่างๆ เหล่านี้ คุณสามารถสร้างสภาพอากาศที่ชัดเจนเพื่อการต่อสู้ที่สมดุล หรือเพิ่มความท้าทายด้วยสภาพแวดล้อมอย่างหมอก ฝน หรือพายุ

การตั้งค่าสภาพอากาศแบบกำหนดเองช่วยให้คุณออกแบบประสบการณ์โดยรวมและทำให้แมตช์มีความไดนามิกหรือสมจริงมากขึ้น

## การตั้งค่า

การตั้งค่าสภาพอากาศจะถูกกำหนดในไฟล์การตั้งค่าเซิร์ฟเวอร์ คุณสามารถเข้าถึงและแก้ไขไฟล์นี้ได้ใน **แผงควบคุมเซิร์ฟเวอร์เกม** ของคุณภายใต้เมนู **Configs** ค้นหาและเปิดไฟล์การตั้งค่า `ServerConfiguration.ini` ภายในไฟล์นี้ ให้ค้นหาพารามิเตอร์ดังต่อไปนี้:

```
EnableRandomWeather = 1
CurrentWeather = 0
CloudCoverage = 20
TimeToChangeWeather = 180
TimeToChangeClouds = 180
```

- `EnableRandomWeather` เปิดหรือปิดการเปลี่ยนแปลงสภาพอากาศแบบไดนามิก

- `0` → ปิดใช้งาน
- `1` → เปิดใช้งาน

- `CurrentWeather` กำหนดประเภทสภาพอากาศปัจจุบัน

- `0` → แจ่มใส
- `1` → ฝนตก
- `2` → หมอก
- `3` → หิมะ
- เพิ่ม `+3` สำหรับเวอร์ชันที่เข้มข้นขึ้น

- `CloudCoverage` ควบคุมปริมาณเมฆบนท้องฟ้า

- ค่าต่ำจะทำให้ท้องฟ้าแจ่มใส
- ค่าสูงจะเพิ่มความหนาแน่นของเมฆ

- `TimeToChangeWeather` กำหนดความถี่ในการเปลี่ยนสภาพอากาศเป็นวินาที

- `TimeToChangeClouds` ควบคุมความถี่ในการอัปเดตสภาพเมฆ

การตั้งค่าสภาพแวดล้อมเพิ่มเติม:

```
WindStrength = 2
WindDirection = 0
ChanceOfLightningOccuring = 100
LightningTimer = 6
CanLightningKill = 1
```

- `WindStrength` ควบคุมความแรงของลมที่มีผลต่อการเล่นเกม
- `WindDirection` กำหนดทิศทางของลม
- `ChanceOfLightningOccuring` ตั้งความน่าจะเป็นที่ฟ้าผ่าจะเกิดขึ้น
- `LightningTimer` ควบคุมความถี่ที่ฟ้าผ่าปรากฏ
- `CanLightningKill` กำหนดว่าฟ้าผ่าสามารถทำอันตรายผู้เล่นได้หรือไม่

หลังจากแก้ไขไฟล์ `ServerConfiguration.ini` ให้บันทึกไฟล์และรีสตาร์ทเซิร์ฟเวอร์ของคุณ การตั้งค่าสภาพอากาศใหม่จะถูกนำไปใช้โดยอัตโนมัติ

## สรุป

ยินดีด้วย! คุณได้ตั้งค่าสภาพอากาศบน **เซิร์ฟเวอร์เกม Over the Top WWI** ของคุณเรียบร้อยแล้ว การปรับค่าต่างๆ เหล่านี้ช่วยให้คุณสร้างสภาพแวดล้อมที่ไดนามิกและเพิ่มประสบการณ์การเล่นเกมให้กับผู้เล่นของคุณ

หากมีคำถามเพิ่มเติมหรือต้องการความช่วยเหลือ อย่าลังเลที่จะติดต่อทีมซัพพอร์ตของเรา ซึ่งพร้อมให้บริการทุกวันเพื่อช่วยเหลือคุณ! 🙂

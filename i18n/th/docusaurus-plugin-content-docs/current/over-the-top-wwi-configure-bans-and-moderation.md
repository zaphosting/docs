## แนะนำ

เครื่องมือดูแลเซิร์ฟเวอร์ใน **Over the Top WWI** ช่วยให้คุณควบคุมพฤติกรรมผู้เล่นและรักษาสภาพแวดล้อมที่ยุติธรรมและสนุกสนานบนเซิร์ฟเวอร์ของคุณ โดยการตั้งค่ารายการแบนและการตั้งค่าการดูแลเซิร์ฟเวอร์ คุณสามารถจำกัดการเข้าถึง จัดการบทลงโทษ และให้คำแนะนำที่ชัดเจนแก่ผู้เล่นได้

การตั้งค่าเหล่านี้เหมาะมากสำหรับเซิร์ฟเวอร์ชุมชนที่ต้องการการดูแลเซิร์ฟเวอร์อย่างต่อเนื่อง

## การตั้งค่า

การตั้งค่าการแบนและการดูแลเซิร์ฟเวอร์จะถูกกำหนดในไฟล์การตั้งค่าเซิร์ฟเวอร์ คุณสามารถเข้าถึงและแก้ไขไฟล์นี้ได้ใน **แผงควบคุมการจัดการเซิร์ฟเวอร์เกม** ของคุณภายใต้เมนู **Configs** ค้นหาและเปิดไฟล์การตั้งค่า `ServerConfiguration.ini` ภายในไฟล์นี้ ให้ค้นหาพารามิเตอร์ต่อไปนี้:

```
BanPathstring = Banlist.txt
TempBanPathstring = TempBanlist.txt
CustomUnBanString =
```

- `BanPathstring` กำหนดไฟล์ที่เก็บรายชื่อผู้เล่นที่ถูกแบนถาวร
- `TempBanPathstring` กำหนดไฟล์ที่เก็บรายชื่อผู้เล่นที่ถูกแบนชั่วคราว

- `CustomUnBanString` ให้คุณกำหนดข้อความแบบกำหนดเองเพื่อบอกผู้เล่นว่าต้องทำอย่างไรเพื่อขอปลดแบน เช่น ลิงก์ Discord หรือเว็บไซต์

### การใช้งานไฟล์แบน

ไฟล์แบนจะอยู่ในไดเรกทอรีเซิร์ฟเวอร์ของคุณและเก็บตัวระบุผู้เล่น

ผู้เล่นที่ถูกแบนแต่ละคนควรถูกระบุในบรรทัดใหม่ในไฟล์ดังนี้:

```
SteamID1
SteamID2
SteamID3
```

ไฟล์เหล่านี้จะถูกเซิร์ฟเวอร์อ่านโดยอัตโนมัติเพื่อบล็อกการเข้าถึงของผู้เล่นที่อยู่ในรายการ การปรับแต่งการตั้งค่าเหล่านี้ช่วยให้คุณกำหนดวิธีจัดการการแบนและวิธีที่ผู้เล่นสามารถอุทธรณ์ได้

หลังจากแก้ไขไฟล์ `ServerConfiguration.ini` ให้บันทึกไฟล์และรีสตาร์ทเซิร์ฟเวอร์ของคุณ การตั้งค่าการดูแลเซิร์ฟเวอร์ที่อัปเดตจะถูกนำไปใช้โดยอัตโนมัติ

## สรุป

ยินดีด้วย! คุณได้ตั้งค่าการแบนและการดูแลเซิร์ฟเวอร์บน **เซิร์ฟเวอร์ Over the Top WWI** ของคุณเรียบร้อยแล้ว ซึ่งช่วยให้คุณจัดการพฤติกรรมผู้เล่นได้อย่างมีประสิทธิภาพและรักษาสภาพแวดล้อมเซิร์ฟเวอร์ที่ควบคุมได้

หากมีคำถามเพิ่มเติมหรือต้องการความช่วยเหลือ อย่าลังเลที่จะติดต่อทีมซัพพอร์ตของเรา ซึ่งพร้อมให้บริการคุณทุกวัน! 🙂

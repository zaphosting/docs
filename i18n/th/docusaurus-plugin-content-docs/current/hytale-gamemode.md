## แนะนำ

เซิร์ฟเวอร์ Hytale ให้คุณตั้งค่าการเล่นเกม เช่น โหมดเกมที่กำหนดประสบการณ์ของผู้เล่น การเปลี่ยนโหมดเกมช่วยให้คุณปรับแต่งเซิร์ฟเวอร์ให้ตรงกับสไตล์การเล่นหรือความชอบของชุมชนได้

:::info แจ้งเตือน Early Access

Hytale เปิดตัวเมื่อวันที่ 13 มกราคม 2026 และตอนนี้อยู่ในช่วง Early Access เนื่องจากเกมยังอยู่ในช่วงพัฒนาอย่างต่อเนื่อง ซอฟต์แวร์เซิร์ฟเวอร์ ไฟล์การตั้งค่า การรองรับม็อด และขั้นตอนการติดตั้งอาจมีการเปลี่ยนแปลงไปตามเวลา

:::

## เปลี่ยนโหมดเกม

### คอนโซลสด
โหมดเกมสามารถเปลี่ยนได้ขณะเซิร์ฟเวอร์กำลังทำงานผ่านคอนโซลสดหรือคำสั่งในเกม โดยต้องมีสิทธิ์ที่จำเป็น วิธีนี้ช่วยให้แอดมินเปลี่ยนโหมดเกมได้โดยไม่ต้องแก้ไขไฟล์การตั้งค่า

การใช้คำสั่งคอนโซลจะอัปเดตโหมดเกมที่ใช้งานทันทีสำหรับเซสชันเซิร์ฟเวอร์นั้น ขึ้นอยู่กับการตั้งค่าเซิร์ฟเวอร์ การเปลี่ยนอาจคงอยู่จนกว่าจะรีสตาร์ทครั้งถัดไป หรือกลับไปใช้ค่าที่กำหนดในไฟล์การตั้งค่า

```
/gamemode <gamemode> <playername>
```

ค่าของ `GameMode` กำหนดว่าโหมดเกมใดจะถูกใช้เมื่อเซิร์ฟเวอร์เริ่มทำงาน ขึ้นอยู่กับประสบการณ์การเล่นที่คุณต้องการ ค่านี้สามารถปรับได้ตามต้องการ

โหมดเกมที่ใช้บ่อยได้แก่:

- `Creative` สำหรับการสร้างและเนื้อหาแบบไม่จำกัด
- `Adventure` สำหรับการเล่นเกมที่มีโครงสร้างและกฎการเล่นที่ชัดเจน

หลังจากเปลี่ยนค่าโหมดเกมแล้ว ต้องรีสตาร์ทเซิร์ฟเวอร์เพื่อให้การตั้งค่าใหม่มีผล



### ไฟล์การตั้งค่า

โหมดเกมสามารถกำหนดได้โดยตรงในไฟล์การตั้งค่าเซิร์ฟเวอร์
วิธีนี้แนะนำเมื่อคุณต้องการให้เซิร์ฟเวอร์เริ่มต้นด้วยโหมดเกมเฉพาะเสมอ หรือเมื่อเตรียมเซิร์ฟเวอร์ใหม่

ไฟล์การตั้งค่าสามารถเข้าถึงได้ผ่านแผงควบคุมโฮสติ้งหรือไฟล์แมเนเจอร์ ในไฟล์นี้จะมีการตั้งค่าที่ควบคุมโหมดเกมที่ใช้งานเมื่อเซิร์ฟเวอร์เริ่มทำงาน หลังจากแก้ไขค่าแล้ว ต้องรีสตาร์ทเซิร์ฟเวอร์เพื่อให้การเปลี่ยนแปลงมีผล

```
"Defaults": {
"World": "default",
"GameMode": "Adventure"
},
```

ค่าของ `GameMode` กำหนดว่าโหมดเกมใดจะถูกใช้เมื่อเซิร์ฟเวอร์เริ่มทำงาน ขึ้นอยู่กับประสบการณ์การเล่นที่คุณต้องการ ค่านี้สามารถปรับได้ตามต้องการ

โหมดเกมที่ใช้บ่อยได้แก่:

- `Creative` สำหรับการสร้างและเนื้อหาแบบไม่จำกัด
- `Adventure` สำหรับการเล่นเกมที่มีโครงสร้างและกฎการเล่นที่ชัดเจน

หลังจากเปลี่ยนค่าโหมดเกมแล้ว ต้องรีสตาร์ทเซิร์ฟเวอร์เพื่อให้การตั้งค่าใหม่มีผล



## สรุป

โดยการตั้งค่าโหมดเกมผ่านไฟล์การตั้งค่าเซิร์ฟเวอร์หรือคอนโซลสด คุณสามารถควบคุมประสบการณ์การเล่นบนเซิร์ฟเวอร์ Hytale ของคุณได้อย่างแม่นยำ หากมีคำถามหรือขอความช่วยเหลือเพิ่มเติม ติดต่อทีมซัพพอร์ตของเราได้เลย พร้อมช่วยเหลือคุณทุกวัน! 🙂

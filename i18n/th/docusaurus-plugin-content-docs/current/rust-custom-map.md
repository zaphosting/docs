## แนะนำ

แผนที่แบบกำหนดเองช่วยให้เจ้าของเซิร์ฟเวอร์ Rust สร้างประสบการณ์การเล่นเกมที่ไม่เหมือนใคร ซึ่งเกินกว่าการใช้แผนที่ Procedural ปกติ หลายชุมชนใช้แผนที่แบบกำหนดเองเพื่อเพิ่มอนุสาวรีย์ใหม่ ๆ ปรับเปลี่ยนภูมิประเทศ จัดกิจกรรมพิเศษ หรือออกแบบสภาพแวดล้อมใหม่ทั้งหมด

แผนที่เหล่านี้มักถูกสร้างด้วยเครื่องมืออย่าง **RustEdit** และแจกจ่ายผ่านแพลตฟอร์มเช่น **Lone Design**, **Codefling** หรือที่เก็บแผนที่ของชุมชน เมื่อคุณมีไฟล์แผนที่หรือ URL ดาวน์โหลดแผนที่แล้ว ก็สามารถตั้งค่าได้โดยตรงในเซิร์ฟเวอร์ของคุณ

## การเตรียมตัว

ก่อนตั้งค่าแผนที่แบบกำหนดเอง คุณต้องมี **URL ดาวน์โหลดแผนที่** ที่ถูกต้อง แผนที่แบบกำหนดเองมักจะถูกโฮสต์บนแพลตฟอร์มแจกจ่ายแผนที่หรือมาจากผู้สร้างแผนที่

URL นี้โดยปกติจะชี้ไปยังไฟล์ `.map` หรือแพ็กเกจแผนที่ที่ดาวน์โหลดได้ ตรวจสอบให้แน่ใจว่า URL นั้นเข้าถึงได้สาธารณะ เพื่อให้เซิร์ฟเวอร์ Rust สามารถดาวน์โหลดได้ตอนเริ่มต้น ตัวอย่าง URL แผนที่:

```
https://example-map-host.com/maps/custommap.map
```

## การตั้งค่า

เพื่อกำหนดค่าแผนที่แบบกำหนดเองบนเซิร์ฟเวอร์ Rust ของคุณ ให้เปิด **แผงควบคุมการจัดการเซิร์ฟเวอร์เกม** แล้วไปที่ **การตั้งค่า**

ค้นหาช่องตั้งค่าที่ชื่อว่า **Level URL** การตั้งค่านี้จะช่วยให้เซิร์ฟเวอร์ดาวน์โหลดและโหลดแผนที่ Rust แบบกำหนดเองตอนเริ่มต้น วาง URL แผนที่แบบกำหนดเองของคุณลงในช่อง **Level URL** เช่น:

```
https://example-map-host.com/maps/custommap.map
```

เมื่อใส่ URL แล้ว ให้บันทึกการตั้งค่าในแผงควบคุม เมื่อเซิร์ฟเวอร์เริ่มทำงาน Rust จะดาวน์โหลดแผนที่จาก URL ที่ให้ไว้และโหลดแผนที่นั้นแทนการสร้างแผนที่ Procedural

ถ้าแผนที่แบบกำหนดเองของคุณต้องการพารามิเตอร์การตั้งค่าเพิ่มเติม (เช่น seed เฉพาะ หรือขนาดโลก) โดยปกติจะมีให้จากผู้สร้างแผนที่ และอาจต้องเพิ่มเข้าไปในการตั้งค่าเซิร์ฟเวอร์ด้วย

หลังจากบันทึกการตั้งค่าแล้ว **รีสตาร์ทเซิร์ฟเวอร์** เพื่อให้แผนที่แบบกำหนดเองถูกดาวน์โหลดและโหลด

## หมายเหตุสำคัญ

แผนที่แบบกำหนดเองต้องเข้ากันได้กับเวอร์ชันเซิร์ฟเวอร์ Rust ที่คุณใช้งาน หากไฟล์แผนที่เก่าหรือเสียหาย เซิร์ฟเวอร์อาจไม่สามารถเริ่มทำงานได้ หรือจะกลับไปใช้แผนที่ Procedural แทน

แผนที่แบบกำหนดเองขนาดใหญ่ก็อาจทำให้เวลาเริ่มต้นเซิร์ฟเวอร์นานขึ้น เพราะเซิร์ฟเวอร์ต้องดาวน์โหลดและเตรียมแผนที่ก่อนที่ผู้เล่นจะเข้าเล่นได้

ถ้าผู้เล่นไม่สามารถเชื่อมต่อหลังเปลี่ยนแผนที่ ให้ตรวจสอบว่า URL แผนที่ถูกต้องและเข้าถึงได้จากอินเทอร์เน็ต

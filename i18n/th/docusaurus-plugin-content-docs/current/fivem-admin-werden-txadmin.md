# การเป็นแอดมิน FiveM ด้วย txAdmin

## แนะนำ  
ในคู่มือนี้ คุณจะได้เรียนรู้วิธีการเป็นแอดมินบนเซิร์ฟเวอร์เกม FiveM โดยใช้ txAdmin ซึ่งรวมถึงขั้นตอนที่จำเป็นในการตั้งค่าสิทธิ์แอดมินผ่านแผงควบคุม txAdmin

## ความต้องการ  
- การเข้าถึงเซิร์ฟเวอร์เกม FiveM ของคุณผ่านเว็บอินเทอร์เฟซหรือแผงควบคุมของ ZAP-Hosting  
- ต้องติดตั้งและรัน txAdmin บนเซิร์ฟเวอร์เกม FiveM ของคุณแล้ว  
- มีความเข้าใจพื้นฐานเกี่ยวกับการจัดการเซิร์ฟเวอร์เกม FiveM  

## คู่มือทีละขั้นตอน

### 1. เข้าถึงแผงควบคุม txAdmin  
1. ล็อกอินเข้าสู่บัญชี ZAP-Hosting ของคุณ  
2. ไปที่แดชบอร์ดเซิร์ฟเวอร์เกม FiveM ของคุณ  
3. คลิกที่ปุ่ม **เปิด txAdmin** เพื่อเปิดอินเทอร์เฟซ txAdmin  
   
   ![Open txAdmin](img/fivem-admin-txadmin/open-txadmin.png)  

### 2. เข้าสู่ระบบ txAdmin  
1. หากเป็นครั้งแรกที่คุณเข้าถึง txAdmin ระบบจะให้คุณสร้างบัญชีแอดมิน  
2. กรอกชื่อผู้ใช้และรหัสผ่านที่ต้องการ จากนั้นบันทึก  
3. หากมีบัญชีอยู่แล้ว ให้กรอกข้อมูลล็อกอินเพื่อเข้าสู่ระบบ  

### 3. เพิ่มสิทธิ์แอดมินของคุณ  
1. ในแดชบอร์ด txAdmin ให้ไปที่แท็บ **Players**  
2. ค้นหาผู้เล่นของคุณโดยใช้ Steam ID หรือชื่อในเกม  
3. คลิกปุ่ม **Manage** ข้างรายการผู้เล่นของคุณ  
4. ในส่วนสิทธิ์ ให้กำหนดบทบาทเป็น **admin** หรือสิทธิ์ที่จำเป็นตามต้องการ  
   
   ![Assign Admin Role](img/fivem-admin-txadmin/assign-admin-role.png)  

### 4. ตั้งค่าสิทธิ์ (ถ้าต้องการ)  
1. ไปที่แท็บ **Settings** ใน txAdmin  
2. เลือก **Permissions** เพื่อปรับแต่งคำสั่งและสิทธิ์เฉพาะสำหรับแอดมิน  
3. ปรับแต่งตามความต้องการแล้วบันทึกการเปลี่ยนแปลง  

### 5. บันทึกและรีสตาร์ทเซิร์ฟเวอร์  
1. หลังจากกำหนดสิทธิ์แล้ว กลับไปที่แท็บ **Dashboard**  
2. คลิกที่ **Restart Server** เพื่อให้การเปลี่ยนแปลงมีผล  
   
   ```bash
   # หรือจะรีสตาร์ทผ่านแผงควบคุม ZAP-Hosting ก็ได้:
   # 1. ไปที่แดชบอร์ดเซิร์ฟเวอร์ของคุณ
   # 2. คลิก "Restart"
   ```  

### 6. ตรวจสอบสถานะแอดมินในเกม  
1. เข้าร่วมเซิร์ฟเวอร์เกม FiveM ของคุณ  
2. เปิดคอนโซลในเกม (โดยปกติใช้ปุ่ม F8)  
3. ทดสอบคำสั่งแอดมิน เช่น:  
   
   ```plaintext
   /admin
   /kick [playerID]
   /ban [playerID]
   ```  
   
4. ถ้าคำสั่งทำงานได้ แสดงว่าคุณมีสิทธิ์แอดมินแล้ว  

> **หมายเหตุ:** หากเจอปัญหา ให้ตรวจสอบ Steam ID หรือการตั้งค่า whitelist ของเซิร์ฟเวอร์ใน txAdmin อีกครั้ง

## สรุป  
เมื่อทำตามขั้นตอนนี้เสร็จสิ้น คุณก็จะกลายเป็นแอดมินบนเซิร์ฟเวอร์เกม FiveM ของคุณด้วย txAdmin ได้อย่างสมบูรณ์ พร้อมจัดการเซิร์ฟเวอร์ได้อย่างมีประสิทธิภาพด้วยสิทธิ์แอดมินเต็มรูปแบบ

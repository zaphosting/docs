## แนะนำ
อยากสร้างโลกของคุณล่วงหน้าอย่างรวดเร็ว มีประสิทธิภาพ และปลอดภัย แทนที่จะสร้างระหว่างเล่นเกมไหม? Minecraft ไม่มีฟังก์ชันนี้มาให้โดยตรง แต่สามารถทำได้ด้วยปลั๊กอินหรือม็อดที่ออกแบบมาเฉพาะ วิธีใช้และสิ่งที่ควรพิจารณาจะอธิบายอย่างละเอียดด้านล่างนี้



## การเตรียมตัว

อันดับแรก คุณต้องมั่นใจว่าเซิร์ฟเวอร์ของคุณรองรับการใช้ปลั๊กอินหรือม็อด โดยต้องใช้ Minecraft เวอร์ชันใดเวอร์ชันหนึ่งต่อไปนี้:

- Minecraft: Forge, Minecraft Fabric (ม็อด)
- Minecraft: Spigot, Minecraft Bukkit, Minecraft: Paper Spigot (ปลั๊กอิน)

ถ้าคุณยังไม่ได้ใช้ Minecraft เวอร์ชันเหล่านี้ คุณสามารถใช้ตัวเลือก [เปลี่ยนเกม](gameserver-gameswitch.md) เพื่อเลือกเกมที่ต้องการได้


## การติดตั้ง
วิธีที่นิยมที่สุดในการสร้างชิ้นส่วนล่วงหน้าคือการใช้ [Chunky](https://github.com/pop4959/Chunky) ซึ่งมีให้สำหรับเซิร์ฟเวอร์หลายแบบทั้งในรูปแบบปลั๊กอินและม็อด ในตัวอย่างนี้จะอธิบายและสาธิตการสร้างชิ้นส่วนด้วย Chunky

| เวอร์ชัน | ดาวน์โหลด |
| -------- | ------------------------------------------------------------ |
| ม็อด | [Forge](https://www.curseforge.com/minecraft/mc-mods/chunky-pregenerator-forge) [Fabric](https://www.curseforge.com/minecraft/mc-mods/chunky-pregenerator) |
| ปลั๊กอิน | [Spigot/Bukkit](https://www.spigotmc.org/resources/chunky.81534/) |

ดาวน์โหลด **Chunky** ในรูปแบบปลั๊กอินหรือม็อดและติดตั้งบนเซิร์ฟเวอร์ของคุณ หากไม่แน่ใจวิธีติดตั้ง [ปลั๊กอิน](minecraft-pluginuploader.md) หรือ [ม็อด](minecraft-forge-fabric-add-mods-modpacks) คู่มือที่เกี่ยวข้องจะช่วยคุณได้



## การใช้งาน

เมื่อคุณติดตั้ง **Chunky** เรียบร้อยแล้ว คุณก็สามารถเริ่มสร้างชิ้นส่วนได้เลย เริ่มด้วยคำสั่ง `chunky start`

ถ้าต้องการหยุดชั่วคราวและกลับมาทำต่อทีหลัง ใช้คำสั่ง `chunky pause` และ `chunky continue` เพื่อดำเนินการต่อ หากต้องการยกเลิกการสร้างทั้งหมด ใช้คำสั่ง `chunky cancel` คำสั่งเหล่านี้สามารถใช้ผ่านคอนโซลสดหรือในเกมโดยผู้ดูแลเซิร์ฟเวอร์ได้เลย

:::warning โหลดเกินพิกัด
การใช้งานทรัพยากรเกินขีดจำกัดเป็นเวลานานอาจทำให้เซิร์ฟเวอร์ถูกระงับชั่วคราว เพื่อป้องกัน ควรสร้างชิ้นส่วนด้วยความเร็วและขนาดที่สมดุล เพื่อให้โหลดระบบอยู่ในระดับที่รับได้

ข้อมูลเพิ่มเติมเกี่ยวกับขีดจำกัดทรัพยากรและวิธีแก้ไขดูได้ใน [คู่มือขีดจำกัดทรัพยากร](gameserver-resourcelimit.md) 
:::



## คำสั่ง

ส่วนนี้รวบรวมคำสั่งทั้งหมดที่ Chunky ใช้สำหรับสร้างและจัดการชิ้นส่วน

#### การจัดการงาน

| คำสั่ง         | คำอธิบาย                                                  |
| --------------- | ------------------------------------------------------------ |
| chunky start    | เริ่มงานสร้างชิ้นส่วนใหม่จากการเลือกปัจจุบัน |
| chunky pause    | หยุดชั่วคราวงานสร้างชิ้นส่วนและบันทึกความคืบหน้า |
| chunky continue | ดำเนินงานสร้างชิ้นส่วนที่กำลังทำหรือบันทึกไว้ต่อ |
| chunky cancel   | หยุดงานสร้างชิ้นส่วนและยกเลิกความคืบหน้า |



#### การเลือก

| คำสั่ง                                    | คำอธิบาย                                                  |
| ------------------------------------------ | ------------------------------------------------------------ |
| chunky world [world]                       | ตั้งค่าโลกที่เลือกปัจจุบัน                            |
| chunky shape `<shape>`                     | ตั้งค่ารูปร่างที่จะสร้าง                                    |
| chunky center [`<x>` `<z>`]                | ตั้งค่าตำแหน่งบล็อกศูนย์กลางปัจจุบัน                       |
| chunky radius `<radius>`                   | ตั้งค่ารัศมีปัจจุบัน                                      |
| chunky worldborder                         | ตั้งค่าศูนย์กลางและรัศมีให้ตรงกับขอบโลกในโลกที่เลือก |
| chunky spawn                               | ตั้งค่าศูนย์กลางเป็นจุดเกิด                            |
| chunky corners `<x1>` `<z1>` `<x2>` `<z2>` | ตั้งค่าการเลือกโดยพิกัดมุม                      |
| chunky pattern `<pattern>`                 | ตั้งค่ารูปแบบการสร้างที่ต้องการ                         |
| chunky slection                            | แสดงการเลือกปัจจุบัน                                |





#### อื่นๆ

| คำสั่ง                   | คำอธิบาย                                           |
| ------------------------- | ----------------------------------------------------- |
| chunky silent             | สลับการแสดงข้อความอัปเดต                     |
| chunky quiet `<interval>` | ตั้งค่าช่วงเวลาที่ไม่แสดงข้อความอัปเดต (วินาที) |
| chunky progress           | แสดงความคืบหน้าการสร้างล่วงหน้าในเกมสำหรับทุกงาน |
| chunky reload             | โหลดการตั้งค่าใหม่                             |
| chunky trim               | ลบชิ้นส่วนที่อยู่นอกการเลือก                       |



## สรุป

เมื่อทำตามขั้นตอนนี้ คุณได้ติดตั้ง Chunky เรียบร้อยและสามารถสร้างโลก (ชิ้นส่วน) ล่วงหน้าได้แล้ว ยินดีด้วย! ถ้าทำตามครบทุกขั้นตอน คุณควรเชื่อมต่อกับเซิร์ฟเวอร์ได้สำเร็จ หากมีคำถามหรืออยากได้ความช่วยเหลือ ติดต่อทีมซัพพอร์ตของเราได้เลย พร้อมช่วยเหลือทุกวัน! 🙂

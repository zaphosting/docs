## แนะนำ

ระบบโพลใน **Over the Top WWI** ช่วยให้ผู้เล่นสามารถโหวตในหลายๆ การกระทำ เช่น การเตะผู้เล่น, เปลี่ยนแผนที่ หรือปรับแต่งการตั้งค่าเกม ฟีเจอร์นี้ช่วยเพิ่มการมีส่วนร่วมของชุมชนและลดความจำเป็นในการแทรกแซงของแอดมินบ่อยๆ

โดยการตั้งค่าระบบโพล คุณสามารถควบคุมได้ว่าการโหวตแบบไหนที่อนุญาตและวิธีการทำงานของมันในระหว่างการเล่นเกม

## การตั้งค่า

ระบบโพลจะถูกตั้งค่าภายในไฟล์การตั้งค่าเซิร์ฟเวอร์ คุณสามารถเข้าถึงและแก้ไขไฟล์นี้ได้ใน **แผงควบคุมการจัดการเซิร์ฟเวอร์เกม** ของคุณภายใต้เมนู **Configs** ค้นหาและเปิดไฟล์การตั้งค่า `ServerConfiguration.ini` ภายในไฟล์นี้ ให้ค้นหาพารามิเตอร์ดังต่อไปนี้:

```
PollPercentThresh = 51
EnableBanPlayerPoll = 0
EnableKickPlayerPoll = 0
EnableChangeMapsPoll = 0
EnableMutePlayerPoll = 0
EnableCustomPoll = 1
ChangeMapImmediately = 1
```

- `PollPercentThresh` กำหนดเปอร์เซ็นต์ของคะแนนโหวตที่ต้องการให้โพลผ่าน
- `EnableBanPlayerPoll` อนุญาตให้ผู้เล่นโหวตแบนผู้เล่นอื่น
- `EnableKickPlayerPoll` อนุญาตให้ผู้เล่นโหวตเตะผู้เล่นอื่น
- `EnableChangeMapsPoll` อนุญาตให้ผู้เล่นโหวตเปลี่ยนแผนที่
- `EnableMutePlayerPoll` อนุญาตให้ผู้เล่นโหวตปิดเสียงผู้เล่นอื่น
- `EnableCustomPoll` เปิดใช้งานตัวเลือกโหวตแบบกำหนดเอง
- `ChangeMapImmediately` กำหนดว่าการโหวตเปลี่ยนแผนที่ที่สำเร็จจะถูกใช้ทันทีหรือไม่

โหมดเกมที่อนุญาตให้โหวต:

```
PollBattle = 1
PollTDM = 1
PollConquest = 1
PollSiege = 1
PollKingoftheHill = 1
PollConvoyAmbush = 1
PollCaptureTheFlag = 1
PollCommanderBattle = 1
```

การตั้งค่าเหล่านี้กำหนดว่าโหมดเกมไหนบ้างที่สามารถเลือกได้ในระหว่างโพล

- `1` → เปิดใช้งาน
- `0` → ปิดใช้งาน

คุณสามารถปิดใช้งานโหมดเฉพาะเพื่อจำกัดตัวเลือกการโหวตและรักษาประสบการณ์การเล่นเกมให้สม่ำเสมอ

หลังจากแก้ไขไฟล์ `ServerConfiguration.ini` ให้บันทึกไฟล์และรีสตาร์ทเซิร์ฟเวอร์ของคุณ การตั้งค่าระบบโพลที่อัปเดตจะถูกนำไปใช้โดยอัตโนมัติ

## สรุป

ยินดีด้วย! คุณตั้งค่าระบบโพลบน **เซิร์ฟเวอร์ Over the Top WWI** ของคุณสำเร็จแล้ว ซึ่งช่วยให้ผู้เล่นมีส่วนร่วมในการตัดสินใจและช่วยให้การจัดการเซิร์ฟเวอร์เป็นไปโดยอัตโนมัติ

หากมีคำถามเพิ่มเติมหรือต้องการความช่วยเหลือ อย่าลังเลที่จะติดต่อทีมซัพพอร์ตของเรา ซึ่งพร้อมให้บริการทุกวันเพื่อช่วยเหลือคุณ! 🙂

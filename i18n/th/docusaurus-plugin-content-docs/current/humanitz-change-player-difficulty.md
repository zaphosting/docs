## แนะนำ

ความยากของผู้เล่นใน HumanitZ กำหนดว่าประสบการณ์การเอาชีวิตรอดจะโหดขนาดไหน ตั้งแต่การตายถาวรไปจนถึงความเร็วในการลดค่าพลังชีวิตและความแข็งแกร่งของโจร การตั้งค่าเหล่านี้ส่งผลโดยตรงต่อความท้าทายของเซิร์ฟเวอร์คุณ

## การตั้งค่า

การตั้งค่าความยากของผู้เล่นถูกควบคุมในไฟล์ `GameServerSettings.ini` คุณสามารถเข้าถึงไฟล์นี้ได้ในแผงควบคุมเซิร์ฟเวอร์ของคุณภายใต้ **Configs** ในไฟล์คอนฟิก ให้ค้นหาพารามิเตอร์ดังต่อไปนี้:

````
;เมื่อตั้งค่าเป็น true ผู้เล่นที่ตายจะสูญเสียตัวละครและต้องสร้างใหม่
PermaDeath=false

;0=ไม่เสียอะไร, 1=เสียกระเป๋าเป้และอาวุธในมือ, 2=ข้อก่อนหน้า + กระเป๋าเสื้อและกระเป๋าเป้, 3=ทั้งหมดก่อนหน้า + อุปกรณ์
OnDeath=2

;เอฟเฟกต์การอดนอน false=ปิดใช้งาน true=เปิดใช้งาน
Sleep=true

;ความเร็วในการลดค่าพลังชีวิต 0=ช้า, 1=ปกติ, 2=เร็ว
VitalDrain=1

;พลังชีวิตของสุนัขคู่หู 0=ต่ำ 1=ค่าเริ่มต้น 2=สูง
CompanionHealth=1

;ความเสียหายของสุนัขคู่หู 0=ต่ำ 1=ค่าเริ่มต้น 2=สูง
CompanionDmg=1

;เวลาที่ใช้ในการเกิดใหม่ของโจรมนุษย์เป็นนาที ตั้งค่าเป็น 0 เพื่อปิดการเกิดใหม่ของโจรมนุษย์
HumanRespawnTimer=90

;ความยากของพลังชีวิตโจรมนุษย์ 0=ง่ายมาก, 1=ง่าย, 2=ค่าเริ่มต้น, 3=ยาก, 4=ยากมาก, 5=ฝันร้าย
HumanHealth=2

;ความยากของความเร็วโจรมนุษย์ 0=ง่ายมาก, 1=ง่าย, 2=ค่าเริ่มต้น, 3=ยาก, 4=ยากมาก, 5=ฝันร้าย
HumanSpeed=2

;ความยากของความเสียหายโจรมนุษย์ 0=ง่ายมาก, 1=ง่าย, 2=ค่าเริ่มต้น, 3=ยาก, 4=ยากมาก, 5=ฝันร้าย
HumanDamage=2
````

ด้านล่างนี้คือคำอธิบายของแต่ละการตั้งค่า:

- `PermaDeath` – เมื่อกำหนดเป็น `true` ผู้เล่นจะสูญเสียตัวละครถาวรเมื่อเสียชีวิตและต้องสร้างใหม่  
- `OnDeath` – กำหนดว่าผู้เล่นจะเสียไอเท็มในอินเวนทอรี่มากแค่ไหนเมื่อเสียชีวิต  
- `Sleep` – เปิดหรือปิดเอฟเฟกต์การอดนอน  
- `VitalDrain` – ควบคุมความเร็วในการลดค่าความหิว กระหาย และค่าพลังชีวิตอื่นๆ  
- `CompanionHealth` – กำหนดระดับพลังชีวิตของสุนัขคู่หู  
- `CompanionDmg` – กำหนดระดับความเสียหายของสุนัขคู่หู  
- `HumanRespawnTimer` – กำหนดเวลาที่ใช้ในการเกิดใหม่ของโจรมนุษย์ (0 ปิดการเกิดใหม่)  
- `HumanHealth` – ปรับความยากของพลังชีวิตโจร  
- `HumanSpeed` – ปรับความยากของความเร็วโจร  
- `HumanDamage` – ปรับความยากของความเสียหายโจร  

แก้ไขค่าตามระดับความยากที่คุณต้องการ หลังจากแก้ไขไฟล์แล้ว ให้บันทึกและรีสตาร์ทเซิร์ฟเวอร์เพื่อให้การตั้งค่าใหม่มีผล



## ตัวอย่างการตั้งค่าความยาก

### การตั้งค่าเอาชีวิตรอดแบบง่าย

การตั้งค่านี้จะสร้างประสบการณ์ที่ผ่อนคลายมากขึ้น ด้วยการลดความเร็วการลดค่าพลังชีวิต ศัตรูที่อ่อนแอ และไม่มีการตายถาวร

```
PermaDeath=false
OnDeath=1
Sleep=false
VitalDrain=0
CompanionHealth=2
CompanionDmg=2
HumanRespawnTimer=180
HumanHealth=1
HumanSpeed=1
HumanDamage=1
```



### การตั้งค่าเอาชีวิตรอดแบบยาก

การตั้งค่านี้เพิ่มความกดดันในการเอาชีวิตรอดด้วยการตายถาวร ความเร็วการลดค่าพลังชีวิตที่เร็วขึ้น และโจรที่แข็งแกร่งขึ้น

```
PermaDeath=true
OnDeath=3
Sleep=true
VitalDrain=2
CompanionHealth=0
CompanionDmg=0
HumanRespawnTimer=60
HumanHealth=4
HumanSpeed=4
HumanDamage=4
```



## สรุป

ยินดีด้วย! โดยการปรับค่าที่เกี่ยวกับความยากในไฟล์ `GameServerSettings.ini` คุณได้ปรับแต่งการตั้งค่าความยากของผู้เล่นบนเซิร์ฟเวอร์ HumanitZ ของคุณเรียบร้อยแล้ว หากมีคำถามเพิ่มเติมหรือต้องการความช่วยเหลือ อย่าลังเลที่จะติดต่อทีมซัพพอร์ตของเรา ซึ่งพร้อมให้บริการทุกวันเพื่อช่วยคุณ! 🙂

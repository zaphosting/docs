## แนะนำ
Whitelist คือฟีเจอร์ด้านความปลอดภัยและการควบคุมการเข้าถึงที่ช่วยให้คุณจำกัดว่าใครสามารถเข้าร่วมเซิร์ฟเวอร์ Hytale ของคุณได้ เมื่อเปิดใช้งาน จะอนุญาตเฉพาะผู้เล่นที่ถูกระบุไว้ใน whitelist เท่านั้นที่สามารถเชื่อมต่อได้ ฟีเจอร์นี้เหมาะมากสำหรับเซิร์ฟเวอร์ส่วนตัว, เซิร์ฟเวอร์ชุมชน, สภาพแวดล้อมการพัฒนา หรือช่วงทดสอบที่ต้องการจำกัดการเข้าถึงเฉพาะผู้เล่นที่เชื่อถือได้เท่านั้น

## เปิดใช้งาน whitelist

คุณสามารถเปิดใช้งาน whitelist ได้ทั้งผ่านคอนโซลเซิร์ฟเวอร์หรือในเกมโดยตรง ตรวจสอบให้แน่ใจว่าคุณมีสิทธิ์ที่จำเป็น เช่น สิทธิ์ operator ก่อนรันคำสั่งเหล่านี้

เพื่อเปิดใช้งาน whitelist ให้ใช้คำสั่งนี้:

```
/whitelist enable
```



เมื่อเปิดใช้งานแล้ว ผู้เล่นที่ไม่ได้อยู่ใน whitelist จะถูกปฏิเสธการเข้าร่วมเซิร์ฟเวอร์ทันทีเมื่อพยายามเชื่อมต่อ เพื่อตรวจสอบว่า whitelist เปิดใช้งานอยู่หรือไม่ ให้ใช้คำสั่ง:

```
/whitelist status
```

คำสั่งนี้จะแสดงสถานะปัจจุบันของ whitelist และช่วยยืนยันว่าการตั้งค่าได้ถูกนำไปใช้เรียบร้อยแล้ว



## ปิดใช้งาน whitelist

ถ้าคุณต้องการให้ผู้เล่นทุกคนสามารถเข้าร่วมเซิร์ฟเวอร์ได้อีกครั้ง คุณสามารถปิด whitelist ได้ตลอดเวลา ใช้คำสั่งนี้เพื่อปิด:

```
/whitelist disable
```

หลังจากปิด whitelist เซิร์ฟเวอร์จะไม่จำกัดการเข้าถึงตาม whitelist อีกต่อไป และผู้เล่นทุกคนจะสามารถเชื่อมต่อได้



## จัดการ whitelist

คุณสามารถจัดการ whitelist ได้แบบไดนามิกขณะเซิร์ฟเวอร์กำลังทำงานอยู่ เพิ่มหรือลบผู้เล่นได้ตลอดเวลาโดยไม่ต้องรีสตาร์ทเซิร์ฟเวอร์ด้วยคำสั่งต่างๆ



### เพิ่มผู้เล่นใน whitelist

เพื่ออนุญาตให้ผู้เล่นคนใดเข้าร่วมเซิร์ฟเวอร์ ให้เพิ่มชื่อผู้เล่นลงใน whitelist ด้วยคำสั่ง:

```
/whitelist add <playername>
```

ผู้เล่นที่ระบุจะได้รับอนุญาตให้เข้าร่วมเซิร์ฟเวอร์ทันที หาก whitelist เปิดใช้งานอยู่



### ลบผู้เล่นออกจาก whitelist

ถ้าต้องการยกเลิกสิทธิ์การเข้าร่วมของผู้เล่น ให้ลบชื่อผู้เล่นออกจาก whitelist ด้วยคำสั่ง:

```
/whitelist remove <playername>
```

เมื่อถูกลบ ผู้เล่นคนนั้นจะไม่สามารถเข้าร่วมเซิร์ฟเวอร์ได้อีกต่อไปในขณะที่ whitelist ยังเปิดใช้งาน



### แสดงรายชื่อผู้เล่นใน whitelist

ถ้าต้องการดูรายชื่อผู้เล่นทั้งหมดที่ได้รับอนุญาตให้เข้าร่วมเซิร์ฟเวอร์ ให้ใช้คำสั่ง:

```
/whitelist list
```

คำสั่งนี้จะแสดงรายชื่อผู้เล่นทั้งหมดที่อยู่ใน whitelist



### ล้าง whitelist

ถ้าคุณต้องการลบผู้เล่นทั้งหมดออกจาก whitelist ในครั้งเดียว คุณสามารถล้าง whitelist ได้ด้วยคำสั่ง:

```
/whitelist clear
```

คำสั่งนี้จะลบข้อมูล whitelist ทั้งหมด แต่จะไม่ปิดการใช้งาน whitelist เอง



## จัดการ whitelist ผ่านไฟล์

นอกจากคำสั่งแล้ว คุณยังสามารถจัดการ whitelist ได้โดยตรงผ่านไฟล์การตั้งค่า วิธีนี้เหมาะสำหรับการเปลี่ยนแปลงจำนวนมากหรือเตรียม whitelist ก่อนเริ่มเซิร์ฟเวอร์ ไฟล์ที่ใช้จัดการคือ `whitelist.json` ซึ่งอยู่ในโฟลเดอร์เซิร์ฟเวอร์

````
{
  "enabled": true,
  "players": [
    "PlayerName1",
    "PlayerName2"
  ]
}
````

## แนะนำ

เซิร์ฟเวอร์เกม Hytale มีคำสั่งมากมายที่ช่วยให้แอดมินจัดการการเล่นเกม, ผู้เล่น, โลก และการตั้งค่าเซิร์ฟเวอร์ได้ คำสั่งเหล่านี้สามารถรันผ่านคอนโซลเซิร์ฟเวอร์สดหรือในเกมโดยตรงได้ ถ้ามีสิทธิ์ที่จำเป็น

![](https://screensaver01.zap-hosting.com/index.php/s/2yJ6G3AWCDfAHMj/download)

หน้านี้ให้ภาพรวมคำสั่งเซิร์ฟเวอร์เกม Hytale ที่ใช้บ่อยและได้รับการรับรองอย่างเป็นทางการ

## การยืนยันตัวตน

คำสั่งยืนยันตัวตนจำเป็นสำหรับการอนุญาตเซิร์ฟเวอร์เพื่อให้สามารถรับการเชื่อมต่อจากผู้เล่นและโต้ตอบกับบริการออนไลน์ของ Hytale ได้

| คำสั่ง               | คำอธิบาย                                                  |
| --------------------- | ------------------------------------------------------------ |
| `/auth login device`  | เริ่มกระบวนการยืนยันตัวตนแบบอุปกรณ์ เซิร์ฟเวอร์จะแสดง URL และรหัสอุปกรณ์ที่ต้องยืนยันผ่านบัญชี Hytale |
| `/auth login browser` | เริ่มกระบวนการยืนยันตัวตนผ่านเบราว์เซอร์ ต้องมีสภาพแวดล้อมกราฟิกบนเซิร์ฟเวอร์ |
| `/auth status`        | แสดงสถานะการยืนยันตัวตนปัจจุบันของเซิร์ฟเวอร์             |
| `/auth logout`        | ออกจากระบบเซิร์ฟเวอร์และลบเซสชันการยืนยันตัวตนที่ใช้งานอยู่ |



## ผู้เล่น & สิทธิ์

คำสั่งเหล่านี้ควบคุมการเข้าถึงระดับแอดมินและการมอบสิทธิ์อย่างละเอียดให้กับผู้ใช้และกลุ่ม

| คำสั่ง                                   | คำอธิบาย                                             |
| ----- | - |
| `/op add <PlayerName>`                    | มอบสิทธิ์ผู้ดูแล (operator) ให้กับผู้เล่นที่ระบุ     |
| `/op remove <PlayerName>`                 | ถอนสิทธิ์ผู้ดูแลจากผู้เล่นที่ระบุ                      |
| `/perm user list <uuid>`                  | แสดงสิทธิ์ทั้งหมดที่มอบให้กับผู้ใช้โดยตรง             |
| `/perm user add <uuid> <permission>`      | มอบสิทธิ์หนึ่งหรือมากกว่านั้นให้กับผู้ใช้โดยตรง       |
| `/perm user remove <uuid> <permission>`   | ถอนสิทธิ์เฉพาะจากผู้ใช้                               |
| `/perm user group list <uuid>`            | แสดงกลุ่มสิทธิ์ทั้งหมดที่ผู้ใช้ถูกมอบหมาย             |
| `/perm user group add <uuid> <group>`     | เพิ่มผู้ใช้เข้าสู่กลุ่มสิทธิ์ที่ระบุ                   |
| `/perm user group remove <uuid> <group>`  | ลบผู้ใช้ออกจากกลุ่มสิทธิ์ที่ระบุ                      |
| `/perm group list <group>`                | แสดงสิทธิ์ทั้งหมดที่มอบให้กับกลุ่ม                     |
| `/perm group add <group> <permission>`    | เพิ่มสิทธิ์ให้กับกลุ่ม                                 |
| `/perm group remove <group> <permission>` | ถอนสิทธิ์จากกลุ่ม                                     |



## รายชื่อขาว (Whitelist)

คำสั่ง whitelist จำกัดการเข้าถึงเซิร์ฟเวอร์เฉพาะผู้เล่นที่ได้รับอนุญาตเท่านั้น

| คำสั่ง                          | คำอธิบาย                                      |
| -------------------------------- | ------------------------------------------------ |
| `/whitelist add <playername>`    | เพิ่มผู้เล่นที่ระบุเข้าสู่รายชื่อขาว            |
| `/whitelist remove <playername>` | ลบผู้เล่นที่ระบุออกจากรายชื่อขาว               |
| `/whitelist list`                | แสดงรายชื่อผู้เล่นที่อยู่ในรายชื่อขาวทั้งหมด    |
| `/whitelist enable`              | เปิดใช้งานการบังคับใช้รายชื่อขาว                |
| `/whitelist disable`             | ปิดการใช้งานรายชื่อขาว                          |
| `/whitelist status`              | แสดงสถานะปัจจุบันของรายชื่อขาว                  |



## การจัดการโลก

คำสั่งโลกควบคุมการสร้าง, โหลด, ปลดโหลด และลบโลก

| คำสั่ง                    | คำอธิบาย                                        |
| -- | -- |
| `/world create <name>`     | สร้างโลกใหม่โดยใช้ชื่อที่ระบุ                     |
| `/world load <name>`       | โหลดโลกที่มีอยู่และทำให้เป็นโลกที่ใช้งานอยู่       |
| `/world unload <name>`     | ปลดโหลดโลกโดยไม่ลบข้อมูล                          |
| `/world remove <name>`     | ลบโลกและข้อมูลทั้งหมดที่เกี่ยวข้อง                  |
| `/world setdefault <name>` | ตั้งค่าโลกเริ่มต้นที่จะโหลดตอนเซิร์ฟเวอร์เริ่มทำงาน |



## การเล่นเกมและสภาพแวดล้อม

คำสั่งเหล่านี้ส่งผลโดยตรงต่อกฎการเล่นเกมและพฤติกรรมของโลก

| คำสั่ง                    | คำอธิบาย                                 |
| -------------------------- | ------------------------------------------- |
| `/list`                    | แสดงผู้เล่นทั้งหมดที่เชื่อมต่ออยู่ในขณะนี้   |
| `/tp <player> <target>`    | เทเลพอร์ตผู้เล่นหนึ่งไปยังผู้เล่นอีกคนหนึ่ง     |
| `/tp <player> <x> <y> <z>` | เทเลพอร์ตผู้เล่นไปยังพิกัดที่ระบุ               |



## เทเลพอร์ต & ตำแหน่ง

คำสั่งใช้จัดการตำแหน่งและการมองเห็นของผู้เล่น

| คำสั่ง                    | คำอธิบาย                                 |
| -- | - |
| `/list`                    | แสดงผู้เล่นทั้งหมดที่เชื่อมต่ออยู่ในขณะนี้   |
| `/tp <player> <target>`    | เทเลพอร์ตผู้เล่นหนึ่งไปยังผู้เล่นอีกคนหนึ่ง     |
| `/tp <player> <x> <y> <z>` | เทเลพอร์ตผู้เล่นไปยังพิกัดที่ระบุ               |



## เครื่องมือและช่วยเหลือ

คำสั่งเครื่องมือให้ความช่วยเหลือและข้อมูลเซิร์ฟเวอร์

| คำสั่ง           | คำอธิบาย                                          |
| ----- | ---- |
| `/help`           | แสดงรายการคำสั่งที่ใช้ได้ทั้งหมด                   |
| `/help <command>` | แสดงคำแนะนำละเอียดสำหรับคำสั่งเฉพาะ               |
| `/status`         | แสดงสถานะเซิร์ฟเวอร์และข้อมูลเวลาทำงานปัจจุบัน   |



## สรุป

ระบบคำสั่งของ Hytale ให้เครื่องมือทรงพลังสำหรับจัดการเซิร์ฟเวอร์, ผู้เล่น และพฤติกรรมการเล่นเกม ด้วยการเข้าใจและใช้คำสั่งที่มีอยู่ แอดมินจะสามารถบริหารและปรับแต่งสภาพแวดล้อมเซิร์ฟเวอร์เกม Hytale ได้อย่างมีประสิทธิภาพ

ถ้ามีคำถามเพิ่มเติมหรือต้องการความช่วยเหลือ อย่าลังเลที่จะติดต่อทีมซัพพอร์ตของเรา ที่พร้อมช่วยเหลือคุณทุกวัน! 🙂

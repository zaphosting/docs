## แนะนำ

ในคู่มือนี้ เราจะมาดูวิธีจัดการเซฟเกม Rimworld Together บนเซิร์ฟเวอร์เกมของคุณ รวมถึงการสำรองข้อมูลเซฟเกมทั้งในเครื่องและบนเซิร์ฟเวอร์ รวมถึงการอัปโหลดไปยังเซิร์ฟเวอร์ของคุณ

:::info
อย่าลืมว่า การย้ายเกมเวอร์ชัน vanilla ที่มีอยู่ไปยังเซิร์ฟเวอร์ Rimworld Together ที่มีม็อด อาจทำให้เกิดปัญหาในการเล่นเกมได้ โดยเฉพาะอย่างยิ่งเมื่อย้ายโลกระหว่างเซิร์ฟเวอร์ที่มีม็อดต่างกัน
:::

## การเตรียมตัว

เพื่อเข้าถึงไฟล์บนเซิร์ฟเวอร์ของคุณ คุณจะต้องใช้ FTP Client เช่น [WinSCP](https://winscp.net/eng/index.php) หรือ [FileZilla](https://filezilla-project.org/) ซึ่งทั้งสองโปรแกรมเป็นโอเพ่นซอร์สและดาวน์โหลดฟรี

คุณต้องเชื่อมต่อกับเซิร์ฟเวอร์ของคุณผ่าน FTP Client โดยใช้ข้อมูลล็อกอินที่อยู่ใน **เครื่องมือ->FTP-Browser** บนเว็บอินเทอร์เฟซของเซิร์ฟเวอร์เกมของคุณ โปรดใช้คู่มือ [การเข้าถึงผ่าน FTP](gameserver-ftpaccess.md) ของเราเพื่อช่วยในการเชื่อมต่อกับเซิร์ฟเวอร์

## การสำรองข้อมูลเซฟเกม

### เซฟเกมในเครื่อง

เซฟเกมในเครื่องคือเซฟที่คุณสร้างขณะโฮสต์เกมมัลติเพลเยอร์บนอุปกรณ์ของคุณเอง เซฟเหล่านี้จะอยู่ใน Windows LocalLow AppData ตามเส้นทางนี้:
```
../AppData/LocalLow/Ludeon Studios/RimWorld by Ludeon Studios/Saves
```

:::tip
คุณสามารถเข้าถึงเส้นทางนี้ได้ง่ายๆ โดยกด `CTRL` + `R` พร้อมกัน แล้วพิมพ์เส้นทางนี้ในหน้าต่างรัน: `%userprofile%/appdata/locallow/Ludeon Studios/RimWorld by Ludeon Studios/Saves` จากนั้นกด **ตกลง** คุณจะถูกพาไปยังโฟลเดอร์นั้นทันที

![](https://screensaver01.zap-hosting.com/index.php/s/LZ6Ljo2DfBoH45s/preview)
:::

ที่นี่คุณจะเห็นเซฟเกมในเครื่องทั้งหมดของคุณ

### การเข้าถึงเซฟเกมผ่าน FTP

การสำรองข้อมูลเซฟเกมจากเซิร์ฟเวอร์ทำได้ง่ายมาก เมื่อคุณเชื่อมต่อกับเซิร์ฟเวอร์เกมผ่าน FTP Client แล้ว ให้ไปที่เส้นทางนี้:
```
../rimworld-together/Saves
```

ที่นี่คุณจะเห็นโฟลเดอร์แต่ละโฟลเดอร์คือเซฟเกม เพียงคลิกขวาที่โฟลเดอร์แล้วเลือกปุ่ม **ดาวน์โหลด** เพื่อบันทึกลงในเครื่องของคุณ

![](https://screensaver01.zap-hosting.com/index.php/s/yjYXnDSHs7g5eCH/preview)

### การสำรองข้อมูลอัตโนมัติ

เรายังมีตัวเลือกให้สำรองข้อมูลเซฟเกม (และไฟล์คอนฟิก) อัตโนมัติผ่านเว็บอินเทอร์เฟซของเรา เพียงเข้าไปที่เว็บอินเทอร์เฟซเซิร์ฟเวอร์เกมของคุณ แล้วไปที่ **เครื่องมือ->สำรองข้อมูล** ที่นี่คุณสามารถตั้งค่าตัวเลือกต่างๆ เพื่อกำหนดเวลาสำรองข้อมูลอัตโนมัติสำหรับเซิร์ฟเวอร์ของคุณ เรามีพื้นที่สำรองข้อมูลฟรี 10GB สำหรับเก็บไฟล์สำรองของคุณ หากต้องการข้อมูลเพิ่มเติมเกี่ยวกับการสำรองข้อมูล ดูคู่มือเฉพาะเรื่อง [การสำรองข้อมูล](gameserver-backups.md) ของเราได้เลย

## การอัปโหลดเซฟเกม

เหมือนกับการสำรองข้อมูล การอัปโหลดเซฟเกมก็ง่ายมาก ก่อนอื่นให้แน่ใจว่าคุณเชื่อมต่อกับเซิร์ฟเวอร์เกมผ่าน FTP Client แล้วไปที่เส้นทางนี้:
```
../rimworld-together/Saves
```

![](https://screensaver01.zap-hosting.com/index.php/s/s4zakSdKniMdGmT/preview)

เพียงลากและวางเซฟเกมของคุณลงในโฟลเดอร์นี้ผ่าน FTP Client แล้วไฟล์จะถูกอัปโหลดไปยังเซิร์ฟเวอร์ของคุณ

:::info
อย่างที่บอกไว้ตอนต้นคู่มือ การย้ายโลกระหว่างเซิร์ฟเวอร์ที่มีม็อดต่างกัน อาจทำให้เกิดปัญหาในการเล่นเกมได้
:::

## แนะนำ

แชทเสียงช่วยให้ผู้เล่นบนเซิร์ฟเวอร์ **Over the Top WWI** ของคุณสื่อสารกันโดยตรงระหว่างเล่นเกม ซึ่งช่วยเพิ่มการประสานงาน การทำงานเป็นทีม และความสมจริงโดยรวม โดยเฉพาะในสถานการณ์ที่ต้องใช้กลยุทธ์

ด้วยการตั้งค่าแชทเสียง คุณสามารถควบคุมได้ว่าจะอนุญาตให้สื่อสารหรือไม่ ปรับคุณภาพเสียง และกำหนดพฤติกรรมของแชทเสียงระหว่างแมตช์

## การตั้งค่า

การตั้งค่าแชทเสียงจะถูกกำหนดในไฟล์คอนฟิกเซิร์ฟเวอร์ คุณสามารถเข้าถึงและแก้ไขไฟล์นี้ได้ใน **แผงควบคุมเซิร์ฟเวอร์เกม** ของคุณภายใต้เมนู **Configs** ค้นหาและเปิดไฟล์คอนฟิก `ServerConfiguration.ini` ภายในไฟล์นี้ ให้หาพารามิเตอร์ดังต่อไปนี้:

```
VoipAllowed = 1
VoipQuality = 0
VoipDelay = 0
```

- `VoipAllowed` เปิดหรือปิดการใช้งานแชทเสียงบนเซิร์ฟเวอร์
- `VoipQuality` ควบคุมระดับคุณภาพเสียง
- `VoipDelay` เพิ่มความหน่วงเป็นวินาทีก่อนที่ผู้เล่นจะใช้แชทเสียงได้หลังจากเกิดใหม่

ปรับค่าตามความชอบและความต้องการด้านประสิทธิภาพของเซิร์ฟเวอร์คุณ การตั้งค่าคุณภาพต่ำอาจเหมาะกับเซิร์ฟเวอร์ที่มีผู้เล่นจำนวนมากหรือแบนด์วิดท์จำกัด

หลังจากแก้ไขไฟล์ `ServerConfiguration.ini` ให้บันทึกไฟล์และรีสตาร์ทเซิร์ฟเวอร์ การตั้งค่าแชทเสียงจะถูกนำไปใช้โดยอัตโนมัติ

## สรุป

ยินดีด้วย! คุณได้ตั้งค่าแชทเสียงบน **เซิร์ฟเวอร์ Over the Top WWI** ของคุณเรียบร้อยแล้ว ซึ่งช่วยให้ผู้เล่นสื่อสารกันได้อย่างมีประสิทธิภาพและเพิ่มการทำงานเป็นทีมระหว่างแมตช์

หากมีคำถามเพิ่มเติมหรือต้องการความช่วยเหลือ อย่าลังเลที่จะติดต่อทีมซัพพอร์ตของเรา ซึ่งพร้อมให้บริการคุณทุกวัน! 🙂

## บทนำ

Git คือระบบควบคุมเวอร์ชันแบบกระจายที่ออกแบบมาเพื่อการจัดการซอร์สโค้ดอย่างมีประสิทธิภาพ ช่วยให้นักพัฒนาสามารถติดตามการเปลี่ยนแปลง ทำงานบนสาขาต่าง ๆ พร้อมกัน และร่วมมือในโปรเจกต์ได้อย่างราบรื่น ด้วยความยืดหยุ่นและประสิทธิภาพ Git จึงกลายเป็นเครื่องมือมาตรฐานสำหรับการควบคุมเวอร์ชันในวงการพัฒนาซอฟต์แวร์ยุคใหม่

กำลังคิดจะเช่าเซิร์ฟเวอร์เกมนี้เองใช่ไหม? เราจะพาคุณผ่านทุกขั้นตอนการตั้งค่าและกำหนดค่า พร้อมทุกสิ่งที่คุณต้องรู้เพื่อให้การใช้งานราบรื่น



## ติดตั้ง Git ด้วย One Click Apps Installer

คุณสามารถติดตั้ง **Git** ได้โดยตรงผ่าน **One Click Apps Installer** ในเว็บแผงควบคุม VPS ของเรา หลังจากตั้งค่าแอปเริ่มต้นเสร็จแล้ว ให้เปิดแคตตาล็อกแอป ค้นหา **Git** และเริ่มการติดตั้งพร้อมตั้งค่าโปรเจกต์ สภาพแวดล้อม และโดเมนที่คุณต้องการ วิธีนี้ช่วยให้คุณติดตั้งและจัดการ **Git** ได้อย่างรวดเร็วและง่ายดายโดยไม่ต้องใช้คำสั่งในคอมมานด์ไลน์ พร้อมทั้งได้ประโยชน์จากการจัดการผ่านเว็บ การรองรับโดเมนแบบกำหนดเอง และการจัดเตรียม SSL เมื่อมีให้บริการ



## การเตรียมตัว

ก่อนตั้งค่า **Git** คุณต้องเตรียมระบบของคุณก่อน ซึ่งรวมถึงการอัปเดตระบบปฏิบัติการเป็นเวอร์ชันล่าสุด การเตรียมตัวเหล่านี้ช่วยให้สภาพแวดล้อมมีความเสถียรและช่วยป้องกันปัญหาระหว่างหรือต่อหลังการติดตั้ง


### อัปเดตระบบ
เพื่อให้แน่ใจว่าระบบของคุณทำงานด้วยซอฟต์แวร์และการปรับปรุงความปลอดภัยล่าสุด คุณควรอัปเดตระบบก่อนเสมอ วิธีนี้ช่วยให้ระบบของคุณมีแพตช์ความปลอดภัยและเวอร์ชันซอฟต์แวร์ล่าสุดก่อนดำเนินการต่อ



## การติดตั้ง

เมื่อเตรียมตัวครบถ้วนแล้ว คุณสามารถดำเนินการติดตั้งแอป Git ได้เลย ดาวน์โหลด Git สำหรับ Windows (x64) ได้ที่นี่: https://git-scm.com/downloads/win

จากนั้นรันไฟล์ติดตั้งที่ดาวน์โหลดมาและทำตามขั้นตอนที่แสดง

![img](https://screensaver01.zap-hosting.com/index.php/s/Y3Rme8q9LHSk4fg/download)

เมื่อการติดตั้งเสร็จสิ้น ให้ตรวจสอบว่า Git ทำงานถูกต้องโดยเปิดเทอร์มินัลหรือคอมมานด์พรอมต์ พิมพ์คำสั่ง `git --version` แล้วกด Enter

ถ้าการติดตั้งสำเร็จ Git จะแสดงหมายเลขเวอร์ชันที่ติดตั้งอยู่ หากได้รับข้อความแสดงข้อผิดพลาดหรือไม่มีผลลัพธ์ แสดงว่า Git อาจติดตั้งไม่สมบูรณ์หรือคำสั่งไม่ได้ถูกเพิ่มในเส้นทางระบบ

```
git --version
git version 2.51.2
```

![img](https://screensaver01.zap-hosting.com/index.php/s/FDDLGnLkStfb7nY/preview)



## การกำหนดค่า

ตั้งค่าชื่อผู้ใช้และอีเมล Git ของคุณด้วยคำสั่งด้านล่าง โดยแทนที่ค่าตัวอย่างด้วยข้อมูลของคุณ ข้อมูลเหล่านี้จะเชื่อมโยงกับทุกคอมมิตที่คุณทำ

```
git config --global user.name "John Doe Paris"
git config --global user.email "doe@e-mail.com"
```

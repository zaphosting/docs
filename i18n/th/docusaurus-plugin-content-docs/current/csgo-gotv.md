## แนะนำ

GOTV เปิดโอกาสให้ถ่ายทอดเกมสดแบบมีหน่วงเวลาเล็กน้อย ซึ่งหมายความว่าคนภายนอกที่ไม่ได้เล่นเกมก็สามารถติดตามสิ่งที่เกิดขึ้นบนเซิร์ฟเวอร์ได้ เทคโนโลยีนี้ยังถูกใช้ในทัวร์นาเมนต์ Counter-Strike ชื่อดัง เพื่อให้ผู้ชมสามารถดูการแข่งขันได้ นอกจากนี้ การถ่ายทอดยังสามารถบันทึกอัตโนมัติ เพื่อให้คุณย้อนดูได้ภายหลัง

## การเปิดใช้งาน

การเปิดใช้งานบริการ GOTV ทำผ่านไฟล์กำหนดค่า **server.cfg** โดยต้องเพิ่มคำสั่งที่จำเป็นดังนี้:

```
tv_enable 1
tv_delay 90
tv_advertise_watchable 1
```

## การตั้งค่า

คำสั่งข้างต้นเป็นคำสั่งบังคับเพื่อเปิดใช้งานและเข้าถึง GOTV master server แต่ยังมีคำสั่งอื่นสำหรับการตั้งค่าขั้นสูงดังนี้:

|            คำสั่ง            |                         คำอธิบาย                         |
| :--------------------------: | :----------------------------------------------------------: |
|  tv_allow_camera_man "0\|1"  |        ผู้ชมสามารถรับบทเป็นช่างกล้องได้        |
| tv_allow_static_shots "0\|1" |    ใช้กล้องที่มีทิศทางคงที่สำหรับถ่ายภาพ    |
|     tv_autorecord "0\|1"     | บันทึกเกมทั้งหมดเป็น GOTV demo อัตโนมัติ ไฟล์ demo จะมีรูปแบบ auto-YYYYYMMDD-hhmm-map.dem |
|     tv_autoretry "0\|1"      | พยายามเชื่อมต่อใหม่หลังจากเกิด network timeout |
| tv_chattimelimit "seconds"  | จำกัดเวลาที่ผู้ชมสามารถพิมพ์ข้อความได้ทุก x วินาที |
|       tv_debug "0\|1"        |             แสดงข้อมูลดีบักของ GOTV             |
|   tv_delaymapchange "0\|1"   | หน่วงเวลาการเปลี่ยนแผนที่จนกว่าการถ่ายทอดจะเสร็จสิ้น |
|      tv_maxclients "n"       |          จำนวนผู้ชมสูงสุดบนเซิร์ฟเวอร์ GOTV          |
|        tv_maxrate "n"        | แบนด์วิดท์สูงสุดที่อนุญาตสำหรับผู้ชม GOTV, 0 == ไม่จำกัด |
|        tv_msg "text"         |           ส่งข้อความไปยังผู้ชมทั้งหมด           |
|        tv_name "name"        | โฮสต์เนม GOTV กำหนดชื่อ GOTV ที่แสดงในเซิร์ฟเวอร์บราวเซอร์และแถบสถานะ |
|       tv_nochat "0\|1"       |           เปิด/ปิดแชท GOTV           |
|    tv_password "password"    |       ป้องกันการถ่ายทอด GOTV ด้วยรหัสผ่าน       |
|     tv_record "filename"     |             เริ่มบันทึก demo GOTV             |
|           tv_stop            |                 หยุดการถ่ายทอด GOTV                 |
|        tv_stoprecord         |            หยุดบันทึก demo GOTV             |
|           tv_title           |           กำหนดชื่อของการถ่ายทอด GOTV           |

## การเชื่อมต่อ

ถ้าฟังก์ชันนี้เปิดใช้งานบนเซิร์ฟเวอร์ สามารถเข้าถึงได้ผ่านที่อยู่ IP และพอร์ต GOTV ที่กำหนดบนเซิร์ฟเวอร์ ในเกมต้องใช้คอนโซลเพื่อเชื่อมต่อ โดยใช้คำสั่งนี้:

```
connect SERVERIP:27020
```

ไฟล์ demo ที่บันทึกทั้งแบบแมนนวลและอัตโนมัติจะถูกเก็บไว้ในไดเรกทอรีหลัก /gxxxxx/CS:GO/game/csgo ของเซิร์ฟเวอร์ CS:GO ซึ่งสามารถดาวน์โหลดผ่านการเข้าถึง FTP ที่จัดเตรียมไว้ หากยังไม่รู้วิธีเข้าถึง FTP ดูได้ที่: [การเข้าถึงผ่าน FTP](gameserver-ftpaccess.md).

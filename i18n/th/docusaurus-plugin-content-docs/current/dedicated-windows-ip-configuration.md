## แนะนำ

การตั้งค่า IP บนเซิร์ฟเวอร์เฉพาะ Windows ของคุณช่วยให้คุณควบคุมการสื่อสารของเซิร์ฟเวอร์ภายในเครือข่ายได้ ซึ่งรวมถึงการกำหนด IP address, การตั้งค่า default gateway และการตั้งค่า DNS server ที่จำเป็นสำหรับการเชื่อมต่อเครือข่าย

การตั้งค่า IP ที่ถูกต้องสำคัญสำหรับเซิร์ฟเวอร์ที่รันบริการต่างๆ เช่น เซิร์ฟเวอร์เกม, เว็บเซิร์ฟเวอร์, ฐานข้อมูล หรือระบบเข้าถึงระยะไกล เพราะจะช่วยให้การสื่อสารในเครือข่ายมีความเสถียรและเข้าถึงได้อย่างน่าเชื่อถือ

:::warning เซิร์ฟเวอร์เข้าใช้งานไม่ได้เพราะไม่มีการเชื่อมต่ออินเทอร์เน็ต?
ถ้าเซิร์ฟเวอร์ของคุณเข้าใช้งานไม่ได้เนื่องจากการตั้งค่าเครือข่ายผิดพลาดหรือไม่มีการเชื่อมต่ออินเทอร์เน็ต การเข้าถึงระยะไกลผ่าน RDP อาจไม่ทำงาน ใช้ [iLO HTML console](dedicated-ilo.md#the-html5-console) เพื่อเชื่อมต่อกับเซิร์ฟเวอร์และแก้ไขการตั้งค่าเครือข่าย
:::

## การเตรียมตัว

ก่อนตั้งค่า IP ให้แน่ใจว่าคุณมีข้อมูลดังนี้พร้อม:

- ที่อยู่ IP ที่ต้องการกำหนด
- Subnet mask
- Default gateway
- DNS servers

ข้อมูลนี้สามารถดูได้ในเว็บอินเทอร์เฟซ โดยเปิด **การจัดการเซิร์ฟเวอร์เฉพาะ** และไปที่ **IP addresses**

## การตั้งค่า

เชื่อมต่อกับเซิร์ฟเวอร์เฉพาะของคุณผ่าน **Remote Desktop (RDP)** เมื่อเชื่อมต่อแล้ว เปิด **แผงควบคุม (Control Panel)** ไปที่ **Control Panel → Network and Internet → Network and Sharing Center**

คลิกที่ **Change adapter settings** ทางด้านซ้าย คุณจะเห็นอินเทอร์เฟซเครือข่ายที่มีอยู่ คลิกขวาที่อะแดปเตอร์เครือข่ายที่ใช้งานอยู่ (โดยปกติจะเป็น **Ethernet**) แล้วเลือก **Properties** ในรายการ เลือก **Internet Protocol Version 4 (TCP/IPv4)** แล้วคลิก **Properties**

![img](https://screensaver01.zap-hosting.com/index.php/s/3EpqZXg5pgWHoLK/download)

เลือก **Use the following IP address** แล้วกรอกค่าที่ต้องการ:

- **IP address** → ที่อยู่ IP ที่ได้รับมอบหมาย
- **Subnet mask** → Subnet mask ของคุณ
- **Default gateway** → เกตเวย์ของคุณ
- **DNS servers →** 1.1.1.1, 8.8.8.8

จากนั้นตั้งค่า DNS servers โดยเลือก **Use the following DNS server addresses** และกรอก DNS servers ที่คุณต้องการ หลังจากกรอกข้อมูลครบแล้ว ยืนยันการตั้งค่าด้วย **OK** และปิดหน้าต่างที่เหลือทั้งหมด



## การตรวจสอบ

เพื่อยืนยันการตั้งค่า เปิด **Command Prompt** และรันคำสั่ง:
```
ipconfig
```
![img](https://screensaver01.zap-hosting.com/index.php/s/AK8F3jppqL8xnsx/download)

คำสั่งนี้จะแสดงอินเทอร์เฟซเครือข่ายทั้งหมดและการตั้งค่า IP ปัจจุบันของพวกมัน มองหาอะแดปเตอร์เครือข่ายที่ใช้งานอยู่และตรวจสอบว่า **IPv4 Address** ตรงกับที่อยู่ IP ที่คุณตั้งค่าก่อนหน้านี้หรือไม่ ถ้าแสดงที่อยู่ IP ถูกต้อง แสดงว่าการตั้งค่าเสร็จสมบูรณ์แล้ว





## สรุป

ยินดีด้วย! คุณตั้งค่า IP บนเซิร์ฟเวอร์เฉพาะ Windows ของคุณสำเร็จแล้ว หากมีคำถามเพิ่มเติมหรือต้องการความช่วยเหลือ ทีมซัพพอร์ตของเราพร้อมให้บริการทุกวันเพื่อช่วยคุณ

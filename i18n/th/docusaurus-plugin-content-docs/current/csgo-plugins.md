## บทนำ

CS:GO ยังคงสืบทอดประเพณีการปรับแต่งเซิร์ฟเวอร์เกมโดยชุมชน ปลั๊กอินช่วยให้เจ้าของเซิร์ฟเวอร์สามารถขยายการเล่นเกม เพิ่มฟีเจอร์ที่ช่วยให้ใช้งานง่ายขึ้น ใช้กฎที่กำหนดเอง หรือสร้างโหมดเกมใหม่ทั้งหมด

สำหรับเซิร์ฟเวอร์เกม CS:GO, **Sourcemod** คือเฟรมเวิร์กที่ใช้กันอย่างแพร่หลายที่สุดสำหรับปลั๊กอินฝั่งเซิร์ฟเวอร์ Sourcemod ทำงานร่วมกับ **Metamod:Source** ซึ่งทำหน้าที่เป็นเลเยอร์ฐานที่ช่วยให้ปลั๊กอินสามารถเชื่อมต่อกับเอนจินเกมได้ โดยการติดตั้งทั้งสองส่วนนี้ ผู้ดูแลเซิร์ฟเวอร์จะสามารถจัดการปลั๊กอินและขยายฟังก์ชันของเซิร์ฟเวอร์ได้อย่างง่ายดาย

## การติดตั้ง Metamod:Source

ต้องติดตั้ง Metamod:Source ก่อน เพราะ Sourcemod ขึ้นอยู่กับมัน เริ่มต้นด้วยการดาวน์โหลดเวอร์ชันล่าสุดของ Metamod:Source จากเว็บไซต์ทางการ แตกไฟล์ที่ดาวน์โหลดมาไว้ในคอมพิวเตอร์ของคุณ

ในไฟล์ที่แตกออกมาจะมีโฟลเดอร์ `addons` ให้คุณอัปโหลดโฟลเดอร์นี้ไปยังเซิร์ฟเวอร์เกม CS:GO ของคุณโดยใช้ [FTP](gameserver-ftpaccess.md) เพื่อให้รวมกับไดเรกทอรีเกมที่มีอยู่แล้วที่ `../csgo/`


หลังจากอัปโหลดไฟล์เสร็จแล้ว ให้รีสตาร์ทเซิร์ฟเวอร์เกม เพื่อยืนยันว่า Metamod ทำงานถูกต้อง ให้เปิดคอนโซลเซิร์ฟเวอร์แล้วรันคำสั่งนี้:

```
meta version
```


ถ้า Metamod ติดตั้งถูกต้อง คอนโซลจะแสดงเวอร์ชันที่ติดตั้งและข้อมูลเพิ่มเติม

<Button label="ดาวน์โหลด Metamod:Source" link="https://www.metamodsource.net/downloads.php" block />



## การติดตั้ง Sourcemod

เมื่อคุณติดตั้ง Metamod:Source เรียบร้อยแล้ว คุณก็สามารถติดตั้ง Sourcemod ได้ ดาวน์โหลดเวอร์ชันเสถียรล่าสุดของ Sourcemod จากเว็บไซต์ทางการและแตกไฟล์ อัปโหลดโฟลเดอร์ `addons` และ `cfg` ที่รวมอยู่ในไฟล์ไปยังเซิร์ฟเวอร์เกม CS:GO ของคุณโดยใช้ [FTP](gameserver-ftpaccess.md) โฟลเดอร์เหล่านี้ต้องวางไว้ในไดเรกทอรีเกม CS:GO ที่ `../csgo/` หลังจากอัปโหลดไฟล์เสร็จแล้ว ให้รีสตาร์ทเซิร์ฟเวอร์เกม Sourcemod จะถูกโหลดโดยอัตโนมัติผ่าน Metamod

เพื่อยืนยันว่า Sourcemod ทำงาน ให้เปิดคอนโซลเซิร์ฟเวอร์แล้วรันคำสั่ง:

```
sm version
```


ถ้าการติดตั้งสำเร็จ เซิร์ฟเวอร์จะส่งกลับเวอร์ชัน Sourcemod ปัจจุบันและข้อมูลบิลด์

<Button label="ดาวน์โหลด Sourcemod" link="https://www.sourcemod.net/downloads.php" block />



## การติดตั้งปลั๊กอิน

ปลั๊กอินสำหรับ Sourcemod มักจะถูกแจกจ่ายในรูปแบบไฟล์ `.smx` ไฟล์ปลั๊กอินที่คอมไพล์แล้วเหล่านี้ต้องถูกอัปโหลดไปยังโฟลเดอร์ `plugins` ภายในการติดตั้ง Sourcemod

```
../csgo/addons/sourcemod/plugins/
```


หลังจากอัปโหลดไฟล์ปลั๊กอินแล้ว ให้รีสตาร์ทเซิร์ฟเวอร์เกม หรือโหลดปลั๊กอินโดยตรงผ่านคอนโซลเซิร์ฟเวอร์ด้วยคำสั่งนี้:

```
sm plugins load <pluginname>
```


ถ้าปลั๊กอินโหลดไม่สำเร็จ ให้ตรวจสอบล็อกเซิร์ฟเวอร์ที่อยู่ในโฟลเดอร์ `logs` ของ Sourcemod เพื่อดูข้อความแสดงข้อผิดพลาดที่เป็นไปได้

ปลั๊กอินบางตัวอาจมีไฟล์คอนฟิกเพิ่มเติม ไฟล์เหล่านี้มักจะถูกวางไว้ใน:

```
../csgo/addons/sourcemod/configs/
```


อย่าลืมทำตามคำแนะนำการติดตั้งที่ผู้พัฒนาปลั๊กอินให้มาเพื่อให้แน่ใจว่าทำงานได้ถูกต้อง

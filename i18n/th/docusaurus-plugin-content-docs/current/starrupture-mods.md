## แนะนำ

ม็อดช่วยให้คุณขยายประสบการณ์การเล่น StarRupture ด้วยการเพิ่มกลไกใหม่ ๆ เนื้อหา ปรับปรุงคุณภาพชีวิต หรือการปรับแต่งเกมเพลย์ที่สร้างโดยชุมชน ม็อด StarRupture หลายตัวถูกแจกจ่ายผ่านแพลตฟอร์มชุมชนอย่าง NexusMods และสามารถติดตั้งได้ทั้งบนเซิร์ฟเวอร์และเครื่องไคลเอนต์ของคุณ

การติดตั้งม็อดบนเซิร์ฟเวอร์จะช่วยให้สภาพแวดล้อมเซิร์ฟเวอร์โหลดเนื้อหาที่ถูกแก้ไข ในขณะที่การติดตั้งบนเครื่องไคลเอนต์จะช่วยให้เกมของคุณใช้ไฟล์เดียวกันเมื่อเชื่อมต่อกับเซิร์ฟเวอร์ที่มีม็อด

## การติดตั้งม็อดบนเซิร์ฟเวอร์

เพื่อจะติดตั้งม็อดบนเซิร์ฟเวอร์เกม StarRupture ของคุณ ให้ดาวน์โหลดไฟล์ม็อดที่ต้องการจากแหล่งที่เชื่อถือได้ เช่น **NexusMods**

![img](https://screensaver01.zap-hosting.com/index.php/s/rM4WR4gW2KLyzxz/preview)

ก่อนอัปโหลดม็อด ให้แน่ใจว่าเซิร์ฟเวอร์ของคุณถูกหยุดทำงานเพื่อหลีกเลี่ยงปัญหาการชนกันของไฟล์ระหว่างการติดตั้ง จากนั้นเชื่อมต่อกับเซิร์ฟเวอร์ของคุณผ่าน [FTP](gameserver-ftpaccess.md) โดยใช้โปรแกรม FTP เมื่อต่อสำเร็จ ให้ไปที่โฟลเดอร์นี้บนเซิร์ฟเวอร์ของคุณ:

```
/gxxxxxxx/StarRupture/Content/Paks
```
อัปโหลดไฟล์ม็อด `.zip` ที่ดาวน์โหลดมาเข้าไปในโฟลเดอร์นี้ หลังจากอัปโหลดเสร็จแล้ว ให้แตกไฟล์ในโฟลเดอร์ `Paks` โดยตรง เมื่อแตกไฟล์เสร็จแล้ว กลับไปที่แผงควบคุมเซิร์ฟเวอร์และสตาร์ทเซิร์ฟเวอร์อีกครั้ง

ในระหว่างการเริ่มต้น เซิร์ฟเวอร์จะตรวจจับและโหลดม็อดที่ติดตั้งโดยอัตโนมัติ

## การติดตั้งม็อดบนไคลเอนต์

ม็อดบางตัวของ StarRupture อาจต้องติดตั้งบนเครื่องไคลเอนต์ด้วย ม็อดที่ติดตั้งในเครื่องส่วนใหญ่จะยังคงทำงานเมื่อเชื่อมต่อกับเซิร์ฟเวอร์อื่น ๆ ก่อนอื่นดาวน์โหลดม็อดที่ต้องการจากแหล่งที่เชื่อถือได้ เช่น NexusMods

จากนั้นเปิด **Steam Library** คลิกขวาที่ **StarRupture** เลื่อนเมาส์ไปที่ **Manage** แล้วเลือก **Browse local files** ในโฟลเดอร์ติดตั้งเกม ให้ไปที่เส้นทางนี้:

```
StarRupture/StarRupture/Content/Paks
```

นำไฟล์ม็อด `.zip` ที่ดาวน์โหลดมาใส่ในโฟลเดอร์นี้ หลังจากคัดลอกไฟล์แล้ว ให้แตกไฟล์ในโฟลเดอร์ `Paks` โดยตรง เมื่อแตกไฟล์เสร็จ ม็อดจะถูกโหลดโดยอัตโนมัติเมื่อคุณเริ่มเกมครั้งถัดไป

## สรุป

ยินดีด้วย! คุณติดตั้งม็อดสำหรับ StarRupture สำเร็จทั้งบนเซิร์ฟเวอร์และไคลเอนต์แล้ว ด้วยม็อดที่ติดตั้ง คุณจะสามารถขยายประสบการณ์การเล่นด้วยฟีเจอร์ใหม่ ๆ กลไก และการปรับปรุงที่สร้างโดยชุมชน

หากมีคำถามเพิ่มเติมหรือต้องการความช่วยเหลือ อย่าลังเลที่จะติดต่อทีมซัพพอร์ตของเรา ซึ่งพร้อมให้บริการคุณทุกวัน! 🙂

## แนะนำ

Mods มีบทบาทสำคัญในการขยายประสบการณ์การเล่นใน **Arma Reforger** ผ่าน **Arma Reforger Workshop** ผู้เล่นและผู้สร้างสามารถเผยแพร่มิชชั่นที่กำหนดเอง, การเปลี่ยนแปลงเกมเพลย์, แผนที่ และเนื้อหาเสริมที่สามารถใช้บนเซิร์ฟเวอร์ได้

การติดตั้ง mods ช่วยให้เจ้าของเซิร์ฟเวอร์ปรับแต่งเกมเพลย์, เพิ่มกลไกใหม่ หรือมอบมิชชั่นเฉพาะตัวให้กับชุมชนของพวกเขา เมื่อทำการติดตั้งแล้ว ผู้เล่นที่เข้าร่วมเซิร์ฟเวอร์จะดาวน์โหลด mods ที่จำเป็นโดยอัตโนมัติก่อนเชื่อมต่อ

## การค้นหา Mods

คุณต้องมี **Workshop ID** ของ mod ที่ต้องการติดตั้งก่อน เปิด **Arma Reforger Workshop** และไปยัง mod ที่ต้องการ แต่ละรายการใน workshop จะมีตัวระบุเฉพาะที่เซิร์ฟเวอร์ใช้ดาวน์โหลด mod โดยอัตโนมัติ

![img](https://screensaver01.zap-hosting.com/index.php/s/jMNGbRAQdjKcDyn/preview)

## การตั้งค่า

เพื่อทำการติดตั้ง workshop mods บนเซิร์ฟเวอร์ของคุณ ให้เปิด **แผงควบคุมการจัดการเซิร์ฟเวอร์เกม** และไปที่ **การตั้งค่า** ค้นหาช่องการตั้งค่าชื่อ **Workshop-Mods** ในช่องนี้ ให้กรอก **Workshop ID** ของ mod ที่ต้องการติดตั้ง ตัวอย่างการตั้งค่ากับ mod เดียว:

```
1234567890
```
ถ้าคุณต้องการติดตั้งหลาย mods ให้กรอก Workshop ID แต่ละตัวและคั่นด้วย **เครื่องหมายจุลภาค** ตัวอย่างการตั้งค่ากับหลาย mods:

```
1234567890,9876543210,1357924680
```
หลังจากกรอก ID mods ที่ต้องการแล้ว **บันทึกการตั้งค่า** และรีสตาร์ทเซิร์ฟเวอร์ ในระหว่างการเริ่มต้น เซิร์ฟเวอร์จะดาวน์โหลด mods ที่ระบุจาก Arma Reforger Workshop โดยอัตโนมัติและโหลดเข้าเซิร์ฟเวอร์



## สรุป

ยินดีด้วย! คุณได้ติดตั้ง **Arma Reforger Workshop mods** บนเซิร์ฟเวอร์ของคุณเรียบร้อยแล้ว หากมีคำถามเพิ่มเติมหรือต้องการความช่วยเหลือ อย่าลังเลที่จะติดต่อทีมซัพพอร์ตของเรา ซึ่งพร้อมให้บริการทุกวันเพื่อช่วยเหลือคุณ! 🙂

## Introduction

แดชบอร์ด **Windrose** คือพื้นที่ศูนย์กลางสำหรับจัดการเซิร์ฟเวอร์เกมของคุณในเว็บอินเทอร์เฟซ ZAP-Hosting ในคู่มือนี้ คุณจะได้รับภาพรวมที่เป็นระบบของส่วนต่างๆ ในแดชบอร์ดและเรียนรู้วิธีใช้เครื่องมือบริหารจัดการ การตรวจสอบ และการโฮสต์ที่สำคัญอย่างมีประสิทธิภาพ

<InlineVoucher />

## Product Administration

ในพื้นที่ **Product Administration** คุณจะพบหน้าที่สำคัญที่สุดสำหรับควบคุมและตรวจสอบเซิร์ฟเวอร์ **Windrose** ของคุณ หน้าต่างๆ เหล่านี้ช่วยให้คุณจัดการสถานะเซิร์ฟเวอร์ปัจจุบัน ตรวจสอบเหตุการณ์ทางเทคนิค และเข้าถึงข้อมูลบริการที่จำเป็นได้อย่างรวดเร็ว

### Dashboard

**แดชบอร์ด** คือหน้าภาพรวมหลักของเซิร์ฟเวอร์ **Windrose** ของคุณ ที่นี่คุณจะเห็นรายละเอียดบริการที่สำคัญที่สุดในทันที เช่น สถานะเซิร์ฟเวอร์ปัจจุบัน ที่ตั้งเซิร์ฟเวอร์ ที่อยู่เชื่อมต่อ และการใช้ทรัพยากร

![img](https://screensaver01.zap-hosting.com/index.php/s/boTwmEC5HSjbKYK/preview)

ขึ้นอยู่กับการตั้งค่าเซิร์ฟเวอร์และการผสานเกมในปัจจุบัน แดชบอร์ดอาจแสดงข้อมูลเพิ่มเติม เช่น แผนที่ที่ใช้งาน ชื่อเซิร์ฟเวอร์ หน่วยความจำที่ใช้ หรือการใช้งานช่องผู้เล่น นอกจากนี้ยังเป็นที่ที่คุณสามารถเริ่ม หยุด หรือรีสตาร์ทบริการได้โดยตรง

แดชบอร์ดมีประโยชน์มากเพราะรวมทั้งการควบคุมเซิร์ฟเวอร์และการตรวจสอบไว้ในที่เดียว ทำให้คุณตรวจสอบได้อย่างรวดเร็วว่าบริการของคุณออนไลน์หรือไม่ และตอบสนองทันทีหากจำเป็นต้องเปลี่ยนแปลง

### Statistics

ในส่วน **Statistics** คุณสามารถตรวจสอบข้อมูลการใช้งานย้อนหลังของเซิร์ฟเวอร์ได้ ซึ่งช่วยให้คุณเข้าใจประสิทธิภาพของบริการโฮสต์ **Windrose** ของคุณในช่วงเวลาต่างๆ ได้ดีขึ้น

![](https://screensaver01.zap-hosting.com/index.php/s/N8jcmdqqG2Xt4Bk/preview)

กราฟที่มีให้มักจะรวมถึงค่าต่างๆ เช่น การใช้ CPU การใช้หน่วยความจำ และการใช้งานช่องผู้เล่น คุณสามารถสลับดูช่วงเวลาต่างๆ เพื่อวิเคราะห์การเพิ่มขึ้นชั่วคราวหรือแนวโน้มระยะยาวได้

| ตัวชี้วัด | จุดประสงค์ |
| --- | --- |
| การใช้ CPU | ช่วยระบุภาระการประมวลผลและปัญหาคอขวดประสิทธิภาพที่อาจเกิดขึ้น |
| การใช้หน่วยความจำ | แสดงจำนวน RAM ที่เซิร์ฟเวอร์ของคุณใช้งาน |
| การใช้งานช่องผู้เล่น | บ่งชี้จำนวนช่องผู้เล่นที่ถูกใช้งานจริง |

:::tip วิเคราะห์ประสิทธิภาพ
ถ้าคุณพบอาการแลค เกมค้าง หรือการตอบสนองช้า หน้าสถิติเป็นจุดเริ่มต้นที่ดีในการแก้ไขปัญหา เพราะช่วยให้คุณทราบว่าปัญหาเกิดจากการใช้ทรัพยากรหรือเกิดขึ้นในช่วงเวลาหนึ่งโดยเฉพาะ
:::

### DDoS Manager

**DDoS Manager** ช่วยให้คุณตรวจสอบข้อมูลเกี่ยวกับการโจมตี DDoS ที่ตรวจพบและถูกบรรเทาต่อบริการของคุณ ซึ่งมีประโยชน์ถ้าคุณต้องการเข้าใจรูปแบบทราฟฟิกที่ผิดปกติหรือสืบสวนการหยุดชะงักของบริการที่เกิดจากทราฟฟิกที่เป็นอันตราย

![](https://screensaver01.zap-hosting.com/index.php/s/ScCCCY52CMLgfyE/preview)

ขึ้นอยู่กับตัวเลือกที่คุณจองและการป้องกันที่มีในที่ตั้งเซิร์ฟเวอร์ที่เลือก คุณอาจดูข้อมูลการโจมตีที่กำลังเกิดขึ้นและรายละเอียดทราฟฟิกแบบเรียลไทม์ได้ด้วย

:::info การใช้งาน DDoS Manager
DDoS Manager จะใช้งานได้เฉพาะเมื่อบริการของคุณมีที่อยู่ IP เฉพาะ หรือมีตัวเลือกเสริม **DDoS Manager Overview** รวมอยู่ในแพ็กเกจ หากไม่มีในแพ็กเกจปัจจุบัน คุณต้องอัปเกรดบริการก่อน
:::

### Log Files

ถ้าเซิร์ฟเวอร์ **Windrose** ของคุณแสดงข้อผิดพลาดหรือทำงานผิดปกติ ส่วน **Log Files** คือที่สำคัญที่สุดที่ควรตรวจสอบ ไฟล์ล็อกจะบันทึกข้อมูลทางเทคนิคจากเซิร์ฟเวอร์ เช่น คำเตือน ข้อความเริ่มต้น และรายละเอียดข้อผิดพลาด

ไฟล์เหล่านี้มีประโยชน์มากเมื่อคุณต้องการหาสาเหตุของการค้าง การเริ่มต้นล้มเหลว หรือปัญหาการตั้งค่า

:::caution ข้อจำกัดการตรวจสอบล็อก
ถ้าข้อมูลล็อกไม่ชัดเจนหรือไม่ช่วยให้คุณระบุปัญหาได้ ควรติดต่อฝ่ายสนับสนุนผ่าน [หน้าสนับสนุนอย่างเป็นทางการ](https://zap-hosting.com/en/customer/support/) พร้อมคำอธิบายปัญหาและข้อความล็อกที่เกี่ยวข้องเพื่อเร่งการแก้ไข
:::

### Event Log

**Event Log** ให้ภาพรวมของการกระทำและเหตุการณ์ที่เกี่ยวข้องกับบริการของคุณ เช่น การเริ่มต้น หยุด ค้าง การติดตั้ง อัปเดต และกิจกรรมสำรองข้อมูล

![](https://screensaver01.zap-hosting.com/index.php/s/xNzsaMbW5BS5KJC/preview)

ส่วนนี้ช่วยได้มากถ้าคุณต้องการตรวจสอบว่าเกิดการเปลี่ยนแปลงเมื่อไหร่ หรือว่าการทำงานอัตโนมัติสำเร็จหรือไม่

### Live Console

**Live Console** แสดงผลลัพธ์เซิร์ฟเวอร์แบบเรียลไทม์ขณะที่เซิร์ฟเวอร์ **Windrose** ของคุณกำลังทำงาน ช่วยให้คุณติดตามสิ่งที่เกิดขึ้นบนเซิร์ฟเวอร์และตรวจจับปัญหาได้ทันที

![](https://screensaver01.zap-hosting.com/index.php/s/J9SsmSrPXnG7P9w/preview)

ถ้าเซิร์ฟเวอร์ซอฟต์แวร์รองรับ คุณยังสามารถส่งคำสั่งผ่านคอนโซลสดได้โดยตรง ซึ่งเหมาะสำหรับการจัดการที่ต้องทำโดยไม่ต้องรีสตาร์ทบริการ

| ฟังก์ชัน | คำอธิบาย |
| --- | --- |
| ผลลัพธ์แบบเรียลไทม์ | แสดงเหตุการณ์และข้อความปัจจุบันของเซิร์ฟเวอร์ |
| การมองเห็นข้อผิดพลาด | ช่วยให้คุณระบุปัญหาได้ทันทีที่เกิดขึ้น |
| การป้อนคำสั่ง | ให้คุณส่งคำสั่งที่รองรับโดยเซิร์ฟเวอร์ได้โดยตรง |

### Service Interruptions

ถ้าเซิร์ฟเวอร์ของคุณไม่สามารถใช้งานได้ชั่วคราวและไม่มีสาเหตุชัดเจนใน event log หรือ log files ปัญหาอาจเกิดจากระบบโฮสต์พื้นฐานมากกว่าบริการของคุณเอง

ในส่วน **Service Interruptions** คุณสามารถตรวจสอบประวัติการหยุดชะงักที่เกี่ยวข้องกับสภาพแวดล้อมโฮสต์ของเซิร์ฟเวอร์เกมได้ สำหรับข้อมูลสถานะแพลตฟอร์มโดยรวม คุณสามารถตรวจสอบที่ [หน้าสถานะอย่างเป็นทางการ](https://status.zap-hosting.com/)

## Settings

พื้นที่ **Settings** ประกอบด้วยหน้าการตั้งค่าหลักสำหรับเซิร์ฟเวอร์ **Windrose** ของคุณ ที่นี่คุณสามารถปรับพฤติกรรมบริการ เปลี่ยนเกมที่ติดตั้งได้ถ้ารองรับ จัดการอัปเดต และตั้งค่าฟีเจอร์อัตโนมัติ

### Settings

หน้าการตั้งค่า **Settings** ให้ภาพรวมที่เรียบง่ายของตัวเลือกเซิร์ฟเวอร์ทั่วไป เหมาะอย่างยิ่งถ้าคุณเป็นมือใหม่ในการเช่าเซิร์ฟเวอร์เกมและต้องการเปลี่ยนแปลงโดยไม่ต้องแก้ไขไฟล์คอนฟิกโดยตรง

ถ้ามีให้สำหรับ **Windrose** ตัวเลือกเหล่านี้อาจรวมถึงการตั้งค่าเกี่ยวกับเกมเพลย์หรือพารามิเตอร์บริการทั่วไป ตัวเลือกที่แน่นอนขึ้นอยู่กับการผสานเกมในปัจจุบัน

### Configs

ส่วน **Configs** ช่วยให้คุณแก้ไขไฟล์คอนฟิกโดยตรงผ่านเว็บอินเทอร์เฟซ ซึ่งให้การควบคุมที่แม่นยำกว่าหน้าการตั้งค่าที่เรียบง่าย เหมาะสำหรับผู้ใช้ขั้นสูง

:::caution ค่าที่ถูกเขียนทับ
บางค่าถูกควบคุมโดยหน้าการตั้งค่า **Settings** ถ้าคุณสังเกตเห็นว่าการเปลี่ยนแปลงด้วยตนเองในตัวแก้ไขคอนฟิกถูกเขียนทับ ให้ตรวจสอบว่าตัวเลือกเดียวกันยังถูกควบคุมที่นั่นหรือไม่
:::

### Games

ในส่วน **Games** คุณสามารถเปลี่ยนไปเล่นเกมหรือเวอร์ชันเกมอื่นที่มีให้ถ้าบริการของคุณรองรับ เกมที่มีราคาช่องผู้เล่นเท่ากันมักเลือกได้โดยตรง ส่วนเกมอื่นอาจต้องปรับแพ็กเกจก่อน

ถ้าคุณต้องการเรียนรู้เพิ่มเติมเกี่ยวกับการเปลี่ยนเกมที่ติดตั้งในบริการ ดูที่คู่มือ [Game Change](gameserver-gameswitch.md)

![](https://screensaver01.zap-hosting.com/index.php/s/xkkECw7o52fAMWk/preview)

### Versions

หน้าการจัดการ **Versions** ช่วยให้คุณจัดการเวอร์ชันเซิร์ฟเวอร์ที่ติดตั้งสำหรับบริการ **Windrose** ของคุณได้ ถ้ารองรับการจัดการเวอร์ชัน คุณสามารถอัปเดตเป็นเวอร์ชันล่าสุดและเปิดใช้งานการอัปเดตอัตโนมัติได้

![](https://screensaver01.zap-hosting.com/index.php/s/BH2JzyRHTeLdKHz/preview)

การอัปเดตอัตโนมัติช่วยให้สภาพแวดล้อมโฮสต์ของคุณทันสมัย แต่คุณควรตรวจสอบความเข้ากันได้หลังการเปลี่ยนเวอร์ชันใหญ่

### Scheduled Tasks

ด้วย **Scheduled Tasks** คุณสามารถตั้งค่าให้ทำงานอัตโนมัติสำหรับเซิร์ฟเวอร์ของคุณ เหมาะสำหรับลดงานดูแลด้วยตนเองและรักษาการทำงานของบริการให้สม่ำเสมอ

![](https://screensaver01.zap-hosting.com/index.php/s/P6DeWiRC3tDqG2z/preview)

งานอัตโนมัติทั่วไปอาจรวมถึง:

| ประเภทงาน | ตัวอย่างการใช้งาน |
| --- | --- |
| เริ่มบริการ | เริ่มเซิร์ฟเวอร์โดยอัตโนมัติในเวลาที่กำหนด |
| หยุดบริการ | ปิดเซิร์ฟเวอร์ในช่วงเวลาบำรุงรักษา |
| รีสตาร์ทบริการ | รีสตาร์ทเซิร์ฟเวอร์เป็นประจำเพื่อความเสถียร |
| สร้างสำรองข้อมูล | สร้างสำรองข้อมูลอัตโนมัติโดยไม่ต้องทำด้วยตนเอง |
| คำสั่งที่กำหนดเอง | รันคำสั่งที่รองรับโดยอัตโนมัติ |

### Restart Planner

**Restart Planner** ถูกออกแบบมาเพื่อกำหนดเวลาการรีสตาร์ทเซิร์ฟเวอร์อัตโนมัติ เหมาะสำหรับการรีสตาร์ทเซิร์ฟเวอร์ **Windrose** เป็นประจำ เช่น ในช่วงเวลาที่มีผู้เล่นน้อย

![](https://screensaver01.zap-hosting.com/index.php/s/Y6WciDS7YP98P4m/preview)

### DNS Settings

ด้วย **DNS Settings** คุณสามารถกำหนดที่อยู่ที่จำง่ายกว่าให้กับบริการของคุณ ทำให้เซิร์ฟเวอร์ของคุณจดจำง่ายและแชร์กับผู้เล่นคนอื่นได้สะดวกขึ้น

![](https://screensaver01.zap-hosting.com/index.php/s/tM5FGTHRSbZA5rN/preview)

คุณอาจใช้ซับโดเมนที่ ZAP ให้มา หรือโดเมนของคุณเอง ขึ้นอยู่กับการตั้งค่าและตัวเลือกที่จองไว้

## Tools

ส่วน **Tools** ประกอบด้วยฟีเจอร์เสริมสำหรับจัดการบริการที่สนับสนุนการเข้าถึงไฟล์ การจัดการฐานข้อมูล และการจัดการสำรองข้อมูล

### FTP Browser

ใน **FTP Browser** คุณจะพบรายละเอียดการเชื่อมต่อที่จำเป็นสำหรับเข้าถึงไฟล์เซิร์ฟเวอร์ การเข้าถึง FTP มีประโยชน์ถ้าคุณต้องการอัปโหลด ดาวน์โหลด หรือแก้ไขไฟล์โดยตรง

![](https://screensaver01.zap-hosting.com/index.php/s/sX85i9HyzR9wmAM/preview)

เพื่อความสะดวกในการจัดการไฟล์ แนะนำให้ใช้โปรแกรม FTP ภายนอก เช่น [FileZilla](https://filezilla-project.org/) หรือ [WinSCP](https://winscp.net/) ถ้าคุณต้องการความช่วยเหลือในการเชื่อมต่อ ดูคู่มือ [Access via FTP](gameserver-ftpaccess.md)

### Databases

ในส่วน **Databases** คุณสามารถสร้างและจัดการฐานข้อมูลที่รวมมากับผลิตภัณฑ์เซิร์ฟเวอร์เกมของคุณได้ ถ้าการตั้งค่าของคุณต้องการ ซึ่งเหมาะสำหรับบริการ ม็อด หรือเครื่องมือที่ต้องเก็บข้อมูลแบบมีโครงสร้าง

![](https://screensaver01.zap-hosting.com/index.php/s/WFLwCbZSQfYjrmD/preview)

คุณมักจะดูข้อมูลรับรองการเข้าถึง สร้างฐานข้อมูลใหม่ และนำเข้าข้อมูลสำรองที่มีอยู่จากส่วนนี้ได้

### Backups

ส่วน **Backups** ช่วยให้คุณสร้างและจัดการสำรองข้อมูลสำหรับเซิร์ฟเวอร์ของคุณ และถ้ารองรับสำหรับข้อมูลฐานข้อมูลที่เกี่ยวข้องด้วย

![](https://screensaver01.zap-hosting.com/index.php/s/nm9za5LTMEcNsga/preview)

:::tip แนวทางปฏิบัติที่ดีที่สุดสำหรับการสำรองข้อมูล
สร้างสำรองข้อมูลด้วยตนเองก่อนทำการเปลี่ยนแปลงใหญ่ เช่น แก้ไขไฟล์คอนฟิก เปลี่ยนเวอร์ชัน หรือเปลี่ยนเกมที่ติดตั้ง เพื่อให้มีจุดคืนค่าที่ปลอดภัยหากเกิดปัญหา
:::

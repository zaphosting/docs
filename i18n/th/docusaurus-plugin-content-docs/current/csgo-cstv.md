## แนะนำ

CSTV เปิดโอกาสให้ถ่ายทอดสดเกมแบบมีหน่วงเวลาเล็กน้อย ซึ่งหมายความว่าคนภายนอกที่ไม่ได้เล่นเกมก็สามารถติดตามเหตุการณ์ที่เกิดขึ้นบนเซิร์ฟเวอร์ได้ เทคโนโลยีนี้ถูกใช้ในทัวร์นาเมนต์ Counter-Strike ชื่อดัง เพื่อให้ผู้ชมได้ดูการแข่งขัน นอกจากนี้ การถ่ายทอดยังสามารถบันทึกอัตโนมัติ เพื่อให้คุณย้อนดูได้ภายหลัง

<InlineVoucher />

## การเปิดใช้งาน

การเปิดใช้งานบริการ CSTV ทำได้ผ่านไฟล์กำหนดค่า **server.cfg** โดยต้องเพิ่มคำสั่งที่จำเป็นดังนี้:

```
tv_enable 1
tv_delay 90
tv_advertise_watchable 1
```

## การตั้งค่า

คำสั่งข้างต้นเป็นคำสั่งบังคับเพื่อเปิดใช้งานเซิร์ฟเวอร์ CSTV และทำให้เข้าถึงได้ แต่ยังมีคำสั่งอื่นๆ สำหรับการตั้งค่าขั้นสูงดังนี้:

|            คำสั่ง            |                         คำอธิบาย                         |
| :--------------------------: | :----------------------------------------------------------: |
|  tv_allow_camera_man "0\|1"  |        ผู้ชมสามารถรับบทเป็นช่างกล้องได้        |
| tv_allow_static_shots "0\|1" |    ใช้กล้องที่มีการตั้งทิศทางคงที่สำหรับถ่ายภาพ    |
|     tv_autorecord "0\|1"     | บันทึกเกมทั้งหมดเป็นไฟล์ CSTV demo อัตโนมัติ โดยไฟล์บันทึกจะมีรูปแบบ auto-YYYYYMMDD-hhmm-map.dem |
|     tv_autoretry "0\|1"      | พยายามเชื่อมต่อใหม่หลังจากเกิดการตัดเน็ตเวิร์ก |
| tv_chattimelimit "seconds"  | จำกัดเวลาที่ผู้ชมสามารถพิมพ์ข้อความได้ทุก x วินาที |
|       tv_debug "0\|1"        |             แสดงข้อมูลดีบักของ CSTV             |
|   tv_delaymapchange "0\|1"   | หน่วงเวลาการเปลี่ยนแผนที่จนกว่าการถ่ายทอดจะเสร็จสิ้น |
|      tv_maxclients "n"       |          จำนวนผู้ชมสูงสุดบนเซิร์ฟเวอร์ CSTV          |
|        tv_maxrate "n"        | แบนด์วิดท์สูงสุดที่อนุญาตสำหรับผู้ชม CSTV, 0 == ไม่จำกัด |
|        tv_msg "text"         |           ส่งข้อความไปยังผู้ชมทั้งหมด           |
|        tv_name "name"        | โฮสต์เนม CSTV กำหนดชื่อ CSTV ที่จะแสดงในเบราว์เซอร์เซิร์ฟเวอร์และแถบสถานะ |
|       tv_nochat "0\|1"       |           เปิด/ปิดแชท CSTV           |
|    tv_password "password"    |       ป้องกันการถ่ายทอด CSTV ด้วยรหัสผ่าน       |
|     tv_record "filename"     |             เริ่มบันทึกไฟล์ demo CSTV             |
|           tv_stop            |                 หยุดการถ่ายทอด CSTV                 |
|        tv_stoprecord         |            หยุดการบันทึกไฟล์ demo CSTV             |
|           tv_title           |           กำหนดชื่อของการถ่ายทอด CSTV           |

## การเชื่อมต่อ

ถ้าฟังก์ชันนี้ถูกเปิดใช้งานบนเซิร์ฟเวอร์ สามารถเข้าถึงได้ผ่านที่อยู่ IP และพอร์ต CSTV ที่กำหนดบนเซิร์ฟเวอร์ โดยต้องใช้คอนโซลในเกมเพื่อเชื่อมต่อด้วยคำสั่งนี้:

```
connect SERVERIP:27020
```

ไฟล์ demo ที่บันทึกทั้งแบบแมนนวลและอัตโนมัติจะถูกเก็บไว้ในไดเรกทอรีหลัก /gxxxxx/CS:GO/game/csgo ของเซิร์ฟเวอร์ CS:GO ซึ่งสามารถดาวน์โหลดผ่านการเข้าถึง FTP ที่จัดเตรียมไว้ หากคุณยังไม่รู้วิธีเข้าถึงผ่าน FTP สามารถดูคู่มือได้ที่นี่: [การเข้าถึงผ่าน FTP](gameserver-ftpaccess.md).

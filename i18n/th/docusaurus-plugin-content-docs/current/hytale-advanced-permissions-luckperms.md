## แนะนำ

การจัดการสิทธิ์บนเซิร์ฟเวอร์ Hytale จะยิ่งสำคัญขึ้นเมื่อคุณเริ่มทำงานกับบทบาททีมงาน, อันดับที่กำหนดเอง หรือสิทธิพิเศษพิเศษสำหรับผู้เล่น ในขณะที่ระบบสิทธิ์ในเกมเริ่มต้นเหมาะสำหรับการดูแลพื้นฐาน แต่มันจะถึงขีดจำกัดอย่างรวดเร็วเมื่อคุณต้องการควบคุมคำสั่งและฟีเจอร์อย่างละเอียด

**LuckPerms** มอบระบบสิทธิ์ที่แข็งแกร่งและปรับแต่งได้สูงสำหรับเซิร์ฟเวอร์ Hytale ช่วยให้คุณกำหนดได้อย่างแม่นยำว่าสิทธิ์และคำสั่งใดที่ผู้เล่นหรือกลุ่มต่างๆ สามารถเข้าถึงได้

## การติดตั้ง LuckPerms

ในการติดตั้งม็อด LuckPerms ให้ดาวน์โหลดไฟล์ `.jar` ของ LuckPerms เวอร์ชันล่าสุดก่อน  
หลังดาวน์โหลดแล้ว เปิดแผงควบคุมเซิร์ฟเวอร์ของคุณและหยุดเซิร์ฟเวอร์เพื่ออัปโหลดม็อดอย่างปลอดภัย

ไปที่ส่วน **ไฟล์** ของเซิร์ฟเวอร์และเปิดโฟลเดอร์ `mods` อัปโหลดไฟล์ `.jar` ของ LuckPerms ลงในโฟลเดอร์นี้ สำหรับคำอธิบายละเอียดเกี่ยวกับการติดตั้งม็อด เราแนะนำให้ดูที่คู่มือ [ติดตั้งม็อด](hytale-mods)

เมื่ออัปโหลดไฟล์เสร็จแล้ว ให้เริ่มเซิร์ฟเวอร์อีกครั้ง หลังจากเริ่มต้น เปิดคอนโซลเซิร์ฟเวอร์และตรวจสอบว่า LuckPerms ทำงานถูกต้องหรือไม่ หากติดตั้งสำเร็จ LuckPerms จะพิมพ์ข้อความเริ่มต้นในคอนโซล



## การมอบสิทธิ์เริ่มต้น

หลังติดตั้ง LuckPerms ต้องตั้งค่าก่อนใช้งานอย่างมีประสิทธิภาพ เพื่อเริ่มจัดการสิทธิ์ คุณต้องมอบสิทธิ์ให้ตัวเองใช้คำสั่ง LuckPerms ในคอนโซลหรือแชทในเกม ให้รันคำสั่งนี้:

```
lp user <playername> permission set luckperms.* true
```

คำสั่งนี้จะมอบสิทธิ์เต็มที่ในการใช้คำสั่ง LuckPerms ให้กับผู้เล่นที่เลือก



## การใช้เว็บเอดิเตอร์ของ LuckPerms

LuckPerms มาพร้อมเว็บเอดิเตอร์ที่ให้อินเทอร์เฟซสะดวกสำหรับสร้างกลุ่มและจัดการสิทธิ์ เพื่อสร้างเซสชันเอดิเตอร์ ให้พิมพ์:

```
lp editor
```

LuckPerms จะพิมพ์ลิงก์เอดิเตอร์ที่สร้างขึ้นในคอนโซล เปิดลิงก์นี้ในเบราว์เซอร์ของคุณเพื่อเข้าเว็บเอดิเตอร์

```
[LP] Preparing a new editor session, please wait...
[LP] Click the link below to open the editor:
https://luckperms.net/editor/XXXXXXXXXX
```

![img](https://screensaver01.zap-hosting.com/index.php/s/5Cx2sGY4axZ6TBo/preview)

ในเอดิเตอร์ ให้คัดลอก **ลิงก์คำสั่ง** ที่แสดง จากนั้นกลับไปที่คอนโซลเซิร์ฟเวอร์และวางลิงก์คำสั่งเพื่อสร้างความเชื่อถือและเชื่อมต่อเซสชันเอดิเตอร์กับเซิร์ฟเวอร์ของคุณ

```
/lp trusteditor XXXX-XXXX
```

เมื่อเอดิเตอร์เชื่อมต่อแล้ว คุณสามารถตั้งค่าผู้ใช้, กลุ่ม และสิทธิ์ได้โดยตรงผ่านเว็บอินเทอร์เฟซ หลังจากแก้ไขเสร็จ คลิก **Apply** เพื่อบันทึกการตั้งค่า หาก LuckPerms มีคำสั่ง apply ให้คัดลอกแล้ววางในคอนโซลเซิร์ฟเวอร์เพื่อยืนยันการเปลี่ยนแปลง



## การสร้างกลุ่มแอดมิน

เปิดลิงก์เอดิเตอร์ในเบราว์เซอร์และไปที่ส่วน **Groups** สร้างกลุ่มใหม่โดยใช้ไอคอนบวกและตั้งชื่อกลุ่มตามต้องการ คุณสามารถตั้งค่ารายละเอียดเพิ่มเติม เช่น ชื่อแสดง, น้ำหนัก, กลุ่มแม่ และคำนำหน้าหรือคำต่อท้ายได้ตามต้องการ

![img](https://screensaver01.zap-hosting.com/index.php/s/xEWHSBkxKy5q8qr/preview)

หลังสร้างกลุ่มแล้ว เลือกกลุ่ม `admin` ในเอดิเตอร์และเพิ่มสิทธิ์ เพื่อมอบสิทธิ์ Hytale เต็มรูปแบบให้กลุ่ม ให้ใส่:

```
hytale.*
```

สิทธิ์นี้จะมอบการเข้าถึงสิทธิ์ Hytale ทั้งหมด หลังเพิ่มสิทธิ์แล้ว ให้กำหนดผู้เล่นเข้ากลุ่ม  
ในเอดิเตอร์ เปิดส่วน **Users** เลือกผู้ใช้ที่ต้องการ และเพิ่มกลุ่ม `admin` ใน **Parent groups**

สุดท้าย คลิก **Apply** เพื่อบันทึกการเปลี่ยนแปลง หากมีคำสั่ง apply ให้วางในคอนโซลเซิร์ฟเวอร์



## คำสั่ง LuckPerms ที่ใช้บ่อย

LuckPerms สามารถจัดการได้ทั้งผ่านเว็บเอดิเตอร์หรือคำสั่งโดยตรง คำสั่งที่ใช้บ่อยสำหรับจัดการสิทธิ์มีดังนี้:

| คำสั่ง                                                      | คำอธิบาย                                                  |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| `/lp user <playername> parent add <group>`                   | เพิ่มผู้เล่นเข้าสู่กลุ่มสิทธิ์                              |
| `/lp user <playername> parent remove <group>`                | ลบผู้เล่นออกจากกลุ่มสิทธิ์                                 |
| `/lp user <playername> permission set <permission>`          | มอบสิทธิ์ให้ผู้เล่น                                        |
| `lp user <playername> permission settemp <permission> true <duration>` | มอบสิทธิ์ชั่วคราวให้ผู้เล่น โดยกำหนดระยะเวลา เช่น `1h` สำหรับ 1 ชั่วโมง หรือ `1m` สำหรับ 1 นาที |
| `/lp user <playername> permission unset <permission>`        | ถอนสิทธิ์จากผู้เล่น                                        |
| `/lp group <group> permission set <permission>`              | มอบสิทธิ์ให้กลุ่ม                                         |
| `/lp group <group> permission unset <permission>`            | ถอนสิทธิ์จากกลุ่ม                                         |



## สรุป

LuckPerms คือโซลูชันทรงพลังสำหรับการจัดการสิทธิ์ขั้นสูงบนเซิร์ฟเวอร์ Hytale ด้วยการใช้กลุ่ม, สิทธิ์ และเว็บเอดิเตอร์ คุณสามารถสร้างระบบบทบาทที่มีโครงสร้างและขยายตัวได้ตามชุมชนของคุณ

เมื่อกำหนดค่าเสร็จ LuckPerms จะช่วยให้การดูแลอันดับทีมงาน, สิทธิพิเศษผู้เล่น และการควบคุมการเข้าถึงเป็นเรื่องง่าย ในขณะที่ยังคงการบริหารเซิร์ฟเวอร์ให้สะอาด, สม่ำเสมอ และประสิทธิภาพดี

หากมีคำถามหรือขอความช่วยเหลือเพิ่มเติม อย่าลังเลที่จะติดต่อทีมซัพพอร์ตของเรา ซึ่งพร้อมช่วยเหลือคุณทุกวัน! 🙂

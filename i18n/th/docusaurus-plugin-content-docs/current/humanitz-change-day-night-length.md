## แนะนำ

วงจรวันและคืนใน HumanitZ มีผลโดยตรงต่อการเล่นเกม การมองเห็น ความยากในการเอาตัวรอด และจังหวะโดยรวม วันสั้นจะสร้างประสบการณ์เอาตัวรอดที่เข้มข้นขึ้น ในขณะที่ช่วงเวลากลางวันที่ยาวขึ้นจะให้เวลาผู้เล่นมากขึ้นในการสำรวจ สร้าง และเก็บทรัพยากร

## การตั้งค่า

ระยะเวลาของวันและคืนถูกควบคุมในไฟล์ `GameServerSettings.ini` คุณสามารถเข้าถึงไฟล์นี้ได้ในแผงควบคุมเซิร์ฟเวอร์ของคุณภายใต้เมนู **Configs** ในไฟล์คอนฟิก ให้ค้นหาพารามิเตอร์ดังนี้:

```
;Day duration in minutes
DayDur=40
;Night duration in minutes
NightDur=20
```

ค่าต่างๆ ถูกกำหนดเป็นนาที:

- `DayDur` – ควบคุมระยะเวลาของช่วงเวลากลางวัน  
- `NightDur` – ควบคุมระยะเวลาของช่วงเวลากลางคืน  

ถ้าต้องการเปลี่ยนความยาววงจร แค่ปรับตัวเลขให้เป็นระยะเวลาที่คุณต้องการ ตัวอย่างสำหรับวันยาวขึ้นและคืนสั้นลง:

```
DayDur=60
NightDur=15
```

หลังจากแก้ไขไฟล์แล้ว ให้บันทึกการเปลี่ยนแปลงและรีสตาร์ทเซิร์ฟเวอร์เพื่อให้การตั้งค่าใหม่มีผล



## สรุป

ยินดีด้วย! โดยการแก้ไขค่า `DayDur` และ `NightDur` ในไฟล์ `GameServerSettings.ini` คุณได้ปรับวงจรวันและคืนบนเซิร์ฟเวอร์ HumanitZ ของคุณเรียบร้อยแล้ว หากมีคำถามเพิ่มเติมหรือต้องการความช่วยเหลือ อย่าลังเลที่จะติดต่อทีมซัพพอร์ตของเรา ซึ่งพร้อมให้บริการทุกวันเพื่อช่วยคุณ! 🙂

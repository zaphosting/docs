## แนะนำ

ด้วยการใช้ phpMyAdmin คุณสามารถจัดการฐานข้อมูล MySQL ได้อย่างรวดเร็วและง่ายดายผ่านอินเทอร์เฟซเว็บ ZAP-Hosting มีฐานข้อมูล MySQL รวมอยู่ในผลิตภัณฑ์เซิร์ฟเวอร์เกม ซึ่งสามารถจัดการได้ทั้งผ่าน phpMyAdmin และโปรแกรมจัดการ MySQL ภายนอกอย่าง Navicat หรือ HeidiSQL เรายังมีคู่มือสำหรับเรื่องนี้ที่สามารถดูได้ที่ [การเข้าถึงฐานข้อมูลภายนอก](gameserver-database-external-access.md)

<InlineVoucher />

## การเข้าสู่ระบบ phpMyAdmin

อันดับแรกเราไปที่รายการฐานข้อมูลที่สร้างขึ้นสำหรับเซิร์ฟเวอร์เกม โดยเปิดเมนู "Databases" ในแถบด้านซ้ายของเซิร์ฟเวอร์ภายใต้แท็บ "Tools"

![](https://screensaver01.zap-hosting.com/index.php/s/ay9X6jBsCRWYdDB/preview)

ในหน้านี้จะแสดงฐานข้อมูลทั้งหมดสำหรับเซิร์ฟเวอร์นี้ การเข้าสู่ระบบ phpMyAdmin ง่ายมากเพราะระบบของเราจะเข้าสู่ระบบให้อัตโนมัติ คุณแค่คลิกที่ปุ่มสีน้ำเงิน "Manage"

:::info
ถ้าการเข้าสู่ระบบไม่ทำงานอัตโนมัติ ต้องเข้าสู่ระบบด้วยตนเองโดยใช้ข้อมูลผู้ใช้ที่เหมาะสม ข้อมูลนี้สามารถดูได้จากพื้นที่ที่ทำเครื่องหมายสีแดงในตัวอย่างนี้
:::


## เครื่องมือจัดการตาราง

ตอนแรกจะแสดงแค่หน้าเริ่มต้นของ phpMyAdmin เท่านั้น เพื่อดูและแก้ไขตารางของฐานข้อมูลนั้น ต้องเปิดฐานข้อมูลก่อน:

![](https://screensaver01.zap-hosting.com/index.php/s/DFTDGKm47ngaB9R/preview)

ตอนนี้จะแสดงตารางทั้งหมดของฐานข้อมูลและเครื่องมือต่างๆ ที่ช่วยให้คุณทำงานบางอย่างได้ เช่น การลบตารางโดยคลิกปุ่มสีแดง "Delete"

![](https://screensaver01.zap-hosting.com/index.php/s/dkPqEj5xgQoHgHD/preview)

เพื่อประหยัดเวลา คุณสามารถทำงานกับหลายตารางพร้อมกันได้ โดยคลิกที่กล่องเลือกทางซ้ายสุดทีละอันจนกว่าจะเลือกตารางที่ต้องการครบ หากต้องการเลือกทุกตารางในฐานข้อมูล สามารถใช้ปุ่ม "Select All" เพื่อเลือกตารางทั้งหมดโดยอัตโนมัติ จากนั้นเลือกการกระทำที่ต้องการจากเมนูแบบเลื่อนลงและกด "OK" เพื่อดำเนินการ

![](https://screensaver01.zap-hosting.com/index.php/s/Qip8TDLbeezT67F/preview)


## แก้ไขเนื้อหาตาราง

ค่าภายในตารางสามารถแก้ไขได้อย่างรวดเร็วและง่ายดายโดยการดับเบิลคลิกที่ช่องนั้นๆ แต่ต้องแน่ใจว่าตารางถูกตั้งค่าให้สามารถแก้ไขได้ ซึ่งจะเห็นได้จากปุ่มการกระทำสามปุ่มที่แสดงในกรณีนี้

![](https://screensaver01.zap-hosting.com/index.php/s/H7twJagPyR7L66w/preview)


## รันคำสั่ง SQL

ด้วย phpMyAdmin คุณยังสามารถรันคำสั่ง SQL ปกติได้ โดยต้องเลือกฐานข้อมูลก่อน จากนั้นเรียกเมนู "SQL" ที่ด้านบนของแถบเมนู

เมื่อเข้าสู่หน้าคำสั่งแล้ว คุณสามารถพิมพ์คำสั่ง SQL ที่ต้องการทั้งหมด จากนั้นกด "OK" เพื่อรันคำสั่ง

![](https://screensaver01.zap-hosting.com/index.php/s/jJcmLGcXAfxQgkH/preview)

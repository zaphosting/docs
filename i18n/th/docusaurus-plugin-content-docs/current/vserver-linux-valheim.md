## แนะนำ
คุณมี VPS Linux และต้องการติดตั้งบริการ Valheim เซิร์ฟเวอร์เฉพาะบนมันใช่ไหม? คุณมาถูกที่แล้ว ในคู่มือนี้เราจะอธิบายขั้นตอนทีละขั้นตอนในการติดตั้งบริการนี้บนเซิร์ฟเวอร์ Linux ของคุณผ่าน SteamCMD เราจะใช้ Ubuntu เป็นตัวอย่าง แต่กระบวนการควรจะคล้ายกันมากสำหรับดิสโทรอื่นๆ

:::tip
รู้ไหมว่าคุณสามารถติดตั้ง **ZAP GS/TS3 Interface** ของเราโดยตรงบน VPS ของคุณได้เลย ทำให้คุณตั้งค่าเซิร์ฟเวอร์เกม พร้อมการเชื่อมต่อโดยตรงกับแดชบอร์ด ZAP-Hosting ของคุณได้ในไม่กี่คลิก! เรียนรู้เพิ่มเติมเกี่ยวกับ [GS/TS3 Interface](vserver-linux-gs-interface.md)
:::

## การเตรียมตัว

เริ่มต้นด้วยการเชื่อมต่อกับ VPS ของคุณผ่าน SSH ใช้คู่มือ [SSH Initial Access](vserver-linux-ssh.md) ของเรา หากคุณต้องการความช่วยเหลือในการทำเช่นนี้

คุณจะต้องทำการตั้งค่า SteamCMD ครั้งแรกด้วยถ้านี่เป็นครั้งแรกที่คุณใช้บนเซิร์ฟเวอร์ Linux ของคุณ กรุณาใช้คู่มือ [SteamCMD Linux Setup](vserver-linux-steamcmd.md) ของเราและตรวจสอบให้แน่ใจว่า SteamCMD ถูกตั้งค่าเรียบร้อยก่อนดำเนินการต่อ

## การติดตั้ง

เริ่มโดยการล็อกอินเป็นผู้ใช้ `steam` และไปที่ไดเรกทอรีโฮมของผู้ใช้ `steam` เพื่อจัดระเบียบไฟล์
```
sudo -u steam -s
cd ~
```

เมื่อเข้าสู่ระบบแล้ว คุณสามารถเริ่มกระบวนการติดตั้งโดยใช้คำสั่งนี้เพื่อเริ่มติดตั้งผ่าน SteamCMD โดยตรงในผู้ใช้ `steam`
```
steamcmd +force_install_dir '/home/steam/Valheim-Server' +login anonymous +app_update 896660 validate +quit
```

โปรดรอให้ดาวน์โหลดเสร็จสมบูรณ์ อาจใช้เวลานานสำหรับเกมที่มีขนาดใหญ่ เมื่อสำเร็จ คุณจะเห็นข้อความยืนยันความสำเร็จปรากฏขึ้น

## การตั้งค่า

ตอนนี้คุณได้ตั้งค่าเซิร์ฟเวอร์ Valheim เสร็จเรียบร้อยแล้ว คุณสามารถปรับแต่งการตั้งค่าเซิร์ฟเวอร์เพิ่มเติมได้โดยแก้ไขไฟล์สตาร์ทโดยตรง

ไปที่ไดเรกทอรีหลักและเปิดไฟล์ `.sh` คุณสามารถแก้ไขพารามิเตอร์ได้ที่นี่
```
nano /home/steam/Valheim-Server/start_server.sh
```

## การเริ่มต้น & การเชื่อมต่อกับเซิร์ฟเวอร์ของคุณ

ถึงเวลาที่จะเริ่มเซิร์ฟเวอร์ของคุณแล้ว ไปที่ไดเรกทอรีเกมหลักและรันไฟล์เชลล์ **start_server.sh**
```
/home/steam/Valheim-Server/start_server.sh
```

คุณควรจะเห็นล็อกปรากฏในพรอมต์คำสั่งซึ่งบ่งบอกว่าการเริ่มต้นสำเร็จ โปรดทราบว่าการเริ่มต้นครั้งแรกอาจใช้เวลานานเนื่องจากการตั้งค่าทั้งหมด หรือคุณสามารถเชื่อมต่อโดยตรงโดยใช้แถบค้นหาด้านล่างในรายการเซิร์ฟเวอร์และค้นหา: `[your_ip_address]:2456`

:::info
ถ้าคุณไม่สามารถเชื่อมต่อและได้รับข้อผิดพลาด `PlayFab` ในคอนโซล คุณอาจต้องปิดการรองรับ crossplay เพื่อแก้ไขปัญหานี้ เนื่องจากเป็นปัญหาปัจจุบันของเวอร์ชัน Linux ให้รันคำสั่ง `nano /home/steam/Valheim-Server/start_server.sh` และลบแฟล็ก `-crossplay` ออก

ถ้าคุณต้องการ crossplay ทางเลือกคือการติดตั้งเวอร์ชัน Windows และใช้ **Wine** เป็นเลเยอร์ความเข้ากันได้ กรุณาใช้คู่มือ [Wine Compatibility Layer Setup](vserver-linux-wine.md) ของเราเพื่อตั้งค่านี้ เมื่อพร้อมแล้ว คุณสามารถติดตั้งเซิร์ฟเวอร์ Valheim เวอร์ชัน Windows ผ่าน SteamCMD โดยใช้คำสั่ง:
```
steamcmd +@sSteamCmdForcePlatformType windows +force_install_dir '/home/steam/Valheim-Server' +login anonymous +app_update 896660 validate +quit
```

เมื่อติดตั้งเสร็จ ใช้คำสั่งนี้เพื่อรันเวอร์ชัน Windows ผ่าน Wine แทน: `xvfb-run wine /home/steam/Valheim-Server/StartServer.bat`
:::

## สรุป

ยินดีด้วย คุณได้ติดตั้งและตั้งค่าเซิร์ฟเวอร์ Valheim บน VPS ของคุณเรียบร้อยแล้ว! ขั้นตอนต่อไป เราแนะนำให้ดูคู่มือ [Setup Linux Service](vserver-linux-create-gameservice.md) ของเรา ซึ่งครอบคลุมการตั้งค่าเซิร์ฟเวอร์เกมเฉพาะของคุณเป็นบริการ ซึ่งจะมีประโยชน์มากมาย เช่น การเปิดเซิร์ฟเวอร์อัตโนมัติเมื่อบูต, อัปเดตเซิร์ฟเวอร์อัตโนมัติ, การจัดการง่าย และการเข้าถึงล็อก รวมถึงฟีเจอร์อื่นๆ อีกเพียบ!

ถ้าคุณมีคำถามหรือปัญหาเพิ่มเติม ติดต่อทีมซัพพอร์ตของเราได้เลย พร้อมช่วยเหลือคุณทุกวัน!

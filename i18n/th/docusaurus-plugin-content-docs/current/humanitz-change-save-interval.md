## แนะนำ

ช่วงเวลาบันทึกกำหนดความถี่ที่เซิร์ฟเวอร์ HumanitZ ของคุณจะบันทึกสถานะโลกโดยอัตโนมัติ การบันทึกเป็นประจำช่วยป้องกันการสูญหายของข้อมูลในกรณีที่เกิดการแครชหรือปิดเครื่องโดยไม่คาดคิด ขณะที่ช่วงเวลาที่นานขึ้นอาจช่วยลดกิจกรรมของดิสก์เล็กน้อย

## การตั้งค่า

ช่วงเวลาบันทึกโลกถูกควบคุมในไฟล์ `GameServerSettings.ini` คุณสามารถเข้าถึงไฟล์นี้ได้ในแผงควบคุมเซิร์ฟเวอร์ของคุณภายใต้เมนู **Configs** ในไฟล์คอนฟิก ให้ค้นหาพารามิเตอร์ดังนี้:

```
;The interval in seconds at which the server saves the world. Set to 0 to disable auto-saving.
SaveIntervalSec=300
```

- `SaveIntervalSec=300` – เซิร์ฟเวอร์จะบันทึกโลกทุก 300 วินาที (5 นาที)  
- `SaveIntervalSec=0` – ปิดการบันทึกโลกอัตโนมัติ  

ค่าที่กำหนดเป็น **วินาที** เช่น:

- `600` = 10 นาที
- `900` = 15 นาที

ถ้าคุณต้องการเพิ่มช่วงเวลาเป็น 10 นาที:

```
SaveIntervalSec=600
```

หลังจากแก้ไขไฟล์แล้ว ให้บันทึกการเปลี่ยนแปลงและรีสตาร์ทเซิร์ฟเวอร์เพื่อให้การตั้งค่าใหม่มีผล

## สรุป

ยินดีด้วย! โดยการแก้ไขค่า `SaveIntervalSec` ในไฟล์ `GameServerSettings.ini` คุณได้ปรับช่วงเวลาบันทึกโลกอัตโนมัติบนเซิร์ฟเวอร์ HumanitZ ของคุณเรียบร้อยแล้ว

หากมีคำถามเพิ่มเติมหรือต้องการความช่วยเหลือ อย่าลังเลที่จะติดต่อทีมซัพพอร์ตของเรา ซึ่งพร้อมให้บริการคุณทุกวัน! 🙂

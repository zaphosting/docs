## บทนำ

การยืนยันตัวตนด้วยฮาร์ดแวร์โดยใช้ Passkey เป็นวิธีทันสมัยและต้านทานการฟิชชิ่งเพื่อปกป้องบัญชีผู้ใช้ของคุณ แทนที่จะใช้รหัสผ่านแบบเดิม จะใช้คู่กุญแจเข้ารหัสลับ กุญแจส่วนตัวจะถูกเก็บไว้อย่างปลอดภัยบนอุปกรณ์ของคุณและจะไม่ถูกส่งออกไปไหน การยืนยันตัวตนจะทำแบบท้องถิ่นโดยใช้การตรวจสอบทางชีวมิติ เช่น ลายนิ้วมือหรือการจดจำใบหน้า หรือถ้าไม่ก็ใช้ PIN ของอุปกรณ์แทน

## ความต้องการ

เพื่อใช้การยืนยันตัวตนด้วยฮาร์ดแวร์ผ่าน Passkey การตั้งค่าของคุณต้องตรงตามข้อกำหนดบางอย่าง คุณต้องมีอุปกรณ์ที่รองรับ เช่น สมาร์ทโฟน แท็บเล็ต หรือคอมพิวเตอร์ ระบบปฏิบัติการและเบราว์เซอร์ต้องรองรับ Passkey หรือ WebAuthn ต้องตั้งค่าการล็อกหน้าจอบนอุปกรณ์ เช่น ลายนิ้วมือ Face ID หรือ PIN ท้องถิ่น

## เปิดใช้งานการยืนยันตัวตน

เพื่อเปิดใช้งานการยืนยันตัวตนด้วยฮาร์ดแวร์ ให้เปิดการตั้งค่าโปรไฟล์ในพื้นที่ลูกค้าแล้วไปที่ส่วนความปลอดภัย (การตั้งค่าโปรไฟล์ → ความปลอดภัย) ในส่วนนี้คุณสามารถตั้งค่าตัวเลือกความปลอดภัยสำหรับบัญชีของคุณ เลือก การยืนยันตัวตนด้วยฮาร์ดแวร์ เพื่อดำเนินการตั้งค่าต่อ

หลังจากเลือกตัวเลือกนี้ จะมีหน้าต่างป๊อปอัปเปิดขึ้น ภายในหน้าต่างนี้ ให้คลิกที่ปุ่มที่เขียนว่า เพิ่มตัวยืนยันตัวตน เพื่อเริ่มกระบวนการตั้งค่า

![img](https://screensaver01.zap-hosting.com/index.php/s/oSEpGD5emG9ToxW/download)

การตั้งค่าจะดำเนินต่อด้วยการสร้าง Passkey สำหรับเข้าสู่ระบบ ZAP-Hosting ในขั้นตอนนี้ คุณจะถูกขอให้กำหนด PIN สำหรับกู้คืน PIN นี้จำเป็นสำหรับการกู้คืนการเข้าถึงในกรณีที่วิธีการยืนยันตัวตนหลักไม่สามารถใช้งานได้ชั่วคราว และควรเก็บไว้ให้ปลอดภัย

![img](https://screensaver01.zap-hosting.com/index.php/s/tiPnCDTAkieNHNP/download)

หลังจากยืนยันขั้นตอนเหล่านี้และตั้งค่าการยืนยันตัวตนด้วย Passkey สำเร็จ คุณจะถูกออกจากระบบโดยอัตโนมัติหนึ่งครั้ง นี่เป็นมาตรการความปลอดภัยปกติและเพื่อให้แน่ใจว่าวิธีการยืนยันตัวตนที่ตั้งค่าใหม่ถูกนำไปใช้เรียบร้อย

ในการเข้าสู่ระบบครั้งถัดไป หลังจากกรอกชื่อผู้ใช้และรหัสผ่าน คุณจะเห็นตัวเลือก การยืนยันตัวตนด้วยฮาร์ดแวร์ เลือกตัวเลือกนี้และยืนยันคำขอยืนยันตัวตนบนอุปกรณ์ของคุณเพื่อเสร็จสิ้นกระบวนการเข้าสู่ระบบ

## ปิดใช้งานการยืนยันตัวตน

หากต้องการปิดใช้งานการยืนยันตัวตนสองขั้นตอน ให้ไปที่ **การตั้งค่าโปรไฟล์ → ความปลอดภัย** หากผู้ใช้ยังสามารถเข้าสู่ระบบได้ ตัวยืนยันตัวตนที่ตั้งค่าไว้สามารถลบออกจากการตั้งค่าความปลอดภัยได้ตลอดเวลา

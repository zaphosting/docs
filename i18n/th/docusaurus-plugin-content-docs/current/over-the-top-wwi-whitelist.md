## แนะนำ

Whitelist ช่วยให้คุณจำกัดการเข้าถึงเซิร์ฟเวอร์ **Over the Top WWI** ของคุณ เพื่อให้เฉพาะผู้เล่นที่ได้รับอนุญาตเท่านั้นที่สามารถเข้าร่วมได้ ซึ่งเหมาะมากสำหรับเซิร์ฟเวอร์ส่วนตัว ชุมชน หรือสภาพแวดล้อมทดสอบที่ต้องการควบคุมการเข้าถึง

เมื่อเปิดใช้งาน whitelist จะมีเพียงผู้เล่นที่อยู่ในไฟล์ whitelist เท่านั้นที่สามารถเชื่อมต่อกับเซิร์ฟเวอร์ได้

<InlineVoucher />

## การตั้งค่า

Whitelist ถูกตั้งค่าในไฟล์คอนฟิกของเซิร์ฟเวอร์ คุณสามารถเข้าถึงและแก้ไขไฟล์นี้ได้ใน **แผงควบคุมเซิร์ฟเวอร์เกม** ของคุณภายใต้เมนู **Configs** ค้นหาและเปิดไฟล์คอนฟิก `ServerConfiguration.ini` ภายในไฟล์นี้ ให้หาพารามิเตอร์ดังนี้:

```
WhiteListIsEnabled = True
WhiteListPathString = Whitelist.txt
```

- `WhiteListIsEnabled` เปิดหรือปิดระบบ whitelist

- `False` → ปิดใช้งาน
- `True` → เปิดใช้งาน

- `WhiteListPathString` กำหนดไฟล์ที่ใช้เก็บ ID ผู้เล่นที่ได้รับอนุญาต

เมื่อเปิดใช้งาน whitelist จะมีเพียงผู้เล่นที่อยู่ในไฟล์ที่ระบุเท่านั้นที่สามารถเข้าร่วมเซิร์ฟเวอร์ได้

## การจัดการ whitelist

ไฟล์ whitelist โดยปกติจะอยู่ในไดเรกทอรีเซิร์ฟเวอร์ของคุณ และจะมีรายชื่อ ID ผู้เล่นที่ได้รับอนุญาต

ผู้เล่นแต่ละคนต้องถูกเพิ่มในบรรทัดใหม่ในไฟล์นี้:

```
SteamID1
SteamID2
SteamID3
```

ตรวจสอบให้แน่ใจว่าใช้ตัวระบุผู้เล่นที่ถูกต้อง มิฉะนั้นจะถูกปฏิเสธการเข้าถึง หลังจากอัปเดตไฟล์ whitelist หรือแก้ไข `ServerConfiguration.ini` ให้บันทึกการเปลี่ยนแปลงและรีสตาร์ทเซิร์ฟเวอร์ ระบบ whitelist จะถูกบังคับใช้ทันที

## สรุป

ยินดีด้วย! คุณตั้งค่า whitelist บน **เซิร์ฟเวอร์ Over the Top WWI** ของคุณสำเร็จแล้ว ซึ่งช่วยให้คุณควบคุมว่าใครสามารถเข้าถึงเซิร์ฟเวอร์ของคุณได้ และรักษาสภาพแวดล้อมที่เป็นส่วนตัวหรือมีการดูแล

หากมีคำถามเพิ่มเติมหรือต้องการความช่วยเหลือ อย่าลังเลที่จะติดต่อทีมซัพพอร์ตของเรา ซึ่งพร้อมให้บริการทุกวัน! 🙂

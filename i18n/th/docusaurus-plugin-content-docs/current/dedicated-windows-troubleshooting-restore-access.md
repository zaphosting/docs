## แนะนำ

มันเกิดขึ้นได้เร็วมาก... คุณพยายามล็อกอินเข้าเซิร์ฟเวอร์ Windows ของคุณ แต่รหัสผ่านไม่ถูกต้องหรือคุณลืมมันไปแล้ว ทันใดนั้นคุณถูกล็อกเอาท์และไม่สามารถเข้าถึงข้อมูลและแอปสำคัญได้ สถานการณ์นี้น่าหงุดหงิดแน่นอน แต่ยังไม่ใช่จุดจบ เพียงแค่ทำตามขั้นตอนด้านล่างนี้ คุณก็จะสามารถกู้คืนการเข้าถึงเซิร์ฟเวอร์เฉพาะ Windows ของคุณและทำงานต่อได้โดยมีการหยุดชะงักน้อยที่สุด

## การเตรียมตัว
การรีเซ็ตรหัสผ่านจากบัญชีผู้ดูแลระบบจะทำโดยใช้ไฟล์ ISO ของ Windows ตามนั้น ไฟล์ ISO เดียวกันที่ใช้ติดตั้งระบบปฏิบัติการ Windows Server ในตอนแรกจะถูกนำมาใช้

ขั้นตอนนี้สามารถทำได้ผ่าน [**การติดตั้งครั้งแรก**](dedicated-setup.md) หรือผ่าน **[ISO](dedicated-iso.md)** เลือกหรือระบุไฟล์ ISO ที่ใช้ในตอนติดตั้งตามวิธีที่คุณเลือก

เมานต์ไฟล์ ISO แล้วรีสตาร์ทระบบ หลังจากนั้นอินเทอร์เฟซการติดตั้ง Windows จะเปิดขึ้นอีกครั้งเมื่อระบบบูต ผลลัพธ์ควรเป็นแบบนี้:

![](https://screensaver01.zap-hosting.com/index.php/s/XGKfQrwdcmcabY6/preview)

## กู้คืนการเข้าถึง

ถึงเวลาที่จะรีเซ็ตการเข้าถึงบัญชีผู้ดูแลระบบแล้ว ทำตามขั้นตอนแรกของการตั้งค่าอีกครั้ง แต่คุณแค่ไปถึงจุดเริ่มต้นของการติดตั้งแล้วเลือก **ซ่อมแซมคอมพิวเตอร์ของคุณ (Repair your computer)**

![](https://screensaver01.zap-hosting.com/index.php/s/qwPgHyqNaQdsqzm/preview)

จะเปิดตัวเลือกขั้นสูงขึ้นมา คลิกที่ตัวเลือก **แก้ไขปัญหา (Troubleshoot)** แล้วเลือก **พรอมต์คำสั่ง (Command Prompt)** หน้าต่างพรอมต์คำสั่ง (cmd.exe) จะเปิดขึ้น

![](https://screensaver01.zap-hosting.com/index.php/s/BEan26iNkmzECJ3/download)

ตอนนี้ให้พิมพ์คำสั่งด้านล่างทีละคำสั่งในพรอมต์คำสั่ง:

```
d:
cd Windows
cd System32
move utilman.exe utilman.exe.bak
copy cmd.exe utilman.exe
net user administrator /active:yes
shutdown -r -t 0
```
:::warning

เลย์เอาต์แป้นพิมพ์สำหรับคอนโซล iLO HTML อาจแตกต่างจากของคุณเอง ทำให้สัญลักษณ์บางตัวอยู่บนปุ่มที่ต่างกัน โปรดระวังเพื่อให้พิมพ์คำสั่งได้ถูกต้อง

:::

หลังจากเซิร์ฟเวอร์รีบูต ที่หน้าจอเข้าสู่ระบบหลัก ให้กดปุ่ม **Win+U** พร้อมกัน ในหน้าต่างพรอมต์คำสั่ง ให้พิมพ์คำสั่งนี้:

```
net user Administrator YourNewPassword
exit
```

![](https://screensaver01.zap-hosting.com/index.php/s/TiKJZPdg2kj5LG3/download)

ตอนนี้รหัสผ่านของคุณถูกเปลี่ยนแล้ว แต่คุณต้องคืนค่าการเปลี่ยนแปลงไฟล์ **Utilman** ด้วย ให้รีสตาร์ทเซิร์ฟเวอร์อีกครั้งแล้วทำตามขั้นตอนนี้ซ้ำ ในพรอมต์คำสั่ง ให้พิมพ์คำสั่งต่อไปนี้:

```
d:
cd Windows
cd System32
del utilman.exe ren utilman.exe.bak utilman.exe
shutdown -r -t 0
```

## สรุป

เมื่อคุณทำขั้นตอนทั้งหมดเสร็จแล้ว คุณจะตั้งรหัสผ่านใหม่สำหรับผู้ใช้ผู้ดูแลระบบได้สำเร็จ ตอนนี้คุณสามารถใช้รหัสผ่านนี้ล็อกอินผ่านการเชื่อมต่อเดสก์ท็อประยะไกลได้อีกครั้ง

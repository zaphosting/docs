## แนะนำ

RCON (Remote Console) คือโปรโตคอลเครือข่ายที่ใช้สำหรับควบคุมเซิร์ฟเวอร์เกมจากระยะไกล ช่วยให้แอดมินสามารถสั่งคำสั่งบนเซิร์ฟเวอร์ได้โดยไม่ต้องเชื่อมต่อเข้าเกมโดยตรง ซึ่งทำให้สามารถจัดการผู้เล่น ปรับแต่งการตั้งค่า หรือเฝ้าดูการทำงานของเซิร์ฟเวอร์จากภายนอกได้

ใน **HumanitZ** คุณสามารถใช้ RCON เพื่อสั่งคำสั่งแอดมิน เช่น เตะผู้เล่น ส่งข้อความ หรือควบคุมพฤติกรรมของเซิร์ฟเวอร์ การเข้าถึงจะปลอดภัยด้วยรหัสผ่านและพอร์ตเฉพาะ

ข้อดีหลักของ RCON คือช่วยให้การบริหารเซิร์ฟเวอร์ **โดยไม่ต้องเชื่อมต่อเป็นผู้เล่นในเกม** ทำให้การจัดการระยะไกลมีความยืดหยุ่นและมีประสิทธิภาพมากขึ้น

## การตั้งค่า

ก่อนจะใช้ RCON ได้ ต้องเปิดใช้งานในคอนฟิกเซิร์ฟเวอร์ก่อน เปิดหน้าการตั้งค่าของเซิร์ฟเวอร์คุณแล้วเปิดใช้งานตัวเลือก rcon หรือจะปรับผ่านไฟล์คอนฟิกก็ได้ เปิดแผงควบคุมเซิร์ฟเวอร์เกมของคุณ ไปที่ Configs แล้วหาไฟล์ `GameServerSettings.ini`

เพิ่มหรือแก้ไขบรรทัดต่อไปนี้:

```
RCONEnabled=true
RCONPassword=your_secure_password
RCONPort=XXXXX
```

พอร์ต RCON ที่กำหนดไว้สามารถดูได้ใน **ภาพรวมพอร์ต** ภายในแดชบอร์ดเซิร์ฟเวอร์ของคุณ ตรวจสอบให้แน่ใจว่ารหัสผ่านและพอร์ตตรงกับการตั้งค่าของคุณ

หลังจากแก้ไขแล้ว ให้บันทึกไฟล์และ **รีสตาร์ทเซิร์ฟเวอร์** เพื่อให้การตั้งค่ามีผล

## การเชื่อมต่อผ่าน RCON

เพื่อเชื่อมต่อกับเซิร์ฟเวอร์ HumanitZ ของคุณผ่าน RCON คุณสามารถใช้เครื่องมือบรรทัดคำสั่ง **rcon-cli** ซึ่งมีให้ดาวน์โหลดจาก GitHub อย่างเป็นทางการ:
https://github.com/gorcon/rcon-cli

เมื่อติดตั้งแล้ว คุณสามารถเชื่อมต่อเซิร์ฟเวอร์ด้วยคำสั่งนี้:

```bash
rcon-cli -a <IP>:<PORT> -p <PASSWORD> command
```

แทนที่ตัวแปรด้วยที่อยู่ IP ของเซิร์ฟเวอร์ พอร์ต RCON และรหัสผ่านของคุณ

คำสั่งนี้ช่วยให้คุณสั่งคำสั่ง RCON ได้โดยตรงจากเครื่องของคุณโดยไม่ต้องเปิดเซสชันแบบโต้ตอบ

## คำสั่ง RCON

เมื่อเชื่อมต่อผ่าน RCON แล้ว คุณสามารถสั่งคำสั่งแอดมินต่าง ๆ บนเซิร์ฟเวอร์ HumanitZ ของคุณได้ คำสั่งที่ใช้ได้ขึ้นอยู่กับการพัฒนาเกม แต่โดยทั่วไปจะมีคำสั่งจัดการผู้เล่นและควบคุมเซิร์ฟเวอร์ดังนี้

| คำสั่ง | คำอธิบาย |
|----------------------------------|------------------------------------------|
| `broadcast <message>`| ส่งข้อความไปยังผู้เล่นทั้งหมด |
| `kick <playername>`| เตะผู้เล่นออกจากเซิร์ฟเวอร์ |
| `ban <playername>` | แบนผู้เล่น |
| `unban <playername>` | ยกเลิกแบนผู้เล่น |
| `save` | บันทึกสถานะเกมปัจจุบัน |
| `restart`| รีสตาร์ทเซิร์ฟเวอร์ |
| `shutdown` | ปิดเซิร์ฟเวอร์ |
| `whitelist add <steamid>`| เพิ่มผู้เล่นในรายชื่อขาว |
| `whitelist remove <steamid>` | ลบผู้เล่นออกจากรายชื่อขาว |

## สรุป

ยินดีด้วย! คุณตั้งค่าและใช้งาน **RCON สำหรับเซิร์ฟเวอร์ HumanitZ ของคุณ** ได้สำเร็จแล้ว ซึ่งช่วยให้คุณจัดการเซิร์ฟเวอร์จากระยะไกล สั่งคำสั่งแอดมิน และควบคุมเซิร์ฟเวอร์ได้โดยไม่ต้องเข้าเกม

ถ้ามีคำถามเพิ่มเติมหรือต้องการความช่วยเหลือ ติดต่อทีมซัพพอร์ตของเราได้เลย พร้อมให้บริการทุกวัน! 🙂

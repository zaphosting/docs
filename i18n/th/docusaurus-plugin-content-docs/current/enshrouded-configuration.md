## แนะนำ

Enshrouded ตอนเปิดตัวมีพารามิเตอร์การตั้งค่าบางอย่างที่คุณสามารถปรับแต่งได้ตามใจชอบ ในคู่มือนี้ เราจะสำรวจพารามิเตอร์การตั้งค่าที่มีอยู่ทั้งหมดในตอนนี้และอธิบายแต่ละอันอย่างละเอียด

## การเข้าถึงไฟล์การตั้งค่าของคุณ

อันดับแรก คุณต้องเข้าถึงไฟล์การตั้งค่าเพื่อแก้ไขพารามิเตอร์ใด ๆ มีหลายวิธีในการแก้ไขไฟล์นี้

#### ผ่านไฟล์ Config บนเว็บอินเทอร์เฟซ

:::note
เซิร์ฟเวอร์ต้องถูกหยุดก่อนจึงจะแก้ไข config ได้ การแก้ไข config แล้วรีสตาร์ทเซิร์ฟเวอร์จะทำให้การเปลี่ยนแปลงทั้งหมดถูกยกเลิก
:::

อีกทางเลือกสำหรับผู้ใช้ที่ต้องการแก้ไขไฟล์ดิบโดยตรง คุณสามารถเข้าถึงได้โดยไปที่ส่วน **Configs** บนเว็บอินเทอร์เฟซของเซิร์ฟเวอร์เกมของคุณแล้วกดปุ่มแก้ไขไฟล์สีน้ำเงินตามภาพด้านล่าง:

![](https://github.com/zaphosting/docs/assets/42719082/dc4699bc-33fa-447d-ac59-2960b3b080f9)

จะเปิดตัวแก้ไขข้อความบนเว็บไซต์ให้คุณแก้ไขได้เลย

</TabItem>

#### ผ่าน FTP

:::note
เซิร์ฟเวอร์ต้องถูกหยุดก่อนจึงจะแก้ไข config ได้ การแก้ไข config แล้วรีสตาร์ทเซิร์ฟเวอร์จะทำให้การเปลี่ยนแปลงทั้งหมดถูกยกเลิก
:::

วิธีสุดท้ายในการเข้าถึงไฟล์ดิบคือผ่าน FTP หากคุณไม่คุ้นเคยกับการใช้ FTP เราแนะนำให้ดูที่ [คู่มือการเข้าถึงผ่าน FTP](gameserver-ftpaccess.md) อย่างไรก็ตาม วิธีนี้ใช้เวลานานกว่า และถ้าคุณต้องการแก้ไขเนื้อหาไฟล์โดยตรง เราแนะนำให้ใช้ส่วน **Configs** บนเว็บอินเทอร์เฟซของเซิร์ฟเวอร์เกมตามที่กล่าวไว้ข้างต้น

</TabItem>

## ตัวเลือกการตั้งค่าเซิร์ฟเวอร์

นี่คือตัวเลือกการตั้งค่าเซิร์ฟเวอร์ที่มีให้โดยเกมสำหรับปรับแต่งเซิร์ฟเวอร์ของคุณ โดยเฉพาะชื่อเซิร์ฟเวอร์ รหัสผ่าน และการตั้งค่าภายในบางอย่าง

:::tip
เราไม่แนะนำให้เปลี่ยนพอร์ต, IP หรือการตั้งค่าไดเรกทอรี เพราะเกมเซิร์ฟเวอร์จะตั้งค่าให้อัตโนมัติ การเปลี่ยนแปลงค่าเหล่านี้อาจทำให้เซิร์ฟเวอร์เกมของคุณเสียหายได้!
:::

| ชื่อพารามิเตอร์ | ตัวอย่าง                                 | คำอธิบาย                                                 |
| -------------- | --------------------------------------- | ----------------------------------------------------------- | 
| name           | ZAP-Hosting Docs Test                   | ชื่อเซิร์ฟเวอร์ของคุณ                                      |
| password       | iLoveZAP!2024                           | ตั้งรหัสผ่านล็อกเซิร์ฟเวอร์ (ส่วนตัว)                      |
| saveDirectory  | ./savegame (ค่าเริ่มต้น)                | ตั้งไดเรกทอรีสำหรับเก็บข้อมูลเซฟ                          |
| logDirectory   | ./logs (ค่าเริ่มต้น)                    | ตั้งไดเรกทอรีสำหรับเก็บบันทึกล็อก                          |
| ip             | 123.123.123.123                         | ที่อยู่ IP สาธารณะที่เซิร์ฟเวอร์ใช้งาน                      |
| gamePort       | 15636 (ค่าเริ่มต้น)                     | พอร์ตสาธารณะที่เซิร์ฟเวอร์ใช้งาน                            |
| queryPort      | 15637 (ค่าเริ่มต้น)                     | พอร์ตสาธารณะที่สามารถเข้าถึงเซิร์ฟเวอร์ได้                 |
| slotCount      | 4                                       | จำนวนผู้เล่นสูงสุดที่สามารถเข้าร่วมเซิร์ฟเวอร์ได้ (สูงสุด 16) |

หลังจากแก้ไขเสร็จแล้ว อย่าลืมบันทึกไฟล์ในที่ที่เหมาะสมและรีสตาร์ทเซิร์ฟเวอร์ของคุณ ครั้งถัดไปที่เซิร์ฟเวอร์เริ่มทำงาน มันจะใช้การตั้งค่าที่คุณกำหนดไว้

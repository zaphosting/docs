## แนะนำ

การตั้งค่าเวลาและสภาพแวดล้อมใน **Over the Top WWI** มีผลต่อการมองเห็น บรรยากาศ และสภาพแวดล้อมการเล่นโดยรวม โดยการปรับค่านี้ คุณสามารถสร้างสนามรบในช่วงกลางวันที่สว่างไสว สถานการณ์มืด หรือสภาพแวดล้อมตามฤดูกาลที่หลากหลายได้

การตั้งค่าเหล่านี้ช่วยให้คุณปรับแต่งรูปลักษณ์และความรู้สึกของเซิร์ฟเวอร์ และเพิ่มความสมจริงให้กับผู้เล่นของคุณ

## การตั้งค่า

การตั้งค่าเวลาและสภาพแวดล้อมจะถูกกำหนดในไฟล์การตั้งค่าเซิร์ฟเวอร์ คุณสามารถเข้าถึงและแก้ไขไฟล์นี้ได้ใน **แผงควบคุมเซิร์ฟเวอร์เกม** ของคุณภายใต้เมนู **Configs** ค้นหาและเปิดไฟล์การตั้งค่า `ServerConfiguration.ini` ภายในไฟล์นี้ ให้ค้นหาพารามิเตอร์ดังต่อไปนี้:

```
TimeOfDay = 8
Season = 0
TemperatureinCelcius = 0
```

- `TimeOfDay` กำหนดชั่วโมงในเกม
- `Season` ควบคุมฤดูกาลปัจจุบัน

- `0` → ค่าเริ่มต้น
- `1` → ฤดูใบไม้ผลิ
- `2` → ฤดูร้อน
- `3` → ฤดูใบไม้ร่วง
- `4` → ฤดูหนาว

- `TemperatureinCelcius` ปรับอุณหภูมิของสภาพแวดล้อม
- ค่านี้มีผลต่อบรรยากาศและเอฟเฟกต์สภาพแวดล้อม

การตั้งค่าเหล่านี้ช่วยให้คุณสร้างสถานการณ์ที่แตกต่างกัน เช่น การต่อสู้ในช่วงเช้าตรู่ การต่อสู้กลางคืน หรือสภาพแวดล้อมตามฤดูกาล

หลังจากแก้ไขไฟล์ `ServerConfiguration.ini` แล้ว ให้บันทึกไฟล์และรีสตาร์ทเซิร์ฟเวอร์ของคุณ การตั้งค่าสภาพแวดล้อมใหม่จะถูกนำไปใช้โดยอัตโนมัติ

## สรุป

ยินดีด้วย! คุณได้ตั้งค่าเวลาและสภาพแวดล้อมบน **เซิร์ฟเวอร์ Over the Top WWI** ของคุณเรียบร้อยแล้ว ซึ่งช่วยให้คุณปรับแต่งบรรยากาศและสร้างประสบการณ์การเล่นเกมที่ไม่เหมือนใครสำหรับผู้เล่นของคุณ

หากมีคำถามเพิ่มเติมหรือต้องการความช่วยเหลือ อย่าลังเลที่จะติดต่อทีมซัพพอร์ตของเรา ซึ่งพร้อมให้บริการคุณทุกวัน! 🙂

## แนะนำ

ข้อความเซิร์ฟเวอร์อัตโนมัติเป็นวิธีง่ายๆ แต่ทรงพลังในการสื่อสารข้อมูลสำคัญกับผู้เล่นโดยตรงในเกม คุณสามารถใช้แสดงกฎเซิร์ฟเวอร์ ประกาศกิจกรรม แชร์ทิปส์ที่เป็นประโยชน์ หรือเตือนความจำเป็นระยะๆ โดยไม่ต้องให้แอดมินมาคอยพิมพ์เอง

ด้วยการใช้ **งานที่ตั้งเวลาไว้ (Scheduled Tasks)** ข้อความเซิร์ฟเวอร์จะถูกส่งเข้าแชทในเกม **CS:GO** อัตโนมัติผ่านการรันคำสั่งที่ตั้งไว้ตามเวลาที่กำหนด

## วิธีการทำงานของข้อความอัตโนมัติ

ข้อความอัตโนมัติจะถูกสร้างขึ้นโดยการตั้งงานที่รันคำสั่งเซิร์ฟเวอร์ในเวลาหรือช่วงเวลาที่กำหนด คำสั่งนี้สามารถใช้ส่งข้อความแชท, ประกาศ หรือแจ้งเตือนในเซิร์ฟเวอร์ได้

เมื่อกำหนดค่าเสร็จ งานจะทำงานอัตโนมัติในเบื้องหลังและแสดงข้อความให้ผู้เล่นทุกคนที่เชื่อมต่ออยู่เห็น ซึ่งช่วยให้การสื่อสารเป็นไปอย่างต่อเนื่องโดยไม่ต้องให้ทีมงานออนไลน์ตลอดเวลา



## การสร้างข้อความอัตโนมัติ

เพื่อสร้างข้อความเซิร์ฟเวอร์อัตโนมัติ ให้เปิดแดชบอร์ดเซิร์ฟเวอร์เกมของคุณแล้วไปที่ส่วน **งานที่ตั้งเวลาไว้ (Scheduled Tasks)** สร้างงานใหม่และเลือกตัวเลือกให้รัน **คำสั่ง (command)** ในช่องคำสั่ง ให้ใส่คำสั่งข้อความที่เซิร์ฟเวอร์เกมของคุณรองรับ

```
<message command> <your message text>
```

แทนที่คำสั่งด้วยคำสั่งเกมหรือคำสั่งจากปลั๊กอินที่คุณต้องการใช้เพื่อส่งข้อความ และ `<your message text>` ด้วยข้อความที่คุณอยากให้ผู้เล่นเห็น

จากนั้นคุณสามารถกำหนดความถี่ในการส่งข้อความ เช่น ทุกไม่กี่นาที ทุกชั่วโมง หรือเวลาที่กำหนดในแต่ละวัน สำหรับคำแนะนำละเอียดเกี่ยวกับการสร้างงานที่ตั้งเวลาไว้ ดูได้ที่ [คู่มือ งานที่ตั้งเวลาไว้](gameserver-scheduled-tasks.md)

## การปรับแต่งช่วงเวลาข้อความ

งานที่ตั้งเวลาไว้ช่วยให้คุณควบคุมได้เต็มที่ว่าอยากให้ข้อความแสดงเมื่อไหร่และบ่อยแค่ไหน ตัวอย่างการใช้งานทั่วไป เช่น:

- แสดงกฎเซิร์ฟเวอร์ทุกไม่กี่นาที
- ประกาศกิจกรรมที่จะมาถึงในเวลาที่กำหนด
- ส่งทิปส์การเล่นให้ผู้เล่นใหม่หรือผู้เล่นที่กลับมา
- เตือนลิงก์ Discord หรือเว็บไซต์

ช่วงเวลาสามารถปรับเปลี่ยนได้ตลอดเวลาโดยไม่ต้องรีสตาร์ทเซิร์ฟเวอร์ ทำให้คุณปรับเปลี่ยนได้อย่างยืดหยุ่นตามการเติบโตของเซิร์ฟเวอร์



## ข้อควรทราบสำคัญ

ฟีเจอร์ข้อความเซิร์ฟเวอร์อัตโนมัติขึ้นอยู่กับความสามารถของเกมและคำสั่งที่มีให้ใช้ แม้ว่าการส่งข้อความตามเวลาที่ตั้งไว้ด้วยคำสั่งเซิร์ฟเวอร์มาตรฐานจะทำได้ แต่ก็อาจจำกัดเรื่องการจัดรูปแบบ เงื่อนไข หรือการปรับแต่งขั้นสูง

ถ้าเกมรองรับปลั๊กอิน, ม็อด หรือส่วนเสริมที่ให้ความยืดหยุ่นและตัวเลือกปรับแต่งมากขึ้น การใช้โซลูชันเหล่านั้นอาจคุ้มค่า เพราะระบบส่งข้อความเฉพาะทางจะมีฟีเจอร์ครบถ้วนกว่าและเหมาะกับเซิร์ฟเวอร์ที่ซับซ้อนหรือปรับแต่งเยอะ



## สรุป

ข้อความเซิร์ฟเวอร์อัตโนมัติเป็นวิธีที่น่าเชื่อถือในการแจ้งข่าวสารและสร้างความผูกพันกับผู้เล่นโดยไม่ต้องใช้แรงงานคน ด้วยการใช้งานที่ตั้งเวลาไว้เพื่อรันคำสั่งส่งข้อความ แอดมินเซิร์ฟเวอร์จะมั่นใจได้ว่าการสื่อสารเป็นไปอย่างสม่ำเสมอทั่วทั้งเซิร์ฟเวอร์

เมื่อเซ็ตอัพเสร็จ ข้อความอัตโนมัติจะทำงานเองและสามารถนำกลับมาใช้ใหม่ได้ง่ายๆ เพียงแค่ปรับคำสั่งและข้อความเฉพาะเกมให้เหมาะสม

ถ้ามีคำถามหรือต้องการความช่วยเหลือ อย่าลังเลที่จะติดต่อทีมซัพพอร์ตของเรา ที่พร้อมช่วยเหลือคุณทุกวัน! 🙂

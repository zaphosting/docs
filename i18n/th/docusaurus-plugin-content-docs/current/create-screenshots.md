## บทนำ

การสร้างและแชร์สกรีนช็อตช่วยเพิ่มประสิทธิภาพการสื่อสารกับฝ่ายซัพพอร์ตด้วยการให้ข้อมูลภาพที่ชัดเจน คู่มือนี้จะอธิบายวิธีสร้างสกรีนช็อตบนคอมพิวเตอร์ของคุณและอัปโหลดโดยตรงไปยังตั๋วซัพพอร์ต

## การสร้างสกรีนช็อต

วิธีที่ดีที่สุดสำหรับการแชร์สกรีนช็อตอย่างปลอดภัยคือการสร้างสกรีนช็อตบนอุปกรณ์ของคุณเองและอัปโหลดโดยตรงไปยังตั๋วซัพพอร์ตผ่านการลากแล้วปล่อย วิธีนี้ช่วยให้สกรีนช็อตของคุณเป็นส่วนตัวและแชร์ได้อย่างปลอดภัยโดยไม่ต้องพึ่งพาลิงก์ภายนอก



### Windows: Snipping Tool

Windows มาพร้อมกับ *Snipping Tool* ในตัว ที่ช่วยให้จับภาพเฉพาะส่วน หน้าต่าง หรือทั้งหน้าจอและบันทึกภาพไว้ในเครื่องได้

1. เปิด **Snipping Tool** โดยค้นหาในเมนู Start
2. คลิก **New** เพื่อเริ่มจับภาพสกรีนช็อต
3. เลือกพื้นที่ที่ต้องการจับภาพ
4. บันทึกสกรีนช็อตไว้ในตำแหน่งที่สะดวกบนคอมพิวเตอร์ของคุณ
5. อัปโหลดไฟล์สกรีนช็อตที่บันทึกไว้โดยตรงไปยังตั๋วซัพพอร์ตของคุณด้วยการลากแล้วปล่อยลงในอินเทอร์เฟซตั๋ว

### Linux: Screenshot Tool หรือ FlameShot

ลีนุกซ์ส่วนใหญ่จะมาพร้อมกับเครื่องมือ *Screenshot* เริ่มต้น หรือจะใช้เครื่องมือเสริมอย่าง *FlameShot* ที่มีฟีเจอร์ขั้นสูงก็ได้

1. เปิดเครื่องมือ Screenshot หรือ FlameShot
2. เลือกพื้นที่หรือหน้าต่างที่ต้องการจับภาพ
3. บันทึกสกรีนช็อตไว้ในเครื่อง
4. อัปโหลดไฟล์สกรีนช็อตที่บันทึกไว้โดยตรงไปยังตั๋วซัพพอร์ตของคุณด้วยการลากแล้วปล่อย

## การอัปโหลดสกรีนช็อตไปยังฝ่ายซัพพอร์ต

หลังจากบันทึกสกรีนช็อตไว้ในเครื่องแล้ว ให้เปิดตั๋วซัพพอร์ตของคุณในแผงควบคุม ZAP-Hosting หรือพอร์ทัลซัพพอร์ต

- ลากและปล่อยไฟล์สกรีนช็อตลงในพื้นที่ข้อความของตั๋ว
- รอจนการอัปโหลดเสร็จสมบูรณ์
- ส่งตั๋วพร้อมไฟล์สกรีนช็อตแนบไปด้วย

วิธีนี้ช่วยหลีกเลี่ยงการแชร์ลิงก์ภายนอกและรักษาความปลอดภัยข้อมูลของคุณ

:::info แชร์สกรีนช็อตอย่างปลอดภัย
หลีกเลี่ยงการแชร์สกรีนช็อตแบบสาธารณะหรือผ่านบริการอัปโหลดของบุคคลที่สาม หากสกรีนช็อตของคุณมีข้อมูลที่ละเอียดอ่อนหรือส่วนตัว อย่าลืมเบลอหรือปิดบังข้อมูลก่อนอัปโหลดนะ
:::

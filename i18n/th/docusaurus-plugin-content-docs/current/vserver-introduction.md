## การตั้งค่า

เรามีตัวเลือกการตั้งค่าต่างๆ สำหรับผลิตภัณฑ์ VPS ของเรา ที่เหมาะกับทุกการใช้งาน การตั้งค่าของเราประกอบด้วยตัวเลือกดังนี้:

| องค์ประกอบ | ตัวเลือกการตั้งค่า |
| -------------------------------- | ------------------------- |
| **CPU** | AMD EPYC ซีรีส์ 7000 |
| **คอร์ CPU**              | 4-64 คอร์ |
| **RAM**              | 4-128 GB RAM     |
| **พื้นที่จัดเก็บ**               | 25-2000 GB                                   |
| **ระบบปฏิบัติการ** | Debian (11,12,13) Ubuntu (20.04,22.04,24.04), Windows Server 2022/2025 และ ISO แบบกำหนดเอง |
| **ที่อยู่ IP** | 1-10 ที่อยู่ IP  |
| **แบนด์วิดท์**     | 1,000-5,000 Mbit/s ขาเข้า/ขาออก |
| **ทราฟฟิก**                      | 33 TB/เดือน       |



## การใช้งาน

ด้วยราคาที่คุ้มค่า ประสิทธิภาพสูง สิทธิ์การเข้าถึงที่กว้างขวาง และความเป็นไปได้ที่มากมาย เว็บโฮสติ้งจึงสามารถใช้ได้กับหลายการใช้งาน เช่น:

- เว็บไซต์
- ฐานข้อมูล
- โซลูชันคลาวด์
- เซิร์ฟเวอร์เกมและเซิร์ฟเวอร์เสียง
- และอื่นๆ!

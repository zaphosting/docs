## แนะนำ
คุณมีเซิร์ฟเวอร์เฉพาะ Linux และต้องการติดตั้งบริการ Soulmask Dedicated Server บนเซิร์ฟเวอร์ของคุณใช่ไหม? คุณมาถูกที่แล้ว ในคู่มือนี้ เราจะอธิบายขั้นตอนทีละขั้นตอนในการติดตั้งบริการนี้บนเซิร์ฟเวอร์ Linux ของคุณผ่านการใช้ SteamCMD เราจะใช้ Ubuntu เป็นตัวอย่าง แต่กระบวนการควรจะคล้ายกันมากสำหรับดิสทริบิวชันอื่นๆ

:::tip
คุณรู้ไหมว่าคุณสามารถติดตั้ง **ZAP GS/TS3 Interface** ของเราโดยตรงบนเซิร์ฟเวอร์เฉพาะของคุณได้เลย ทำให้คุณสามารถตั้งค่าบริการเซิร์ฟเวอร์เกม พร้อมการเชื่อมต่อโดยตรงกับแดชบอร์ด ZAP-Hosting ของคุณได้ในไม่กี่คลิก! เรียนรู้เพิ่มเติมเกี่ยวกับ [GS/TS3 Interface](dedicated-linux-gs-interface.md)
:::

## การเตรียมตัว

เริ่มต้นด้วยการเชื่อมต่อกับเซิร์ฟเวอร์เฉพาะของคุณผ่าน SSH หากคุณต้องการความช่วยเหลือในการทำเช่นนี้ ให้ใช้คู่มือ [SSH Initial Access](dedicated-linux-ssh.md) ของเรา

คุณจะต้องทำการตั้งค่า SteamCMD ครั้งแรกหากนี่เป็นครั้งแรกที่คุณใช้บนเซิร์ฟเวอร์ Linux ของคุณ โปรดใช้คู่มือ [SteamCMD Linux Setup](dedicated-linux-steamcmd.md) ของเราและตรวจสอบให้แน่ใจว่า SteamCMD ถูกตั้งค่าอย่างสมบูรณ์ก่อนดำเนินการต่อ

## การติดตั้ง

เริ่มต้นด้วยการล็อกอินเป็นผู้ใช้ `steam` และไปที่ไดเรกทอรีหลัก `home/steam` ของผู้ใช้เพื่อจัดระเบียบให้เรียบร้อย
```
sudo -u steam -s
cd ~
```

เมื่อเข้าสู่ระบบแล้ว คุณสามารถเริ่มกระบวนการติดตั้งโดยใช้คำสั่งต่อไปนี้เพื่อเริ่มการติดตั้งผ่าน SteamCMD โดยตรงในผู้ใช้ `steam`
```
steamcmd +force_install_dir '/home/steam/Soulmask-Server' +login anonymous +app_update 3017300 validate +quit
```

โปรดอดทนรอจนกว่าการดาวน์โหลดจะเสร็จสมบูรณ์ อาจใช้เวลานานสำหรับเกมที่มีขนาดใหญ่ เมื่อสำเร็จ คุณจะเห็นข้อความยืนยันความสำเร็จปรากฏขึ้น

## การกำหนดค่า

ในขั้นตอนนี้ คุณได้ตั้งค่าเซิร์ฟเวอร์ Soulmask เสร็จเรียบร้อยแล้ว คุณสามารถปรับแต่งการตั้งค่าเซิร์ฟเวอร์เพิ่มเติมได้ผ่านไฟล์การกำหนดค่าที่อยู่ในไดเรกทอรีของเซิร์ฟเวอร์ของคุณ

คุณจะสามารถปรับพารามิเตอร์ทั้งหมดได้โดยการเข้าถึงและแก้ไขไฟล์การกำหนดค่า **Engine.ini** ที่อยู่ในโฟลเดอร์ Saved
```
nano /home/steam/Soulmask-Server/Engine/Saved/Config/LinuxServer/Engine.ini
```

## การเริ่มต้น & การเชื่อมต่อกับเซิร์ฟเวอร์ของคุณ

ถึงเวลาที่จะเริ่มเซิร์ฟเวอร์ของคุณแล้ว ไปที่ไดเรกทอรีเกมหลักและรันไฟล์ปฏิบัติการ **StartServer.sh** โดยใช้คำสั่งด้านล่างนี้
```
/home/steam/Soulmask-Server/StartServer.sh
```

คุณควรจะเห็นบันทึกปรากฏในพรอมต์คำสั่งของคุณ ซึ่งบ่งบอกว่าการเริ่มต้นเซิร์ฟเวอร์สำเร็จ โปรดทราบว่าการเริ่มต้นครั้งแรกอาจใช้เวลานานเนื่องจากการตั้งค่าทั้งหมด นอกจากนี้ คุณยังสามารถเชื่อมต่อโดยตรงได้โดยใช้แถบค้นหาด้านล่างในรายการเซิร์ฟเวอร์และค้นหาด้วย: `[your_ip_address]:18888`

## สรุป

ยินดีด้วย คุณได้ติดตั้งและกำหนดค่าเซิร์ฟเวอร์ Soulmask บนเซิร์ฟเวอร์เฉพาะของคุณสำเร็จแล้ว! ขั้นตอนต่อไป เราแนะนำให้ดูคู่มือ [Setup Linux Service](dedicated-linux-create-gameservice.md) ของเรา ซึ่งครอบคลุมการตั้งค่าเซิร์ฟเวอร์เกมเฉพาะของคุณเป็นบริการ ซึ่งจะมอบประโยชน์มากมาย เช่น การเปิดเซิร์ฟเวอร์อัตโนมัติเมื่อบูต, การอัปเดตเซิร์ฟเวอร์อัตโนมัติ, การจัดการและเข้าถึงบันทึกได้ง่าย และอื่นๆ อีกมากมาย!

หากคุณมีคำถามหรือปัญหาเพิ่มเติม โปรดติดต่อทีมซัพพอร์ตของเรา ซึ่งพร้อมช่วยเหลือคุณทุกวัน!

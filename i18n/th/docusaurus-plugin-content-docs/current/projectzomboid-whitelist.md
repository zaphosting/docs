## แนะนำ

รายชื่อขาวคือโหมดควบคุมการเข้าถึงที่ต้องให้คุณสร้างบัญชีผู้ใช้ล่วงหน้าก่อนที่พวกเขาจะเข้าร่วมได้ ใน Project Zomboid วิธีนี้มักทำโดยการปิดการลงทะเบียนแบบเปิดในไฟล์ตั้งค่าเซิร์ฟเวอร์ แล้วเพิ่มผู้ใช้ผ่านคำสั่งคอนโซล

## การเปิดใช้งานรายชื่อขาว

เชื่อมต่อกับเซิร์ฟเวอร์ของคุณผ่าน FTP แล้วเปิดไฟล์ตั้งค่าเซิร์ฟเวอร์ ซึ่งโดยปกติจะอยู่ที่ `Zomboid/Server/` และตั้งชื่อตามเซิร์ฟเวอร์ของคุณ เช่น `servertest.ini` หรือ `<ServerName>.ini` ค้นหาการตั้งค่า `Open` แล้วตั้งค่าเป็น false

```ini
Open=false
```

บันทึกไฟล์แล้วรีสตาร์ทเซิร์ฟเวอร์ เมื่อปิดการลงทะเบียนแบบเปิดแล้ว จะมีแค่บัญชีที่คุณสร้างเท่านั้นที่สามารถเข้าร่วมได้

## การจัดการผู้เล่นในรายชื่อขาว

เปิด Live Console ในการจัดการเซิร์ฟเวอร์เกมของ ZAP-Hosting แล้วสร้างบัญชีสำหรับผู้เล่นที่อนุญาต ใช้คำสั่ง adduser พร้อมชื่อผู้ใช้และรหัสผ่าน

```text
/adduser "username" "password"
```

ถ้าต้องการลบสิทธิ์การเข้าถึงของผู้ใช้ ให้ลบออกจากรายชื่อขาวด้วยคำสั่งเฉพาะ

```text
/removeuserfromwhitelist "username"
```

ถ้าคุณเคยเปิดการลงทะเบียนแบบเปิดและต้องการแปลงบัญชีที่เชื่อมต่ออยู่ให้เป็นผู้เล่นในรายชื่อขาว ใช้คำสั่งที่เพิ่มผู้ใช้ที่เชื่อมต่อทั้งหมดในตอนนี้ลงในรายชื่อขาว

```text
/addalltowhitelist
```

## การตรวจสอบการทำงานของรายชื่อขาว

หลังจากตั้งค่า `Open=false` แล้ว ผู้ใช้ใหม่ที่ไม่ได้ถูกเพิ่มด้วย `/adduser` จะไม่สามารถเข้าร่วมได้ ทดสอบด้วยบัญชีที่อยู่ในรายชื่อขาวเพื่อยืนยันการเข้าถึง

ถ้าผู้ใช้ที่ไม่ได้อยู่ในรายชื่อขาวยังเข้าร่วมได้ ให้ตรวจสอบว่าคุณแก้ไขไฟล์ `<ServerName>.ini` ที่ใช้งานจริงและรีสตาร์ทเซิร์ฟเวอร์แล้ว

## สรุป

ถ้าทำตามขั้นตอนทั้งหมดข้างต้นถูกต้อง รายชื่อขาวของคุณจะเปิดใช้งานแล้ว และคุณสามารถควบคุมได้อย่างแม่นยำว่าใครสามารถเข้าร่วมเซิร์ฟเวอร์ได้ หากการเข้าถึงยังไม่เป็นไปตามที่คาดไว้ ให้รีสตาร์ทเซิร์ฟเวอร์อีกครั้งและตรวจสอบไฟล์หรือผลลัพธ์คำสั่งเพื่อยืนยันว่าการเปลี่ยนแปลงถูกนำไปใช้แล้ว

หากมีคำถามเพิ่มเติมหรือต้องการความช่วยเหลือ อย่าลังเลที่จะติดต่อทีมซัพพอร์ตของเรา ซึ่งพร้อมให้บริการทุกวันเพื่อช่วยคุณ! 🙂

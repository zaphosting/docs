## Introducción

El Cashbox es el área central para gestionar el saldo de tu cuenta. Te permite depositar dinero, administrar métodos de pago guardados, configurar recargas automáticas, canjear vales y revisar todas las entradas relacionadas con el saldo en un solo lugar.



## Opciones disponibles para depositar dinero

El Cashbox ofrece varias opciones para depositar dinero en tu cuenta. Todos los métodos de pago disponibles se muestran directamente en esta sección. Para iniciar un depósito, haz clic en **Depositar dinero**.

![img](https://screensaver01.zap-hosting.com/index.php/s/sqwjs4DkT4BBQxP/preview)





## Guardar una tarjeta de crédito

Puedes guardar una tarjeta de crédito para pagos futuros o para usar con recargas automáticas. Esto permite depósitos más rápidos sin tener que ingresar tus datos de pago cada vez. Para añadir una tarjeta de crédito, haz clic en **Agregar tarjeta de crédito**. Se abrirá una ventana donde podrás ingresar los datos de tu tarjeta. Completa todos los campos requeridos y confirma el proceso para guardar la tarjeta.

![](https://screensaver01.zap-hosting.com/index.php/s/mLQGYd69pKaS63y/preview)



## Activar pagos automáticos

Los pagos automáticos permiten que el saldo de tu cuenta se recargue automáticamente cuando cae por debajo de un umbral definido. Para activar esta función, haz clic en **Recargar automáticamente**. Luego podrás definir el saldo mínimo que activará la recarga y la cantidad que se abonará a tu cuenta.

![](https://screensaver01.zap-hosting.com/index.php/s/qFRi5rJypmdMM58/preview)





## Canjear vales y ZAP Coins

Los vales pueden canjearse directamente en el Cashbox y se acreditan en tu cuenta como ZAP Coins. Para canjear un vale, haz clic en **Usar vale** e ingresa el código del vale en el campo proporcionado.

![img](https://screensaver01.zap-hosting.com/index.php/s/TrRACJwJB8jkW8Z/preview)



## Ver entradas de la cuenta

Todas las transacciones de la cuenta, como depósitos, cargos y canjes de vales, se listan al final de la página del Cashbox.

Puedes filtrar y ordenar las entradas por mes para obtener una visión clara del historial de tu cuenta.

![img](https://screensaver01.zap-hosting.com/index.php/s/2gjfYHtTZBAjBb4/preview)

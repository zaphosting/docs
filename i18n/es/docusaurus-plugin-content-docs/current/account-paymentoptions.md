## Resumen
ZAP-Hosting ofrece una gran variedad de opciones de pago con las que puedes alquilar y activar servidores al instante. El proceso de pago es simple y rápido. Las siguientes opciones de pago están disponibles con nosotros:
*  **ZAP Coins:** Los ZAP Coins se añaden a través de la Caja (ingresar dinero) y pueden usarse para casi todos los productos.
* **Tarjeta de crédito:** Pago con tarjeta de crédito.
* **PayPal:** Pago con PayPal.
* **PaySafeCard:** Las PaySafeCards son tarjetas prepago que se pueden comprar en más de 25,000 puntos de venta en Alemania. Para pagar se necesita el código de la tarjeta.
* **Bitcoin:** Paga con Bitcoin. La transacción será procesada por el proveedor Coinify.
* **Transferencia bancaria:** Solo disponible para ingresar ZAP Coins. Tras recibir el pago, los ZAP Coins se activarán.

## Proceso de pago
El proceso de pago es sencillo: se configura el servidor en la página de configuración, y al final de la página se selecciona la opción de pago deseada. Tras hacer clic en "Comprar ahora", si es necesario, se redirige al proveedor de pago seleccionado para completar la transacción. Una vez recibido el pago, el servidor se configura en cuestión de segundos.

## ZAP Coins: Ingresar saldo
Puedes ingresar saldo en ZAP Coins usando tu caja a través de cualquiera de las opciones de pago disponibles. Los servidores con contrato también pueden pagarse con ZAP Coins. En la interfaz web puedes ingresar ZAP Coins en el apartado **Ingresar saldo** dentro de tu Caja.

:::info
No está permitido el pago parcial con ZAP Coins. Si ya tienes ZAP Coins, por ejemplo porque tienes un código de ZAP Coins, puedes añadir crédito libremente. Consejo: Si pagas con ZAP Coins, fíjate en nuestras Ofertas de Ingreso de Saldo: Si la oferta está activa, recibirás crédito adicional al aumentar tu depósito.

Las transferencias bancarias pueden usarse para ingresar ZAP Coins. En cuanto recibamos la transferencia, el crédito se activará y estará disponible.
:::

## Caja
La Caja se puede acceder desde la interfaz web en el apartado **Caja**. Además de la opción de ingresar dinero, la Caja ofrece las siguientes funciones adicionales:
* **Añadir tarjeta de crédito:** Se pueden registrar tarjetas de crédito. Al registrar una tarjeta, no será necesario ingresar todos los datos en cada pago con tarjeta.
* **Ingresar dinero:** La función de ingreso también está disponible desde la Caja.
* **Canjear cupón:** Aquí puedes canjear cupones que otorgan ZAP Coins. Tras canjear el código, los ZAP Coins se acreditan inmediatamente en tu cuenta.
* **Recarga automática de ZAP Coins:** Puedes configurar la recarga automática de ZAP Coins. Esto es útil si tienes un servidor con contrato. Para recargar crédito automáticamente, debes usar PayPal.
* **Resumen de transacciones:** Un resumen de todas las transacciones realizadas hasta ahora.

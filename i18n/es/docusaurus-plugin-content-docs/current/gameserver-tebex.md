## Conectando Tebex

Configurar Tebex con ZAP es muy sencillo. La instalación y configuración solo requieren unos pocos pasos, que se explican con más detalle a continuación. 

### Selecciona un juego compatible

Elige un juego que sea compatible con Tebex y añádelo a tu servidor de juegos. Haz clic en tu servidor de juegos y selecciona `Juegos` en la barra lateral. Escoge un juego compatible en `Juegos disponibles` y pulsa el botón de descarga para instalarlo.

### Solicita la invitación de Tebex

Si el juego que elegiste soporta Tebex, verás una pestaña llamada "Tebex Shop" en la barra lateral. Haz clic en la imagen que aparece al cargar la página.

![](https://screensaver01.zap-hosting.com/index.php/s/SztAWeeNX8fx5kF/preview)

Tebex enviará una invitación al correo electrónico vinculado a tu cuenta de cliente ZAP-Hosting. Sigue las instrucciones del email para activar tu cuenta de Tebex. Te guiarán por la configuración básica de tu tienda.

### Instala el plugin de Tebex

Este proceso suele ser automático. Si no tenemos un plugin para una configuración automática en nuestros servidores o si falta la confirmación (pingback) de Tebex, tendrás que hacer la configuración manual.

### Configura el plugin de Tebex

Instala manualmente el plugin adecuado para tu servidor de juegos. El proceso de configuración varía según el juego. Puedes encontrar instrucciones básicas sobre cómo instalar los plugins [aquí](https://docs.tebex.io/store/integrating-with-your-game-server-or-website/minecraft-java-edition).

## Verifica la conexión con Tebex

Finalmente, tendrás que confirmar que instalaste el plugin del servidor de juegos correctamente.

![](https://screensaver01.zap-hosting.com/index.php/s/DkDtPMtij2QTKGS/preview)

En algunos casos, puede que necesites introducir la clave API del plugin de tu tienda Tebex para establecer la conexión entre ZAP y tu tienda Tebex. Entonces podrás ver que la conexión fue exitosa:
![](https://screensaver01.zap-hosting.com/index.php/s/4YS2SwHfSTgn4DE/preview)

## Administración del Producto

En la sección de **Administración del Producto** encontrarás todas las páginas necesarias para gestionar tu Voicebot. Las opciones para administrar tu Voicebot se detallan a continuación.

### Panel

El **Panel** del VoiceBot sirve como la vista central de gestión. Proporciona toda la información básica y esencial sobre el VoiceBot en un formato claro, incluyendo el estado en línea, detalles de acceso, enlace a la interfaz web externa y uso del disco.

![img](https://screensaver01.zap-hosting.com/index.php/s/YnXaQDPywcoxFPP/preview)

### Interfaz Web

La mayoría de las funciones del VoiceBot están disponibles a través de una interfaz web externa. Esta interfaz permite seleccionar la fuente de música deseada y enviar sugerencias para mejorar el bot. También es posible cambiar la contraseña para acceder a la interfaz web y reinstalar el VoiceBot.

![img](https://screensaver01.zap-hosting.com/index.php/s/cmKrJ2YpzmZcm2L/preview)

### Servidor

Antes de usar el bot, es necesario definir en qué servidor de TeamSpeak debe ejecutarse. Se debe especificar la dirección IP y el puerto del servidor, junto con un nombre libremente seleccionable para el bot. Si el servidor está protegido con contraseña, debe proporcionarse para que el bot pueda acceder al servidor.

![img](https://screensaver01.zap-hosting.com/index.php/s/sbbRr9f5HexcMsG/preview)

### Configuraciones Avanzadas

Funciones avanzadas como mover usuarios automáticamente o reproducir un mensaje de bienvenida cuando un usuario se une al servidor pueden activarse configurando los ajustes correspondientes. Estas funciones permiten una gestión del servidor más personalizada y automatizada.

![img](https://screensaver01.zap-hosting.com/index.php/s/yz94fyZssG53Rjd/preview)

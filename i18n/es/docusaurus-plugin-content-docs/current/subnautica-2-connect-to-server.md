## Introducción
¿No sabes cómo conectarte a tu servidor de **Subnautica 2** o qué información necesitas para empezar? Esta guía te muestra cómo obtener los datos necesarios del servidor, establecer la conexión y solucionar problemas comunes de conexión.

:::info Aviso de Acceso Anticipado
Subnautica 2 puede estar aún en desarrollo activo según la etapa actual de lanzamiento. Por ello, las funciones multijugador, el comportamiento del navegador de servidores y los flujos de conexión pueden cambiar con el tiempo.
:::

## Obtén los datos del servidor

Necesitarás toda la información relevante para conectarte a tu servidor. Puedes encontrar los datos en el panel de control web de tu servidor de juegos. La información importante incluye:

- Dirección IP y puerto
- Nombre del servidor
- Contraseña del servidor

Con esta información podrás identificar tu servidor y asegurarte de conectarte al correcto.

| Dato | Descripción |
| --- | --- |
| `Dirección IP` | La dirección pública de tu servidor de Subnautica 2 |
| `Puerto` | El puerto del juego usado para la conexión |
| `Nombre del servidor` | El nombre que aparece en el juego si hay lista de servidores o navegador de sesiones |
| `Contraseña del servidor` | Solo necesaria si tu servidor está protegido con contraseña |

## Establecer conexión con el servidor

La forma exacta de conectarte a un servidor de **Subnautica 2** depende de las opciones multijugador disponibles en la versión del juego que uses. En general, necesitarás la **dirección IP** y el **puerto** del servidor, y en algunos casos también la **contraseña**.

### Unirse usando el menú multijugador dentro del juego

Inicia **Subnautica 2** y abre la sección multijugador o de unión a servidor desde el menú principal. Si el juego ofrece una opción de conexión directa, introduce la `dirección IP` y el `puerto` de tu servidor tal como aparecen en el panel web de ZAP-Hosting. Si hay contraseña configurada, introdúcela cuando se te pida.

Tras confirmar los datos, el juego intentará establecer la conexión con tu servidor.

:::note Disponibilidad de Conexión Directa
La redacción exacta de menús y botones puede variar según la versión actual del juego. Si no ves opción de conexión directa, verifica si el juego requiere unirse mediante lista de servidores, sistema de invitaciones o navegador de sesiones.
:::

### Unirse usando una lista de servidores o navegador de sesiones

Si **Subnautica 2** usa un navegador de servidores integrado en lugar de un campo de conexión directa, busca tu servidor usando el `nombre del servidor` configurado. Por eso es importante verificar el nombre en tu panel web antes de intentar conectarte.

Una vez encuentres el servidor correcto en la lista:

1. Selecciona la entrada del servidor
2. Verifica que el nombre mostrado coincida con tu servidor
3. Únete al servidor
4. Introduce la contraseña si se solicita

:::tip Usa Nombres de Servidor Únicos
Si tu servidor no aparece fácil de identificar en la lista, asigna un nombre claro y único en la configuración del servidor. Esto facilita encontrar la entrada correcta y evita conectarte al servidor equivocado.
:::

### Si el servidor no es visible inmediatamente

Si tu servidor no aparece en el navegador dentro del juego, no siempre significa que esté offline. A veces las listas tardan en actualizarse o el juego solo muestra servidores tras una inicialización exitosa.

Debes verificar lo siguiente:

- Que el servidor esté online en el panel web de ZAP-Hosting
- Que se esté usando el puerto de juego correcto
- Que el servidor haya arrancado completamente
- Que el nombre del servidor esté configurado correctamente
- Que se introduzca la contraseña si es necesaria

## Problemas comunes y soluciones

La conexión a un servidor de juegos puede verse afectada por varios factores. A continuación, los problemas y soluciones más comunes.

### No visible

La falta de visibilidad del servidor puede ocurrir si la inicialización no se completó correctamente. Esto puede deberse a una configuración errónea o archivos corruptos. Normalmente, más información está disponible en la consola del servidor o en los archivos de registro.

:::caution Es Necesario que el Servidor Arranque
No intentes conectarte antes de que el servidor haya completado su proceso de arranque. Si el servidor aún está cargando o reiniciándose, los jugadores pueden no poder conectarse o el servidor puede no aparecer en ninguna lista.
:::

### Datos de conexión incorrectos

Si no puedes establecer conexión, verifica que estés usando la `dirección IP`, `puerto` y `contraseña` correctos. Incluso un pequeño error tipográfico puede impedir la conexión.

| Verifica | Qué comprobar |
| --- | --- |
| `Dirección IP` | Que coincida con la dirección mostrada en tu panel de servidor |
| `Puerto` | Que coincida con el puerto asignado para el juego |
| `Contraseña` | Que esté escrita exactamente como fue configurada, respetando mayúsculas y minúsculas |
| `Nombre del servidor` | Que coincida con el nombre configurado si buscas en un navegador |

### El servidor está online pero la conexión falla

Si el servidor parece estar online pero no puedes conectarte, revisa la consola y los registros del servidor en tu panel web. Los errores allí pueden indicar si el servidor no cargó correctamente o si hay un problema de configuración que bloquea las conexiones.

:::tip Revisa los Registros Primero
La consola y los archivos de registro suelen ser la forma más rápida de identificar por qué falla una conexión. Te ayudan a confirmar si el servidor está escuchando correctamente y si los intentos de conexión llegan al servidor.
:::

### No encuentras solución o no es útil

¿Sigues con problemas tras intentar estas soluciones? Nuestro equipo de soporte está disponible todos los días para ayudarte. Solo crea un **[Ticket](https://zap-hosting.com/en/customer/support/)** en nuestra web y explica tu problema con el mayor detalle posible. Incluye la IP, puerto y una breve descripción de lo que ocurre al intentar conectarte.

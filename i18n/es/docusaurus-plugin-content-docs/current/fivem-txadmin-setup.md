## Introducción

txAdmin es un panel web completamente gratuito y con todas las funciones para gestionar y monitorear tu servidor de juegos FiveM. Ofrece un montón de características diseñadas para que administrar un servidor FiveM sea pan comido. Configurar txAdmin es totalmente gratis, así que empieza ya a usar el panel web para servidores FiveM más sencillo y funcional.

## Acceso

Puedes encontrar un enlace y los datos de acceso para entrar a txAdmin en el área de txAdmin dentro del panel de control de tu servidor de juegos. Ahí tienes toda la info que necesitas para acceder fácilmente a la interfaz de txAdmin y empezar a configurarlo.

![img](https://screensaver01.zap-hosting.com/index.php/s/69LdTK3FyNZNXid/download)

Copia el nombre de usuario y la contraseña, y haz clic en el enlace (dirección IP:puerto) para acceder a la interfaz de txAdmin e iniciar sesión. La interfaz de txAdmin se abrirá en una nueva ventana, donde tendrás que ingresar con los datos de acceso proporcionados.

![](https://screensaver01.zap-hosting.com/index.php/s/pp8GLQBoX4LoqTA/preview)



## Configuración

Ahora que has iniciado sesión con éxito, puedes comenzar a configurar txAdmin y tu servidor. Deberías ver la página principal de txAdmin (Panel). En la esquina superior izquierda, verás un mensaje que indica que tu servidor aún necesita ser configurado. Haz clic en **Ir a la página de configuración** para comenzar el proceso.

![](https://screensaver01.zap-hosting.com/index.php/s/oXakf3qoJaim7ex/download)



### Bienvenida y nombre del servidor

A continuación, define un nombre para tu servidor que quieras usar en tu proyecto. Este nombre no es para la lista de servidores, sino solo para usar dentro de la interfaz de txAdmin y para mensajes en chat/Discord.

![](https://screensaver01.zap-hosting.com/index.php/s/FCmd5xQ89wSPHfe/preview)



### Tipo de despliegue

En Tipo de despliegue, ahora tienes que elegir cómo quieres configurar tu servidor. Tienes las siguientes opciones: **Recetas populares**, Datos de servidor existentes, **Plantilla URL remota** y **Plantilla personalizada**. Cada opción se explica con más detalle abajo para que elijas la que mejor se adapte a tus necesidades.

![](https://screensaver01.zap-hosting.com/index.php/s/52HfyJSNLscApNE/preview)



### Plantilla popular

Las Plantillas populares son paquetes predefinidos que puedes usar al instante, siendo esta la forma más fácil de poner tu servidor FiveM en marcha. Si es tu primer servidor, esta opción es súper recomendada. Estas plantillas incluyen todas las configuraciones y recursos necesarios para montar un servidor funcional rápido, permitiéndote enfocarte en la personalización y el gameplay en lugar de la configuración inicial. Solo elige la plantilla que mejor te sirva.

![](https://screensaver01.zap-hosting.com/index.php/s/PSsf22NeebNBRw7/preview)



### Datos de servidor existentes

Usa esta opción si ya has operado un servidor con txAdmin y ahora quieres alojarlo con nosotros. En ese caso, simplemente sube los datos del servidor vía FTP y especifica la ruta durante la configuración. Si aún no sabes cómo usar FTP en general, te recomendamos echar un vistazo a nuestra [guía de acceso FTP](gameserver-ftpaccess.md). Esta guía te ayudará a transferir tus archivos sin problemas para que tu servidor esté listo rápido en nuestra plataforma.

![img](https://screensaver01.zap-hosting.com/index.php/s/KS4raRtHWmmw5iN/preview)





### Plantilla URL remota

Si quieres aplicar una plantilla que no está enlazada dentro de txAdmin, esta función te permite especificar una receta personalizada mediante una URL remota. Esta receta se descargará y ejecutará en tu servidor. Esta opción es ideal para desplegar configuraciones personalizadas o setups específicos que no están incluidos en las opciones predeterminadas de txAdmin. Al proporcionar la URL, txAdmin descargará y aplicará la plantilla automáticamente, asegurando que tu servidor se configure según tus requisitos exactos.

![](https://screensaver01.zap-hosting.com/index.php/s/jrGzTGp9FwLc82i/preview)

### Plantilla personalizada

Esta opción es para usuarios que ya crearon su propia receta y quieren recargarla. Te permite aplicar tus configuraciones personalizadas rápida y fácilmente sin tener que configurar todo desde cero otra vez. Solo selecciona tu receta existente y txAdmin se encarga del resto, asegurando que tu servidor quede configurado tal como quieres.

![](https://screensaver01.zap-hosting.com/index.php/s/Z75q5RKakwfpHGy/preview)



### Desplegador de recetas

Después de elegir un Tipo de despliegue y seguir los pasos indicados, puedes terminar haciendo clic en **Ir al desplegador de recetas**. Ahí tendrás que completar los pasos finales: **Revisar receta**, **Ingresar parámetros**, **Ejecutar receta** y **Configurar server.cfg**.

En el primer paso, puedes personalizar aún más la receta seleccionada o añadida. Si no necesitas hacer ajustes, pasa al Paso 2. Aquí puedes ingresar tu propia [Clave de licencia](fivem-licensekey.md) y revisar la información de la base de datos. Cuando todo esté listo, haz clic en **Ejecutar receta**. Este proceso puede tardar un momento. Finalmente, puedes ajustar el archivo de configuración del servidor (`server.cfg`) según necesites.

![](https://screensaver01.zap-hosting.com/index.php/s/wFMD576sBQAAdxZ/download)

Para terminar, haz clic en **Guardar y ejecutar servidor**. Con esto finaliza la configuración de txAdmin y ya puedes conectarte a tu servidor.


## Soporte adicional

Para soporte específico de txAdmin, contacta con el equipo de txAdmin a través de su [sitio oficial](https://txadm.in/) o únete a la comunidad en [Discord](https://discord.gg/txAdmin/). La comunidad y el equipo de soporte están listos para ayudarte con cualquier duda o problema que tengas usando txAdmin, asegurando que tengas la mejor experiencia gestionando tu servidor.

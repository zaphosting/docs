## Crear cuenta
Crear una cuenta en ZAP-Hosting es el primer paso para acceder y gestionar todos los servicios y funciones. El proceso de registro es rápido y solo requiere unos pocos datos.

Para empezar el registro, visita la web de ZAP-Hosting y navega a la página de inicio. Allí encontrarás el área de login con un botón **¡Regístrate!**. Haz clic en **¡Regístrate!** para abrir la ventana emergente de registro.

![img](https://screensaver01.zap-hosting.com/index.php/s/5b6Wzc27CcHW3Zx/download)

Dentro de la ventana emergente, debes definir la siguiente información:

- Nombre de usuario
- Dirección de email
- Contraseña

Después de ingresar tus datos, confirma el captcha de seguridad. Este paso es obligatorio para protegernos contra registros automáticos. Una vez completados todos los campos y confirmado el captcha, envía el formulario para crear tu cuenta.

![img](https://screensaver01.zap-hosting.com/index.php/s/k6P3wSKWWRRiYax/preview)

## Activación de la cuenta

Después de enviar el formulario de registro, se envía un email de activación a la dirección proporcionada. Este email contiene un código de activación que debes confirmar para completar la configuración de tu cuenta.

La entrega del email de activación puede tardar unos minutos. Si el mensaje no aparece en tu bandeja de entrada, revisa también la carpeta de spam o correo no deseado.

:::warning Funcionalidad limitada sin activación
Puedes iniciar sesión inmediatamente después del registro. Sin embargo, para acceder a todas las funciones y servicios, debes confirmar el email de activación.
:::

## 产品管理

在 **产品管理** 部分，您会找到管理服务器所需的所有页面。以下详细列出了管理服务器的各项选项。

### 控制面板

服务器的 **控制面板** 是管理的核心。这里清晰展示了服务的所有基础和重要信息，包括服务器位置、状态、IP:端口、已用内存、当前游戏、服务器名称、地图以及可用槽位数。此外，还显示了 CPU、内存和 RAM 的使用情况。

![img](https://screensaver01.zap-hosting.com/index.php/s/tSHRbW66eDXBxJn/preview)

在控制面板中，您还可以启动、停止或重启服务，全面掌控服务器运行状态。此外，您可以访问实时控制台，获取服务器状态的实时信息，并在必要时直接采取操作。

### 统计数据

在 **统计数据** 中，您可以全面查看服务器的负载情况。这里包含了可用槽位、CPU 和内存等多个方面的负载信息。您可以选择不同时间段查看数据，例如最近的 **60 分钟、24 小时、3 天、7 天** 或 **30 天**。

![](https://screensaver01.zap-hosting.com/index.php/s/N8jcmdqqG2Xt4Bk/preview)

这些信息让您深入了解服务器的使用情况。此外，这些统计数据常用于识别潜在的性能问题并缩小排查范围。例如，您可以追踪负载上升的时间点，判断是否与某些事件或变更有关。这在突发问题时尤为重要，能帮助您有针对性地排查并系统分析可能的原因。

### DDoS 管理器

查看过去的 DDoS 攻击记录，更好地评估潜在威胁和攻击模式。开启服务器正在进行的 DDoS 攻击通知，确保事件发生时您能迅速响应。

根据服务器所在位置及可用的 DDoS 保护，您还可以实时监控活跃攻击，包括详细的流量分析，显示合法流量和被防护系统拦截的流量量。这种透明度帮助您了解防护效果，及早识别潜在风险。

:::info 缺少 DDoS 管理器访问权限
使用 DDoS 管理器需要专用 IP 地址或额外选项 `DDoS Manager Overview`。如果当前套餐未包含这些组件，您可以随时通过升级服务添加。
:::

![](https://screensaver01.zap-hosting.com/index.php/s/ScCCCY52CMLgfyE/preview)

:::caution DDoS 管理器使用要求
仅当您拥有专用 IP 地址或已购买 `DDoS Manager Overview` 额外选项时，才能使用 DDoS 管理器。如果当前套餐缺少这些组件，您可以随时通过升级添加。
:::

### 日志文件

遇到技术问题但不确定原因？首选解决方案是查看 **日志文件**。这些文件包含大量服务器信息，如当前事件、提示或错误消息。

尤其是错误消息，有助于识别和深入理解问题根源。通过有针对性地查看日志文件，常能发现并解决问题。

:::caution 需要日志文件帮助？
如果日志信息无法帮助您或难以理解，您可以提交 **[工单](https://zap-hosting.com/en/customer/support/)** 说明问题。我们的支持团队每日在线，能协助您进一步分析问题。
:::

### 事件日志

**事件日志** 提供了服务器管理相关的所有活动的全面概览。您可以随时追踪服务器何时启动、停止或崩溃，以及何时执行了更新、备份或新安装。

![](https://screensaver01.zap-hosting.com/index.php/s/xNzsaMbW5BS5KJC/preview)

### 实时控制台

**实时控制台** 实时显示服务器上的所有当前事件。您可以看到玩家活动信息、错误消息和系统消息。这一视图让您全面了解服务器动态，及早发现潜在问题。

![](https://screensaver01.zap-hosting.com/index.php/s/J9SsmSrPXnG7P9w/preview)

除了监控事件，您还可以在 **实时控制台** 直接向服务器发送命令，用于调整设置、执行操作或快速响应突发问题。

### 服务中断

服务器曾短暂离线但原因不明？查看事件日志和日志文件也无线索？问题可能不在您的服务器本身，而是托管服务器所在的主机系统。

当监控系统检测到整体中断时，技术人员会自动收到通知并尽快处理问题。

本节提供了游戏服务器所在主机系统的中断历史概览。更多信息及最新服务状态请访问 [状态页面](https://status.zap-hosting.com/)。

## 设置

在 **设置** 部分，您会找到配置服务器所需的所有页面。以下详细列出了可用的配置选项。

### 设置

在 **设置** 中，您可以看到服务器最常用的配置选项及简短说明。此页面特别适合服务器管理新手使用。

如果您更熟悉操作，想进行更精细的调整，建议使用配置编辑器。相关信息请参见 **Configs** 分类。

### 配置文件

配置文件编辑器允许您直接在网站上以传统方式编辑配置文件，而非简化视图。此功能让您根据个人喜好自定义服务器设置。

通过编辑器，您可以直接访问所有可用选项，精准调整配置。此功能特别适合高级用户，能完全掌控服务器设置并明确所需更改。

:::caution 配置值可能被覆盖
如果发现部分更改被覆盖，请注意某些选项值来自设置页面。若想直接在配置文件中修改这些值，需先在设置页面禁用相关选项。
:::

### 游戏

在 **游戏** 中，您可以随时更改现有游戏的游戏模式，或安装其他游戏。价格相同的游戏可直接在 **可用游戏** 中选择。其他游戏则需先调整槽位价格。详情请参见 [游戏切换](gameserver-gameswitch.md) 指南。

![](https://screensaver01.zap-hosting.com/index.php/s/xkkECw7o52fAMWk/preview)

### 版本

在 **版本** 中，您可以查看和管理当前游戏服务器版本。随时更新到最新版本，或启用自动更新，确保服务器始终保持最新状态。

![](https://screensaver01.zap-hosting.com/index.php/s/BH2JzyRHTeLdKHz/preview)

此外，您还可以开启自动更新的邮件通知功能，及时了解更新情况，确保信息同步。

### 计划任务

创建计划任务，在指定时间自动执行，帮助您自动化服务器上的各种流程。您可以设置任务执行一次或循环执行。支持的任务类型包括启动、停止或重启服务，在线时重启服务器，创建备份，或执行自定义命令。

![](https://screensaver01.zap-hosting.com/index.php/s/P6DeWiRC3tDqG2z/preview)

### 重启计划

想要在特定时间安排服务器重启？重启计划功能允许您设置定时自动重启服务器。您可以指定具体时间或定期间隔执行重启。

![](https://screensaver01.zap-hosting.com/index.php/s/Y6WciDS7YP98P4m/preview)

### DNS 设置

通过 ZAP DNS 设置，您可以为服务创建易记的地址。您可以使用可用域名的子域，或在自己的域名上创建 DNS 记录，从而获得更贴合项目的地址。

![](https://screensaver01.zap-hosting.com/index.php/s/tM5FGTHRSbZA5rN/preview)

## 工具

在导航面板的 **工具** 部分，您会找到多个子项，帮助您管理服务器，包括 **FTP 浏览器**、**数据库** 和 **备份**。以下为这些功能的详细介绍。

### FTP 浏览器

使用 **FTP 浏览器**，您可以获取连接 FTP 访问所需的所有信息。FTP 访问允许您直接操作游戏服务器文件。建议使用外部 FTP 软件，如 [FileZilla](https://filezilla-project.org/) 或 [WinSCP](https://winscp.net/eng/index.php) 以获得最佳体验。如果您不熟悉 FTP 使用方法，可以参考 [通过 FTP 访问](gameserver-ftpaccess.md) 指南。

![](https://screensaver01.zap-hosting.com/index.php/s/sX85i9HyzR9wmAM/preview)

**FileZilla：** FileZilla 是最简单且知名的 FTP 客户端之一。安装后，启动程序，在顶部栏输入面板显示的访问凭据，点击连接即可直接登录服务器，管理文件。

**WinSCP：** WinSCP 主要支持 SCP 和 SFTP，但也支持标准 FTP 连接。功能比 FileZilla 更丰富，适合高级用户使用。

### 数据库

除了游戏服务器产品外，还包含 10 个数据库。这些数据库用于存储数据等场景。在 **数据库** 中，您可以创建数据库并查看访问凭据，还能启用备份选项及导入已有数据库备份。

![](https://screensaver01.zap-hosting.com/index.php/s/WFLwCbZSQfYjrmD/preview)

### 备份

在 **备份** 中，您可以为服务器及其数据库创建手动或自动备份，方便在出现问题或数据丢失时恢复数据。

您可以设置是否自动备份，以及备份频率（日常或每周）。还可选择在存储空间不足时删除最旧备份。

![](https://screensaver01.zap-hosting.com/index.php/s/nm9za5LTMEcNsga/preview)

默认情况下，游戏服务器产品提供 10 GB 备份存储空间。若需更多空间，可付费升级，确保备份空间充足，且能访问较旧备份。

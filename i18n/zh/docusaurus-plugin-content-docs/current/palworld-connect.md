## 获取服务器 IP

首先，你需要知道你的 Palworld 服务器的 IP 和端口，才能进行直连。只需登录你的 [ZAP-Hosting 控制面板](https://zap-hosting.com/en/customer/)，就能找到完整的 IP 和端口信息。

![](https://screensaver01.zap-hosting.com/index.php/s/DRxKHE7bP3FbFMo/preview)

如果你是用外部独立服务器运行 Palworld，IP 就是主机的 IP，端口则是你在配置文件中设置的端口（默认是 8211）。更多端口相关信息，请参考我们的[服务器配置](palworld-configuration.md)指南。

## 直连服务器

先从你的游戏启动器启动 Palworld。在主菜单中选择 **加入多人游戏**。

![](https://screensaver01.zap-hosting.com/index.php/s/TZzLAS29rnpPmi6/preview)

进入专用服务器浏览器后，滚动到页面底部。在底部的搜索框中输入你的 IP 和端口。准备好后，点击 **连接** 按钮，即可加入你的服务器。

:::note
一定要用底部的搜索框，顶部搜索框是用来按服务器名称搜索服务器列表的。
:::

![](https://screensaver01.zap-hosting.com/index.php/s/CG5gARbSxC3RpTb/preview)

如果连接失败并出现超时错误，请确认输入的 IP 和端口是否正确，且服务器处于在线状态。你可以通过 Palworld 服务器的控制面板中的控制台部分来辅助排查问题。

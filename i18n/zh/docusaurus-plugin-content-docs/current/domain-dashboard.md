## 产品管理

在**产品管理**部分，你会找到管理域名所需的所有页面。下面详细列出了管理域名的各项功能。

### 仪表盘

你的域名**仪表盘**是管理的核心。这里清晰展示了所有关于域名的基础且重要的信息。例如，域名详情、域名服务器的注册/到期日期以及 AUTH 码。此外，你还可以通过仪表盘使用我们简单易用的 DNS 模式（[EasyDNS](domain-easydns.md)、[ProfiDNS](domain-profidns.md)、[ExpertDNS](domain-expertdns)）。

![img](https://screensaver01.zap-hosting.com/index.php/s/xQww62Noja46TED/preview)

### 事件日志

**事件日志**为你提供了域名管理相关的所有操作的全面记录。你可以随时追踪 DNS 记录或域名服务器的变更时间。

![img](https://screensaver01.zap-hosting.com/index.php/s/2Tfirza5nkm6jzr/preview)

## 设置

在**设置**部分，你会找到配置域名所需的所有页面。下面详细列出了可用的配置选项。

### DNS 管理

在 DNS 管理中管理你的 DNS 记录。这里你可以创建所需的[域名记录](domain-records.md)，并根据需求进行配置。

![img](https://screensaver01.zap-hosting.com/index.php/s/eSFEN9sLkKfcHPb/preview)

### 联系人管理

你使用域名联系人（Handle）来定义注册域名时存储的联系信息。这里包含域名持有者的姓名、地址、邮箱和电话号码等信息。

这些信息用于明确将域名归属到某个人或组织。你可以在这里选择已有的联系人，也可以创建新的联系人，以使用不同的联系信息。

![img](https://screensaver01.zap-hosting.com/index.php/s/DWKmQqPQ3B2DwD9/preview)

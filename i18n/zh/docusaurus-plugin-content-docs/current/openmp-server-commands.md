## 介绍

Open.mp 服务器自带一套实用的内置服务器命令，既可以在游戏内使用，也可以通过服务器控制台执行。在本指南中，我们将详细介绍这些命令，提供描述和使用示例。

<InlineVoucher />

## 设置访问权限（RCON）

要在游戏内执行服务器命令，你需要启用 RCON 功能。请打开你的配置文件 `config.json`，调整相关参数以开启该功能并设置密码。请参考我们的[服务器配置](openmp-configuration.md)指南，了解如何访问你的服务器配置文件。

准备好文件后，找到以下参数，开启功能并设置 RCON 密码。

| 参数名称                      | 示例                                   | 说明                                                                                         |
| ----------------------------- | ------------------------------------ | -------------------------------------------------------------------------------------------- | 
| rcon.enable                   | true                                 | 切换是否启用 RCON 功能                                                                       |
| rcon.password                 | ZAP-IS-AWESOME                       | 设置访问 RCON 所需的密码                                                                     |

:::info
务必通过配置文件设置 RCON 密码，否则可能导致任何人都能访问 RCON，极其不安全。
:::

完成修改后，保存文件并重启服务器。下次服务器启动时，RCON 功能将可用且已准备就绪。

:::tip
想了解更多关于 RCON 的内容？欢迎查看我们的[RCON](openmp-rcon.md)指南，内容更详细。
:::

## 可用命令

下表列出了 Open.mp 服务器的一些重要命令，包含使用示例和说明，这些命令可在游戏内、服务器控制台或通过 RCON 程序执行。如果你想查看所有当前命令，推荐阅读[open.mp 的服务器控制指南](https://www.open.mp/docs/server/ControllingServer)，里面涵盖了所有可用的服务器命令。

:::tip
通过控制台或 RCON 程序直接执行命令时，无需使用 `/rcon` 前缀。只有在游戏内执行命令时才需要加上。
:::

:::note
确保已启用 RCON，否则这些命令在游戏内或通过 RCON 程序都无法使用。
:::

### 通用命令

| 命令语法                     | 接受参数        | 说明                                                                 | 
| ----------------------------- | --------------- | -------------------------------------------------------------------- | 
| /rcon cmdlist                 | -               | 返回所有服务器命令列表                                               | 
| /rcon varlist                 | -               | 返回当前所有服务器变量列表                                           | 
| /rcon exit                    | -               | 关闭服务器                                                         | 
| /rcon echo [文本]             | 字符串          | 向服务器控制台发送消息（游戏内不显示）                             | 
| /rcon say [文本]              | 字符串          | 向所有玩家发送游戏内消息（显示为“Admin: [文本]”）                 | 
| /rcon players                 | -               | 显示当前所有在线玩家信息                                            |
| /rcon reloadlog               | -               | 重新加载服务器日志文件                                              |

### 管理命令

| 命令语法                     | 接受参数        | 说明                                                                 | 
| ----------------------------- | --------------- | -------------------------------------------------------------------- | 
| /rcon kick [id]               | 整数            | 将目标玩家踢出服务器                                               | 
| /rcon ban [id]                | 整数            | 禁止目标玩家进入服务器                                             | 
| /rcon banip [ip]              | 字符串          | 禁止某个 IP 地址访问服务器                                         | 
| /rcon unbanip [ip]            | 字符串          | 解除对某个 IP 地址的封禁                                           | 
| /rcon reloadbans              | -               | 重新加载封禁列表                                                   |

## 获取服务器信息


你需要所有连接服务器所需的相关信息。服务器信息可以在你的游戏服务器控制面板的仪表盘上找到。重要信息包括：

- 服务器名称
- 搜索会话 ID（实时控制台）


利用以上信息，你应该能找到你的服务器，确保连接到正确的服务器。

## 建立服务器连接


连接游戏服务器通常有一种或多种方式。下面详细介绍并说明可用的方法。

<Tabs>
    <TabItem value="connect_solution_server_browser_ingame" label="游戏内服务器浏览器" default>

要连接服务器，你需要启动或继续你的**职业生涯**。然后点击**车队**，打开**车队大厅**。服务器列表会打开，你可以用服务器名称搜索你的服务器。

:::warning 服务器看不到？
如果用服务器名称找不到你的服务器，可以改用实时控制台里的搜索会话 ID 来搜索服务器。复制搜索会话 ID，去掉“/101”部分再搜索。
:::

## 可能遇到的问题及解决方案

服务器不可见通常是因为初始化未成功完成。可能是配置错误或文件损坏导致。你可以在服务器控制台或日志文件中找到更多线索。

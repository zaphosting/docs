## 介绍

Web空间套餐提供通过外部Plesk界面创建和管理网站及邮箱的功能。Web空间充当网页存储。文件和文件夹可以通过FTP连接上传。

每个通过Zap-Hosting预订的Web空间都会分配一个自动生成的子域名。该子域名由客户编号、主机服务器和域名“zap-webspace.com”组成。文档“绑定域名”展示了如何将自己的域名连接到Web空间。

## 入门步骤

订单完成后，Web空间会显示在Zap-Hosting控制面板中。

### 打开Plesk

登录信息可以在打开的页面查看。密码不会以明文保存。要显示密码，需要点击“眼睛”图标。

![Bildschirmfoto vom 2022-05-13 05-16-48](https://screensaver01.zap-hosting.com/index.php/s/B3stJeKMAsYRRer/preview)

:::info
如果你不喜欢自动生成的密码，可以在同一页面直接点击“更改密码”按钮进行修改。
:::

查看用户名和密码后，点击“**打开管理面板**”按钮即可进入Plesk界面。跳转到Plesk界面后，需要输入之前查看的登录信息。

![Bildschirmfoto vom 2022-05-13 05-17-21](https://screensaver01.zap-hosting.com/index.php/s/tZLEJZTzo7kqn4J/preview)

:::info
用户界面的语言可以从德语切换为英语，默认语言为德语。
:::

### 绑定域名

你可以在Plesk界面的菜单项“**添加域名**”中输入自己的域名。

![Bildschirmfoto vom 2022-05-13 05-18-15](https://screensaver01.zap-hosting.com/index.php/s/LHJkPqAsC7zGysW/preview)

在“**添加域名**”中，直接输入你的域名URL，并选择文件位置、SSL等相关设置。

![Bildschirmfoto vom 2022-05-13 05-19-06](https://screensaver01.zap-hosting.com/index.php/s/JXYg4ycDpwqYxCN/preview)

点击“**确定**”按钮确认更改。

:::info
确认可能需要一些时间。如果更改时出现错误，可能的原因是你的域名已经作为别名存在于域中。
:::

如果更改无误，Plesk界面会显示确认信息。要使该域名可用于Web空间，必须在域名管理中配置DNS设置。通过我们订购的域名如何操作，请查看这里。

![Bildschirmfoto vom 2022-05-13 05-19-40](https://screensaver01.zap-hosting.com/index.php/s/TecSRSimmCrRmYj/preview)

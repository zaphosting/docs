## 我从哪里获取 Mods？
因为 BeamMP 是基于 BeamNG 构建的，所以它当然可以使用 BeamNG 的 Mods。
有多个地方可以下载 Mods，官方资源可以在 beamng.com/resources 找到。

## 上传数据
首先，你需要通过 FTP 连接到你的 BeamMP 游戏服务器。登录凭据可以在你的 ZAP 面板的“FTP 浏览器”中找到。

![](https://screensaver01.zap-hosting.com/index.php/s/98Yz5M7K6YDP9Fd/preview)

使用你喜欢的 FTP 软件连接，打开目录

```
/g*****/beammp/Resources/Client/
```

将你的 mod 的 .zip 文件上传到这里。

:::info
服务器重启后，mod 会自动生效，并自动下载到你的客户端。
:::

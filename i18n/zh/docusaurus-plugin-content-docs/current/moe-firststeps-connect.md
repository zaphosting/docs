## 获取服务器信息


你需要准备所有连接服务器所需的相关信息。服务器信息可以在你的游戏服务器控制面板的仪表盘上找到。重要信息包括：

- IP 地址和端口
- 服务器名称
- 服务器密码（可选）


利用以上信息，你就能找到你的服务器，确保连接到正确的服务器。

## 建立服务器连接


连接游戏服务器通常有一种或多种方式。下面详细介绍了可用的方法。

<Tabs>
    <TabItem value="connect_solution_server_browser_ingame" label="游戏内服务器浏览器" default>

要通过游戏内服务器浏览器连接服务器，先在主菜单打开服务器浏览器。点击 **自定义服务器**。在搜索框中输入服务器名称搜索你的服务器。找到后选中服务器，点击 **连接** 即可建立连接。

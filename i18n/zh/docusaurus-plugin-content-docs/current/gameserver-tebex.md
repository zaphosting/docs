## 连接 Tebex

用 ZAP 设置 Tebex 超简单。安装和配置只需几个步骤，下面会详细说明。

### 选择支持的游戏

选一个 Tebex 支持的游戏并添加到你的游戏服务器。点击你的游戏服务器，侧边栏选择 `Games`。在 `Available Games` 里选一个支持的游戏，点击下载按钮安装。

### 请求 Tebex 邀请

如果你选的游戏支持 Tebex，侧边栏会出现 “Tebex Shop” 标签。点击页面加载时出现的图片。

![](https://screensaver01.zap-hosting.com/index.php/s/SztAWeeNX8fx5kF/preview)

Tebex 会向你 ZAP-Hosting 账户绑定的邮箱发送邀请邮件。请按照邮件里的指引激活你的 Tebex 账户，系统会引导你完成店铺的基础配置。

### 安装 Tebex 插件

这个过程通常是自动的。如果我们的服务器没有自动安装插件，或者没收到 Tebex 的 pingback，你就得手动安装。

### 配置 Tebex 插件

手动安装适合你游戏的服务器插件。不同游戏的安装流程不一样。基础安装说明可以看这里：[官方文档](https://docs.tebex.io/store/integrating-with-your-game-server-or-website/minecraft-java-edition)。

## 验证与 Tebex 的连接

最后你得确认游戏服务器插件安装成功。

![](https://screensaver01.zap-hosting.com/index.php/s/DkDtPMtij2QTKGS/preview)

有时候你需要输入 Tebex 网店插件的 API Key，才能让 ZAP 和你的 Tebex 店铺成功连接。连接成功后你会看到这个界面：
![](https://screensaver01.zap-hosting.com/index.php/s/4YS2SwHfSTgn4DE/preview)

## 获取服务器信息

你需要准备好所有连接服务器所需的重要信息。服务器信息可以在你的游戏服务器控制面板的仪表盘里找到。重要信息包括：

- IP 地址和端口
- 服务器名称
- 服务器密码（可选）

凭借以上信息，你就能找到你的服务器，确保连接到正确的服务器。

## 建立服务器连接

连接游戏服务器通常有一种或多种方式。下面详细介绍可用的方法。

<Tabs>
    <TabItem value="connect_solution_server_browser_ingame" label="游戏内服务器浏览器" default>

要通过游戏内服务器浏览器连接你的服务器，先在主菜单打开服务器浏览器。点击服务器列表右上角的 **通过 IP 连接**，输入你的服务器 IP 地址和端口。按下 `Enter` 键确认，即可建立连接。

## 可能遇到的问题及解决方案

连接游戏服务器通常有一种或多种方式。下面详细介绍可用的方法。

#### 服务器不可见

如果服务器初始化未成功完成，可能导致服务器不可见。这可能是由于配置错误或文件损坏引起的。通常可以在服务器控制台或日志文件中找到更多信息。

## 介绍

Hytale的抢先体验版本带来了一个丰富多彩的世界，里面充满了各种物品、制作路径、工具、武器和实用道具。无论是新手还是老玩家，面对海量物品及其隐藏属性，尤其是在游戏早期没有基础游戏提供详细物品数据时，都会感到有点手足无措。

游戏本身目前还没有在工具提示或背包界面中包含全面的物品元数据，这让理解物品的特性、用途和需求变得比较困难，除非借助外部资料。

为了解决这个问题，**高级物品信息**模组通过一个可搜索的界面，为游戏内物品提供**额外的上下文信息**，让玩家无需猜测即可直接获取详细数据。

## 高级物品信息提供了什么

高级物品信息增加了一个专门的GUI，展示游戏中物品的丰富信息，包括：

- 物品识别及唯一ID  
- 视觉图标和品质等级  
- 等级需求和物品等级  
- 最大堆叠数量和耐久度  
- 是否为燃料  
- 是否为消耗品或具有特殊属性  
- 是否作为工具、武器、护甲、滑翔翼或实用组件  
- 物品可制作或获得的提示信息


## 如何使用模组

安装HyFixes请按照[安装模组](hytale-mods)指南中的标准流程操作。该指南说明了模组应放置的位置以及服务器如何加载它们。安装完成后，可以通过游戏内命令打开用户界面：

```
/advancedinfo
```

打开GUI后，玩家可以输入或粘贴物品名称进行搜索，点击或悬停结果，查看模组提供的丰富元数据。

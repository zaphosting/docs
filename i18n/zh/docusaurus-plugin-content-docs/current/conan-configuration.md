## 服务器配置

Conan Exiles 服务器支持高度自定义。服务器的配置通过配置文件完成。你可以在游戏服务器管理的网页界面中的“Configs”找到它们。打开 **ServerSettings.ini** 配置文件即可开始设置。



![](https://screensaver01.zap-hosting.com/index.php/s/HSqXbCnS62s9mEx/preview)



下面是所有相关命令的分类总览，你可以根据自己的需求自由配置服务器。某些命令默认可能不在配置文件中，可以直接后续添加。

## 通用设置

通用设置中包含所有自定义服务器名称、密码、防作弊等相关命令。

| 启用 PVP                       | 启用或禁用 PVP。1 = 启用，0 = 禁用                           | PVPEnabled=                     |

## 时间设置

时间设置中包含所有调整游戏时间的相关命令，允许你自由设定白天和黑夜的长短。

| 选项               | 说明                                                         | 配置命令                |
| ------------------ | ------------------------------------------------------------ | ----------------------- |
| 日夜循环速度       | 整个24小时日夜循环的速度倍率。下方选项可单独调整时间段，倍率会叠加。 | DayCycleSpeedScale=     |
| 白天时间速度       | 白天时间为游戏内7:00至16:59，调整此倍率可延长或缩短白天时长。 | DayTimeSpeedScale=      |
| 夜晚时间速度       | 夜晚时间为游戏内19:00至4:59，调整此倍率可延长或缩短夜晚时长。 | NightTimeSpeedScale=    |
| 黎明/黄昏时间速度  | 黎明为5:00至6:59，黄昏为17:00至18:59，调整此倍率影响黎明和黄昏时长。 | DawnDuskSpeedScale=     |
| 启用时间补偿       | 启用或禁用客户端时间补偿机制。true=启用，false=禁用          | UseClientCatchUpTime=   |
| 时间补偿起始时间   | 新玩家可从指定时间开始游戏，直到服务器时间“追上”他们。建议不要设为夜晚时间。 | ClientCatchUpTime=      |

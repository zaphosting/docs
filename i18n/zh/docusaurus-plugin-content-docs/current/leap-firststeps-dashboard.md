## 产品管理

在 **产品管理** 部分，你会找到管理服务器所需的所有页面。下面详细列出了管理服务器的各种选项。



### 仪表盘

你的服务器 **仪表盘** 是管理的核心。这里清晰展示了服务的所有基础和关键信息，比如服务器位置、状态、IP:端口、已用内存、当前游戏、服务器名称、地图和可用槽位数。此外，还会显示 CPU、内存和 RAM 的使用情况。

![img](https://screensaver01.zap-hosting.com/index.php/s/MsSgmBkz37XnaFJ/preview)

在仪表盘中，你还可以启动、停止或重启服务，全面掌控服务器运行状态。你也能访问实时控制台，获取服务器状态的实时信息，并在必要时直接操作。



### DDoS 管理器


查看过去的 DDoS 攻击记录，更好地评估潜在威胁和攻击模式。开启服务器正在遭受 DDoS 攻击时的通知，确保事件发生时你能快速响应。

根据服务器所在位置及可用的 DDoS 保护，你还可以实时监控当前攻击情况。这里会显示详细的流量信息，包括正常流量和被防护系统拦截的流量量。这样的透明度帮助你了解防护效果，及早识别潜在风险。

:::info 缺少 DDoS 管理器访问权限？
使用 DDoS 管理器需要拥有专用 IP 地址或额外购买 “DDoS 管理器概览” 选项。如果你当前套餐未包含这些组件，可以随时通过升级服务添加。
:::

![img](https://screensaver01.zap-hosting.com/index.php/s/ScCCCY52CMLgfyE/preview)



:::warning
只有当你拥有专用 IP 地址或额外购买了 “DDoS 管理器概览” 选项时，才能使用 DDoS 管理器。如果当前套餐缺少这些组件，可以随时通过升级添加。
:::



### 日志文件

遇到技术问题，不确定原因？第一步可以查看 **日志文件**。这些文件包含大量服务器信息，比如当前事件、提示或错误信息。

尤其是错误信息，能帮助你识别并更好地理解问题根源。通常，通过有针对性地查看日志文件，可以发现并解决问题。

:::warning
日志信息看不懂或没帮助？别担心！我们每天都有客服支持。只需提交一个 **[工单](https://zap-hosting.com/en/customer/support/)**，告诉我们你的问题，我们会尽快帮你解决！:)
:::


### 事件日志

**事件日志** 为你提供了服务器管理相关活动的全面记录。你可以随时查看服务器何时启动、停止或崩溃，以及何时进行了更新、备份或新安装。

![img](https://screensaver01.zap-hosting.com/index.php/s/xNzsaMbW5BS5KJC/preview)



### 服务中断

服务器曾短暂离线，但你不清楚原因？查看事件日志和日志文件也没头绪？问题可能不在你的服务器本身，而是在承载你服务器的主机系统。

如果我们的监控系统检测到整体中断，技术人员会自动收到通知，并尽快处理问题。

本节提供了你游戏服务器所在主机系统的中断历史概览。更多信息及最新服务状态，请访问我们的新 [状态页面](https://status.zap-hosting.com/)。












## 设置

在 **设置** 部分，你会找到配置服务器所需的所有页面。下面详细列出了可用的配置选项。



### 设置

设置页汇总了服务器最常用的配置选项，并附有简短说明。特别推荐给刚入门服务器管理的新手。

如果你更熟悉操作，想做更精准的调整，建议使用配置文件编辑器。相关信息请查看 **配置文件** 分类。



### 配置文件

配置文件编辑器允许你直接在网站上以经典方式编辑配置文件，而非简化视图。这个选项让你能根据个人喜好自定义服务器设置。

通过编辑器，你可以直接访问所有可用选项，精确调整配置。对高级用户来说，这尤其有用，能完全掌控服务器设置，明确知道要改哪些内容。

:::warning
发现部分修改被覆盖？请注意，某些选项值来自设置页。如果想直接在配置文件中修改，需先在设置页关闭相关选项。
:::


### 游戏

在 **游戏** 里，你可以随时更换现有游戏的游戏模式，或安装其他游戏。相同槽位价格的游戏可直接在 **可用游戏** 中选择。其他游戏则需先调整槽位价格。更多详情请参考 [游戏切换](gameserver-gameswitch.md) 指南。

![img](https://screensaver01.zap-hosting.com/index.php/s/xkkECw7o52fAMWk/preview)



### 版本

在 **版本** 中，你可以查看和管理当前游戏服务器版本。随时更新到最新版本，或开启自动更新，确保服务器始终保持最新状态。

![img](https://screensaver01.zap-hosting.com/index.php/s/BH2JzyRHTeLdKHz/preview)

此外，你还可以开启自动更新的邮件通知功能，随时掌握更新动态，确保信息及时。



### 计划任务

创建计划任务，在指定时间自动执行，帮你自动化服务器上的各种流程。你可以设置任务只执行一次，或按需重复。支持的任务类型包括启动、停止或重启服务，在线时重启服务器，创建备份或执行自定义命令。

![](https://screensaver01.zap-hosting.com/index.php/s/P6DeWiRC3tDqG2z/preview)

### 重启计划器

想在特定时间安排服务器重启？重启计划器帮你设置定时自动重启。你可以设定具体时间或定期间隔执行重启。

![](https://screensaver01.zap-hosting.com/index.php/s/Y6WciDS7YP98P4m/preview)



## 工具

导航栏的 **工具** 部分包含多个子项，帮助你管理服务器，包括 **FTP 浏览器**、**数据库** 和 **备份**。下面为你详细介绍这些功能。



### FTP 浏览器

通过 **FTP 浏览器**，你可以获取连接 FTP 的所有必要信息。FTP 访问让你直接操作游戏服务器文件。我们推荐使用外部 FTP 客户端，如 **FileZilla** 或 **WinSCP**，以获得最佳体验。如果你不熟悉 FTP 使用，建议查看我们的 [FTP 访问](gameserver-ftpaccess.md) 指南。

![img](https://screensaver01.zap-hosting.com/index.php/s/sX85i9HyzR9wmAM/preview)

**FileZilla：** FileZilla 是最简单且最知名的 FTP 客户端。安装后启动，在顶部栏输入本页稍上方提供的登录信息，点击“连接”，即可直接登录服务器，管理文件。

**WinSCP：** WinSCP 主要用于 SCP 和 SFTP（FTP 的加密版本），但同样支持普通 FTP。功能比 FileZilla 更丰富，适合进阶用户。



### 数据库

除了游戏服务器产品，我们还提供 10 个免费数据库。当需要将数据存储到数据库时非常有用。在 **数据库** 中，你可以创建数据库并查看访问凭证，还能开启备份功能，导入已有数据库备份。

![img](https://screensaver01.zap-hosting.com/index.php/s/WFLwCbZSQfYjrmD/preview)



### 备份

在 **备份** 中，你可以为服务器及其数据库创建手动或自动备份。这样在出现问题或数据丢失时，更容易恢复数据。

你可以设置是否自动备份，以及备份频率（日常或每周）。还可以选择在存储空间紧张时，自动删除最旧的备份。

![img](https://screensaver01.zap-hosting.com/index.php/s/nm9za5LTMEcNsga/preview)

我们默认为游戏服务器产品提供 10 GB 备份存储空间。如果需要更多空间，可以付费升级。这样你就能确保备份空间充足，必要时还能访问旧备份。

## 产品管理

在**产品管理**部分，你会找到管理服务器所需的所有页面。下面详细列出了管理服务器的各种选项。



### 仪表盘

服务器的**仪表盘**是管理的核心。这里清晰展示了服务的所有基础且重要的信息，比如服务器位置、状态、IP:端口、内存使用情况、当前游戏、服务器名称、地图和可用槽位数。此外，还会显示 CPU、内存和 RAM 的使用率。

![](https://screensaver01.zap-hosting.com/index.php/s/oYDKFk586rJmHwE/preview)

在仪表盘中，你还可以启动、停止或重启服务器，全面掌控其运行状态。你也能访问实时控制台，获取服务器状态的实时信息，并在必要时直接操作。



### DDoS 管理器


查看过去的 DDoS 攻击记录，更好地评估潜在威胁和攻击模式。开启服务器正在遭受 DDoS 攻击时的通知，确保你能快速响应突发事件。

根据服务器所在位置及可用的 DDoS 保护，你还可以实时监控正在进行的攻击。这包括详细的流量分析，显示合法流量和被防护系统拦截的流量量。这样的透明度帮助你了解防护效果，及早识别潜在风险。

:::info 缺少 DDoS 管理器访问权限？
使用 DDoS 管理器需要拥有专用 IP 地址或额外购买“DDoS 管理器概览”选项。如果你当前套餐未包含这些组件，可以随时通过升级服务添加。
:::

![](https://screensaver01.zap-hosting.com/index.php/s/ScCCCY52CMLgfyE/preview)



:::warning
只有当你拥有专用 IP 地址或额外购买了“DDoS 管理器概览”选项时，才能使用 DDoS 管理器。如果当前套餐缺少这些组件，可以随时通过升级添加。
:::



### 日志文件

遇到技术问题却不确定原因？第一步可以查看**日志文件**。这些文件包含大量服务器信息，比如当前事件、提示或错误信息。

尤其是错误信息，能帮助你定位并更好地理解问题根源。通常，通过有针对性地查看日志文件，可以发现并解决问题。

:::warning
看不懂日志里的信息或没帮助？别担心！我们每天都在线客服。只需提交一个**[工单](https://zap-hosting.com/en/customer/support/)**，告诉我们你的问题，我们会尽快帮你解决！:)
:::


### 事件日志

**事件日志**为你提供了服务器管理相关活动的全面概览。你可以随时查看服务器何时启动、停止或崩溃，以及何时进行了更新、备份或新安装。

![](https://screensaver01.zap-hosting.com/index.php/s/xNzsaMbW5BS5KJC/preview)



### 实时控制台

**实时控制台**让你实时查看服务器上的所有当前事件。这里能看到玩家活动信息、错误消息和系统提示。这个视图让你全面了解服务器动态，及早发现潜在问题。

![](https://screensaver01.zap-hosting.com/index.php/s/J9SsmSrPXnG7P9w/preview)

除了监控事件，你还可以在**实时控制台**直接向服务器发送命令。这对于调整设置、执行特定操作或快速应对突发问题非常有用。



### 服务中断

服务器曾短暂离线但你不清楚原因？查看事件日志和日志文件也没发现线索？问题可能不在你的服务器本身，而是托管服务器的主机系统。

如果我们的监控系统检测到整体中断，技术团队会自动收到通知并尽快处理。

本节提供了你游戏服务器所在主机系统的中断历史概览。更多信息及最新服务状态，请访问我们的新[状态页面](https://status.zap-hosting.com/)。










## 设置

在**设置**部分，你会找到配置服务器所需的所有页面。下面详细列出了可用的配置选项。



### 设置

这里汇总了服务器最常用的设置选项，并附有简短说明。特别推荐给刚接触服务器管理的新手使用。

如果你更熟悉操作，想做更精准的调整，建议使用配置文件编辑器。相关内容可在**配置文件**分类中找到。



### 配置文件

配置文件编辑器允许你直接在网站上以经典方式编辑配置文件，而非简化视图。这个功能让你可以根据个人喜好自定义服务器设置。

通过编辑器，你能直接访问所有可用选项，精准调整。对高级用户来说，这尤其有用，可以完全掌控服务器配置，明确知道要改哪些内容。

:::warning
发现部分修改被覆盖？请注意，某些选项值来自设置页面。如果想直接在配置文件中修改，需先在设置页面关闭对应选项。
:::


### 游戏

在**游戏**中，你可以随时更换现有游戏的游戏版本，或者安装其他游戏。相同槽位价格的游戏可以直接在**可用游戏**中选择。其他游戏则需先调整槽位价格。更多信息请参考[游戏切换](gameserver-gameswitch.md)指南。

![](https://screensaver01.zap-hosting.com/index.php/s/xkkECw7o52fAMWk/preview)



### 版本

在**版本**中，你可以查看和管理当前游戏服务器版本。随时更新到最新版本，或开启自动更新，确保服务器始终保持最新状态。

![](https://screensaver01.zap-hosting.com/index.php/s/BH2JzyRHTeLdKHz/preview)

此外，你还可以开启自动更新的邮件通知功能，及时了解更新情况，确保信息同步。



### 计划任务

创建计划任务，自动在指定时间执行各种服务器操作。你可以设置任务只执行一次或循环执行。支持的任务类型包括启动、停止或重启服务，在线时重启服务器，创建备份或执行自定义命令。

![](https://screensaver01.zap-hosting.com/index.php/s/P6DeWiRC3tDqG2z/preview)

### 重启计划器

想在特定时间安排服务器重启？重启计划器帮你设置定时自动重启。你可以设定具体时间或周期性重启。

![](https://screensaver01.zap-hosting.com/index.php/s/Y6WciDS7YP98P4m/preview)

### DNS 设置
通过 ZAP DNS 设置，你可以为服务创建易记的地址。可以使用我们域名的子域名，或者在你自己的域名上创建 DNS 记录，打造更符合你个人品牌的地址。
![](https://screensaver01.zap-hosting.com/index.php/s/tM5FGTHRSbZA5rN/preview)

## 工具

导航栏中的**工具**部分包含多个子项，帮助你管理服务器，包括**FTP 浏览器**、**数据库**和**备份**。下面为你详细介绍这些功能。



### FTP 浏览器

通过**FTP 浏览器**，你可以获取连接 FTP 的所有必要信息。FTP 访问让你直接操作游戏服务器文件。我们推荐使用外部 FTP 客户端，如 **FileZilla** 或 **WinSCP**，以获得最佳体验。如果你不熟悉 FTP 使用，建议查看[FTP 访问指南](gameserver-ftpaccess.md)。

![](https://screensaver01.zap-hosting.com/index.php/s/sX85i9HyzR9wmAM/preview)

**FileZilla：** FileZilla 是最简单且最知名的 FTP 客户端。安装后启动，在顶部栏输入本页稍上方提供的登录信息，点击“连接”，即可直接登录服务器，管理文件。

**WinSCP：** WinSCP 主要用于 SCP 和 SFTP（FTP 的加密版本），但同样支持普通 FTP。功能比 FileZilla 更丰富，适合进阶用户。



### 数据库

除了游戏服务器产品，我们还免费提供 10 个数据库。当需要将数据存储到数据库时非常有用。在**数据库**中，你可以创建数据库并查看访问凭证。还可以开启备份选项，导入已有数据库备份。

![](https://screensaver01.zap-hosting.com/index.php/s/WFLwCbZSQfYjrmD/preview)



### 备份

在**备份**中，你可以为服务器及其数据库创建手动或自动备份。这样在出现问题或数据丢失时，更容易恢复数据。

你可以设置是否自动备份，以及备份频率（日常或每周）。还可以选择是否在存储空间不足时删除最旧的备份。

![](https://screensaver01.zap-hosting.com/index.php/s/nm9za5LTMEcNsga/preview)

我们默认为游戏服务器产品提供 10 GB 备份存储空间。如果需要更多空间，可以付费升级。这样你就能确保备份空间充足，必要时还能访问旧备份。

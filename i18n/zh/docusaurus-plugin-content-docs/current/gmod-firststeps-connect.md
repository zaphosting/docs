## 获取服务器信息


你需要所有连接服务器所需的相关信息。这些信息可以在你的游戏服务器控制面板的仪表盘上找到。重要信息包括：

- IP 地址和端口
- 服务器名称
- 服务器密码（可选）


利用以上信息，你就能找到你的服务器，确保连接到正确的服务器。

## 建立服务器连接


连接游戏服务器通常有多种方式。下面详细介绍并说明可用的方法。

<Tabs>
    <TabItem value="connect_solution_server_browser_ingame" label="游戏内服务器浏览器" default>

通过游戏内服务器浏览器连接服务器，先在主菜单打开服务器浏览器。点击 **查找多人游戏**，然后打开 **传统浏览器**。进入后，使用 **搜索框** 输入你的服务器名称，找到后点击 **连接**。

![img](https://screensaver01.zap-hosting.com/index.php/s/XFpND5FA4ZAxfHq/download)

</TabItem>

<TabItem value="connect_solution_server_browser_steam" label="Steam 服务器浏览器">



通过 Steam 服务器浏览器连接服务器，先打开 Steam 的服务器浏览器。点击 Steam 顶部导航的 `查看`，然后选择 `游戏服务器` 进入。

在 Steam 浏览器中，使用你的游戏服务器的 `IP 地址` 和 `端口` 搜索或添加服务器。选择服务器列表中的 `收藏夹` 标签，点击底部的加号。弹出窗口中填写你的服务器信息。

![img](https://screensaver01.zap-hosting.com/index.php/s/MMsokw2ZyCreeCN/download)

</TabItem>

<TabItem value="connect_solution3" label="游戏内控制台">

通过控制台连接服务器，按 `~` 键打开控制台。打开后执行以下命令：

```
connect IP:Port
```

![img](https://screensaver01.zap-hosting.com/index.php/s/gpqBB72jx7r2A48/preview)

</TabItem>
</Tabs>

## 介绍

**Windrose** 仪表盘是您在 ZAP-Hosting 网页界面中管理游戏服务器的核心区域。本指南将为您提供仪表盘各部分的结构化概览，并教您如何高效使用最重要的管理、监控和托管工具。

<InlineVoucher />

## 产品管理

在**产品管理**区域，您可以找到控制和监控您的 **Windrose** 服务器的关键页面。这些页面帮助您管理当前服务器状态，查看技术事件，并快速访问重要的服务信息。

### 仪表盘

**仪表盘** 是您 **Windrose** 服务器的主概览页面。这里通常可以一目了然地看到最重要的服务详情，例如当前服务器状态、位置、连接地址和资源使用情况。

![img](https://screensaver01.zap-hosting.com/index.php/s/boTwmEC5HSjbKYK/preview)

根据当前服务器设置和游戏集成，仪表盘还可能显示额外信息，如当前地图、服务器名称、已用内存或玩家槽位使用情况。您也可以直接在这里启动、停止或重启服务。

仪表盘特别实用，因为它将服务器控制和监控功能合二为一，让您能快速检查服务是否在线，并在需要时立即做出响应。

### 统计数据

在**统计数据**中，您可以查看服务器的历史使用数据，帮助您更好地了解 **Windrose** 托管服务的性能变化。

![](https://screensaver01.zap-hosting.com/index.php/s/N8jcmdqqG2Xt4Bk/preview)

可用图表通常包括 CPU 使用率、内存使用率和槽位利用率。您通常可以切换不同时间范围，分析短期峰值或长期趋势。

| 指标 | 作用 |
| --- | --- |
| CPU 使用率 | 帮助识别处理负载和可能的性能瓶颈 |
| 内存使用率 | 显示服务器使用了多少 RAM |
| 槽位利用率 | 表示有多少可用玩家槽位被实际使用 |

:::tip 性能分析
如果您发现延迟、崩溃或响应缓慢，统计页面是排查问题的最佳起点之一。它能帮助您判断问题是否与资源使用相关，或是否仅在特定时间发生。
:::

### DDoS 管理器

**DDoS 管理器** 允许您查看针对您的服务检测到并已缓解的 DDoS 攻击信息。如果您想了解异常流量模式或调查因恶意流量导致的服务中断，这非常有用。

![](https://screensaver01.zap-hosting.com/index.php/s/ScCCCY52CMLgfyE/preview)

根据您预订的选项和所选位置的可用防护，您还可能实时查看活跃攻击信息和流量详情。

:::info DDoS 管理器可用性
只有当您的服务包含专用 IP 地址或额外选项 **DDoS 管理器概览** 时，才可使用 DDoS 管理器。如果当前套餐未包含，您需要先升级服务。
:::

### 日志文件

如果您的 **Windrose** 服务器出现错误或异常行为，**日志文件** 是最重要的检查点之一。日志文件包含服务器的技术输出，可能包括警告、启动信息和错误详情。

这些文件在识别崩溃原因、启动失败或配置问题时尤其有用。

:::caution 日志查看限制
如果日志输出不清晰或无法帮助您定位问题，请通过官方 [支持页面](https://zap-hosting.com/en/customer/support/) 联系客服。请提供清晰的问题描述和相关日志信息，以加快排查速度。
:::

### 事件日志

**事件日志** 提供与您的服务相关的操作和事件概览，包括服务器启动、停止、崩溃、安装、更新和备份活动。

![](https://screensaver01.zap-hosting.com/index.php/s/xNzsaMbW5BS5KJC/preview)

如果您想确认某次变更的时间或自动任务是否成功执行，这部分特别有帮助。

### 实时控制台

**实时控制台** 显示您的 **Windrose** 服务器运行时的实时输出，方便您监控服务器当前状态，及时发现问题。

![](https://screensaver01.zap-hosting.com/index.php/s/J9SsmSrPXnG7P9w/preview)

如果服务器软件支持，您还可以通过实时控制台直接发送命令，适合无需重启服务即可执行的管理操作。

| 功能 | 描述 |
| --- | --- |
| 实时输出 | 显示当前服务器事件和消息 |
| 错误可见性 | 帮助您即时发现问题 |
| 命令输入 | 允许直接发送支持的服务器命令 |

### 服务中断

如果服务器曾短暂不可用，且事件日志或日志文件中无明显原因，问题可能出在底层主机系统，而非您的单独服务。

在**服务中断**部分，您可以查看与游戏服务器主机环境相关的中断历史。若需更广泛的平台状态信息，也可访问官方 [状态页面](https://status.zap-hosting.com/)。

## 设置

**设置** 区域包含您的 **Windrose** 服务器的主要配置页面。您可以在这里调整服务行为、更换已安装游戏（如支持）、管理更新和配置自动化功能。

### 设置

**设置** 页面提供常用服务器选项的简化概览，特别适合游戏服务器托管新手，无需直接编辑配置文件即可进行更改。

如果 **Windrose** 支持，这些选项可能包括与游戏玩法相关的设置或其他常见服务参数。具体选项视当前游戏集成而定。

### 配置文件

**配置文件** 部分允许您通过网页界面直接编辑配置文件，提供比简化设置页面更精细的控制，通常更适合高级用户。

:::caution 被覆盖的值
部分值可能由简化的 **设置** 页面管理。如果您发现配置编辑器中的手动更改被覆盖，请检查该选项是否仍由设置页面控制。
:::

### 游戏

在**游戏**部分，您可以切换到另一个可用游戏或游戏变体（如果服务支持）。同价位的游戏通常可直接选择，其他游戏可能需要先调整套餐。

想了解更多关于更换已安装游戏的信息，请参考 [游戏切换](gameserver-gameswitch.md) 指南。

![](https://screensaver01.zap-hosting.com/index.php/s/xkkECw7o52fAMWk/preview)

### 版本

**版本** 页面允许您管理 **Windrose** 服务的已安装服务器版本（如果支持版本管理）。您通常可以更新到最新版本并启用自动更新。

![](https://screensaver01.zap-hosting.com/index.php/s/BH2JzyRHTeLdKHz/preview)

自动更新有助于保持托管环境的最新状态，但在重大版本变更后，您仍应验证兼容性。

### 计划任务

通过**计划任务**，您可以自动化服务器的周期性操作，减少手动管理工作，保持服务稳定运行。

![](https://screensaver01.zap-hosting.com/index.php/s/P6DeWiRC3tDqG2z/preview)

常见自动化操作包括：

| 任务类型 | 示例用途 |
| --- | --- |
| 启动服务 | 在指定时间自动启动服务器 |
| 停止服务 | 在维护时间段关闭服务器 |
| 重启服务 | 定期重启以保持稳定 |
| 备份创建 | 自动创建定期备份 |
| 自定义命令 | 自动执行支持的命令 |

### 重启计划

**重启计划** 专门用于安排自动服务器重启，适合您想在低活跃时段定期重启 **Windrose** 服务器的场景。

![](https://screensaver01.zap-hosting.com/index.php/s/Y6WciDS7YP98P4m/preview)

### DNS 设置

通过**DNS 设置**，您可以为服务分配更易记的地址，方便自己记忆和与其他玩家分享。

![](https://screensaver01.zap-hosting.com/index.php/s/tM5FGTHRSbZA5rN/preview)

根据您的设置和预订选项，您可能可以使用 ZAP 提供的子域名或您自己的域名。

## 工具

**工具** 部分包含支持文件访问、数据库管理和备份处理的额外服务管理功能。

### FTP 浏览器

在**FTP 浏览器**中，您可以找到访问服务器文件所需的连接详情。FTP 访问适合需要直接上传、下载或编辑文件的场景。

![](https://screensaver01.zap-hosting.com/index.php/s/sX85i9HyzR9wmAM/preview)

为了更方便地管理文件，建议使用外部 FTP 客户端，如 [FileZilla](https://filezilla-project.org/) 或 [WinSCP](https://winscp.net/)。如果需要连接帮助，请查看 [通过 FTP 访问](gameserver-ftpaccess.md) 指南。

### 数据库

在**数据库**部分，您可以创建和管理随游戏服务器产品提供的数据库（如果您的配置需要）。这对服务、模组或工具需要结构化数据存储时非常有用。

![](https://screensaver01.zap-hosting.com/index.php/s/WFLwCbZSQfYjrmD/preview)

您通常可以在此查看访问凭据、新建数据库以及导入已有备份。

### 备份

**备份**部分允许您为服务器及其关联数据库数据（如支持）创建和管理备份。

![](https://screensaver01.zap-hosting.com/index.php/s/nm9za5LTMEcNsga/preview)

强烈建议定期备份，因为它能让您在配置错误、文件损坏或意外问题后恢复服务。

:::tip 备份最佳实践
在进行重大更改（如编辑配置文件、切换版本或更换游戏）前，先创建手动备份。这样如果出现问题，您有安全的恢复点。
:::

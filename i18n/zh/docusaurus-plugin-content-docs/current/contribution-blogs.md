在本页了解如何向我们 ZAP-Hosting.com 博客投稿的流程。流程按时间顺序排列，建议从这里开始阅读。

## 第一步：提出内容建议

流程的第一步是创建并提交你想写的博客文章建议。请通过我们的[博客建议表单](https://forms.gle/jft8sb4mmKMkbt5W9)完成。你需要提供关于你的创意的关键信息。

表单中会问你几个关于博客建议细节的问题，包括主题、计划使用的大致结构、内容摘要以及你的联系方式。

:::tip
建议先查看我们的[指南页面](contribution-blogs-guidelines.md)，了解我们追求的内容和文档风格。
:::

请尽可能详细填写表单，以提高博客创意被采纳的几率。

### 我们在找什么主题？

我们的博客欢迎各种实用且有趣的内容，涵盖游戏、服务器脚本、技巧、热情分享等多种主题。我们希望博客内容多样化，涵盖丰富主题和风格，让博客既有料又好看。

如前所述，所有内容必须原创，遵守版权法规。严禁未经授权使用 AI/生成工具或其他知识产权内容。

### 提交建议

了解完信息后，请前往我们的**[博客建议表单](https://forms.gle/jft8sb4mmKMkbt5W9)**，尽量详细填写并提交。你还需要确认接受我们的贡献计划条款，详见[贡献条款](contribution-terms.md)。

:::caution
请务必正确填写你的 ZAP ID，因为我们会通过它联系你关于博客建议的事宜！
:::

提交建议后，我们会在3-5天内回复。若建议**被采纳**，你会通过网站的**工单系统**收到回复，联系到你提供的 ZAP ID。若未被采纳，我们不会回复。

:::tip
**遇到问题？** 欢迎加入我们的[ZAP-Hosting Discord 服务器](https://zap-hosting.com/discord)，ZAP 社区和我们都乐意帮你解答。或者通过网站[提交工单](https://zap-hosting.com/en/customer/support/)，并在留言中注明“贡献计划 - 博客”，我们会转给贡献团队。
:::

工单回复中会给你反馈和调整建议。双方确认细节后，你就可以开始写博客，进入下一步。

## 第二步：撰写博客文章

这一步，你将根据与 ZAP 贡献团队确认的信息撰写博客。你可以用任何工具写作，灵活自由。

目前我们支持**Markdown**和**HTML5**格式。

:::tip
强烈推荐用 Markdown 编辑器 **[StackEdit](https://stackedit.io/app#)**，它功能强大且支持 Markdown。喜欢 HTML5 的话，可以用 **[HTML5 Editor](https://html5-editor.net/)**。写好后可直接导出到 Google Drive 或其他云盘，生成分享链接给我们。更多 Markdown 规范请看我们的博客[指南](contribution-blogs-guidelines.md)。
:::

:::note
请确保你已经通过[博客建议表单](https://forms.gle/jft8sb4mmKMkbt5W9)提交建议，并收到 ZAP 贡献团队通过工单的回复确认。未按流程或未获批准的内容，我们无法保证奖励发放。
:::

写作时请严格按照确认的结构和内容进行。我们没有硬性截止时间，但如果拖得太久，可能会联系你确认进度。

有任何问题，随时回复工单联系我们，我们很乐意帮忙！:D

## 第三步：提交草稿

提交草稿前，请用下面的清单自查：
- 结构清晰。
- 内容符合建议和与贡献团队的讨论。
- 严格遵守我们的[指南](contribution-blogs-guidelines.md)。

确认草稿完成后，请回复工单提交。工单不支持直接上传文件，建议上传到云盘（如 Google Drive）并分享链接。推荐用 Google Docs 或 Microsoft Word 写作，方便快速生成分享链接。

我们会在2-3天内回复。审核后可能会要求你做调整，届时请根据反馈修改并重新提交草稿，直到双方都满意。

## 第四步：发布博客 & 奖励发放

最终稿确认后，我们会回复工单，告知博客发布时间安排及奖励发放细节。

:::info
奖励金额取决于内容质量、篇幅及其他由 ZAP 贡献团队评估的标准。详情请看我们的[奖励说明](contribution-rewards.md)。
:::

恭喜你为 ZAP-Hosting 指南贡献内容！我们非常感谢你的付出！💚

## 获取服务器信息


你需要所有连接服务器所需的相关信息。这些信息可以在你的游戏服务器控制面板的仪表盘中找到。重要信息包括：

- IP 地址和端口
- 服务器名称
- 服务器密码（可选）


利用以上信息，你就能找到你的服务器，确保连接到正确的服务器。

## 建立服务器连接


连接游戏服务器通常有一种或多种方式。下面详细介绍可用的方法。

<Tabs>
    <TabItem value="connect_solution_server_browser_ingame" label="服务器浏览器" default>

要连接服务器，启动游戏或打开启动器。进入启动器中的 **服务器** 菜单。选择服务器列表中的 **互联网** 分类。使用 **搜索框** 输入你的服务器名称进行搜索。找到后选中服务器，点击 **加入** 按钮即可。

![img](https://screensaver01.zap-hosting.com/index.php/s/ogEprRpMdWe7pFn/download)

</TabItem>

<TabItem value="connect_solution_server_browser_direct" label="直接连接" default>

要连接服务器，启动游戏或打开启动器。进入启动器中的 **服务器** 菜单。点击右下角的 **直接连接** 按钮。输入你的服务器 IP 地址和端口，点击 **加入** 按钮即可。

![img](https://screensaver01.zap-hosting.com/index.php/s/pRHcaR7RcWrym9B/download)

</TabItem>

</Tabs>

如果服务器初始化未成功完成，可能导致服务器不可见。这可能是由于配置错误或文件损坏等原因。通常可以在服务器控制台或日志文件中找到更多信息。

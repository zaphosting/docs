## 产品管理

在**产品管理**部分，你会找到管理服务器所需的所有页面。下面详细列出了管理服务器的各项功能。



### 控制面板

服务器的**控制面板**是管理的核心。这里清晰展示了服务的所有基础和重要信息，比如服务器位置、状态、IP:端口、已用内存、当前游戏、服务器名称、地图和可用槽位数。此外，还会显示CPU、内存和RAM的使用情况。

![](https://screensaver01.zap-hosting.com/index.php/s/WdqSnnxYH6Yz29N/preview)

在控制面板中，你还可以启动、停止或重启服务器，全面掌控服务器运行状态。此外，你可以访问实时控制台，获取服务器状态的实时信息，并在必要时直接操作。



### 统计数据

在**统计数据**中，你可以全面查看服务器的负载情况。这里包含了可用槽位、CPU和内存的使用信息。你可以选择不同时间段查看数据，比如最近的**60分钟、24小时、3天、7天**或**30天**。

![](https://screensaver01.zap-hosting.com/index.php/s/N8jcmdqqG2Xt4Bk/preview)

这些信息让你对服务器的使用情况有清晰了解，也有助于发现潜在的性能问题并定位原因。比如，你可以追踪使用率何时上升，是否与某些事件或变更有关。这在遇到突发问题时尤其重要，能帮助你有针对性地排查和分析问题。



### DDoS 管理器


查看历史DDoS攻击，更好地评估潜在威胁和攻击模式。开启服务器DDoS攻击通知，确保事件发生时你能快速响应。

根据服务器所在位置及可用的DDoS防护，你还可以实时监控当前攻击情况。包括详细的流量分析，显示合法流量和被防护系统拦截的流量。这样的透明度帮助你了解防护效果，及早识别潜在风险。

:::info 缺少DDoS管理器访问权限？
使用DDoS管理器需要专用IP地址或额外购买“DDoS管理器概览”选项。如果当前套餐未包含这些组件，你可以随时通过升级服务添加。
:::

![](https://screensaver01.zap-hosting.com/index.php/s/ScCCCY52CMLgfyE/preview)



:::warning
只有当你拥有专用IP地址或购买了“DDoS管理器概览”附加选项时，才能使用DDoS管理器。如果当前套餐缺少这些组件，可以随时升级添加。
:::

![](https://screensaver01.zap-hosting.com/index.php/s/TbwafKd3Kmq4WrS/preview)

Tebex 使用简单，提供丰富功能来自定义和优化你的商店。你可以调整商店布局、添加支付方式，甚至发起营销活动，吸引更多玩家。

点击图片即可创建属于你的游戏服务器商店。设置快速简单，让你马上开始销售虚拟物品。

遇到技术问题，不确定原因？第一步可以查看**日志文件**。这些文件包含大量服务器信息，如当前事件、提示或错误信息。

尤其是错误信息，能帮助你识别并更好地理解问题原因。通过针对性查看日志，常常能发现并解决问题。

### 事件日志

**事件日志**为你提供服务器管理相关活动的全面记录。你可以随时查看服务器何时启动、停止或崩溃，以及何时执行了更新、备份或新安装。

![](https://screensaver01.zap-hosting.com/index.php/s/xNzsaMbW5BS5KJC/preview)



### 实时控制台

**实时控制台**让你实时查看服务器上的所有当前事件。你可以看到玩家活动信息、错误消息和系统消息。这让你全面掌握服务器动态，及早发现潜在问题。

![](https://screensaver01.zap-hosting.com/index.php/s/J9SsmSrPXnG7P9w/preview)

除了监控事件，你还可以直接在**实时控制台**发送命令给服务器。这对于调整设置、执行操作或快速响应问题非常有用。



### 服务中断

服务器曾短暂离线，原因不明？查看事件日志和日志文件也没发现线索？问题可能不在你的服务器，而是托管服务器的主机系统。

如果我们的监控系统检测到整体中断，技术人员会自动收到通知并尽快处理。

本节提供了托管你游戏服务器的主机系统中断历史记录。更多信息和最新服务状态，请访问我们的新[状态页面](https://status.zap-hosting.com/)。












## 设置

在**设置**部分，你会找到配置服务器所需的所有页面。下面详细列出了可用的配置选项。



### 设置

设置页面汇总了服务器最常用的配置选项，并附有简短说明。特别推荐给刚接触服务器管理的新手。

如果你更有经验，想做更精准的调整，建议使用配置文件编辑器。相关信息请查看**配置文件**分类。



### 配置文件

配置文件编辑器允许你直接在网站上以传统方式编辑配置文件，而非简化视图。这个功能让你可以根据个人喜好精细调整服务器设置。

通过编辑器，你能直接访问所有可用选项，精准定制。对高级用户来说，这尤其有用，他们知道自己想改什么。

:::warning
发现部分修改被覆盖？请注意，某些选项值来自设置页面。如果想直接在配置文件中修改，需先在设置页面禁用相关选项。
:::

## 工具

在导航面板的**工具**部分，你会找到一系列子项，帮助你管理服务器，包括**FTP 浏览器**、**数据库**和**备份**。下面为你详细介绍这些功能。



### FTP 浏览器

通过**FTP 浏览器**，你可以获取连接FTP所需的所有信息。FTP访问让你直接操作游戏服务器文件。我们推荐使用外部FTP客户端，如**Filezilla**或**WinSCP**，以获得最佳体验。如果你不熟悉FTP使用，建议查看[FTP访问指南](gameserver-ftpaccess.md)。

![](https://screensaver01.zap-hosting.com/index.php/s/sX85i9HyzR9wmAM/preview)

**FileZilla:** FileZilla 是最简单且最知名的FTP客户端。安装后启动，在顶部栏输入你的登录信息（本页稍上方可见），点击“连接”即可直接登录服务器，管理文件。

**WinSCP:** WinSCP 本为SCP和SFTP（FTP的加密版本）设计，但同样支持普通FTP。功能比FileZilla更丰富，适合进阶用户。



### 数据库

除了游戏服务器产品，我们还提供10个免费数据库。当需要将数据存储到数据库时非常有用。在**数据库**中，你可以创建数据库并查看访问凭证，还能启用备份功能和导入已有数据库备份。

![](https://screensaver01.zap-hosting.com/index.php/s/WFLwCbZSQfYjrmD/preview)



### 备份

在**备份**中，你可以为服务器及其数据库创建手动或自动备份，方便在出现问题或数据丢失时恢复。

你可以设置是否自动备份，以及备份频率（日常或每周）。还可以选择是否在存储空间不足时删除最旧备份。

![](https://screensaver01.zap-hosting.com/index.php/s/nm9za5LTMEcNsga/preview)

我们默认为游戏服务器产品提供10GB备份存储空间。如需更多空间，可付费升级，确保你有足够空间保存备份，并能访问较旧的备份。

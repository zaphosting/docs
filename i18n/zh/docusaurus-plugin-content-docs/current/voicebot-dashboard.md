## 产品管理

在 **产品管理** 部分，你会找到管理 Voicebot 所需的所有页面。下面详细列出了管理 Voicebot 的各项选项。

### 仪表盘

VoiceBot 的 **仪表盘** 是核心管理总览。它以清晰的格式展示 VoiceBot 的所有基础和关键信息，包括在线状态、登录详情、外部网页界面链接以及磁盘使用情况。

![img](https://screensaver01.zap-hosting.com/index.php/s/YnXaQDPywcoxFPP/preview)

### 网页界面

VoiceBot 的大部分功能都通过外部网页界面提供。这个界面允许你选择想要的音乐来源，并提交改进机器人的建议。你还可以更改访问网页界面的密码，以及重新安装 VoiceBot。

![img](https://screensaver01.zap-hosting.com/index.php/s/cmKrJ2YpzmZcm2L/preview)

### 服务器

使用机器人之前，需要先定义它运行在哪个 TeamSpeak 服务器上。必须填写服务器的 IP 地址和端口，以及为机器人自由命名。如果服务器设置了密码，也需要填写，这样机器人才能访问服务器。

![img](https://screensaver01.zap-hosting.com/index.php/s/sbbRr9f5HexcMsG/preview)

### 高级设置

通过配置相应设置，可以激活一些高级功能，比如自动移动用户或在用户加入服务器时播放欢迎消息。这些功能让服务器管理更个性化、更自动化。

![img](https://screensaver01.zap-hosting.com/index.php/s/yz94fyZssG53Rjd/preview)

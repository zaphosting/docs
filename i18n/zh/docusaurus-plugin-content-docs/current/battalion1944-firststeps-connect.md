## 获取服务器信息

你需要准备好所有连接服务器所需的重要信息。这些信息可以在你的游戏服务器控制面板的仪表盘里找到。重要信息包括：

- IP 地址和端口
- 服务器名称
- 服务器密码（可选）

凭借以上信息，你就能找到你的服务器，确保连接到正确的服务器。

## 建立服务器连接

连接游戏服务器通常有一种或多种方式。下面详细介绍并说明可用的方法。

<Tabs>
 

<TabItem value="connect_solution_server_browser_steam" label="服务器浏览器（Steam）">



通过 Steam 服务器浏览器连接服务器，先打开 Steam 的服务器浏览器。点击 Steam 顶部导航栏的 `查看`，然后选择 `游戏服务器` 进入。

通过 Steam 浏览器使用你的游戏服务器的 `IP 地址` 和 `端口` 搜索或添加服务器。操作方法是在服务器列表中选择 `收藏夹` 标签页，点击底部的加号。会弹出一个窗口，输入你的服务器信息即可。

![img](https://screensaver01.zap-hosting.com/index.php/s/MMsokw2ZyCreeCN/download)

</TabItem>

<TabItem value="connect_solution3" label="控制台（游戏内）">

通过控制台连接服务器，按 `~` 键打开控制台。打开后执行以下命令：

```
connect IP:Port
connect IP:Port; password yourdefinedpassword // 如果服务器设置了密码
```

</TabItem>
</Tabs>

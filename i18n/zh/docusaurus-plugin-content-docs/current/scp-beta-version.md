## 介绍

想玩**SCP：秘密实验室**的Beta测试版，不知道怎么操作？我们这就告诉你需要知道的内容，以及如何在游戏和服务器上成功安装这个版本！

<InlineVoucher />


## 游戏端

要使用游戏的Beta测试版，先在你的Steam库里选中游戏，打开**属性**，然后切换到**Beta**标签页。在Beta参与的下拉菜单里，选择**Beta版本**，而不是**无**。

![img](https://screensaver01.zap-hosting.com/index.php/s/BSn8E3D7JBgM6Dy/preview)



## 服务器端

SCP秘密实验室的Beta测试版服务器是作为一个独立游戏提供的。这意味着如果你想用这个版本，必须切换游戏。这个选项可以在游戏服务器管理后台的“游戏”栏目里找到。如果你需要帮助，我们推荐你看看我们的[更换游戏](gameserver-gameswitch.md)指南。



![img](https://screensaver01.zap-hosting.com/index.php/s/BZwaxoknbmFKCJB/preview)



:::warning 插件支持
Beta版本不支持使用Exiled扩展插件。
:::

:::danger 更换游戏（SCP: EXILED 到 SCP: BETA）
从**SCP: Exiled**切换到**SCP: Beta**需要更换主机系统，因为两者条件不同。这个过程中，现有数据会被清空，如果你没有预订专属IP，IP地址也会发生变化。
:::

## 介绍

**ProfiDNS 选项** 提供了对 DNS 设置进行高级配置的可能性。这样您不仅可以链接到 ZAP-Hosting 自家的服务，还能连接到其他外部服务。可能性简直无限大。

<InlineVoucher />

## 访问与使用场景

要访问并激活 EasyDNS 选项，您需要在网页后台打开您的域名管理界面。进入后，您会看到 DNS 管理下的以下选项：

- EasyDNS
- ProfiDNS
- ExpertDNS

默认情况下会选中 **EasyDNS** 选项。所以您需要先选择并激活 **ProfiDNS** 选项。

![2022-05-07_21-51_1](https://screensaver01.zap-hosting.com/index.php/s/tgQaM3iP2oYZsDC/preview)

您可以在域名管理的设置里，**DNS 管理** 下管理 DNS 记录。可以新建记录，也可以调整或删除已有记录。

![2022-05-07_21-50_1](https://screensaver01.zap-hosting.com/index.php/s/DHodS8rAggnMmSF/preview)

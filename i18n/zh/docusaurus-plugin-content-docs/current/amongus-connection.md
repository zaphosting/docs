## 准备工作 - 下载

为了让你加入服务器，我们需要一个能实现连接的文件。
首先打开 [Impostor](https://impostor.github.io/Impostor/)，在这里输入你的信息。

创建文件时，我们需要 IP、端口和一个名称等信息。
![](https://screensaver01.zap-hosting.com/index.php/s/bQ7KzTAJcbAsLJd/preview)

在这里输入你在 ZAP-Hosting 面板上的数据，举个例子是这样的：

![](https://screensaver01.zap-hosting.com/index.php/s/7scfddD5WPFr25b/preview)

其中 31.214.245.215 是你的 IP，2000 是端口。

服务器名称可以随意填写，输入完所有数据后点击“Download Server File”。
这时会下载一个名为 "regioninfo.json" 的文件。

接着按下 `Win` + `R`，输入

```
%APPDATA%\..\LocalLow\Innersloth\Among Us
```

然后回车，会打开一个文件夹。把 regioninfo.json 移进去，覆盖原有文件。

![](https://screensaver01.zap-hosting.com/index.php/s/KTrfij6ww64DPCq/preview)


## 游戏内创建服务器

在你的电脑上启动 Among Us。
主菜单点击“online”，然后点击“Create Game”。

![](https://screensaver01.zap-hosting.com/index.php/s/SfMwr82R9BbDK2P/preview)

这里你可以设置服务器的各种参数：

![](https://screensaver01.zap-hosting.com/index.php/s/imzCR9WZenyrTSX/preview)

设置完成后点击“confirm”。
你现在已经连接到你的服务器啦！

在大厅里你可以选择服务器是公开还是私密。
切换只需点击“Private”或“Public”图标：

如果服务器设置为“Public”，其他玩家也能用同样方式连接。
他们不需要在游戏主菜单点击“Create Game”，而是点击“Find Game”。
这里要确保聊天频道设置为“others”，且筛选条件和服务器设置完全匹配。
否则服务器不会显示出来。

如果是“Private”服务器，玩家需要在主菜单点击“Enter Code”。
然后输入服务器创建者在大厅看到的代码。

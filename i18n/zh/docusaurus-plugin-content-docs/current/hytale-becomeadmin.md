## 介绍
在 Hytale 服务器上赋予管理员权限，可以让值得信赖的玩家协助管理服务器。管理员，也称为运营者，可以执行服务器命令、管理玩家并协助服务器的日常维护。这让服务器拥有者更轻松地分配常规任务，确保游戏体验流畅。

## 赋予权限



Hytale 服务器上的管理员拥有运营者身份，享有更高的命令权限。现有运营者可以通过**服务器控制台**或**游戏内命令**来赋予玩家运营者权限。要给玩家赋予运营者权限，请在服务器控制台或游戏内（需有足够权限）输入：

```
/op add <playername>
```



## 移除权限

如果某位玩家不再需要管理员权限，或者不应继续拥有提升的权限，可以随时移除其运营者身份：

```
/op remove <playername>
```

移除后，该玩家将恢复为普通权限，无法再使用管理员命令。



## 总结

通过赋予运营者权限成为 Hytale 服务器管理员，是服务器管理的核心环节，能让值得信赖的玩家协助执行管理和维护任务。如有任何疑问或需要帮助，欢迎随时联系我们的客服团队，我们每天都在线为你服务！🙂

## 介绍
想要清除服务器进度，让你和你的玩家重新开始吗？这可以通过清档选项实现，下面将详细说明。

<InlineVoucher />



## 清档类型

服务器清档有三种类型，分别是**部分清档**、**黄金清档**和**完全清档**，具体说明如下：

- **部分清档：** 这种清档会部分删除存档文件内容，移除建筑物、自定义地图区域，但保留玩家进度。
- **黄金清档：** 这种清档会部分删除存档文件内容，移除建筑物、自定义地图区域，但保留玩家进度。
- **完全清档：** 这种清档会完全删除存档文件内容，移除建筑物、自定义地图区域以及玩家进度。



## 执行清档

游戏内的服务器设置允许你调用可用的清档选项。为此，连接到你的服务器并打开**服务器设置**菜单。

![img](https://screensaver01.zap-hosting.com/index.php/s/4F7ni5erqNfQwfn/download)

在**常规设置**中找到**部分、黄金和完全清档区域**。根据需要激活所需的清档选项并确认。更改将在下一次重启时生效。

:::warning  需要管理员权限
要访问游戏内的服务器设置，需要拥有[管理员权限](scum-becomeadmin.md)。
:::

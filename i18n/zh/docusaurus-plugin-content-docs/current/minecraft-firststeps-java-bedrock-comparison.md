## 平台支持

很多新手最关心的问题是：“我能在哪些设备上玩？”Minecraft *Java*版，作为原版游戏，只能在Windows、macOS和Linux上玩，且对硬件要求较高（尤其是加了着色器和各种mod后），而Minecraft *基岩*版几乎支持所有设备，从手机到电脑再到主机，且支持全平台联机。

## 更新机制

两个版本都会持续更新，但测试版本的叫法不同：*Java*版叫“快照（snapshot）”，*基岩*版叫“测试版（beta）”（仅限Windows、Xbox和安卓）。两个版本的世界通常可以跨版本访问，但某些重大更新（比如1.18到1.19）可能会直接影响世界生成机制和种子，导致老世界无法体验新内容。这种情况下，建议新开一个世界。对于原版世界，所有连接服务器的客户端必须使用相同的Minecraft版本才能连上。

:::caution 
使用快照/测试版时要小心，世界存档可能会损坏。强烈建议你定期备份世界！
:::

## 自定义和Mod

第一次启动游戏，很多玩家都会先换个皮肤。你可以用在线工具自制皮肤，或者在[Minecraft市场](https://www.minecraft.net/en-us/marketplace)（*基岩版*内置商店）用Minecoins（虚拟货币）买皮肤。*Java*版则有大量免费皮肤网站，也可以自己设计，通过**Minecraft启动器**的`Skins`标签上传，或者用默认皮肤。

*Java*版对mod支持更开放，有多种“启动器”（比如[Forge](https://files.minecraftforge.net/net/minecraftforge/forge/)和[Fabric](https://fabricmc.net/use/installer/)），社区维护着丰富的mod和内容。想深入了解*Java*版mod，看看我们的[安装Mods / Mod包指南](minecraft-forge-fabric-add-mods-modpacks.md)。

*基岩*版则通过内置[商店](https://www.minecraft.net/en-us/catalog)更方便地安装自定义地图、材质包和游戏模式。

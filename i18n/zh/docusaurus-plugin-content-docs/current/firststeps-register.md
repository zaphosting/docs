## 创建账号
创建 ZAP-Hosting 账号是访问和管理所有服务与功能的第一步。注册过程很快，只需填写少量信息。

要开始注册，访问 ZAP-Hosting 官网并进入首页。在那里你会看到登录区域，带有一个 **注册！** 按钮。点击 **注册！** 打开注册弹窗。

![img](https://screensaver01.zap-hosting.com/index.php/s/5b6Wzc27CcHW3Zx/download)


在弹窗中，你需要填写以下信息：

- 用户名
- 邮箱地址
- 密码

填写完信息后，确认安全验证码。这一步是为了防止自动注册。所有字段填写完成并确认验证码后，提交注册表单即可创建账号。

![img](https://screensaver01.zap-hosting.com/index.php/s/k6P3wSKWWRRiYax/preview)

## 账号激活

提交注册表单后，系统会发送一封激活邮件到你填写的邮箱。邮件中包含一个激活码，必须确认该激活码才能完成账号设置。

激活邮件的发送可能需要几分钟。如果邮件没有出现在收件箱，请务必检查垃圾邮件或广告邮件文件夹。

:::warning 未激活功能受限
注册后你可以立即登录，但要使用全部功能和服务，必须确认激活邮件。
:::

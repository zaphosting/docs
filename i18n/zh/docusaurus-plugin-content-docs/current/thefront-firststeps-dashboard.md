## 产品管理

在 **产品管理** 部分，您会找到管理服务器所需的所有页面。下面详细列出了管理服务器的各项选项。



### 控制面板

服务器的 **控制面板** 是管理的核心。这里清晰展示了服务的所有基础且重要的信息。例如，服务器位置、状态、IP:端口、已用内存、当前游戏、服务器名称、地图以及可用槽位数量。此外，还会显示 CPU、内存和 RAM 的使用情况。

![img](https://screensaver01.zap-hosting.com/index.php/s/yLQnHnt86XdpnH2/preview)

在控制面板中，您还可以启动、停止或重启服务，全面掌控服务器运行状态。除此之外，您可以访问实时控制台，获取服务器状态的实时信息，并在必要时直接操作。



### 统计数据

在 **统计数据** 中，您可以全面了解服务器的负载情况。这里包含了可用槽位、CPU 和内存的负载信息。您可以查看不同时间段的数据，比如最近的 **60 分钟、24 小时、3 天、7 天** 或 **30 天**。

![img](https://screensaver01.zap-hosting.com/index.php/s/N8jcmdqqG2Xt4Bk/preview)

这些信息让您对服务器的使用情况有清晰的洞察。此外，统计数据常用于识别潜在的性能问题并缩小排查范围。例如，您可以追踪负载何时上升，是否与某些事件或变更相关。这在突发问题时尤为重要，能帮助您采取针对性措施进行故障排查和系统分析。



### DDoS 管理器


查看过去的 DDoS 攻击记录，更好地评估潜在威胁和攻击模式。开启服务器正在进行的 DDoS 攻击通知，确保事件发生时您能快速响应。

根据服务器所在位置及其可用的 DDoS 保护，您还可以实时监控活跃攻击。这包括详细的流量分析，显示合法流量和被防护系统拦截的流量量。这样的透明度帮助您了解防护效果，及早识别潜在风险。

:::info 缺少 DDoS 管理器访问权限？
使用 DDoS 管理器需要专用 IP 地址或额外购买 “DDoS 管理器概览” 选项。如果您当前套餐未包含这些组件，可以随时通过升级服务添加。
:::

![img](https://screensaver01.zap-hosting.com/index.php/s/ScCCCY52CMLgfyE/preview)



:::warning
仅当您拥有专用 IP 地址或已购买 “DDoS 管理器概览” 额外选项时，才能使用 DDoS 管理器。如果当前套餐缺少这些组件，您可以随时通过升级添加。
:::

### Tebex 商店

通过 TEBEX，您可以将游戏服务器变现，出售虚拟物品和等级，赚取真实货币。它是专为游戏服务器管理员设计的强大且可靠的工具，帮助您为玩家提供更优质的游戏体验。

![img](https://screensaver01.zap-hosting.com/index.php/s/TbwafKd3Kmq4WrS/preview)

TEBEX 使用简单，提供丰富功能来自定义和优化您的商店。您可以自定义商店布局、添加支付方式，甚至发起营销活动，吸引更多玩家。

点击图片即可创建属于您的 TEBEX 游戏服务器商店。设置快速便捷，让您马上开始销售虚拟物品。



### 日志文件

遇到技术问题，不确定原因？首选解决方案之一是查看 **日志文件**。这些文件包含大量服务器信息，如当前事件、提示或错误信息。

尤其是错误信息，能帮助您识别并更好地理解问题根源。通常，通过针对性查看日志文件，可以发现并解决问题。

:::warning
日志信息看不懂或无帮助？别担心！我们每天都提供支持服务。只需创建一个 **[工单](https://zap-hosting.com/en/customer/support/)**，向我们说明您的问题。我们会尽快查看并协助您解决！:)
:::


### 事件日志

**事件日志** 为您提供服务器管理相关活动的全面记录。您可以随时追踪服务器何时启动、停止或崩溃，以及何时执行了更新、备份或新安装。

![img](https://screensaver01.zap-hosting.com/index.php/s/xNzsaMbW5BS5KJC/preview)



### 服务中断

服务器曾短暂离线，原因不明？查看事件日志和日志文件也没头绪？问题可能不在您的服务器本身，而是托管服务器的主机系统。

当我们的监控系统检测到整体中断时，技术人员会自动收到通知，并尽快处理问题。

本节提供您游戏服务器所在主机系统的中断历史概览。更多信息及最新服务状态，请访问我们的新 [状态页面](https://status.zap-hosting.com/)。












## 设置

在 **设置** 部分，您会找到配置服务器所需的所有页面。下面详细列出了可用的配置选项。



### 设置

设置页汇总了服务器最常用的配置选项，并附有简短说明。特别推荐给刚入门的服务器管理员使用。

如果您更有经验，想进行更精准的调整，建议使用配置文件编辑器。相关信息请参见 **配置文件** 分类。



### 配置文件

配置文件编辑器允许您直接在网站上以传统方式编辑配置文件，而非简化视图。此功能让您根据个人喜好自定义服务器设置。

通过编辑器，您可以直接访问所有可用选项，进行精细调整。对高级用户来说，这尤其有用，能完全掌控服务器配置，明确知道要修改哪些内容。

:::warning
发现部分修改被覆盖？请注意，某些选项值来自设置页。如果想直接在配置文件中修改这些内容，需先在设置页禁用相关选项。
:::


### 游戏

在 **游戏** 中，您可以随时更换现有游戏的游戏模式，或安装其他游戏。相同槽位价格的游戏可直接在 **可用游戏** 中选择。其他游戏则需先调整槽位价格。更多信息请参见 [游戏切换](gameserver-gameswitch.md) 指南。

![img](https://screensaver01.zap-hosting.com/index.php/s/xkkECw7o52fAMWk/preview)



### 版本

在 **版本** 中，您可以查看和管理当前游戏服务器版本。随时更新至最新版本，或启用自动更新，确保服务器始终保持最新状态。

![img](https://screensaver01.zap-hosting.com/index.php/s/BH2JzyRHTeLdKHz/preview)

此外，您还可以开启自动更新的邮件通知功能，及时了解更新信息，确保第一时间掌握最新动态。



### 计划任务

创建定时任务，在指定时间自动执行，帮助您自动化服务器上的各种流程。您可以设置任务执行一次或循环执行。支持的任务类型包括启动、停止或重启服务，在线时重启服务器，创建备份或执行自定义命令。

![](https://screensaver01.zap-hosting.com/index.php/s/P6DeWiRC3tDqG2z/preview)

### 重启计划器

想要在特定时间安排服务器重启？重启计划器帮您设置定时自动重启。您可以设定具体时间或周期性重启。

![](https://screensaver01.zap-hosting.com/index.php/s/Y6WciDS7YP98P4m/preview)



## 工具

导航栏的 **工具** 部分包含多个子项，帮助您管理服务器，包括 **FTP 浏览器**、**数据库** 和 **备份**。下面为您详细介绍这些功能。



### FTP 浏览器

通过 **FTP 浏览器**，您可以获取连接 FTP 的所有必要信息。FTP 访问让您直接操作游戏服务器文件。我们推荐使用外部 FTP 客户端，如 **FileZilla** 或 **WinSCP**，以获得最佳体验。如果您不熟悉 FTP 使用，建议查看我们的 [FTP 访问](gameserver-ftpaccess.md) 指南。

![img](https://screensaver01.zap-hosting.com/index.php/s/sX85i9HyzR9wmAM/preview)

**FileZilla：** FileZilla 是最简单且最知名的 FTP 客户端。安装后启动，在顶部栏输入您在本页稍上方找到的登录信息。点击“连接”即可快速登录服务器，管理文件。

**WinSCP：** WinSCP 主要用于 SCP 和 SFTP（FTP 的加密版本），但同样支持普通 FTP。它功能更丰富，适合进阶用户。



### 数据库

除了游戏服务器产品，我们还提供 10 个免费数据库。当需要将数据存储到数据库时非常有用。在 **数据库** 中，您可以创建数据库并查看访问凭证。还可启用备份功能，导入已有数据库备份。

![img](https://screensaver01.zap-hosting.com/index.php/s/WFLwCbZSQfYjrmD/preview)



### 备份

在 **备份** 中，您可以为服务器及其数据库创建手动或自动备份。这样在出现问题或数据丢失时，更容易恢复数据。

您可以设置是否自动备份，以及备份频率（日常或每周）。还可选择在存储空间不足时删除最旧的备份。

![img](https://screensaver01.zap-hosting.com/index.php/s/nm9za5LTMEcNsga/preview)

我们默认为游戏服务器产品提供 10 GB 备份存储空间。如需更多空间，可付费升级。确保您始终有足够空间保存备份，并能访问旧备份。

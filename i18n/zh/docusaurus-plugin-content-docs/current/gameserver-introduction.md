## 配置选项

我们为游戏服务器产品提供多种配置选项，满足各种使用需求。我们的配置通常包含以下可选项：

| 配置项                       | 选择选项 |
| --------------------------------- | ---- |
| 插槽数（Slots）                 | 根据游戏支持的最大玩家数 |
| 内存加速（额外内存）            | 最高可额外添加32 GB内存，叠加默认分配内存 |
| 额外磁盘空间                   | 最高可额外添加200 GB磁盘空间，基础为50 GB |
| CPU & 主机服务器               | 标准（SSD，2.0-3.4 GHz，DDR3/4内存）或高级（M.2 SSD，游戏级CPU 3.4-4.4GHz，DDR4内存） |
| 独立IPv4地址                  | 拥有独立IPv4地址 / 不拥有独立IPv4地址 |

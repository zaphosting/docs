## 点赞 vs 爆发点赞

RedM 点赞和 RedM 爆发点赞都是提升RedM服务器曝光度和人气的方式。RedM 点赞是持续性的投票，适合长期保持服务器在排名中的可见度。而 RedM 爆发点赞则是短期内大量投票，能在短时间内快速提升服务器排名。



## 获取点赞

RedM 点赞可以在[我们的商店](https://zap-hosting.com/en/shop/product/fivem-upvotes/)购买。下单时你可以选择点赞数量、有效期、RedM用户名和支付方式。

:::info 产品名称说明
点赞最初是为 FiveM 设计的，因此至今仍称为 **FiveM 点赞**。不过它们同样适用于 **RedM**。
:::

:::warning 支付流程提醒
RedM 点赞无法使用 ZAP Coins 购买，因为它是外部产品。支付需通过 PayPal 或信用卡等支持的方式完成。
:::

使用 RedM 点赞时，需要绑定你的 RedM 账号。点赞会分配到该账号，你可以在游戏内根据需求分配点赞。

![](https://screensaver01.zap-hosting.com/index.php/s/2fT6CyCfzo4wEe5/download)


确认绑定后即可完成订单。信息会立即传送给 RedM 并进行处理。处理完成后，点赞即可在界面中管理，点赞的有效期也从此时开始计时。



## 分配点赞

点赞一旦在界面显示，就可以通过 RedM 客户端分配。只需在服务器列表中点击你想分配点赞的服务器旁的点赞图标即可。
确认后会弹出窗口，提示点赞已成功分配。

![](https://screensaver01.zap-hosting.com/index.php/s/zD7b9awkCDW7bcx/download)



## 重置点赞

如果想把点赞重新分配给其他服务器，或者分配时出现错误，可以通过控制面板重置点赞。点赞重置后需要几分钟时间才能被 Cfx.re 采纳。

![](https://screensaver01.zap-hosting.com/index.php/s/LjcYptAkZ6dfH8Y/preview)

:::danger 重置冷却时间
Cfx.re 对点赞分配实行**冷却期**限制。每台服务器每天只能分配一次点赞。分配后需等待**24小时**才能再次分配。
:::

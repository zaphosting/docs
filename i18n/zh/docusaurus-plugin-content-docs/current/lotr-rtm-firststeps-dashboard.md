## 产品管理

在**产品管理**部分，你会找到管理服务器所需的所有页面。下面详细列出了管理服务器的各项功能。



### 控制面板

服务器的**控制面板**是管理的核心。这里清晰展示了服务的所有基础和关键信息，比如服务器位置、状态、IP:端口、已用内存、当前游戏、服务器名称、地图以及可用槽位数。此外，还会显示CPU、内存和RAM的使用情况。

![img](https://screensaver01.zap-hosting.com/index.php/s/igcm7GkoB8KmsyD/preview)

在控制面板中，你还可以启动、停止或重启服务器，全面掌控服务器运行状态。同时，你可以访问实时控制台，获取服务器状态的实时信息，并在必要时直接操作。



### DDoS 管理器


查看历史DDoS攻击记录，更好地评估潜在威胁和攻击模式。开启服务器DDoS攻击通知，确保在事件发生时能快速响应。

根据服务器所在位置及可用的DDoS防护，你还可以实时监控正在进行的攻击。这里包含详细的流量分析，显示正常流量和被防护系统拦截的流量量。这样的透明度帮助你了解防护效果，及早识别潜在风险。

:::info 缺少DDoS管理器访问权限？
使用DDoS管理器需要拥有专用IP地址或额外购买“DDoS管理器概览”选项。如果你当前套餐未包含这些组件，可以随时通过升级服务添加。
:::

![](https://screensaver01.zap-hosting.com/index.php/s/ScCCCY52CMLgfyE/preview)



:::warning
只有当你拥有专用IP地址或购买了“DDoS管理器概览”附加选项时，才能使用DDoS管理器。如果当前套餐缺少这些组件，可以随时通过升级添加。
:::





### 日志文件

遇到技术问题却不确定原因？第一步可以查看**日志文件**。这些文件包含大量服务器信息，如当前事件、提示或错误信息。

尤其是错误信息，能帮助你识别并更好地理解问题根源。通常，通过针对性地查看日志文件，可以发现并解决问题。

:::warning
看不懂日志里的信息或没帮助？别担心！我们每天都有客服支持。只需提交一个**[工单](https://zap-hosting.com/en/customer/support/)**，告诉我们你的问题，我们会尽快帮你解决！:)
:::


### 事件日志

**事件日志**为你提供了服务器管理相关的所有活动的全面概览。你可以随时查看服务器何时启动、停止或崩溃，以及何时进行了更新、备份或新安装。

![](https://screensaver01.zap-hosting.com/index.php/s/xNzsaMbW5BS5KJC/preview)



### 实时控制台

**实时控制台**让你实时查看服务器上的所有当前事件。这里可以看到玩家活动信息、错误消息和系统消息。这一视图让你全面了解服务器动态，及早发现潜在问题。

![](https://screensaver01.zap-hosting.com/index.php/s/J9SsmSrPXnG7P9w/preview)

除了监控事件，你还可以在**实时控制台**直接向服务器发送命令。这对于调整设置、执行特定操作或快速应对突发问题非常有用。



### 服务中断

服务器曾短暂离线但你不清楚原因？查看事件日志和日志文件也没发现线索？问题可能不在你的服务器本身，而是托管服务器的主机系统。

如果我们的监控系统检测到整体中断，技术人员会自动收到通知并尽快处理问题。

本节提供了托管你游戏服务器的主机系统的中断历史概览。更多信息和最新服务状态，请访问我们的新[状态页面](https://status.zap-hosting.com/)。



## 设置

在**设置**部分，你会找到配置服务器所需的所有页面。下面详细列出了可用的服务器配置选项。



### 设置

设置页面汇总了服务器最常用的配置选项，并附有简短说明。特别推荐服务器管理新手使用此页面。



### 配置文件

配置文件编辑器允许你直接在网站上以传统方式编辑配置文件，而非简化视图。此功能让你根据个人喜好自定义服务器设置。

通过编辑器，你可以直接访问所有可用选项，精准调整。对高级用户来说，这尤其有用，他们知道自己想改哪些设置。

:::warning
发现部分修改被覆盖？请注意，某些选项值来自设置页面。如果想直接在配置文件中修改这些内容，需先在设置页面禁用相关选项。

想在特定时间安排服务器重启？重启计划功能允许你设置定时自动重启服务器。你可以设定具体时间或定期间隔执行重启。

## 工具

在导航栏的**工具**部分，你会找到一系列子项，帮助你管理服务器，包括**FTP浏览器**、**数据库**和**备份**。下面为你详细介绍这些功能。



### FTP 浏览器

通过**FTP浏览器**，你可以获取连接FTP所需的所有信息。FTP访问让你直接操作游戏服务器文件。我们推荐使用外部FTP客户端，如**FileZilla**或**WinSCP**，以获得最佳体验。如果你不熟悉FTP使用，可以参考[FTP访问指南](gameserver-ftpaccess.md)。

![](https://screensaver01.zap-hosting.com/index.php/s/sX85i9HyzR9wmAM/preview)

**FileZilla：** FileZilla是最简单且最知名的FTP客户端。安装后启动，在顶部栏输入你在本页稍上方找到的登录信息，点击“连接”，即可直接登录服务器，管理文件。

**WinSCP：** WinSCP主要用于SCP和SFTP（FTP的加密版本），但同样支持普通FTP。它功能更丰富，适合进阶用户。



### 数据库

除了游戏服务器产品，我们还提供10个免费数据库。当需要将数据存储到数据库时非常有用。在**数据库**中，你可以创建数据库并查看访问凭证，还能启用备份功能和导入已有数据库备份。

![](https://screensaver01.zap-hosting.com/index.php/s/WFLwCbZSQfYjrmD/preview)



### 备份

在**备份**中，你可以为服务器及其数据库创建手动或自动备份。这样在出现问题或数据丢失时，更容易恢复数据。

你可以设置是否自动备份，以及备份频率（日常或每周）。还可以选择是否在存储空间不足时自动删除最旧的备份。

![](https://screensaver01.zap-hosting.com/index.php/s/nm9za5LTMEcNsga/preview)

我们默认为游戏服务器产品提供10GB备份存储空间。如需更多空间，可付费升级。这样你就能确保备份空间充足，必要时还能访问较旧的备份。

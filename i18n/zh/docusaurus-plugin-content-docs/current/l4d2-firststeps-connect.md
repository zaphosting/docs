## 获取服务器信息

你需要准备好所有连接服务器所需的重要信息。这些信息可以在你的游戏服务器控制面板的仪表盘上找到。重要信息包括：

- IP 地址和端口
- 服务器名称
- 服务器密码（可选）

有了以上信息，你就能找到你的服务器，确保连接到正确的服务器。

## 建立服务器连接

连接游戏服务器通常有一种或多种方式。下面详细介绍并说明可用的方法。

<Tabs>

<TabItem value="connect_solution3" label="控制台（游戏内）">

Left 4 Dead 2 默认关闭了开发者控制台。打开设置，点击“键盘/鼠标”，然后启用控制台。要通过控制台连接服务器，按 `~` 键打开控制台。打开后，执行以下命令：

```
connect IP:Port
```

如果服务器初始化未成功完成，可能导致服务器不可见。这可能是由于配置错误或文件损坏造成的。通常可以在服务器控制台或日志文件中找到更多信息。

## 需求

连接这款游戏的服务器需要额外的软件。FiveM 要求你使用他们的官方客户端。想下载客户端，访问官方 [网站](https://fivem.net/) 的 **FiveM (Cfx.re)**。

![img](https://screensaver01.zap-hosting.com/index.php/s/Li9iCjESkpzdAP4/download)

下载后会得到 `FiveM.exe` 文件，运行它。程序加载完成后，Windows 资源管理器会自动打开，你需要选择 GTA5 游戏和 `GTA5.exe` 文件所在的安装目录。



:::warning 安装问题
遇到安装问题？请确保你的系统满足 [最低要求](https://docs.fivem.net/docs/client-manual/system-requirements/)，并查看 [客户端问题手册](https://docs.fivem.net/docs/support/client-issues/)。
:::



## 获取服务器信息

你需要所有连接服务器所需的相关信息。服务器信息可以在你的游戏服务器控制面板的仪表盘中找到。重要信息包括：

- IP 地址和端口
- 服务器名称

利用以上信息，你就能找到你的服务器，确保连接到正确的服务器。

![img](https://screensaver01.zap-hosting.com/index.php/s/yRMWxYtmWF7Xwxe/preview)

## 建立服务器连接

连接游戏服务器通常有一种或多种方式。下面详细介绍可用的方法。

<Tabs>
    <TabItem value="connect_solution_server_browser_ingame" label="游戏内服务器浏览器" default>

通过游戏内服务器浏览器连接服务器，打开主菜单中的服务器浏览器，点击 **Play** 选项。点击搜索框，输入你的游戏服务器名称。找到服务器后点击它，然后在服务器界面点击 **Connect** 按钮。

![img](https://screensaver01.zap-hosting.com/index.php/s/qzW3DkWTjASeXqG/download)

</TabItem>



<TabItem value="connect_solution3" label="游戏内控制台">

通过控制台连接服务器，按 `F8` 打开控制台。打开后执行以下命令：

```
connect IP:Port
```

![img](https://screensaver01.zap-hosting.com/index.php/s/n864XaxYmG5fSGi/preview)

</TabItem>
</Tabs>



## 可能遇到的问题及解决方案

连接游戏服务器通常有一种或多种方式。下面详细介绍可用的方法。

#### 服务器不可见

如果服务器初始化未成功完成，可能导致服务器不可见。比如配置错误或文件损坏。更多信息通常可以在服务器控制台或日志文件中找到。除此之外，确保服务器列表中没有使用错误的筛选条件，避免服务器被隐藏。

#### 没有解决方案或无效

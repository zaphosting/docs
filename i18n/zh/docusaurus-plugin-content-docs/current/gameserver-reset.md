## 介绍

你的服务器不再像以前那样正常运行，或者你想从头开始重新来过但不知道怎么做？别担心，你可以通过重新安装来重置你的服务器。本指南将告诉你如何操作以及需要注意的事项。

## 重置服务器

重置（重新安装）服务器的选项可以在游戏服务器管理后台的设置页面找到。进入后，你会看到两个按钮：**重置所有选项** 和 **重新安装服务器**，你需要点击这两个按钮才能彻底重置你的服务器。

![img](https://screensaver01.zap-hosting.com/index.php/s/H8Y9Tz8imigEBDZ/download)

:::info
执行重新安装时，你可以选择是否同时重置数据库。如果不想重置数据库，记得取消勾选该选项。
:::

:::danger
如果服务器中还有重要数据，务必先备份，避免数据丢失。
:::

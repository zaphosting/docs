Mods 让你能更自由地定制服务器。下面会告诉你在哪儿能找到最热门的服务器 Mods，以及如何安装它们。

## 准备工作

通过 [Steam 创意工坊](https://steamcommunity.com/app/107410/workshop/)，你可以直接在服务器上安装 Mods。Steam 创意工坊是发现和管理新鲜有趣 Mods 的绝佳平台。先去创意工坊逛逛，找找有没有你感兴趣的 Mods。如果发现喜欢的，一定要记下它的 Workshop ID，通常可以从链接 URL 里找到。这个 Workshop ID 是后续安装必备的。

## 安装步骤

接下来就是安装 Mods 了。得益于 Steam 创意工坊功能，安装过程非常简单。登录你的游戏服务器管理后台，找到 Steam 创意工坊选项。在那里，你可以把之前记下的 Workshop ID 填入指定输入框。如果要添加多个 Mods，记得用逗号分隔开。

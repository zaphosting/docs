## 获取服务器信息

你需要准备好所有连接服务器所需的重要信息。这些信息可以在你的游戏服务器面板的仪表盘上找到。重要信息包括：

- IP 地址和端口
- 服务器名称
- 服务器密码（可选）

凭借以上信息，你就能找到你的服务器，确保连接到正确的服务器。

![img](https://screensaver01.zap-hosting.com/index.php/s/RJb99LD9eMYdGwm/preview)

## 建立服务器连接

连接游戏服务器通常有一种或多种方式。下面详细介绍可用的方法。

<Tabs>
    <TabItem value="connect_solution_server_browser_ingame" label="游戏内服务器浏览器" default>

通过游戏内服务器浏览器连接服务器，先打开游戏。进入游戏后，点击 **多人游戏** 打开服务器列表。在服务器列表右上角的搜索栏中输入你的服务器名称，找到后选择服务器并建立连接。

![img](https://screensaver01.zap-hosting.com/index.php/s/NfpZo6FgFtxDP7x/download)

</TabItem>

<TabItem value="connect_solution_direct_ingame" label="游戏内直接连接">

你也可以不用搜索服务器名称，直接通过 `V` 键打开直接连接界面。输入你的服务器 IP 地址和端口，按 `Enter` 确认，立即连接服务器。

![img](https://screensaver01.zap-hosting.com/index.php/s/Ng6KMdExbkNQfoA/download)

</TabItem>


</Tabs>

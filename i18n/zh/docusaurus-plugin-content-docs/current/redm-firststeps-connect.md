## 需求

连接这款游戏的服务器需要额外的软件。RedM 要求你使用他们的官方客户端。想下载客户端，去 **RedM (Cfx.re)** 的官方[网站](https://redm.net/)吧。

![img](https://screensaver01.zap-hosting.com/index.php/s/7NRnjo2PdgQPKP3/download)

下载后会得到 `RedM.exe` 文件，运行它。程序启动后，Windows 资源管理器会自动打开，你需要选择你电脑上安装了 `Red Dead Redemption 2` 游戏和 `RDR2.exe` 文件的目录。



## 获取服务器信息

你需要所有连接服务器所需的相关信息。这些信息可以在你的游戏服务器控制面板的仪表盘里找到。重要信息包括：

- IP 地址和端口
- 服务器名称

用上面这些信息，你就能找到你的服务器，确保连接到正确的服务器。

![img](https://screensaver01.zap-hosting.com/index.php/s/9wWySCZdiHQt8Sj/preview)

## 建立服务器连接

连接游戏服务器通常有一种或多种方式。下面详细介绍可用的方法。

<Tabs>
    <TabItem value="connect_solution_server_browser_ingame" label="游戏内服务器浏览器" default>

通过游戏内服务器浏览器连接服务器，先在主菜单点击 **Play** 进入服务器浏览器。点击搜索框，输入你的游戏服务器名称。找到服务器后点击它，然后点击服务器界面中的 **Connect** 按钮即可连接。

![img](https://screensaver01.zap-hosting.com/index.php/s/kZqRygE8m9P9HLJ/download)

</TabItem>



<TabItem value="connect_solution3" label="游戏内控制台">

通过控制台连接服务器，按 `F8` 打开控制台。打开后输入以下命令：

```
connect IP:Port
```

![img](https://screensaver01.zap-hosting.com/index.php/s/n864XaxYmG5fSGi/preview)

</TabItem>
</Tabs>



## 可能遇到的问题及解决方案

连接游戏服务器通常有一种或多种方式。下面详细介绍可用的方法。

#### 服务器不可见

如果服务器初始化未成功完成，可能导致服务器不可见。这可能是配置错误或文件损坏导致的。相关信息通常可以在服务器控制台或日志文件中找到。除此之外，确认服务器列表中没有使用错误的筛选条件，避免服务器被隐藏。

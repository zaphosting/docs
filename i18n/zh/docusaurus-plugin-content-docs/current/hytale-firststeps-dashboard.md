## 产品管理

在 **产品管理** 部分，你会找到管理服务器所需的所有页面。下面详细列出了管理服务器的各项选项。

### 控制面板

服务器的 **控制面板** 是管理的核心。这里清晰展示了服务的所有基础且重要的信息，比如服务器位置、状态、IP:端口、已用内存、当前游戏、服务器名称、地图和可用槽位数。此外，还会显示 CPU、内存和 RAM 的使用情况。

![img](https://screensaver01.zap-hosting.com/index.php/s/x9f7brkqDKjsEz5/preview)

在控制面板中，你还可以启动、停止或重启服务，全面掌控服务器运行状态。同时，你可以访问实时控制台，获取服务器状态的实时信息，并在必要时直接操作。

### 统计数据

在 **统计数据** 中，你可以全面查看服务器的负载情况。这里包含了可用槽位、CPU 和内存等多方面的信息。你可以选择不同时间段查看数据，比如最近的 **60 分钟、24 小时、3 天、7 天** 或 **30 天**。

![](https://screensaver01.zap-hosting.com/index.php/s/N8jcmdqqG2Xt4Bk/preview)

这些信息让你深入了解服务器的使用情况。统计数据还能帮助你识别潜在的性能问题并缩小排查范围。例如，你可以追踪负载上升的时间点，判断是否与某些事件或变更有关。这在突发问题时尤其重要，能帮助你有针对性地排查和分析原因。

### DDoS 管理器

查看历史 DDoS 攻击，帮助你更好地评估潜在威胁和攻击模式。开启服务器的 DDoS 攻击通知，确保事件发生时你能快速响应。

根据服务器所在位置及可用的 DDoS 保护，你还可以实时监控正在进行的攻击，包括详细的流量分析，显示合法流量和被防护系统拦截的流量。这样的透明度帮助你了解防护效果，及早发现潜在风险。

:::info DDoS 管理器访问权限缺失？
使用 DDoS 管理器需要专用 IP 地址或额外购买“DDoS 管理器总览”选项。如果当前套餐未包含这些组件，你可以随时通过升级服务添加。
:::

![](https://screensaver01.zap-hosting.com/index.php/s/ScCCCY52CMLgfyE/preview)

:::warning
只有当你拥有专用 IP 地址或已购买“DDoS 管理器总览”选项时，才能使用 DDoS 管理器。如果当前套餐缺少这些组件，可以随时升级添加。
:::

### 日志文件

遇到技术问题却不确定原因？第一步可以查看 **日志文件**。这些文件包含大量服务器信息，如当前事件、提示或错误消息。

尤其是错误信息，能帮助你识别并更好地理解问题根源。通过针对性地查看日志，很多问题都能被发现并解决。

:::warning
看不懂日志里的信息或没帮助？别担心！我们每天都在线客服。只需提交一个 **[工单](https://zap-hosting.com/en/customer/support/)**，告诉我们你的问题，我们会尽快帮你排查解决！:)
:::

### 事件日志

**事件日志** 为你提供服务器管理相关活动的全面记录。你可以随时查看服务器何时启动、停止或崩溃，何时执行了更新、备份或新安装。

![](https://screensaver01.zap-hosting.com/index.php/s/xNzsaMbW5BS5KJC/preview)

### 实时控制台

**实时控制台** 让你实时查看服务器上的所有当前事件。你可以看到玩家活动信息、错误消息和系统提示。这让你全面掌握服务器动态，及早发现潜在问题。

![](https://screensaver01.zap-hosting.com/index.php/s/dktDADkq3bzG3F6/preview)

除了监控事件，你还可以直接在 **实时控制台** 发送命令给服务器，用于调整设置、执行操作或快速响应突发问题。

### 服务中断

服务器曾短暂离线但你不清楚原因？查看事件日志和日志文件也没发现线索？问题可能不在你的服务器，而是托管服务器的主机系统。

当我们的监控系统检测到整体中断时，技术人员会自动收到通知并尽快处理。

本节提供了托管你游戏服务器的主机系统的中断历史记录。更多信息及最新服务状态，请访问我们的新 [状态页面](https://status.zap-hosting.com/)。

## 设置

在 **设置** 部分，你会找到配置服务器所需的所有页面。下面详细列出了可用的配置选项。

### 设置

设置页面汇总了服务器最常用的配置选项，并附有简短说明。对于服务器管理新手，强烈推荐先从这里开始。

如果你更熟悉操作，想做更精准的调整，建议使用配置编辑器。关于这部分的更多信息，请查看 **配置文件** 分类。

### 配置文件

配置文件编辑器允许你直接在网站上以传统方式编辑配置文件，而非简化视图。这个功能让你可以根据个人喜好自定义服务器设置。

通过编辑器，你能直接访问所有可用选项，精准调整。对高级用户来说，这尤其有用，能完全掌控服务器配置，明确知道要改哪些内容。

:::warning
发现部分修改被覆盖？请注意，某些选项值来自设置页面。如果想直接在配置文件中修改这些选项，需先在设置页面关闭对应选项。
:::

### 游戏

在 **游戏** 选项中，你可以随时更换当前游戏版本，或安装其他游戏。相同槽位价格的游戏可直接在 **可用游戏** 中选择。其他游戏则需先调整槽位价格。更多详情请参考 [游戏切换](gameserver-gameswitch.md) 指南。

![](https://screensaver01.zap-hosting.com/index.php/s/xkkECw7o52fAMWk/preview)

### 版本

在 **版本** 页面，你可以查看和管理当前游戏服务器版本。随时更新到最新版本，或开启自动更新，确保服务器始终保持最新。

![](https://screensaver01.zap-hosting.com/index.php/s/BH2JzyRHTeLdKHz/preview)

此外，你还可以启用自动更新的邮件通知，随时掌握更新动态，确保信息及时。

### 计划任务

创建定时任务，自动在指定时间执行，帮你自动化服务器上的各种流程。你可以设置任务执行一次或循环执行。支持的任务类型包括启动、停止或重启服务，在线时重启服务器，创建备份或执行自定义命令。

![](https://screensaver01.zap-hosting.com/index.php/s/P6DeWiRC3tDqG2z/preview)

### 重启计划

想在特定时间安排服务器重启？重启计划功能允许你设置定时自动重启服务器。你可以设定具体时间或周期性重启。

![](https://screensaver01.zap-hosting.com/index.php/s/Y6WciDS7YP98P4m/preview)

### DNS 设置

通过 ZAP DNS 设置，你可以为服务创建易记的地址。你可以使用我们域名的子域名，或者在你自己的域名上创建 DNS 记录，从而拥有更个性化、更易识别的地址。

![](https://screensaver01.zap-hosting.com/index.php/s/tM5FGTHRSbZA5rN/preview)

## 工具

在导航栏的 **工具** 部分，你会找到一系列子项，帮助你管理服务器，包括 **FTP 浏览器**、**数据库** 和 **备份**。下面为你详细介绍这些功能。

### FTP 浏览器

通过 **FTP 浏览器**，你可以获取连接 FTP 的所有必要信息。FTP 访问让你直接操作游戏服务器文件。我们推荐使用外部 FTP 客户端，如 **FileZilla** 或 **WinSCP**，以获得最佳体验。如果你不熟悉 FTP 使用，可以参考 [FTP 访问指南](gameserver-ftpaccess.md)。

![](https://screensaver01.zap-hosting.com/index.php/s/sX85i9HyzR9wmAM/preview)

**FileZilla：** FileZilla 是最简单且最知名的 FTP 客户端。安装后启动，在顶部栏输入你在本页稍上方找到的登录信息，点击“连接”即可登录服务器，管理文件。

**WinSCP：** WinSCP 主要用于 SCP 和 SFTP（FTP 的加密版本），但同样支持普通 FTP。它功能更丰富，适合进阶用户。

### 数据库

除了游戏服务器产品，我们还提供 10 个免费数据库。当需要将数据存储到数据库时非常有用。在 **数据库** 页面，你可以创建数据库、查看访问凭证，还能开启备份功能并导入已有数据库备份。

![](https://screensaver01.zap-hosting.com/index.php/s/WFLwCbZSQfYjrmD/preview)

### 备份

在 **备份** 页面，你可以为服务器及其数据库创建手动或自动备份，方便在出现问题或数据丢失时恢复。

你可以设置自动备份的频率（每日或每周），并决定是否在存储空间不足时自动删除最旧的备份。

![](https://screensaver01.zap-hosting.com/index.php/s/nm9za5LTMEcNsga/preview)

我们默认为游戏服务器产品提供 10 GB 备份存储空间。如需更多空间，可付费升级，确保你有足够空间保存备份，并能访问较旧的备份。

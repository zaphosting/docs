## 介绍
不知道如何连接您的 **Hytale** 服务器或需要准备什么？别担心，我们帮您搞定！本指南将带您了解所需工具和信息，以及实际连接流程和关键注意事项，确保您顺利无忧地连接服务器。跟着我们的步骤，您很快就能连接成功！

## 获取服务器信息
在连接之前，您需要从 ZAP-Hosting 游戏服务器管理界面获取相关服务器信息。对于 Windrose，最重要的信息是**邀请代码**。

您可以在游戏服务器管理的服务器配置文件中找到名为 `InviteCode.txt` 的邀请代码文件。

:::info 邀请代码要求
对于 Windrose，邀请代码是连接服务器的关键。如果使用错误或过期的代码，连接可能会失败。
:::

以下信息在开始前非常有用：

| 信息 | 用途 |
| --- | --- |
| `InviteCode.txt` 内容 | 连接 Windrose 服务器所需的邀请代码 |
| 服务器名称 | 帮助您识别正确的服务器 |
| 可选服务器密码 | 根据配置可能需要输入 |

## 建立服务器连接
拿到邀请代码后，您可以直接通过游戏连接到 Windrose 服务器。

### 在服务器管理中查找邀请代码
打开您的 ZAP-Hosting 游戏服务器管理界面，进入服务器配置区域。找到名为 `InviteCode.txt` 的文件并打开。

文件中包含当前服务器的邀请代码。请准确复制该代码。

:::note 占位符说明
如果本指南中出现类似 `[your_invite_code]` 的值，请用您自己 `InviteCode.txt` 文件中的实际代码替换。
:::

### 在 Windrose 中加入服务器
启动 Windrose，打开游戏的服务器连接或多人游戏菜单。找到允许您通过邀请代码加入服务器的选项。

将 `InviteCode.txt` 中的邀请代码输入到指定字段并确认连接。如果服务器设置了保护，您可能还需要在此步骤输入密码。

| 字段 | 输入内容 |
| --- | --- |
| 邀请代码 | `[your_invite_code]` |
| 密码 | 如果配置了，输入 `[your_server_password]` |

提交邀请代码后，游戏应能成功连接到您的 Windrose 服务器。

:::tip 精准复制代码
为避免连接问题，建议直接从 `InviteCode.txt` 复制粘贴邀请代码。即使是小小的输入错误也会导致连接失败。
:::

## 可能的问题及解决方案
如果连接服务器时遇到问题，可以检查以下常见原因。

### 邀请代码无效
如果邀请代码被拒绝，首先确认您是否完整且准确地复制了 `InviteCode.txt` 的内容。确保代码前后没有多余空格。

邀请代码也可能因服务器重启、更新或配置调整而更改。此时，请重新打开 `InviteCode.txt` 并使用最新代码。

### 服务器尚未准备好
如果服务器仍在启动或未完成初始化，即使邀请代码正确，玩家也可能无法连接。

请检查游戏服务器管理中的服务器控制台和日志，确认服务器已成功启动并完全在线。

:::caution 等待服务器完全启动
请勿在服务器加载或重启过程中尝试连接。即使邀请代码正确，也可能导致连接失败。
:::

### 配置或文件问题
如果服务器依然无法访问，可能存在配置错误或文件损坏，导致服务器无法正常启动。请检查您最近的配置更改并查看日志中的错误信息。

必要时，恢复可用备份或撤销最近的配置更改。

### 无法解决或无帮助
如果在检查邀请代码、服务器状态和配置后仍无法连接，请通过官方 [支持页面](https://zap-hosting.com/en/customer/support/) 联系 ZAP-Hosting 支持团队。

提交请求时，请尽量提供详细信息，例如：

- 您遇到的具体问题
- 服务器是否成功启动
- `InviteCode.txt` 中的邀请代码是否可见
- 控制台或日志中的相关错误信息

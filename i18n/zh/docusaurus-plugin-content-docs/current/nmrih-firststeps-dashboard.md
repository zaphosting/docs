## 产品管理

在 **产品管理** 版块，你会找到管理服务器所需的所有页面。下面详细列出了管理服务器的各项选项。



### 控制面板

你的服务器 **控制面板** 是管理的核心。这里清晰展示了服务的所有基础且重要信息，比如服务器位置、状态、IP:端口、已用内存、当前游戏、服务器名称、地图和可用槽位数。此外，还会显示 CPU、内存和 RAM 的使用情况。

![img](https://screensaver01.zap-hosting.com/index.php/s/kMQ9rgXosqSYr83/preview)

在控制面板中，你还可以启动、停止或重启服务，全面掌控服务器运行状态。同时，你可以访问实时控制台，获取服务器状态的实时信息，并在必要时直接操作。





### 统计数据

在 **统计数据** 里，你能全面查看服务器的负载情况。这里包含了可用槽位、CPU 和内存等多方面的负载信息。你可以选择不同时间段查看数据，比如最近的 **60 分钟、24 小时、3 天、7 天** 或 **30 天**。

![img](https://screensaver01.zap-hosting.com/index.php/s/N8jcmdqqG2Xt4Bk/preview)

这些信息让你对服务器的使用情况有清晰了解。同时，统计数据也有助于发现潜在的性能问题并缩小排查范围。比如，你可以追踪负载何时上升，是否与某些事件或变更有关。这在突发问题时尤其重要，能帮你有针对性地排查并系统分析可能的原因。



### DDoS 管理器


查看历史 DDoS 攻击，更好地评估潜在威胁和攻击模式。开启服务器遭遇 DDoS 攻击时的通知，确保你能迅速响应。

根据服务器所在位置及其可用的 DDoS 保护，你还可以实时监控正在进行的攻击。这里会显示详细的流量信息，包括合法流量和被防护系统拦截的流量。这样的透明度帮助你了解防护效果，及早识别潜在风险。

:::info 缺少 DDoS 管理器访问权限？
使用 DDoS 管理器需要专用 IP 地址或额外购买“DDoS 管理器总览”选项。如果你当前套餐未包含这些组件，可以随时通过升级服务添加。
:::

![img](https://screensaver01.zap-hosting.com/index.php/s/ScCCCY52CMLgfyE/preview)





:::warning
只有当你拥有专用 IP 地址或已购买“DDoS 管理器总览”附加选项时，才能使用 DDoS 管理器。如果当前套餐缺少这些组件，可以随时通过升级添加。
:::



### 日志文件

遇到技术问题却不确定原因？第一步可以查看 **日志文件**。这些文件包含大量服务器信息，比如当前事件、提示或错误消息。

尤其是错误消息，能帮助你定位并更好理解问题根源。通常，针对日志文件的有针对性查看能发现并解决很多问题。

:::warning
日志信息看不懂或没帮助？别担心！我们每天都有客服支持。只需提交一个 **[工单](https://zap-hosting.com/en/customer/support/)**，告诉我们你的问题，我们会尽快帮你排查解决！:)
:::


### 事件日志

**事件日志** 为你提供了服务器管理相关的所有活动总览。你可以随时追踪服务器何时启动、停止或崩溃，以及何时进行了更新、备份或新安装。

![img](https://screensaver01.zap-hosting.com/index.php/s/xNzsaMbW5BS5KJC/preview)



### 实时控制台

**实时控制台** 让你实时查看服务器上的所有当前事件。这里能看到玩家活动信息、错误消息和系统提示。这一视图让你全面掌握服务器动态，及早发现潜在问题。

![img](https://screensaver01.zap-hosting.com/index.php/s/bDqw5G9fLi5mxAg/preview)

除了监控事件，你还可以在 **实时控制台** 直接向服务器发送命令。这对于调整设置、执行特定操作或快速应对突发问题非常有用。



### 服务中断

服务器曾短暂离线但你不清楚原因？查看事件日志和日志文件也没头绪？问题可能不在你的服务器本身，而是托管服务器的主机系统。

如果我们的监控系统检测到整体中断，技术团队会自动收到通知并尽快处理。

本节提供了托管你游戏服务器的主机系统中断历史概览。更多信息和最新服务状态，请访问我们的新 [状态页面](https://status.zap-hosting.com/)。












## 设置

在 **设置** 版块，你会找到配置服务器所需的所有页面。下面详细列出了可用的配置选项。



### 设置

设置页汇总了服务器最常用的配置选项，并附有简短说明。特别推荐给刚入门的服务器管理员使用。

如果你更有经验，想做更精准的调整，建议使用配置文件编辑器。相关信息请查看 **配置文件** 分类。



### 配置文件

配置文件编辑器允许你直接在网站上以经典方式编辑配置文件，而非简化视图。这个功能让你能根据个人喜好定制服务器设置。

通过编辑器，你可以直接访问所有可用选项，精准调整。对高级用户来说，这尤其有用，能完全掌控服务器配置，明确自己想改什么。

:::warning
发现部分修改被覆盖？请注意，某些选项值来自设置页。如果想直接在配置文件中修改，需先在设置页关闭对应选项。
:::

## 工具

导航栏的 **工具** 版块包含多个子项，帮助你管理服务器，包括 **FTP 浏览器**、**数据库** 和 **备份**。下面为你详细介绍这些功能。



### FTP 浏览器

通过 **FTP 浏览器**，你能获取连接 FTP 所需的全部信息。FTP 访问让你直接操作游戏服务器文件。我们推荐使用外部 FTP 软件，如 **Filezilla** 或 **WinSCP**，以获得最佳体验。如果你不熟悉 FTP 使用，建议查看我们的 [FTP 访问](gameserver-ftpaccess.md) 指南。

![img](https://screensaver01.zap-hosting.com/index.php/s/sX85i9HyzR9wmAM/preview)

**FileZilla：** FileZilla 是最简单且最知名的 FTP 客户端。安装启动后，在顶部栏输入你在本页稍上方找到的登录信息，点击“连接”即可直接登录服务器，管理文件。

**WinSCP：** WinSCP 主要用于 SCP 和 SFTP（FTP 的加密版本），但同样支持普通 FTP。它功能更丰富，适合进阶用户。



### 数据库

除了游戏服务器产品，我们还免费提供 10 个数据库。当需要将数据存储到数据库时非常有用。在 **数据库** 里，你可以创建数据库并查看访问凭证，还能开启备份功能和导入已有数据库备份。

![img](https://screensaver01.zap-hosting.com/index.php/s/WFLwCbZSQfYjrmD/preview)



### 备份

在 **备份** 里，你可以为服务器及其数据库创建手动或自动备份。这样在出现问题或数据丢失时，更容易恢复数据。

你可以设置是否自动备份，以及备份频率（日常或每周）。还可以选择是否在空间不足时删除最旧备份。

![img](https://screensaver01.zap-hosting.com/index.php/s/nm9za5LTMEcNsga/preview)

我们默认为游戏服务器产品提供 10 GB 备份存储空间。如需更多空间，可付费升级。这样你就能确保备份空间充足，必要时还能访问旧备份。

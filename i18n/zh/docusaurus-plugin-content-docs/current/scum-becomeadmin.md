## 介绍
赋予管理员权限让你可以轻松全面地管理服务器，拥有完全控制权。作为管理员，你可以直接在游戏内使用游戏提供的所有功能和选项。下面将详细介绍如何为你的服务器分配管理员权限的所有步骤。

<InlineVoucher />



## 配置

管理员权限的配置可以通过你的**游戏服务器设置页面**完成。打开设置，找到**管理员列表（Admin List）**选项。输入你的**SteamID64**，即可将自己设为管理员。如果你想添加更多玩家为管理员，请用**逗号**分隔每个SteamID。

## 管理员命令

下面是一些你作为管理员可以在服务器上使用的常用命令列表。

| 命令               | 说明                                                         | 示例                         |
| ----------------- | ------------------------------------------------------------ | ---------------------------- |
| #LIST             | 显示SCUM所有管理命令的完整列表。                             | #list                       |
| #LISTPLAYERS      | 显示当前服务器上所有玩家的列表。                             | #listplayers                |
| #KICK             | 将指定玩家踢出服务器。                                       | #kick playername            |
| #BAN              | 永久封禁指定玩家。                                           | #ban playername             |
| #LOCATION         | 显示指定玩家的精确位置。                                     | #location playername        |
| #LISTASSETS       | 列出所有游戏内资产，可按关键词筛选。                         | #listassets wood            |
| #SPAWNITEM        | 在玩家面前生成指定物品。                                     | #spawnitem cal_9mm          |
| #SPAWNCHARACTER   | 在玩家面前生成指定NPC、动物或傀儡。                          | #spawncharacter bp_cow2     |
| #TELEPORT         | 传送你到指定坐标。                                           | #teleport 143,1222,1333     |
| #TELEPORTTOPLAYER | 传送你到指定玩家位置。                                       | #teleport playername        |
| #SETFAMEPOINTS    | 设置指定玩家的声望点数。                                     | #setfamepoints 400 playername |

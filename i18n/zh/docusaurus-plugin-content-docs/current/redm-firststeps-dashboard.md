## 产品管理

在 **产品管理** 部分，你会找到管理服务器所需的所有页面。下面详细列出了管理服务器的各种选项。

### 控制面板

你的服务器 **控制面板** 是管理的核心。这里清晰展示了服务的所有基础且关键的信息，比如服务器位置、状态、IP:端口、已用内存、当前游戏、服务器名称、地图以及可用槽位数。此外，还会显示 CPU、内存和 RAM 的使用情况。

![img](https://screensaver01.zap-hosting.com/index.php/s/9wWySCZdiHQt8Sj/preview)

在控制面板中，你还可以启动、停止或重启服务，全面掌控服务器运行状态。同时，你可以访问实时控制台，获取服务器状态的实时信息，并在必要时直接采取操作。

### 统计数据

在 **统计数据** 中，你可以全面了解服务器的负载情况。这里会显示各种指标，比如槽位使用率、CPU 和内存负载。你可以查看不同时间段的数据，比如最近 **60 分钟、24 小时、3 天、7 天** 或 **30 天**。

![](https://screensaver01.zap-hosting.com/index.php/s/N8jcmdqqG2Xt4Bk/preview)

这些信息让你对服务器的使用情况有清晰的洞察。此外，统计数据常用于识别潜在的性能问题并缩小排查范围。比如，你可以追踪负载何时上升，是否与某些事件或变更有关。这在突发问题时尤其重要，能帮助你有针对性地排查并系统分析可能的原因。

### DDoS 管理器

查看历史 DDoS 攻击，帮助你更好地评估潜在威胁和攻击模式。开启服务器正在进行的 DDoS 攻击通知，确保事件发生时你能快速响应。

根据服务器所在位置及可用的 DDoS 保护，你还可以实时监控活跃攻击。这里会显示详细的流量信息，包括合法流量和被防护系统拦截的流量。这样的透明度帮助你了解防护效果，及早发现潜在风险。

:::info DDoS 管理器访问权限缺失？
使用 DDoS 管理器需要专用 IP 地址或额外选项“DDoS 管理器总览”。如果你当前套餐未包含这些组件，可以随时通过升级服务添加。
:::

![](https://screensaver01.zap-hosting.com/index.php/s/ScCCCY52CMLgfyE/preview)

### Tebex 商店

通过 Tebex，你可以将游戏服务器变现，出售虚拟物品和等级，赚取真实货币。它是专为游戏服务器管理员设计的强大且可靠的工具，帮助你为玩家提供更优质的游戏体验。

![](https://screensaver01.zap-hosting.com/index.php/s/TbwafKd3Kmq4WrS/preview)

Tebex 使用简单，功能丰富，支持自定义商店布局、添加支付选项，甚至发起营销活动，吸引更多玩家。

点击图片即可快速创建你的游戏服务器商店。设置快捷简单，让你迅速开始销售虚拟物品。

### 日志文件

遇到技术问题却不确定原因？第一步可以查看 **日志文件**。这些文件包含大量服务器信息，比如当前事件、提示或错误消息。

尤其是错误消息，能帮助你定位并更好地理解问题根源。通常，通过有针对性地查看日志文件，可以发现并解决问题。

:::info
看不懂日志信息或没帮助？别担心！我们每天都有客服支持。只需创建一个 **[工单](https://zap-hosting.com/en/customer/support/)**，告诉我们你的问题，我们会尽快帮你解决！:)
:::

### 事件日志

**事件日志** 提供了服务器管理相关活动的全面记录。你可以随时查看服务器何时启动、停止或崩溃，以及何时进行了更新、备份或新安装。

![](https://screensaver01.zap-hosting.com/index.php/s/xNzsaMbW5BS5KJC/preview)

### 实时控制台

**实时控制台** 让你实时查看服务器上的所有当前事件。这里显示玩家活动信息、错误消息和系统通知。这个视图让你全面了解服务器动态，及早发现潜在问题。

![](https://screensaver01.zap-hosting.com/index.php/s/dktDADkq3bzG3F6/preview)

除了监控事件，你还可以在 **实时控制台** 直接发送命令给服务器，方便调整设置、执行操作或快速响应突发问题。

### 服务中断

服务器曾短暂离线却不知原因？查看事件日志和日志文件也没头绪？问题可能不在你的服务器，而是托管服务器的主机系统。

如果我们的监控系统检测到整体中断，技术团队会自动收到通知并尽快处理。

本节提供了托管你游戏服务器的主机系统中断历史记录。更多信息和最新服务状态请访问我们的新 [状态页面](https://status.zap-hosting.com/)。

## 设置

在 **设置** 部分，你会找到配置服务器所需的所有页面。下面详细列出了可用的配置选项。

### 设置

设置页汇总了服务器最常用的配置选项，并附有简短说明。特别推荐给服务器管理新手使用。

![](https://screensaver01.zap-hosting.com/index.php/s/4X7ERpBn7y9XW4e/preview)

这里你可以调整：

- 基础设置（服务器名、项目名、项目描述等）
- EasyAdmin（语言、菜单按钮、Discord Webhook、举报和管理员命令）
- RedM 设置（坐标显示、PVP）
- 加载界面（YouTube 视频 ID）

如果你更熟悉操作，想做更精准的调整，建议使用配置文件编辑器。相关信息见 **配置文件** 分类。

### 配置文件

配置文件编辑器允许你直接在网站上以传统方式编辑配置文件，而非简化视图。此功能让你根据个人喜好精细定制服务器设置。

使用编辑器，你可以直接访问所有可用选项，精准调整。适合高级用户，想完全掌控服务器配置，且明确自己想改什么。

:::info
发现部分修改被覆盖？请注意，某些选项值来自设置页。如果想直接在配置文件中修改，需先在设置页禁用相关选项。
:::

### 游戏

在 **游戏** 中，你可以随时更换现有游戏的游戏版本，或安装其他游戏。相同槽位价格的游戏可直接在 **可用游戏** 中选择。其他游戏则需先调整槽位价格。详情见 [游戏切换](gameserver-gameswitch.md) 指南。

![](https://screensaver01.zap-hosting.com/index.php/s/k5SKzGYwf6DdpMT/preview)

### 版本

在 **版本** 中，你可以查看和管理当前游戏服务器版本。随时更新到最新版本，或开启自动更新，确保服务器始终保持最新。

![](https://screensaver01.zap-hosting.com/index.php/s/HdY9zcZy3HtiA2C/preview)

此外，还可开启自动更新邮件通知，随时掌握更新动态。

### 资源

本游戏支持通过 **资源** 一键安装知名且流行的资源/插件。概览中展示可用内容，点击安装按钮即可添加。重启服务器后，资源将自动安装、激活并可用。

![](https://screensaver01.zap-hosting.com/index.php/s/bP48gnjNPdSkA9q/preview)

利用此功能，你能快速轻松扩展服务器内容，无需自行调整，节省时间并降低出错或兼容性风险。

### 计划任务

创建定时任务，自动在指定时间运行，帮你自动化服务器上的各种流程。可设置任务执行一次或循环执行。支持的任务类型包括启动、停止或重启服务，在线时重启服务器，创建备份或执行自定义命令。

![](https://screensaver01.zap-hosting.com/index.php/s/P6DeWiRC3tDqG2z/preview)

### 重启计划

想在特定时间安排服务器重启？重启计划功能让你设置定时自动重启。支持指定时间或定期重启。

![](https://screensaver01.zap-hosting.com/index.php/s/Y6WciDS7YP98P4m/preview)

### DNS 设置

通过 ZAP DNS 设置，你可以为服务创建易记地址。可使用我们域名的子域名，或在你自己的域名上创建 DNS 记录，打造更具个人特色的地址。

![](https://screensaver01.zap-hosting.com/index.php/s/tM5FGTHRSbZA5rN/preview)

## 工具

导航栏的 **工具** 部分包含多个子项，帮助你管理服务器，包括 **FTP 浏览器**、**数据库** 和 **备份**。下面为你详细介绍这些内容。

### FTP 浏览器

通过 **FTP 浏览器**，你能获得连接 FTP 所需的全部信息。FTP 访问让你直接操作游戏服务器文件。建议使用外部 FTP 软件，如 **FileZilla** 或 **WinSCP**，以获得最佳体验。如果不熟悉 FTP 使用，建议查看 [FTP 访问指南](gameserver-ftpaccess.md)。

![](https://screensaver01.zap-hosting.com/index.php/s/fZSYpSi4KwPos5P/preview)

**FileZilla：** FileZilla 是最简单且广为人知的 FTP 客户端。安装后启动，输入本页上方提供的访问数据，点击“连接”即可直接登录服务器，管理文件。

**WinSCP：** WinSCP 主要用于 SCP 和 SFTP（FTP 的加密版本），但同样支持 FTP。功能比 FileZilla 更丰富，适合高级用户。

### 数据库

除了游戏服务器产品，我们还提供 10 个免费数据库。数据库用于存储数据等场景。在 **数据库** 中，你可以创建数据库、查看访问凭证，开启备份功能并导入已有数据库备份。

![](https://screensaver01.zap-hosting.com/index.php/s/WFLwCbZSQfYjrmD/preview)

### 备份

在 **备份** 中，你可以为服务器及其数据库创建手动或自动备份，方便在出现问题或数据丢失时恢复。

设置中可指定是否自动备份，以及每日或每周备份频率。还可选择是否在存储空间不足时删除最旧备份。

![](https://screensaver01.zap-hosting.com/index.php/s/nm9za5LTMEcNsga/preview)

默认情况下，我们为游戏服务器产品提供 10 GB 备份存储空间。需要更多空间可付费升级，确保你始终有足够空间存储备份，并能访问旧备份。

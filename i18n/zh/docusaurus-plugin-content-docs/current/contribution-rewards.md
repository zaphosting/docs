在我们的贡献计划中，我们很高兴能为成功贡献的你提供**现金奖励**作为超棒的福利！我们通过 PayPal 或 ZAP-Hosting 账户余额发放奖励。奖励会在你的贡献被 ZAP-Hosting 贡献团队审核通过并准备发布后，作为最后一步发放。

:::info
另一个超赞的福利是你首次贡献后会在我们的 [ZAP-Hosting Discord 服务器](https://discord.com/invite/zaphosting) 获得独家 **@Community Contributor** 身份组，炫耀你对社区的支持，同时还能解锁一些独家内容。想拿到这个身份，提交时记得填写你的 Discord ID 哦。
:::

我们会根据多种标准来计算最终奖励，且不同类型的贡献对应不同的计算方式。下面的章节里，你可以看到针对各类贡献的详细奖励信息。

:::info
所有最终决定（包括奖励金额）均由 ZAP-Hosting 贡献团队全权裁定。
:::

## 贡献类型

:::tip
评估时，我们会先计算一个基础奖励金额，适用于 PayPal 选项。如果作者选择用 ZAP-Hosting 账户余额作为奖励，我们会在基础奖励上额外加 30% 作为奖金。

举个例子：我们评估你的贡献基础奖励为 40 欧元。选择 PayPal 你拿 40 欧元，选择 ZAP-Hosting 账户余额则拿 46 欧元（40 * 1.3）。
:::

## 评估标准

我们会根据以下多项标准来评估最终奖励金额：
- 技术质量：内容是否专业撰写？是否使用了最新且准确的信息？关键词使用是否恰当？
- 话题独特性：网络上是否已有大量类似话题？话题是否独特？
- 详尽程度：内容是否详尽，直接覆盖读者需要知道的所有信息？
- 指南遵守度：作者在多大程度上遵守了我们的撰写指南？
- 审核后修改：为了保证高质量指南，修改量有多大？ZAP 贡献团队介入帮助了多少？

基于这些标准，我们会估算一个合理的奖励金额，力求公平透明。如果你对奖励或工作有疑问，欢迎随时反馈，我们很乐意和你进一步沟通。

## 领取奖励

当你提交并与 ZAP-Hosting 贡献团队确认完成贡献后，我们会评估并通知你奖励金额。指南贡献会通过 GitHub Pull Request 通知，博客贡献则通过你 ZAP-Hosting.com 账户的工单通知。

### ZAP-Hosting 账户余额（ZAP Coins）
选择账户余额奖励时，我们会直接将批准的金额充值到你指定的 ZAP-Hosting.com 账户。

### PayPal
选择 PayPal 付款时，你需要完成一些额外步骤，我们会在准备好后联系你。

:::important
务必在 PayPal 确认工单中同时填写你的 ZAP ID 和工单 ID，否则你的付款会被延迟！
:::

你需要使用 [PayPal 发票创建器](https://www.paypal.com/invoice/create?fromWidget=newuser) 并按以下要求填写：

- 收件人填写我们的公司信息：`ZAP-Hosting GmbH, Hafenweg 8, 48155 Münster, Deutschland`
- 描述中必须包含你的 ZAP ID 和付款确认工单 ID，格式如下：`Contribution Program Payout: Ticket #[yourticket_id], [your_zapid]`
- 发票发送至：`m.kluck@zap-hosting.com`

一旦付款完成，你即同意将你创作的所有独家知识产权转让给 ZAP-Hosting GmbH。此转让在你收到付款后生效，且已在你贡献建议阶段同意的服务条款（TOS）中明确说明。

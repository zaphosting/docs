## 介绍

我们的联盟计划允许任何人通过推荐新客户到 ZAP-Hosting 来赚取佣金。你可以通过访问我们网站上的 [联盟控制面板](https://zap-hosting.com/en/customer/affiliate/) 来使用该功能。

通过联盟控制面板，你可以管理联盟链接、横幅广告，并获取专属的联盟优惠码，新客户使用该码购买服务器时，可享受整个租用期内永久 **20% 折扣**。

## 联盟计划如何运作

你可以在任何你觉得能吸引流量的地方发布和推广所有联盟生成的内容，比如你的网站、论坛或社交媒体账号。完全由你决定。

当有人点击你的链接并在我们[官网](https://zap-hosting.com/)购买产品时，你就有资格获得销售佣金。

:::tip
用户也可以在结账时使用你的联盟优惠码来享受折扣。
:::

## 佣金

通过联盟计划，佣金金额取决于通过你的链接购买的产品类型。某些产品的佣金高达 **50%**。

你可以通过联盟页面的[条款部分](https://zap-hosting.com/en/customer/affiliate/conditions/)查看最新的佣金百分比（%）信息。

## 创建联盟链接和横幅

要创建新的联盟链接或横幅，只需访问主[联盟页面](https://zap-hosting.com/en/customer/affiliate/)并向下滚动。你可以查看已有的链接和横幅，也可以通过绿色的 **创建广告素材** 按钮创建新的。

![](https://screensaver01.zap-hosting.com/index.php/s/zHjcrXACxAoA7qJ/preview)

这会打开一个菜单，你可以在其中自定义多个选项。首先，你需要选择这是一个链接还是横幅。

以下是各项参数的详细说明：

#### 别名

在此字段中，你可以指定用于你的横幅或链接的唯一别名 URL 结尾。别名不能重复，必须是可用的。

例如，填写 `ilovezap`，则你的横幅/链接将通过 `https://zap-hosting.com/ilovezap` 访问。

如果你不指定别名，系统会自动为你生成一个随机别名。

#### 选择模板或链接文本

此字段取决于你选择的广告素材类型。

如果你选择了 **横幅**，你需要从下拉菜单中选择一个模板。我们提供多种默认横幅供你选择，满足不同需求和风格。

如果你选择了 **链接**，你可以指定链接文本，用于生成链接的 HTML 代码。例如，填写 `click this`，生成的 HTML 代码会显示文本 `click me`，并链接到你创建的联盟链接。

示例：`<a href="https://zap-hosting.com/[your_affiliate_link]">click me</a>`

#### 选择目标产品

在这些字段中，你可以选择多种选项，定制你的新联盟链接将跳转到的目标页面。

建议开启 **商店产品**，这样客户点击链接后会直接进入所选产品的服务器配置页面，而不是产品详情页。减少用户跳转页面数，能提高下单概率。

接着，你可以通过下拉菜单选择 **产品**。这是用户访问你的联盟链接时将被重定向的页面。如果你选择的是游戏服务器，还需要额外选择 **游戏**，否则只需选择产品即可。

#### 完成创建流程

填写完所有内容后，点击绿色的 **创建** 按钮完成横幅或链接的创建。如果一切正常，你会看到成功提示，新创建的横幅或链接会显示在页面下方的 **你的横幅/链接** 区域。

![](https://screensaver01.zap-hosting.com/index.php/s/THYSkKPHtSpMgiy/preview)

## 结算与转账

你可以将联盟计划余额转入现金箱（ZAP Coins）或通过 PayPal 提现。再次访问[联盟页面](https://zap-hosting.com/en/customer/affiliate/)，点击蓝色的 **付款/转账** 按钮。

![](https://screensaver01.zap-hosting.com/index.php/s/GnzqQrQtC3jtzt9/preview)

然后根据下面的说明选择转账方式。

### ZAP Coins

:::warning
一旦将余额转入 ZAP Coins，就无法再提现！
:::

在菜单中选择 **ZAP Coin 余额** 选项。选择是转出全部余额还是部分余额。准备好后，点击绿色的 **转账** 按钮确认。

![](https://screensaver01.zap-hosting.com/index.php/s/HyCXmc2KzqSY4yL/preview)

### 现金（PayPal）

我们目前只支持通过 PayPal 提现。要通过 PayPal 提现，你需要创建并发送一张发票给我们，发票中需注明你的 ZAP 客户 ID。

访问 [PayPal 发票创建器](https://www.paypal.com/invoice/create?fromWidget=newuser)，并按要求填写发票：

- 收件人填写我们的公司信息：`ZAP-Hosting GmbH, Hafenweg 8, 48155 Münster, Deutschland`
- 描述必须包含：`ZAP Affiliate Payout [your_zapid]`
- 发票发送至：`m.kluck@zap-hosting.com`

:::note
你的 ZAP ID 可在登录后网站右上角找到，格式如 `zap123456`。
:::

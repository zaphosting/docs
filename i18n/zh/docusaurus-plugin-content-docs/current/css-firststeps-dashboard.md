## 产品管理

在**产品管理**部分，你会找到管理服务器所需的所有页面。下面详细列出了管理服务器的各项选项。

### 控制面板

服务器的**控制面板**是管理的核心。这里清晰展示了关于你的服务的所有基础且重要的信息。例如，服务器位置、状态、IP:端口、已用内存、当前游戏、服务器名称、地图和可用槽位数。此外，还会显示CPU、内存和RAM的使用情况。

![](https://screensaver01.zap-hosting.com/index.php/s/z3tAg8PFiS8N9r3/preview)

在控制面板中，你还可以启动、停止或重启服务，全面掌控服务器运行状态。此外，你可以访问实时控制台，获取服务器状态的实时信息，并在必要时直接采取行动。

### 统计数据

在**统计数据**中，你可以全面了解服务器的负载情况。这里包含了可用槽位、CPU和内存等多方面的负载信息。你可以查看不同时间段的数据，比如最近的**60分钟、24小时、3天、7天**或**30天**。

![](https://screensaver01.zap-hosting.com/index.php/s/N8jcmdqqG2Xt4Bk/preview)

这些信息让你深入了解服务器的使用情况。统计数据还能帮助你识别潜在的性能问题并缩小排查范围。例如，你可以追踪负载何时上升，是否与某些事件或更改有关。这在突发问题时尤为重要，能帮助你有针对性地排查并系统分析可能的原因。

### DDoS 管理器

查看过去的DDoS攻击记录，更好地评估潜在威胁和攻击模式。开启服务器DDoS攻击通知，确保事件发生时你能快速响应。

根据服务器所在位置及可用的DDoS防护，你还可以实时监控正在进行的攻击。包括详细的流量分析，显示合法流量和被防护系统拦截的流量量。这种透明度帮助你了解防护效果，及早识别潜在风险。

:::info 缺少DDoS管理器访问权限？
使用DDoS管理器需要专用IP地址或额外购买“DDoS管理器概览”选项。如果当前套餐未包含这些组件，你可以随时通过升级服务添加。
:::

![](https://screensaver01.zap-hosting.com/index.php/s/ScCCCY52CMLgfyE/preview)

:::warning
只有当你拥有专用IP地址或已购买“DDoS管理器概览”选项时，才能使用DDoS管理器。如果当前套餐缺少这些组件，可以随时升级添加。
:::

### 日志文件

遇到技术问题却不确定原因？第一步可以查看**日志文件**。这些文件包含大量服务器信息，如当前事件、提示或错误消息。

尤其是错误消息，能帮助你识别并更好地理解问题根源。通过有针对性地查看日志文件，往往能发现并解决问题。

:::warning
看不懂日志信息或无从下手？别担心！我们每天都有客服支持。只需创建一个**[工单](https://zap-hosting.com/en/customer/support/)**，告诉我们你的问题，我们会尽快帮你解决！:)
:::

### 事件日志

**事件日志**为你提供了服务器管理相关活动的全面记录。你可以随时追踪服务器何时启动、停止或崩溃，以及何时执行了更新、备份或新安装。

![](https://screensaver01.zap-hosting.com/index.php/s/xNzsaMbW5BS5KJC/preview)

### 实时控制台

**实时控制台**让你实时查看服务器上的所有当前事件。你可以看到玩家活动信息、错误消息和系统提示。这一视图帮助你全面了解服务器动态，及早发现潜在问题。

![](https://screensaver01.zap-hosting.com/index.php/s/J9SsmSrPXnG7P9w/preview)

除了监控事件，你还可以直接在**实时控制台**发送命令给服务器，用于调整设置、执行操作或快速响应突发问题。

### 服务中断

服务器曾短暂离线但你不清楚原因？查看事件日志和日志文件也没发现线索？问题可能不在你的服务器，而是在承载服务器的主机系统。

当我们的监控系统检测到整体中断时，技术人员会自动收到通知并尽快处理。

本节提供了承载你游戏服务器的主机系统的中断历史记录。更多信息和最新服务状态请访问我们的新[状态页面](https://status.zap-hosting.com/)。

## 设置

在**设置**部分，你会找到配置服务器所需的所有页面。下面详细列出了可用的配置选项。

### 设置

设置页面汇总了服务器最常用的配置选项，并附有简短说明。特别推荐服务器管理新手使用此页面。

如果你更熟悉操作，想做更精准的调整，建议使用配置文件编辑器。更多信息请查看**配置文件**分类。

### 配置文件

配置文件编辑器允许你直接在网站上以经典方式编辑配置文件，而非简化视图。此功能让你根据个人喜好自定义服务器设置。

通过编辑器，你能直接访问所有可用选项，精准调整。适合高级用户，想完全掌控服务器设置并清楚自己要改什么。

:::warning
发现部分修改被覆盖？请注意，某些选项值来自设置页面。如果想直接在配置文件中修改，需先在设置页面关闭相关选项。
:::

### 游戏

在**游戏**中，你可以随时更换现有游戏的游戏模式，或安装其他游戏。相同槽位价格的游戏可直接在**可用游戏**中选择。其他游戏则需先调整槽位价格。更多信息请参见[游戏切换](gameserver-gameswitch.md)指南。

![](https://screensaver01.zap-hosting.com/index.php/s/xkkECw7o52fAMWk/preview)

### 版本

在**版本**中，你可以查看和管理当前游戏服务器版本。随时更新到最新版本，或启用自动更新，确保服务器始终保持最新。

![](https://screensaver01.zap-hosting.com/index.php/s/BH2JzyRHTeLdKHz/preview)

你还可以开启自动更新的邮件通知，随时掌握更新动态，确保信息及时。

### 插件

本游戏支持通过**插件**一键安装知名且流行的资源/插件。概览中展示了可用内容，点击安装按钮即可添加。重启服务器后，插件会自动安装、激活并可用。

![](https://screensaver01.zap-hosting.com/index.php/s/QY4xxqFKTXbYXoc/preview)

此功能让你快速轻松扩展游戏服务器内容，无需自行调整，节省时间并降低出错或兼容性风险。

### Steam 创意工坊

想为游戏增添新内容？Steam创意工坊提供丰富的模组和地图。只需输入Steam创意工坊合集ID，重启服务器后即可加载新内容，丰富游戏体验。

![](https://screensaver01.zap-hosting.com/index.php/s/AaL9sQeye5gKzWQ/preview)

### 计划任务

创建定时任务，自动在指定时间运行，帮你自动化服务器上的各种流程。可设置任务执行一次或循环执行。支持的任务类型包括启动、停止或重启服务，在线时重启服务器，创建备份或执行自定义命令。

![](https://screensaver01.zap-hosting.com/index.php/s/P6DeWiRC3tDqG2z/preview)

### 重启计划

想在特定时间安排服务器重启？重启计划功能允许你设置定时自动重启服务器。可设定具体时间或定期间隔重启。

![](https://screensaver01.zap-hosting.com/index.php/s/Y6WciDS7YP98P4m/preview)

### DNS 设置

通过ZAP DNS设置，你可以为服务创建易记的地址。可以使用我们的子域名，也可以在你自己的域名上创建DNS记录，打造更个性化、更易识别的地址。

![](https://screensaver01.zap-hosting.com/index.php/s/tM5FGTHRSbZA5rN/preview)

## 工具

导航栏中的**工具**部分包含多个子项，帮助你管理服务器，包括**FTP浏览器**、**数据库**和**备份**。下面为你详细介绍这些功能。

### FTP 浏览器

通过**FTP浏览器**，你可以获取连接FTP所需的所有信息。FTP访问让你直接操作游戏服务器文件。推荐使用外部FTP客户端，如**FileZilla**或**WinSCP**，以获得最佳体验。如果不熟悉FTP使用，建议查看[FTP访问指南](gameserver-ftpaccess.md)。

![](https://screensaver01.zap-hosting.com/index.php/s/sX85i9HyzR9wmAM/preview)

**FileZilla：** FileZilla是最简单且最知名的FTP客户端。安装后启动，在顶部栏输入本页稍上方提供的登录信息，点击“连接”即可登录服务器，管理文件。

**WinSCP：** WinSCP主要用于SCP和SFTP（FTP的加密版本），但同样支持普通FTP。功能比FileZilla丰富，适合进阶用户。

### 数据库

除了游戏服务器产品，我们还提供10个免费数据库。当需要将数据存储到数据库时非常有用。在**数据库**中，你可以创建数据库并查看访问凭证，还能启用备份功能和导入已有数据库备份。

![](https://screensaver01.zap-hosting.com/index.php/s/WFLwCbZSQfYjrmD/preview)

### 备份

在**备份**中，你可以为服务器及其数据库创建手动或自动备份，方便在出现问题或数据丢失时恢复。

设置中可指定是否自动备份，以及每日或每周备份频率。还可选择在存储空间紧张时自动删除最旧备份。

![](https://screensaver01.zap-hosting.com/index.php/s/nm9za5LTMEcNsga/preview)

我们默认为游戏服务器产品提供10GB备份存储空间。如需更多空间，可付费升级，确保你始终有足够空间保存备份，并能访问旧备份。

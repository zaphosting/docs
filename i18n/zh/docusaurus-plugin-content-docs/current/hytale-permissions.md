## 介绍

在 Hytale 服务器上管理权限可以让你控制玩家能做什么和不能做什么。这包括给信任的玩家、版主和管理员分配不同级别的访问权限。正确设置权限对于维护一个平衡且安全的服务器环境至关重要，让玩家能安心享受游戏，避免意外的漏洞利用或滥用行为。

Hytale 的服务器软件支持分层权限等级，决定每个玩家可以执行哪些命令和操作。你可以通过实时控制台或配置文件来管理这些权限，具体取决于你的服务器设置。

:::

## 权限等级概览

权限定义了玩家在服务器上允许执行的操作。最基础的普通玩家只拥有标准的游戏权限，比如移动、与世界互动和聊天。更高级的权限等级，比如操作员或管理员，可以访问影响游戏玩法、其他玩家、服务器配置和管理工具的服务器命令。

权限系统主要由两个部分组成：

- **用户权限**，直接应用于特定玩家
- **组权限**，允许将权限打包并一次性分配给多个玩家

每个玩家在内部通过 UUID 识别，管理权限时需要用到这个 ID。



## 管理用户权限

用户权限允许你为单个玩家授予或撤销特定权限。

### 查看用户权限

显示直接分配给某个用户的所有权限。此命令会列出指定用户当前拥有的所有权限。

```
/perm user list <uuid>
```



### 给用户添加权限

直接授予一个或多个权限给用户。指定的权限会立即生效，无需重启服务器。

```
/perm user add <uuid> <permissions>
```



### 从用户移除权限

撤销用户的指定权限，仅移除列出的权限，其他权限保持不变。

```
/perm user remove <uuid> <permissions>
```



## 管理用户组分配

除了直接权限，用户还可以通过组继承权限。

### 查看用户所属组权限

显示用户当前所属的权限组。

```
/perm user group list <uuid>
```



### 将用户添加到组

将用户分配到一个权限组。加入后，用户会继承该组定义的所有权限。

```
/perm user group add <uuid> <group>
```



### 从组中移除用户

将用户从权限组中移除。移除后，用户将不再继承该组的权限。

```
/perm user group remove <uuid> <group>
```



## 管理组权限

组权限允许集中管理权限，并在多个用户间复用。

### 查看组权限

显示指定组分配的所有权限，概览该组拥有的所有权限。

```
/perm group list <group>
```



### 给组添加权限

为组添加一个或多个权限。所有分配到该组的用户会立即继承新增权限。

```
/perm group add <group> <permissions>
```



### 从组移除权限

仅移除组中的指定权限，不影响其他权限。

```
/perm group remove <group> <permissions>
```

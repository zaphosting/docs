## 介绍

Teamspeak服务器可以进行个性化配置。接下来我们将为你讲解如何配置你的服务器。我们会说明如何自定义通用设置，比如服务器名称、图标、横幅、用户权限、频道和服务器组权限。

<InlineVoucher />

## 通用设置

**网页界面**

如果你想通过网页界面进行服务器的通用配置，可以在Teamspeak服务器管理后台的“设置”中找到相关选项。设置页面大致如下：

![](https://screensaver01.zap-hosting.com/index.php/s/fLYXn5sx38BBC92/preview)


**Teamspeak客户端**

如果你想直接通过Teamspeak客户端进行服务器的通用配置，可以右键点击最顶层频道 **TeamSpeak ]I[ Server**，然后点击 **编辑虚拟服务器** 来找到相关选项。

![](https://screensaver01.zap-hosting.com/index.php/s/epQ2qzRiex9BmpE/preview)


由于网页界面设置尽量做到易懂，但并非所有内容都能调整，本指南重点讲解如何通过Teamspeak客户端配置服务器。



### 名称

在 **服务器名称** 字段中，你可以定义你的Teamspeak服务器名称。这个名称会显示在服务器列表、网站以及其他地方。



### 横幅

如果你购买了自定义横幅选项，可以在 **横幅图片URL** 字段添加你自己的横幅。请确保提供的是图片的直接链接，否则图片无法识别。在 **URL** 字段中，你可以设置点击横幅时跳转的目标地址。

:::info
如果没有购买自定义横幅，系统会自动用默认的ZAP横幅替代。
:::



### 插槽（保留插槽）

插槽数量由我们设置，无法更改。但你可以设置保留插槽。也就是说，当Teamspeak服务器满员时，拥有相应权限的用户仍然可以连接服务器。



### 欢迎信息

你可以添加自定义的欢迎信息，连接时会显示给用户。示例如下：

![](https://screensaver01.zap-hosting.com/index.php/s/AWFcf4HHJ7jesdc/preview)


### 安全等级

所需的安全等级决定了用户需要多长时间提升其身份的安全等级，直到该身份被认为足够安全。基本上，这能减少垃圾信息和滥用的可能性。安全等级设置越高，所需时间越长。以下是时间概览：

- 等级 0-23 - 几秒内生成
- 等级 23-29 - 几分钟内生成
- 等级 29-34 - 几小时内生成
- 等级 35-39 - 几天内生成
- 等级 40-43 - 几个月内生成
- 等级 44+ - 几年内生成



### 用户管理（踢出、封禁、封禁列表、举报列表）

你可以通过右键点击用户来踢出或封禁。封禁时，身份和IP地址会被记录在封禁列表中。封禁列表可在 **导航栏 -> 额外功能** 中找到，你可以在这里解除已有封禁。

举报列表也在这里。这是Teamspeak的一个功能，允许用户在服务器没有管理员时举报其他用户、服务器本身等。



## 个性化

你可以在Teamspeak服务器上创建更多频道和服务器组，进一步个性化你的服务器。下面我们会详细介绍如何操作。


### 频道

要创建更多频道，你需要在现有频道下方 **右键点击**，然后选择 **创建频道**。会弹出一个窗口，你可以在其中配置频道名称、密码、图标、描述、频道类型等选项。



![](https://screensaver01.zap-hosting.com/index.php/s/Bkx2q69a5ceNiyD/preview)


### 服务器组

要创建额外的服务器组，点击 **导航栏** 中的 **权限**，然后进入 **服务器组**。这里会显示已有服务器组的概览、权限和分配的用户。在服务器组列表底部点击 **加号**，即可创建新组。



![](https://screensaver01.zap-hosting.com/index.php/s/QqcaaRse6kLNwPQ/preview)


## 权限

到这里，你应该已经了解如何调整Teamspeak服务器的通用设置，以及如何通过创建频道和服务器组来个性化你的服务器。接下来，我们讲讲如何调整权限。



### 权限：用户

用户权限取决于其所属的服务器组。默认情况下，用户会被分配到 **普通** 组。如果你想给用户添加更多权限，可以右键点击用户，选择服务器组，然后分配想要的组。

![](https://screensaver01.zap-hosting.com/index.php/s/sXG3qFXXJXc6Kjr/preview)

### 权限：频道

每个频道都可以定义权限。例如，你可以设置哪些用户可以 **进入、订阅、描述、查看、修改、删除** 频道，以及 **访问频道文件浏览器**。



![](https://screensaver01.zap-hosting.com/index.php/s/9sCRd7NgMNHy9Do/preview)



每个字段的权限值取决于服务器组。举个例子，如果你创建了一个名为 **成员** 的服务器组，只有该组成员可以访问频道，那么权限值必须与该服务器组的权限值一致。假设服务器组权限值是50，那么各个字段都要设置为 **50**。



### 权限：服务器组

服务器组权限可以自定义多个类别，包括：

- 通用选项
- 管理组
- 添加、移除用户，管理权限密钥
- 管理用户（移动、踢出、封禁）
- 管理频道
- 基础设置


![](https://screensaver01.zap-hosting.com/index.php/s/RxcYJCsar7C3KnM/preview)



你可以根据需要调整这些权限。如果想进行更深入的权限修改，可以进入Teamspeak客户端的选项，找到“应用程序”，然后启用 **高级权限系统**。

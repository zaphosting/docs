## 获取服务器信息

你需要准备好所有连接服务器所需的重要信息。这些信息可以在你的游戏服务器控制面板的仪表盘中找到。重要信息包括：

- IP 地址和端口
- 服务器名称
- 服务器密码（可选）

凭借以上信息，你就能找到你的服务器，确保连接到正确的服务器。

## 建立服务器连接

连接游戏服务器通常有多种方式。下面详细介绍并说明可用的方法。

<Tabs>
    <TabItem value="connect_solution_server_browser_ingame" label="游戏内服务器浏览器" default>

要通过游戏内服务器浏览器连接服务器，进入主菜单，点击 **查找服务器** 按钮打开服务器浏览器。点击 **添加服务器**，在输入框中填写你的服务器 IP:端口。点击 **将此地址添加到收藏夹**，即可将服务器加入收藏。

![img](https://screensaver01.zap-hosting.com/index.php/s/xmQbcE2k8omcSaS/download)

</TabItem>

<TabItem value="connect_solution_server_browser_steam" label="Steam 服务器浏览器">



要通过 Steam 服务器浏览器连接服务器，打开 Steam，点击顶部导航栏的 `查看`，然后选择 `游戏服务器` 进入服务器浏览器。

在服务器列表中切换到 `收藏夹` 标签，点击底部的加号，弹出窗口中输入你的服务器 IP 地址和端口，添加你的服务器。

![img](https://screensaver01.zap-hosting.com/index.php/s/MMsokw2ZyCreeCN/download)

</TabItem>

<TabItem value="connect_solution3" label="游戏内控制台">

在 Counter-Strike: Source 中，控制台默认是关闭的。要启用它，**进入选项 → 键盘，选择高级**。打开控制台后，按 `~` 键打开控制台。输入以下命令连接服务器：

```
connect IP:Port
```

![img](https://screensaver01.zap-hosting.com/index.php/s/HJR9y7BdiLziABo/preview)

</TabItem>
</Tabs>



## 可能遇到的问题及解决方案

## 产品管理

在 **产品管理** 部分，你会找到管理服务器所需的所有页面。下面详细列出了管理服务器的各项选项。



### 控制面板

服务器的 **控制面板** 是管理的核心。这里清晰展示了关于你的服务的所有基础且重要的信息。例如，服务器位置、状态、IP:端口、已用内存、当前游戏、服务器名称、地图以及可用槽位数。此外，还会显示 CPU、内存和 RAM 的使用情况。

![img](https://screensaver01.zap-hosting.com/index.php/s/PGKR7SPGASZQ3Cc/preview)

在控制面板中，你还可以启动、停止或重启你的服务，全面掌控服务器运行状态。此外，你可以访问实时控制台，获取服务器状态的实时信息，并在必要时直接采取操作。



### 统计数据

在 **统计数据** 中，你可以全面查看服务器的负载情况。这里包含了关于可用槽位、CPU 和内存负载等多方面的信息。你可以选择不同时间段查看数据，比如最近的 **60 分钟、24 小时、3 天、7 天** 或 **30 天**。

![](https://screensaver01.zap-hosting.com/index.php/s/N8jcmdqqG2Xt4Bk/preview)

这些信息让你对服务器的使用情况有清晰的洞察。此外，这些统计数据常常有助于识别潜在的性能问题并缩小排查范围。比如，你可以追踪负载何时上升，以及是否与某些事件或变更相关。这在突发问题时尤其重要，能帮助你有针对性地排查并系统分析可能的原因。



### DDoS 管理器


查看历史 DDoS 攻击，更好地评估潜在威胁和攻击模式。开启服务器正在遭受 DDoS 攻击时的通知，确保你能在事件发生时迅速响应。

根据服务器所在位置及其可用的 DDoS 防护，你还可以实时监控正在进行的攻击。这包括详细的流量分析，显示合法流量和被防护系统拦截的流量量。这样的透明度帮助你了解防护效果，并能及早识别潜在风险。

:::info 缺少 DDoS 管理器访问权限？
使用 DDoS 管理器需要专用 IP 地址或额外选项“DDoS 管理器总览”。如果你当前的套餐没有包含这些组件，可以随时通过升级服务添加。
:::

![](https://screensaver01.zap-hosting.com/index.php/s/ScCCCY52CMLgfyE/preview)



:::warning
只有当你拥有专用 IP 地址或已购买“DDoS 管理器总览”额外选项时，才能使用 DDoS 管理器。如果当前套餐缺少这些组件，可以随时通过升级添加。
:::



### 日志文件

遇到技术问题却不确定原因？第一步可以查看 **日志文件**。这些文件包含了大量服务器信息，比如当前事件、提示或错误信息。

尤其是错误信息，能帮助你识别并更好地理解问题根源。通常，通过有针对性地查看日志文件，可以发现并解决问题。

:::warning
日志信息你看不懂或者没帮助？别担心！我们每天都有客服支持。只需创建一个 **[工单](https://zap-hosting.com/en/customer/support/)**，告诉我们你的问题。我们会尽快帮你排查并解决！:)
:::


### 事件日志

**事件日志** 为你提供了服务器管理相关的所有活动的全面记录。你可以随时追踪服务器何时启动、停止或崩溃，以及何时进行了更新、备份或新安装。

![](https://screensaver01.zap-hosting.com/index.php/s/xNzsaMbW5BS5KJC/preview)



### 实时控制台

**实时控制台** 让你实时查看服务器上的所有当前事件。这里显示玩家活动信息、可能的错误和系统消息。这个视图让你全面了解服务器动态，及早发现潜在问题。

![](https://screensaver01.zap-hosting.com/index.php/s/J9SsmSrPXnG7P9w/preview)

除了监控事件，你还可以在 **实时控制台** 直接向服务器发送命令。这对调整设置、执行特定操作或快速响应突发问题非常有用。



### 服务中断

服务器曾短暂离线但你不清楚原因？查看事件日志和日志文件也没发现线索？问题可能不在你的服务器本身，而是托管服务器的主机系统。

如果我们的监控系统检测到整体中断，技术团队会自动收到通知并尽快处理问题。

本节提供了托管你游戏服务器服务的主机系统的中断历史记录。更多信息和最新服务状态，请访问我们的新 [状态页面](https://status.zap-hosting.com/)。












## 设置

在 **设置** 部分，你会找到配置服务器所需的所有页面。下面详细列出了可用的配置选项。



### 设置

设置页为你提供了服务器最常用的配置选项总览，并附有简短说明。对于服务器管理新手，强烈推荐先使用设置页。

如果你更有经验，想做更精准的调整，建议使用配置文件编辑器。关于这点，你可以在 **配置文件** 分类中找到更多信息。



### 配置文件

配置文件编辑器允许你直接在网站上以经典方式编辑配置文件，而非简化视图。这个选项让你能根据个人喜好自定义服务器设置。

通过编辑器，你可以直接访问所有可用选项，进行精准调整。对高级用户来说，这尤其有用，能完全掌控服务器设置，明确知道要改哪些内容。

:::warning
发现部分修改被覆盖？请注意，某些选项值来自设置页。如果想直接在配置文件中修改这些内容，必须先在设置页禁用相关选项。
:::


### 游戏

在 **游戏** 里，你可以随时更换现有游戏的游戏模式，或者安装其他游戏。相同槽位价格的游戏可以直接在 **可用游戏** 中选择。其他游戏则需先调整槽位价格。更多信息请参考 [游戏切换](gameserver-gameswitch.md) 指南。

![](https://screensaver01.zap-hosting.com/index.php/s/xkkECw7o52fAMWk/preview)



### 版本

在 **版本** 中，你可以查看和管理当前游戏服务器版本。随时更新到最新版本，或开启自动更新，确保服务器始终保持最新状态。

![](https://screensaver01.zap-hosting.com/index.php/s/BH2JzyRHTeLdKHz/preview)

此外，你还可以启用自动更新的邮件通知功能，随时掌握更新动态，确保信息及时。



### 计划任务

创建计划任务，在指定时间自动执行，帮你自动化服务器上的各种流程。你可以设置任务执行一次或循环执行。支持的任务类型包括启动、停止或重启服务，在线时重启服务器，创建备份或执行自定义命令。

![](https://screensaver01.zap-hosting.com/index.php/s/P6DeWiRC3tDqG2z/preview)

### 重启计划器

想在特定时间安排服务器重启？重启计划器帮你设置定时自动重启。你可以设定具体时间或定期间隔执行重启。

![](https://screensaver01.zap-hosting.com/index.php/s/Y6WciDS7YP98P4m/preview)

### DNS 设置
通过 ZAP DNS 设置，你可以为你的服务创建易记的地址。你可以使用我们域名的子域名，或者在你自己的域名上创建 DNS 记录，从而拥有更贴合你个人的地址。
![](https://screensaver01.zap-hosting.com/index.php/s/tM5FGTHRSbZA5rN/preview)

## 工具

导航栏中的 **工具** 部分包含多个子项，帮助你管理服务器，包括 **FTP 浏览器**、**数据库** 和 **备份**。下面为你详细介绍这些功能。



### FTP 浏览器

通过 **FTP 浏览器**，你可以获取连接 FTP 的所有必要信息。FTP 访问让你直接操作游戏服务器文件。我们推荐使用外部 FTP 客户端，如 **FileZilla** 或 **WinSCP**，以获得最佳体验。如果你不熟悉 FTP 使用，可以参考 [FTP 访问](gameserver-ftpaccess.md) 指南。

![](https://screensaver01.zap-hosting.com/index.php/s/sX85i9HyzR9wmAM/preview)

**FileZilla：** FileZilla 是最简单且最知名的 FTP 客户端。安装后启动，在顶部栏输入你在本页稍上方找到的登录信息，点击“连接”，即可直接登录服务器，管理文件。

**WinSCP：** WinSCP 主要用于 SCP 和 SFTP（FTP 的加密版本），但同样支持普通 FTP。它功能比 FileZilla 更丰富，更适合进阶用户。



### 数据库

除了游戏服务器产品，我们还提供 10 个免费数据库。当需要将数据存储到数据库时非常有用。在 **数据库** 中，你可以创建数据库并查看访问凭证。你还可以启用备份选项，导入已有的数据库备份。

![](https://screensaver01.zap-hosting.com/index.php/s/WFLwCbZSQfYjrmD/preview)



### 备份

在 **备份** 中，你可以为服务器及其数据库创建手动或自动备份。这样在出现问题或数据丢失时，更容易恢复数据。

你可以设置是否自动备份，以及备份频率（日常或每周）。还可以选择是否在存储空间不足时删除最旧的备份。

![](https://screensaver01.zap-hosting.com/index.php/s/nm9za5LTMEcNsga/preview)

我们默认为游戏服务器产品提供 10 GB 的备份存储空间。如果需要更多空间，可以付费升级。这样你就能确保有足够空间存储备份，并在需要时访问旧备份。

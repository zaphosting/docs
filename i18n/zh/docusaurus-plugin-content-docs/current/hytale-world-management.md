## 介绍

在 Hytale 服务器上，世界管理就是控制玩家探索、建造和互动的持久游戏环境。一个世界包含所有地形数据、建筑、玩家进度和定义共享体验的设置。能够创建、重命名、重置或整理世界，是维护私服和公服社区的核心功能。

:::info 早期体验提示

Hytale 于 2026 年 1 月 13 日发布，目前处于早期体验阶段。由于游戏仍在积极开发中，服务器软件、配置文件、模组支持和安装流程可能会持续更新。

:::

## 新建世界

创建新世界会给你的服务器一个全新的环境，从零开始生成地形、建筑和游戏进度。通过实时控制台，可以使用相关的世界命令来创建或切换世界。  
当指定一个新世界名时，服务器会加载已有同名世界，或者如果找不到匹配数据，则生成一个新世界。

```
/world create <worldname>
```

该命令会创建一个名为 `<worldname>` 的新世界。如果该名称没有现存世界数据，服务器将生成一个全新的世界环境。



## 设置默认世界

你可以将某个特定世界设置为默认世界，这样服务器启动时会自动加载它。当服务器存在多个世界时，设置默认世界非常实用，可以确保始终使用主要环境。

通过实时控制台，无需手动编辑配置文件即可更新默认世界。设置完成后，服务器下次重启时会优先加载该世界。

```
/world setdefault <worldname>
```

该命令将名为 `mainworld` 的世界设置为默认世界。重启服务器后，该世界会自动加载。



## 删除现有世界

除了创建和加载世界，实时控制台还可以用来删除服务器上的现有世界。当某些世界过时、不再使用或只是测试用时，删除它们可以释放空间。删除世界会永久清除其相关数据，除非你有备份，否则无法恢复。

```
/world remove <worldname>
```

该命令会从服务器删除名为 `worldname` 的世界。如果该世界当前处于激活状态，必须先卸载，否则根据服务器实现，命令可能会被拒绝。

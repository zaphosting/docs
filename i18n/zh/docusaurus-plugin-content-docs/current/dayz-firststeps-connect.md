## 需求

游戏默认不需要额外软件。但如果你想在服务器上使用模组，推荐使用 [DZSA Launcher](https://dayzsalauncher.com/)。



## 获取服务器信息

你需要所有连接服务器所需的相关信息。这些信息可以在你的游戏服务器控制面板的仪表盘上找到。重要信息包括：

- IP 地址和端口
- 服务器名称
- 服务器密码（可选）

凭借以上信息，你应该能找到你的服务器，确保连接到正确的服务器。

![img](https://screensaver01.zap-hosting.com/index.php/s/WPtBDPCB2AQ4bZZ/preview)

## 建立服务器连接

连接游戏服务器通常有一种或多种方式。下面详细介绍可用的方法。

<Tabs>
    <TabItem value="connect_solution_server_browser_ingame" label="游戏内服务器浏览器">

要通过游戏内服务器浏览器连接服务器，打开主菜单中的服务器浏览器。进入服务器列表后，点击 **Community（社区）**。使用筛选选项通过服务器名称搜索你的服务器。

![img](https://screensaver01.zap-hosting.com/index.php/s/Mbb4DFjeEKrZK8M/download)



:::warning DayZ 服务器未显示
游戏内的服务器列表并不总是很可靠。你的服务器可能暂时未显示。这种情况下，请使用其他连接方式。
:::



</TabItem>



<TabItem value="connect_solution_server_browser_expext" label="服务器浏览器（实验版 & 外部）">

如果你使用的是 DayZ 实验版，可以通过 DayZ Launcher 连接服务器。启动启动器，打开 **Servers（服务器）**。点击 **Direct connect（直接连接）** 按钮，输入服务器的 IP 地址和端口。

![img](https://screensaver01.zap-hosting.com/index.php/s/ffaPNzmwcwe428c/download)

</TabItem>

<TabItem value="connect_solution_dzsa" label="DZSA Launcher（外部）">

官方服务器列表不稳定，或者你在用模组？那就用 DZSA Launcher 吧。打开启动器，左侧用筛选器通过服务器名称搜索你的服务器。

:::warning 注册你的服务器
你的服务器默认未在 DZSA Launcher 注册。我们强烈建议你注册，因为很多玩家都用这个服务器列表。点击[这里](https://dayzsalauncher.com/#/servercheck)注册你的服务器。
:::

![img](https://screensaver01.zap-hosting.com/index.php/s/o5GqcZ4EcWondpn/download)

</TabItem>
</Tabs>



## 可能遇到的问题及解决方案

连接游戏服务器通常有一种或多种方式。下面详细介绍可用的方法。

#### 服务器不可见

如果初始化未成功完成，服务器可能无法显示。这可能是配置错误或文件损坏导致。通常可以在服务器控制台或日志文件中找到更多信息。



#### 没有解决方案或无效

排查后仍有问题？没关系，我们的客服团队每天在线帮你解决。只需在我们网站创建一个 **[工单](https://zap-hosting.com/en/customer/support/)**，详细描述你的问题。我们会尽快查看并帮你解决！

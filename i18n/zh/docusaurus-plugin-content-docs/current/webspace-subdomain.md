## 介绍

我们的 Webspace 现在支持快速设置子域名，方便你轻松访问你的站点。在本指南中，你将学习如何使用 Webspace 的 DNS 子域名功能，生成 ZAP Cloud 子域名或设置你自己的域名使用。

![](https://screensaver01.zap-hosting.com/index.php/s/RJTfQf9ARDkC3r4/preview)

<InlineVoucher />

## 可用性

子域名功能对新老客户均开放。新客户在安装过程中会自动设置一个 `xxx`**.zap.cloud** 子域名。老客户则会保留兼容性的传统随机子域名 `[zapid].`**plesk**`xx`**.zap-webspace.com**，但当然也可以创建自己的自定义 ZAP Cloud 域名，后续章节会详细介绍。

:::info
你的 Webspace 还会自动带有一个 `xxx`**.zap-webspace.com** 子域名，用于访问管理面板。
:::

## 子域名管理（DNS）

管理你的 Webspace 子域名非常简单。前往 **设置->DNS 设置** 页面开始操作。

在这里，你可以管理免费 ZAP Cloud 子域名，也可以使用你在 ZAP-Hosting 账户中设置的自有域名。

![](https://screensaver01.zap-hosting.com/index.php/s/QZ4aJJLkKz59g9w/preview)

:::info
目前每个产品最多可激活 3 个子域名（不包括随机生成的 ZAP Cloud 子域名）。
:::

### 随机 ZAP Cloud 子域名

所有 Webspace 默认都会有一个随机子域名。新客户会看到随机的 `xxx`**.zap.cloud** 子域名，老客户则保留传统的 `zapid.`**plesk**`xx`**.zap-webspace.com** 随机子域名。

我们建议老客户如果想要更好的体验，可以创建自定义 ZAP Cloud 子域名，或者你也可以[联系支持团队](https://zap-hosting.com/en/customer/support/)帮你把标准子域名刷新成随机的 ZAP Cloud 子域名。

![](https://screensaver01.zap-hosting.com/index.php/s/XbrGeT5mH98tPEj/preview)

### 自定义 ZAP Cloud 子域名

你可以根据需求创建自己的 ZAP Cloud 子域名，只需输入想用的子域名，并从下拉菜单中选择 **.zap.cloud**（默认选项）。

:::note
如果出现错误，可能是子域名已被占用，或者你输入了禁止使用的词汇或字符，这些限制是为了保护我们的客户。
:::

点击 **创建条目** 按钮后，你应该能在列表中看到新子域名。生效可能需要几分钟时间。

![](https://screensaver01.zap-hosting.com/index.php/s/odqKSyzXRLi5zRx/preview)

![](https://screensaver01.zap-hosting.com/index.php/s/yWmt4j3nWEgbN6K/preview)

### 自有域名

:::info
要使用自己的域名创建子域名，请确保该域名已关联到你的 ZAP-Hosting 账户，可以通过[在我们网站购买](https://zap-hosting.com/en/shop/product/domain/)或从其他提供商转入域名实现。
:::

使用自有域名创建子域名的步骤与自定义 ZAP Cloud 子域名相同。输入想用的子域名，并从下拉菜单中选择你的域名。

:::note
如果出现错误，可能是子域名已被占用，或者你输入了禁止使用的词汇或字符，这些限制是为了保护我们的客户。
:::

点击 **创建条目** 按钮后，你应该能在列表中看到新子域名。生效可能需要几分钟时间。

![](https://screensaver01.zap-hosting.com/index.php/s/Xoe8c4T9TNpby27/preview)

![](https://screensaver01.zap-hosting.com/index.php/s/AX9yFSb3nRNoKiF/preview)

## 故障排查

### ZAP Cloud 子域名

创建自定义 ZAP Cloud 子域名时，如果子域名已被占用或包含禁止词汇/字符，会出现错误。

如果遇到其他 ZAP Cloud 子域名相关问题，请通过[在我们网站提交工单](https://zap-hosting.com/en/customer/support/)联系支持团队。

### 自有域名

如果在创建子域名时看不到自己的域名，通常是因为该域名未关联到你的 ZAP-Hosting 账户。

请确保域名已激活并关联到你的账户，可能需要从其他提供商转入。

如果子域名功能的任何部分仍有问题，请通过[在我们网站提交工单](https://zap-hosting.com/en/customer/support/)联系支持团队，我们会乐意帮你解决。

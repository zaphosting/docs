## 介绍

制作并分享截图能通过提供清晰的视觉信息，帮助提升客服沟通效率。本指南将告诉你如何在电脑上制作截图，并直接上传到客服工单。

## 制作截图

分享截图的最佳流程是先在本地设备上制作截图，然后通过拖拽直接上传到你的客服工单。这样能确保截图隐私安全，不用依赖外部链接分享。



### Windows：截图工具（Snipping Tool）

Windows 自带的*截图工具*可以捕捉指定区域、窗口或整个屏幕，并将图片保存到本地。

1. 在开始菜单搜索并打开 **截图工具**。
2. 点击 **新建** 开始截图。
3. 选择你想捕捉的区域。
4. 将截图保存到电脑上方便的位置。
5. 通过拖拽将保存好的截图文件直接上传到你的客服工单界面。

### Linux：截图工具或 FlameShot

大多数 Linux 发行版自带默认的*截图工具*，或者你也可以用第三方工具 *FlameShot*，功能更强大。

1. 打开截图工具或 FlameShot。
2. 选择要捕捉的区域或窗口。
3. 将截图保存到本地。
4. 通过拖拽将保存好的截图文件直接上传到你的客服工单。

## 上传截图到客服工单

截图保存到本地后，打开 ZAP-Hosting 面板或客服门户中的工单。

- 将截图文件拖拽到工单消息区域。
- 等待上传完成。
- 提交带有截图的工单。

这种方式避免了分享外部链接，保障你的数据安全。

:::info 安全分享截图小贴士
避免公开分享截图或通过第三方上传服务。如果截图包含敏感或隐私信息，上传前务必进行模糊处理或遮盖。
:::

## 制作截图

有时候需要给客服发送截图。  
为了实现这一点，截图必须通过链接分享。  
这个链接可以是你自己的云盘链接，也可以是“网页上传器”，也叫网页截图工具。

如果你的云盘或网页截图工具没有直接截图功能，通常可以用简单的工具在电脑上制作截图。  
Windows 下，直接用“截图工具”（Snipping Tool）就够了。用它可以把截图保存到本地，然后再上传。  
Linux 下，通常自带的“截图”工具或者其他工具比如“FlameShot”都能用来制作本地截图，之后再上传。

下面给你简单介绍两款市场上最受欢迎且免费的截图工具：

:::info
不建议公开分享私人或敏感数据。如果截图中有敏感信息，记得打码或用示例数据替换。
:::

### Imgur

Imgur 提供了一个上传自己文件（图片/视频）的功能。  
上传完成后，会生成一个链接，方便分享给别人。

重点：Imgur 本身没有截图功能，所以截图要先做好，再上传（见上面“制作截图”部分）。

[Imgur官网](https://imgur.com/upload)

上传界面示例：

![](https://screensaver01.zap-hosting.com/index.php/s/4sCEeKkyGEm3EXd/preview)

上传截图后，还能做一些简单编辑。

编辑完后，可以选择发布为社区帖子（公开展示）或者隐藏（不公开）。

完成后，会显示一个可分享的链接：

![](https://screensaver01.zap-hosting.com/index.php/s/dJCpsKtka9rHX6T/preview)

### Lightshot

Lightshot 不支持直接上传图片/截图，上传只能通过它的客户端完成。  
客户端可以直接从官网下载安装，还提供了标记、涂鸦、打码等实用功能。

客户端下载链接：[下载](https://app.prntscr.com/en/)  
注意：没有 Linux 版本。

客户端界面示例：

![](https://screensaver01.zap-hosting.com/index.php/s/iGEccwZZDXqLGKp/preview)

下载安装后，默认按“Print”键就能截图或选取区域。

选好区域后，还能对截图进行编辑。

编辑完成后，点击“云端”按钮即可上传截图。

上传成功后，复制链接分享给别人就行啦。

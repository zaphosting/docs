## 介绍

DDoS（分布式拒绝服务）攻击是恶意尝试通过大量网络流量淹没目标服务器、服务或网络，从而干扰其正常流量。为了确保客户服务不中断，ZAP-Hosting 采用经过验证的 DDoS 防护方案，并根据数据中心所在地进行定制。

我们的 DDoS 防护系统自动且实时运行，在恶意流量影响服务器性能或可用性之前进行过滤。

## DDoS 防护方案

根据各数据中心的地理位置和网络架构，ZAP-Hosting 部署了不同的 DDoS 防护系统。每个站点都配备了专门挑选的缓解方案，以在性能、可靠性和对常见攻击模式的响应速度之间达到最佳平衡。
<details>
  <summary>PletX</summary>

PletX 是一家专注于高性能主机基础设施的德国网络提供商，提供原生网络层过滤，并与专业缓解服务商合作。

**可用地点：** 法兰克福 / 艾格尔斯霍芬，德国

</details>

<details>
  <summary>OVH</summary>

OVH 是欧洲最大的基础设施提供商之一，运营着自己的全球 DDoS 防护系统，配备全天候监控。防护通过分布式清洗网络实现，能在早期过滤流量。

**可用地点：** 伦敦，英国 / 新加坡，新加坡

</details>

| 防护功能                                           | PletX | OVH  |
| -------------------------------------------------- | :---: | :--: |
| 持续在线防护                                       |   ✓   |  ✓   |
| 所有服务的基础防护                                 |   ✓   |  ✓   |
| 网络层和应用层过滤                                 |   ✓   |  ✓   |
| 额外的游戏专用过滤                                 |   ✓   |  ✓   |
| 缓解期间无停机                                     |   ✓   |  ✓   |
| DDoS 管理器中的实时监控可视化                      |   ✓   |  X   |

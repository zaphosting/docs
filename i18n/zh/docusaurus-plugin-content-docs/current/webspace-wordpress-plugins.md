## 介绍

WordPress 本身的标准功能已经非常丰富了，但你还可以通过额外的插件进一步增强和定制它，让平台更贴合你的需求。这些插件带来无限可能，帮你添加新功能、提升性能，打造一个完全符合你目标的独一无二的网站。在本指南中，你将学会如何在你的 Webspace 产品上安装 WordPress 插件。

<InlineVoucher/>

## 安装

首先登录 WordPress 后台，点击侧边栏的 **插件** 菜单。进入后，找到顶部的 **安装插件** 按钮并点击。

页面会跳转到插件总览。在这里，你可以浏览按“精选”、“流行”、“推荐”和“收藏”分类的插件。右侧有个搜索栏，可以用来查找特定插件。

![](https://screensaver01.zap-hosting.com/index.php/s/8yqT94dqFQZGCzp/download)

浏览可用插件，选择你想安装的插件。这里我们以安装热门的 **Contact Form 7** 插件为例。

![](https://screensaver01.zap-hosting.com/index.php/s/s45ag8yptMro4AD/download)

## 激活

安装后的插件默认是未激活状态。你会看到一个 **启用** 按钮，点击它即可激活插件。

![](https://screensaver01.zap-hosting.com/index.php/s/fqAdDb5YGWXZ7sB/download)

## 更新

插件一定要保持最新状态。WordPress 会帮你轻松搞定这件事。当有更新时，WordPress 会直接在插件列表里显示，你只需一键安装即可。小版本更新 WordPress 会自动完成，但对于较大或手动更新，建议先备份你的网站和数据库。

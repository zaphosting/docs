## 介绍

Elementor 是一款强大且易用的 WordPress 页面构建器，让你无需任何编码知识就能轻松打造专业网站。

:::info
Elementor 提供免费和付费版本。免费版（Elementor Core）包含了启动网站所需的所有基础功能。Elementor Pro 在免费版基础上增加了更多功能和专业小部件。想了解具体差异，可以查看官方文档 **[Elementor](https://elementor.com/help/elementor-pro-vs-free/)**。
:::

## 安装

使用和安装 Elementor 之前，你需要先在你的 Webspace 上安装并运行 WordPress。如果还没安装，建议先参考我们的 **[WordPress 安装指南](webspace-wordpress.md)**。

确认 WordPress 已正常运行后，就可以开始安装 Elementor 了。Elementor 是作为 WordPress 插件安装的。关于如何安装 WordPress 插件，可以参考我们的 [**WordPress 插件指南**](webspace-wordpress-plugins.md)。

在 WordPress 后台导航到插件，点击页面顶部的 **添加新插件** 按钮。接着搜索 Elementor，找到 **Elementor Website Builder - More than just a page builder** 并点击安装。

![img](https://screensaver01.zap-hosting.com/index.php/s/xcj9stZtAmY9cgJ/preview)



## 配置

安装完成后，接下来就是配置。你会看到 Elementor 的欢迎窗口，按照提示完成以下步骤：创建 Elementor 账号、安装 “Hello” 主题、选择所需功能，最后点击 “准备就绪”。只需跟着步骤操作，就能完成 Elementor 的完整设置。

![img](https://screensaver01.zap-hosting.com/index.php/s/6QN5trndZgfSano/download)

创建 Elementor 账号不是必须的，除非你打算直接使用 Elementor Pro。之后你可以安装主题并添加所需的额外功能。

![img](https://screensaver01.zap-hosting.com/index.php/s/YgXwPiEnBZTQsC4/preview)

## 设计与布局

### 创建页面

新建页面并用 **Elementor** 自由设计。打开侧边栏的 **Elementor**，选择 **“创建页面”**。编辑器提供丰富的**小部件和配置选项**，让你灵活调整设计和内容。

![img](https://screensaver01.zap-hosting.com/index.php/s/mdMbnXNkngXWJHt/download)

### 模板

创建可复用的**容器、页面或区块模板**。最大优势是只需在一个地方修改，所有使用该模板的地方都会自动更新。节省时间的同时保证网站设计统一。

![img](https://screensaver01.zap-hosting.com/index.php/s/mdMbnXNkngXWJHt/download)

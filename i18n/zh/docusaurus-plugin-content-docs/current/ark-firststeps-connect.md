## 获取服务器信息

你需要准备好所有连接服务器所需的相关信息。服务器信息可以在你的游戏服务器控制面板的仪表盘中找到。重要信息包括：

- IP 地址和端口
- 服务器名称
- 服务器密码（可选）

利用以上信息，你就能找到你的服务器，确保连接到正确的服务器。

## 建立服务器连接

连接游戏服务器通常有多种方式。下面详细介绍了可用的方法。

<Tabs>
    <TabItem value="connect_solution_server_browser_ingame" label="游戏内服务器浏览器" default>

通过游戏内服务器浏览器连接服务器，打开主菜单中的服务器浏览器。点击 **Join Ark**，然后在顶部的 **Name Filter** 输入你的游戏服务器名称。接着选择服务器，点击 **Join** 按钮即可加入。

![img](https://screensaver01.zap-hosting.com/index.php/s/HyDC2cQJ6ejFEcA/download)



</TabItem>

<TabItem value="connect_solution_server_browser_steam" label="Steam 服务器浏览器">



通过 Steam 服务器浏览器连接服务器，打开 Steam 的服务器浏览器。在 Steam 导航栏顶部点击 `View`，然后选择 `Game Server` 版块打开。

使用游戏服务器的 `IP 地址` 和 `端口`，在 Steam 浏览器中搜索或添加你的服务器。操作方法是选择服务器列表中的 `Favorites` 标签页，点击底部的加号。弹出窗口中填写服务器信息即可。

![img](https://screensaver01.zap-hosting.com/index.php/s/MMsokw2ZyCreeCN/download)

</TabItem>

<TabItem value="connect_solution_console" label="控制台">

通过控制台连接服务器，按 `TAB` 键打开控制台。打开后执行以下命令：

```
open IP:Port
```

</TabItem>

<TabItem value="connect_solution_epic" label="控制台（Epic Games）">

通过控制台连接服务器，按 `TAB` 键打开控制台。打开后执行以下命令：

```
open IP:Port
```

:::warning EPIC 启动器端口使用说明 
如果你想通过 EPIC 启动器连接游戏服务器，端口号需要在实际端口基础上加 2（+2）。比如你的端口是 27015，那么需要使用 27017。
:::

:::danger 密码保护服务器兼容性限制 
目前通过 EPIC 启动器无法连接设置了服务器密码保护的 ARK 服务器。
:::
</TabItem>
</Tabs>

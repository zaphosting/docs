## はじめに

**EasyDNSオプション**を使えば、ZAPホスティングのサービスと簡単にリンク（転送）できます。数クリックで、ドメインをゲームサーバーやホスティング製品に接続可能です。

<InlineVoucher />

## アクセス方法と使い方

EasyDNSオプションを利用・有効化するには、まずウェブインターフェースでドメインの管理画面を開きます。そこでDNS管理の項目に以下のオプションが表示されます：

- EasyDNS
- ProfiDNS
- ExpertDNS

EasyDNSを有効にして、ドメインにリンクしたいサービスを選択してください。例として、ホスティングやゲームサーバー製品が選べます。選択後、**保存**をクリックして変更を完了させましょう。

![](https://screensaver01.zap-hosting.com/index.php/s/spg7YHqAJb6Wqky/preview)

スクリーンショットの例では、ドメインがvServer経由でインストールしたGS/TS3インターフェースを使ったゲームサーバーにリンクされています。サブドメイン欄に名前を入力し、サービスを選択してください。

:::warning DNS設定の変更について
DNSの変更はグローバルに反映されるまで最大**24時間**かかる場合があります。その間は転送が利用できませんのでご注意ください。
:::

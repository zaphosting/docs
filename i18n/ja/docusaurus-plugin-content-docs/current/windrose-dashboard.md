## Introduction

**Windrose**ダッシュボードは、ZAP-Hostingのウェブインターフェースでゲームサーバーを管理する中心的なエリアです。このガイドでは、利用可能なダッシュボードのセクションを体系的に紹介し、最も重要な管理、監視、サーバーのレンタルツールの効率的な使い方を学びます。

<InlineVoucher />

## Product Administration

**Product Administration**エリアでは、**Windrose**サーバーの制御と監視に必要な主要ページを見つけられます。これらのページは、現在のサーバー状態の管理、技術的なイベントの確認、重要なサービス情報への迅速なアクセスに役立ちます。

### Dashboard

**ダッシュボード**は、**Windrose**サーバーのメインの概要ページです。ここでは、通常、現在のサーバーステータス、サーバー所在地、接続アドレス、リソース使用状況など、最も重要なサービス情報を一目で確認できます。

![img](https://screensaver01.zap-hosting.com/index.php/s/boTwmEC5HSjbKYK/preview)

現在のサーバー設定やゲーム統合によっては、アクティブなマップ、サーバー名、使用メモリ、プレイヤースロットの使用状況などの追加情報も表示されることがあります。また、サービスの開始、停止、再起動を直接行える場所でもあります。

ダッシュボードは、サーバーの制御と監視を一か所で行えるため特に便利です。これにより、サービスがオンラインかどうかをすばやく確認し、必要に応じて即座に対応できます。

### Statistics

**統計**では、サーバーの過去の使用データを確認できます。これにより、**Windrose**サーバーのサーバーのレンタルパフォーマンスを時間をかけて理解しやすくなります。

![](https://screensaver01.zap-hosting.com/index.php/s/N8jcmdqqG2Xt4Bk/preview)

利用可能なグラフには通常、CPU使用率、メモリ使用量、スロット利用率などの値が含まれます。短期的なピークや長期的な傾向を分析するために、異なる期間を切り替えて表示することも可能です。

| 指標 | 目的 |
| --- | --- |
| CPU使用率 | 処理負荷やパフォーマンスのボトルネックを特定するのに役立ちます |
| メモリ使用量 | サーバーが使用しているRAMの量を示します |
| スロット利用率 | 利用中のプレイヤースロット数を示します |

:::tip パフォーマンス分析
ラグ、クラッシュ、応答遅延が発生した場合、統計ページはトラブルシューティングの最適な出発点の一つです。問題がリソース使用に関連しているか、特定の時間帯だけに発生したかを判断できます。
:::

### DDoS Manager

**DDoSマネージャー**では、サービスに対して検出および緩和されたDDoS攻撃に関する情報を確認できます。異常なトラフィックパターンを理解したり、悪意のあるトラフィックによるサービス中断を調査したりする際に役立ちます。

![](https://screensaver01.zap-hosting.com/index.php/s/ScCCCY52CMLgfyE/preview)

ご契約のオプションや選択したサーバー所在地の保護状況によっては、リアルタイムでの攻撃情報やトラフィック詳細も閲覧可能です。

:::info DDoSマネージャーの利用条件
DDoSマネージャーは、サービスに専用IPアドレスが含まれているか、追加オプションの**DDoS Manager Overview**が有効な場合にのみ利用可能です。現在のプランに含まれていない場合は、サービスのアップグレードが必要です。
:::

### Log Files

**Windrose**サーバーでエラーが発生したり、予期しない動作をしている場合、**ログファイル**セクションは最も重要な確認場所の一つです。ログファイルにはサーバーの技術的な出力が含まれ、警告、起動メッセージ、エラーの詳細が記録されています。

これらのファイルは、クラッシュ、起動失敗、設定問題の原因を特定する際に特に役立ちます。

:::caution ログ確認の注意点
ログ出力が不明瞭で問題の特定に役立たない場合は、公式の[support page](https://zap-hosting.com/en/customer/support/)からサポートに連絡してください。問題の明確な説明と関連するログメッセージを添えるとトラブルシューティングがスムーズになります。
:::

### Event Log

**イベントログ**は、サービスに関連するアクションやイベントの概要を提供します。サーバーの起動、停止、クラッシュ、インストール、アップデート、バックアップ活動などが含まれます。

![](https://screensaver01.zap-hosting.com/index.php/s/xNzsaMbW5BS5KJC/preview)

変更がいつ行われたかや、自動タスクが正常に実行されたかを確認したい場合に特に役立ちます。

### Live Console

**ライブコンソール**は、**Windrose**サーバーが稼働中のリアルタイムのサーバー出力を表示します。現在サーバーで何が起きているかを監視でき、問題を即座に検出するのに役立ちます。

![](https://screensaver01.zap-hosting.com/index.php/s/J9SsmSrPXnG7P9w/preview)

サーバーソフトウェアが対応していれば、ライブコンソールから直接コマンドを送信することも可能です。サービスを再起動せずに管理操作を行いたい場合に便利です。

| 機能 | 説明 |
| --- | --- |
| リアルタイム出力 | 現在のサーバーイベントやメッセージを表示 |
| エラーの可視化 | 発生中の問題をすぐに特定可能 |
| コマンド入力 | 対応するサーバーコマンドを直接送信可能 |

### Service Interruptions

サーバーが一時的に利用できなかった場合で、イベントログやログファイルに明確な原因がない場合は、個別のサービスではなく基盤となるホストシステムに問題がある可能性があります。

**サービス中断**セクションでは、ゲームサーバーのホスト環境に関連する中断履歴を確認できます。より広範なプラットフォームのステータス情報は、公式の[status page](https://status.zap-hosting.com/)もご覧ください。

## Settings

**設定**エリアには、**Windrose**サーバーの主要な構成ページが含まれています。ここでサービスの動作を調整したり、対応していればインストール済みゲームの変更、アップデート管理、自動化機能の設定が可能です。

### Settings

**設定**ページは、一般的なサーバーオプションの簡易的な概要を提供します。ゲームサーバーのサーバーのレンタルが初めての方が、設定ファイルを直接編集せずに変更を加えたい場合に特に便利です。

**Windrose**で利用可能な場合、ゲームプレイに関連する設定やその他の一般的なサービスパラメータが含まれることがあります。具体的なオプションは現在のゲーム統合によって異なります。

### Configs

**Configs**セクションでは、ウェブインターフェースを通じて設定ファイルを直接編集できます。簡易設定ページよりも詳細な制御が可能で、上級ユーザーに適しています。

:::caution 上書きされる値について
一部の値は簡易設定ページで管理されている場合があります。設定エディタで手動変更が上書きされる場合は、同じオプションが簡易設定ページで制御されていないか確認してください。
:::

### Games

**Games**では、サービスが対応していれば別の利用可能なゲームやゲームバリアントに切り替えられます。同じスロット料金のゲームは通常直接選択可能ですが、他のゲームはパッケージの調整が必要な場合があります。

インストール済みゲームの変更方法について詳しくは、[Game Change](gameserver-gameswitch.md)ガイドをご覧ください。

![](https://screensaver01.zap-hosting.com/index.php/s/xkkECw7o52fAMWk/preview)

### Versions

**Versions**ページでは、**Windrose**サービスのインストール済みサーバーバージョンを管理できます（バージョン管理がサポートされている場合）。通常、最新バージョンへのアップデートや自動更新の有効化が可能です。

![](https://screensaver01.zap-hosting.com/index.php/s/BH2JzyRHTeLdKHz/preview)

自動更新はホスティング環境を最新に保つのに役立ちますが、大きなバージョン変更後は互換性を必ず確認してください。

### Scheduled Tasks

**Scheduled Tasks**では、サーバーの定期的な操作を自動化できます。手動管理の手間を減らし、サービスを安定して稼働させたい場合に便利です。

![](https://screensaver01.zap-hosting.com/index.php/s/P6DeWiRC3tDqG2z/preview)

よくある自動化タスク例：

| タスク種類 | 使用例 |
| --- | --- |
| サービス開始 | 指定時間にサーバーを自動起動 |
| サービス停止 | メンテナンス時間帯にサーバーを停止 |
| サービス再起動 | 安定性のため定期的に再起動 |
| バックアップ作成 | 手動操作なしで定期的にバックアップ作成 |
| カスタムコマンド | 対応コマンドを自動実行 |

### Restart Planner

**Restart Planner**は、サーバーの自動再起動をスケジュールするための専用機能です。例えば、利用者が少ない時間帯に**Windrose**サーバーを定期的に再起動したい場合に役立ちます。

![](https://screensaver01.zap-hosting.com/index.php/s/Y6WciDS7YP98P4m/preview)

### DNS Settings

**DNS設定**では、サービスにより覚えやすいアドレスを割り当てられます。これにより、サーバーの共有やアクセスが簡単になります。

![](https://screensaver01.zap-hosting.com/index.php/s/tM5FGTHRSbZA5rN/preview)

ZAP提供のサブドメインか、ご自身のドメインのいずれかを利用できる場合があります。設定や契約オプションによって異なります。

## Tools

**Tools**セクションには、ファイルアクセス、データベース管理、バックアップ処理をサポートする追加のサービス管理機能が含まれています。

### FTP Browser

**FTPブラウザ**では、サーバーファイルにアクセスするための接続情報を確認できます。FTPアクセスは、ファイルのアップロード、ダウンロード、編集が必要な場合に便利です。

![](https://screensaver01.zap-hosting.com/index.php/s/sX85i9HyzR9wmAM/preview)

ファイル管理を簡単にするために、[FileZilla](https://filezilla-project.org/)や[WinSCP](https://winscp.net/)などの外部FTPクライアントの使用を推奨します。接続方法については、[Access via FTP](gameserver-ftpaccess.md)ガイドをご覧ください。

### Databases

**Databases**では、ゲームサーバー製品に含まれるデータベースを作成・管理できます。これは、サービス、MOD、ツールで構造化データの保存が必要な場合に役立ちます。

![](https://screensaver01.zap-hosting.com/index.php/s/WFLwCbZSQfYjrmD/preview)

通常、アクセス資格情報の確認、新規データベースの作成、既存バックアップのインポートが可能です。

### Backups

**バックアップ**セクションでは、サーバーおよび対応していれば関連データベースのバックアップを作成・管理できます。

![](https://screensaver01.zap-hosting.com/index.php/s/nm9za5LTMEcNsga/preview)

定期的なバックアップは、設定ミス、ファイル破損、予期せぬ問題発生時にサービスを復元できるため強く推奨されます。

:::tip バックアップのベストプラクティス
設定ファイルの編集、バージョン切り替え、インストールゲームの変更など大きな変更を行う前に手動バックアップを作成してください。問題発生時の安全な復元ポイントになります。
:::

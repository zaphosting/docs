## Introduction
**Subnautica 2**のゲームサーバーにどうやって接続するか、また必要な情報がわからない場合に役立つガイドです。ここでは必要なサーバー情報の取得方法、接続手順、よくある接続トラブルの対処法を説明します。

:::info 早期アクセスに関する注意
Subnautica 2は現在のリリース段階によってはまだ開発中の可能性があります。そのため、マルチプレイヤー機能やサーバーブラウザの挙動、接続方法が今後変更されることがあります。
:::

## サーバー情報の取得

サーバーに接続するために必要な情報をすべて用意してください。これらの情報はゲームサーバーのコントロールパネル内のダッシュボードで確認できます。重要な情報は以下の通りです。

- IPアドレスとポート
- サーバー名
- サーバーパスワード

上記の情報を使って、正しいサーバーを特定し、確実に接続できるようにしましょう。

| 項目 | 説明 |
| --- | --- |
| `IPアドレス` | Subnautica 2サーバーの公開アドレス |
| `ポート` | 接続に使用するゲームポート |
| `サーバー名` | サーバーリストやセッションブラウザで表示される名前 |
| `サーバーパスワード` | サーバーがパスワード保護されている場合に必要 |

## サーバーへの接続方法

**Subnautica 2**のサーバーへの接続方法は、使用しているゲームバージョンのマルチプレイヤー機能によって異なります。基本的にはサーバーの**IPアドレス**と**ポート**、場合によっては**サーバーパスワード**が必要です。

### ゲーム内マルチプレイヤーメニューから参加する

Subnautica 2を起動し、メインメニューからマルチプレイヤーまたはサーバー参加セクションを開きます。ゲームに直接接続オプションがあれば、ZAP-Hostingのコントロールパネルに表示されている`IPアドレス`と`ポート`を正確に入力してください。パスワードが設定されている場合は、接続時に`サーバーパスワード`を入力します。

情報を確認して確定すると、ゲームがサーバーへの接続を試みます。

:::note 直接接続オプションの有無について
メニューやボタンの表記はゲームのバージョンによって異なる場合があります。直接接続オプションが見当たらない場合は、サーバーリストや招待システム、セッションブラウザを使って参加する必要があるか確認してください。
:::

### サーバーリストやセッションブラウザから参加する

Subnautica 2が手動の直接接続欄ではなく、内蔵のサーバーブラウザを使う場合は、設定した`サーバー名`でサーバーを検索してください。接続前にコントロールパネルでサーバー名を確認しておくことが重要です。

リストから正しいサーバーを見つけたら：

1. サーバーを選択
2. 表示されている名前が自分のサーバーと一致しているか確認
3. サーバーに参加
4. パスワード入力を求められたら入力

:::tip ユニークなサーバー名を使おう
リストでサーバーが見つけにくい場合は、設定でわかりやすくユニークなサーバー名を付けましょう。これにより正しいサーバーを簡単に見つけられ、間違ったサーバーに接続するリスクを減らせます。
:::

### サーバーがすぐに見つからない場合

ゲーム内ブラウザにサーバーが表示されなくても、必ずしもサーバーがオフラインとは限りません。サーバーリストの更新に時間がかかる場合や、ゲームがサーバーの初期化完了後にのみ表示することもあります。

以下を確認してください：

- ZAP-Hostingのコントロールパネルでサーバーがオンライン状態である
- 正しいゲームポートを使用している
- サーバーが完全に起動している
- サーバー名が正しく設定されている
- パスワードが必要なら正しく入力している

## よくある問題と対処法

ゲームサーバーへの接続は様々な要因で影響を受けることがあります。よくある問題とその解決策を以下にまとめました。

### サーバーが見えない

初期化が正常に完了していない場合、サーバーが見えなくなることがあります。これは設定ミスやファイルの破損などが原因の場合があります。詳細はサーバーコンソールやログファイルで確認できます。

:::caution サーバー起動完了が必須
サーバーの起動処理が完了する前に接続しようとしないでください。サーバーがまだ読み込み中や再起動中の場合、プレイヤーは接続できなかったり、リストに表示されないことがあります。
:::

### 接続情報の誤り

接続できない場合は、`IPアドレス`、`ポート`、`サーバーパスワード`が正しいか必ず確認してください。入力ミスがあると接続は失敗します。

| 確認項目 | チェック内容 |
| --- | --- |
| `IPアドレス` | サーバーダッシュボードに表示されているアドレスと一致しているか |
| `ポート` | 割り当てられたゲームポートと一致しているか |
| `パスワード` | 大文字・小文字も含めて正確に入力されているか |
| `サーバー名` | ブラウザで検索する場合は設定した名前と一致しているか |

### サーバーはオンラインだが接続できない

サーバーがオンライン表示でも接続できない場合は、コントロールパネルのサーバーコンソールやログを確認してください。読み込み失敗や設定ミスで接続がブロックされている可能性があります。

:::tip まずはログをチェック
接続失敗の原因を特定するには、サーバーコンソールやログファイルが最も早い手段です。サーバーが正しく待機しているか、接続試行が届いているかを確認できます。
:::

### 解決策が見つからない場合

トラブルシューティングを行っても問題が解決しない場合は、サポートチームが毎日対応しています。ZAP-Hostingのウェブサイトから**[チケット](https://zap-hosting.com/en/customer/support/)**を作成し、サーバーのIP、ポート、接続時の状況を詳しく説明してください。

## はじめに

アクティビティフィードは、FiveMゲームサーバー内でプレイヤーに直接情報を届けるのに役立ちます。Mastodonアカウントを連携することで、サーバーのアップデートやお知らせ、短いメッセージをリアルタイムで表示でき、プレイヤーが外部プラットフォームをチェックする手間を省けます。

この連携により、サーバーとコミュニティ間の明確なコミュニケーションチャネルが生まれ、重要な情報が常に一か所で見やすくなります。

## 準備

アクティビティフィードを設定する前に、Mastodonアカウントが必要です。このアカウントから投稿された更新情報が後でサーバーに表示されます。

まだアカウントを持っていない場合は、お好きな[Mastodon](https://joinmastodon.org/)インスタンスで登録してください。登録後はプロフィールが公開されていて投稿できる状態であることを確認しましょう。これらの投稿がアクティビティフィードに表示されます。



## 設定方法

Mastodonのプロフィールを開き、フルユーザー名をコピーします。  
ユーザー名は通常、以下の形式です：

```
username@instance.domain
```

例として、ユーザー名は `zaphosting@mstdn.instance` です。次に、FiveMゲームサーバーの**Configs**セクションを開き、`server.cfg`ファイルを編集します。ファイルの一番下までスクロールして、以下の行を追加してください：

```
sets activitypubFeed username
```

`username` の部分は、ご自身のMastodonユーザー名に置き換えます。

![](https://screensaver01.zap-hosting.com/index.php/s/oQWC9pxrweM8FsY/preview)

ファイルを保存してサーバーを再起動しましょう。再起動後、アクティビティフィードが利用可能になるまで最大20分かかることがあります。利用可能になると、サーバーを開いたときに新しい**Feed**タブが表示され、連携したMastodonアカウントの投稿が見られます。



## まとめ

アクティビティフィードを設定することで、サーバーのアップデートやお知らせがプレイヤーに直接、わかりやすく表示されます。この設定は透明性を高め、コミュニティの交流を強化し、外部リンクやプラットフォームに頼らずにニュースや情報を効率的に共有する方法を提供します。

## はじめに

Hytaleのアーリーアクセス版は、多彩なアイテムやクラフトパス、ツール、武器、ユーティリティで溢れる豊かな世界を提供しています。新規プレイヤーもベテランも、膨大なアイテム数とその隠れた特性に圧倒されがち。特にゲーム序盤では、基本ゲームに詳細なアイテムデータがまだ用意されていないため、理解が難しいことも多いです。

ゲーム自体にはまだツールチップやインベントリ画面での包括的なアイテムメタデータがなく、アイテムの特徴や用途、必要条件を外部情報なしで把握するのは大変です。

そこで、**Advanced Item Info**モッドは、ゲーム内アイテムの**追加コンテキスト情報**を検索可能なインターフェースで提供し、プレイヤーが詳細データを直接確認できるようにしてくれます。もう勘に頼る必要なし！

## Advanced Item Infoでできること

Advanced Item Infoは専用のGUIを追加し、ゲーム内のアイテムに関する豊富な情報を表示します。具体的には：

- アイテム識別とユニークID  
- ビジュアルアイコンと品質ランク  
- レベル要件とアイテムレベル  
- 最大スタック数と耐久度  
- 燃料として使えるかどうか  
- 消費アイテムか特殊効果があるか  
- ツール、武器、防具、グライダー、ユーティリティ部品としての機能  
- クラフト可能場所や入手方法のツールチップ情報


## モッドの使い方

HyFixesのインストールは、[Install Mods](hytale-mods)ガイドにある標準的なモッド導入手順に従ってください。このガイドではモッドの配置場所やサーバーでの読み込み方法を解説しています。インストール後は、ゲーム内で以下のコマンドを入力するとUIが開きます：

```
/advancedinfo
```

GUIが開いたら、アイテム名を入力またはペーストして検索し、結果をクリックまたはホバーしてモッドが提供する詳細メタデータをチェックできます。

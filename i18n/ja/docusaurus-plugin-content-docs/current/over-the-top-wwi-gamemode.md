## はじめに

**Over the Top WWI** のゲームモードは、試合の進行やプレイヤーが達成すべき目標を決定します。ゲームモードが変わることで、ゲームのテンポや戦略、全体的なプレイ体験が大きく変わります。

ゲームモードを調整することで、大規模戦闘に特化したり、防御重視のプレイや目標達成型のシナリオに合わせてサーバーをカスタマイズできます。コミュニティの好みに合わせて最適な設定を選びましょう。

## 設定方法

ゲームモードはサーバーの設定ファイル内で変更します。**ゲームサーバー管理パネル**の**Configs**から設定ファイルにアクセスし、`ServerConfiguration.ini` を開いてください。ファイル内の以下のパラメータを探します：

```
CurrentGameMode = 2
```

この値がサーバーで有効なゲームモードを示します。設定可能な値は以下の通りです：

- `2` → コンケスト（Conquest）
- `3` → シージ（Siege）

選択した値が各試合の基本ルールを決定します。

- **コンケスト (2)** はマップ上の目標を奪い合い、維持することに重点を置き、常に動き回りチーム連携が求められます。
- **シージ (3)** は防御寄りのモードで、一方のチームが攻撃し、もう一方が戦略的な拠点を守ります。

どのモードを選ぶかは、好みのプレイスタイルやプレイヤー数に合わせて決めましょう。

`ServerConfiguration.ini` の値を変更したら、ファイルを保存してサーバーを再起動してください。次の試合やマップローテーションから新しいゲームモードが適用されます。

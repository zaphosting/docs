## Introduction
**Hytale**サーバーへの接続方法や必要な準備がわからないですか？心配いりません！必要なツールや情報から実際の接続手順、スムーズで問題のない接続を実現するためのポイントまで、すべてご案内します。このガイドに従えば、すぐに接続できます！

## Obtain server details
接続する前に、ZAP-Hostingのゲームサーバーのウェブインターフェースから必要なサーバー情報を取得してください。Windroseの場合、最も重要なのは**招待コード**です。

招待コードはゲームサーバー管理のサーバー設定ファイル内の`InviteCode.txt`にあります。

:::info 招待コードの重要性
Windroseでは、招待コードがサーバー接続のためのキーとなります。間違ったコードや古いコードを使うと接続に失敗する可能性があります。
:::

開始前に役立つ情報は以下の通りです：

| 情報 | 用途 |
| --- | --- |
| `InviteCode.txt`の内容 | Windroseサーバーに接続するために必要 |
| サーバー名 | 正しいサーバーを識別するのに役立つ |
| 任意のサーバーパスワード | 設定によっては必要になる場合あり |

## Establish server connection
招待コードを入手したら、ゲーム内から直接Windroseサーバーに接続できます。

### サーバー管理で招待コードを確認する
ZAP-Hostingのゲームサーバーウェブインターフェースを開き、サーバー設定エリアに移動します。`InviteCode.txt`というファイルを見つけて開いてください。

このファイル内に現在の招待コードが記載されています。表示されている通りに正確にコピーしてください。

:::note プレースホルダーの説明
このガイド内で`[your_invite_code]`のような値が出てきたら、ご自身の`InviteCode.txt`ファイルにある実際のコードに置き換えてください。
:::

### Windroseでサーバーに参加する
Windroseを起動し、ゲームのサーバー接続またはマルチプレイヤーメニューを開きます。招待コードを使ってサーバーに参加するオプションを探してください。

`InviteCode.txt`の招待コードを入力欄に入れて接続を確定します。サーバーが保護されている場合は、この段階でパスワードの入力も求められることがあります。

| 項目 | 入力内容 |
| --- | --- |
| 招待コード | `[your_invite_code]` |
| パスワード | 設定している場合は`[your_server_password]` |

招待コードを送信すると、ゲームがWindroseサーバーへの接続を開始します。

:::tip コードは正確にコピー
接続トラブルを避けるため、可能な限り`InviteCode.txt`から招待コードをコピー＆ペーストしてください。小さなタイプミスでも接続失敗の原因になります。
:::

## Potential issues and solutions
接続がすぐにできない場合は、以下のよくある原因を確認してください。

### 招待コードが使えない
招待コードが拒否される場合は、まず`InviteCode.txt`の内容を正確にコピーしたか確認してください。コードの前後に余分なスペースがないかもチェックしましょう。

サーバーの再起動やアップデート、設定変更後に招待コードが変わっている可能性もあります。その場合は`InviteCode.txt`を再度開いて最新のコードを使ってください。

### サーバーがまだ準備中
サーバーが起動中または初期化が完了していない場合、正しい招待コードでも接続できないことがあります。

ゲームサーバー管理のコンソールやログで、サーバーが正常に起動し完全にオンラインになっているか確認してください。

:::caution 完全起動を待つこと
サーバーが読み込み中や再起動中に接続を試みないでください。招待コードが正しくても接続失敗する原因になります。
:::

### 設定やファイルの問題
サーバーに接続できない場合、設定ミスやファイル破損で正常に起動できていない可能性があります。最近の変更を見直し、ログにエラーがないか確認してください。

必要に応じて正常に動作していたバックアップから復元するか、設定変更を元に戻してください。

### 解決しない場合
招待コード、サーバーステータス、設定を確認しても接続できない場合は、公式の[サポートページ](https://zap-hosting.com/en/customer/support/)からZAP-Hostingサポートチームにお問い合わせください。

問い合わせ時には以下の情報をできるだけ詳しく伝えてください：

- 発生している具体的な問題
- サーバーが正常に起動しているかどうか
- `InviteCode.txt`の招待コードが見えるかどうか
- コンソールやログに表示される関連エラーメッセージ

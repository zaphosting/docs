## はじめに

スクリーンショットを作成・共有することで、サポートのやり取りが視覚的にわかりやすくなり、問題解決がスピーディーになります。このガイドでは、パソコンでのスクリーンショットの撮り方と、サポートチケットに直接アップロードする方法を解説します。

## スクリーンショットの作成

スクリーンショットを安全に共有するベストな方法は、端末上でローカルに作成し、そのままサポートチケットにドラッグ＆ドロップでアップロードすること。これにより、外部リンクに頼らずプライバシーを守りながら安全に共有できます。



### Windows：Snipping Tool（スニッピングツール）

Windowsには標準で*Snipping Tool*が搭載されており、画面の一部やウィンドウ、全画面をキャプチャしてローカルに保存できます。

1. スタートメニューで**Snipping Tool**を検索して起動。
2. **新規作成**をクリックしてスクリーンショットを開始。
3. キャプチャしたい範囲を選択。
4. スクリーンショットをわかりやすい場所に保存。
5. 保存した画像ファイルをサポートチケットの画面にドラッグ＆ドロップでアップロード。

### Linux：ScreenshotツールまたはFlameShot

多くのLinuxディストリビューションには標準の*Screenshot*ツールが付属しています。さらに、高機能な*FlameShot*などのサードパーティ製ツールもおすすめです。

1. ScreenshotツールかFlameShotを起動。
2. キャプチャしたい範囲やウィンドウを選択。
3. スクリーンショットをローカルに保存。
4. 保存した画像ファイルをサポートチケットにドラッグ＆ドロップでアップロード。

## スクリーンショットのサポートへのアップロード

スクリーンショットをローカルに保存したら、ZAP-Hostingのパネルやサポートポータルでサポートチケットを開きます。

- スクリーンショットファイルをチケットのメッセージ欄にドラッグ＆ドロップ。
- アップロードが完了するまで待機。
- スクリーンショット付きでチケットを送信。

この方法なら外部リンクを共有せずに済み、データの安全性もバッチリです。

:::info 安全なスクリーンショット共有のポイント
スクリーンショットを公開したり、第三者のアップロードサービスを使って共有するのは避けましょう。もし個人情報や機密情報が含まれている場合は、アップロード前に必ずぼかしや編集で隠してくださいね。
:::

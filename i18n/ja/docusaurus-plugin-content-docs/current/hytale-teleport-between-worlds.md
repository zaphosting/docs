## はじめに

Hytaleサーバーでのワールド間テレポートは、プレイヤーが異なるワールド環境間をシームレスに移動できる機能です。この機能は、ハブ、アドベンチャーマップ、クリエイティブエリア、または別々のゲームプレイゾーンなど、複数のワールドをホストするサーバーでよく使われます。

:::info 早期アクセスのお知らせ

Hytaleは2026年1月13日にリリースされ、現在は早期アクセス版です。ゲームはまだ開発段階にあるため、サーバーソフトウェア、設定ファイル、モッディングサポート、インストール手順は今後も変更される可能性があります。

:::

## テレポート方法

ワールド間テレポートコマンドは、サーバーのコンソールまたはゲーム内チャットでオペレーターが実行できます。コンソールから実行すると、指定したプレイヤーに即座に効果が適用されます。ゲーム内で実行する場合は、コマンドを発行するプレイヤーがオペレーター権限または同等の権限を持っている必要があります。

現在のワールドから別のワールドへプレイヤーをテレポートするには、以下のコマンドを使います：

```
/world teleport <playername> <worldname>
```

`<playername>` はテレポートさせたいプレイヤー名を指定します。 `<worldname>` は移動先のワールド名を指定します。

ワールド間のテレポートには、移動先のワールドがサーバー上で利用可能である必要があります。対象のワールドが現在ロードされていない場合、テレポートコマンドは成功しません。ワールドをテレポート可能にするには、以下のコマンドでロードしてください：

```
/world load <worldname>
```


:::tip ワールド名の確認

ワールド名がわからない場合は、`/world list` コマンドで利用可能なワールドの一覧を表示できます。 
:::


## まとめ

ワールド間テレポートは、マルチワールドのHytaleサーバー管理において強力なツールです。コンソールやゲーム内チャットから適切なワールドコマンドを使い、移動先のワールドがロードされていることを確認することで、管理者は異なるサーバー環境間のスムーズで柔軟な移動を提供できます。

ご不明点やサポートが必要な場合は、いつでもお気軽にサポートチームまでご連絡ください！毎日対応しています 🙂

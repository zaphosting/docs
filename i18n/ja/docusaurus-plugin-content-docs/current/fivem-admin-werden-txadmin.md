# txAdminでFiveMの管理者になる方法

## はじめに  
このガイドでは、txAdminを使ってFiveMサーバーの管理者になる方法を解説するよ。txAdminのインターフェースを使って管理者権限を設定する手順をしっかり押さえよう。

## 必要条件  
- ZAP-HostingのウェブインターフェースまたはパネルからFiveMゲームサーバーにアクセスできること  
- FiveMゲームサーバーにtxAdminがインストールされて稼働していること  
- FiveMゲームサーバー管理の基本的な知識があること  

## ステップバイステップガイド

### 1. txAdminパネルにアクセス  
1. ZAP-Hostingアカウントにログイン。  
2. FiveMゲームサーバーのダッシュボードに移動。  
3. **Open txAdmin** ボタンをクリックしてtxAdminインターフェースを起動。  
   
   ![Open txAdmin](img/fivem-admin-txadmin/open-txadmin.png)  

### 2. txAdminにログイン  
1. 初めてtxAdminにアクセスする場合は、管理者アカウントの作成を求められるよ。  
2. 好きなユーザー名とパスワードを入力して保存しよう。  
3. すでにアカウントがある場合は、そのままログイン情報を入力してログイン。  

### 3. 管理者権限を追加  
1. txAdminのダッシュボードで **Players** タブに移動。  
2. Steam IDかゲーム内の名前で自分のプレイヤーを検索。  
3. プレイヤーの横にある **Manage** ボタンをクリック。  
4. 権限セクションで、自分に **admin** ロールや必要な権限を割り当てよう。  
   
   ![Assign Admin Role](img/fivem-admin-txadmin/assign-admin-role.png)  

### 4. 権限の詳細設定（オプション）  
1. txAdminの **Settings** タブに移動。  
2. **Permissions** を選択して、管理者が使えるコマンドや権限を細かく調整。  
3. 必要に応じて設定を変更し、保存しよう。  

### 5. 保存してサーバーを再起動  
1. 権限を割り当てたら、**Dashboard** タブに戻る。  
2. **Restart Server** をクリックして変更を反映。  
   
   ```bash
   # またはZAP-Hostingパネルから再起動も可能：
   # 1. サーバーダッシュボードに移動
   # 2. 「Restart」をクリック
   ```  

### 6. ゲーム内で管理者権限を確認  
1. FiveMゲームサーバーに参加。  
2. ゲーム内コンソール（通常はF8キー）を開く。  
3. 以下の管理者コマンドを試してみよう：  
   
   ```plaintext
   /admin
   /kick [playerID]
   /ban [playerID]
   ```  
   
4. コマンドが正常に実行できれば、管理者権限が付与されている証拠だよ。  

> **注意:** 問題があれば、Steam IDやtxAdminのサーバーホワイトリスト設定を再確認してね。

## まとめ  
この手順で、txAdminを使ってFiveMゲームサーバーの管理者権限を無事に設定できたはず。これで管理者としてサーバーを効率よく運営できるよ！

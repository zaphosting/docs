## はじめに

自動サーバーメッセージは、ゲーム内でプレイヤーに重要な情報を直接伝えるシンプルかつ効果的な方法です。サーバールールの表示、イベント告知、役立つヒントの共有、定期的なリマインダーなど、管理者が手動で入力しなくても自動で配信できます。

**スケジュールタスク**を使うことで、あらかじめ設定したコマンドを繰り返し実行し、**Unturned**のゲーム内チャットに自動でメッセージを送信できます。

## 自動メッセージの仕組み

自動メッセージは、指定した時間や間隔でサーバーコマンドを実行するタスクをスケジューリングして作成します。コマンドはチャットメッセージの送信、ブロードキャスト、サーバー通知などに使えます。

設定が完了すると、タスクはバックグラウンドで自動的に動作し、接続中の全プレイヤーにメッセージを表示します。スタッフが常にオンラインでなくても、安定した情報発信が可能です。



## 自動メッセージの作成方法

自動サーバーメッセージを作成するには、ゲームサーバーのダッシュボードを開き、**スケジュールタスク**のセクションに移動します。新しいスケジュールタスクを作成し、**コマンド実行**のオプションを選択してください。コマンド欄に、ゲームサーバーが対応しているメッセージ送信コマンドを入力します。

```
<message command> <your message text>
```

`<message command>`はメッセージ送信用のゲームコマンドや拡張機能のコマンドに置き換え、`<your message text>`はプレイヤーに表示したいメッセージテキストに置き換えてください。

メッセージの送信間隔も設定可能で、数分ごと、毎時、特定の時間帯など自由に指定できます。スケジュールタスクの詳しい作成方法は[スケジュールタスク](gameserver-scheduled-tasks.md)ガイドを参照してください。

## メッセージの間隔カスタマイズ

スケジュールタスクは、メッセージの表示タイミングや頻度を自由にコントロールできます。よくある使い方は：

- 数分ごとにサーバールールを表示
- 固定時間にイベント告知を配信
- 新規・復帰プレイヤー向けにゲームプレイのヒントを送信
- Discordやウェブサイトのリンクをリマインド

間隔はサーバーを再起動せずにいつでも変更できるので、サーバーの成長に合わせて柔軟に調整可能です。



## 注意点

自動サーバーメッセージの機能は、ゲームや利用可能なコマンドの仕様に依存します。基本的なスケジュールメッセージは標準のサーバーコマンドで実装できますが、フォーマットや条件分岐、高度なカスタマイズには制限がある場合があります。

もしゲームがプラグインやMOD、拡張機能をサポートしていて、より柔軟で多機能なメッセージシステムが利用できるなら、そちらを使うのもおすすめです。専用のメッセージングシステムは、標準コマンドよりリッチな機能を提供し、複雑なカスタムサーバーに適しています。



## まとめ

自動サーバーメッセージは、手間なくプレイヤーに情報を届けて盛り上げる信頼できる方法です。スケジュールタスクでメッセージコマンドを実行することで、サーバー全体で安定したコミュニケーションを実現できます。

一度設定すれば自動で動作し、ゲーム固有のメッセージコマンドやテキストを変えるだけで簡単に再利用可能です。

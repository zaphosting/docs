## アカウント作成
ZAP-Hostingのアカウントを作るのは、すべてのサービスや機能にアクセスして管理するための第一歩。登録は超簡単で、必要な情報も少しだけ。

登録を始めるには、ZAP-Hostingのサイトにアクセスしてスタートページへ。ログインエリアにある**サインアップ！**ボタンをクリックすると、登録用のポップアップウィンドウが開きます。

![img](https://screensaver01.zap-hosting.com/index.php/s/5b6Wzc27CcHW3Zx/download)


ポップアップ内で以下の情報を入力してください：

- ユーザー名
- メールアドレス
- パスワード

情報を入力したら、セキュリティキャプチャを確認します。これは自動登録を防ぐためのステップです。すべての項目を埋めてキャプチャをクリアしたら、登録フォームを送信してアカウントを作成しましょう。

![img](https://screensaver01.zap-hosting.com/index.php/s/k6P3wSKWWRRiYax/preview)

## アカウントの有効化

登録フォームを送信すると、入力したメールアドレスに有効化メールが届きます。このメールにはアクティベーションコードが含まれていて、アカウント設定を完了するために確認が必要です。

有効化メールの到着には数分かかることがあります。受信トレイに見当たらない場合は、迷惑メールフォルダもチェックしてみてください。

:::warning 有効化なしだと機能制限あり
登録後すぐにログインはできますが、すべての機能やサービスを使うには有効化メールの確認が必須です。
:::

## 🔐 ZAP-Hostingに登録しよう
めっちゃ簡単！まずはウェブサイトにアクセスしてスタートページへ。そこでユーザー名、メールアドレス、パスワードを入力するログイン画面が見つかるよ。

![](https://screensaver01.zap-hosting.com/index.php/s/bLBnpoAWESigiK7/preview)

:::info
登録後すぐにログインできるけど、フル機能を使うにはアクティベーションメールの確認を忘れずにね。
:::

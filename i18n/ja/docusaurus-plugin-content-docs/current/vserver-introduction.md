## 構成内容

VPSにはいろんな構成オプションがあって、どんな使い方にも合うように用意してるよ。基本的には以下のような選択肢があるよ：

| コンポーネント | 構成オプション |
| -------------------------------- | ------------------------- |
| **CPU** | AMD EPYC 7000シリーズ |
| **CPUコア数**              | 4〜64コア |
| **RAM**              | 4〜128 GB RAM     |
| **ディスク容量**               | 25〜600 GB                                      |
| **OS** | Debian (11,12,13)、Ubuntu (20.04,22.04,24.04)、Windows Server 2022/2025、カスタムISOもOK |
| **IPアドレス数** | 1〜10個  |
| **帯域幅**     | 上り/下り 1,000〜5,000 Mbit/s |
| **トラフィック**                      | 月間33 TB       |



## 利用シーン

コスパ良し、パフォーマンス良し、アクセス権限も充実でできることが多いから、VPSはこんな用途にピッタリ：

- ゲームサーバーやボイスサーバー
- その他いろいろ！

## 構成内容

VPS製品には、あらゆる用途に合うように様々な構成オプションを用意してるよ。基本的には以下のような選択肢があるよ：

| コンポーネント | 構成オプション |
| -------------------------------- | ------------------------- |
| **CPU** | AMD EPYC 7000シリーズ |
| **CPUコア**              | 4〜64コア |
| **RAM**              | 4〜128 GB RAM     |
| **ディスク容量**               | 25〜2000 GB                                   |
| **OS** | Debian (11,12,13)、Ubuntu (20.04,22.04,24.04)、Windows Server 2022/2025、カスタムISOもOK |
| **IPアドレス** | 1〜10個のIPアドレス  |
| **帯域幅**     | 上り/下り 1,000〜5,000 Mbit/s |
| **トラフィック**                      | 月間33 TB       |

## 製品管理

**製品管理** セクションでは、ボイスボットの管理に必要なすべてのページが見つかるよ。ボイスボット管理のオプションを詳しく下にまとめているよ。

### ダッシュボード

ボイスボットの**ダッシュボード**は、中央管理の概要画面として機能するよ。オンライン状態、ログイン情報、外部ウェブインターフェースへのリンク、ディスク使用量など、ボイスボットに関する基本かつ重要な情報をわかりやすく表示しているよ。

![img](https://screensaver01.zap-hosting.com/index.php/s/YnXaQDPywcoxFPP/preview)

### ウェブインターフェース

ボイスボットのほとんどの機能は外部のウェブインターフェースから利用できるよ。このインターフェースでは、好きな音楽ソースを選んだり、ボット改善の提案を送ったりできる。ウェブインターフェースのパスワード変更やボイスボットの再インストールもここから可能だよ。

![img](https://screensaver01.zap-hosting.com/index.php/s/cmKrJ2YpzmZcm2L/preview)

### サーバー

ボットを使う前に、どのTeamSpeakサーバーで動かすかを設定する必要があるよ。サーバーのIPアドレスとポートを指定し、ボットの名前は自由に決められる。もしサーバーがパスワードで保護されている場合は、そのパスワードも入力してボットがアクセスできるようにしよう。

![img](https://screensaver01.zap-hosting.com/index.php/s/sbbRr9f5HexcMsG/preview)

### 詳細設定

ユーザーを自動で移動させたり、ユーザーがサーバーに参加したときにウェルカムメッセージを流したりといった高度な機能は、ここで設定をオンにして使えるよ。これらの機能で、よりパーソナライズされた自動化されたサーバー管理が可能になるんだ。

![img](https://screensaver01.zap-hosting.com/index.php/s/yz94fyZssG53Rjd/preview)

## さらに役立つリンク
**さらに役立つリンク** セクションでは、製品に関連する追加のリンクをまとめているよ。

### よくある質問
**よくある質問** セクションでは、お客様からよく寄せられる質問をまとめているよ。ここで役立つ回答や詳しい情報を見つけて、素早く効率的に疑問を解消しよう。

### ZAP-Hostingドキュメント
当社の製品に関する充実したドキュメントを用意しているよ。これはよくある質問や悩みのナレッジベースとして役立つんだ。さまざまなガイドやヘルプが揃っていて、製品の使い方や管理に役立つよ。

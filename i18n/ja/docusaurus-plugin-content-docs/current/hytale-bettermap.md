## はじめに

広大で複雑なワールドを探索するのはHytaleの醍醐味のひとつ。ワールドが大きくなり、スポーン地点から遠くへ移動するほど、探索済みエリアや注目ポイント、ルートの管理がどんどん難しくなっていきます。

![img](https://screensaver01.zap-hosting.com/index.php/s/WGaiERKfa3EFy48/preview)

**BetterMap**モッドは、永続的に保存されるマップデータ、カスタマイズ可能なズームレベル、ウェイポイント、そしてパフォーマンス最適化されたレンダリングを備えた高度なゲーム内マップを提供し、ワールド探索を強化します。記憶や外部ツールに頼らず、BetterMapを使えばもっと効率的に世界をナビゲートし、マップ情報を仲間と共有できます。

## BetterMapがHytaleに加えるもの

BetterMapは標準の探索体験を拡張し、フル機能のマッピングシステムを導入します。探索したエリアは自動的に保存され、セッションをまたいでも利用可能。時間をかけて信頼できるワールドの全体像を作り上げられます。

マップはズームレベルを調整でき、詳細なローカルビューと広域の周辺地域ビューを簡単に切り替え可能。長距離の旅の計画や特定のバイオームや建造物の位置特定に超便利です。

ウェイポイントは拠点、村、ダンジョン、資源豊富な場所など重要なロケーションをマークできます。これにより方向感覚が向上し、無駄な移動時間を減らせます。

さらにBetterMapは共有マッピング機能もサポート。マルチプレイヤーサーバーではマップデータをプレイヤー間で共有でき、チームでの協力や探索済みワールドの共通理解がグッと深まります。



## BetterMapのインストール

BetterMapはCurseForge経由で配布されているモッドです。モッドは当社のModsインストーラーかFTP経由でインストール可能。詳しくは[モッドのインストール](hytale-mods.md)ガイドをチェックしてください。

インストール後はHytaleサーバーを再起動。ロードが完了するとゲーム内でマップインターフェースが使えるようになり、探索エリアの記録が自動で始まります。


## BetterMapの使い方

ゲーム内に入ると、BetterMapは視覚的なマップインターフェースと、マップの動作やウェイポイントを直接操作できるコマンドを提供します。

設定されたキーでマップインターフェースを開くと、探索済みの地形がリアルタイムで表示されます。ズームイン・アウトで詳細なローカルビューと広域ビューを切り替えられ、ナビゲーションや計画に役立ちます。

また、BetterMapはゲーム内チャットから実行できるコマンドも用意。フルインターフェースを開かずにマップやウェイポイントを素早く管理したい時に便利です。




## ウェイポイント管理

ウェイポイントはBetterMapの超強力な機能のひとつで、マップインターフェースとコマンドの両方で管理可能。現在地にウェイポイントを作成するには以下を使います：

```
/bettermap waypoint add <name>
```
これで現在の座標と指定した名前で新しいウェイポイントが作成されます。

![img](https://screensaver01.zap-hosting.com/index.php/s/99KXn7XmSbQSmNy/preview)


既存のウェイポイントを削除するには：
```
/bettermap waypoint remove <name>
```

すべてのウェイポイントを一覧表示するには：
```
/bettermap waypoint list
```

ウェイポイントは編集してマップを整理しやすくできます。多くのサーバーでは色やアイコンのカスタマイズも可能で、モッドのバージョンによってはマップインターフェースかコマンドパラメータで設定できます。

```
/bettermap waypoint edit <name> color red
```

ウェイポイントマネージャーを使えば、個別のウェイポイントの有効・無効切り替え、名前変更、古くなった場所の削除も簡単。ワールドの変化に合わせて柔軟に管理できます。

マルチプレイヤーサーバーでは共有ウェイポイントでチームの探索を連携。あるプレイヤーが発見した場所を他のメンバーと共有し、重要エリアの共同マップを作れます。

## まとめ

BetterMapは保存されたマップデータ、ズーム可能なビュー、ウェイポイント、共有マッピング、効率的なレンダリングを組み合わせて、Hytaleのワールド探索を大幅にパワーアップ。BetterMapを使えば、方向感覚が向上し、計画も立てやすくなり、シングルプレイでもマルチプレイでももっと楽しく探索できます。

## はじめに

Hytaleのゲームサーバーで権限を管理することで、プレイヤーができること・できないことをコントロールできるよ。信頼できるプレイヤーやモデレーター、管理者にそれぞれ違うアクセスレベルを割り当てることが可能。権限設定を正しく行うことは、不正利用や悪用を防ぎつつ、みんなが安心してゲームを楽しめるバランスの取れた安全なサーバー環境を維持するために超重要だよ。

Hytaleのサーバーソフトは階層的な権限レベルをサポートしていて、プレイヤーごとに実行できるコマンドやアクションを決められる。サーバーの設定によっては、ライブコンソールや設定ファイルから管理できるよ。

## 権限レベルの概要

権限はプレイヤーがサーバー上でどんな行動ができるかを決めるもの。基本的には通常プレイヤーは移動や世界とのインタラクト、チャットなどの標準的なゲームプレイ権限だけを持っている。オペレーターや管理者などの上位権限になると、ゲームプレイや他プレイヤー、サーバー設定、モデレーションツールに影響を与えるコマンドを使えるようになるよ。

権限システムは主に2つの要素で構成されている：

- **ユーザー権限**：特定のプレイヤーに直接適用される権限
- **グループ権限**：複数のプレイヤーにまとめて割り当てられる権限

プレイヤーは内部的にUUIDで識別されていて、コマンドで権限を管理する際にこのUUIDが必要になるよ。



## ユーザー権限の管理

ユーザー権限は特定のプレイヤーに対して個別に権限を付与・剥奪できるよ。

### ユーザー権限の確認

指定したユーザーに直接割り当てられているすべての権限を表示するコマンド。現在そのユーザーに適用されている権限が全部見れるよ。

```
/perm user list <uuid>
```



### ユーザーに権限を追加

特定のユーザーに1つ以上の権限を直接付与するコマンド。指定した権限は即時反映され、サーバー再起動は不要だよ。

```
/perm user add <uuid> <permissions>
```



### ユーザーから権限を削除

ユーザーから指定した権限だけを剥奪するコマンド。他の権限はそのまま残るよ。

```
/perm user remove <uuid> <permissions>
```



## ユーザーのグループ割り当て管理

直接の権限に加えて、ユーザーはグループを通じて権限を継承できるよ。

### ユーザーの所属グループ確認

ユーザーが現在どの権限グループに所属しているかを表示するコマンド。

```
/perm user group list <uuid>
```



### ユーザーをグループに追加

ユーザーを権限グループに割り当てるコマンド。追加すると、そのグループに設定されたすべての権限を継承するよ。

```
/perm user group add <uuid> <group>
```



### ユーザーをグループから削除

ユーザーを権限グループから外すコマンド。外すと、そのグループの権限はもう継承されなくなるよ。

```
/perm user group remove <uuid> <group>
```



## グループ権限の管理

グループは権限を一括管理して複数ユーザーに使い回せる便利な仕組み。

### グループ権限の確認

指定したグループに割り当てられているすべての権限を表示するコマンド。グループの権限全体を把握できるよ。

```
/perm group list <group>
```



### グループに権限を追加

グループに1つ以上の権限を追加するコマンド。このグループに所属する全ユーザーが即座に新しい権限を継承するよ。

```
/perm group add <group> <permissions>
```



### グループから権限を削除

グループから指定した権限だけを削除するコマンド。他の権限には影響しないよ。

```
/perm group remove <group> <permissions>
```

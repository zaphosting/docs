## Criar conta
Criar uma conta na ZAP-Hosting é o primeiro passo para acessar e gerenciar todos os serviços e recursos. O processo de registro é rápido e só precisa de alguns dados.

Para começar o registro, acesse o site da ZAP-Hosting e vá para a página inicial. Lá você vai encontrar a área de login com um botão **Cadastrar-se!**. Clique em **Cadastrar-se!** para abrir a janela popup de registro.

![img](https://screensaver01.zap-hosting.com/index.php/s/5b6Wzc27CcHW3Zx/download)


Na janela popup, você deve informar os seguintes dados:

- Nome de usuário
- Endereço de e-mail
- Senha

Depois de preencher seus dados, confirme o captcha de segurança. Essa etapa é necessária para proteger contra registros automáticos. Quando todos os campos estiverem preenchidos e o captcha confirmado, envie o formulário para criar sua conta.

![img](https://screensaver01.zap-hosting.com/index.php/s/k6P3wSKWWRRiYax/preview)

## Ativação da conta

Após enviar o formulário de registro, um e-mail de ativação será enviado para o endereço informado. Esse e-mail contém um código de ativação que deve ser confirmado para finalizar a criação da conta.

O envio do e-mail de ativação pode levar alguns minutos. Se a mensagem não aparecer na sua caixa de entrada, confira também a pasta de spam ou lixo eletrônico.

:::warning Funcionalidade limitada sem ativação
Você pode fazer login logo após o registro. Mas para acessar todos os recursos e serviços, é preciso confirmar o e-mail de ativação.
:::

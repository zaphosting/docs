## Introdução

Elementor é um construtor de páginas poderoso e fácil de usar para WordPress que permite criar sites profissionais sem precisar saber nada de programação.

:::info
O Elementor está disponível em versões gratuita e paga. Com a versão gratuita do Elementor (Elementor Core), você já tem tudo que precisa para começar seu site. O Elementor Pro adiciona à versão gratuita funções extras e widgets Pro. Você pode conferir as diferenças exatas na documentação oficial do **[Elementor](https://elementor.com/help/elementor-pro-vs-free/)**.
:::

## Instalação

Para usar e instalar o Elementor, você precisa ter o WordPress já instalado e rodando na sua hospedagem de sites. Se ainda não tiver, esse é o primeiro passo. Se não souber exatamente como fazer isso, recomendamos dar uma olhada no nosso **[guia Instalar WordPress](webspace-wordpress.md)**.

Assim que o WordPress estiver funcionando, você pode começar a instalar o Elementor. Ele é instalado como um plugin no WordPress. Saiba como instalar plugins no WordPress no nosso **[guia Plugins WordPress](webspace-wordpress-plugins.md)**.

No WordPress, vá em Plugins, clique no botão **Adicionar novo plugin** no topo da página. Na próxima tela, pesquise por Elementor e clique em instalar na opção **Elementor Website Builder - More than just a page builder**.

![img](https://screensaver01.zap-hosting.com/index.php/s/xcj9stZtAmY9cgJ/preview)



## Configuração

Com a instalação concluída, você pode seguir para a configuração. Na próxima etapa, você verá a janela de boas-vindas do Elementor. Ela vai te guiar pelos passos: criar uma conta Elementor, instalar o tema “Hello”, escolher as funções desejadas e, por fim, a confirmação “Good to go”. É só seguir as instruções para configurar tudo certinho.

![img](https://screensaver01.zap-hosting.com/index.php/s/6QN5trndZgfSano/download)

Criar uma conta Elementor não é obrigatório, a não ser que você queira usar o Elementor Pro direto. Depois, é só instalar o tema e adicionar as funções extras que precisar.

![img](https://screensaver01.zap-hosting.com/index.php/s/YgXwPiEnBZTQsC4/preview)



## Começando

Agora você pode começar a criar seu site com a ajuda do Elementor. Primeiro, vale a pena entender direitinho como usar o Elementor, quais widgets estão disponíveis, como editar o design e layout, e como criar páginas e templates personalizados.

### Widgets Disponíveis

Tanto a versão gratuita quanto a paga do Elementor oferecem vários **widgets** para você personalizar seu site.

#### Widgets Básicos
A versão gratuita do Elementor oferece uma seleção de widgets para você criar e personalizar seu site sem custo extra:

- Inner Section, Heading, Image, Text Editor, Video, Button, Star Rating, Divider, Google Maps, Icon, Image Box, Icon Box, Basic Gallery, Image Carousel, Icon List, Counter, Spacer, Testimonial, Tabs, Accordion, Toggle, Social Icons, Progress Bar, Sound Cloud, Shortcode, HTML, Menu Anchor, Alert, Sidebar, Text Path, Container e Link in Bio.

#### Widgets Pro

Leve o design do seu site para o próximo nível com widgets Pro adicionais. A versão Elementor Pro inclui os seguintes widgets extras:

- Posts, Share Buttons, Portfolio, Slides, Form, Login, Nav Menu, Animated Headline, Price Table, Price List, Gallery, Flip Box, Call to Action, Media Carousel, Testimonial Carousel, Nested Carousel, Loop Carousel, Table of Content, Countdown, Facebook Page, Blockquote, Template, Reviews, Facebook Button, Facebook Embed, Facebook Comments, PayPal Button, Lottie Widget, Code Highlight, Video Playlist, Hotspot, Progress Tracker, Stripe Button, Mega Menu, Off-Canvas






## Design e Layout

### Criar Página

Crie uma nova página e personalize do jeito que quiser com o **Elementor**. Para isso, abra o **Elementor** na barra lateral e selecione **"Criar Página"**. O editor oferece vários **widgets e opções de configuração** para você customizar o design e o conteúdo com flexibilidade.

![img](https://screensaver01.zap-hosting.com/index.php/s/mdMbnXNkngXWJHt/download)

### Templates

Crie templates para **containers, páginas ou seções** que podem ser usados várias vezes. A grande vantagem é que qualquer alteração feita no template atualiza automaticamente todas as áreas onde ele é usado. Isso economiza tempo e garante um design padronizado para seu site.

![img](https://screensaver01.zap-hosting.com/index.php/s/mdMbnXNkngXWJHt/download)

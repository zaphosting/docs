## Introdução
Não sabe como conectar ao seu **Subnautica 2** ou quais informações são necessárias para começar? Este guia mostra como obter os dados do servidor, estabelecer a conexão e resolver problemas comuns de conexão.

:::info Aviso de Acesso Antecipado
Subnautica 2 pode ainda estar em desenvolvimento ativo dependendo da fase atual do lançamento. Por isso, recursos multiplayer, comportamento do navegador de servidores e fluxos de conexão podem mudar com o tempo.
:::

## Obtenha os dados do servidor

Você precisará de todas as informações relevantes para conectar ao seu servidor. Os dados do seu servidor estão disponíveis no dashboard do painel web do seu servidor de jogos. Informações importantes incluem:

- Endereço IP e porta
- Nome do servidor
- Senha do servidor

Com essas informações, você poderá identificar seu servidor e garantir que está conectando ao correto.

| Dado | Descrição |
| --- | --- |
| `Endereço IP` | O endereço público do seu servidor de Subnautica 2 |
| `Porta` | A porta do jogo usada para a conexão |
| `Nome do servidor` | O nome exibido no jogo se houver lista de servidores ou navegador de sessões |
| `Senha do servidor` | Necessária apenas se seu servidor estiver protegido por senha |

## Estabeleça a conexão com o servidor

A forma exata de conectar a um servidor de **Subnautica 2** depende das opções multiplayer disponíveis na versão do jogo que você está usando. Em geral, você precisará do **endereço IP** e da **porta** do servidor, e em alguns casos também da **senha do servidor**.

### Entrar usando o menu multiplayer do jogo

Inicie o **Subnautica 2** e abra a seção multiplayer ou de conexão a servidor no menu principal. Se o jogo oferecer uma opção de conexão direta, insira o `endereço IP` e a `porta` do seu servidor exatamente como aparecem no painel web da ZAP-Hosting. Se houver senha configurada, digite a `senha do servidor` quando solicitado.

Após confirmar os dados, o jogo tentará estabelecer a conexão com seu servidor.

:::note Disponibilidade da Conexão Direta
A nomenclatura exata dos menus e botões pode variar conforme a versão atual do jogo. Se não encontrar a opção de conexão direta, verifique se o jogo exige entrar via lista de servidores, sistema de convite ou navegador de sessões.
:::

### Entrar usando lista de servidores ou navegador de sessões

Se o **Subnautica 2** usar um navegador de servidores embutido em vez de um campo de conexão direta, procure seu servidor pelo `nome do servidor` configurado. Por isso, é importante conferir o nome do servidor no painel web antes de tentar conectar.

Quando encontrar o servidor correto na lista:

1. Selecione a entrada do servidor
2. Verifique se o nome exibido corresponde ao seu servidor
3. Entre no servidor
4. Digite a senha se for solicitada

:::tip Use Nomes Únicos para Servidores
Se seu servidor não aparecer fácil de identificar na lista, defina um nome claro e único na configuração do servidor. Isso facilita encontrar a entrada correta e evita conectar ao servidor errado.
:::

### Se o servidor não aparecer imediatamente

Se seu servidor não aparecer no navegador do jogo, isso nem sempre significa que ele está offline. Às vezes, as listas de servidores demoram para atualizar ou o jogo só mostra servidores após inicialização bem-sucedida.

Verifique o seguinte:

- O servidor está online no painel web da ZAP-Hosting
- A porta do jogo está correta
- O servidor foi totalmente iniciado
- O nome do servidor está configurado corretamente
- A senha foi digitada se for necessária

## Problemas comuns e soluções

A conexão a um servidor de jogos pode ser afetada por vários fatores. Os problemas e soluções mais comuns estão descritos abaixo.

### Servidor não visível

A falta de visibilidade do servidor pode ocorrer se a inicialização não foi concluída com sucesso. Isso pode ser causado por configuração incorreta ou arquivos corrompidos. Informações adicionais geralmente estão disponíveis no console do servidor ou nos arquivos de log.

:::caution Servidor Precisa Estar Iniciado
Não tente conectar antes que o servidor tenha concluído seu processo de inicialização. Se o servidor ainda estiver carregando ou reiniciando, jogadores podem não conseguir conectar ou o servidor pode não aparecer em nenhuma lista.
:::

### Dados de conexão incorretos

Se não conseguir conectar, verifique se está usando o `endereço IP`, `porta` e `senha do servidor` corretos. Mesmo um pequeno erro de digitação pode impedir a conexão.

| Verificação | O que conferir |
| --- | --- |
| `Endereço IP` | Confere com o endereço mostrado no dashboard do servidor |
| `Porta` | Confere com a porta do jogo atribuída |
| `Senha` | Digitada exatamente como configurada, incluindo maiúsculas e minúsculas |
| `Nome do servidor` | Confere com o nome configurado se estiver buscando no navegador |

### Servidor está online, mas conexão falha

Se o servidor parece estar online mas você ainda não consegue conectar, revise o console e os logs do servidor no painel web. Erros podem indicar se o servidor falhou ao carregar ou se há problemas de configuração bloqueando conexões.

:::tip Verifique os Logs Primeiro
O console e os arquivos de log do servidor são geralmente a forma mais rápida de identificar o motivo da falha na conexão. Eles ajudam a confirmar se o servidor está ouvindo corretamente e se as tentativas de conexão estão chegando.
:::

### Sem solução ou ajuda insuficiente

Ainda com problemas após tentar resolver? Nosso time de suporte está disponível diariamente para ajudar. Basta abrir um **[Ticket](https://zap-hosting.com/en/customer/support/)** no nosso site e explicar seu problema com o máximo de detalhes possível. Inclua o IP do servidor, porta e uma breve descrição do que acontece ao tentar conectar.

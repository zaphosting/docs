## Step-by-Step Guide

### Step 1: Access the ZAP-Hosting Panel

1. Open your preferred web browser.
2. Navigate to the [ZAP-Hosting Panel](https://panel.zap-hosting.com).
3. Log in using your username and password.

![](https://screensaver01.zap-hosting.com/index.php/s/SCREENSHOT_PLACEHOLDER/preview)

### Step 2: Locate Test 2 Service

1. After logging in, go to the **Services** section in the main menu.
2. Use the search bar or browse to find the **Test 2** product.
3. Click on the **Test 2** item to open its management page.

### Step 3: Configure Test 2 Settings

Adjust the following configuration options according to your needs:

| Setting          | Description                      | Default Value | Recommended Value      |
|------------------|--------------------------------|---------------|-----------------------|
| Option A         | Enables feature A               | Disabled      | Enabled               |
| Option B         | Sets the operational mode       | Mode 1        | Mode 2                |
| Timeout Duration | Time before automatic reset    | 30 minutes    | 15 minutes            |

To modify these settings:

1. Click **Edit Configuration**.
2. Change the values as needed.
3. Save the changes by clicking **Apply**.

:::tip
Refer to the [ZAP-Hosting Configuration Guide](configuration-guide.md) for detailed explanations of each option.
:::

### Step 4: Start the Test 2 Process

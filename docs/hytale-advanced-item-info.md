## What Advanced Item Info provides

Advanced Item Info adds a dedicated GUI that displays enriched information for items across the game. This includes:

- Item identification and unique ID  
- Visual icon and quality tier  
- Level requirements and item level  
- Maximum stack size and durability  
- Whether the item is fuel 
- Whether the item is consumable or has special properties  
- Whether the item functions as a tool, weapon, armor, glider, or utility component  
- Tooltip context for where the item can be crafted or obtained


## Using the mod

To install HyFixes, follow the standard mod installation procedure described in the [Install Mods](hytale-mods) guide. This guide explains where mods must be placed and how they are loaded by the server. Once installed, the user interface can be opened with the following in-game command:

```
/advancedinfo
```

When the GUI is open, players can type or paste item names to search, click or hover over results, and view the rich metadata provided by the mod.
